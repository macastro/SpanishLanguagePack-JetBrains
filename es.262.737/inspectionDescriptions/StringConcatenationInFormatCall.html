Informa de la concatenación de cadenas no constantes utilizadas como argumento de cadena de formato.

Aunque ocasionalmente es intencionado, esto suele ser un uso incorrecto de los métodos de formateo e incluso puede causar problemas de seguridad si las variables utilizadas en la cadena concatenada contienen caracteres especiales como %.

Además, a veces esto puede ser el resultado de concatenar erróneamente argumentos de formato de cadena al escribir + cuando se debería haber escrito ,.

Ejemplo:


  static String formatGreeting(String userName) {
    return String.format("Hello, " + userName);
  }

Aquí, userName se interpretará como parte de la cadena de formato, lo que podría provocar una IllegalFormatException (por ejemplo, si userName es "%") o consumir una gran cantidad de memoria (por ejemplo, si userName es "%2000000000%"). Esta llamada probablemente debería reemplazarse por String.format("Hello, %s", userName);.

Esta inspección comprueba las llamadas a los métodos de formateo de java.util.Formatter, java.lang.String, java.io.PrintWriter o java.io.PrintStream.
