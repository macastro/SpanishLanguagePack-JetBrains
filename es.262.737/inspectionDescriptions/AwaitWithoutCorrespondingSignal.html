Reporta las llamadas a Condition.await() para las cuales no se puede encontrar una llamada correspondiente a Condition.signal() o Condition.signalAll().

Llamar a Condition.await() en un hilo sin un Condition.signal() correspondiente puede causar que el hilo quede deshabilitado hasta que sea interrumpido o se produzca un "despertar espurio".

Esta inspección solo reporta las llamadas dirigidas a campos de la clase actual.

Ejemplo:


  class Queue {
    private final Condition isEmpty = ...;

    void add(Object elem) {
      // ...
      // isEmpty.signal();
      // ...
    }

    void remove(Object elem) throws InterruptedException {
      // ...
      isEmpty.await(); // 'await()' no contiene una llamada 'signal()'/'signalAll()' correspondiente
      // ...
    }
  }
