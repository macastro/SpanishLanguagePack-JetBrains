Informa sobre las clases cuyos nombres son demasiado cortos, demasiado largos o no siguen el patrón de expresión regular especificado.

Ejemplo: si la inspección está habilitada para pruebas y la longitud mínima especificada para el nombre de la clase es 8 (valor predeterminado), la siguiente clase de prueba generará una advertencia porque la longitud de su nombre es 6, que es menor que 8: public class MyTest{}.

La corrección rápida para renombrar este tipo de clases solo está disponible en el editor.

Configuración de la inspección:

Utilice la lista en la sección Opciones para especificar qué clases deben inspeccionarse. Desactive la casilla de verificación de las clases para las que desee omitir la inspección.

Para cada tipo de clase, utilice los campos de entrada proporcionados para especificar la longitud mínima, la longitud máxima y la expresión regular que debe tener el nombre de la clase. Especifique 0 en los campos de longitud para omitir la comprobación correspondiente.

Las expresiones regulares deben especificarse en el formato estándar de java.util.regex.
