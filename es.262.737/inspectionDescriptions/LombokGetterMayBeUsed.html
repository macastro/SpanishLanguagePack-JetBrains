Informa sobre los métodos getter estándar que pueden reemplazarse por la anotación @Getter de lombok.

Ejemplo:


  import java.util.Date;

  public class MyClass {
    /**
     * Fecha de procesamiento.
     */
    private Date processDate;

    /**
     * Devuelve la fecha.
     *
     * @return La fecha
     */
    public Date getProcessDate() {
      return processDate;
    }
  }

Después de aplicar la corrección rápida o limpieza:


  import lombok.Getter;
  import java.util.Date;

  @Getter
  public class MyClass {
    /**
     * Fecha de procesamiento.
     * -- GETTER --
     * Devuelve la fecha.
     *
     * @return La fecha
     */
    private Date processDate;
  }

Solo se informa si la biblioteca lombok está configurada. Para eliminar lombok, consulte el sitio web de lombok.
