Reporta condiciones de desigualdad que, según el análisis de flujo de datos, solo se cumplen para un único valor de operando. Estas condiciones pueden reemplazarse por condiciones de igualdad para que el código sea más claro.

Ejemplo:


  if (x >= 10) {
    ...
    if (x <= 10) { // puede reemplazarse por 'x == 10'
    }
  }
