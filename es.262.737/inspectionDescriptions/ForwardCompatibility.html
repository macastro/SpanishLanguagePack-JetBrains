Informa sobre los constructos de código Java que podrían no compilar en versiones futuras de Java.

Se informan los siguientes problemas:

  • Uso de assert, enum o _ como identificadores
  • Uso de los identificadores restringidos var, yield o record como nombres de tipos
  • Llamadas no calificadas al método yield
  • Modificadores en la sentencia requires java.base dentro de module-info.java
  • Puntos y coma redundantes entre sentencias import

Ejemplo:


  // Esta clase, antes válida, no compilará en Java 14
  // porque 'yield' se ha convertido en un identificador restringido.
  public class yield {}

Corregir estos problemas a tiempo puede simplificar la migración a versiones futuras de Java.
