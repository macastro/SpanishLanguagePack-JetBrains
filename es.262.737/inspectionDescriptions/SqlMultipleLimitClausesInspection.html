Informa sobre el uso de múltiples cláusulas de límite de filas en una sola consulta.

Ejemplo (Microsoft SQL Server):

create table foo(a int);
select top 1 * from foo order by a offset 10 rows fetch next 20 rows only;

La cláusula SELECT TOP se utiliza para especificar que solo se debe devolver 1 registro. La cláusula FETCH especifica el número de filas a devolver después de que se haya procesado la cláusula OFFSET. Sin embargo, dado que ya existe la cláusula de límite SELECT TOP, la cláusula FETCH puede ser redundante.
