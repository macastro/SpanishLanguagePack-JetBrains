Informa sobre las extensiones instanciadas por la plataforma IntelliJ que han sido declaradas como objetos de Kotlin.

El ciclo de vida de las extensiones es gestionado por la plataforma IntelliJ. El uso de objetos de Kotlin para el registro de extensiones puede provocar la creación de instancias de extensión innecesarias e impedir que el complemento se pueda descargar.

Ejemplo

Registro de la extensión:


<annotator language="myLang" implementationClass="com.example.MyAnnotator"/>

Implementación de la extensión:


// Incorrecto:
object MyAnnotator : Annotator {
  ...
}

// Correcto:
class MyAnnotator : Annotator {
  ...
}
