Volcar a Fuente de Datos DDL

Una fuente de datos DDL es una fuente de datos virtual con un esquema basado en varios scripts SQL. Almacenar estos archivos en el Sistema de Control de Versiones (VCS) es una forma de mantener tu base de datos bajo el VCS.

Para crear una fuente de datos DDL, haz clic derecho en una fuente de datos en el Explorador de Base de Datos y selecciona Importar/Exportar | Volcar a Fuente de Datos DDL.
