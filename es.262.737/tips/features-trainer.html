Prueba los cursos de entrenamiento interactivos

PyCharm incluye un Entrenador de Características del IDE integrado. Proporciona cursos de entrenamiento interactivos para ayudarte a aprender cómo realizar algunas tareas comunes, como refactorizar código, navegar por proyectos, hacer debug y trabajar con control de versiones.

Asegúrate de que el plugin IDE Features Trainer esté habilitado en la pestaña Instalados de la página Plugins en la configuración del IDE.

Para abrir la ventana de herramientas Aprender e iniciar tus ejercicios, selecciona View | Tool Windows | Learn o Help | Learn IDE Features desde el menú principal. La ventana de herramientas lista los tutoriales disponibles.
