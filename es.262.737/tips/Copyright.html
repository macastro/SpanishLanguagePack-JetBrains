Agregar avisos de copyright

En PyCharm, puedes configurar diferentes avisos de copyright para diferentes conjuntos de archivos. Para mayor comodidad, puedes usar variables. Por ejemplo, usa $today.year para mantener el año actualizado.

En el diálogo Configuración (&shortcut:ShowSettings;), selecciona Editor | Copyright | Perfiles de Copyright, crea un nuevo perfil y agrega el texto para tu aviso de copyright usando variables.
