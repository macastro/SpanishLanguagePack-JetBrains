Control de Versiones

PyCharm proporciona integración lista para usar con los siguientes sistemas de control de versiones: Git, Mercurial, Subversion y Perforce.

Versiona tu aplicación para colaborar en ella, así como para eliminar los riesgos de almacenar toda tu base de código localmente. Presiona &shortcut:ActivateVersionControlToolWindow; para ver el historial de cambios en el proyecto o presiona &shortcut:CheckinProject; para enviar tus cambios locales al repositorio.
