Buscar en Todas Partes

Presiona Shift dos veces para buscar archivos, acciones, símbolos, elementos de UI, branches de Git y comentarios en todo tu proyecto. Presionar doble Shift nuevamente, extenderá la búsqueda a elementos que no pertenecen al proyecto.

Usa pestañas o atajos directos &shortcut:GotoClass; para clases, &shortcut:GotoFile; para archivos, &shortcut:GotoSymbol; para símbolos, y &shortcut:GotoAction; para acciones para filtrar los resultados de búsqueda.
