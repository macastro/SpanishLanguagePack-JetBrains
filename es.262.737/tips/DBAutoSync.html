Sincronización del estado de la base de datos

Para sincronizar el estado de la base de datos automáticamente, en el Explorador de Base de Datos (View | Tool Windows | Database Explorer), haz clic en el botón Propiedades del Origen de Datos () y selecciona el origen de datos que deseas cambiar. En la pestaña Opciones, selecciona la casilla Sincronización automática bajo Introspección.

Si la casilla Sincronización automática está desmarcada, la vista del origen de datos en el Explorador de Base de Datos (View | Tool Windows | Database Explorer) se sincroniza con el estado real de la base de datos solo cuando haces clic en el icono Actualizar o presionas &shortcut:Refresh;.
