Habilitar modo de solo lectura para conexiones

Para proteger tu base de datos de modificaciones accidentales, puedes activar el modo de solo lectura para una conexión.

Para habilitar el modo de solo lectura para una conexión, haz clic en File | Data sources y selecciona la fuente de datos necesaria en la lista Data Sources. En la pestaña Options, selecciona la casilla Read-only.
