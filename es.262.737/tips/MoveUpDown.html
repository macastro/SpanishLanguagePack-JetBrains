Mover declaraciones arriba y abajo

Las acciones Code | Move Statement Up/Down son útiles para reorganizar líneas de código, por ejemplo para acercar una declaración de variable al uso de la variable.

Selecciona un fragmento de código y presiona &shortcut:MoveStatementUp; o &shortcut:MoveStatementDown;.

Cuando no hay nada seleccionado en el editor, se moverá la línea en la posición del cursor.
