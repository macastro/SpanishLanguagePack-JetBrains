Rellena el cuerpo del método cuando este se genera (por ejemplo, al aplicar la acción de intención Crear método a partir del uso).
Esta plantilla integrada es editable. Además de expresiones Java y comentarios, puede utilizar variables predefinidas que se expandirán a sus valores correspondientes.
Las variables predefinidas aceptan los siguientes valores:
${RETURN_TYPE}   El tipo de retorno del método creado
${DEFAULT_RETURN_VALUE}   El valor devuelto por defecto por el método
${METHOD_NAME}   El nombre del método creado
${CLASS_NAME}   El nombre calificado de la clase en la que se crea el método
${SIMPLE_CLASS_NAME}   El nombre no calificado de la clase en la que se implementa el método
También están disponibles todas las variables predefinidas de la plantilla Encabezado de archivo (pestaña Incluir). (Por ejemplo: ${USER}, ${DATE}, etc.)
