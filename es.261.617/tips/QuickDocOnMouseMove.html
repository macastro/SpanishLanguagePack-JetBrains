Ver documentación rápida al pasar el cursor

No necesitas invocar la documentación rápida explícitamente &shortcut:QuickJavaDoc;. Si pasas el puntero del mouse sobre el símbolo deseado, la ventana emergente de documentación rápida se mostrará automáticamente.

Para habilitar esta función, selecciona la casilla Mostrar documentación rápida al pasar el cursor en la página Editor | Edición de código del diálogo Configuración (&shortcut:ShowSettings;).
