Conmutador para navegación

Usa el conmutador &shortcut:Switcher; para navegar entre archivos abiertos y ventanas de herramientas. Mantén Ctrl presionado para dejar abierto el popup del conmutador. Para navegar, usa Tab y Shift+Tab.

Presiona Delete o BackSpace para cerrar una pestaña del editor u ocultar una ventana de herramientas.
