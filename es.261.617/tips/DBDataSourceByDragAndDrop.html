Archivos SQL como fuente de datos DDL

Los archivos SQL que contienen declaraciones CREATE pueden usarse como fuentes de datos. Como resultado, puedes construir diagramas, referenciar todas las tablas, columnas y otros objetos definidos en dichos archivos en el editor.

Para crear una fuente de datos DDL, arrastra el archivo DDL al Explorador de Base de Datos (View | Tool Windows | Database Explorer) desde tu administrador de archivos.
