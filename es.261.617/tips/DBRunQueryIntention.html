Trabajar con archivos SQL

Haz doble clic en un archivo SQL para abrirlo en el IDE. Para ejecutar una consulta desde este archivo, llama a las acciones de intención (Option+Enter para macOS, Alt+Enter para Windows y Linux) y selecciona Ejecutar consulta en consola. En la lista Sesiones, selecciona una consola existente o crea una nueva.

Ten en cuenta que una nueva consola de consultas significa una nueva conexión a una fuente de datos.
