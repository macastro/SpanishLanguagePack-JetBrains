Una plantilla integrada para un método getter de PHP. Por ejemplo, se utiliza cuando se invoca la corrección rápida "Add getter and setter".
Las variables predefinidas aceptan los siguientes valores:
${STATIC}   Toma el valor 'static' si la propiedad es estática; de lo contrario, devuelve una cadena vacía.
${GET_OR_IS}   Prefijo 'get/is'. Se evalúa como 'is' si la propiedad tiene un tipo booleano (type hint); de lo contrario, se evalúa como la cadena 'get'.
${NAME}   Nombre del método en estilo CamelCase sin el prefijo "get" y con el primer carácter en mayúscula. Por ejemplo, el nombre de la propiedad '_my_property' se convierte en 'MyProperty'.
${CLASS_NAME}   El nombre de la clase contenedora.
${RETURN_TYPE}   Incluye el type hint del tipo de retorno si el nivel del lenguaje es PHP 7.0 o superior.
${FIELD_NAME}   El nombre original de la propiedad sin cambios.
${TYPE_HINT}   Type hint de la propiedad; si no se especifica, devuelve una cadena vacía.
${DS}   Símbolo de dólar, que se evalúa como el carácter '$' normal.
