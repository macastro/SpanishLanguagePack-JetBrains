Esta es una plantilla integrada que se utiliza para completar el cuerpo de un método Groovy cada vez que el programa genera uno (por ejemplo, mediante la acción de intención Crear método a partir del uso).
La plantilla es editable. Además de expresiones Groovy y comentarios, puede utilizar variables predefinidas que se expandirán a sus valores correspondientes.
Las variables predefinidas aceptarán los siguientes valores:
${RETURN_TYPE}   El tipo de retorno del método creado
${DEFAULT_RETURN_VALUE}   El valor devuelto por defecto por el método
${METHOD_NAME}   El nombre del método creado
${CLASS_NAME}   El nombre cualificado de la clase en la que se crea el método
${SIMPLE_CLASS_NAME}   El nombre no cualificado de la clase en la que se implementa el método
