Se aplica al nuevo archivo package-info.java creado al invocar Nuevo | package-info.java en la ventana de herramientas Proyecto.

Esta plantilla integrada es editable. Además de expresiones y comentarios de Java, puede utilizar variables predefinidas (enumeradas a continuación), que se expandirán a sus valores correspondientes como si fueran macros.

También puede especificar variables personalizadas. Las variables personalizadas utilizan el formato ${VARIABLE_NAME}, donde VARIABLE_NAME es el nombre de la variable (por ejemplo, ${MY_CUSTOM_FUNCTION_NAME}). Antes de que el IDE cree un nuevo archivo utilizando variables personalizadas, aparecerá un cuadro de diálogo donde podrá definir el valor de dichas variables para la plantilla.

Mediante el uso de la directiva #parse, puede incluir plantillas de la pestaña Incluir. Para incluir una plantilla, especifique el nombre completo de la misma como un parámetro entre comillas (por ejemplo, #parse("File Header.java")).
Las variables predefinidas aceptan los siguientes valores:
${PACKAGE_NAME}   Nombre del paquete en el que se crea el nuevo archivo package-info.java
${USER}   Nombre de inicio de sesión del sistema del usuario actual
${DATE}   Fecha actual del sistema
${TIME}   Hora actual del sistema
${YEAR}   Año actual
${MONTH}   Mes actual
${MONTH_NAME_SHORT}   Las primeras 3 letras del nombre del mes actual (Jan, Feb, etc.)
${MONTH_NAME_FULL}   Nombre completo del mes actual (January, February, etc.)
${DAY}   Día actual del mes
${HOUR}   Hora actual
${MINUTE}   Minuto actual
${PROJECT_NAME}   Nombre del proyecto actual
