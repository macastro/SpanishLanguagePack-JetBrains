Contiene un fragmento de código que se puede incluir en plantillas de archivo (pestaña Plantillas) usando la directiva #parse.
Las plantillas son editables. Además de texto estático, código y comentarios, también puedes usar variables predefinidas que luego se expandirán como macros a sus valores correspondientes.
Las variables predefinidas tomarán los siguientes valores:
${FILE_NAME}   El nombre del archivo actual
${USER}   El nombre de inicio de sesión del sistema del usuario actual
${USER_NAME}   El nombre completo del usuario actual
${ORGANIZATION_NAME}   El nombre de la organización del proyecto Xcode
${DATE}   La fecha actual del sistema
${TIME}   La hora actual del sistema
${YEAR}   El año actual
${MONTH}   El mes actual
${MONTH_NAME_SHORT}   Las primeras 3 letras del nombre del mes actual. Ejemplo: Jan, Feb, etc.
${MONTH_NAME_FULL}   El nombre completo del mes actual. Ejemplo: January, February, etc.
${DAY}   El día actual
${DAY_NAME_SHORT}   Las primeras 3 letras del nombre del día actual. Ejemplo: Mon, Tue, etc.
${DAY_NAME_FULL}   El nombre completo del día. Ejemplo: Monday, Tuesday, etc.
${HOUR}   La hora actual
${MINUTE}   El minuto actual
${PROJECT_NAME}   El nombre del proyecto actual
${COMPILER_VERSION_MAJOR}   La versión principal del compilador Swift
${COMPILER_VERSION_MINOR}   La versión menor del compilador Swift
${PREFER_EXPLICIT_RETURN}   El valor de la opción Preferir return explícito (Preferencias | Editor | Estilo de código | Swift | Generación de código)
