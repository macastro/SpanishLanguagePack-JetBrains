Reporta llamadas a métodos donde el número de argumentos pasados (incluyendo bloques) no coincide con la firma del método. La firma del método se obtiene de una de las siguientes fuentes:
  • Declaración del método
  • Firma de tipo RBS

En algunos casos, se puede usar una corrección rápida para agregar argumentos faltantes o eliminar argumentos excedentes.

Ejemplo:

def example
end
# Se proporcionaron demasiados argumentos
example(true)
