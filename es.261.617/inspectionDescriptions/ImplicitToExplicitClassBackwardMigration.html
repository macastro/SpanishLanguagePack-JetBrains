Reporta archivos fuente compactos y sugiere reemplazarlos con clases regulares.

Ejemplo (en el archivo Sample.java):


  void main() {
      String a = "Hello, world!";
  }

Después de aplicar la corrección rápida:


public class Sample {
    void main() {
        String a = "Hello, world!";
    }
}

Esta inspección ayuda a degradar para compatibilidad hacia atrás con versiones anteriores de Java.
