Reporta el uso de la propiedad kotlin.coroutine.coroutineContext en código que depende de la biblioteca kotlinx.coroutines.

Cuando tanto kotlin.coroutine.coroutineContext como kotlinx.coroutines.CoroutineScope.coroutineContext están presentes en el código, puede causar confusión y bugs potenciales.

La biblioteca kotlinx.coroutines proporciona la función currentCoroutineContext() como una alternativa más clara, que debería ser preferida incluso cuando no hay conflictos obvios.

Para más detalles, consulta la documentación de kotlin.coroutine.coroutineContext y kotlinx.coroutines.currentCoroutineContext.

Ejemplo:


  suspend fun getCurrentJob(): Job? {
    return coroutineContext[Job]
  }

Después de aplicar la corrección rápida:


  suspend fun getCurrentJob(): Job? {
    return currentCoroutineContext()[Job]
  }
