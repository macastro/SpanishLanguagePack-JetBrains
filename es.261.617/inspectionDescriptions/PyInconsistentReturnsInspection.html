Resalta declaraciones return inconsistentes en funciones. Según PEP8, todas las declaraciones return en una función deberían devolver una expresión, o ninguna debería hacerlo.

Recomendación de PEP8: Todas las declaraciones return en una función deberían devolver una expresión, o ninguna debería hacerlo. Si cualquier declaración return devuelve una expresión, entonces cualquier declaración return que no devuelva un valor debería declararlo explícitamente como return None, y debería existir una declaración return explícita al final de la función (si es alcanzable): 


# Correcto：
def foo(x):
    if x >= 0:
        return math.sqrt(x)
    else:
        return None
def bar(x):
    if x < 0:
        return None
    return math.sqrt(x)


# Incorrecto：
def foo(x):
    if x >= 0:
        return math.sqrt(x)
def bar(x):
    if x < 0:
        return
    return math.sqrt(x)
