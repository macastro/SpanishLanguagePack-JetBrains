Reporta funciones de extensión públicas para Driver/Finder/IdeaFrameUI en clases de test del módulo intellij.driver.tests.

Las funciones de extensión públicas en el módulo intellij.driver.tests crean problemas de mantenibilidad porque:

  • Son visibles para todos los tests en todos los contextos
  • Pueden llevar a dependencias de test poco claras y mayor dificultad de mantenimiento

Soluciones recomendadas:

  • Mover las funciones a un objeto companion privado
  • Reubicar en el objeto de página apropiado:
    • Objetos de página genéricos para funciones ampliamente aplicables
    • Objetos de página específicos para funciones dependientes del contexto
