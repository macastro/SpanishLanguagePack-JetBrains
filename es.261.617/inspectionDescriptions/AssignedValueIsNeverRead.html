Reporta casos donde un valor asignado nunca se usa en el código después de la asignación.

Ejemplo:


  fun foo(): Int = 42
  fun example() {
      var local = 0
      print(local)
      local = 42  // El valor asignado nunca se lee
  }
  • Después del quick fix, la asignación se elimina:

  fun foo(): Int = 42
  fun example() {
      var local = 0
      print(local)
  }
  • Si la expresión del lado derecho (RHS) puede tener efectos secundarios, el quick fix también sugiere conservar la expresión RHS para preservar estos efectos secundarios: 

  fun foo(): Int = 42
  fun example() {
      var local = 0
      print(local)
      foo()  // La llamada a la función se conserva por sus efectos secundarios
  }
