Informa sobre las cláusulas catch que capturan excepciones inapropiadas.

Algunas excepciones (por ejemplo, java.lang.NullPointerException o java.lang.IllegalMonitorStateException) indican errores de programación y, casi con toda seguridad, no deberían capturarse en el código de producción.

Ejemplo:


  try {
    return component.getMousePosition(true) != null;
  } catch (NullPointerException e) { // advertencia: Se ha capturado la excepción prohibida 'NullPointerException'
    return false;
  }

Utilice la lista de Excepciones prohibidas para especificar qué excepciones deben notificarse.
