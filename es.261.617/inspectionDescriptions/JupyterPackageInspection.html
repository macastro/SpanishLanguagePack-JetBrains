Reporta la situación en la que el paquete jupyter no está instalado para el intérprete de Python seleccionado. Sin el paquete Jupyter correctamente instalado, no es posible ejecutar notebooks de Jupyter.

Haz clic en el enlace correspondiente en el banner de advertencia para instalar el paquete faltante. También puedes instalar este paquete en la Configuración/Preferencias del proyecto o en la ventana de herramientas de Paquetes de Python.

Para obtener información detallada, consulta Instalación de paquetes de Python.
