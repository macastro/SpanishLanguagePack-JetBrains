Reporta secuencias de escape Unicode innecesarias. Cuando la codificación del archivo puede manejar el carácter sin necesidad de escape, las secuencias de escape Unicode serán innecesarias. Esta inspección no reportará caracteres de control Unicode, excepto los saltos de línea y tabulaciones horizontales.

Ejemplo:


  String s = "\u0078";

Después de aplicar la corrección rápida:


  String s = "x";
