Reporta expresiones lambda que no son llamadas o asignadas.

Las expresiones lambda que no son llamadas o asignadas a una variable no tienen efecto en la ejecución del programa y pueden ser involuntarias.

Ejemplo:


  fun foo() {
    {} // Advertencia: La expresión lambda nunca es llamada
  }

Después de aplicar la corrección rápida:


  fun foo() {
    run {} // Lambda ahora es llamada a través de la función 'run'
  }
