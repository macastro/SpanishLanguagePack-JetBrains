Reporta etiquetas redundantes que no pueden ser referenciadas desde declaraciones 'break', 'continue' o 'return'.

Estas etiquetas pueden ser eliminadas de forma segura.

Ejemplo:


fun foo(c: Boolean) {
    b@ while (true) a@ {
        while (true) {
            break@b
        }
    }
}

Después de aplicar la corrección rápida:


fun foo(c: Boolean) {
    b@ while (true) {
        while (true) {
            break@b
        }
    }
}
