Reporta declaraciones if que tienen tres o más ramas y que pueden ser reemplazadas por una expresión when con cuerpo.

Ejemplo:


  fun translateNumber(n: Int): String {
    return if (n == 1) {
      "one"
    } else if (n == 2) {
      "two"
    } else {
      "???"
    }
  }

La corrección rápida convertirá la expresión if a when:


  fun translateNumber(n: Int): String {
    return when (n) {
      1 -> {
        "one"
      }
      2 -> {
        "two"
      }
      else -> {
        "???"
      }
    }
  }
