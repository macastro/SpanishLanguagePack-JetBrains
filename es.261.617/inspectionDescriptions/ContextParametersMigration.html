Reporta receptores de contexto que deberían migrarse a parámetros de contexto.

A partir de Kotlin 2.2.0, los receptores de contexto han sido reemplazados por parámetros de contexto. Esta inspección ayuda a migrar proyectos que tienen receptores de contexto habilitados pero aún no han migrado a la sintaxis de parámetros de contexto.

Ejemplo: 


context(MutableList<String>, C)
private fun foo() {
    add("new")
    bar()
    baz()
}
class C {
    fun bar() {}
}
fun C.baz() {}
Después de la migración: 

context(strings: MutableList<String>, c: C)
private fun foo() {
    strings.add("new")
    c.bar()
    c.baz()
}
class C {
    fun bar() {}
}
fun C.baz() {}
