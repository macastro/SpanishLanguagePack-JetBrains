Reporta valores de retorno de campos de array o tipos mutables (como Collection, Date, Map, Calendar, etc.) o asignaciones desde parámetros de método a campos de array o tipos mutables.

Dado que este tipo es mutable, esta estructura puede llevar a modificaciones inesperadas del estado del objeto desde fuera de la clase propietaria. Aunque esta estructura puede ser útil por razones de rendimiento, es inherentemente propensa a errores.

Se reportan los siguientes tipos mutables:

  • java.util.Date
  • java.util.Calendar
  • java.util.Collection
  • java.util.Map
  • com.google.common.collect.Multimap
  • com.google.common.collect.Table

La corrección rápida añade una llamada al método .clone() del campo para arrays, o utiliza wrappers de colecciones no modificables.

Ejemplo:


  import java.util.*;
  class Log {
    private String[] messages = {"one", "two", "three"};
    private Map<String, String> map = new HashMap<>();
    String[] getMessages() {
      return messages; // Advertencia: retorna el campo String[] 'messages'
    }
    Map<String, String> mapping() {
      return map; // Advertencia: retorna el campo Map<String, String> 'map'
    }
  }

Después de aplicar la corrección rápida:


  import java.util.*;
  class Log {
    String[] messages = {"one", "two", "three"};
    private Map<String, String> map = new HashMap<>();
    String[] getMessages() {
      return messages.clone();
    }
    Map<String, String> mapping() {
      return Collections.unmodifiableMap(map);
    }
  }

Usa la opción Ignorar asignaciones y valores de retorno en métodos private para ignorar asignaciones y valores de retorno en métodos private.
