Reporta expresiones no utilizadas, lo que generalmente indica código incompleto o la falta de una declaración return.

Si la expresión no utilizada es la última declaración en una función y su tipo coincide con el tipo de valor de retorno de la función, una corrección rápida agregará automáticamente la palabra clave return.

Ejemplo:


fun example(): Int {
    val x = 10
    x // Advertencia: expresión no utilizada
}

Después de aplicar la corrección rápida:


fun example(): Int {
    val x = 10
    return x
}
