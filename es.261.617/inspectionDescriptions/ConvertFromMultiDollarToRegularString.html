Reporta todas las cadenas con prefijo de interpolación.

La corrección rápida de la inspección eliminará el prefijo de interpolación, actualizará las entradas de interpolación y escapará los caracteres de signo de dólar inseguros.

Ejemplo:


val foo = $$"""
{
    "foo": $ref,
    "bar": {
        "baz": "$20"
        "boo": "$${100 + 200}"
    }
}
"""

Después de aplicar la corrección rápida:


val foo = """
{
    "foo": ${'$'}ref,
    "bar": {
        "baz": "$20"
        "boo": "${100 + 200}"
    }
}
"""
