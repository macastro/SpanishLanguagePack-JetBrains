Reporta las llamadas al método openSession() si el recurso org.hibernate.Session devuelto no se cierra de forma segura.

Por defecto, esta inspección asume que el recurso puede ser cerrado por cualquier método cuyo nombre contenga 'close' o 'cleanup'.

Ejemplo:


  void doHibernateQuery(SessionFactory factory) {
    Session session = factory.openSession(); //advertencia
    session.createQuery("...");
  }

Configure esta inspección utilizando las siguientes opciones:

  • Si se permite abrir recursos org.hibernate.Session dentro de un bloque try. Este estilo es menos deseable porque es más verboso que abrir el recurso antes del bloque try.
  • Si el recurso puede cerrarse mediante cualquier llamada a un método que pase el recurso como argumento.
