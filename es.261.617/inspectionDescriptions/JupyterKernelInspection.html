Reporta situaciones donde el intérprete de Python no está configurado como kernel del notebook. Esta verificación puede prevenir diferencias entre el code insight y el comportamiento en tiempo de ejecución de las celdas de código del notebook de Jupyter.

Si ve la siguiente advertencia, significa que el kernel del servidor recién configurado no coincide con el intérprete de Python actual. Por ejemplo, podría tener instalado un conjunto diferente de paquetes.

Puede hacer clic en el enlace correspondiente en el banner de advertencia para registrar su intérprete de Python. Después, podrá ver ese intérprete en la lista de kernels del servidor configurado.

Para obtener información detallada, consulte Administrar servidores de notebook de Jupyter.
