Reporta instancias de inicializadores de instancia que pueden ser marcados como static sin producir ningún error.

Un inicializador de instancia puede ser static si no hace referencia a ningún miembro no static de su clase. Los inicializadores static se ejecutan después de que la clase se resuelve, mientras que los inicializadores de instancia se ejecutan para cada instanciación de esta clase.

Esta inspección no reporta inicializadores de instancia vacíos e inicializadores en clases anónimas.

Ejemplo:


  class A {
     public static String CONSTANT;
     {
        CONSTANT = "Hello";
     }
  }

Después de aplicar la solución rápida:


  class A {
     public static String CONSTANT;
     static {
        CONSTANT = "Hello"; // ahora se instancia solo una vez por clase
     }
  }
