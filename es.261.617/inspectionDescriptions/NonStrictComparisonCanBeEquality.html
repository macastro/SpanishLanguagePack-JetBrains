Reporta condiciones de desigualdad que solo pueden ser satisfechas por un único valor de operando según el análisis de flujo de datos. Tales condiciones pueden ser reemplazadas por condiciones de igualdad para hacer el código más claro.

Ejemplo:


  if (x >= 10) {
    ...
    if (x <= 10) { // puede ser reemplazado por 'x == 10'
    }
  }
