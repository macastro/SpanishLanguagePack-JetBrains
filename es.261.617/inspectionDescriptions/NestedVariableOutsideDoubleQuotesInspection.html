Detecta instancias de variables anidadas en archivos `.env` que se usan fuera de comillas dobles y pueden no analizarse correctamente. Esta verificación puede asegurar que las variables anidadas estén correctamente encerradas para evitar problemas de configuración.

Esta verificación ayuda a mantener la sintaxis correcta en archivos `.env` al forzar el uso de comillas dobles para variables anidadas, previniendo comportamientos inesperados durante la evaluación de variables.

Corrección rápida: Esta verificación proporciona una corrección rápida para encerrar automáticamente las variables anidadas entre comillas dobles. Haz clic en la corrección rápida proporcionada para resolver automáticamente el problema.
