Informa de los siguientes problemas en los descriptores utilizados para desplegar módulos Web en un servidor:
  • Referencias a clases no instanciables
  • Referencias a clases que no extienden las clases requeridas
  • Referencias a clases con un ámbito inadecuado
  • Etiquetas y valores de atributos vacíos
  • Valores de etiquetas y atributos que no coinciden con el patrón requerido (por ejemplo, identificadores Java)
  • Etiquetas que no contienen las etiquetas hijas o los atributos requeridos
  • Etiquetas que definen objetos con nombres duplicados

Ejemplo:



<web-app>
  <filter-mapping>
    <filter-name>CustomFilter</filter-name> // Error: se debe definir la subetiqueta servlet-name o url-pattern
  </filter-mapping>
  <servlet-mapping>
    <servlet-name>CustomServletName</servlet-name>
    <url-pattern>/ baseUrl</url-pattern> // Error: el patrón URI no puede contener espacios
  </servlet-mapping>
</web-app>
