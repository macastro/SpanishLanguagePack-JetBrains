Reporta el uso incorrecto de anotaciones de ruta.

Esta inspección puede detectar situaciones donde cadenas con un tipo de anotación de ruta se usan en contextos que requieren otro tipo de anotación de ruta. Ayuda a asegurar el uso correcto de las anotaciones @MultiRoutingFileSystemPath y @NativePath.

La inspección resalta los siguientes problemas:

  • Cuando cadenas anotadas con @NativePath se usan en constructores o métodos factory de Path
  • Cuando cadenas anotadas con @NativePath se pasan a parámetros de método anotados con @MultiRoutingFileSystemPath
  • Cuando cadenas anotadas con @MultiRoutingFileSystemPath se pasan a parámetros de método anotados con @NativePath
  • Cuando literales de cadena se usan en contextos que requieren cualquiera de estas anotaciones

Se proporcionan correcciones rápidas para agregar las anotaciones apropiadas donde sea necesario.

Esta inspección ayuda a prevenir errores de runtime que pueden ocurrir al usar incorrectamente rutas de diferentes formatos. La anotación @MultiRoutingFileSystemPath se usa para rutas que deberían soportar diferentes sistemas de archivos, mientras que la anotación @NativePath se usa para rutas específicas del sistema de archivos nativo.
