Reporta llamadas a map { it.await() } en colecciones Deferred.

Esto puede ser reemplazado por una llamada a awaitAll() más concisa y potencialmente más eficiente.

Para más detalles, consulta la documentación de kotlinx.coroutines.awaitAll.

Ejemplo:


  suspend fun waitForResult(results: List<Deferred<String>>): List<String> {
    return results.map { it.await() }
  }

Después de aplicar la corrección rápida:


  suspend fun waitForResult(results: List<Deferred<String>>): List<String> {
    return results.awaitAll()
  }
