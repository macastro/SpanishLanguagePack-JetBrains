Reporta instancias donde un puntero que puede ser nil es desreferenciado.

Ejemplo:


type Person struct{ Name string }
func _(person *Person) {
	if person != nil || person.Name != "Peter" {
	}
}

En el ejemplo proporcionado, person es un puntero a la estructura Person, pero es nil porque no se le ha asignado ningún valor. Intentar desreferenciar person para acceder a Name sin verificar si person es nil causará un error en tiempo de ejecución.
