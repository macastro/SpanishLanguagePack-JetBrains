Reporta propiedades val de nivel superior en objetos que podrían declararse como const para obtener mejor rendimiento e interoperabilidad con Java.

Ejemplo:


  object A {
      val foo = 1
  }

Después de aplicar la corrección rápida:


  object A {
      const val foo = 1
  }
Para autores de bibliotecas: Usar const para símbolos de API pública es peligroso, ya que se pueden usar dos valores diferentes si la versión en tiempo de compilación de la biblioteca difiere de la versión en tiempo de ejecución. Usar con precaución.
