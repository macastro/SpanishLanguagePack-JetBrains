Reporta operaciones de escritura dentro de métodos @SafeVarargs donde el parámetro vararg es reasignado a otros objetos. En este caso, el array recién asignado no está relacionado con la aridad variable del sitio de llamada y puede que ya no sea seguro.

Ejemplo:


    @SafeVarargs
    public final <T> void test(T[] tt, T... t) {
        t = tt;
        System.out.println(t[0]);
    }
