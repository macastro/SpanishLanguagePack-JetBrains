Esta inspección está diseñada para identificar segmentos de código potencialmente vulnerables donde los datos obtenidos de fuentes potencialmente no confiables pueden ser utilizados en operaciones que constituyen un riesgo de seguridad.

Seguimiento del flujo de datos:

Esta inspección rastrea el flujo de datos desde su origen hasta su uso en operaciones críticas. Esto incluye el análisis de la transmisión de datos a través de métodos, objetos y variables.

Considera un ejemplo donde la entrada del usuario se utiliza directamente en una consulta SQL sin el preprocesamiento adecuado. Esta inspección emitirá una advertencia sobre una potencial inyección SQL.

String userInput = getUserInput();
String sqlQuery = "SELECT * FROM users WHERE username = '" + userInput + "'";
