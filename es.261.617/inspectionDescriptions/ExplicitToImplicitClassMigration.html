Reporta clases ordinarias que pueden convertirse en archivos fuente compactos

Ejemplo:


  public class Sample {
      public static void main(String[] args) {
          System.out.println("Hello, world!");
      }
  }

Después de aplicar la corrección rápida:


  void main() {
      IO.println("Hello, world!");
  }

Configurar inspección:

  • Selecciona Reemplazar 'System.out.print()' y 'System.out.println()' con métodos de la clase 'java.lang.IO' para convertir estas llamadas a métodos IO.
