Reporta las llamadas al método java.util.Map.get(). Dichas llamadas pueden sustituirse por una forma de acceso por clave más corta y clara.

Ejemplo:


def map = ["foo": "bar"]
def str = map.get("foo") // map.get("foo") se puede reemplazar por map["foo"]

Tras aplicar la corrección rápida:


def map = ["foo": "bar"]
def str = map["foo"]
