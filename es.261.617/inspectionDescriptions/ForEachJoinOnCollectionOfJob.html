Reporta llamadas a forEach { it.join() } en colecciones de Job.

Esto puede ser reemplazado por una llamada más concisa y completamente equivalente a joinAll().

Para más detalles, consulta la documentación de kotlinx.coroutines.joinAll.

Ejemplo:


  suspend fun waitForJobs(jobs: List<Job>) {
    jobs.forEach { it.join() }
  }

Después de aplicar la corrección rápida:


  suspend fun waitForJobs(jobs: List<Job>) {
    jobs.joinAll()
  }
