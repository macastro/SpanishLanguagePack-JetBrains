Reporta caracteres latinos no básicos en literales y comentarios. Esta inspección proporciona una corrección rápida para reemplazar dichos caracteres con escapes Unicode en literales y con entidades HTML en comentarios.

Ejemplo:


  // © 2021
  char c = '©';
  String s = "Áî";

Después de aplicar la corrección rápida:


  // &copy; 2021
  char c = '\u00a9';
  String s = "\u00c1\u00ee";
Por defecto, esta inspección no resalta en el editor, sino que solo proporciona correcciones.
