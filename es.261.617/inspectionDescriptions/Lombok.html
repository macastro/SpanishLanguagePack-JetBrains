Proporciona inspecciones generales para las anotaciones de Lombok.

Las advertencias y errores reportados por esta inspección están diseñados para coincidir 1:1 con las advertencias y errores reportados en tiempo de compilación por el procesador de anotaciones de Lombok, por ejemplo:

  • Anotaciones aplicadas de forma redundante (reportadas como advertencias)

    Ejemplo:

    
      @ToString
      class Person {
        final String name;
        final int age;
        @Override
        public String toString() {
            return "Person{name='" + name + "', age=" + age + '}';
        }
      }
    La corrección rápida sugiere eliminar la anotación redundante.
  • Anotaciones aplicadas incorrectamente (reportadas como errores)

    Ejemplo:

    
      @Value
      record Person(String name, int age) {}
    La corrección rápida sugerirá eliminar la anotación inválida.
