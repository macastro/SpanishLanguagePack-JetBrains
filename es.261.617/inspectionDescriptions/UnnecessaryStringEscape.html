Reporta caracteres de escape innecesarios en literales de String y char (opcional).

Aunque no necesita ser escapado, el carácter de tabulación escapado (\t) no será reportado, porque las tabulaciones son difíciles de distinguir de los espacios.

Ejemplo:


  String s = "\'Scare\' quotes";
  String t = """
    All you need is\n\tLove\n""";

Después de aplicar la corrección rápida:


  String s = "'Scare' quotes";
  String t = """
    All you need is
    \tLove
    """;
