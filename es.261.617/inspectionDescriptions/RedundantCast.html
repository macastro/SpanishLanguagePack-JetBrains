Reporta expresiones de conversión innecesarias.

Ejemplo:


  static Object toObject(String s) {
    return (Object) s;
  }

Después de aplicar la corrección rápida:


    static Object toObject(String s) {
    return s;
  }

Usa la casilla de verificación a continuación para ignorar conversiones aclaratorias, como conversiones en llamadas de colección que deberían ser Object: 


  static void removeFromList(List<String> l, Object o) {
    l.remove((String)o);
  }
