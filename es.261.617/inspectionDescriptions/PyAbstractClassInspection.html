Reporta definiciones y usos inválidos de clases abstractas.

Ejemplo:


from abc import abstractmethod, ABC
class Figure(ABC):
    @abstractmethod
    def do_figure(self):
        pass
class Triangle(Figure):  # 'Triangle' 类中未定义所有 abstract 方法
    def do_triangle(self):
        pass
Triangle()  # 无法实例化抽象类 'Triangle'

Después de aplicar la corrección rápida, el IDE implementa los métodos abstractos para la clase Triangle:


from abc import abstractmethod, ABC
class Figure(ABC):
    @abstractmethod
    def do_figure(self):
        pass
class Triangle(Figure):
    def do_figure(self):
        pass
    def do_triangle(self):
        pass
Triangle()
