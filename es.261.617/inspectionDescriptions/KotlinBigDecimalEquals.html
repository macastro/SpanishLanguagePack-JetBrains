Reporta llamadas de == o equals() que comparan dos números java.math.BigDecimal. Esto suele ser un error, ya que dos números java.math.BigDecimal solo son iguales si son iguales tanto en valor como en escala.

Ejemplo:


  // 条件为 false
  val condition = BigDecimal("2.0") ==
      BigDecimal("2.00")

Después de aplicar la corrección rápida:


  // 条件为 true
  val condition = BigDecimal("2.0").compareTo(
      BigDecimal("2.00")) == 0
