Informa sobre problemas en la firma de funciones integradas.

Esta inspección informa sobre un número incorrecto de argumentos, palabras clave no válidas, tipos de datos incorrectos y otros problemas.

Ejemplo (MySQL):

CREATE TABLE foo (a INT, b INT, c INT)

SELECT IFNULL() FROM foo; -- error
SELECT IFNULL(a) FROM foo; -- error
SELECT IFNULL(a, b) FROM foo; -- correcto
SELECT IFNULL(a, b, c) FROM foo; -- error

En MySQL, la función IFNULL() acepta estrictamente dos argumentos. Por lo tanto, solo la consulta SELECT IFNULL(a, b) FROM foo; es correcta.
