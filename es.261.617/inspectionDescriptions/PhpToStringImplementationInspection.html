Reporta los intentos de convertir un objeto que no tiene una implementación del método __toString a string.

Antes de PHP 7.4, esto provocaba un error fatal de nivel E_RECOVERABLE_ERROR. En PHP 7.4, se lanza un ERROR fatal irrecuperable. A partir de PHP 8.0, el valor de retorno sigue la semántica de tipos estándar de PHP y, si es posible y el modo de tipado estricto está desactivado, se convierte mediante coerción a una cadena de caracteres.

Si la opción Comprobar si existe __toString para cada tipo de expresión está activada, la inspección comprobará todos los tipos posibles de la expresión e informará si al menos uno de ellos no contiene una implementación del método __toString.

Para obtener más información, consulta __toString (php.net).
