Reporta variables sin tipo especificado

Cuando el código no puede resolver el tipo que contiene una variable, debe especificarse manualmente

Ejemplo:


  var int_val = 1
  val string_val = ""

Después de aplicar la corrección rápida:


  var int_val: int = 1
  val string_val: String = ""

O cuando se usa la opción corta:


  var int_val := 1
  val string_val := ""

Desde 1.1.0
