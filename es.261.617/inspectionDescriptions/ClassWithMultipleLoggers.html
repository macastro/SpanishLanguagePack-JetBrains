Informa sobre las clases que declaran múltiples loggers. Un paso importante para proporcionar una implementación de registro (logging) unificada en una aplicación es asegurarse de que cada clase tenga un único logger dedicado.

Ejemplo:


  public class Critical {
    protected static final Logger LOG = Logger.getLogger(Critical.class);

    protected static final Logger myLogger = Logger.getLogger(getClass());
  }

Utilice la tabla de abajo para especificar los nombres de clase de Logger. Esta inspección informará sobre las clases que declaren múltiples campos que tengan uno de los tipos de las clases especificadas.
