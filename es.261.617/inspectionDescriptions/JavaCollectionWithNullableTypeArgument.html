Reporta casos donde se proporcionan argumentos de tipo nullable a colecciones Java que no aceptan null.

Algunas colecciones Java proporcionadas en la biblioteca estándar de Java no soportan elementos, claves o valores null. Los argumentos de tipo de tales colecciones no deberían ser instanciados con tipos nullable.

Ejemplo:


val map = ConcurrentHashMap<String, String?>()

Después de aplicar la corrección rápida:


val map = ConcurrentHashMap<String, String>()
