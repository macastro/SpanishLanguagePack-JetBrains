Informa sobre expresiones de incremento o decremento que están anidadas dentro de otras expresiones. Dichas expresiones pueden generar confusión e infringir el principio general de diseño que establece que cualquier construcción solo debe hacer una cosa.

La corrección rápida extrae la operación de incremento o decremento en una sentencia de expresión separada.

Ejemplo:


  int i = 10;
  while (i-- > 0) {
    System.out.println(i);
  }

Después de aplicar la corrección rápida:


  int i = 10;
  while (i > 0) {
    i--;
    System.out.println(i);
  }
  i--;
