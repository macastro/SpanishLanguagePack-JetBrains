Reporta métodos cuyos nombres son idénticos a su clase. Si bien dicho nombre es permitido por el lenguaje Java, por convención se reserva para definir constructores. Usarlo para métodos probablemente es un error o una mala práctica.

Ejemplo:


  clase MyClass {
    int val;

    // El método MyClass tiene el mismo nombre que la clase contenedora.
    // Es probable que 'void' se haya agregado por error.
    void MyClass(int val) {
      this.val = val;
    }
  }

Cuando es pertinente, un arreglo rápido convierte el método en un constructor:


  clase MyClass {
    int val;

    MyClass(int val) {
      this.val = val;
    }
  }

Otro arreglo rápido vuelve a nombrar el método.
