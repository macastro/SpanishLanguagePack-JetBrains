Reporta variables de instancia leídas antes de ser inicializadas.

Esta inspección ignora las revisiones de equivalencia con null.

Ejemplo:


  class Foo {
    int bar;

    Foo() {
      System.out.println(bar);
    }
  }

Tenga en cuenta que esta inspección usa un algoritmo de flujo de datos muy conservador y que puede reportar incorrectamente variables de instancia que no están inicializadas. Las variables que se reportan como inicializadas siempre estarán inicializadas.

Use la opción Ignorar si tiene las siguientes anotaciones para especificar anotaciones especiales. La inspección ignorará los campos anotados con una de estas anotaciones.

Use la opción Ignorar campos primitivos para ignorar los campos primitivos no inicializados.
