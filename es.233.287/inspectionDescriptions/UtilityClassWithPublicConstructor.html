Informa de clases de utilidad con constructores public.

Las clases de utilidad tienen todos los campos y métodos declarados como static. Crear un constructor public en dichas clases resulta confuso y puede provocar la instanciación accidental de la clase.

Ejemplo:


  public final class UtilityClass {
    public UtilityClass(){
    }
    public static void foo() {}
  }

Después de aplicar la corrección rápida:


  public final class UtilityClass {
    private UtilityClass(){
    }
    public static void foo() {}
  }
