Informa las sentencias switch que se pueden reemplazar automáticamente con sentencias o expresiones switch mejoradas.

Ejemplo:


  double getPrice(String fruit) {
    // La sentencia switch se puede reemplazar con 'switch' mejorada
    switch (fruit) {
      case "Apple":
        return 1.0;
      case "Orange":
        return 1.5;
      case "Mango":
        return 2.0;
      default:
        throw new IllegalArgumentException();
    }
  }

Después de aplicar la revisión rápida:


  double getPrice(String fruit) {
    return switch (fruit) {
      case "Apple" -> 1.0;
      case "Orange" -> 1.5;
      case "Mango" -> 2.0;
      default -> throw new IllegalArgumentException();
    };
  }

  • Usa la opción Mostrar advertencia solo si la conversión a expresión es posible para no advertir sobre la conversión a sentencia switch.
  • Usa la opción Número máximo de sentencias en una rama para convertir a expresión switch para advertir sobre la conversión a expresión solo si cada rama tiene menos del número dado de sentencias.
