Informa de los métodos marcados con la anotación @DataProvider que no devuelven Object[][] o Iterator<Object>.

Si se devuelve otro tipo, TestNG lanza una excepción.

Ejemplo:


public class TestNgTest {
  @DataProvider(name = "Idiomas")
  List<String> getData() {
    return List.of("Java", "Kotlin");
  }

  @Test(dataProvider = "Idiomas")
  public void testData(String idioma) {
    System.out.println(idioma);
  }
}
