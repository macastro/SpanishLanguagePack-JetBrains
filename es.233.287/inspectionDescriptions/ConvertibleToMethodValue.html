Informa sobre funciones anónimas que pueden ampliarse mediante eta e informa también de marcadores de posición que pueden eliminarse de las ampliaciones mediante eta.

Ejemplo:


  def f(x: Any, y: Int): Unit = ()
  val x = f(_, _)
  val y: (Int, Int) => Unit = f _

Después de aplicar la corrección rápida:


  def f(x: Any, y: Int): Unit = ()
  val x = f _
  val y: (Int, Int) => Unit = f
