Reporta asignaciones compuestas si el tipo del operando de la derecha no es compatible con el tipo de la variable.

Durante dichas asignaciones compuestas, ocurre un casteo implícito, lo que puede resultar en conversiones con pérdida de datos.

Ejemplo:


  long c = 1;
  c += 1.2;

Después de aplicar la corrección rápida:


  long c = 1;
  c += (long) 1.2;
