Informa las llamadas a un método de superclase desde una clase anónima, interna o local, si existe un método con la misma firma en el código circundante a la clase. En este caso, puede parecer que se llama a un método del código circundante, cuando en realidad es una llamada a un método de la superclase.

Para aclarar la intención del código, se recomienda añadir un calificador explícito super a la llamada del método.

Ejemplo:


  clase Parent {
    void ambiguous(){}
  }

  clase Example {
    void ambiguous(){}

    clase Inner extiende Parent {
      void example(){
        ambiguous(); //advertencia
      }
    }
  }

Después de aplicar la solución rápida:


  clase Parent {
    void ambiguous(){}
  }

  clase Example {
    void ambiguous(){}

    clase Inner extiende Parent {
      void example(){
        super.ambiguous();
      }
    }
  }
