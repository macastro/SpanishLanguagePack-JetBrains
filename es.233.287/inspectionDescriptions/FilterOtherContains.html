Informa sobre filter o filterNot en Set y sugiere usar intersect o diff en su lugar.

Ejemplo:


  val set = Set(1, 2)
  val others = Set(2, 3)
  set.filter(others.contains(_))
  set.filterNot(others.contains)
  set.filter(x => !others.contains(x))
  set.filterNot(x => !others.contains(x))

Después de aplicar la corrección rápida:


  val set = Set(1, 2)
  val others = Set(2, 3)
  set.intersect(others)
  set.diff(others)
  set.diff(others)
  set.intersect(others)
