Informa sobre la copia manual del contenido de una matriz que se puede reemplazar por una llamada a System.arraycopy().

Ejemplo:


  for (int i = 0; i < array.length; i++) {
    newArray[i] = array[i];
  }

Una vez aplicada la corrección rápida:


  System.arraycopy(array, 0, newArray, 0, array.length);
