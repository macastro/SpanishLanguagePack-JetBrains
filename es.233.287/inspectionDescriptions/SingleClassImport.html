Informa acerca de sentencias import que importan clases individuales (en lugar de paquetes completos).

Algunos estándares de codificación prohíben este tipo de sentencias import.

Puedes configurar IntelliJ IDEA para detectar y corregir tales sentencias con su comando Optimizar importaciones. Ve a Configuración | Editor | Estilo de código | Java | Importaciones y desactiva la casilla de verificación Usar importación de clase única. Por lo tanto, esta inspección es principalmente útil para informes sin conexión sobre bases de código que no pretendes cambiar.
