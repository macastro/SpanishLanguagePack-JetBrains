Reporta llamadas explicitas a funciones get o set que pueden remplazarse por un operador de indexación [].

Kotlin permite implementar de manera personalizada el conjunto de operadores predefinidos en tipos. Para sobrecargar un operador, puede marcar la función correspondiente con el modificador operator: 


  operator fun get(index: Int) {}
  operator fun set(index: Int, value: Int) {}
Las funciones anteriores corresponden al operador de indexación.

Ejemplo:

  class Test {
      operator fun get(i: Int): Int = 0
  }

  fun test() {
      Test().get(0) // 'get()' reemplazable
  }

Después de aplicar el quick-fix:

  class Test {
      operator fun get(i: Int): Int = 0
  }

  fun test() {
      Test()[0]
  }
