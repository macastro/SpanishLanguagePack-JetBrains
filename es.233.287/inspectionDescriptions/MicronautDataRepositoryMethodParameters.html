Informa de tipos de parámetros de método de repositorio de Datos de Micronaut incorrectos.

Ejemplo:


  public class Persona {
    private int id;
    private String apellido;
    private Direccion direccion;
  }

  public class Direccion {
    private int codigoPostal;
  }

  public interface RepositorioDeClientes extends CrudRepository<Persona, Integer> {
    List<Persona> findByDirección(String apellido);  // Se espera tipo 'Dirección'
    List<Persona> findByDirecciónCódigoPostal(String str);   // Se espera tipo 'int'
  }
