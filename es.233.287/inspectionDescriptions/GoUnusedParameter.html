Reporta parámetros de función no utilizados. 
func main() {
 printAll(
  42,
  "bird",
 )
}

func printAll(
 i int,
 s string,
) {
 fmt.Println(i)
}

Llamamos a la función printAll pasando 42 y bird como argumentos. La función printAll acepta dos parámetros int y string pero utiliza sólo el primero de ellos. Por lo tanto, el s string está en gris.
