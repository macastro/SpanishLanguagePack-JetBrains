Informa sobre los objetos (incluido objeto de datos) que implementan java.io.Serializable pero que no implementan readResolve

Ejemplo:


  import java.io.Serializable

  object Foo : Serializable

La corrección rápida implementa el método readResolve:


  import java.io.Serializable

  object Foo : Serializable {
      private fun readResolve() = Foo
  }
