Reporta llamadas a java.lang.System.loadLibrary(), java.lang.System.load(), java.lang.Runtime.loadLibrary() y java.lang.Runtime.load() que toman una cadena construida dinámicamente como nombre de la biblioteca.

Las cadenas de nombre de biblioteca construidas son una fuente común de brechas de seguridad. Por defecto, esta inspección ignora las constantes de tiempo de compilación.

Ejemplo:


  void test(int i) {
    System.loadLibrary("foo" + i);
  }

Usa la configuración de inspección para considerar cualquier campo static final como constante. Ten cuidado, porque las cadenas como las siguientes serán ignoradas cuando la opción esté habilitada: 


  private static final String LIBRARY = getUserInput();
