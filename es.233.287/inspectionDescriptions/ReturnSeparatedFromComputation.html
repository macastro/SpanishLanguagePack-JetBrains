Reporta declaraciones return que retornan una variable local donde el valor de la variable se calcula en algún otro lugar dentro del mismo método.

El arreglo rápido realiza un ajuste en línea de la variable retornada moviendo la declaración return a la ubicación en la que se calcula el valor de la variable. Cuando el valor retornado no puede ajustarse en línea en la declaración return, el arreglo rápido intenta mover la declaración return lo más cerca posible al cálculo del valor retornado.

Ejemplo:


  int n = -1;
  for (int i = 0; i < a.length; i++) {
    if (a[i] == b) {
      n = i;
      break;
    }
  }
  return n;

Después de aplicar el arreglo rápido:


  int n = -1;
  for (int i = 0; i < a.length; i++) {
    if (a[i] == b) {
      return i;
    }
  }
  return n;
