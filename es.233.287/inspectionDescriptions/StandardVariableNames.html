Reporta variables con nombres 'estándar' que no corresponden a sus tipos. Tales nombres pueden ser confusos. Existen los siguientes nombres estándar para tipos específicos:
  • i, j, k, m, n - int
  • f - float
  • d - double
  • b - byte
  • c, ch - char
  • l - long
  • s, str - String

Se sugiere solo en el editor la sugerencia rápida de cambio de nombre.

Usa la opción para ignorar nombres de parámetros que sean idénticos al nombre de parámetro de un método super directo.
