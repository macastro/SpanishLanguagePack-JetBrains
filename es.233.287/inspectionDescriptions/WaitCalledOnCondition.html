Reporta llamadas a wait() realizadas en un objeto java.util.concurrent.locks.Condition. Esto es probablemente un error de programación y alguna variante del método await() debía usarse en su lugar.

Ejemplo:


  void acquire(Condition released) throws InterruptedException {
    while (acquired) {
      released.wait();
    }
  }

El código correcto se vería así:


  void acquire(Condition released) throws InterruptedException {
    while (acquired) {
      released.await();
    }
  }
