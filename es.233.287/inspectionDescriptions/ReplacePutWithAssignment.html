Reporta llamadas a la función map.put que pueden ser reemplazadas con operador de indexación ([]).

El uso de sintaxis dulce hace más simple tu código.

La corrección rápida reemplaza la llamada put con la asignación.

Ejemplo:


  fun foo(map: MutableMap<Int, String>) {
      map.put(42, "foo")
  }

Después de aplicar la corrección rápida:


  fun foo(map: MutableMap<Int, String>) {
      map[42] = "foo"
  }
