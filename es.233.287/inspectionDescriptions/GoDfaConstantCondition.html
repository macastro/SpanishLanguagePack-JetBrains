Informa sobre condiciones en el código que siempre evalúan a true o false.

Ejemplo:


func _(file *os.File) {
    if file == nil {
        return
    }
    if file == nil {
    }
}

En el ejemplo dado, la segunda condición if file == nil es redundante. Siempre evaluará a false ya que repite una verificación ya realizada en el código.
