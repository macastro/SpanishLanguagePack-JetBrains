Reporta concatenaciones de cadenas no constantes usadas como argumento para una llamada a MessageFormat.format().

Aunque a veces es intencional, usualmente se trata de un mal uso del método de formateo e incluso puede causar excepciones inesperadas si las variables usadas en la cadena concatenada contienen caracteres especiales como {.

Además, algunas veces esto puede ser el resultado de concatenar por error un argumento de formato de cadena al escribir un + cuando se pretendía una ,.

Ejemplo:


  String formatGreeting(String userName, int balance) {
    return MessageFormat.format("Hola, " + userName + "! Su saldo es {0}.", balance);
  }

Aquí, userName será interpretado como parte de la cadena de formato, lo que puede resultar en una IllegalArgumentException (por ejemplo, si userName es "{"). Esta llamada probablemente debería reemplazarse con MessageFormat.format("Hola, {0}! Su saldo es {1}.", userName, balance).
