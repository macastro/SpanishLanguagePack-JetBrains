else rama incluye condicional negada if sentencia.

. Normalmente, invirtiendo el orden de las ramas if y else se incrementa la claridad de este tipo de sentencias.

Hay una corrección rápida para invertir la sentencia if actual.

Ejemplo:


    void m(Object o1, Object o2) {
        if (o1 != o2) {
            System.out.println(1);
        }
        else {
            System.out.println(2);
        }
    }

Después de aplicar la corrección rápida:


    void m(Object o1, Object o2) {
        if (o1 == o2) {
            System.out.println(2);
        } else {
            System.out.println(1);
        }
    }

Usa la opción Ignorar comparación '!= null' para ignorar comparaciones de la forma != null.

Usa la opción Ignorar comparación '!= 0' para ignorar comparaciones de la forma!= 0.
