Informa sobre la asignación o modificación de campos static desde un método de instancia.

Aunque legítimas, esas asignaciones son difíciles de hacer de forma segura y a menudo son resultado de marcar campos como static inadvertidamente.

Ejemplo:


  class Counter {
    private static int count = 0;

    void increment() {
      // Advertencia: actualización de un campo static
      // desde un método de instancia
      count++;
    }
  }
