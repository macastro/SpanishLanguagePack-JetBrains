Informa de los métodos de servicios remotos que tienen tipos de parámetros no serializables, tipos de retorno o tipos de excepción lanzada.

El arreglo rápido extiende la interfaz correspondiente para marcar el tipo de referencia como serializable.

Deshabilita la opción Informar interfaces para omitir la comprobación de tipos que son interfaces. No se producirá ningún problema si la clase real que tiene una instancia pasada al método es serializable.
