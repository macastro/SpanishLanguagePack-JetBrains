Reporta instanciación de clases de utilidades utilizando la palabra clave new.

En clases de utilidades, todos los campos y métodos son static. La instanciación de tales clases probablemente sea innecesaria e indica un error.

Ejemplo:


  class MyUtils {
    public static double cube(double x) {
      return x * x * x;
    }
  }
  class Main {
    public static void main(String[] args) {
      // Instanciación de clase de utilidades
      MyUtils utils = new MyUtils();
    }
  }

Para evitar que las clases de utilidades sean instanciadas, se recomienda usar un constructor private.
