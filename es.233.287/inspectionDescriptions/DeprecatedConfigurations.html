Detecta el uso de métodos de configuración que fueron descontinuados.

Los métodos de configuración pueden ser descontinuados por dos posibles razones:
  • La API de Gradle ha evolucionado, por lo que se deben evitar las opciones de método antiguas;
  • Algún plugin proporciona una mejor versión del método de configuración obsoleto.
Lo más probable es que haya una alternativa para el método obsoleto.

Ejemplo:


plugins {
  id 'java'
}

dependencies {
  archive 'org.gradle.api:gradle:1.0' // reports 'archive'
}
