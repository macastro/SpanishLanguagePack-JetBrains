Crea el cuerpo de un método -isEqual: cuando se genera, por ejemplo, al llamar a la acción Código | Reemplazar método o Código | Generar | -isEqual: y -hash.
La plantilla es editable. Junto con las expresiones y comentarios de Objective-C, puedes usar variables predefinidas que serán expandidas con los valores correspondientes.
Las variables predefinidas tomarán los siguientes valores:
${CALL_SUPER} una llamada al super método, para métodos void — [super nombreMétodo] con o sin parámetros; para otros métodos — retorno [super nombreMétodo] con o sin parámetros
${RETURN_TYPE} el tipo de retorno del método creado
${DEFAULT_RETURN_VALUE} el valor retornado por el método por defecto
${CONTAINING_CLASS} el nombre de la clase que reemplaza el método
${CUSTOM_CODE} específico del tipo isEqualToTipo: el nombre del método de comparación, si se genera
Todas las variables predefinidas de la plantilla Archivo de cabecera de C (la pestaña Incluye) también están disponibles (${USER}, ${DATE}, etc.)
