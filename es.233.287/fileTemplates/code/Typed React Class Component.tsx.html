Esta plantilla integrada se utiliza para crear componentes de clase React, por ejemplo, a través de la refactorización Extraer componente. Puedes personalizar la plantilla, por ejemplo, usar Component en lugar de PureComponent o cambiar las definiciones de tipos de prop. También puedes utilizar variables predefinidas. Estas variables se reemplazarán por los valores correspondientes cuando se aplique la plantilla. Esta plantilla se utiliza en TypeScript y Flow.
Las variables predefinidas tomarán los siguientes valores:
${NAME} nombre del componente
${COMPONENT_BODY} cuerpo del componente
${PROP_TYPES} definición de tipo de propiedades
${PROP_TYPES} los valores de las propiedades predeterminados que utilizará React cuando no se proporcione ningún valor de la propiedad
