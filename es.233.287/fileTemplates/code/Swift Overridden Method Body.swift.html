Rellena el cuerpo del método a generar cuando un método sobreescrito (por ejemplo, usando la acción Código | Generar | Sobreescribir métodos).
La plantilla se puede editar. Además de expresiones y comentarios Swift, también puedes utilizar variables predefinidas que se expandirán al valor correspondiente.
Las variables predefinidas toman los siguientes valores:
${CALL_SUPER} Llamada al método super, para métodos Void: super.methodName() (con o sin parámetros). Para el resto de métodos: return super.methodName() (con o sin parámetros).
${RETURN_TYPE} Tipo de retorno del método que se crea
${DEFAULT_RETURN_VALUE} Valor que devuelve el método por defecto
${SIGNATURE} Firma del método
Todas las variables predefinidas de la plantilla Cabecera de archivo Swift (pestaña Incluye) también están disponibles (por ejemplo: ${USER}, ${DATE} ).
