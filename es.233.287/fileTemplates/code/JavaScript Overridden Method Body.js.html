Esta plantilla integrada se utiliza para rellenar el cuerpo de un método anulado cada vez que se genera, por ejemplo, cuando se llama a la función Código | Anular método....
La plantilla es editable. Junto con expresiones y comentarios de JavaScript, también puede utilizar variables predefinidas. Estas variables se reemplazarán por los valores correspondientes cuando se aplique la plantilla.
Las variables predefinidas tomarán los siguientes valores:
${CALL_SUPER} una llamada a un método super, para métodos void - super.nombre_método(); con o sin parámetros; para otros métodos - return super.nombre_método(); con o sin parámetros
${PLAIN_CALL_SUPER} una llamada a un método super, super.nombre_método(); con o sin parámetros;
${RETURN_TYPE} tipo de retorno de un método creado
${DEFAULT_RETURN_VALUE} valor devuelto por el método por defecto
${METHOD_NAME} nombre del método que se anula
Todas las variables predefinidas de la plantilla Encabezado de archivo (pestaña Incluir) también están disponibles (por ejemplo, ${USER}, ${DATE}, etc.)
