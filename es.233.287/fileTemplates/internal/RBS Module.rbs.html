Una plantilla integrada para un módulo RBS. Se utiliza cada vez que crea un nuevo módulo RBS seleccionando Nuevo | Archivo/clase RBS en el menú emergente de una de las vistas del proyecto. Junto con las declaraciones, expresiones y comentarios de RBS, también puede utilizar las variables predefinidas que se enumeran a continuación. Cada variable se expande como macros en el valor correspondiente. Por medio de la directiva #parse, puede incluir plantillas de la pestaña Includes especificando el nombre completo de la plantilla deseada como parámetro entre comillas.
Las variables predefinidas toman los siguientes valores:
${FILE_NAME} Nombre del archivo actual
${USER} Nombre de inicio de sesión del sistema del usuario actual
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${DAY} Día actual del mes
${HOUR} Hora actual
${MINUTE} Minuto actual
${PRODUCT_NAME} Nombre actual del IDE
${PROJECT_NAME} Nombre del proyecto actual
