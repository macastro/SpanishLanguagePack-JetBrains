Plantilla incorporada para pruebas de PHPUnit (versión 5 o anterior). Se utiliza cada vez que creas una nueva prueba de PHPUnit (Selecciona Nuevo | Prueba PHP en el menú emergente de una de las vistas del proyecto, luego selecciona 'PHPUnit < 6' como biblioteca de pruebas).
La plantilla es editable. Además de las expresiones, declaraciones y comentarios de PHP, también puedes utilizar las variables predefinidas que se indican a continuación. Cada variable se expandirá al valor correspondiente, tal como lo hace una macro.
Mediante la directiva #parse, se pueden incluir plantillas de la pestaña Incluir especificando el nombre completo de la plantilla deseada como parámetro entre comillas.
Las variables predefinidas toman los siguientes valores:
${TESTED_NAME} El nombre de la clase bajo prueba
${TESTED_NAMESPACE} Espacio de nombres de la clase bajo prueba
${NAME} Nombre de la clase de prueba
${NAMESPACE} Espacio de nombres de la clase de prueba
${FILE_NAME} Nombre del archivo actual
${USER} Nombre de inicio de sesión del sistema del usuario actual
${DATE} Fecha del sistema actual
${TIME} Hora del sistema actual
${YEAR} Año actual
${MONTH} Mes actual
${DAY} Día del mes actual
${HOUR} Hora actual
${MINUTE} Minuto actual
${PRODUCT_NAME} Nombre del IDE actual
${DS} El signo de dólar, evalúa al carácter '$' normal.
${PROJECT_NAME} Nombre del proyecto actual
