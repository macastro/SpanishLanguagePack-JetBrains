Aplicación Go es una plantilla integrada que se utiliza cada vez que se crea un nuevo archivo Go. A diferencia de la plantilla Archivo Go, Aplicación Go tiene una función main predefinida. Para usar esta plantilla, ve a Archivo | Nuevo, selecciona Archivo Go en la categoría Go y luego haz clic en Aplicación simple.

La plantilla es editable. En la plantilla, puedes utilizar las siguientes variables predefinidas. Estas variables se ampliarán más adelante en el editor al valor correspondiente.

También puedes especificar variables personalizadas. Las variables personalizadas utilizan el formato ${VARIABLE_NAME}, donde VARIABLE_NAME es el nombre de la variable (por ejemplo, ${MY_CUSTOM_FUNCTION_NAME}). Antes de que el IDE use variables personalizadas para crear un archivo nuevo, verás un cuadro de diálogo donde podrás definir el valor de las variables personalizadas de la plantilla.

Puedes incluir plantillas de la pestaña Incluir usando la directiva #parse. Para incluir una plantilla, especifica el nombre completo de la plantilla entre comillas como parámetro (por ejemplo, #parse("File Header")).
Lista de variables predefinidas
${GO_PACKAGE_NAME_WITH_TEST_SUFFIX} El nombre del paquete en el que se creará el nuevo archivo de prueba. El archivo debe tener el sufijo _test. El nombre tiene el formato package_test.
${GO_PACKAGE_NAME} El nombre del paquete en el que se creará el nuevo archivo.
${DATE} La fecha actual del sistema.
${DAY} El día del mes actual.
${DAY_NAME_SHORT} Las primeras 3 letras del nombre del día actual (por ejemplo, Lun, Mar etc.).
${DAY_NAME_FULL} El nombre completo del día actual (por ejemplo, Lunes, Martes, etc.).
${DIR_PATH} La ruta del directorio donde se guardará el nuevo archivo. Esta ruta es relativa a la raíz del proyecto.
${DS} El signo de dólar ($). Esta variable se utiliza para escapar el carácter dólar y así evitar que se trate como prefijo de variable de plantilla.
${FILE_NAME} El nombre del nuevo archivo.
${HOUR} La hora actual.
${MINUTE} El minuto actual.
${SECOND} El segundo actual.
${MONTH} El mes actual.
${MONTH_NAME_SHORT} Las primeras 3 letras del nombre del mes actual (por ejemplo, Ene, Feb etc.).
${MONTH_NAME_FULL} El nombre completo del mes actual (por ejemplo, Enero, Febrero etc.).
${NAME} El nombre de la nueva entidad (archivo, tipo, interfaz, etc.).
