Habilitar el modo de sólo lectura para conexiones

Para proteger las bases de datos de cambios accidentales, puede habilitar el modo de sólo lectura para las conexiones.

Para habilitar el modo de sólo lectura para las conexiones, haga clic en "Archivo | Fuentes de datos" y seleccione la fuente de datos que necesita en la lista "Fuentes de datos". En la pestaña "Opciones", marque la casilla de selección "Sólo lectura".
