Administración de las solicitudes pull

Puedes administrar las solicitudes pull de GitHub recibidas directamente desde PyCharmCommunity. Para ello, selecciona "Git | GitHub | Mostrar solicitudes pull" del menú principal. PyCharmCommunity te permite asignar y fusionar solicitudes pull, ver la línea de tiempo y los comentarios en línea, enviar comentarios y revisiones, y aprobar cambios sin tener que abandonar el IDE.
