Graba, reproduce y depura la aplicación con Mozilla rr

Mozilla rr es una herramienta que se puede usar para grabar, reproducir y depurar una aplicación. El propósito de Mozilla rr es ayudar a encontrar errores que no sean evidentes. La herramienta rr graba la ejecución de un programa. Esto permite que una ejecución que haya fallado se pueda reproducir después en el depurador tantas veces como sea necesario.

Selecciona "Grabar y depurar <configuration_name>" en el menú "Ejecutar" del gutter.

Mozilla rr solo se puede usar en máquinas Linux.

Para obtener información detallada sobre Mozilla rr, consulta el sitio web oficial de Mozilla rr.
