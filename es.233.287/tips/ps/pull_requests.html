Gestión de solicitudes de extracción

Administre las solicitudes de extracción entrantes de GitHub directamente desde PhpStorm. Desde el menú principalGit | GitHub | Mostrar solicitud de extracción Seleccionar . PhpStorm le permite asignar y fusionar solicitudes de extracción, mostrar una línea de tiempo y comentarios en línea, enviar comentarios y reseñas y aceptar cambios sin siquiera salir del IDE.
