Informa de instanciaciones de Throwable o sus subclases cuando el Throwable creado nunca se lanza.

El código informado indica errores que son difíciles de detectar en las pruebas.

Además, esta inspección informa de llamadas a métodos que devuelven instancias de Throwable o sus subclases, cuando la instancia Throwable resultante no se lanza.

Ejemplo:


  fun check(condition: Boolean) {
      if (!condition) /* falta el throw aquí */ IllegalArgumentException("condition is not met");
  }

  fun createError() = RuntimeException()

  fun foo() {
      /* falta el throw aquí */ createError()
  }
