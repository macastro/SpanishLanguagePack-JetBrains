Informa las instanciaciones de objetos java.util.Set cuyos tipos de contenido son clases enumeradas. Dichos objetos Set pueden remplazarse con objetos java.util.EnumSet.

Las implementaciones de EnumSet pueden ser mucho más eficientes en comparación con otros conjuntos, ya que la estructura de datos subyacente es un bit de vector. Utiliza la solución rápida para reemplazar el inicializador con una llamada a EnumSet.noneOf(). Esta solución rápida no está disponible cuando el tipo de variable es una subclase de Set.

Ejemplo:


  public enum MyEnum { FOO, BAR; }

  Set<MyEnum> enums = new HashSet<MyEnum>();

Después de aplicar la solución rápida:


  public enum MyEnum { FOO, BAR; }

  Set<MyEnum> enums = EnumSet.noneOf(MyEnum.class);
