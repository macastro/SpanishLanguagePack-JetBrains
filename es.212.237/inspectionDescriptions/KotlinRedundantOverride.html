Reporta declaraciones de anulación redundantes.

Se puede omitir una anulación si no modifica la semántica de la firma heredada, por ejemplo, cambiando la visibilidad.

Ejemplo:


  open class Foo {
      open fun singleExpression() {
      }
  }

  class Bar : Foo() {
      override fun singleExpression() = super.singleExpression()
  }

Después de aplicar la corrección rápida:


  class Bar : Foo() {
  }
