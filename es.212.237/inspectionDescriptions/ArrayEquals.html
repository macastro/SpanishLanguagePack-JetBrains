Informa de llamadas a equals() que comparan dos matrices.

Cuando se llama a equals() en una matriz, se compara la identidad, que es equivalente a utilizar ==. Utiliza Arrays.equals() para comparar los contenidos de dos matrices, o Arrays.deepEquals() para matrices multidimensionales.

Ejemplo:


  void sample(int[] first, int[] second){
    if (first.equals(second)) return;
  }

Después de aplicar la corrección rápida:


  void sample(int[] first, int[] second){
    if (Arrays.equals(first, second)) return;
  }
