Informa sobre la concatenación de cadenas que se puede convertir en una plantilla de cadena.

Se recomienda utilizar plantillas de cadena porque hacen que el código sea más fácil de leer.

Ejemplo:


  fun example() {
      val capitals = mapOf("Francia" to "Paris", "España" to "Madrid")
      for ((country, capital) in capitals) {
          print(capital + " es una capital de " + country)
      }
  }

Después de aplicar la solución rápida:


  fun example() {
      val capitals = mapOf("Francia" to "Paris", "España" to "Madrid")
      for ((country, capital) in capitals) {
          print("$capital es una capital de $country")
      }
  }
