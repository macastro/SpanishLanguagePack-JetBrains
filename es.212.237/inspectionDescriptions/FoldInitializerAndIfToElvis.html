Reporta una expresión if que verifica si una variable es nula o no justo después de inicializarla, y que se puede convertir en un operador Elvis en el inicializador.

Ejemplo:


  fun test(foo: Int?, bar: Int): Int {
      var i = foo
      if (i == null) {
          return bar
      }
      return i
  }

La corrección rápida convierte la expresión if con un inicializador en una expresión Elvis:


  fun test(foo: Int?, bar: Int): Int {
      var i = foo ?: return bar
      return i
  }
