Informa sobre la asignación o modificación de campos que se declaran en una superclase desde un constructor de una subclase.

Se considera preferible inicializar los campos de una superclase en su propio constructor y delegar en dicho constructor en una subclase. Esto también permitirá declarar un campo final si no se cambia después de la construcción.

Ejemplo:


  class Super {
    int x;
  }
  class Sub extends Super {
    Sub(int _x) {
      // Advertencia: x se declara en una superclase
      x = _x;
    }
  }

Para evitar el problema, declara un constructor de superclase:


  class Super {
    final int x;

    Super(int _x) {
      x = _x;
    }
  }
  class Sub extends Super {
    Sub(int _x) {
      super(_x);
    }
  }
