Reporta campos que no tienen garantía de ser inicializados después de que el objeto se deserializa por el método readObject().

La inspección no reporta campos transitorios.

Nota: Esta inspección usa un algoritmo de flujo de control muy conservador, y puede reportar campos como no inicializados de forma incorrecta.

Ejemplo:


class DataObject implements Serializable {
  String s; // s no se inicializa en readObject
  int i;

  private void readObject(ObjectInputStream stream) throws IOException {
    i = stream.readInt();
  }
}
