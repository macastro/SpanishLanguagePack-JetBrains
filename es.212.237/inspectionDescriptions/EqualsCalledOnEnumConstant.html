Informa sobre llamadas a equals() en constantes de enumeración.

Dichas llamadas pueden reemplazarse por una comparación de identidad (==) porque dos constantes de enumeración son iguales sólo cuando tienen la misma identidad.

Hay disponible una corrección rápida para cambiar la llamada a una comparación.

Ejemplo:


  boolean foo(MyEnum value) {
    return value.equals(MyEnum.FOO);
  }

Después de aplicar la corrección rápida:


  boolean foo(MyEnum value) {
    return value == MyEnum.FOO;
  }
