Reporta parámetros de método que siempre tienen el mismo valor constante.

Ejemplo:


  static void printPoint(int x, int y) { // x siempre es 0
    System.out.println(x + ", " + y);
  }

  public static void main(String[] args) {
    printPoint(0, 1);
    printPoint(0, 2);
  }

La corrección rápida inlinae el valor constante. Esto podría simplifcar la implemntación del método.

Usa la opción Ignorar cuando no se puede ofrecer una corrección rápida para suprimir las inspecciones cuando:

  • el parámetro es modificado dentro del método
  • el valor del parámetro que se está pasando es una referencia a un campo inaccesible (solo Java)
  • el parámetro es un vararg (solo Java)

Usa la opción Visibilidad máxima del método para controlar la máxima visibilidad de los métodos que se van a reportar.

Usa el campo Cuenta mínima de uso de métodos para reportar el parámetro para especificar el número mínimo de usos del método con el mismo valor de parámetro.
