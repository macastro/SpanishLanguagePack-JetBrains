Informa de comparaciones con llamadas a String.indexOf() que pueden ser reemplazadas con una llamada al método String.contains()..

Ejemplo:


  boolean b = "abcd".indexOf('e') >= 0;

Tras aplicar la corrección rápida:


  boolean b = "abcd".contains('e');

Esta inspección sólo informa si el nivel de idioma del proyecto o módulo es 5 o superior.
