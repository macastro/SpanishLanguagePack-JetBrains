Informa de las llamadas a java.lang.Thread.sleep() que se producen dentro de los bucles.

Dichas llamadas son indicativas de "espera activa". La espera activa suele ser ineficiente y puede dar lugar a interbloqueos inesperados ya que los subprocesos en espera activa no liberan los recursos bloqueados.

Ejemplo:


  class X {
     volatile int x;
     public void waitX() throws Exception {
      while (x > 0) {
        Thread.sleep(10);// advertencia: llamada a 'Thread.sleep()' en un bucle, probablemente espera activa
      }
    }
  }
