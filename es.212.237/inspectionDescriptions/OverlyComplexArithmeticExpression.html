Informa de las expresiones aritméticas con un número excesivo de términos. Estas expresiones pueden resultar difíciles de entender y podrían contener errores.

Los parámetros, las referencias de campo y otras expresiones primarias se cuentan como un término.

Ejemplo: 

int calc(int a, int b) {
        return a + a + a + b + b + b + b; // La línea contiene 7 términos y se informará.
    }

Usa el campo siguiente para especificar el número de términos permitidos en expresiones aritméticas.
