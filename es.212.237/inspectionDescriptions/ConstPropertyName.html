Informa nombres de propiedad const que no siguen las convenciones de nomenclatura recomendadas.

Una nomenclatura constante permite una lectura y comprensión de código más sencilla. De acuerdo con la guía de estilo oficial de Kotlin, las propiedades const deben usar nombres separados por guión bajo y en mayúsculas.

Ejemplo:


  const val Planck: Double = 6.62607015E-34

El arreglo rápido renombra la propiedad:


  const val PLANCK: Double = 6.62607015E-34
