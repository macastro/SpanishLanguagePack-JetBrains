Reporta cláusulas catch que capturan una excepción inapropiada.

Algunas excepciones, por ejemplo java.lang.NullPointerException o java.lang.IllegalMonitorStateException, representan errores de programación y por lo tanto casi con seguridad no deberían ser capturadas en un código de producción.

Ejemplo: 


  try {
    return component.getMousePosition(true) != null;
  } catch (NullPointerException e) { // warning: Prohibited exception 'NullPointerException' caught
    return false;
  }

Utiliza la lista de Excepciones prohibidas para especificar qué excepciones deberían ser reportadas.
