Inspecciona Clases Internas Calificadas Innecesariamente

Reporta toda referencia a clases internas que están calificadas innecesariamente con el nombre de la clase contenedora.

Tal calificación se puede eliminar de forma segura, que a veces añade una importación para la clase interna.

Ejemplo:


  class X {
     X.Y foo;
     class Y{}
  }

Después de aplicar la corrección rápida:


  class X {
     Y foo;
     class Y{}
  }

Usa la opción Ignorar referencias para las que se necesita una importación para ignorar referencias a clases internas, donde eliminar la calificación añade una importación.
