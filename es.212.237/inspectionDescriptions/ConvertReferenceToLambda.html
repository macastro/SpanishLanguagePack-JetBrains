Reporta una expresión de referencia de función que puede reemplazarse con un literal de función (lambda).

En algunas ocasiones, pasar una lambda se ve más directo y consistente con el resto del código. Además, el arreglo puede ser útil si necesitas reemplazar una llamada simple con algo más complejo.

Ejemplo:


  fun Int.isEven() = this % 2 == 0

  fun example() {
      val numbers = listOf(1, 2, 4, 7, 9, 10)
      val evenNumbers = numbers.filter(Int::isEven)
  }

Después de aplicar la corrección rápida:


  fun Int.isEven() = this % 2 == 0

  fun example() {
      val numbers = listOf(1, 2, 4, 7, 9, 10)
      val evenNumbers = numbers.filter { it.isEven() }
  }
