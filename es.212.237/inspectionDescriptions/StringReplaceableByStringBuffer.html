Reporta variables declaradas como java.lang.String que se agregan repetidamente. Dichas variables podrían declararse de manera más eficiente como java.lang.StringBuffer o java.lang.StringBuilder.

Ejemplo:


  String s = "";
  for (int i = 0; i < names.length; i++) {
    String name = names[i] + (i == names.length - 1 ? "" : " ");
    s = s + name;
  }

Tal bucle puede ser reemplazado con:


  StringBuilder s = new StringBuilder();
  for (int i = 0; i < names.length; i++) {
    String name = names[i] + (i == names.length - 1 ? "" : " ");
    s.append(name);
  }

O incluso con:


  String s = String.join(" ", names);

Utiliza la opción para hacer que esta inspección solo informe cuando la variable se agrega en un bucle.
