Informa el uso de miembros static calificados innecesariamente por el nombre de clase.

Cuando un miembro static puede ser utilizado en la clase circundante o en la superclase de la clase circundante, no es necesario calificarlo con la clase. Dicha calificación se puede eliminar de forma segura.

Ejemplo:


  class Foo {
    static void foo() {}
    static int x;

    void bar() {
      Foo.foo();
      System.out.println(Foo.x);
    }

    static void baz() { Foo.foo(); }
  }

Después de aplicar la solución rápida:


  class Foo {
    static void foo() {}
    static int x;

    void bar() {
      foo();
      System.out.println(x);
    }

    static void baz() { foo(); }
  }

Utiliza las opciones de inspección para alternar qué denunciar:

  • Acceso a un campo static:
    void bar() { System.out.println(Foo.x); }

  • Llamada a un método static:
    void bar() { Foo.foo(); }

Además, puedes configurar la inspección para que solo informe el uso de miembros static en un contexto static. En ese caso, solo se informaría static void baz() { Foo.foo(); }.
