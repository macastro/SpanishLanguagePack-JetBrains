Notifica condiciones de comprobación de nulos y sugiere reemplazarlas con cadenas Optional.

Ejemplo:


  return str == null ? "" : str.trim();

Después de aplicar la corrección rápida:


  return Optional.ofNullable(str).map(String::trim).orElse("");

Aunque el reemplazo no siempre resulta más breve, podría ser útil para realizar refactorizaciones adicionales (por ejemplo, para cambiar el valor de retorno del método a Optional).

Tenga en cuenta que cuando una rama no nula de la condición devuelve null, el paso de asignación correspondiente producirá un Optional vacío, lo cual posiblemente cambie la semántica. Si no puede comprobarse estáticamente que se preservará la semántica, el nombre de la acción de corrección rápida contendrá el aviso "(puede cambiar la semántica)" y se desactivará el resaltado de inspección.
