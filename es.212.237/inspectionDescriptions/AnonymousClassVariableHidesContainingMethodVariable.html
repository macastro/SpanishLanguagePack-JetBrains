Informa sobre campos en una clase anónima que tienen el mismo nombre que variables locales o parámetros del método contenedor o expresión lambda.

Como resultado de dicha denominación, es posible que uses accidentalmente el campo de clase anónimo donde se propone la variable o parámetro con el mismo nombre del método contenedor.

Se sugiere una corrección rápida para cambiar el nombre del campo.

Ejemplo:


  class Test {
    public Test(String value) {
      Object foo = new Object() {
        private String value = "TEST";
        public void foo() {
          System.out.println(value); //se accede al campo y no al parámetro
        }
      };
    }
  }
