Reporta un operador unario seguido de un calificador de punto como -1.inc().

Código como -1.inc() puede llevar a error porque - tiene una precedencia menor que .inc(). Como resultado, -1.inc() se evalúa como -2 y no como 0 como podría esperarse.

El arreglo rápido Envolver el operador unario y el valor con () puede usarse para enmendar el código automáticamente.
