Reporta las clases de utilidad que no son final o abstract.

Las clases de utilidad tienen todos los campos y métodos declarados como static. Hacerlas final evita que sean subclaseadas accidentalmente.

Ejemplo:


  public class UtilityClass {
    public static void foo() {}
  }

Después de aplicar la corrección rápida:


  public final class UtilityClass {
    public static void foo() {}
  }
