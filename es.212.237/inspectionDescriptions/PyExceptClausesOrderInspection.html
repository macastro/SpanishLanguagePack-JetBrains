Informa sobre los casos en los que las cláusulas de except no están en el orden correcto, desde las más específicas a las más genéricas, o cuando una clase de excepción se captura dos veces.

Si no se corrige el orden, es posible que algunas excepciones no sean capturadas por el controlador más específico.

Ejemplo:


try:
    call()
except ValueError:
    pass
except UnicodeError:
    pass

El IDE recomienda mover la cláusula hacia arriba. Cuando se aplica la corrección rápida, el código cambia a:


try:
    call()
except UnicodeError:
    pass
except ValueError:
    pass
