Reporta ciclos for con una variable de índice incrementada manualmente.

Los ciclos for con una variable de índice incrementada manualmente pueden simplificarse con la función withIndex().

El arreglo rápido "Usar withIndex() en lugar del incremento manual del índice" puede utilizarse para enmendar el código automáticamente.

Ejemplo:


  fun foo(list: List<String>): Int? {
      var index = 0
      for (s in list) { <== puede simplificarse
          val x = s.length * index
          index++
          if (x > 0) return x
      }
      return null
  }

Después de aplicar el arreglo rápido:


  fun foo(list: List<String>): Int? {
      for ((index, s) in list.withIndex()) {
          val x = s.length * index
          if (x > 0) return x
      }
      return null
  }
