Reporta llamadas recursivas de equals(==).

En Kotlin, == compara valores de objetos llamando al método equals bajo el capó. Por otro lado, === compara objetos por referencia.

=== se usa comúnmente en la implementación del método equals. Pero === puede mezclarse erróneamente con ==, lo que lleva a una recursión infinita.

Ejemplo:


  class X {
      override fun equals(other: Any?): Boolean {
          if (this == other) return true
          return false
      }
  }

Después de que se aplique la corrección rápida:


  class X {
      override fun equals(other: Any?): Boolean {
          if (this === other) return true
          return false
      }
  }
