Informa métodos booleanos cuyos nombres no empiezan con una palabra de pregunta.

Los métodos booleanos que sobrescriben métodos de biblioteca son ignorados por esta inspección.

Ejemplo:

boolean empty(List<String> list) {
  return list.isEmpty();
}

Una corrección rápida que renombra dichos métodos solo está disponible en el editor.

Configurar la inspección:

  • Utiliza la lista Prefijos de nombre de método booleano para especificar palabras de pregunta aceptables para empezar a nombrar métodos booleanos.
  • Utiliza la opción Ignorar métodos con tipo de retorno 'java.lang.Boolean' para ignorar métodos con tipo de retorno java.lang.Boolean.
  • Utiliza la opción Ignorar métodos booleanos en un @interface para ignorar métodos booleanos en types de anotación (@interface).
  • Utiliza Ignorar métodos que sobrescriben/implementan un supermétodo para ignorar métodos que tienen super.
