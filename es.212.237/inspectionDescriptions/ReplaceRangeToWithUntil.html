Reporta llamadas a rangeTo o al operador .. en lugar de llamadas a until.

El uso de las funciones correspondientes hace que el código sea más simple.

La corrección rápida reemplaza la llamada a rangeTo o .. con until.

Ejemplo:


  fun foo(a: Int) {
      for (i in 0..a - 1) {

      }
  }

Después de aplicar la corrección rápida:


  fun foo(a: Int) {
      for (i in 0 until a) {

      }
  }
