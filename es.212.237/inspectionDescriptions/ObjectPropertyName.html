Reporta propiedades que no siguen los acuerdos para nombrar.

Las siguientes propiedades son reportadas:

  • Propiedades de alto nivel
  • Propiedades en objetos y objetos partner

Puedes especificar el patrón requerido en las opciones de inspección.

Acuerdos para nombrar recomendados: deben empezar con una letra mayúscula, usar mayúsculas y minúsculas y sin subrayados.

Ejemplo:


  // propiedad de alto nivel
  val NOMBRE_DE_USUARIO_CAMPO = "NombreDeUsuario"
  // propiedad de alto nivel con referencia al objeto singular
  val PersonComparator: Comparator<Person> = /*...*/

  class Persona {
    companion object {
      // propiedad en objeto partner
      val SIN_NOMBRE = Persona()
    }
  }
