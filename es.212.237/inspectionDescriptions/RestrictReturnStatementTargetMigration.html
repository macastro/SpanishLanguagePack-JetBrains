Reporta las etiquetas que no apuntan a funciones.

Se prohíbe declarar una etiqueta objetivo que no indique una función.

La corrección rápida elimina la etiqueta.

Ejemplo:


  fun testValLabelInReturn() {
      L@ val fn = { return@L }
      fn()
  }

Después de aplicar la corrección rápida:


  fun testValLabelInReturn() {
      L@ val fn = { return }
      fn()
  }

Esta inspección solo informa si el nivel de lenguaje del proyecto o módulo es 1.4 o superior.
