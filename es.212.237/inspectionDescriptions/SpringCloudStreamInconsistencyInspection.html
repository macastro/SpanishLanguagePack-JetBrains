Informa errores de inconsistencias en los métodos manejadores de flujos.

Los métodos manejadores de flujos de entrada y salida anotados con @StreamListener o @StreamEmitter tienen varias restricciones. La violación de las restricciones resulta en errores en tiempo de ejecución.

Por ejemplo:

  • El canal definido en el atributo value de @StreamListener nunca se puede combinar con parámetros del método anotados con @Input o @Output
  • Se debe especificar un canal de entrada para un método anotado con @StreamListener
  • Se debe especificar un canal de salida para un método que pueda devolver un valor
  • No se puede especificar un canal de salida para un método que no devuelve un valor
  • El atributo condition de @StreamListener no se puede establecer para métodos que devuelven un valor
