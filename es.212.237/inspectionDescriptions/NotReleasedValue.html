지역 변수 또는 매개변수에 저장된 소유한 객체 중 나중에 릴리스되지 않았거나 릴리스되기 전에 다른 값으로 덮어써지는 객체를 보고합니다.
 - 이름이 alloc 또는 new로 시작하거나 copy를 포함하는 메서드를 사용해 객체를 생성했거나 retain 메시지를 객체에 전송한 경우, 객체의 소유권을 가지게 됩니다.
 - 객체의 소유권을 포기하려면 release 또는 autorelease를 사용하세요.
Objective-C 메모리 관리 규칙에 따르면 소유한 각 객체는 릴리스되거나 자동으로 릴리스되어야 합니다. 메모리 관리 프로그래밍 가이드를 참조하세요.
자동 참조 집계(ARC)를 사용하면 이 검사가 비활성화됩니다.
