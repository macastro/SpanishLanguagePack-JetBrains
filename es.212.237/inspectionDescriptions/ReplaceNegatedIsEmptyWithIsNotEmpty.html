Reporta la negación isEmpty() e isNotEmpty() para colecciones y String, o isBlank() e isNotBlank() para String.

Usar las funciones correspondientes simplifica tu código.

La corrección rápida reemplaza la llamada de negación con la llamada correspondiente de la librería estándar.

Ejemplo:


  fun main() {
      val list = listOf(1,2,3)
      if (!list.isEmpty()) {
          // hacer algo
      }
  }

Después de aplicar la corrección rápida:


  fun main() {
      val list = listOf(1,2,3)
      if (list.isNotEmpty()) {
          // hacer algo
      }
  }
