Informa sobre las propiedades de extensión que entran en conflicto con las sintéticas producidas automáticamente a partir de los métodos Java get o set.

Estas propiedades deben eliminarse o renombrarse para evitar que el código se rompa por futuros cambios en el compilador.

La solución rápida elimina una propiedad de extensión.

Ejemplo:


  val File.name: String
      get() = getName()

La solución rápida agrega la anotación @Deprecated:


  @Deprecated("Es reemplazado con extensión sintética automática", ReplaceWith("name"), level = DeprecationLevel.HIDDEN)
  val File.name: String
      get() = getName()
