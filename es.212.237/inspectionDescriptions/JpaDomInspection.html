Informa los siguientes problemas dentro de archivos de configuración XML:
  • Referencias a clases no instanciables
  • Referencias a clases que no extienden la clase requerido
  • Referencias a clases con alcance inapropiado
  • Valores de tags y atributos que no coinciden con el patrón requerido (por ejemplo, identificadores de Java)
  • Tags que no incluyen tags o atributos requeridos
  • Tags que definen objetos con nombres duplicados

Ejemplo:



<persistence>
    <persistence-unit name="example-persistence-unit">
        <provider>org.example.SomeRandomClass</provider> // Error: la clase del proveedor debe implementar la interfaz PersistenceProvider
    </persistence-unit>
</persistence>
