Inspección sobre tipos que implementan clases anotadas con una anotación JSF especificada y no implementan la interfaz esperada.

  • @FacesComponent debe extender <javax/jakarta>.faces.component.FacesComponent
  • @FacesRenderer debe extender <javax/jakarta>.faces.render.Renderer
  • @FacesConverter debe extender <javax/jakarta>.faces.convert.Converter
  • @FacesValidator debe extender <javax/jakarta>.faces.validator.Validator

Ejemplos:



@FacesConverter(value="...")  // La clase debe extender 'javax.faces.convert.Converter'
public class FooConverter {
}

@FacesRenderer(rendererType="..", componentFamily="..")  // La clase debe extender 'javax.faces.render.Renderer'
public class FooRenderer {
}

@FacesValidator(value="...")  // La clase debe extender 'javax.faces.validator.Validator'
public class FooValidator {
}
