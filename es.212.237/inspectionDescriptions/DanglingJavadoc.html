Informa sobre comentarios Javadoc que no pertenecen a ninguna clase, método o campo. La herramienta Javadoc ignora los comentarios Javadoc aislados y no los incluye en la documentación HTML que genera.

Ejemplo:


  class A {
    /**
     * Comentario aislado
     */
    /**
     * Javadoc del método
     */
     public void m(){}
  }

Hay una corrección rápida disponible para eliminar por completo dichos comentarios o convertirlos en un comentario de bloque. Después de aplicar la corrección rápida:


  class A {
    /*
      Comentario aislado
    */
    /**
     * Javadoc del método
     */
    public void m(){}
  }

Use la opción Ignorar comentario de encabezado de archivo en formato JavaDoc para ignorar comentarios al principio de los archivos Java. Normalmente, estos son mensajes de derechos de autor.
