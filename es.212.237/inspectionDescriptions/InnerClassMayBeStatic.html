Reporta clases internas que pueden ser hechas static.

Una clase interna static no mantiene una referencia implícita a su instancia envolvente. Esto previene una causa común de fugas de memoria y usa menos memoria por instancia de la clase.

Ejemplo:


  public class Outer {
    class Inner { // no static
      public void foo() {
        bar("x");
      }

      private void bar(String string) {}
    }
  }

Después de que se aplica la solución rápida:


  public class Outer {
    static class Inner {
      public void foo() {
        bar("x");
      }

      private void bar(String string) {}
    }
  }
