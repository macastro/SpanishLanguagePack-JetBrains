Reporta los siguientes errores de especificación JPA:
  • Varios atributos están configurados para un método o campo
  • Atributo ilegal utilizado en un objeto incrustable (JPA 1.0)
  • Metadatos de atributos persistentes no se especifican ni en un campo o propiedad getter
  • No hay setter o getter con acceso basado en propiedades

Ejemplo:


  @Entity
  @Table(name = "TABLE")
  public class JavaEntity {

    private Integer id; // Error: ambos getter y setter deben estar presentes

    public Integer getId() { ... }



    private String data;

    public String getData() { ... }

    @Basic // Error: la anotación de persistencia debe colocarse en campo o método getter
    public void setData(String data) { ... }
  }
