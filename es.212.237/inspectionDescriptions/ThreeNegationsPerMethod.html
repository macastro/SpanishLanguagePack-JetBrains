Informa sobre métodos que presentan tres o más negaciones. Estos métodos podrían resultar confusos.

Ejemplo:


  void doSmth(int a, int b, boolean flag1, boolean flag2) {
    if (!flag && !flag2) {
      if (a != b) {
        doOther();
      }
    }
  }

La eliminación de las negaciones facilita la comprensión del método:


  void doSmth(int a, int b, boolean flag1, boolean flag2) {
    if (flag1 || flag2 || a == b) return;
    doOther();
  }

Cómo configurar la inspección:

  • Utiliza la opción Ignorar negaciones en los métodos "equals()" para desactivar la inspección en los métodos equals().
  • Utiliza la opción Ignorar negaciones en las sentencias "assert" para desactivar la inspección en las sentencias assert.
