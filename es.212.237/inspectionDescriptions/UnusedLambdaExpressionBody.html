Reporta llamadas con un valor de retorno no utilizado cuando la función llamada devuelve una lambda de un cuerpo de expresión.

Si hay = entre el encabezado de la función y el bloque del cuerpo, el código de la función no se evaluará, lo que puede provocar un comportamiento incorrecto.

Se puede utilizar Eliminar token = de la declaración de función para modificar el código automáticamente.

Ejemplo:


  fun printHello() = { println("Hello") }

  fun main() {
      printHello() // Esta función no imprime nada
  }

Después de aplicar la corrección rápida:


  fun printHello() { println("Hello") }

  fun main() {
      printHello()
  }
