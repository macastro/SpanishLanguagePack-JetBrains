Reporta declaraciones de variables de Collection hechas usando la clase de collection como un tipo, en lugar de una interfaz apropiada. La advertencia no se emite si el debilitamiento del tipo de variable causa un error de compilación.

Ejemplo:


  // Advertencia: clase de colección concreta ArrayList usada.
  int getTotalLength(ArrayList<String> list) {
    return list.stream().mapToInt(String::length).sum();
  }

  // Sin advertencia, como el método trimToSize() no está
  // disponible en la interfaz List
  void addData(ArrayList<String> data) {
    data.add("Hola");
    data.add("Mundo");
    data.trimToSize();
  }
Se sugiere una solución rápida para usar la interfaz de colección apropiada (por ejemplo, Collection, Set o List).
