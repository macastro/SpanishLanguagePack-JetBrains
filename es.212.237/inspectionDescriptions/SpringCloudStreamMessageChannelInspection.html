Informa de los atributos de canal no resueltos en anotaciones @StreamListener, @SendTo, @Output y @Input.

Usar un nombre de enlace no válido causa un error del tipo "no se ha encontrado el bean" en tiempo de ejecución.

Ejemplo:


  @EnableBinding(Sink.class)
  public class LogSource {
    @StreamListener(Sink.class)
	  public void log1() {
    }

    @StreamListener("invalid")  // Informa de 'No se encuentra el canal'
	  public void log2() {
    }
  }
