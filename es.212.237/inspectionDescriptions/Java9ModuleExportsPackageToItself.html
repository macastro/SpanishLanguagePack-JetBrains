Reporta los paquetes que son exportados o abiertos en el mismo módulo Java 9 en el que fueron definidos. El arreglo rápido elimina tales directivas de module-info.java.

Ejemplo:


  module com.mycomp {
    exports com.mycomp.main to com.mycomp;
  }

Después de aplicar el arreglo rápido:


  module main {
  }
