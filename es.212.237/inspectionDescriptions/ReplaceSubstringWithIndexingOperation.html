Informa llamadas como "abc".substring(0, 1) que pueden remplazarse con "abc"[0].

Obtener el elemento por índice hace tu código más simple.

La corrección rápida reemplaza la llamada de substring con el operador de indexación.

Ejemplo:


  fun foo() {
      "abc".substring(0, 1)
  }

Después de aplicar la corrección rápida:


  fun foo() {
      "abc"[0]
  }
