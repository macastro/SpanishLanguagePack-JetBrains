Informa de llamadas sospechosas a System.arraycopy().

Estas llamadas son sospechosas cuando:

  • el origen o destino no es de un tipo de arreglo
  • el origen y destino son de diferentes tipos
  • la longitud del fragmento copiado es mayor que src.length - srcPos
  • la longitud del fragmento copiado es mayor que dest.length - destPos
  • los rangos siempre se intersectan cuando el origen y el destino son el mismo arreglo

Ejemplo:


  void foo() {
    int[] src = new int[] { 1, 2, 3, 4 };
    System.arraycopy(src, 0, src, 1, 2); // advertencia: Copiando al mismo arreglo con rangos intersecados
  }
