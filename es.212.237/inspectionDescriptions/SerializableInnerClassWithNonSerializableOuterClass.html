Reporta clases internas no estáticas que implementan el tipo Serializable y se declaran dentro de una clase que no implementa Serializable.

Es poco probable que dichas clases se serialicen correctamente debido a referencias implícitas a la clase externa.

Ejemplo:


  class A {
    class Main implements Serializable {
    }
  }

Usa las siguientes opciones para configurar la inspección:

  • Lista las clases cuyos herederos no deben ser reportados por esta inspección. Esto es para clases que heredan Serializable de una superclase pero no están pensadas para serialización.
  • Si ignorar o no clases anónimas Serializable.
