Informa sobre métodos no booleanos cuyos nombres empiezan con una palabra interrogativa. Dichos nombres de método pueden resultar confusos.

Este control ignora los métodos no booleanos que sobrescriben los métodos de la biblioteca.

Ejemplo: 


  public void hasName(String name) {
    assert names.contains(name);
  }

Solo en el editor está disponible una solución rápida que cambia el nombre de dichos métodos.

Configurar la inspección:

  • Usa la lista Prefijos de los nombres de método booleano para especificar las palabras interrogativas que deben usarse solo para los métodos booleanos.
  • Usa la opción Ignorar métodos con tipo de retorno 'java.lang.Boolean' para ignorar los métodos con tipo de retorno java.lang.Boolean.
  • Usa la opción Ignorar métodos que sobrescriben/implementan un método super para ignorar los métodos que tienen supermétodos.
