Reporta cadenas de un solo carácter que se utilizan como argumento en llamadas a String.indexOf() y String.lastIndexOf().

Se sugiere un arreglo rápido para reemplazar tales literales de cadena con literales de carácter equivalentes, obteniendo alguna mejora de rendimiento.

Ejemplo:


  return s.indexOf("x");

Después de aplicar el arreglo rápido:


  return s.indexOf('x');
