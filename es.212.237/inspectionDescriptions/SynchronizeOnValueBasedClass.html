Informa de intentos de sincronización en una instancia de una clase basada en valores que producen advertencias en tiempo de compilación y lanzan excepciones en tiempo de ejecución a partir de Java 16.

Por ejemplo, java.lang.Double está anotado con jdk.internal.ValueBased, por lo que el siguiente código producirá una advertencia en tiempo de compilación: 


Double d = 20.0;
synchronized (d) { ... } // advertencia de javac
