Inspecciona código que sobrescribe una clave de un Map, un elemento de un Set o un elemento de un arreglo en una secuencia de llamadas add/put usando un método factory de Java 9 como Set.of (lo cual resultará en una excepción de tiempo de ejecución).

Esto por lo general sucede por un error de copiar y pegar.

Ejemplo:


  map.put("A", 1);
  map.put("B", 2);
  map.put("C", 3);
  map.put("D", 4);
  map.put("A", 5); // se duplica la clave "A", se sobrescribe la entrada escrita anteriormente
