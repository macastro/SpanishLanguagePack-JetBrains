Informa del constructor principal private en clases de datos.

Las clases data tienen un método de fábrica copy() que se puede usar de forma similar a un constructor. Un constructor no debe marcarse como private para proporcionar suficiente seguridad.

Ejemplo:


  data class Usuario private constructor(val name: String)

La solución rápida cambia el modificador de visibilidad del constructor a public:


  data class Usuario(val name: String)
