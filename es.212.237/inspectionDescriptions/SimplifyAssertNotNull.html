Informa las llamadas `assert` que verifican un valor distinto de nulo de la variable declarada.

Usar `!!` o `?:` simplifica tu código.

La corrección rápida reemplaza `assert` con el operador `!!` o `?:` en el inicializador de la variable.

Ejemplo:


  fun foo(p: Array<String?>) {
      val v = p[0]
      assert(v != null, { "Debe ser distinto de nulo" })
  }

Después de aplicar la corrección rápida:


  fun foo(p: Array<String?>) {
      val v = p[0] ?: error("Debe ser distinto de nulo")
  }
