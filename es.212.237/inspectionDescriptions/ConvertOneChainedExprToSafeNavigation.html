Informa de llamadas encadenadas que pueden simplificarse con el operador de navegación segura (&.). Un grupo de llamadas encadenadas es un patrón común para comprobar si hay nil en cada componente de llamada.

Ejemplo:

# Mala práctica
if user && user.profile && user.profile.thumbnails
end
Hay disponible una corrección rápida para simplificar la expresión con &.. Después de aplicar la corrección rápida, el resultado es el siguiente: 
# Buena práctica
if user&.profile&.thumbnails
end
