Informa iteraciones sobre keySet() de una instancia de java.util.Map, donde las llaves iteradas se usan para recuperar los valores del mapa.

Tal iteración podría ser más eficiente cuando se reemplaza con una iteración sobre el entrySet() o values() (si la llave no se usa realmente).

De manera similar, keySet().forEach(key -> ...) se puede reemplazar con forEach((key, value) -> ...) si los valores se recuperan dentro de una lambda.

Ejemplo:


  for (Object key : map.keySet()) {
   Object val = map.get(key);
  }

Después de aplicar la solución rápida:


  for (Object val : map.values()) {}
