Reporta las sentencias return, throw, break, continue, y yield que se utilizan dentro de los bloques finally. Esto provoca que el bloque finally no se complete normalmente, sino que se complete abruptamente. Cualquier excepción lanzada desde los bloques try y catch de la misma sentencia try-catch quedará suprimida.

Ejemplo:


  void x() {
    try {
      throw new RuntimeException();
    } finally {
      // si bar() devuelve true, la RuntimeException quedará suprimida
      if (bar()) return;
    }
  }
