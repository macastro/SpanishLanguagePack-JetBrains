Reporta expresiones when con las ramas constantes true o false.

El arreglo rápido "Simplicar cuando" puede ser utilizado para modificar el código automáticamente.

Ejemplos:


  fun redundante() {
      when { // <== redundante, el arreglo rápido simplifica la expresión when a "println("true")"
          true -> println("true")
          else -> println("false")
      }
  }
