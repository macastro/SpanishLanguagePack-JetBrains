Informa las clases de prueba JUnit 3 que pueden convertirse a las clases de prueba JUnit 4.

Ejemplo:


  public class MainTestCase extends junit.framework.TestCase {
    public void test() {
      Assert.assertTrue(true);
    }
  }

Después de aplicar la corrección rápida:


  public class MainTestCase {
    @org.junit.Test
    public void test() {
      Assert.assertTrue(true);
    }
  }

Esta inspección sólo informa si el nivel del lenguaje del proyecto o módulo es 5 o superior y JUnit 4 está disponible en la ruta de clase.
