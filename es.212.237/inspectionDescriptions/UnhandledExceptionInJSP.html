Informa sobre las excepciones lanzadas por los scriptlets JSP de nivel superior.

Ejemplo:


<%!
String getFormattedDate() {
 ...
 throw new IllegalArgumentException(...); // Error
}
%>
<html>
<head>
 <meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
 <title>¡Hola, JSP!</title>
</head>
<body>
<i>Hoy es <%= getFormattedDate() %></i>
</body>
</html>

NOTA: Estas páginas JSP se compilarán porque el contenedor del servlet envuelve todo el código JSP en un bloque try {} catch() {}, pero producirán excepciones de tiempo de ejecución cuando se implementen en el servidor.
