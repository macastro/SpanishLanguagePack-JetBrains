Reporta expresiones `instanceof` donde la expresión que es verificada tiene un tipo de clase/interfaz que no extiende/implementa el tipo de clase/interfaz en el lado derecho de la expresión `instanceof`, ni tiene subclases que lo hagan.

A pesar de que podría ser para código de biblioteca por ejemplo, tal construcción es probablemente errónea, porque ninguna instancia de ninguna clase declarada en el proyecto podría pasar esta prueba `instanceof`.

Ejemplo:


  class Foo { }

  interface Bar { }
  
  class Main {
      void test(Foo f, Bar b) {
          if (f instanceof Bar) { // problema
              System.out.println("fail");
          }
          if (b instanceof Foo) { // problema
              System.out.println("fail");
          }
      }
  }
