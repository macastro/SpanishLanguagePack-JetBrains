Informa de los métodos con demasiados puntos de retorno. Los métodos con demasiados puntos de retorno pueden resultar confusos y difíciles de refactorizar.

Ejemplo:


  int foo(int a) {
    if (a > 0) {
      return a
    }
    if (a < 0) return -a
    return 0
  }

Utiliza el campo proporcionado a continuación para especificar el número máximo aceptable de puntos de retorno que puede tener un método.
