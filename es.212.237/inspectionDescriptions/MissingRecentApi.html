Informa sobre el uso de las API de la plataforma IntelliJ que se introdujeron en una versión superior a la especificada en la versión actual. La versión se especifica en <idea-version> @since-build (en plugin.xml).

El uso de estas API puede provocar problemas de compatibilidad con las versiones anteriores del IDE y del plugin.

Para evitar posibles problemas al ejecutar el plugin en versiones anteriores del IDE, incrementa since-build en consecuencia o elimina el uso de esta API.

Para obtener más información, consulta los rangos de número de compilación en la documentación del SDK del plugin para la plataforma IntelliJ.

Configuración de la inspección:

Si no se especifican los atributos @since/until-build de <idea-version> en plugin.xml, establece Desde/Hasta de forma explícita.
