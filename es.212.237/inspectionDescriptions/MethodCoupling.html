Inspecciona métodos que hacen referencia a demasiadas otras clases. Los métodos con un acoplamiento demasiado alto pueden ser muy frágiles y probablemente deberían dividirse en métodos más pequeños.

Cada clase referenciada se cuenta solo una vez, sin importar cuántas veces se haga referencia a ella.

Configurar la inspección:

  • Utilice el campo Límite de acoplamiento del método para especificar el acoplamiento máximo permitido para un método.
  • Utilice la opción Incluir acoplamientos a clases del sistema Java para contar referencias a clases de paquetes java o javax.
  • Utilice la opción Incluir acoplamientos a clases de biblioteca para contar referencias a clases de bibliotecas de terceros.
