Reporta llamadas sospechosas a Collection.toArray().

Los siguientes tipos de llamadas se consideran sospechosas:

  • cuando el tipo del argumento de la matriz no es el mismo que el tipo de matriz al que se lanza el resultado.
  • cuando el tipo del argumento de la matriz no coincide con el parámetro de tipo en la declaración de la colección.

Ejemplo:


void m1(List list) {
  Number[] ns = (Number[]) list.toArray(new String[0]);
}

void m2(List<Number> list) {
  Number[] ns = list.toArray(new String[0]);
}
