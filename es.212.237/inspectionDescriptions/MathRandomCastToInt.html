Informa sobre las llamadas a Math.random() en las que se realiza una conversión inmediata a int.

Convertir un double entre 0.0 (incluido) y 1.0 (excluido) a int siempre redondea hacia abajo a cero. El valor debe multiplicarse primero por algún factor antes de convertirlo a int para obtener un valor entre cero (incluido) y el factor de multiplicación (excluido). Otra posible solución es utilizar el método nextInt() de java.util.Random.

Ejemplo:

  int r = (int)Math.random() * 10;

Después de aplicar la corrección rápida:

  int r = (int)(Math.random() * 10);
