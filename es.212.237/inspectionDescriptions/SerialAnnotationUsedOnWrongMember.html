Reporta métodos y campos en las clases Serializable y Externalizable que no son aptos para ser anotados con la anotación java.io.Serial.

Ejemplos:


class Test implements Serializable {
  @Serial // El campo anotado no es parte del mecanismo de serialización porque no es final
  private static long serialVersionUID = 7874493593505141603L;

  @Serial // El método anotado no es parte del mecanismo de serialización porque no es privado
  void writeObject(ObjectOutputStream out) throws IOException {
  }
}


class Test implements Externalizable {
  @Serial // El método anotado no es parte del mecanismo de serialización ya que está dentro de la clase Externalizable
  private void writeObject(ObjectOutputStream out) throws IOException {
  }
}

Para información sobre todos los casos válidos, consulte la documentación para java.io.Serial.
