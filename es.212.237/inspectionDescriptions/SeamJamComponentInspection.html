어노테이션이 추가된 Seam 클래스의 구성 문제를 보고합니다.

예:


@Name("authenticator")
class SeanComponent {
  @Create // static 메서드는 허용되지 않습니다
  private static void create() {...}

   @Destroy // static 메서드는 허용되지 않습니다
   private static void destroy() {...}

   @DataModel // getter가 아닙니다
   private Map method() {...}

   @Factory // public이어야 합니다
   private Map getUniverse2() {...}

   @Unwrap // 매개변수가 없어야 합니다
   private static Map createUniverse(int days) { ... }
}
