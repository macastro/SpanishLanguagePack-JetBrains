Reporta expresiones que pueden ser reemplazadas con constantes "mágicas".

Ejemplo 1:


  // El literal básico "2" es usado, advertencia:
  Font font = new Font("Arial", 2)

Ejemplo 2:


  // La constante predefinida es usada, es bueno:
  Font font = new Font("Arial", Font.ITALIC)

Cuando sea posible, el arreglo rápido inserta una constante predefinida apropiada.

El comportamiento de esta inspección es controlado por la anotación org.intellij.lang.annotations.MagicConstant. Algunos métodos de la librería Java estándar están pre-anotados, pero puedes usar esta anotación en tu código también.
