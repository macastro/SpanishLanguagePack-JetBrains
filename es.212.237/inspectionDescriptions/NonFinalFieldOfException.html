Reporta los campos de las subclases de java.lang.Exception que no están declarados como final.

Los datos de los objetos de excepción no deben ser modificados, ya que esto puede provocar la pérdida del contexto de error para su posterior depuración y registro.

Ejemplo:


  public class EditorException extends Exception {
    private String message; // advertencia: campo no final 'message' de la clase de excepción
  }
