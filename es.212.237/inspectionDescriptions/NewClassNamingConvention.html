Informa de las clases con nombres demasiado cortos, demasiado largos o que no sigan el patrón de expresión regular especificado.

Ejemplo: Si la inspección está habilitada para pruebas y la longitud mínima del nombre de la clase está establecida en 8 (valor predeterminado), se generará una advertencia para la siguiente clase de prueba porque la longitud del nombre de la clase es de 6, que es inferior a 8: public class MyTest{}.

Solo se puede usar una corrección rápida para cambiar el nombre de este tipo de clases en el editor.

Configuración de la inspección:

Usa la lista en la sección Opciones para especificar las clases que se deben inspeccionar. Desmarca las casillas correspondientes a las clases a las que quieras omitirles la inspección.

Para cada tipo de clase, usa los campos de entrada proporcionados para especificar la longitud mínima, la longitud máxima y la expresión regular esperadas para el nombre de la clase. Especifica 0 para el campo de longitud para omitir la comprobación correspondiente.

Las expresiones regulares se deben especificar en el formato estándar java.util.regex.
