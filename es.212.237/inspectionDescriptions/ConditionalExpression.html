Reporta el uso de operadores condicionales ternarios y sugiere reemplazarlos con sentencias if/else.

Algunos estándares de codificación prohíben el uso de operadores condicionales.

Ejemplo:


  Object result = (condition) ? foo() : bar();

Después de aplicar la corrección rápida:


  Object result;
  if (condition) {
    comp = foo();
  }
  else {
    comp = bar();
  }

Configuración de la inspección:

Para ignorar asignaciones simples y devoluciones, y permitir la siguiente sintaxis, usa la opción de "Ignorar asignaciones simples y devoluciones".


  String s = (foo == null) ? "" : foo.toString();

Para ignorar expresiones condicionales en el contexto donde la sentencia if es imposible (por ejemplo, cuando se usa la expresión condicional como un argumento de llamada de constructor de super()), usa la opción de "Ignorar ubicaciones donde la sentencia if es imposible".
