Informa de las llamadas a funciones de la biblioteca que podrían sustituirse por otras más simples.

El uso de funciones correspondientes simplifica el código.

La corrección rápida sustituye las llamadas a funciones por otras.

Ejemplo:


  fun test(list: List<Any>) {
      list.filter { it is String }
  }

Después de aplicar la corrección rápida:


  fun test(list: List<Any>) {
      list.filterIsInstance<String>()
  }
