Informa sobre los parámetros de tipo declarados para extender una clase final.

Sugiere reemplazar el parámetro de tipo por el tipo de la clase final especificada, dado que dichas clases no pueden extenderse.

Ejemplo:


  void foo() {
    List<? extends Integer> list; // Advertencia: la clase Integer es una clase final
  }

Después de aplicar la solución rápida:


  void foo() {
    List<Integer> list;
  }
