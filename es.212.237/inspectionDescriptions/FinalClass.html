Informa sobre las clases declaradas como final. No se informa sobre las clases finales que extienden una clase o interfaz sealed. Estas clases no pueden heredarse y pueden indicar una falta de diseño orientado a objetos. Algunos estándares de codificación desaconsejan las clases final.

Ejemplo:


  public final class Main {
  }

Después de aplicar la solución rápida:


  public class Main {
  }
