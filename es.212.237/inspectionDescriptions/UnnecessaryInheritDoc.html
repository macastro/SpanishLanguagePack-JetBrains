{@inheritDoc} informar comentarios Javadoc que solo contienen la etiqueta. Dado que Javadoc copia los comentarios de la superclase cuando no hay comentarios, los comentarios que solo contienen {@inheritDoc} no agregan nada.

También informes del uso de {@inheritDoc} en lugares no válidos, como dentro de campos.

Sugiere eliminar los comentarios Javadoc innecesarios.

Ejemplo:


  class Ejemplo implementa Comparable<Ejemplo> {
    /**
     * {@inheritDoc}
     */
    @Override
    public int compareTo(Ejemplo o) {
      return 0;
    }
  }

Después de aplicar la corrección rápida:


  class Ejemplo implementa Comparable<Ejemplo> {
    @Override
    public int compareTo(Ejemplo o) {
      return 0;
    }
  }
