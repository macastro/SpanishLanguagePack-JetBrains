Informa declaraciones if que se pueden reemplazar por declaraciones switch .

El resultado del reemplazo suele ser más corto y claro.

Ejemplo:


  void test(String str) {
    if (str.equals("1")) {
      System.out.println(1);
    } else if (str.equals("2")) {
      System.out.println(2);
    } else if (str.equals("3")) {
      System.out.println(3);
    } else {
      System.out.println(4);
    }
  }

Después de aplicar la corrección rápida:


  void test(String str) {
    switch (str) {
      case "1" -> System.out.println(1);
      case "2" -> System.out.println(2);
      case "3" -> System.out.println(3);
      default -> System.out.println(4);
    }
  }

Esta inspección solo informa si el nivel de lenguaje del proyecto o módulo es 7 o superior.

Utiliza el campo Número mínimo de ramas de condición 'if' para especificar el número mínimo de ramas de condición if para una declaración if que debe informarse. Ten en cuenta que la rama terminal else (sin if ) no se cuenta.

Utiliza la opción Sugerir switch en números para habilitar la sugerencia de declaraciones switch en números y caracteres primitivos y encapsulados.

Utiliza la opción Sugerir switch en enums para habilitar la sugerencia de declaraciones switch en constantes de enum .

Utiliza la opción Sugerir solo en expresiones null-safe para sugerir declaraciones switch que no pueden introducir un NullPointerException únicamente.
