Informa sobre las llamadas toString() a objetos de una clase que extiende Number. Estas llamadas suelen ser incorrectas en un entorno internacionalizado y debería usarse algún formato específico de la configuración regional en su lugar.

Ejemplo:


  void print(Double d) {
    System.out.println(d.toString());
  }
Una forma posible de solucionar este problema podría ser: 

  void print(Double d) {
    System.out.printf("%f%n", d);
  }
Esto formatea el número usando la configuración regional predeterminada que se establece durante el inicio de JVM y se basa en el entorno del host.
