Informa de cualquier instancia de paréntesis innecesarios.

Los paréntesis se consideran innecesarios si el orden de evaluación de una expresión permanece sin cambios después de eliminar los paréntesis.

Ejemplo:


  int n = 3 + (9 * 8);

Tras aplicar la corrección rápida:


  int n = 3 + 9 * 8;

Configurar la inspección:

  • Utiliza la opción Ignorar paréntesis de aclaración para ignorar los paréntesis que ayudan a aclarar una expresión binaria. Los paréntesis son de aclaración si la expresión entre paréntesis es una expresión instanceof que forma parte de una expresión más grande o tiene un operador diferente al de la expresión primaria.
  • Utiliza la opción Ignorar paréntesis alrededor de la condición de las expresiones condicionales para ignorar los paréntesis alrededor de la condición de las expresiones condicionales. Algunos estándares de codificación especifican que todas estas condiciones deben ir entre paréntesis.
  • Utiliza la opción Ignorar paréntesis alrededor de un único parámetro lambda sin tipo formal para ignorar los paréntesis alrededor de un único parámetro lambda dentro de una expresión lambda.
