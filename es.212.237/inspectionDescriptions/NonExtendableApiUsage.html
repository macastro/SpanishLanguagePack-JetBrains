Reporta clases, interfaces y métodos que extienden, implementan o anulan elementos de API marcados con @ApiStatus.NonExtendable.

La anotación @ApiStatus.NonExtendable indica que la clase, interfaz o método no debe ser extendido, implementado o anulado. Como es bastante común hacer un cast de dichas interfaces y clases en la implementación de la biblioteca interna, si un cliente proporciona una implementación diferente, obtendrá ClassCastException. Añadir nuevos métodos abstractos a dichas clases e interfaces romperá la compatibilidad con las implementaciones del cliente.
