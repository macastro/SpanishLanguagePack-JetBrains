Informa de las siguientes violaciones de las especificaciones de EJB:
  • El tipo de la propiedad inyectada es inválido
  • Los EJB referenciados son inconsistentes con el `ejb-jar.xml`

Ejemplo:


@LocalBean
@Stateless
public class ExampleBean {

  private AnotherBean anotherBean;

  @Inject
  public void getAnotherBean() { // Error: La inyección debe realizarse en un campo o un método setter
    return anotherBean;
  }

  ...
}
