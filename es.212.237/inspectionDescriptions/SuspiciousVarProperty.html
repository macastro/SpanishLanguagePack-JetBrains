Informa sobre propiedades var con un setter y getter predeterminados que no hacen referencia al campo de respaldo.

Estas propiedades no afectan a la llamada de su setter; por lo tanto, será más claro cambiar dicha propiedad a val y eliminar el inicializador.

Se puede utilizar la corrección rápida Cambiar a val y eliminar el inicializador para modificar el código automáticamente.

Ejemplo:


  // Esta propiedad siempre devuelve '1' y no importa que la propiedad sea 'var'
  var foo: Int = 0
      get() = 1
