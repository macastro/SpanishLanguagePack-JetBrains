Inspecciona la delegación de la interfaz a una propiedad var.

Solo se utiliza el valor inicial de una propiedad para la delegación, cualquier asignación posterior no la afecta.

Ejemplo:


  class Example(var texto: CharSequence): CharSequence by texto

La corrección rápida reemplaza una propiedad por una inmutable:


  class Example(val texto: CharSequence): CharSequence by texto

Forma alternativa, si por alguna razón confía en la mutabilidad:


  class Example(texto: CharSequence): CharSequence by texto {
      var texto = texto
  }
