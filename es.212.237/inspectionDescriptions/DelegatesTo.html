Inspecciona las anotaciones @DelegatesTo.Target sin utilizar y los valores de atributo de anotaciones @DelegatedTo.target que no se pueden resolver.

Ejemplo:


  // Objetivo 't1' no utilizado y objetivo 't2' no resuelto
  def m(
      @DelegatesTo.Target('t1') target,
      @DelegatesTo(target = 't2') Closure c
  ) {}
