Reporta métodos de interfaz que entran en conflicto con los métodos protegidos clone() y finalize() de la clase java.lang.Object.

En una interfaz, es posible declarar estos métodos con un tipo de retorno que es incompatible con los métodos de java.lang.Object. Una clase que implementa tal interfaz no será compilable. Cuando la interfaz es funcional, sigue siendo posible crear una lambda a partir de ella, pero no se recomienda.

Ejemplo:


  // Advertencia: esta interfaz no puede ser implementada
  // por ninguna clase, solo por una lambda o referencia de método
  interface MiInterfaz {
    double clone();
  }
