Reportan nombres de receptores, como me, this, self, o nombres que difieren para otros nombres de receptores de este tipo.

Ejemplo:

func (self *MeterSnapshot) Rate5() float64 { return math.Float64frombits(self.rate5) }

De acuerdo con Nombres de receptor en github.com/golang, no debes usar nombres genéricos como "me", "this", o "self". Estos identificadores son típicos de lenguajes orientados a objetos y le pueden dar a los métodos un significado especial.
