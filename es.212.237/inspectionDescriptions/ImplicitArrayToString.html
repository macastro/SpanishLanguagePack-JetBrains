Información de arreglos usados en concatenaciones de String o pasados como parámetros a métodos de java.io.PrintStream, tales como System.out.println().

Por lo general, el contenido del arreglo debe usarse y no el objeto arreglo en sí.

Ejemplo:


  void print(Object[] objects) {
    System.out.println(objects);
  }

Después de aplicar la solución rápida:


  void print(Object[] objects) {
    System.out.println(Arrays.toString(objects));
  }
