Informa de los miembros protegidos en clases finales.

Puesto que las clases finales no se pueden heredar, marcar el método como protegido puede resultar confuso. Es preferible declarar estos miembros como privados.

Ejemplo:

record Bar(int a, int b) {
  protected int sum() { 
     return a + b;
  }
}

Tras aplicar la corrección rápida:

record Bar(int a, int b) {
  int sum() { 
     return a + b;
  }
}

Como se muestra en el ejemplo, una clase se puede marcar como final de forma explícita o implícita.
