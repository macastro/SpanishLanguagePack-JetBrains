Informa de variables, métodos o clases con nombres dudosos o poco descriptivos. Estos nombres no ayudan a entender el código y, probablemente, se crearon como algo temporal, pero posteriormente se olvidaron.

Ejemplo:


int aa = 42;

El quick-fix de cambio de nombre se sugiere únicamente en el editor.

Usa la opción para listar los nombres que deben informarse.
