Informa de else redundante en if con return

Ejemplo:


  fun foo(arg: Boolean): Int {
      if (arg) return 0
      else { // Este else es redundante, el código entre llaves se puede desplazar hacia la izquierda
          someCode()
      }
  }

Después de aplicar la corrección rápida:


  fun foo(arg: Boolean): Int {
      if (arg) return 0
      someCode()
  }
