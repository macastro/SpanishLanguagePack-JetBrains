Informa sobre las clases que hacen referencia a demasiadas otras clases.

Las clases con un acoplamiento demasiado alto pueden ser muy frágiles y probablemente deban dividirse en clases más pequeñas.

Configurar la inspección:

  • Utilice el campo Límite de acoplamiento de clases para especificar el acoplamiento máximo permitido para una clase.
  • Utilice la opción Incluir acoplamientos a clases del sistema Java para especificar si se deben contar las referencias a clases del sistema (las incluidas en los paquetes java. o javax.)
  • Utilice la opción Incluir acoplamientos a clases de biblioteca para especificar si deben contarse las referencias a cualquier clase de biblioteca.
