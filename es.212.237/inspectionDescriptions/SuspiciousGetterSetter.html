class Point {

    public void setX(int x) { // Advertencia: setter 'setX()' asigna al campo 'y'
      this.y = x;
    }

    public int getY() { // Advertencia: getter 'getY()' retorna el campo 'x'
      return x;
    }
  }

Utiliza la casilla de abajo para reportar situaciones cuando un campo en la clase tiene un nombre que coincide con el nombre de un getter o setter.
