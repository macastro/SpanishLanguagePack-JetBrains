Informa sobre el espaciado incorrecto para pares clave-valor en los comandos ARG, ENV y LABEL.

Si bien no se especifica explícitamente en la especificación de Dockerfile, no se permiten algunas combinaciones de espaciado para pares clave-valor. La compilación de Docker fallará después de alcanzar la instrucción del problema.

Ejemplos:

  • El comando ARG no permite espacios alrededor de '='
  • ENV y LABEL no permiten espacios después de '='

  # todos los comandos siguientes fallarán
  ARG answer = 42
  ARG version= "1.0.0"
  LABEL "maintained.by"= someone@gmail.com
  ENV JAVA_HOME= "/docker-java-home"

Después de aplicar la corrección rápida:


  ARG answer=2
  ARG version="1.0.0"
  LABEL "maintained.by"=someone@gmail.com
  ENV JAVA_HOME="/docker-java-home"
