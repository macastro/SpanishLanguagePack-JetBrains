Reporta problemas en anotaciones de método @Contract. Los tipos de problemas que pueden ser reportados son:
  • Errores en sintaxis de contrato
  • Contratos que no se ajustan a la firma del método (número de parámetros equivocado)
  • Implementaciones de método que contradicen el contrato (ej. devolver true cuando el contrato dice false)

Ejemplo:


  // el método no tiene parámetros, pero el contrato espera 1
  @Contract("_ -> fail")
  void x() {
    throw new AssertionError();
  }
