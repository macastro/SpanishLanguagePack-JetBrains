Reporta métodos public en clases que no son expuestos en una interfaz.

Exponer todos los métodos public a través de una interfaz es importante para mantener un acoplamiento débil, y podría ser necesario para ciertos estilos de programación basada en componentes.

Ejemplo:


interface Persona {
  String getNombre();
}

class PersonaImplementacion implements Persona {
  private String nombre;

  // ok: el método está expuesto en la interfaz
  @Override
  public String getNombre() {
    return nombre;
  }

  // advertencia: el método es public
  // pero no está expuesto en la interfaz
  public void setNombre() {
    this.nombre = nombre;
  }
}

Usa la lista Ignorar si está anotada con para especificar anotaciones especiales. Los métodos anotados con una de estas anotaciones serán ignorados por esta inspección.

Usa la opción Ignorar si la clase contenedora no implementa una interfaz que no sea de biblioteca para ignorar métodos de clases que no implementan ninguna interfaz del proyecto.
