Reporta llamadas a métodos de fábrica `MethodHandle` y `VarHandle` que no coinciden con ningún método o campo.

También reporta argumentos para `MethodHandle.invoke()` y métodos similares, que no coinciden con la firma `MethodHandle` y argumentos para `VarHandle.set()` que no coinciden con el tipo `VarHandle`.

Ejemplos:

  MethodHandle mh = MethodHandles.lookup().findVirtual(
      MyClass.class, "foo", MethodType.methodType(void.class, int.class));
  // el argumento debe ser un valor int
  mh.invoke(myObj, "abc");


  // el argumento debe ser String.class
  VarHandle vh = MethodHandles.lookup().findVarHandle(
      MyClass.class, "text", int.class);


  VarHandle vh = MethodHandles.lookup().findVarHandle(
      MyClass.class, "text", String.class);
  // el argumento debe ser un valor String
  vh.set(myObj, 42);
