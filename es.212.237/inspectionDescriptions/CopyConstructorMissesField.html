Informa sobre constructores de copias que no copian todos los campos de la clase.

Los campos final con inicializadores y los campos transient se consideran innecesarios de copiar.

Ejemplo:


  class Point {

    private int x;
    private int y;

    Point(int x, int y) {
      this.x = x;
      this.y = y;
    }

    Point(Point other) {
      // Los campos x e y no est√°n inicializados
    }
  }
