Informa de los negativos unarios innecesarios. Dichas expresiones pueden ser difíciles de entender y podrían contener errores.

Por ejemplo:

void unaryMinus(int i) {
    int x = - -i;
  }

Aquí se sugieren las siguientes soluciones rápidas:

  • Eliminar los operadores - antes de la variable i:

    void unaryMinus(int i) {
    int x = i;
  }

  • Reemplazar los operadores - con el operador de decremento prefijo:

    void unaryMinus(int i) {
    int x = --i;
  }

Otro ejemplo:

void unaryMinus(int i) {
    i += - 8;
  }

Tras aplicar la corrección rápida:

void unaryMinus(int i) {
    i -= 8;
  }
