Reporta expresiones `return` sin etiqueta dentro de una lambda inline.

Tales expresiones pueden ser confusas porque puede no estar claro a qué espectro pertenece `return`.

El "arreglo rápido" Cambiar a `return@...` puede usarse para modificar el código automáticamente.

Ejemplo:


  fun test(list: List<Int>) {
      list.forEach {
          // Esta expresión `return` regresa desde la función `test`
          // Puedes cambiarla a `return@forEach` para cambiar el espectro
          if (it == 10) return
      }
  }

Después de que el arreglo rápido sea aplicado:


  fun test(list: List<Int>) {
      list.forEach {
          if (it == 10) return@test
      }
  }
