Informa de las clases que implementan Serializable y no declaran un campo serialVersionUID.

Sin un campo serialVersionUID, cualquier cambio en la clase hará que las versiones serializadas anteriormente sean ilegibles.

Ejemplo:


  class Main implements Serializable {
  }

Tras aplicar la corrección rápida:


  class Main implements Serializable {
    private static final long serialVersionUID = -1446398935944895849L;
  }
Al usar un nivel de lenguaje de JDK 14 o superior, la corrección rápida también agregará la anotación java.io.Serial.

Utilice las siguientes opciones para configurar la inspección:

  • Enumerar las clases cuyos herederos no deben ser informados por esta inspección. Esto está pensado para clases que heredan Serializable de una superclase pero no están pensadas para la serialización.
  • Si ignorar las clases anónimas Serializable.
