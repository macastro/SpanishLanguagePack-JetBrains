Reporta clases de utilidades que pueden ser convertidas a enumeraciones.

Algunas guías de estilo de codificación requieren la implementación de clases de utilidades como enumeraciones para evitar problemas de cobertura de código en constructores private.

Ejemplo:


  class StringUtils {
    public static final String EMPTY = "";
  }

Después de aplicar la corrección rápida:


  enum StringUtils {
    ;
    public static final String EMPTY = "";
  }
