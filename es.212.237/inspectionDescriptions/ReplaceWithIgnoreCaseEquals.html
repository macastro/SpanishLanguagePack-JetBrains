Informa de comparaciones que no distinguen mayúsculas o minúsculas que se pueden reemplazar con equals(..., ignoreCase = true).

Al usar equals(), no tienes que asignar cadenas adicionales con toLowerCase() o toUpperCase() para comparar cadenas.

La corrección rápida reemplaza la comparación que no distingue entre mayúsculas y minúsculas que utiliza toLowerCase() o toUpperCase() con equals(..., ignoreCase = true).

Nota: Puede cambiar la semántica de algunos lugares.

Ejemplo:


  fun main() {
      val a = "KoTliN"
      val b = "KOTLIN"
      println(a.toLowerCase() == b.toLowerCase())
  }

Después de que se aplica la corrección rápida:


  fun main() {
      val a = "KoTliN"
      val b = "KOTLIN"
      println(a.equals(b, ignoreCase = true))
  }
