Informa de llamadas a un simple getter de propiedad desde dentro de la clase de la propiedad.

Un simple getter de propiedad se define como aquel que simplemente devuelve el valor de un campo, y no hace otros cálculos. Tales llamadas a getters simples pueden ser alineadas de forma segura usando la corrección rápida. Algunos estándares de codificación también sugieren evitar el uso de getters simples por razones de claridad del código.

Ejemplo:


  public class Salient {
    private String name;

    public String getName() {
      return name;
    }

    @Override
    public String toString() {
      return getName();
    }
  }

Después de aplicar la corrección rápida:


  public class Salient {
    private String name;

    public String getName() {
      return name;
    }

    @Override
    public String toString() {
      return name;
    }
  }

Utiliza las siguientes opciones para configurar el análisis:

  • Si informar solo llamadas getter en this, no en objetos del mismo tipo pasados como parámetro.
  • Si ignorar los getters no private.
