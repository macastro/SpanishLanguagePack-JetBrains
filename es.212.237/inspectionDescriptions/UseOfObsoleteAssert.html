Informa de cualquier llamada a métodos de la clase junit.framework.Assert. Esta clase está obsoleta y las llamadas pueden ser reemplazadas por llamadas a métodos de la clase org.junit.Assert.

Por ejemplo:


  import org.junit.*;
  public class NecessaryTest {
    @Test
    public void testIt() {
      junit.framework.Assert.assertEquals("expected", "actual");
    }
  }

Después de aplicar la corrección rápida, el resultado se ve como sigue:


  import org.junit;
  public class NecessaryTest {

    public void testIt() {
      Assert.assertEquals("expected", "actual");
    }
  }
