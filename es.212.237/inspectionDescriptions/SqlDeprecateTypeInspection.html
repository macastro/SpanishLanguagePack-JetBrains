Informa sobre los usos de los tipos que están desfasados y que podrían desaparecer en versiones futuras del DBMS.

Tipos notificados:

  • LONG en Oracle (ver Entidades desfasadas y sin soporte en docs.oracle.com).
  • TEXT, NTEXT e IMAGE en Microsoft SQL Server (ver Entidades de motor de base de datos desfasadas en SQL Server 2016 en docs.microsoft.com).

Ejemplo (Oracle):

CREATE TABLE ot.foo(
a NUMBER GENERATED BY DEFAULT AS IDENTITY,
b LONG NOT NULL
);
