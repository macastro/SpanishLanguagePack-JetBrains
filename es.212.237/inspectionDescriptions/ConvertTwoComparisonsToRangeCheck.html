Informa de dos comparaciones consecutivas que se pueden convertir en una comprobación de intervalo.

Comprobar con un intervalo simplifica el código al eliminar la duplicación del asunto de la prueba.

Ejemplo:


  fun checkMonth(month: Int): Boolean {
      return month >= 1 && month <= 12
  }

La corrección rápida reemplaza la comprobación basada en la comparación por una de intervalo:


  fun checkMonth(month: Int): Boolean {
      return month in 1..12
  }
