Reporta entradas de declaraciones de desestructuración que coinciden con el nombre de una propiedad diferente de la clase de datos desestructurada.

Ejemplo:


  data class Foo(val a: String, val b: Int, val c: String)

  fun bar(f: Foo) {
      val (a, c) = f
  }

La corrección rápida cambia el nombre de la variable para que coincida con el nombre del campo de clase correspondiente:


  data class Foo(val a: String, val b: Int, val c: String)

  fun bar(f: Foo) {
      val (a, b) = f
  }
