Informa de problemas con los campos AtomicLongFieldUpdater, AtomicIntegerFieldUpdater o AtomicReferenceFieldUpdater (el paquete java.util.concurrent.atomic).

Los problemas informados son idénticos a los problemas de tiempo de ejecución que pueden ocurrir con los actualizadores de campo atómicos: campo especificado no encontrado, campo especificado no accesible, campo especificado tiene un tipo incorrecto, etc.

Ejemplos:

  • 
  public class A {
    private static volatile int value = 0;
    private static final AtomicIntegerFieldUpdater updater =
    AtomicIntegerFieldUpdater.newUpdater((A.class), "value"); // advertencia: El campo 'value' tiene el modificador 'static'
  }
  
  • 
  public class B {
    private static final AtomicIntegerFieldUpdater updater =
      AtomicIntegerFieldUpdater.newUpdater(B.class, "value"); // advertencia: no se ha encontrado ningún campo con nombre 'value' en la clase 'B'
  }
