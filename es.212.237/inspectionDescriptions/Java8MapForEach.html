Sugiere reemplazar for(Entry<?,?> entrada : map.entrySet()) {...} o map.entrySet().forEach(entrada -> ...) por map.forEach((key, value) -> ...).

Ejemplo


  void print(Map<String, Integer> map) {
    map.entrySet().forEach(entry -> {
      String str = entry.getKey();
      System.out.println(str + ":" + entry.getValue());
    });
  }

Después de aplicar la corrección rápida:


  void print(Map<String, Integer> map) {
    map.forEach((str, value) -> System.out.println(str + ":" + value));
  }

Cuando la opción No informar bucles esté habilitada, se informará sólo de los casos entrySet().forEach(). Sin embargo, la acción de corrección rápida también estará disponible para los bucles for.
