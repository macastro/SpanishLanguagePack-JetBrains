Informa parámetros de tipos que pueden tener varianza in o out.

Usar varianza in y out proporciona inferencia de tipos más precisa en Kotlin y semántica de código más clara.

Ejemplo:


  class Box<T>(val obj: T)

  fun consumeString(box: Box<String>) {}
  fun consumeCharSequence(box: Box<CharSequence>) {}

  fun usage(box: Box<String>) {
      consumeString(box)
      consumeCharSequence(box) // Error de compilación
  }

La corrección rápida añade el modificador de varianza correspondiente:


  class Box<out T>(val obj: T)

  fun consumeString(box: Box<String>) {}
  fun consumeCharSequence(box: Box<CharSequence>) {}

  fun usage(box: Box<String>) ++{
      consumeString(box)
      consumeCharSequence(box) // OK
  }
