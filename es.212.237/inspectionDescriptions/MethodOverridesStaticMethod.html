Avisos de métodos static con una firma idéntica a un método static de una superclase. Este método puede verse como un sobrescrito cuando en realidad oculta el método de la superclase porque los métodos static en Java no pueden sobrescribirse.

Ejemplo:


  class Padre {
    static void método(){}
  }

  class Ejemplo extends Padre {
    static void método(){} //advertencia
  }
