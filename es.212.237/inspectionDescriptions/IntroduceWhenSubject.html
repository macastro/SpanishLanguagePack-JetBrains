Informa una expresión when que puede ser simplificada introduciendo un argumento de sujeto.

Ejemplo:


  fun test(obj: Any): String {
      return when {
          obj is String -> "string"
          obj is Int -> "int"
          else -> "unknown"
      }
  }

La solución rápida introduce un argumento de sujeto:


  fun test(obj: Any): String {
      return when (obj) {
          is String -> "string"
          is Int -> "int"
          else -> "unknown"
      }
  }
