Informa una función que coincide con una de las convenciones del operador pero le falta la palabra clave operator.

Al añadir el modificador operator, podrías permitir a los consumidores de funciones escribir código Kotlin idiomático.

Ejemplo:


  class Complex(val real: Double, val imaginary: Double) {
      fun plus(otro: Complex) =
          Complex(real + otro.real, imaginary + otro.imaginary)
  }

  fun uso(a: Complex, b: Complex) {
      a.plus(b)
  }

El arreglo rápido añade la palabra clave de modificador operator:


  class Complex(val real: Double, val imaginary: Double) {
      operator fun plus(otro: Complex) =
          Complex(real + otro.real, imaginary + otro.imaginary)
  }

  fun uso(a: Complex, b: Complex) {
      a + b
  }
