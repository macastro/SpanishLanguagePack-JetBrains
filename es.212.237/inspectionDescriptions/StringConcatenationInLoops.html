Informa de la concatenación de String en bucles.

Como cada concatenación de String copia toda la cadena, normalmente es preferible reemplazarla con llamadas explícitas a StringBuilder.append() o StringBuffer.append().

Ejemplo:


  String str = "";
  for(int i=0; i<10; i++) {
    str += i;
  }

Después de aplicar la corrección rápida:


  String str = "";
  StringBuilder strBuilder = new StringBuilder(str);
  for(int i = 0; i<10; i++) {
    strBuilder.append(i);
  }
  str = strBuilder.toString();

En ocasiones, la corrección rápida permite convertir una variable String en una StringBuilder o introducir una nueva StringBuilder. Tened cuidado si en el código original se maneja especialmente el valor nulo, ya que el reemplazo podría cambiar su semántica. Si nulo es posible, se sugieren correcciones seguras que no admiten valores nulos y que generan las comprobaciones necesarias de valores nulos. Además, no está garantizado que el reemplazo automático mejore siempre el rendimiento.
