Informa de llamadas a isEmpty, isBlank, isNotEmpty o isNotBlank en una instrucción if para asignar un valor predeterminado.

La corrección rápida reemplaza la condición if por llamadas a ifEmpty o ifBlank.

Ejemplo:


  función test(lista: List): List {
      return si (lista.isEmpty()) {
          println()
          foo()
      } más {
          lista
      }
  }

Después de aplicar la corrección rápida:


  fun test(list: List<Int>): List<Int> {
      return list.ifEmpty {
          println()
          foo()
      }
  }

Esta inspección solo informa si la versión del lenguaje Kotlin del proyecto o módulo es 1.3 o superior.
