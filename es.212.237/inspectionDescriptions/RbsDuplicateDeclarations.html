Informa sobre declaraciones duplicadas de clases, módulos, interfaces, constantes, variables globales, alias de tipo y métodos.
  • Las declaraciones duplicadas producen DuplicatedDeclarationError.
  • Las definiciones de métodos duplicados producen DuplicatedMethodDefinitionError.
  • Las definiciones de métodos de interfaz duplicados producen DuplicatedInterfaceMethodDefinitionError.

Hay una solución rápida disponible para navegar hasta la declaración duplicada.

Ejemplo:

# Declaración de interfaz duplicada
interface _Example
end

# Declaración de interfaz duplicada
interface _Example
end
