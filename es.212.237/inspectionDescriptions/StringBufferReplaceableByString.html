Informa sobre el uso de `StringBuffer`, `StringBuilder` o `StringJoiner` que pueden sustituirse por una única concatenación de `String`.

El uso de concatenación de `String` hace que el código sea más corto y simple.

Esta inspección solo informa cuando el reemplazo sugerido no introduce un problema de rendimiento significativo en las JVM modernas. En la mayoría de los casos, la concatenación de `String` puede mejorar el rendimiento.

Ejemplo:


  StringBuilder result = new StringBuilder();
  result.append("i = ");
  result.append(i);
  result.append(";");
  return result.toString();

Después de aplicar la corrección rápida:


  String result = "i = " + i + ";";
  return result;
