Reporta cálculo de código hash incorrecto para arreglos.

Para calcular correctamente el código hash de un arreglo, use:

  • Arrays.hashcode() para arreglos lineales
  • Arrays.deepHashcode() para arreglos multidimensionales
Estos métodos también deben usarse con Objects.hash() cuando la secuencia de valores de entrada incluye arreglos, por ejemplo: Objects.hash(string, Arrays.hashcode(array))
