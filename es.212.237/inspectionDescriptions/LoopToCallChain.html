Informa los bucles for que pueden reemplazarse con una secuencia de operaciones de la biblioteca estándar (como map, filter y otras).

Ejemplo:


fun foo(list: List<String>): List<Int> {
  val result = ArrayList<Int>()
  for (s in list) {
     if (s.length > 0)
       result.add(s.hashCode())
     }
  return result
}

Después de aplicar la reparación rápida:


fun foo(list: List<String>): List<Int> {
  val result = list
    .filter { it.length > 0 }
    .map { it.hashCode() }
  return result
}
