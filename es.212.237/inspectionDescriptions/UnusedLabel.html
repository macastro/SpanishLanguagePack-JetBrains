Informa de etiquetas que no son objetivos de sentencias break o continue.

Ejemplo:


  etiqueta: for (int i = 0; i < 10; i++) {
    if (i == 3) {
      break;
    }
  }

Después de aplicar la corrección rápida, se elimina la etiqueta:


  for (int i = 0; i < 10; i++) {
    if (i == 3) {
      break;
    }
  }
