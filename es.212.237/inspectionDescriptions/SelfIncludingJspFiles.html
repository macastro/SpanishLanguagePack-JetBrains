Informa cualquier directiva de inclusión en un archivo JSP que hace referencia al archivo JSP contenedor.

Archivo JSP de ejemplo llamado main.jsp:


  <jsp:include page="./main.jsp" /> // Error: intento de incluir el archivo contenedor
  ...

NOTA: La autoinclusión de este tipo conduce a una recursión infinita, por lo que el archivo JSP no se puede compilar.
