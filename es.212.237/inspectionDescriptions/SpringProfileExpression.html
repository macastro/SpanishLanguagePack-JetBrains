Reporta expresiones incorrectas de @Profile:

  • Los profiles de Spring no deben estar vacíos
  • Los operadores '&' y '|' no se deben mezclar sin paréntesis en las expresiones de profiles de Spring

Ejemplos:


  // Configurado correctamente
  @Configuration
  @Profile("a & (b | c)")
  public class MyConfiguration {...}

  // Vacío
  @Configuration
  @Profile() // Reporta "Una expresión de Profile debe contener texto"
  public class MyConfiguration {...}

  // Operadores mezclados sin paréntesis
  @Configuration
  @Profile("a & b | c") // Reporta "Expresión de Profile mal formada"
  public class MyConfiguration {...}
