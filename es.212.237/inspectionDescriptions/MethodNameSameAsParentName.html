Informa de los métodos que tienen el mismo nombre que el de la superclase de la clase del método, ya que dichos nombres de métodos pueden resultar confusos.

Esta inspección no comprueba interfaces o superclases que se encuentren en un nivel profundo de la jerarquía.

Ejemplo: 


  class Parent {}
  class Child extends Parent {
     public Parent Parent() {
       return null;
     }
  }

La corrección rápida para cambiar el nombre de tales métodos solo se pueda utilizar en el editor.
