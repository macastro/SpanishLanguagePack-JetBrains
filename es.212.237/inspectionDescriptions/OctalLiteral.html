Informa sobre literales enteros octales. Algunos estándares de codificación prohíben el uso de literales octales, ya que pueden confundirse fácilmente con literales decimales.

Ejemplo:


  int i = 015;
  int j = 0_777;

Esta inspección tiene dos soluciones rápidas diferentes. Tras aplicar la solución rápida Convertir el literal octal a literal decimal, el código cambia a:


  int i = 13;
  int j = 511;

Tras aplicar la solución rápida Eliminar el cero inicial para convertir a decimal, el código cambia a:


  int i = 15;
  int j = 777;
