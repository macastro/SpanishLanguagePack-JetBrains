Inspecciona las declaraciones de clases y objetos con un cuerpo vacío.

Utiliza la corrección rápida "Eliminar cuerpo de clase vacío redundante" para limpiar el código.

Ejemplos:


  class EmptyA() {} // <== cuerpo vacío

  class EmptyB {
      companion object {} // <== cuerpo vacío
  }

  fun emptyC() {
     object {} // <== objeto anónimo, está bien (no reportado)
  }

Después de aplicar la corrección rápida:


  class EmptyA()

  class EmptyB {
      companion object
  }

  fun emptyC() {
     object {}
  }
