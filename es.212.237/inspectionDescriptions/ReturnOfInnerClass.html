Reporta instrucciones return que devuelven una instancia de una clase anónima, local o interna. Tales instancias mantienen una referencia implícita a la instancia externa, la cual puede evitar que la instancia externa sea recolectada por la basura. Cualquier invocador de un método que devuelva tal instancia podría causar una fuga de memoria manteniendo una referencia a la instancia devuelta.

Configurar la inspección:

  • Usa la opción Ignorar devoluciones de métodos no públicos para ignorar devoluciones de métodos protected o privados de paquete. Las devoluciones de métodos private siempre son ignoradas.
