관용적 오류 처리를 위한 보일러플레이트 코드를 생성합니다.

error 타입의 변수 및 표현식에 호출될 수 있습니다.

  • 변수 err를 다음 구문으로 바꿉니다:

    if err != nil {
    return value1, value2, ...
  }

  • expr 표현식을 다음 구문으로 바꿉니다:

    if err := expr; err != nil {
    return value1, value2, ...
  }

각 값은 err 변수이거나 둘러싸는 함수의 해당 반환 타입의 디폴트 값입니다. 여러 개의 값을 사용하는 표현식에 이 템플릿을 호출할 수 있습니다.
