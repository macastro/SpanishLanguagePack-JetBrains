Esta es una plantilla incorporada para comentarios PHPDoc utilizados en la implementación de hooks de Drupal. Incluye fragmentos de código que pueden incluirse en la plantilla de archivo (pestaña "Plantillas") mediante la directiva #parse. La plantilla es editable. Además de texto estático, código y comentarios, puede utilizar variables predefinidas que se expandirán a su valor correspondiente como lo hacen las macros.
Las variables predefinidas usan los siguientes valores:
${HOOK_NAME} Nombre del hook.
