Plantilla incorporada de comentarios PHPDoc para propiedades de clase PHP. Incluye fragmentos de código que se pueden incluir en plantillas de archivos (pestaña Plantillas) mediante la directiva #parse.
La plantilla es editable. Además del texto, código y comentarios estáticos, se pueden utilizar variables predefinidas, las cuales se expandirán a sus valores correspondientes como lo hacen las macros.
Las variables predefinidas toman los siguientes valores:
${NAME} Nombre de la propiedad.
${CLASS_NAME} Nombre de la clase.
${STATIC} Toma el valor 'static' si la propiedad es estática, o una cadena vacía en caso contrario.
${NAMESPACE} Nombre de espacio de la propiedad totalmente calificado sin la barra inicial.
${DS} El signo de dólar se evalúa como el carácter ‘$' normal.
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${MONTH_NAME_SHORT} Las primeras 3 letras del nombre del mes actual, Ej.: Jan, Feb, etc.
${MONTH_NAME_FULL} El nombre completo del mes actual, Ej.: January, February, etc.
${DAY} Día del mes actual
${HOUR} Hora actual
${MINUTE} Minuto actual
${SECOND} Segundo actual
${PROJECT_NAME} Nombre del proyecto actual
${USER} Usuario actual
${CARET} Almacena la posición a la que se debe mover el cursor después de agregar el comentario.
