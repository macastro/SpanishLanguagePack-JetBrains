Esta es una plantilla preincorporada para los comentarios de PHPDoc de las funciones de PHP. Contiene un fragmento de código que se puede incluir en las plantillas de archivo (pestaña Plantillas) con la ayuda de las directivas #parse.
La plantilla es editable. Además de texto estático, código y comentarios, puede usar variables predefinidas que se expandirán al valor correspondiente, como las macros.
Las variables predefinidas toman los siguientes valores:
${NAME} Nombre de la función.
${TYPE_HINT} Sugerencia de tipo para el valor devuelto de la función. Si no se deduce el tipo de valor devuelto, se evalúa como "void".
${PARAM_DOC}

Comentario de PHPDoc de los parámetros.

Se crea como una serie de líneas "* @param tipo nombre". Si no hay parámetros, se evalúa como una cadena vacía.

${THROWS_DOC}

Comentario de documentación de excepciones.

Se genera como una serie de líneas "* @throws type". Si no se genera ninguna excepción, se evalúa como una cadena vacía.

${STATIC} Toma el valor "static" si la función/método es estático o una cadena vacía en caso contrario. Por ejemplo: 
#if (${STATIC} == "static") * @static
#end
${DS} El signo de dólar se evalúa como el carácter "$" normal.
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${MONTH_NAME_SHORT} Primeras 3 letras del nombre del mes actual. Por ejemplo: Jan, Feb, etc.
${MONTH_NAME_FULL} Nombre completo del mes actual. Por ejemplo: January, February, etc.
${DAY} Día del mes actual
${HOUR} Hora actual
${MINUTE} Minuto actual
${SECOND} Segundo actual
${PROJECT_NAME} Nombre del proyecto actual
${USER} Usuario actual
${CARET} Almacena dónde debe moverse el cursor después de añadir comentarios.
