Plantilla integrada para comentarios PHPDoc para utilizar en traits PHP. Incluye fragmentos de código que se pueden incluir en la plantilla de archivo (pestaña Plantillas) usando la directiva #parse.
La plantilla es editable. Además de texto estático, código y comentarios, puedes usar variables predefinidas, que se expandirán a sus valores correspondientes como las macros.
Las variables predefinidas adquieren los siguientes valores:
${NAME} Nombre del trait.
${NAMESPACE} Nombre de clase del trait completamente calificado sin la barra inicial.
${DS} El símbolo del dólar se evalúa como el carácter '$' normal.
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${MONTH_NAME_SHORT} Los tres primeros caracteres del nombre del mes actual. Por ejemplo, Ene, Feb, etc.
${MONTH_NAME_FULL} Nombre completo del mes actual. Por ejemplo, Enero, Febrero, etc.
${DAY} Día del mes actual
${HOUR} Hora actual
${MINUTE} Minuto actual
${SECOND} Segundo actual
${PROJECT_NAME} Nombre del proyecto actual
${USER} Usuario actual
${CARET} Almacena la ubicación a la que debe moverse el cursor después de añadir el comentario.
