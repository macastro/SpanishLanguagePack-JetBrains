Plantilla integrada para un comentario PHPDoc en la clase PHP. Esta incluye fragmentos de código que pueden incluirse en una plantilla de archivo (pestaña "Plantillas") con la ayuda de la directiva #parse.
Las plantillas se pueden editar. Además del texto estático, el código y los comentarios, también puede utilizar variables predefinidas que se expanden al valor correspondiente como lo hacen las macros.
Las variables predefinidas reciben los siguientes valores:
${NAME} Nombre de la clase.
${NAMESPACE} Nombre completamente calificado del espacio de nombres de la clase sin la barra diagonal principal.
${DS} El símbolo del dólar se evalúa al carácter "$" normal.
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${MONTH_NAME_SHORT} Las 3 primeras letras del nombre del mes actual, p. ej., ene, feb, etc.
${MONTH_NAME_FULL} El nombre completo del mes actual, p. ej., enero, febrero, etc.
${DAY} Día del mes actual
${HOUR} Hora actual
${MINUTE} Minuto actual
${SECOND} Segundo actual
${PROJECT_NAME} Nombre del proyecto actual
${USER} Usuario actual
${CARET} Contiene la posición a la que se debe mover el cursor después de agregar el comentario.
