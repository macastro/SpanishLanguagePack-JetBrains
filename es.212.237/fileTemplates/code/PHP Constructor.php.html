Una plantilla incorporada para el constructor de clases PHP.
Las variables predefinidas toman los siguientes valores:
${PARAM_LIST} Lista de parámetros del constructor separados por comas, uno para cada propiedad seleccionada.
${PARAM_DOC} Comentarios PHPDoc del parámetro.
Generados como varias líneas que contienen '* @param tipo nombre'. Se evalúa como una cadena vacía si no hay parámetros.
${THROWS_DOC} Comentarios PHPDoc de excepción.
Generados como varias líneas que contienen '* @throws tipo'. Se evalúa como una cadena vacía si no se lanzan excepciones.
${CLASS_NAME} Nombre de la clase.
${BODY} Cuerpo del constructor que contiene asignaciones '$this->property=$property'.
