Esta es una plantilla preconstruida que se usa para escribir los cuerpos de los métodos dinámicos de FreeMarker cada vez que son generados por el programa.
Esta plantilla es editable. Así como las expresiones y comentarios de Java, también puedes usar variables predefinidas que se expanden a valores relevantes.
Las variables predefinidas tienen los siguientes valores:
${RETURN_TYPE} El tipo de retorno del método creado
${DEFAULT_RETURN_VALUE} El valor que el método regresa por defecto
${METHOD_NAME} El nombre del método creado
${CLASS_NAME} El nombre calificado de la clase que crea el método
