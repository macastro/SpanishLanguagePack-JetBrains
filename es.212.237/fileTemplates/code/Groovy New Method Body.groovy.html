Esta es una plantilla incorporada que sirve para rellenar el cuerpo del método Groovy cada vez que es generado por el programa, por ejemplo, al utilizar la acción de intención \"Crear método desde el uso\".
La plantilla se puede editar. Puedes utilizar expresiones y comentarios de Groovy y además, las variables predefinidas que se expandirán después a los valores correspondientes.
Las variables predefinidas tendrán los siguientes valores:
${RETURN_TYPE} Un tipo de retorno de un método creado
${DEFAULT_RETURN_VALUE} Un valor retornado por el método de forma predeterminada
${METHOD_NAME} Nombre del método creado
${CLASS_NAME} Nombre completo de la clase donde se crea el método
${SIMPLE_CLASS_NAME} Nombre no completo de la clase donde se implementa el método
