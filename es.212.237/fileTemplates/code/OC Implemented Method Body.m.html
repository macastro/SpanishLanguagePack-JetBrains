Crea el cuerpo de un método implementado cuando se genera, por ejemplo, cuando se llama la acción Código | Implementar método.
La plantilla es editable. Junto con expresiones y comentarios Objective-C, puedes usar variables predefinidas que luego se expandirán en los valores correspondientes.
Las variables predefinidas tomarán los siguientes valores:
${RETURN_TYPE} el tipo de retorno del creado método
${DEFAULT_RETURN_VALUE} el valor retornado por el método por defecto
${CONTAINING_CLASS} el nombre de la clase donde se implementa el método
Todas las variables predefinidas desde la plantilla Encabezado de archivo en C (la pestaña Incluye) también están disponibles (${USER}, ${DATE}, etc.)
