Una plantilla incorporada para los métodos setter de PHP. Se utiliza, por ejemplo, cuando se invoca el arreglo rápido "Agregar getter y setter".
Las variables predefinidas toman los siguientes valores:
${STATIC} Toma el valor 'static' si la propiedad es estática y una cadena vacía en caso contrario.
${NAME} El nombre del método en formato CamelCase sin prefijo "set" y con la primera letra en mayúsculas. Por ejemplo, el nombre de propiedad '_my_property' se convierte en 'MyProperty'.
${CLASS_NAME} El nombre de la clase contenedora.
${SCALAR_TYPE_HINT} Contiene una sugerencia de tipo escalar si el nivel de lenguaje es PHP 7.0 o superior.
${FIELD_NAME} El nombre de la propiedad original intacto.
${PARAM_NAME} El nombre del parámetro del setter. Es casi idéntico a ${FIELD_NAME}, pero se elimina el guión bajo inicial, si existe. Por ejemplo, el nombre de propiedad '_myProperty' se convierte en 'myProperty'.
${TYPE_HINT} La sugerencia de tipo para la propiedad. Una cadena vacía si no se especifica.
${VOID_RETURN_TYPE} Indica si el tipo de retorno void se puede utilizar en el nivel de lenguaje PHP del proyecto.
${DS} El signo de dólar se evalúa como un carácter '$' normal.
