Cada vez que se genera mediante un programa (por ejemplo, cuando se usa la acción de intención Crear función a partir del uso) rellena el cuerpo de la función de Kotlin. Esta plantilla se puede editar. Además de expresiones y comentarios de Kotlin, puedes usar variables predefinidas que se expandirán al valor correspondiente.
Las variables predefinidas toman los siguientes valores:
${RETURN_TYPE} El tipo de valor devuelto de la función creada.
${FUNCTION_NAME} El nombre de la función creada.
${CLASS_NAME} El nombre calificado de la clase en la que se crea la función.
${SIMPLE_CLASS_NAME} El nombre no calificado de la clase en la que se implementa la función.
