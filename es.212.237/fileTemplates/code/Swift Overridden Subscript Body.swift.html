Aqui es donde se escribe el cuerpo del subíndice cuando se generan subíndices que se anularán mediante acciones como Código | Generación | Anulación de métodos.
Esta plantilla es editable. Además de las expresiones y comentarios de Swift, puede utilizar variables predefinidas que también se expandirán a su valor correspondiente.
Las variables predefinidas toman los siguientes valores:
${CALL_SUPER} Llamada al superíndice - return super[] (con parámetros)
${RETURN_TYPE} Tipo de valor devuelto para el subíndice que se crea
${DEFAULT_RETURN_VALUE} Valor predeterminado devuelto por el subíndice
${SIGNATURE} Firma del subíndice
${HAS_SET} "true" si el subíndice tiene un establecedor
${HAS_SUPER_SET} "true" si el subíndice que se anula tiene un establecedor
${GETTER_EFFECTS} Efectos de obtención para el captador de propiedades (async, throws)
${GET_ATTRIBUTES} Atributos del captador del subíndice
${SET_ATTRIBUTES} Atributos del establecedor del subíndice
