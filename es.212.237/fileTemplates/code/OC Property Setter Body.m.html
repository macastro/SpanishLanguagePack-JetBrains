Crea el cuerpo de un método setter sobreescrito para una propiedad cuando se genera, por ejemplo, al llamar a la acción Código | Sobreescribir método.
La plantilla es editable. Junto con expresiones y comentarios de Objective-C, puedes utilizar variables predefinidas que luego se expandirán a los valores correspondientes.
Las variables predefinidas tomarán los siguientes valores:
${CALL_SUPER} una llamada a un supermétodo, para métodos void — [super methodName] con o sin parámetros; para otros métodos — return [super methodName] con o sin parámetros
${RETURN_TYPE} el tipo de retorno del método creado
${DEFAULT_RETURN_VALUE} el valor devuelto por el método de forma predeterminada
${CONTAINING_CLASS} el nombre de la clase donde se sobreescribe el método
${IVAR_IS_AVAILABLE} "true" cuando la propiedad tiene una variable de instancia
${IVAR} el nombre de la variable de instancia
${PARAM} el nombre de un parámetro setter
${SETTER_SEMANTICS} en modo ARC: "copyArc" cuando se establece el atributo copy, "assign" en caso contrario
en modo no ARC: "copy" cuando se establece el atributo copy, "retain" cuando se establece el atributo retain, "assign" en caso contrario
Todas las variables predefinidas de la plantilla Encabezado de archivo C (la pestaña Incluye) también están disponibles (${USER}, ${DATE}, etc.)
