Crea una definición de pasos para Cucumber para un archivo Java.
Las variables predefinidas toman los valores siguientes:
${STEP_KEYWORD} La anotación de definición de paso. Normalmente, la palabra clave se usa para invocar pasos (como Given, Then, And).
${STEP_REGEXP} La expresión regular que se genera para aceptar el paso.
${PARAMETERS} Los parámetros del método generado.
${BODY} El cuerpo del método generado.
