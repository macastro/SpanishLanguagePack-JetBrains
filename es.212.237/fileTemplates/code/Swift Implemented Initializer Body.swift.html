Complete el cuerpo del inicializador que se implementará cuando se genere con acciones como Code | Generate | Implement Methods.
La plantilla es editable. Además de las expresiones y comentarios de Swift, puede utilizar variables predefinidas que se expandirán en el valor correspondiente.
Las variables predefinidas reciben los siguientes valores:
${SIGNATURE} Firma del inicializador
Todas las variables predefinidas en la plantilla Swift File Header (pestaña Includes) están disponibles también (p. ej.: ${USER}, ${DATE}).
