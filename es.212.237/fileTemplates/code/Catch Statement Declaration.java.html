Rellena el parámetro catch cuando se genera, por ej. cuando se llama a la función Código | Rodear con...
Esta plantilla integrada se puede editar. Por ejemplo, puede hacer que el parámetro generado sea final y usar siempre el nombre predefinido a pesar del nombre completo de la excepción capturada.
Las variables predefinidas toman los siguientes valores:
${EXCEPTION} Nombre de la variable Exception especificada como parámetro catch derivado del tipo de excepción capturado, e, si el nombre completo termina con Exception.
${EXCEPTION_TYPE} Tipo del parámetro catch.
