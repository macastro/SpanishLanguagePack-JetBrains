Ingrese el cuerpo del método que implementará cuando el método generado se crea mediante acciones tales como Código | Generar | Implementar métodos.
Esta plantilla es editable. Puede utilizar expresiones de Swift y comentarios además de las variables predefinidas que se expanden a sus valores correspondientes.
Las variables predefinidas reciben los siguientes valores:
${RETURN_TYPE} El tipo de retorno del método creado
${DEFAULT_RETURN_VALUE} El valor predeterminado sugerido para el valor de retorno (puede estar vacío)
${SIGNATURE} La firma del método
También se pueden utilizar todas las variables predefinidas en la plantilla Encabezado del archivo Swift (en la pestaña Inclusiones), como ${USER}, ${DATE}, etc.
