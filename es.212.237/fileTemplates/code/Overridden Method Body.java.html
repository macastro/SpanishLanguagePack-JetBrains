Rellena el cuerpo de un método sobreescrito cada vez que se genera mediante el programa, por ejemplo, cuando se llama a la función Código | Sobreescribir método....
Esta plantilla integrada es editable. Junto con las expresiones y los comentarios de Java, también puedes utilizar las variables predefinidas que se ampliarán a los valores correspondientes.
Las variables predefinidas toman los siguientes valores:
${CALL_SUPER} Llamada al método super, para métodos void - super.nombre_método(); con o sin parámetros para otros métodos - return super.nombre_método(); con o sin parámetros
${PLAIN_CALL_SUPER} Llamada al método super, super.nombre_método(); con o sin parámetros
${RETURN_TYPE} Tipo de retorno del método creado
${DEFAULT_RETURN_VALUE} Valor devuelto por el método de forma predeterminada
${METHOD_NAME} Nombre del método que se sobrescribe
${CLASS_NAME} Nombre completo de la clase en la que se sobrescribe el método
${SIMPLE_CLASS_NAME} Nombre no calificado de la clase en la que se implementa el método
Todas las variables predefinidas de la plantilla Encabezado de archivo (pestaña Incluye) también están disponibles (por ejemplo, ${USER}, ${DATE}, etc.)
