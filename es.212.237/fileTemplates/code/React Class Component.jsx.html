Esta plantilla incorporada se utiliza para crear componentes de clase React (por ejemplo, a través de la refactorización Extraer componente). Puede personalizar esta plantilla, por ejemplo, utilizando Componente en lugar de ComponentePuro o cambiando las definiciones de los tipos de prop. También puede utilizar variables predefinidas. Estas variables se reemplazarán con los valores correspondientes cuando se aplique la plantilla.
Las variables predefinidas reciben los siguientes valores:
${NAME} nombre del componente
${COMPONENT_BODY} cuerpo del componente
${PROP_TYPES} definición de tipos prop (utilizando el módulo prop-types)
${PROP_TYPES} valores de prop predeterminados que utilizará React cuando no se proporcione el valor de prop.
