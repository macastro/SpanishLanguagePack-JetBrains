Una plantilla incorporada para métodos de prueba de PHPUnit. Por ejemplo, se utiliza cuando se selecciona la casilla de verificación para generar métodos de prueba en el cuadro de diálogo "Nueva prueba".
Las variables predefinidas reciben los siguientes valores:
${NAME} Nombre del método.
${CAPITALIZED_NAME} Nombre del método con la primera letra en mayúscula.
${TESTED_NAME} Nombre de la clase que se prueba.
