Crea el cuerpo de una función implementada cuando esta se genera, por ejemplo, mediante la acción Código | Implementar Métodos.
Al editar la plantilla, puedes usar expresiones de C++, comentarios y las siguientes variables predefinidas:
${RETURN_TYPE} el tipo de retorno de la función.
${DEFAULT_RETURN_VALUE} el valor de retorno predeterminado de la función.
Todas las variables predefinidas de la plantilla Encabezado del Archivo C (la pestaña Incluye), como ${USER} y ${DATE}, también están disponibles.
