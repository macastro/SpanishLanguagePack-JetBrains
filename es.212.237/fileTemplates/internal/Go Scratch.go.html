Go Scratch es una plantilla preestablecida que se utiliza cada vez que se crea un archivo Go scratch nuevo. Los archivos scratch son archivos completamente funcionales, ejecutables y depurables que se pueden utilizar para elaborar ideas de código. Para crear un archivo scratch, abra Archivo | Nuevo | Archivo scratch. En el cuadro de diálogo Nuevo archivo scratch, busque Go y selecciónelo.

La plantilla se puede editar. En la plantilla, se pueden utilizar las siguientes variables predefinidas. Estas variables se expandirán más tarde en el editor al valor correspondiente.

También es posible especificar variables personalizadas. Las variables personalizadas utilizan el siguiente formato: ${VARIABLE_NAME}, donde VARIABLE_NAME es el nombre de la variable (por ejemplo, ${MY_CUSTOM_FUNCTION_NAME}). Antes de que el IDE use variables personalizadas para crear un archivo nuevo, se mostrará un cuadro de diálogo donde se puede definir el valor de las variables personalizadas de la plantilla.

Mediante el uso de la directiva #parse, se pueden incluir plantillas que se encuentran en la pestaña Incluir. Para incluir una plantilla, especifique el nombre completo de la plantilla entre comillas como parámetro (por ejemplo, #parse("File Header")).
Lista de variables predefinidas
${GO_PACKAGE_NAME_WITH_TEST_SUFFIX} Nombre del paquete en el que se crea el archivo de prueba nuevo. El archivo debe tener el sufijo _test. El nombre tiene el siguiente formato: package_test.
${GO_PACKAGE_NAME} Nombre del paquete en el que se crea el archivo nuevo.
${DATE} Fecha del sistema actual.
${DAY} Día del mes actual.
${DAY_NAME_SHORT} Primeras 3 letras del nombre del día actual (por ejemplo, Lun, Mar, etc.).
${DAY_NAME_FULL} Nombre completo del día actual (por ejemplo, Lunes, Martes, etc.).
${DIR_PATH} Ruta del directorio donde se almacenará el archivo nuevo. Esta ruta es relativa a la raíz del proyecto.
${DS} El signo de dólar ($). Esta variable se utiliza para escapar del carácter del dólar para que no se trate como prefijo de las variables de plantilla.
${FILE_NAME} Nombre del archivo nuevo.
${HOUR} Hora actual.
${MINUTE} Minuto actual.
${SECOND} Segundo actual.
${MONTH} Mes actual.
${MONTH_NAME_SHORT} Primeras 3 letras del nombre del mes actual (por ejemplo, Ene, Feb, etc.).
${MONTH_NAME_FULL} Nombre completo del mes actual (por ejemplo, Enero, Febrero, etc.).
${NAME} Nombre de la entidad nueva (archivo, tipo, interfaz, etc.).
