Se aplica a los nuevos archivos package-info.java creados invocando Nuevo | package-info.java en la ventana de herramientas Proyecto.

Esta plantilla integrada es editable. Junto con las expresiones y los comentarios de Java, también puedes utilizar las variables predefinidas (que se enumeran a continuación) que luego se expandirán como macros en valores correspondientes.

También es posible especificar variables personalizadas. Las variables personalizadas utilizan el siguiente formato: ${NOMBRE_DE_VARIABLE}, donde NOMBRE_DE_VARIABLE es un nombre para tu variable (por ejemplo, ${MI_NOMBRE_DE_FUNCIÓN_PERSONALIZADA}). Antes de que el IDE cree un nuevo archivo con variables personalizadas, verás un cuadro de diálogo donde puedes definir valores para variables personalizadas en la plantilla.

Utilizando la directiva #parse, puedes incluir plantillas desde la pestaña Incluye. Para incluir una plantilla, especifica el nombre completo de la plantilla como un parámetro entre comillas (por ejemplo, #parse("File Header.java").
Las variables predefinidas toman los siguientes valores:
${NOMBRE_DEL_PAQUETE} Nombre del paquete en el que se crea un nuevo archivo package-info.java
${USUARIO} Nombre de inicio de sesión del sistema del usuario actual
${FECHA} Fecha del sistema actual
${HORA} Hora del sistema actual
${AÑO} Año actual
${MES} Mes actual
${NOMBRE_DEL_MES_CORTO} Primeras 3 letras del nombre del mes actual (ene., feb., etc.)
${NOMBRE_DEL_MES_COMPLETO} Nombre completo del mes actual (enero, febrero, etc.)
${DÍA} Día actual del mes
${HORA} Hora actual
${MINUTO} Minuto actual
${NOMBRE_DEL_PROYECTO} Nombre del proyecto actual
