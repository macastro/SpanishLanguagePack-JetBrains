Una plantilla incorporada para rasgos de PHP. Se utilizará cada vez que se cree un nuevo rasgo de PHP, seleccionándolo del menú emergente Nuevo | Clase de PHP en una de las vistas de proyecto.
La plantilla es editable. Además de las declaraciones, expresiones y comentarios PHP, también puedes utilizar las variables predefinidas que aparecen en la lista siguiente. Cada variable se expandirá en el valor correspondiente, del mismo modo que una macro.
Puedes utilizar la directiva #parse para incluir plantillas de la pestaña Incluye, especificando como parámetros el nombre completo de la plantilla deseada entre comillas.
Las variables predefinidas toman los siguientes valores:
${FILE_NAME} Nombre del archivo actual
${USER} Nombre de inicio de sesión del sistema del usuario actual
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${DAY} Día del mes actual
${HOUR} Hora actual
${MINUTE} Minuto actual
${PRODUCT_NAME} Nombre actual del IDE
${DS} El símbolo del dólar se evalúa como el carácter '$' normal.
${PROJECT_NAME} Nombre del proyecto actual
