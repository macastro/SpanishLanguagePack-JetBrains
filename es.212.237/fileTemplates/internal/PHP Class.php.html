Una plantilla incorporada para las clases PHP. Se utiliza cada vez que se crea una nueva clase PHP seleccionando Nuevo | Clase PHP en el menú emergente de una de las vistas de proyecto.
La plantilla es editable. Además de las sentencias, expresiones y comentarios de PHP, también puedes utilizar las variables predefinidas que se muestran a continuación. Cada variable se expandirá al valor correspondiente como una macro.
Se pueden incluir plantillas que se encuentran en la pestaña Incluye mediante la directiva #parse especificando el nombre completo de la plantilla deseada como parámetro entre comillas.
Las variables predefinidas toman los siguientes valores:
${FILE_NAME} Nombre del archivo actual
${USER} Nombre de inicio de sesión del sistema del usuario actual
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${DAY} Día del mes actual
${HOUR} Hora actual
${MINUTE} Minuto actual
${PRODUCT_NAME} Nombre del IDE actual
${PROJECT_NAME} Nombre del proyecto actual
${DS} El símbolo de dólar se evalúa en el carácter '$' normal.
