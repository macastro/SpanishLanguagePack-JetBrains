Se aplica a nuevas interfaces Java que se crean al invocar Nuevo | Clase de Java | Interfaz en la ventana de herramientas Proyecto.

Esta plantilla integrada se puede editar. Junto con las expresiones y los comentarios de Java, también puede usar las variables predefinidas (que se enumeran a continuación) que luego se expandirán como macros en valores correspondientes.

También es posible especificar variables personalizadas. Las variables personalizadas usan el siguiente formato: ${NOMBRE_VARIABLE}, donde NOMBRE_VARIABLE es un nombre para su variable (por ejemplo, ${MY_CUSTOM_FUNCTION_NAME}). Antes de que el IDE cree un nuevo archivo con variables personalizadas, verá un cuadro de diálogo donde puede definir valores para variables personalizadas en la plantilla.

Mediante el uso de la directiva #parse, puede incluir plantillas de la pestaña Incluidos. Para incluir una plantilla, especifique el nombre completo de la plantilla como parámetro entre comillas (por ejemplo, #parse("File Header.java").
Las variables predefinidas toman los siguientes valores:
${PACKAGE_NAME} Nombre del paquete en el que se crea una nueva interfaz
${NAME} Nombre de la nueva interfaz que especificó en el cuadro de diálogo Crear clase nueva
${USER} Nombre de inicio de sesión del sistema del usuario actual
${DATE} Nombre de inicio de sesión del sistema del usuario actual
${TIME} Tiempo actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${MONTH_NAME_SHORT} Primeras 3 letras del nombre del mes actual (ene, feb, etc.)
${MONTH_NAME_FULL} Nombre completo del mes actual (enero, febrero, etc.)
${DAY} Día actual del mes
${HOUR} Hora actual
${MINUTE} Minuto actual
${PROJECT_NAME} Nombre del proyecto actual
