Una plantilla incorporada para un archivo RBS. Se utiliza cada vez que se crea un nuevo archivo RBS al seleccionar Nuevo | Archivo/Clase RBS desde el menú emergente en una de las vistas de proyecto. Junto con las declaraciones, expresiones y comentarios RBS, también puedes utilizar las variables predefinidas que se muestran a continuación. Cada variable se expande como macros al valor correspondiente. Por medio de la directiva #parse, puede incluir plantillas de la pestaña Incluir especificando el nombre completo de la deseada en los parámetros entre comillas.
Las variables predefinidas toman los siguientes valores:
${FILE_NAME} Nombre del archivo actual
${USER} Nombre de usuario actual en el sistema
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${DAY} Día actual del mes
${HOUR} Hora actual
${MINUTE} Minuto actual
${PRODUCT_NAME} Nombre del IDE actual
${PROJECT_NAME} Nombre del proyecto actual
