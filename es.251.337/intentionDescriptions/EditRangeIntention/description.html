Puede agregar la anotación @Range o editar sus valores. La anotación @Range puede agregarse a parámetros de método, campos o valores de retorno de método, y es utilizada por herramientas de análisis estático (como inspecciones) para validar el código.

Se puede especificar un rango para valores de tipo long, int, short, char o byte. El rango se define especificando un límite inferior from (inclusivo) y un límite superior to (inclusivo).
