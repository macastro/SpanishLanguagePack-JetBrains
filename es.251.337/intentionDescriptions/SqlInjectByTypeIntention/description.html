Inyectar un dialecto y asociarlo con el tipo SQL seleccionado. Aplicar este dialecto a todas las coincidencias del mismo tipo (por ejemplo, query).

En el ejemplo Before, puede invocar la acción de intención Inyectar por tipo en el valor 'SELECT * FROM foo', y luego inyectar este tipo de objeto y asociarlo con cualquier dialecto SQL. El tipo de objeto para este valor es query (especificado por q query DEFAULT 'SELECT * FROM foo'). Por lo tanto, la próxima vez que utilice el tipo query en el código, el dialecto seleccionado se inyectará automáticamente.
