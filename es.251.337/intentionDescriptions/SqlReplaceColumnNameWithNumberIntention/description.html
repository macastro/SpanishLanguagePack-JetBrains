Reemplazar referencias posicionales con nombres de columnas.

Algunos DBMS permiten referenciar columnas en las cláusulas ORDER BY y GROUP BY mediante números de secuencia. Todos los dialectos soportan ORDER BY, excepto Cassandra, ClickHouse, Hive, Spark y HSQLDB. MySQL, Maria, PostgreSQL (y sus subdialectos), Exasol, SQLite y Snowflake soportan GROUP BY.

Ejemplo (PostgreSQL):

CREATE TABLE tab(a INT, b INT);
SELECT * FROM tab ORDER BY 1;

Puede usar esta intención para reemplazar el 1 con el nombre de la columna en la consulta SELECT. Después de aplicar esta intención, la consulta se verá así.

SELECT * FROM tab ORDER BY a;
