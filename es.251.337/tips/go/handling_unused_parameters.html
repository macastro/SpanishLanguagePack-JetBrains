Corrección rápida: Eliminar nombre del parámetro

Para los parámetros no utilizados, GoLand resalta tanto el nombre como el tipo del parámetro (si ambos pueden eliminarse). Si el tipo se utiliza para implementar una especificación de método, solo se resalta el nombre.

En este último caso, puedes hacer clic en el nombre y presionar &shortcut:ShowIntentionActions; para elegir entre las siguientes correcciones rápidas: Eliminar nombre del parámetro o Renombrar a _. El mismo comportamiento está disponible para funciones que se utilizan como tipos de función.
