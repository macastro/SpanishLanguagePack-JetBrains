Autocompletado de código para el paquete 'time'

El paquete time utiliza diseños predefinidos para Time.Format y time.Parse.

En GoLand, cuando presionas &shortcut:CodeCompletion;, verás marcadores de posición para elementos de fecha y hora que cumplen con el estándar ISO-8601. El autocompletado también sugiere diseños de tiempo estándar fuera de los literales de cadena.
