Corrección rápida: Manejo de errores en Go

La corrección rápida Manejar error puede asignar el resultado de la llamada a una variable y generar una comprobación if error not nil.

La corrección rápida Envolver el manejo de errores en una clausura, disponible para las declaraciones defer y go, genera código similar a la corrección rápida Manejar error, pero envuelve todo en una clausura.
