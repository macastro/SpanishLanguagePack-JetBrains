Abrir y depurar volcados de memoria de Go

Al depurar un programa, puede ser necesario examinar el flujo de ejecución del código para comprender el estado actual del programa. Un volcado de memoria de Go es un archivo que contiene un volcado de la memoria de un proceso que se está ejecutando y sus estados en un momento dado durante la ejecución de un programa.

Es posible abrir y depurar volcados de memoria de Go utilizando GoLand. Para abrir un volcado de memoria de Go, vaya a Run | Open Core Dump. Especifique la ruta al ejecutable y al archivo de volcado y haga clic en OK.

Nota: Los archivos de volcado de memoria de Go solo se pueden crear en máquinas Linux. Los archivos de volcado se pueden mostrar en cualquier sistema operativo compatible con GoLand.
