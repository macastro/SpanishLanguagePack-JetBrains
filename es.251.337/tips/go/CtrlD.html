Duplicar bloques de código o líneas

Presiona &shortcut:EditorDuplicate; en el editor para duplicar el bloque de código seleccionado o la línea actual si no hay ninguna selección.

Si deseas insertar un salto de línea entre la parte duplicada y el texto original, presiona &shortcut:GotoAction; y escribe Duplicate Entire Lines. Puedes asignar un atajo de teclado para esta acción en la configuración de Mapeo de teclas &shortcut:ShowSettings;.
