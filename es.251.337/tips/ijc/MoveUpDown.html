Mover sentencias arriba y abajo

Código | Mover sentencia arriba/abajo es útil para reorganizar líneas de código, por ejemplo, para acercar las declaraciones de variables a donde se utilizan.

Selecciona un fragmento de código y presiona &shortcut:MoveStatementUp; o &shortcut:MoveStatementDown;.

Si no hay ninguna selección en el editor, se moverá la línea de código donde se encuentra el cursor.
