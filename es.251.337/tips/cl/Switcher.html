Navegador de cambio

Utiliza el navegador &shortcut:Switcher; para navegar entre archivos abiertos y ventanas de herramientas. Mantén presionada la tecla Control para mantener abierta la ventana emergente del navegador. Para navegar, utiliza Tab y Shift+Tab.

Presiona Delete o BackSpace para cerrar las pestañas del editor u ocultar las ventanas de herramientas.
