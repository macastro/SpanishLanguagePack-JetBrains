Para comentar o descomentar código con un comentario de una línea, presiona &shortcut:CommentByLineComment;. Se comentará/descomentará la línea actual y el cursor se moverá a la siguiente línea.

Alternativamente, puedes presionar &shortcut:CommentByBlockComment; para poner la selección dentro de un comentario de bloque o descomentar el comentario de bloque completo si el cursor está dentro de él.
