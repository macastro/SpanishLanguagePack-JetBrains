Para alternar entre ventanas de herramientas y archivos abiertos, utiliza el Switcher &shortcut:Switcher;. Elige la vista deseada con las teclas de dirección y presiona &shortcut:EditorEnter; para abrirla.

También puedes pulsar la tecla Delete o BackSpace para cerrar una pestaña del editor u ocultar una ventana de herramienta.
