PyCharm permite configurar el intérprete de Python en diferentes etapas del desarrollo:

  • Al crear un proyecto (Archivo | Nuevo Proyecto, o Pantalla de Bienvenida y Crear Nuevo Proyecto).

  • En un proyecto existente, usando el widget Intérprete de Python en la barra de estado, o seleccionando Proyecto | Intérprete de Python en la Configuración del proyecto.
