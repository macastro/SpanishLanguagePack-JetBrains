Búsqueda en cualquier lugar

Presionar Shift dos veces permite buscar archivos, acciones, símbolos, elementos de IU, ramas de Git y comentarios en todo el proyecto. Presionando Shift dos veces nuevamente, se buscarán también elementos que no están incluidos en el proyecto.

Para restringir los resultados, se pueden usar tabulaciones o bien usar directamente el atajo necesario (por ejemplo, &shortcut:GotoClass; para clases, &shortcut:GotoFile; para archivos, &shortcut:GotoSymbol; para símbolos, &shortcut:GotoAction; para acciones).
