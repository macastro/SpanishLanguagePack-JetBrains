Habilitar el modo de solo lectura para conexiones

Para proteger tu base de datos de modificaciones accidentales, puedes activar el modo de solo lectura para una conexión.

Para habilitar el modo de solo lectura para una conexión, haz clic en Archivo | Fuentes de datos y selecciona la fuente de datos necesaria en la lista Fuentes de Datos. En la pestaña Opciones, selecciona la casilla de verificación Solo lectura.
