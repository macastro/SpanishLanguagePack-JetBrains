Consolas de bases de datos

Las Consolas de bases de datos son archivos SQL que ya están adjuntados a una fuente de datos. Cuando crea una fuente de datos, se crea automáticamente una consola de edición de consultas (query console). Para abrir una consola, haga clic con el botón derecho en la fuente de datos y seleccione Navegación | Ir a consola de edición de consultas | Abrir consola predeterminada.
