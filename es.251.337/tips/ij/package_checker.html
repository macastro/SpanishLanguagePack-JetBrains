Analizar paquetes de software

Utilice el complemento Package Checker integrado para asegurarse de que sus proyectos Maven y Gradle no tengan dependencias vulnerables. Desde el menú principal, seleccione Código | Analizar código y luego haga clic en Mostrar dependencias vulnerables.

Los resultados del análisis se mostrarán en una ventana de herramientas separada, donde cada dependencia encontrada tendrá una descripción detallada.
