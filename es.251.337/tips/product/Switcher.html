Switcher para navegación

Usa el switcher &shortcut:Switcher; para navegar entre archivos abiertos y ventanas de herramientas. Mantén Control presionado para mantener abierto el popup del switcher. Para la navegación, usa Tab y Shift+Tab.

Presiona Delete o BackSpace para cerrar una pestaña de editor u ocultar una ventana de herramientas.
