Mejora el código con acciones de contexto

Pulsa &shortcut:ShowIntentionActions; en el editor para corregir un error resaltado o una advertencia, mejorar u optimizar una construcción de código. Para algunas acciones de intención, puedes abrir una vista previa o una descripción de la acción pulsando &shortcut:QuickJavaDoc; (Ver | Documentación rápida).
