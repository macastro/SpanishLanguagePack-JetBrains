Completar nombres de variables

La finalización de código puede sugerir un nombre para una variable cuando la declares. Por ejemplo, comienza a escribir private FileOutputStream. El IDE te mostrará una lista con sugerencias.

Puedes personalizar los prefijos de nombre para las variables locales, parámetros, campos de instancia y estáticos en el diálogo Configuraciones/Preferencias (&shortcut:ShowSettings;) bajo Estilo del código.
