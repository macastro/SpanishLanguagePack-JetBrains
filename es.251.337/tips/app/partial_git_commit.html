Commit changes partially

You can split changes made to the same file between different commits. En la ventana de herramienta Commit &shortcut:ActivateCommitToolWindow;, haz clic en Diff. Selecciona la casilla de verificación junto a cada fragmento de código modificado o nuevo que quieras confirmar y haz clic en Commit. Los cambios no seleccionados permanecerán en la lista de cambios actual.
