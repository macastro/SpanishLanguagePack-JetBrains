Editar múltiples líneas simultáneamente

Presiona &shortcut:EditorToggleColumnMode; para activar el modo de selección de columnas. En este modo, puedes realizar los mismos cambios en múltiples líneas. Esta función es especialmente útil cuando necesitas formatear texto como una lista y agregar el mismo marcador en cada línea. Usa el mismo atajo de teclado para desactivar este modo.
