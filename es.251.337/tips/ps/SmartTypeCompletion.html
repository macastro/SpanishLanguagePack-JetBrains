Enderezamiento de código por tipo

Enderezar de código por tipo analiza el tipo esperado para la expresión completa y ayuda a encontrar métodos y variables aplicables en el contexto actual. Este enderezamiento de código funciona después de la palabra clave return, en asignaciones y en listas de argumentos de invocación de métodos. Se puede presionar &shortcut:SmartTypeCompletion; (Ctrl + Cmd + P) (Ctrl + Mayús + P) para filtrar la lista de finalización.
