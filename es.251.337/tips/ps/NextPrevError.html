Navegación entre errores

Pulsa &shortcut:GotoNextError; / &shortcut:GotoPreviousError; para navegar entre los errores de sintaxis resaltados.

Pulsa &shortcut:PreviousOccurence; / &shortcut:NextOccurence; para navegar entre mensajes de error o resultados de búsqueda.

Para omitir advertencias, haz clic con el botón derecho en la barra lateral de inspección / barra de marcadores y selecciona "Ir solo a problemas de mayor prioridad".
