Archivos SQL como fuentes de datos DDL

Los archivos SQL que contienen sentencias CREATE pueden utilizarse como fuentes de datos. Esto le permite construir diagramas en el editor haciendo referencia a todas las tablas, columnas y otros objetos definidos en dichos archivos.

Para crear una fuente de datos DDL, arrastre el archivo DDL desde el administrador de archivos al Explorador de Base de Datos (Ver | Ventanas de Herramientas | Explorador de Base de Datos).
