Ver archivos recientes

Si ha cerrado accidentalmente la consola de consultas, un script SQL o cualquier otra pestaña, presione &shortcut:RecentFiles;. El diálogo Archivos recientes mostrará una lista de los archivos editados recientemente.

Presione &shortcut:RecentFiles; (Ver | Archivos recientes) para ver la lista de archivos abiertos recientemente.
