Nombres de clase y archivo en cumplimiento

Al cambiar el nombre de una clase, WebStorm sugiere cambiar también el nombre del archivo con el mismo nombre. Si se acepta esta sugerencia, WebStorm actualiza el nombre del archivo incluido en las sentencias de importación de otros archivos.

Si no se acepta esta sugerencia, se puede cambiar el nombre del archivo posteriormente mediante la acción de intención "Cambiar nombre de archivo...".
