Documentación externa

Para abrir el navegador con la página de documentación del elemento en la posición del cursor, presione &shortcut:ExternalJavaDoc; o seleccione Herramientas | Navegador web y vista previa.

Asegúrese de tener configurado el navegador predeterminado en la configuración. Para verificar el navegador predeterminado, presione &shortcut:ShowSettings; para abrir la configuración y navegue hasta Herramientas | Navegador web y vista previa.
