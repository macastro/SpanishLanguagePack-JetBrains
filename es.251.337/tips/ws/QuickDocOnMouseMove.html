Colocar el cursor para mostrar un documento rápido

No es necesario invocar el documento rápido explícitamente mediante &shortcut:QuickJavaDoc;. Al colocar el puntero del ratón sobre el símbolo de destino, se mostrará automáticamente una ventana emergente con el documento rápido.

Esta acción está habilitada de forma predeterminada. Para deshabilitar esta acción, abre el cuadro de diálogo "Configuración" (&shortcut:ShowSettings;), ve a "Editor | Edición de código" y desactiva la casilla de verificación "Mostrar documento rápido al colocar el cursor sobre".
