Selector de navegación

Para moverse entre los archivos abiertos y las ventanas de herramientas, usa el selector &shortcut:Switcher;. Para salir del selector, mantén presionada la tecla Control. Para navegar, usa Tab y Shift+Tab.

Para cerrar una pestaña de editor u ocultar una ventana de herramientas, presiona Eliminar o Retroceso.
