Añadir avisos de copyright

En PyCharm, puede configurar distintos avisos de copyright para diferentes grupos de archivos. Para mayor comodidad, puede utilizar variables. Por ejemplo, utilice $today.year para mantener el año actualizado.

En el cuadro de diálogo Ajustes (&shortcut:MostrarAjustes;), seleccione Editor | Copyright | PerfilesDeCopyright, cree un nuevo perfil y agregue el texto para el aviso de copyright utilizando variables.
