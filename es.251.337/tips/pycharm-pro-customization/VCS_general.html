Control de Versiones

PyCharm ofrece integración lista para usar con los siguientes sistemas de control de versiones: Git, Mercurial, Subversion y Perforce.

Versiona la aplicación para colaborar en ella, así como para eliminar los riesgos de almacenar todo el código base localmente. Presiona &shortcut:ActivateVersionControlToolWindow; para ver el historial de cambios en el proyecto o presiona &shortcut:CheckinProject; para enviar los cambios locales al repositorio.
