Usar múltiples cursores de texto

En PyCharm Community, puedes usar múltiples cursores de texto. Mantén presionado Shift+Alt+Ctrl (Windows y Linux) o Shift+Alt(Option)+Cmd (macOS) y haz clic en diferentes ubicaciones en el editor para establecer cursores de texto adicionales.

Luego puedes usar las teclas de flecha izquierda o derecha con Shift para seleccionar múltiples fragmentos de texto.

Presiona &shortcut:EditorEscape; para eliminar todos los cursores de texto excepto el último agregado.
