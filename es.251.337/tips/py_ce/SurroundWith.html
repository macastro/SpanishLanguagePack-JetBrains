Rodear código

Para rodear rápidamente un bloque de código específico con un constructo válido, selecciona el bloque de código pertinente en el editor y presiona &shortcut:SurroundWith; (menú Código | Rodear).

La lista de opciones o envoltorios disponibles variará según el contexto y el lenguaje. Por ejemplo, puedes realizar acciones como rodear un bloque html con etiquetas.
