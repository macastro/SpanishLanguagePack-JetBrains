Revisar ortografía y gramática

La revisión de ortografía y gramática está habilitada por defecto. Estas comprobaciones son proporcionadas por el plugin Grazie que viene incluido con el IDE. Para configurar los ajustes de revisión, presiona &shortcut:ShowSettings; para abrir la configuración, luego navega a Editor | Lenguaje Natural | Gramática.

Todo el texto, incluyendo comentarios, cadenas de texto y literales, se verifica contra diccionarios predefinidos o personalizados. Los errores ortográficos se resaltan para que puedas seleccionar la palabra correcta, aceptar la ortografía actual o deshabilitar la revisión.

Para corregir un error ortográfico o gramatical, presiona &shortcut:ShowIntentionActions; para ver la lista de acciones disponibles.
