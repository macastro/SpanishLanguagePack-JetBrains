Barra de navegación

La barra de navegación es otra forma de acceder rápidamente a la ventana de herramientas del Proyecto, donde puede explorar la estructura del proyecto, abrir archivos específicos y navegar a elementos específicos del código en el archivo actual.

Presione &shortcut:ShowNavBar; para mostrar la barra de navegación y use las teclas de flecha para alternar entre archivos y carpetas.
