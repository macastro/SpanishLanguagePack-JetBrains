Añadir avisos de copyright

PyCharm Community le permite configurar diferentes avisos de copyright para cada conjunto de archivos. Puede utilizar variables para mayor comodidad. Por ejemplo: $today.year mantendrá el año actualizado.

En el cuadro de diálogo "Configuración" (&shortcut:ShowSettings;), seleccione "Editor | Copyright | Perfiles de copyright" para crear nuevos perfiles o añadir texto de aviso de copyright utilizando variables.
