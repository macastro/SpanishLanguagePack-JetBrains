Selector para la navegación

Utiliza el selector &shortcut:Switcher; para navegar entre archivos y ventanas de herramientas abiertas. Mantén Ctrl presionado para dejar la ventana emergente del selector abierta. Para navegar, utiliza Tab y Shift+Tab.

Presiona Delete o BackSpace para cerrar una pestaña del editor u ocultar una ventana de herramientas.
