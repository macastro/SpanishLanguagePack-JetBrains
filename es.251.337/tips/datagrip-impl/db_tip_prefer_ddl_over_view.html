DDL del objeto de base de datos, datos y ubicación en el Explorador de Base de Datos

Presiona &shortcut:GotoDeclaration; para abrir el DDL del objeto seleccionado. Por ejemplo, si presionas este atajo en el nombre de una tabla en la sentencia SELECT, verás el DDL de esta tabla (la sentencia CREATE TABLE).

Presiona &shortcut:EditSource; para abrir los datos del objeto. Por ejemplo, si presionas este atajo en el nombre de una tabla en la sentencia SELECT, verás el contenido de la tabla.

Presiona Alt+Shift+B para navegar al objeto en la ventana de herramientas Explorador de Base de Datos.
