Diagramas

Puedes crear un diagrama para un origen de datos, un esquema o una tabla.

En el Explorador de base de datos (Ver | Ventanas de herramientas | Explorador de base de datos), haz clic derecho en un objeto de base de datos y selecciona Diagramas | Mostrar visualización. Para ver un diagrama en una ventana emergente, selecciona Diagramas | Mostrar emergente de visualización.
