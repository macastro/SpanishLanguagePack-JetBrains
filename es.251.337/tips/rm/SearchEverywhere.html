Buscar en todas partes

Presiona Shift dos veces para buscar archivos, acciones, símbolos, elementos de la interfaz, ramas Git y comentarios en tu proyecto. Presiona Shift dos veces nuevamente para extender la búsqueda a elementos fuera del proyecto.

Puedes limitar el alcance de la búsqueda utilizando pestañas o los atajos directos: &shortcut:GotoClass; para clases, &shortcut:GotoFile; para archivos, &shortcut:GotoSymbol; para símbolos y &shortcut:GotoAction; para acciones.
