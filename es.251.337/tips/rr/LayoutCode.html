Reformatear código

Para reformatear el código según el estilo de código actual, seleccione Código | Reformatear Código desde el menú principal, o presione &shortcut:ReformatCode;.

Por defecto, RustRover utilizará el formateador incorporado. Si prefiere Rustfmt, vaya a Configuración | Rust | Rustfmt y marque la casilla Usar Rustfmt en lugar del formateador incorporado.
