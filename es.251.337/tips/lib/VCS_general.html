Control de Versiones

DataSpell brinda integración de fábrica con los siguientes sistemas de control de versiones: Git, Mercurial, Subversion y Perforce.

Realiza el versionado de tu aplicación para colaborar en ella, así como para eliminar el riesgo de almacenar todo tu código base localmente. Presiona &shortcut:ActivateVersionControlToolWindow; para ver el historial de cambios en el proyecto o presiona &shortcut:CheckinProject; para enviar tus cambios locales al repositorio.
