Reporta asignaciones para o modificaciones de parámetros de método.

Aunque ocasionalmente se pretende, esta construcción puede ser confuso y por lo tanto es prohibida en algunos proyectos Java.

La solución rápida agrega una declaración para una nueva variable.

Ejemplo:


  void printTrimmed(String s) {
    s = s.trim();
    System.out.println(s);
  }

Después de que se aplique la solución rápida:


  void printTrimmed(String s) {
    String trimmed = s.trim();
    System.out.println(trimmed);
  }

Utiliza la opción Ignorar si la asignación es una transformación del parámetro original para ignorar las asignaciones que modifican el valor del parámetro en base a su valor anterior.
