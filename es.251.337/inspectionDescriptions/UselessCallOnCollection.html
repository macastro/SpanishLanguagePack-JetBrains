Informa sobre llamadas a filter… en colecciones ya filtradas de la biblioteca estándar.

Algunas funciones de la biblioteca estándar como filterNotNull() o filterIsInstance solo tienen sentido cuando se llaman en receptores de un tipo diferente al tipo resultante. En otros casos, estas llamadas pueden ignorarse ya que el resultado será el mismo.

La corrección rápida Eliminar llamada redundante está disponible para arreglar el código automáticamente.

Ejemplo:


  fun test(list: List<String>) {
      val x = list.filterNotNull() // La corrección rápida simplificará a 'list'
      val y = list.filterIsInstance<String>() // La corrección rápida simplificará a 'list'
  }
