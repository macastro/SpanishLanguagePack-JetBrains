La inspección global está diseñada para identificar secciones del código que se consideran vulnerables, donde los datos obtenidos de una fuente potencialmente no confiable pueden usarse en operaciones que conllevan un riesgo de seguridad.

Seguimiento del flujo de datos:

La inspección rastrea el flujo de datos desde la fuente hasta su uso en operaciones críticas. Esto incluye el análisis de la transferencia de datos a través de métodos, objetos y variables.

Considere un ejemplo en el que la entrada del usuario se usa directamente en una consulta SQL sin el preprocesamiento adecuado. La inspección emitirá una advertencia sobre una posible inyección SQL.

String userInput = getUserInput();
String sqlQuery = "SELECT * FROM users WHERE username = '" + userInput + "'";
