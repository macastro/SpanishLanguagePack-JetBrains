Informa sobre las diferencias entre los parámetros formales declarados y los argumentos reales, además de reportar argumentos incorrectos (como argumentos nombrados duplicados) y orden incorrecto de argumentos.

Ejemplo:


class Foo:
    def __call__(self, p1: int, *, p2: str = "%"):
        return p2 * p1
bar = Foo()
bar.__call__() # parámetros faltantes
bar(5, "#") # argumento inesperado

El código correcto se muestra a continuación:


class Foo:
    def __call__(self, p1: int, *, p2: str = "%"):
        return p2 * p1
bar = Foo()
bar.__call__(5)
bar(5, p2="#")
