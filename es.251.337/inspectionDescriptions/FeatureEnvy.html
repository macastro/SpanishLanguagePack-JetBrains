Informa sobre el code smell de Feature Envy. Se emitirá una advertencia cuando un método llame a métodos de otra clase tres o más veces. Las llamadas a clases de biblioteca, clases padre y clases contenedoras están excluidas de esta comprobación. Feature Envy generalmente indica que la funcionalidad está en la clase incorrecta.

Ejemplo:


  class JobManager {
    // Advertencia: este método llama a
    // tres métodos de la clase Job
    // Es mejor mover esta cadena de llamadas
    // a la propia clase Job.
    void performJob(Job job) {
      job.beforeStart();
      job.process();
      job.afterProcessing();
    }
  }
