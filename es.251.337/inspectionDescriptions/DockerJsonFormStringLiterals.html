Reporta cadenas de texto con comillas simples en formato de array JSON.

En la sintaxis de arrays JSON, se deben usar comillas dobles (") alrededor de las palabras, no comillas simples ('). De lo contrario, la construcción de Docker fallará.

Ejemplo:


  # Todos los siguientes comandos fallarán
  RUN ['/bin/bash', '-c', 'echo hello']
  ADD ['binaryA.jar', 'binary2.jar', 'destination/']
  COPY ['binaryA.jar', 'binary2.jar', 'destination/']

Después de aplicar la corrección rápida:


  RUN ["/bin/bash", "-c", "echo hello"]
  ADD ["binaryA.jar", "binary2.jar", "destination/"]
  COPY ["binaryA.jar", "binary2.jar", "destination/"]
