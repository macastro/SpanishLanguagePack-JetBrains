Reporta los operadores == y != usados para probar la igualdad de arreglos. En la mayoría de los casos, probar la igualdad del contenido del arreglo es lo deseado, lo cual puede hacerse con el método java.util.Arrays.equals().

Se sugiere una reparación rápida para reemplazar == con java.util.Arrays.equals().

Ejemplo:


  void foo(Object[] x, Object[] y) {
    boolean comparison = x == y;
  }

Después de aplicar la reparación rápida:


  void foo(Object[] x, Object[] y) {
    boolean comparison = Arrays.equals(x, y);
  }
