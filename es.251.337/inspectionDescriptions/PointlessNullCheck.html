Informa sobre comprobaciones de null seguidas de llamadas a métodos que siempre devuelven false cuando se pasa null (por ejemplo, Class.isInstance).

En estos casos, dado que la llamada al método siempre devolverá false, estas comprobaciones parecen redundantes.

Ejemplo:


  if (x != null && myClass.isInstance(x)) { ... }

Después de aplicar la corrección rápida:


  if (myClass.isInstance(x)) { ... }
