Informa sobre clases, métodos y campos obsoletos que se utilizan en el código.

Ejemplo:


  class MyCode {
  @Deprecated
  void oldMethod() {}// Advertencia: "Miembro obsoleto todavía en uso"
  void newMethod() {
    oldMethod(); // Uso obsoleto
  }
  }

Se ignorarán los usos dentro de elementos obsoletos.

Nota: Por razones de rendimiento, los miembros no privados solo se verifican cuando sus nombres aparecen con poca frecuencia en el proyecto.
