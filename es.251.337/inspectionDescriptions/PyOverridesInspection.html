Informa cuando un método decorado con @override no tiene un método coincidente en su clase superior

Ejemplo:


from typing import override
class Parent:
    def foo(self) -> int:
        return 1
    def bar(self, x: str) -> str:
        return x
class Child(Parent):
    @override
    def foo(self) -> int:
        return 2
    @override # Falta el método super para la función sobrescrita
    def baz(self) -> int:
        return 1
