Informa sobre operandos de cadenas vacías en la concatenación de cadenas. La concatenación con cadenas vacías se puede usar para convertir objetos no String o tipos primitivos a String, pero sería más claro usar el método String.valueOf().

Se sugiere simplificar la concatenación mediante la corrección rápida.

Ejemplo:


  void foo(int x, int y) {
    String s = "" + x + " ; " + y;
  }

Después de aplicar la corrección rápida:


  void foo(int x, int y) {
    String s = x + " ; " + y;
  }

La opción Informar solo casos donde la cadena vacía puede eliminarse sin cambios adicionales ignora los casos donde eliminar la cadena vacía requeriría agregar una conversión String.valueOf() de otro operando.
