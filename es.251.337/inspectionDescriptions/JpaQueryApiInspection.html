Símbolos no resueltos en nombres de informes, consultas nativas y métodos API relacionados:
  • Consultas con nombres desconocidos
  • Parámetros de consulta desconocidos

Ejemplo:


  @Entity
  @NamedQuery(name = "SelectByConditionQuery", query = "...")
  public class JavaEntity { ... }
  class Main {
    public static void executeQuery() {
      Persistence.createEntityManagerFactory("")
        .createEntityManager()
        .createNamedQuery("SelectAllQuery") // Error: nombre de consulta desconocido
        ...
    }
  }
