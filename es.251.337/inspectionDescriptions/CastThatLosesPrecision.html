Informa sobre operaciones de conversión entre tipos de datos primitivos que pueden causar pérdida de precisión.

Estas conversiones no son necesariamente un problema, pero pueden causar errores difíciles de rastrear si la pérdida de precisión es inesperada.

Ejemplo:


  int a = 420;
  byte b = (byte) a;

Use la opción Ignorar conversiones de int a char para ignorar las conversiones de int a char. Este tipo de conversión se usa frecuentemente al implementar operaciones de I/O, ya que el método read() de la clase java.io.Reader devuelve un int.

Use la opción Ignorar conversiones de int 128-255 a byte para ignorar las conversiones de valores constantes de int a byte (128-255). Estos valores se desbordarán a números negativos que aún caben en un byte.
