Informa sobre funciones de alcance intercambiables (let, run, apply, also).

El código se puede simplificar usando la función correspondiente.

Esta corrección rápida permite intercambiar entre las diferentes funciones de alcance.

Ejemplo:


  val x = "".let {
      it.length
  }

Después de aplicar la corrección rápida:


  val x = "".run {
      length
  }
