Informa sobre problemas relacionados con el uso de ngSrc en etiquetas img (NgOptimizedDirective).

Se informará sobre los siguientes problemas:

  • Etiquetas img que usan src en lugar de ngSrc
  • Falta de atributos width y height o fill cuando se usa ngSrc
  • Presencia simultánea de atributos width o height y fill en el mismo elemento cuando se usa ngSrc
