Informa sobre propiedades setter Bean con @Required que no han sido inyectadas o autowired.

Ejemplo:


<beans>
  <bean class="beans.MyComponent" id="foo"/>
  <!-- Informa "Falta la propiedad requerida: 'port'" -->
</beans>


public class MyComponent {
   @Required
   public void setPort(int port) // Informa "Propiedad requerida 'port' no inyectada"
   {...}
}
