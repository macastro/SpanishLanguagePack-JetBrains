Informa sentencias case sin bloques else. Un bloque else siempre debe estar presente para especificar el resultado por defecto si no se encuentra ninguna coincidencia.

Ejemplo:

# Mala práctica
case status
when :active
  perform_action
end
Hay una corrección rápida disponible para insertar un bloque else. Después de aplicar la corrección rápida, el resultado se verá así: 
# Buena práctica
case status
when :active
  perform_action
else
  # escribe código aquí
end
