Reporta los mismos problemas HTML en los comentarios Javadoc que han sido reportados por DocLint desde Java 8.

La inspección detecta los siguientes problemas:

  • Etiqueta autocerrada, no cerrada, desconocida, fuera de lugar o vacía
  • Atributo desconocido o incorrecto
  • Texto fuera de lugar

Ejemplo:


/**
 * Etiqueta desconocida: List<String>
 * Etiqueta no cerrada: error</b>
 * Texto o etiqueta fuera de lugar: <ul><li>uno</li>, <li>dos</li></ul>
 * Atributo incorrecto o vacío: <a href=""></a>
 * Etiqueta autocerrada: <br/>
 * ...
 */
void sample(){ }
