Informa sobre archivos Python 2 que contienen caracteres no ASCII sin una declaración de codificación en la parte superior.

Ejemplo:


class A(object):
# №5
    def __init__(self):
        pass

En este ejemplo, el IDE informa sobre un símbolo no ASCII en el comentario y la falta de una declaración de codificación. Aplica la corrección rápida sugerida para añadir la declaración de codificación faltante:


# coding=utf-8
class A(object)
# №5
    def __init__(self):
        pass
