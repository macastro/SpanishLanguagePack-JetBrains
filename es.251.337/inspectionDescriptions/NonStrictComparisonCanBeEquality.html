El informe muestra condiciones de desigualdad que, según el análisis de flujo de datos, solo pueden satisfacerse con un único valor operativo. Estas condiciones pueden reemplazarse por condiciones de igualdad para hacer el código más claro.

Ejemplo:


  if (x >= 10) {
    ...
    if (x <= 10) { // can be replaced with 'x == 10'
    }
  }
