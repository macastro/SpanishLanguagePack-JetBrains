Informa sobre declaraciones import que hacen referencia al paquete java.lang.

Las clases de java.lang siempre se importan implícitamente, por lo que estas declaraciones import son redundantes y pueden causar confusión.

IntelliJ IDEA puede detectar y corregir automáticamente estas declaraciones mediante el comando Optimizar imports. Esta inspección es útil para informes sin conexión de bases de código que no se pretenden modificar.
