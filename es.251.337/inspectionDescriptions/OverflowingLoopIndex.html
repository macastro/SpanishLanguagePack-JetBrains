Informa sobre bucles que no pueden completarse sin desbordamiento de índice o bucles que no se ejecutan en absoluto. Esto suele ocurrir debido a errores en la operación de actualización.

Ejemplo:


  void foo(int s) {
    for (int i = s; i > 12; i++) { // i-- debería estar aquí
      System.out.println(i);
    }
  }
