Informa sobre llamadas a equals() en constantes enum.

Estas llamadas pueden reemplazarse por comparaciones de identidad (==), ya que dos constantes enum son iguales solo cuando tienen la misma identidad.

La llamada puede cambiarse a una comparación mediante una corrección rápida.

Ejemplo:


  boolean foo(MyEnum value) {
    return value.equals(MyEnum.FOO);
  }

Después de aplicar la corrección rápida:


  boolean foo(MyEnum value) {
    return value == MyEnum.FOO;
  }
