Informa sobre llamadas a funciones y métodos que no manejan resultados de tipo error.

La API de estas funciones indica que la ejecución puede fallar y devolver un error. Las llamadas que no manejan el resultado de error pueden indicar un uso incorrecto de la API.

Ejemplo:


os.Remove("non-existing") // Error ignorado

Después de aplicar la corrección rápida manejar error:

err := os.Remove("non-existing") // Error manejado
if err != nil {
  return err
}
