Reporta variables locales que se declaran con la palabra clave var pero nunca se modifican.

Kotlin recomienda usar la palabra clave val para declarar variables prácticamente inmutables, asegurando que sus valores nunca cambiarán.

Ejemplo:


  fun example() {
      var primeNumbers = listOf(1, 2, 3, 5, 7, 11, 13)
      var fibonacciNumbers = listOf(1, 1, 2, 3, 5, 8, 13)
      print("Same numbers: " + primeNumbers.intersect(fibonacciNumbers))
  }

La corrección rápida reemplazará la palabra clave var por val:


  fun example() {
      val primeNumbers = listOf(1, 2, 3, 5, 7, 11, 13)
      val fibonacciNumbers = listOf(1, 1, 2, 3, 5, 8, 13)
      print("Same numbers: " + primeNumbers.intersect(fibonacciNumbers))
  }
