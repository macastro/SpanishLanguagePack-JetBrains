Informa sobre métodos que tienen el mismo nombre que su clase. Aunque Java permite este tipo de nomenclatura, por convención está reservada para definir constructores. Su uso en métodos puede ser un error o una mala práctica.

Ejemplo:


  class MyClass {
    int val;
    // El nombre del método MyClass es igual al de su clase contenedora.
    // Posiblemente se añadió 'void' por error.
    void MyClass(int val) {
      this.val = val;
    }
  }

Cuando sea apropiado, el método puede convertirse en un constructor mediante una corrección rápida:


  class MyClass {
    int val;
    MyClass(int val) {
      this.val = val;
    }
  }

El método también puede renombrarse mediante otra corrección rápida.
