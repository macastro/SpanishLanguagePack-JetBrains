Reporta una lista de variables o constantes exportadas separadas por comas en la declaración.

Ejemplo:

const C1, C3, C2, C44, C9, C11, C6 = 1, 2, 3, 1, 3, 2, 1

Esta declaración dificulta conocer el valor de cada constante. Para hacer esta declaración más legible, puede aplicar la corrección rápida Extraer a su propia declaración. Después de aplicar la corrección rápida a cada constante, la declaración se verá así:

const (
 C3  = 2
 C2  = 3
 C44 = 1
 C9  = 3
 C11 = 2
 C6  = 1
 C1  = 1
)
