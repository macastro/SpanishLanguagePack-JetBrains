Reporta los usos de self como calificador donde no es requerido por un cierre de escape.

Sugiere eliminar self redundante.

Ejemplo: 


  struct URL {
      func loadAsync() {
          print(self.path)

          DispatchQueue.global().async {
              self.loadSync()
          }
      }
  }

Después de que la corrección rápida sea aplicada: 


  struct URL {
      func loadAsync() {
          print(path)

          DispatchQueue.global().async {
              loadSync()
          }
      }
  }
