Reporta el uso de código pseudo-funcional de Guava cuando la Java Stream API está disponible.

Aunque la Guava Iterable API proporciona funcionalidad similar a la Java Streams API, es ligeramente diferente y puede carecer de algunas características. En particular, las variantes de flujo especializadas en tipos primitivos como IntStream tienen mejor rendimiento que las variantes generales.

Ejemplo:


List<String> transformedIterable = Iterables.transform(someList, someTransformFunction);//advertencia: código de estilo pseudo-funcional

Después de aplicar la corrección rápida:

List<String> transformedIterable = someList.stream().map(someTransformFunction).collect(Collectors.toList());

Nota: La semántica del código puede cambiar; por ejemplo, el Iterable.transform de Guava produce un iterable con evaluación diferida, pero el reemplazo utiliza evaluación inmediata.
