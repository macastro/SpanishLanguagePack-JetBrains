Informa de atributos autowire en elementos <bean> y sugiere inyectar propiedades del bean explícitamente si es posible.

Ejemplo:


  public class MyComponent {
    public void setOtherBean(OtherBean bean){...}
  }


  <beans>
    <bean class="beans.OtherBean" id="bar"/>
    <bean autowire="byType" class="beans.MyComponent"/>
    <!-- "Dependencia autocableada innecesaria" -->
    <!-- sugiere "Hacer dependencia autocableada explícita" -->
  </beans>

Después de aplicar la corrección rápida:


  <beans>
    <bean class="beans.OtherBean" id="bar"/>
    <bean class="beans.MyComponent" id="foo">
      <property name="otherBean" ref="otherBean"/>
    </bean>
  </beans>
