Se generan dos tipos de advertencias:
  • Variable no definida: no se encuentra la definición de la variable en los archivos del proyecto, en las rutas include configuradas o en las variables predefinidas de PHP.
  • Variable posiblemente no definida: existe una o más rutas que llegan a la línea donde se usa la variable cuando ésta aún no está definida.

Use las siguientes opciones para personalizar la inspección:

  • Reportar variables posiblemente no definidas: desmarque la casilla para deshabilitar las advertencias de variable posiblemente no definida.
  • Habilitar inspección en espacio global: por defecto, la inspección está habilitada para el espacio global. Desmarque la casilla para reportar solo variables no definidas en el ámbito local de las funciones.

  • Buscar definiciones de variables fuera del archivo actual: si esta opción está marcada, el IDE solo reportará variables globales como no definidas cuando no haya sentencias include o require en el archivo actual y la definición de la variable no se encuentre en el proyecto. Tenga en cuenta que el IDE no verifica el contenido real de los archivos incluidos, solo verifica la existencia de sentencias include o require.

    De lo contrario, si no está seleccionada, el IDE solo buscará definiciones de variables en el archivo actual e ignorará las sentencias include o require si existen.
