Una interfaz en la lista implements de una clase o en la lista extends de una interfaz que ya está implementada por una superclase o extendida por una superinterfaz. Esta declaración es redundante y puede eliminarse de forma segura.

Ejemplo:


  class X implements One, Two {
  }
  interface One {}
  interface Two extends One {}

Después de aplicar la corrección rápida:


  class X implements Two {
  }
  interface One {}
  interface Two extends One {}

Utilice esta opción para no reportar Serializable o Externalizable en las listas extends o implements.
