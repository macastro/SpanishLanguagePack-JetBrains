Inspecciona las interfaces en la lista de implements de la clase, o las interfaces en la lista de extends que ya están implementadas por la superclase o extendidas por la superinterfaz. Esta declaración es innecesaria y puede eliminarse de forma segura.

Ejemplo:


  class X implements One, Two {
  }
  interface One {}
  interface Two extends One {}

Después de aplicar la corrección rápida:


  class X implements Two {
  }
  interface One {}
  interface Two extends One {}

Utiliza esta opción para no informar sobre Serializable o Externalizable en las listas de extends o implements.
