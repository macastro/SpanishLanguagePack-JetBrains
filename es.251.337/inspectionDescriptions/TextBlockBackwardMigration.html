Reporta bloques de texto que pueden ser reemplazados por literales de cadena regulares. Esta inspección ayuda con la degradación para mantener la compatibilidad con versiones anteriores de Java.

Ejemplo:


  Object obj = engine.eval("""
                           function hello() {
                             print('"Hello, world"');
                           }
                           hello();
                           """);

Después de aplicar la corrección rápida:


  Object obj = engine.eval("function hello() {\n" +
                           "  print('\"Hello, world\"');\n" +
                           "}\n" +
                           "\n" +
                           "hello();\n");
