Informa sobre expresiones que pueden reemplazarse mediante una llamada a java.util.Objects#equals.

Ejemplo:


  void f(Object a, Object b) {
    boolean result = a != null && a.equals(b);
  }

Después de aplicar la corrección rápida:


  void f(Object a, Object b) {
    boolean result = Objects.equals(a, b);
  }

La sustitución de expresiones tales como a != null && a.equals(b) por Objects.equals(a, b) cambia ligeramente la semántica. Utiliza la opción Resaltar expresiones similares a "a != null && a.equals(b)" para habilitar o deshabilitar este comportamiento.

Esta inspección solo informa si el nivel de lenguaje del proyecto o módulo es 7 o superior.
