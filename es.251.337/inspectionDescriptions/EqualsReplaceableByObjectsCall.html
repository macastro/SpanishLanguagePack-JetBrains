La expresión puede reemplazarse con una llamada a java.util.Objects#equals.

Ejemplo:


  void f(Object a, Object b) {
    boolean result = a != null && a.equals(b);
  }

Después de aplicar la corrección rápida:


  void f(Object a, Object b) {
    boolean result = Objects.equals(a, b);
  }

Reemplazar expresiones como a != null && a.equals(b) con Objects.equals(a, b) cambia ligeramente la semántica. Este comportamiento puede habilitarse o deshabilitarse usando la opción Resaltar expresiones similares a 'a != null && a.equals(b)'.
