Informa sobre campos no final y no private accedidos en un contexto sincronizado.

No se puede garantizar que los campos no private sean siempre accedidos de manera sincronizada, y estos accesos "parcialmente sincronizados" pueden llevar accidentalmente a estructuras de datos inconsistentes.

Ejemplo:


  class Bar {
    public String field1;
  }
  public Bar myBar;
  synchronized public void sample() {
   myBar.field1 = "bar";
  }
