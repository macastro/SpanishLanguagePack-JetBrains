Utiliza el procesador STR para informar sobre expresiones de plantillas de cadenas y proporciona una corrección rápida para volver a la concatenación de cadenas simple.

Ejemplo:


  String name = "Bob";
  String greeting = STR."Hello, \{name}. You are 29 years old.";

Después de aplicar la corrección rápida:


  String name = "Bob";
  String greeting = "Hello, " + name + ". You are 29 years old.";
