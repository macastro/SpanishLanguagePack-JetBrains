Informa sobre las clases permitidas que no extienden la clase base sealed.

Groovy no requiere que todas las clases permitidas estén disponibles en tiempo de compilación y se compilen junto con la clase base. El compilador no advierte a los usuarios cuando procesa subclases permitidas que no extienden, lo cual contradice la naturaleza de las clases sealed.

Ejemplo:


  class A permits B {} // informa sobre B
  class B {}
