Informa sobre métodos final en clases final.

Dado que las clases final no pueden ser heredadas, marcar los métodos como final puede ser redundante y confuso.

Ejemplo:

record Bar(int a, int b) {
  public final int sum() { 
     return a + b;
  }
}

Después de aplicar la corrección rápida:

record Bar(int a, int b) {
  public int sum() { 
     return a + b;
  }
}

Como se muestra en este ejemplo, la clase puede estar marcada como final de manera explícita o implícita.
