El informe puede extraer las partes comunes de las sentencias if de las ramas.

Estas partes comunes son independientes de la condición y dificultan la comprensión de las sentencias if.

Ejemplo:


  if (x > 12) {
    doSomethingBefore();
    doSomethingDifferent1();
    doSomethingAfter();
  } else {
    doSomethingBefore();
    doSomethingDifferent2();
    doSomethingAfter();
  }

Después de aplicar la corrección rápida:


  doSomethingBefore();
  if (x > 12) {
    doSomethingDifferent1();
  } else {
    doSomethingDifferent2();
  }
  doSomethingAfter();

Esta función se actualizó en la versión 2018.1
