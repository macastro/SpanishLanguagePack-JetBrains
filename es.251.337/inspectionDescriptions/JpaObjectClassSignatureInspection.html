Reporta errores en las definiciones de clases de objetos persistentes en los siguientes casos:
  • La clase no es una clase de nivel superior
  • La clase es final
  • La clase está anotada múltiples veces
  • La clase está definida como entidad persistente y como listener de entidad

Ejemplo:



  @Entity
  @EntityListeners(value = JavaEntity.class) // Error: las anotaciones @Entity y @EntityListener apuntan a la misma clase
  public final class JavaEntity { // Error: una entidad persistente no debe ser final
    ...
  }
