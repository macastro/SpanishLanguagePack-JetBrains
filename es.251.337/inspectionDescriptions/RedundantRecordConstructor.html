Informa sobre constructores redundantes declarados en registros Java.

Ejemplo 1:


  record Point(int x, int y) {
    public Point {} // se puede eliminar
  }
  record Point(int x, int y) {
    public Point(int x, int y) { // se puede eliminar
      this.x = x;
      this.y = y;
    }
  }

La corrección rápida elimina los constructores redundantes.

Ejemplo 2:


   // se puede convertir a constructor compacto
  record Range(int from, int to) {
    public Range(int from, int to) {
      if (from > to) throw new IllegalArgumentException();
      this.from = from;
      this.to = to;
    }
  }

La corrección rápida convierte este código en un constructor compacto.
