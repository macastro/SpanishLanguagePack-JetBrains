Informa sobre llamadas de registro SLF4J o Log4j2 en una clase, como logger.info("message: {}", key) con mensajes de registro similares. Estas llamadas pueden ser indistinguibles entre sí, lo que dificulta entender de dónde proviene un mensaje de registro específico.

Ejemplo (Java):


  private static void request1(String text) {
      log.info("Message: {}", text); //llamadas similares
      doSomething1();
  }
  private static void request2(int i) {
      log.info("Message: {}", i); //llamadas similares
      doSomething2();
  }
  • Utiliza la opción longitud mínima de secuencia similar para establecer la longitud mínima de la secuencia antes de que se informen las llamadas
  • Utiliza la opción no informar llamadas con nivel de registro 'error' para ignorar los mensajes con nivel de registro 'error' que contengan excepciones. Ocultar estas advertencias puede ser útil ya que aún es posible localizar el punto de llamada usando el seguimiento de pila
