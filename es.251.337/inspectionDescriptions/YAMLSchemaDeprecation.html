Reportar claves obsoletas en archivos YAML.

La verificación de obsolescencia solo se realiza cuando existe un esquema JSON asociado al archivo YAML correspondiente.

Tenga en cuenta que el mecanismo de obsolescencia aún no está definido en la especificación del esquema JSON, y esta verificación utiliza la extensión no estándar deprecationMessage.

Ejemplo de obsolescencia en el esquema:


  {
    "properties": {
      "SomeDeprecatedProperty": {
        "deprecationMessage": "Baz",
        "description": "Foo bar"
      }
    }
  }

Ejemplo con la advertencia correspondiente:


  SomeDeprecatedProperty: some value
