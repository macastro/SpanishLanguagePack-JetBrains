Informa sobre declaraciones try con múltiples recursos que pueden dividirse automáticamente en varias declaraciones try-with-resources independientes.

Esta transformación es útil para refactorizaciones posteriores (por ejemplo, para extraer declaraciones try anidadas en métodos separados).

Ejemplo:


  try (FileInputStream in = new FileInputStream("in.txt");
     FileOutputStream out = new FileOutputStream("out.txt")) {
     /*lectura y escritura*/
  }

Después de aplicar la corrección rápida:


  try (FileInputStream in = new FileInputStream("in.txt")) {
    try (FileOutputStream out = new FileOutputStream("out.txt")) {
      /*lectura y escritura*/
    }
  }
