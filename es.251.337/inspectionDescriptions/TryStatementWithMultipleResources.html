Reporta declaraciones try con múltiples recursos que pueden dividirse automáticamente en varias declaraciones try-with-resource.

Esta conversión puede ser útil para una refactorización posterior (por ejemplo, para extraer la declaración try anidada en un método separado).

Ejemplo:


  try (FileInputStream in = new FileInputStream("in.txt");
     FileOutputStream out = new FileOutputStream("out.txt")) {
     /*lectura y escritura*/
  }

Después de aplicar la corrección rápida:


  try (FileInputStream in = new FileInputStream("in.txt")) {
    try (FileOutputStream out = new FileOutputStream("out.txt")) {
      /*lectura y escritura*/
    }
  }
