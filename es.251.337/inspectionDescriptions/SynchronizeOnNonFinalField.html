Informa sobre expresiones de bloqueo synchronized que hacen referencia a campos no final. Es poco probable que tales declaraciones contengan una semántica útil, ya que diferentes hilos pueden adquirir diferentes bloqueos incluso cuando operan sobre el mismo objeto.

Ejemplo:


  private Object o;
  public void foo() {
    synchronized (o) // Sincronización en un campo no final
    { }
  }
