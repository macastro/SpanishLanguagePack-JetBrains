Informa sobre conflictos en declaraciones de contenedores parciales debido a un número incompatible de variables de tipo. Se produce un error cuando se declaran contenedores parciales con diferentes cantidades de variables de tipo.

Es posible corregir el número de variables de tipo en la declaración o navegar a las declaraciones en conflicto mediante una corrección rápida.

Ejemplo:

# La clase 'Example' tiene 1 variable de tipo, lo que entra en conflicto con otras declaraciones
class Example[A]
end
# La clase 'Example' tiene 2 variables de tipo, lo que entra en conflicto con otras declaraciones
class Example[Y, Z]
end

Después de aplicar la corrección rápida:

class Example[A, B]
end
class Example[Y, Z]
end
