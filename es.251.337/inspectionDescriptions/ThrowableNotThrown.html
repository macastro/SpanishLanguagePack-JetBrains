Informa cuando se crea una instancia de Throwable o sus subclases que nunca se llega a lanzar realmente. Además, esta inspección informa sobre las llamadas a métodos que devuelven una instancia de Throwable o sus subclases cuando el resultado no se lanza.

No se informan las llamadas a métodos anotados con @CanIgnoreReturnValue cuando se usa Error Prone o AssertJ.

Ejemplo:


    void check(String s) {
        if (s == null) {
            new NullPointerException("s");
        }
        // ...
    }
