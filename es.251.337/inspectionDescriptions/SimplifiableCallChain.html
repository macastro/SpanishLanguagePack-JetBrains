Informa sobre cadenas de doble llamada que pueden ser reemplazadas por una única llamada.

Esto ayuda a evitar la ejecución redundante de código.

La corrección rápida reemplaza la cadena de llamadas por una única llamada.

Ejemplo:


  fun main() {
      listOf(1, 2, 3).filter { it > 1 }.count()
  }

Después de aplicar la corrección rápida:


  fun main() {
      listOf(1, 2, 3).count { it > 1 }
  }
