La inspección reporta bucles for sin inicialización ni componentes de actualización y sugiere convertirlos a bucles while. Esto hace que el código sea más legible.

Ejemplo:


  for(; exitCondition(); ) {
    process();
  }

Después de aplicar la corrección rápida:


  while(exitCondition()) {
    process();
  }

La corrección rápida también está disponible para otros bucles for, por lo que puede reemplazar cualquier bucle for con un bucle while.

Si desea ignorar los bucles for con condiciones no significativas o inexistentes, utilice la opción Ignorar bucles for 'infinitos' sin condición.
