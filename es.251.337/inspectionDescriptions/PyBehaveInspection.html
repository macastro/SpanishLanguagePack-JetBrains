Reporta usos incorrectos del framework Python Behave BDD:

  • El archivo de definición de pasos no tiene from behave import *.
  • El directorio que contiene las definiciones de pasos no está nombrado como steps.

Si ocurre alguno de estos casos, el IDE resaltará los pasos correspondientes en el archivo .feature. Puede aplicar una corrección rápida para crear la definición del paso o renombrar el directorio que contiene este paso.

Para más información, consulte Crear definición de pasos.
