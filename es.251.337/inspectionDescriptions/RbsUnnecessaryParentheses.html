Informa paréntesis innecesarios dentro de los tipos de funciones. Las funciones sin parámetros no requieren paréntesis.

Hay una solución rápida disponible para eliminar los paréntesis innecesarios.

Ejemplo:

class Example
  # Paréntesis innecesarios en el tipo de función
  def call: () -> untyped
end

Después de la solución rápida:

class Example
  def call: -> untyped
end
