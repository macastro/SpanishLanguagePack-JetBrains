Informa sobre cualquier referencia a variables locales no Serializable que se encuentren fuera de lambdas Serializable, clases locales y clases anónimas.

Cuando se hace referencia a variables locales desde una clase anónima, su valor se almacena en un campo implícito de esa clase. Lo mismo ocurre con las clases locales y lambdas. Si el tipo de variable no es Serializable, la serialización fallará.

Ejemplo:


  interface A extends Serializable {
      abstract void foo();
  }
  class B {}
  class C {
      void foo() {
          B b = new B();
          A a = new A() {
              @Override
              public void foo() {
                  System.out.println(b); // advertencia
              }
          };
      }
  }
