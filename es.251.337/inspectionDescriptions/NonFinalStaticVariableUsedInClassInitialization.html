Informa sobre el uso de variables final static durante la inicialización de clases.

En estos casos, la semántica del código puede depender del orden de creación de las clases. Además, estas situaciones pueden llevar al uso de variables antes de su inicialización y generalmente resultan en errores difíciles y confusos.

Ejemplo:


  class Foo {
    public static int bar = 0;
    static {
      System.out.println(bar);
    }
  }
