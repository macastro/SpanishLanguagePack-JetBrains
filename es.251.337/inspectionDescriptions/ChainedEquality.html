Informa sobre comparaciones de igualdad encadenadas.

Este tipo de comparaciones pueden ser confusas: a == b == c significa (a == b) == c, pero probablemente a == b && a == c era la intención.

Ejemplo:


  boolean chainedEquality(boolean a, boolean b, boolean c) {
    return a == b == c;
  }

Puede usar paréntesis para hacer la comparación menos confusa:


  boolean chainedEquality(boolean a, boolean b, boolean c) {
    return (a == b) == c;
  }
