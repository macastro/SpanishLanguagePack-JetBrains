Informa clases locales.

Una clase local es una clase anidada nombrada declarada dentro de un bloque de código. Las clases locales son poco comunes y, por lo tanto, pueden resultar confusas. Además, algunos estándares de código desaconsejan el uso de clases locales.

Ejemplo:


class Ejemplo {
    void test() {
        class Local { // aquí
        }
        new Local();
    }
}

Después de aplicar la corrección rápida:


class Ejemplo {
    void test() {
        new Local();
    }

    private static class Local { // aquí
    }
}
