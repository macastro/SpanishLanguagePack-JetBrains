Reportar clase local.

Una clase local es una clase anidada con nombre que se declara dentro de un bloque de código. Las clases locales son poco comunes y pueden resultar confusas. Además, algunos estándares de código desaconsejan su uso.

Ejemplo:


  class Example {
    void test() {
      class Local { // aquí
      }
      new Local();
    }
  }

Después de aplicar la corrección rápida:


  class Example {
    void test() {
      new Local();
    }
    private static class Local { // aquí
    }
  }
