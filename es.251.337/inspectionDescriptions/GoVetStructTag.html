Informa sobre etiquetas de estructura que no cumplen con las convenciones de etiquetas de estructura de Go.

Según estas convenciones, la cadena de etiquetas es una concatenación de pares opcionales key:"value" separados por espacios. Cada clave es una cadena no vacía de caracteres no de control, excluyendo space (U+0020 ' '), quote (U+0022 '"') y colon (U+003A ':'). Cada valor está entrecomillado usando el carácter U+0022 '"' y la sintaxis de literales de cadena de Go.

Además, esta verificación comprueba si los campos con etiquetas están exportados.

Ejemplo de etiquetas válidas:

type Example struct {
  Field int `json:"field" xml:"demo"`
}
