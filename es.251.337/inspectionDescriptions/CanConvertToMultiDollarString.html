El informe muestra situaciones que pueden beneficiarse de la conversión de cadenas a versiones con prefijos de interpolación de múltiples símbolos de dólar.

Los prefijos de interpolación pueden hacer que las cadenas sean más fáciles de entender, ya que las cadenas con prefijo pueden usar símbolos de dólar literales sin necesidad de escape adicional.

Ejemplo:


val jsonSchema: String = """
{
  "${'$'}id": "https://example.com/product.schema.json",
}"""
La corrección rápida convertirá todos los caracteres de dólar escapados que se puedan reemplazar de forma segura en caracteres de dólar regulares: 

val jsonSchema: String = $$"""
{
  "$id": "https://example.com/product.schema.json",
}"""
