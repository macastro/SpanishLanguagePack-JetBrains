Cuando solo se puede usar una ruta String, se informa sobre la creación redundante de File en uno de los siguientes constructores: FileInputStream, FileOutputStream, FileReader, FileWriter, PrintStream, PrintWriter, Formatter.

Ejemplo:


InputStream is = new FileInputStream(new File("in.txt"));

Después de aplicar la corrección rápida:


InputStream is = new FileInputStream("in.txt");

Novedades de la versión 2020.3
