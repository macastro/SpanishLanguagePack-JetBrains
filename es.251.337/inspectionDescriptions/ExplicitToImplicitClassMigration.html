Informar clases ordinarias que se pueden convertir en clases declaradas implícitamente

Ejemplo:


  public class Sample {
      public static void main(String[] args) {
          System.out.println("Hello, world!");
      }
  }

Después de aplicar la corrección rápida:


  public static void main(String[] args) {
      System.out.println("Hello, world!");
  }
