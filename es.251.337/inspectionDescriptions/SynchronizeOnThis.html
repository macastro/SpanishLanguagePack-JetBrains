Informa sobre la sincronización de expresiones this o class. El informe incluye bloques synchronized y llamadas a wait(), notify() o notifyAll().

Sincronizar expresiones this o class puede no ser una buena idea por varias razones:

  1. Hace que la sincronización sea parte de la interfaz externa de la clase, lo que dificulta cambiar a otros mecanismos de bloqueo en el futuro,
  2. Es difícil rastrear quién exactamente está bloqueando un objeto dado,
  3. Permite ataques de denegación de servicio, ya sea intencionales o accidentales durante la herencia.

Como alternativa, considere sincronizar sobre un objeto de bloqueo private final, sobre el cual tiene control total de acceso.

Ejemplo:


  public void print() {
    synchronized(this) { // Advertencia: el bloqueo sobre 'this' puede tener efectos secundarios impredecibles
      System.out.println("synchronized");
    }
  }
