Reporta el uso de API de IntelliJ Platform que fue introducida en versiones anteriores a la especificada en @since-build del elemento en plugin.xml.

El uso de esta API puede hacer que el plugin sea incompatible con versiones anteriores del IDE.

Para evitar posibles problemas al ejecutar el plugin en versiones anteriores del IDE, incremente el valor de since-build según corresponda o elimine el uso de esta API.

Consulte Rangos de números de compilación en la documentación del SDK de plugins de la plataforma IntelliJ para más detalles.

Configuración de la inspección:

Si los atributos @since/until-build del elemento <idea-version> no están especificados en plugin.xml, establezca explícitamente Since/Until.
