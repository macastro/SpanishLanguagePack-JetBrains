Informar de columnas, alias y otros elementos de subconsulta que no están referenciados en la expresión de consulta externa.

Ejemplo (PostgreSQL):

CREATE TABLE for_subquery(id INT);
SELECT a, q FROM (SELECT 1 AS a, 10 AS b, 2 + 3 AS q, id
      FROM for_subquery) x;

Hacemos referencia a los alias a y q de una subconsulta. Pero el alias b y la columna id no están referenciados en la sentencia SELECT exterior. Por lo tanto, b y id aparecen atenuados.
