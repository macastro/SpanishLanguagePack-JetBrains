Reporta el uso de API de funciones en vista previa, es decir, el uso de módulos, paquetes, clases, interfaces, métodos, constructores o constantes enum en los espacios de nombres java.* o javax.* anotados con @PreviewFeature.

Una función en vista previa es una nueva característica del lenguaje Java, la Máquina Virtual Java o la API de Java SE que está completamente especificada e implementada, pero no es permanente. El concepto de funciones en vista previa se define en JEP 12.

Si el código depende de una API en vista previa, puede dejar de compilarse en futuras versiones del JDK si la función se modifica o elimina.

Esta inspección solo informa cuando el nivel de lenguaje del proyecto o módulo está establecido en Preview.
