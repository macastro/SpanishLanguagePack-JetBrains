Informa de usos de API de funciones de vista previa, es decir, un módulo, paquete, clase, interfaz, método, constructor, campo o constante de enumeración en el espacio de nombres java.* o javax.* anotado con @PreviewFeature.

Una función de vista previa es una nueva función del lenguaje Java, la máquina virtual Java o la API Java SE que está completamente especificada, implementada por completo y aún es impermanente. La noción de una función de vista previa se define en JEP 12.

Si alguna parte del código depende de una API de vista previa, puede dejar de compilarse en versiones futuras de JDK si la función se cambia o elimina.

La inspección solo informa si el nivel de idioma del proyecto o módulo es Vista previa.
