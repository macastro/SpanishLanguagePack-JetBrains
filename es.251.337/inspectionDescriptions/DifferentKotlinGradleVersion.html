Se informa que el plugin IDE actual no es compatible correctamente con la versión del plugin Gradle.

Esto puede resultar en inconsistencias entre el IDE y la compilación de Gradle en términos de informes de errores o comportamiento del código.

Ejemplo:


  dependencies {
    classpath "org.jetbrains.kotlin:kotlin-gradle-plugin:0.0.1"
  }

Para corregir este problema, modifique la versión del plugin Kotlin Gradle para que coincida con la versión de Kotlin incluida en el plugin IDE.
