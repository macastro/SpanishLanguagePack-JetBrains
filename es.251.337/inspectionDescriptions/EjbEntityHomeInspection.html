Informa sobre definiciones incorrectas de los siguientes elementos relacionados con EJB:
  • Clases que no implementan las interfaces esperadas (EJBLocalObject, EJBObject, etc.)
  • Modificadores de clase EJB incorrectos
  • Clases que no definen un constructor público sin parámetros
  • Uso de clases en lugar de interfaces o interfaces en lugar de clases

Ejemplo:


public final class SessionEJB implements javax.ejb.SessionBean {
  private SessionEJB() {} // Error: la clase debe definir un constructor público sin parámetros
}
