Informa sobre las llamadas a métodos como Assert.assertEquals(), Assert.assertTrue() y otros que pueden migrarse al estilo declarativo Hamcrest usando Assert.assertThat().

Por ejemplo:


  public class SubstantialTest {
    @Test
    public void testContents(Collection<String> c, String s) {
      Assert.assertTrue(c.contains(s));
      Assert.assertEquals(c, s);
      Assert.assertNotNull(c);
      Assert.assertNull(c);
      Assert.assertFalse(c.contains(s));
    }
  }

Se proporciona una corrección rápida para realizar la migración:


  public class SubstantialTest {
    @Test
    public void testContents(Collection<String> c, String s) {
      assertThat(c, hasItem(o));
      assertThat(o, is(c));
      assertThat(c, notNullValue());
      assertThat(c, nullValue());
      assertThat(c, not(hasItem(o)));
    }
  }

Esta inspección requiere que la biblioteca Hamcrest esté disponible en el classpath.

La opción importar estáticamente los métodos matcher permite especificar si desea que la corrección rápida importe estáticamente los métodos matcher de Hamcrest.
