Informa sobre declaraciones if que pueden ser reemplazadas por declaraciones switch.

El resultado del reemplazo suele ser más conciso y claro.

Ejemplo:


  void test(String str) {
    if (str.equals("1")) {
      System.out.println(1);
    } else if (str.equals("2")) {
      System.out.println(2);
    } else if (str.equals("3")) {
      System.out.println(3);
    } else {
      System.out.println(4);
    }
  }

Después de aplicar la corrección rápida:


  void test(String str) {
    switch (str) {
      case "1" -> System.out.println(1);
      case "2" -> System.out.println(2);
      case "3" -> System.out.println(3);
      default -> System.out.println(4);
    }
  }

Esta inspección solo informa cuando el nivel de lenguaje del proyecto o módulo es 7 o superior.

El campo Número mínimo de ramas de condición 'if' permite especificar el número mínimo de ramas de condición if que debe tener una declaración if para ser reportada. Ten en cuenta que la rama else final (sin if) no se cuenta.

Usa la opción Habilitar sugerencias de switch para números para activar las sugerencias de declaraciones switch para números primitivos y empaquetados, y caracteres.

Usa la opción Habilitar sugerencias de switch para enums para activar las sugerencias de declaraciones switch para constantes enum.

Usa la opción Habilitar sugerencias solo para expresiones null-safe para sugerir solo declaraciones switch que no puedan introducir una NullPointerException.
