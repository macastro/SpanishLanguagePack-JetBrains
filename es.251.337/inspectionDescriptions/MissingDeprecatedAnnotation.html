Informa sobre declaraciones de módulos, clases, campos o métodos que tienen la etiqueta Javadoc @deprecated pero carecen de la anotación @java.lang.Deprecated.

Ejemplo:


  /**
   * @deprecated use {@code example()} instead
   */
  void sample(){ }

Después de aplicar la corrección rápida:


  /**
   * @deprecated use {@code example()} instead
   */
  @Deprecated
  void sample(){ }

Utilice la casilla de verificación a continuación para informar sobre miembros que usan la anotación @Deprecated pero no tienen explicación en la etiqueta Javadoc @deprecated.
