Informa sobre declaraciones de tipo redundantes en literales compuestos.

Ejemplo:

nums := [][]int{[]int{1}, []int{2}}

Esto es un slice de slices de tipo int. En este caso, puede usar una definición más corta. Puede corregir este código manualmente o usar la corrección rápida Eliminar tipo redundante. Después de aplicar la corrección rápida, el código se verá así:

nums := [][]int{{1},{2}}

Para obtener más información sobre literales compuestos, consulte la Especificación del Lenguaje Go: Literales Compuestos en go.dev.
