Informa sobre modificaciones de variables con asignaciones simples (por ejemplo, y = y + x) que pueden reemplazarse por operadores de asignación compuesta.

Esta corrección rápida reemplazará la asignación con un operador de asignación compuesta.

Ejemplo:


  fun foo() {
      val list = mutableListOf(1, 2, 3)
      list = list + 4
  }

Después de aplicar la corrección rápida:


  fun foo() {
      val list = mutableListOf(1, 2, 3)
      list += 4
  }
