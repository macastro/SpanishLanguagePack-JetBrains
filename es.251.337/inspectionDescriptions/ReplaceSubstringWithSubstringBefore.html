Informa de llamadas como s.substring(0, s.indexOf(x)) que se pueden reemplazar con s.substringBefore(x).

Usar substringBefore() simplifica el código.

La corrección rápida reemplaza la llamada substring con substringBefore.

Ejemplo:


  fun foo(s: String) {
      s.substring(0, s.indexOf('x'))
  }

Después de aplicar la corrección rápida:


  fun foo(s: String) {
      s.substringBefore('x')
  }
