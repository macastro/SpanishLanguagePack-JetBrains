Se detectó un recurso/fuente de datos que no se puede resolver según los proveedores conocidos. El proveedor debe especificarse en la sección required_providers o debe provenir del espacio de nombres hashicorp.

Por ejemplo, el siguiente código generará un error: 


terraform {
  required_version = "1.1.3"
}
resource "abbey_demo" "demo" {
  email      = "user@mail.company"
  permission = "user"
}

Terraform intentará importar el proveedor hashicorp/abbey para el recurso abbey_demo.

Para corregir este problema, necesitamos especificar las coordenadas exactas del proveedor: 


terraform {
  required_version = "1.1.3"
  required_providers {
    abbey = {
      source  = "abbeylabs/abbey"
      version = "0.2.9"
    }
  }
}
resource "abbey_demo" "demo" {
  email      = "user@mail.company"
  permission = "user"
}

Para más información, consulte la documentación de Terraform.
