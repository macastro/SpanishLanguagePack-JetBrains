Informa sobre cualquier declaración break innecesaria.

Una declaración break es innecesaria si no hay más declaraciones que se ejecutarían después de eliminarla.

Ejemplo:


  switch (e) {
    case A -> {
        System.out.println("A");
        break; // Se informa 'break' como declaración innecesaria
    }
    default -> {
        System.out.println("Default");
        break; // Se informa 'break' como declaración innecesaria
    }
  }
