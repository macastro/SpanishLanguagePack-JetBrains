El informe sugiere reemplazar llamadas como s.substring(0, x) por s.take(x).

Usar take() hace que el código sea más simple.

Esta corrección rápida reemplazará la llamada a substring por take().

Ejemplo:


  fun foo(s: String) {
      s.substring(0, 10)
  }

Después de aplicar la corrección rápida:


  fun foo(s: String) {
      s.take(10)
  }
