Informa sobre referencias a métodos de proveedores de datos que no existen o no son accesibles.

Ejemplo:


public class InstanceDataProviderFromForeignClass {
  // El método data() no existe en la clase A
  @Test(dataProvider = "data", dataProviderClass = A.class)
  public void test() {
  }
}
class A { }

Después de aplicar la corrección rápida:


//El método data() requerido se crea en la clase A
class A {
  @DataProvider
  public Object[][] data() {
    return new Object[][]{};
  }
}
