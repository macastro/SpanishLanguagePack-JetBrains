El informe sugiere reemplazar la llamada s.substring(0, s.length - x) por s.dropLast(x).

El código puede simplificarse usando la función correspondiente.

Esta corrección rápida reemplazará la llamada substring por dropLast.

  fun foo(s: String) {
      s.substring(0, s.length - 5)
  }
