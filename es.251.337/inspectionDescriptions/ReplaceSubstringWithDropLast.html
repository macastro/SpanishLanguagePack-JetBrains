Reporta llamadas como s.substring(0, s.length - x) que se pueden reemplazar con s.dropLast(x).

Usar funciones correspondientes simplifica tu código.

El arreglo rápido reemplaza la llamada substring con dropLast.

Ejemplo:


  fun foo(s: String) {
      s.substring(0, s.length - 5)
  }

Después de aplicar el arreglo rápido:


  fun foo(s: String) {
      s.dropLast(5)
  }
