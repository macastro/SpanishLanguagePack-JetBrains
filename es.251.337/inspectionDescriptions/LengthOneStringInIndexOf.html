Informa sobre cadenas de un solo carácter utilizadas como argumentos en las llamadas a String.indexOf() y String.lastIndexOf().

Se sugiere reemplazar estos literales de cadena por literales de carácter equivalentes mediante una corrección rápida para obtener algunas mejoras de rendimiento.

Ejemplo:


  return s.indexOf("x");

Después de aplicar la corrección rápida:


  return s.indexOf('x');
