Informa sobre recursos Java ME javax.microedition.rms.RecordStore que no se abren en un bloque try y no se cierran en el correspondiente bloque finally.

Este tipo de recursos pueden tener fugas involuntarias si se lanza una excepción antes de que se cierren.

Esta inspección es aplicable a Java ME y otros entornos con recursos muy limitados. La aplicación indiscriminada de esta inspección puede tener un impacto negativo en la claridad y el diseño del código.

Ejemplo:


  void foo1() throws RecordStoreException {
      RecordStore rs = RecordStore.openRecordStore("bar", true); // advertencia
  }
  void foo2() throws RecordStoreException {
      RecordStore rs = RecordStore.openRecordStore("bar", true); // sin advertencia
      try {
          /* ... */
      } finally {
          rs.closeRecordStore();
      }
  }
