Informa declaraciones de propiedades que se pueden unir con la siguiente asignación.

Ejemplo:


  val x: String
  x = System.getProperty("")

El arreglo rápido une la declaración con la asignación:


  val x = System.getProperty("")

Ajustar la inspección:

Puedes desactivar la opción Reportar con inicialización compleja de propiedades de miembros para omitir las propiedades con inicialización compleja. Esto cubre dos casos:

  1. El inicializador de propiedades es complejo (es una expresión compuesta/de control de flujo o de varias líneas)
  2. La propiedad se inicializa primero y luego se usa inmediatamente en el código subsecuente (por ejemplo, para llamar a métodos de inicialización adicionales)
