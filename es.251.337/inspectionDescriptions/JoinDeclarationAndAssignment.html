El informe puede unir declaraciones de propiedades con las siguientes asignaciones.

Ejemplo:


  val x: String
  x = System.getProperty("")

La corrección rápida unirá la declaración con la asignación:


  val x = System.getProperty("")

Configuración de la inspección:

Puede deshabilitar la opción Informar sobre propiedades de miembro con inicialización compleja para omitir propiedades con inicialización compleja. Esto incluye dos casos:

  1. El inicializador de la propiedad es complejo (es una expresión multilínea o compuesta/de control de flujo)
  2. La propiedad se inicializa primero y luego se usa inmediatamente en el código subsiguiente (por ejemplo, para llamar a otros métodos de inicialización)
