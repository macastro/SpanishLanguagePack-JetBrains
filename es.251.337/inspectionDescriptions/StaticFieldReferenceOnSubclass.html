Informa sobre accesos a campos static que están calificados por una subclase de la clase declarante en lugar de por la propia clase declarante.

Java permite tal calificación, pero este tipo de acceso puede indicar una confusión sutil entre herencia y sobrescritura.

Ejemplo:


  class Parent {
    static int foo = 0;
  }
  class Child extends Parent { }
  void bar() {
    System.out.println(Child.foo);
  }

Después de aplicar la corrección rápida, el resultado se muestra así:


  class Parent {
    static int foo = 0;
  }
  class Child extends Parent { }
  void bar() {
    System.out.println(Parent.foo);
  }
