Informa sobre casos donde se puede usar Math.max() o Math.min() en lugar de calcular manualmente el valor mínimo o máximo de dos números.

Ejemplo:


  public int min(int a, int b) {
    return b < a ? b : a;
  }

Después de aplicar la corrección rápida:


  public int min(int a, int b) {
    return Math.min(a, b);
  }

Con la opción Deshabilitar para float y double, esta inspección puede deshabilitarse para los tipos double y float. Esto es útil ya que la corrección rápida puede alterar ligeramente la semántica de los tipos float/ double cuando se maneja NaN. Sin embargo, si no se considera NaN, esta corrección rápida en realidad corrige errores sutiles en la mayoría de los casos.
