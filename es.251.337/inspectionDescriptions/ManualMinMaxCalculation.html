Reporta casos en los que se puede calcular el mínimo o el máximo de dos números usando una llamada Math.max() o Math.min(), en lugar de hacerlo manualmente.

Ejemplo:


  public int min(int a, int b) {
    return b < a ? b : a;
  }

Después de aplicar la corrección rápida:


  public int min(int a, int b) {
    return Math.min(a, b);
  }

Usa la opción Deshabilitar para float y double para deshabilitar esta inspección para tipos double y float. Esto es útil porque la corrección rápida puede cambiar ligeramente la semántica para los tipos float y double al manejar el NaN. Sin embargo, en la mayoría de los casos, esto realmente solucionará un error sutil en el que no se tiene en cuenta NaN.
