Se informa sobre los constructores de clases anotadas con @Singleton a menos que se declaren como no estrictos.

Ejemplo:


  @Singleton
  class Foo{
      Foo(){
      }
  }

Hay dos posibles correcciones rápidas: eliminar el constructor o declarar @Singleton como no estricto.

Después de aplicar la corrección rápida:


  @Singleton
  class Foo{
  }

o:


  @Singleton(strict = false)
  class Foo{
      Foo(){
      }
  }
