Informa sobre llamadas redundantes a requireNotNull o checkNotNull en expresiones que no pueden ser null.

Ejemplo:


  fun foo(i: Int) {
      requireNotNull(i) //Esta 'i' nunca puede ser null, por lo que la llamada a 'requireNotNull' es redundante.
      ...
  }

Después de aplicar la corrección rápida:


  fun foo(i: Int) {
      ...
  }
