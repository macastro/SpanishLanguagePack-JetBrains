Reporta las declaraciones throw dentro de bloques finally. Aunque a veces es intencional, tales declaraciones throw pueden ocultar excepciones lanzadas desde bloques try y catch, complicando el proceso de depuración.

Ejemplo:


  class Fascinating {
    void doTask(Task t) {
      try {
        t.activity();
      } finally {
        if (!t.finish()) {
          throw new IllegalStateException();
        }
      }
    }
  }

Utiliza la opción Advertir en cada lugar donde se pueda lanzar una excepción declarada para recibir advertencias también cuando se llaman métodos que declaran excepciones.
