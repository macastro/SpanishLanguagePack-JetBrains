El informe muestra una expresión when que puede simplificarse mediante la introducción de un argumento principal.

Ejemplo:


  fun test(obj: Any): String {
      return when {
          obj is String -> "string"
          obj is Int -> "int"
          else -> "unknown"
      }
  }

La corrección rápida introducirá un argumento principal:


  fun test(obj: Any): String {
      return when (obj) {
          is String -> "string"
          is Int -> "int"
          else -> "unknown"
      }
  }
