Informa sobre el uso de los decoradores @classmethod o @staticmethod en métodos externos a la clase.

Ejemplo:


class State(object):
    @classmethod
    def my_state(cls, name):
        cls.name = name
@classmethod
def change_state(self):
    pass

El método change_state no debería usar el decorador @classmethod, o debería moverse dentro de la declaración de la clase state.

Si se aplica la operación eliminar decorador, el código se convierte en:


class State(object):
    @classmethod
    def my_state(cls, name):
        cls.name = name
def change_state(self):
    pass
