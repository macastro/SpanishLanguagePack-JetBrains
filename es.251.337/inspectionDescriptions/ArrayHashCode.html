Informa sobre el cálculo incorrecto del código hash de arrays.

Para calcular correctamente el código hash de los arrays, realice lo siguiente:

  • Use Arrays.hashcode() para arrays unidimensionales
  • Use Arrays.deepHashcode() para arrays multidimensionales
Estos métodos también deben usarse con Objects.hash() cuando la secuencia de entrada contiene arrays, por ejemplo: Objects.hash(string, Arrays.hashcode(array))
