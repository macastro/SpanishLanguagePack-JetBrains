Informa cuando una clase tiene un método equals() con un tipo de parámetro diferente a Any?, sin tener otro método equals() que use Any? como tipo de parámetro.

Ejemplo:


  class Foo {
      fun equals(other: Foo?): Boolean {
          return true
      }
  }

Para corregir este problema, cree un método equals() que tome un parámetro de tipo Any?.
