Informe sobre el uso de asignaciones como condiciones en sentencias if, while, for o do o en expresiones condicionales.

Aunque a veces es intencional, este uso puede resultar confuso y podría indicar un error tipográfico (por ejemplo, usar = en lugar de ==).

Esta corrección rápida reemplazará = por ==.

Ejemplo:


  void update(String str, boolean empty) {
    // Advertencia: 'empty' está siendo reasignado,
    // no comparado con str.isEmpty()
    if (empty = str.isEmpty()) {
      ...
    }
  }

Después de aplicar la corrección rápida:


  void update(String str, boolean empty) {
    if (empty == str.isEmpty()) {
      ...
    }
  }
