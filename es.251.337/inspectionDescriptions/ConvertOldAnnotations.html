Información de anotaciones TestNG org.testng.annotations.Configuration

Es mejor configurar una suite de pruebas con anotaciones modernas @BeforeXXX/@AfterXXX.

Ejemplo:


  public class Test {
    @Configuration(beforeSuite = true, afterTest = true)
    public void afterBefore(){
    }
  }

Después de aplicar la corrección rápida:


  public class Test {
    @AfterTest()
    @BeforeSuite()
    public void afterBefore(){
    }
  }
