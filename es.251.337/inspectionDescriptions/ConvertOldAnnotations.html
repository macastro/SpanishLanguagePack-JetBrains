Informa sobre la anotación TestNG org.testng.annotations.Configuration.

Es recomendable utilizar las anotaciones modernas @BeforeXXX/@AfterXXX para configurar el conjunto de pruebas.

Ejemplo:


  public class Test {
    @Configuration(beforeSuite = true, afterTest = true)
    public void afterBefore(){
    }
  }

Después de aplicar la corrección rápida:


  public class Test {
    @AfterTest()
    @BeforeSuite()
    public void afterBefore(){
    }
  }
