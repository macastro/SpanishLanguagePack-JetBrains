Informa sobre bloques synchronized que se bloquean en objetos inicializados con literales.

Los literales de cadena se almacenan en el pool de strings, y los literales de Character, Boolean y Number pueden asignarse desde el caché. Por lo tanto, es posible que otras partes del sistema (que utilizan objetos inicializados con el mismo literal) contengan referencias al mismo objeto. Esto puede provocar situaciones inesperadas de interbloqueo si se considera que el objeto bloqueado es privado.

Ejemplo:


  class Main {
    final String mutex = "Mutex";
    void method() {
      synchronized (mutex) {
      }
    }
  }

La opción advertir sobre todos los literales posibles informa sobre cualquier sincronización de objetos String, Character, Boolean y Number.
