Informa sobre métodos no synchronized que sobrescriben métodos synchronized.

Si un método de la superclase está declarado como synchronized, el método sobrescrito no se sincronizará automáticamente. Esto puede provocar condiciones de carrera inesperadas cuando se utiliza la subclase.

Ejemplo:


  class Super {
    synchronized void process() {}
  }
  class Sub extends Super {
    // Método no synchronized 'process()' sobrescribe un método synchronized
    void process() {}
  }
