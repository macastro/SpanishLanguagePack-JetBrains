Informa sobre miembros protected en clases final.

Dado que las clases final no pueden ser heredadas, marcar métodos como protected puede resultar confuso. Es mejor declarar estos miembros como private o con visibilidad de paquete.

Ejemplo:

record Bar(int a, int b) {
  protected int sum() { 
     return a + b;
  }
}

Después de aplicar la corrección rápida:

record Bar(int a, int b) {
  int sum() { 
     return a + b;
  }
}

Como se muestra en este ejemplo, una clase puede marcarse como final de manera explícita o implícita.
