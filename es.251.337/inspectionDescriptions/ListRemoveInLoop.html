Informa sobre llamadas a List.remove(index) en bucles que pueden ser reemplazadas por List.subList().clear().

Esta sustitución es más eficiente para la mayoría de implementaciones de List cuando se eliminan múltiples elementos.

Ejemplo:


  void removeRange(List<String> list, int from, int to) {
    for (int i = from; i < to; i++) {
        list.remove(from);
    }
  }

Después de aplicar la corrección rápida:


  void removeRange(List<String> list, int from, int to) {
    if (to > from) {
      list.subList(from, to).clear();
    }
  }

La corrección rápida añade automáticamente una comprobación de rango para prevenir posibles IndexOutOfBoundsException cuando el valor mínimo es mayor que el máximo. Si este caso no es posible en el código, esta comprobación puede eliminarse.
