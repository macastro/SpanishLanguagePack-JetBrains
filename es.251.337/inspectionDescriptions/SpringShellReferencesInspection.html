Informa sobre referencias no válidas en la anotación Spring Shell org.springframework.shell.standard.ShellMethodAvailability.

Ejemplo:


@ShellComponent
public class MyCommands {
    @ShellMethod("sum")
    @ShellMethodAvailability("sumAvailability") // el valor de la anotación se resaltará como un símbolo no resuelto
    public int sum(int a, int b) {
        return a + b;
    }
    @ShellMethodAvailability("sumMethod") // el valor de la anotación se resaltará como un símbolo no resuelto
    public Availability isSumAvailable() {
        return Availability.available();
    }
}
