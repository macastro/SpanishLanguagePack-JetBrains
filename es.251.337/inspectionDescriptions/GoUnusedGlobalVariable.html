Informa sobre variables globales que están definidas pero nunca se utilizan en el código.

Si tiene variables sin usar, el código no compilará. Para obtener más información sobre variables e importaciones sin usar, consulte Unused imports and variables en go.dev. 

func main() {
 a := 422
}

El código del ejemplo no compilará. Por lo tanto, se resaltará como un error. Para estos casos, puede aplicar dos correcciones rápidas: eliminar variable y renombrar a _. La primera corrección elimina la variable, la segunda convierte la variable en un identificador en blanco.

Después de aplicar la corrección rápida renombrar a _:

func main() {
 _ := 422
}
