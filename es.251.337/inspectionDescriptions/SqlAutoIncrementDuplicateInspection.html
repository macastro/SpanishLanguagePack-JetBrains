Reporta tablas que contienen dos columnas con un incremento automático. En los dialectos de MySQL, Microsoft SQL Server y Db2, una tabla sólo puede tener un campo con una opción de incremento automático, y este campo debe ser una clave.

Ejemplo (MySQL):

CREATE TABLE my_table
(
    id INT AUTO_INCREMENT,
    c2 INT AUTO_INCREMENT,
);

La restricción de AUTO_INCREMENT para c2 se resaltará porque c1 ya tiene esta restricción. Para corregir la advertencia, puede establecer `id` como clave primaria y eliminar AUTO_INCREMENT para `c2`.

CREATE TABLE my_table
(
    id INT AUTO_INCREMENT PRIMARY KEY,
    c2 INT,
);
