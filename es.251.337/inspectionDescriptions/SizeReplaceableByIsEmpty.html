Reporta comparaciones de .size() o .length() con un literal de 0 que pueden ser remplazadas con una llamada a .isEmpty().

Ejemplo:


  boolean emptyList = list.size() == 0;

Después de aplicar la corrección rápida:


  boolean emptyList = list.isEmpty();

Usa la tabla Clases ignoradas para añadir clases para las cuales no deben ser reemplazadas las comparaciones de .size() o .length().

Usa la opción Ignorar expresiones que pueden ser reemplazadas con !isEmpty() para ignorar cualquier expresión que pueda ser reemplazada con !isEmpty().
