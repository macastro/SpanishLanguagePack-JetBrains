Informa sobre el uso innecesario del modificador non-sealed en métodos, campos o variables.

Este modificador solo es válido para clases, interfaces y traits.

Ejemplo:


  non-sealed boolean foo() {} // Modificador innecesario
  non-sealed Object bar // Modificador innecesario
  // El modificador es obligatorio, por lo que no está resaltado
  non-sealed class A {}
