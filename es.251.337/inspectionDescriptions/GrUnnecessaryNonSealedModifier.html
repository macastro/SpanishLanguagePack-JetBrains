Informa sobre modificadores non-sealed no necesarios usados en métodos, campos o variables.

Este modificador sólo tiene efecto en clases, interfaces y rasgos.

Ejemplo:


  non-sealed boolean foo() {} // el modificador es innecesario
  non-sealed Object bar // el modificador es innecesario

  // el modificador es necesario y por lo tanto no se resalta
  non-sealed class A {}
