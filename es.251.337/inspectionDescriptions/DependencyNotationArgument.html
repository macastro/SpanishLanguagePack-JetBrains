Detecta notaciones incorrectas de dependencias.

Gradle admite los siguientes tipos/formatos:
  • Instancias de Dependency;
  • Valores de tipo String o CharSequence, por ejemplo 'org.gradle:gradle-core:1.0';
  • Mapeos, por ejemplo [group: 'org.gradle', name: 'gradle-core', version: '1.0'];
  • FileCollection, por ejemplo files('some.jar', 'someOther.jar');
  • Proyectos, por ejemplo project(':some:project:path');
  • ClassPathNotation, por ejemplo gradleApi();
  • Listas de notaciones de dependencias, por ejemplo ['org.gradle:gradle-core:1.0'];
  • (Gradle 7.0+) Accesores de catálogo de versiones, por ejemplo libs.groovy.core.
Consulte la documentación de Gradle

Ejemplo:


dependencies {
  implementation(1) // reports '1'
}
