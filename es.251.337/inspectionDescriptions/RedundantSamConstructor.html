El uso del constructor SAM (Single Abstract Method) puede ser reemplazado por una expresión lambda.

Ejemplo:


  fun main() {
      foo(Runnable { println("Hi!") })
  }
  fun foo(other: Runnable) {}

Después de aplicar la corrección rápida:


  fun main() {
      foo( { println("Hi!") })
  }
  fun foo(other: Runnable) {}
