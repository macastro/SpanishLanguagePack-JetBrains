Informa sobre asignaciones o modificaciones de campos static dentro de métodos de instancia.

Aunque es legal, estas asignaciones son difíciles de realizar de manera segura y generalmente son el resultado de haber marcado un campo como static de forma no intencionada.

Ejemplo:


  class Counter {
    private static int count = 0;
    void increment() {
      // Advertencia: desde un método de instancia
      // actualizando un campo static
      count++;
    }
  }
