Informa sobre casos donde se llama a equals() para comparar un String con una cadena vacía. En estos casos, es mejor usar .isEmpty() ya que expresa de manera más clara lo que se está verificando.

Ejemplo:


  void checkString(String s){
    if ("".equals(s)) throw new IllegalArgumentException();
  }

Después de aplicar la corrección rápida:


  void checkString(String s){
    if (s != null && s.isEmpty()) throw new IllegalArgumentException();
  }

"".equals(str) devuelve false cuando str es null. Por seguridad, cuando el argumento de equals() puede ser null, la corrección rápida de esta inspección inserta una comprobación explícita de null. Use esta opción para que la inspección ignore tales casos.
