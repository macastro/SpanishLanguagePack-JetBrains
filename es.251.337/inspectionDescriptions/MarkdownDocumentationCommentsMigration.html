Informa sobre comentarios Javadoc que pueden convertirse a comentarios de documentación Markdown.

Ejemplo:


  /**
   * For blubbering of the florz.
   *
   * {@link com.deeps.Querp querping} clase.
   */
  class FlorzBlubber {
  }

Tras aplicar la corrección rápida:


  /// For blubbering of the florz.
  /// [querping][com.deeps.Querp] clase.
  class FlorzBlubber {
  }
