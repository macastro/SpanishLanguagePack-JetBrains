Reporta el uso de interfaces vacías como tipos o restricciones de tipos.

Una interfaz vacía representa el conjunto de todos los tipos. Go 1.18 introduce el alias más explícito any para interface{}, que es equivalente a interface{} en todos los aspectos.

La inspección se activa cuando se usa una interfaz vacía como tipo o restricción de tipo. Los alias de interfaces vacías y las interfaces que solo incorporan otras interfaces vacías no activan la inspección.

Considere usar el alias más explícito any.
