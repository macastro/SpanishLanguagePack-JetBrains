Informa de los parámetros @PathVariable declarados en la firma de un método y ausentes en la ruta de URL o viceversa. La solución rápida añade el parámetro ausente.

Ejemplo:


  @Get("/path/{myVariable}/")
  public String handler(@PathVariable String name_is_not_equal_to_myVariable) {
    return "...";
  }

Después de aplicar la solución rápida, el resultado es similar a:


  @Get("/path/{myVariable}/")
  public String handler(@PathVariable String myVariable) {
    return "...";
  }
