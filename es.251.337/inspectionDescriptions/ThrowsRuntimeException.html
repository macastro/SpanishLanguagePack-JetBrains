Informa sobre la declaración de excepciones no verificadas (java.lang.RuntimeException o una de sus subclases) en la cláusula throws del método.

La declaración de excepciones no verificadas no es necesaria y puede eliminarse o moverse a la etiqueta Javadoc @throws.

Ejemplo:


  public class InvalidDataException extends RuntimeException {}
  class TextEditor {
    void readSettings() throws InvalidDataException {} // Advertencia: Excepción no verificada 'InvalidDataException' declarada en la cláusula 'throws'
  }
