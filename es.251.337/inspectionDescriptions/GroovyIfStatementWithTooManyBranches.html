Reporta sentencias if con demasiadas ramas. Este tipo de sentencias pueden ser confusas y generalmente indican una abstracción insuficiente en el diseño.

Ejemplo:


if (a) {
  print "foo"
} else if (b) {
  print "bar"
} else if (c) {
  print "baz"
} else if (d) {
  print "Too many branches"
}

Utiliza el campo número máximo de ramas para especificar la cantidad máxima esperada de ramas.
