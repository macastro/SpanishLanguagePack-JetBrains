Informa sobre problemas detectados con valores mutables como listas o diccionarios utilizados como valores predeterminados en argumentos.
Los valores predeterminados de los argumentos se evalúan solo una vez durante la definición de la función, lo que significa que modificar el valor predeterminado de ese argumento afectará a todas las llamadas posteriores a esta función.

Ejemplo:


def func(s, cache={}):
    cache[s] = None

Después de aplicar la corrección rápida, el código se convierte en:


def func(s, cache=None):
    if cache is None:
        cache = {}
    cache[s] = None
