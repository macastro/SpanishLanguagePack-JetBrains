Informa los casos cuando se insertan valores NULL en columnas que aceptan sólo valores NOT NULL.

Ejemplo (Microsoft SQL Server):

CREATE TABLE br2 (
id INT NOT NULL,
col1 NVARCHAR (20) NOT NULL,
col2 NVARCHAR (20) NOT NULL,
);
--
INSERT INTO br2 (id, col1, col2)
VALUES (1, NULL, NULL);

No se pueden insertar valores NULL en col1 y col2 porque están definidas como NOT NULL. Si se ejecuta el script tal cual está, se recibirá un error. Para corregir este código, se debe reemplazar NULL en la parte de VALUES con algunos valores (por ejemplo, 42 y 'bird').

INSERT INTO br2 (id, col1, col2)
VALUES (1, 42, 'bird');
