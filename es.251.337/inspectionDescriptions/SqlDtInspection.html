Informa sobre errores en literales de fecha y hora. Esta inspección está disponible para MySQL, Oracle, Db2 y H2.

Ejemplo (MySQL):

SELECT TIME '10 -12:13:14' FROM dual;
SELECT TIME ' 12 : 13 : 14 ' FROM dual;
SELECT TIME '12 13 14' FROM dual;
SELECT TIME '12-13-14' FROM dual;
SELECT TIME '12.13.14' FROM dual;
SELECT TIME '12:13:' FROM dual;
SELECT TIME '12:13' FROM dual;
SELECT TIME '12:' FROM dual;

En este ejemplo, las fechas ignoran el estándar de literales de fecha y hora de MySQL. Por lo tanto, estos serán resaltados. Para obtener más información sobre literales de fecha y hora en MySQL, consulte Date and Time Literals en dev.mysql.com.

Los siguientes literales de fecha y hora son válidos para MySQL.

SELECT TIME '12:13:14' FROM dual;
SELECT TIME '12:13:14.555' FROM dual;
SELECT TIME '12:13:14.' FROM dual;
SELECT TIME '-12:13:14' FROM dual;
SELECT TIME '10 12:13:14' FROM dual;
SELECT TIME '-10 12:13:14' FROM dual;
