Informa sobre las expresiones switch o las declaraciones switch mejoradas con cuerpos de expresión. Estos pueden convertirse en bloques de código.

Ejemplo:


  String message = switch (errorCode) {
    case 404 -> "Not found!";
    ...
  };

Después de aplicar la corrección rápida:


  String message = switch (errorCode) {
    case 404 -> {
                yield "Not found!";
            }
    ...
  };
