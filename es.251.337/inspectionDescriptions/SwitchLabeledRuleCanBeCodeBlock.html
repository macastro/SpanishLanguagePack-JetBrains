Informa de reglas de expresiones switch o sentencias switch mejoradas con un cuerpo de expresión. Se pueden convertir en bloques de código.

Ejemplo:


  String message = switch (errorCode) {
    case 404 -> "Not found!";
    ...
  };

Después de aplicar la corrección rápida:


  String message = switch (errorCode) {
    case 404 -> {
                yield "Not found!";
            }
    ...
  };
