Detecta archivos JSON que pueden interpretarse como parte de una especificación OpenAPI/Swagger.

Estos archivos no contienen características explícitas de la especificación, pero hacen referencia a un archivo de especificación principal cercano.

El siguiente ejemplo contiene dos archivos ubicados en el mismo directorio. El primero es un archivo de especificación principal normal. El segundo hace referencia al primero, por lo que se sugiere tratarlo como una especificación.

Archivo de especificación principal openapi.yaml:


openapi: 3.1.0
components:
  schemas:
    CustomSchema:
      description: Custom schema object
      properties:
        foo:
          $ref: 'common.components.json#/components/schemas/CommonSchema'

Archivo candidato de especificación common.components.json:


{
  "components": { // 'Mark file as OpenAPI specification' highlighting
    "schemas": {
      "CommonSchema": {
        "description": "Common schema object reused in several specifications"
      }
    }
  }
}
