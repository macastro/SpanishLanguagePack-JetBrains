Informa sobre cualquier instancia de java.util.concurrent.ScheduledThreadPoolExecutor donde corePoolSize se establece en cero, ya sea mediante el método setCorePoolSize o mediante el constructor del objeto.

Un ScheduledThreadPoolExecutor sin hilos principales no ejecutará ningún contenido.

Ejemplo:


  void foo(int corePoolSize) {
      if (corePoolSize != 0) return;
      ThreadPoolExecutor executor = new ScheduledThreadPoolExecutor(corePoolSize); // Advertencia
      executor.setCorePoolSize(corePoolSize); // Advertencia
  }
