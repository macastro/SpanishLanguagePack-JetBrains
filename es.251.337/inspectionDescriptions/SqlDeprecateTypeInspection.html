Informa sobre el uso de tipos que están obsoletos y que podrían desaparecer en futuras versiones del DBMS.

Tipos reportados:

  • LONG en Oracle (consulte la página Deprecated and Desupported Features en docs.oracle.com).
  • TEXT, NTEXT e IMAGE en Microsoft SQL Server (consulte la página Deprecated Database Engine Features in SQL Server 2016 en docs.microsoft.com).

Ejemplo (Oracle):

CREATE TABLE ot.foo(
a NUMBER GENERATED BY DEFAULT AS IDENTITY,
b LONG NOT NULL
);
