Informa sobre bucles que no producen efectos secundarios después de la primera iteración, lo que podría indicar un error de programación.

Estos bucles pueden ejecutarse cero veces, una vez o infinitamente. Si no existe la posibilidad de un bucle infinito, se puede reemplazar con una sentencia if. En caso contrario, el programa podría bloquearse.

Ejemplo:


    public void foo(String baseName, String names) {
        int suffix = 1;
        String name = baseName;
        while (names.contains(name)) {
            // Error: suffix no se actualiza, causando que el cuerpo del bucle sea idempotente
            name = baseName + suffix;
        }
    }
