Informa sobre las llamadas al método toString() que utilizan la implementación predeterminada de java.lang.Object.

Esta implementación predeterminada raramente es intencional y podría ser un uso incorrecto.

Esta inspección ignora las llamadas a toString() en objetos cuyo tipo es java.lang.Object, una interfaz o una clase abstracta.

Ejemplo:


  class Bar {
      void foo1(Bar bar) {
          String s = bar.toString(); // advertencia
          /* ... */
      }
      void foo2(Object obj) {
          String s = obj.toString(); // sin advertencia aquí
          /* ... */
      }
  }
