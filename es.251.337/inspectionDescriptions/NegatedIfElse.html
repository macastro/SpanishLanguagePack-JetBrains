Informa sobre declaraciones if que contienen una rama else y tienen una condición negada.

Invertir el orden de las ramas if y else generalmente mejora la claridad de tales declaraciones.

Se puede corregir invirtiendo la declaración if actual.

Ejemplo:


    void m(Object o1, Object o2) {
        if (o1 != o2) {
            System.out.println(1);
        }
        else {
            System.out.println(2);
        }
    }

Después de aplicar la corrección rápida:


    void m(Object o1, Object o2) {
        if (o1 == o2) {
            System.out.println(2);
        } else {
            System.out.println(1);
        }
    }

Use la opción Ignorar comparaciones '!= null' para ignorar las comparaciones de la forma != null.

Use la opción Ignorar comparaciones '!= 0' para ignorar las comparaciones de la forma != 0.
