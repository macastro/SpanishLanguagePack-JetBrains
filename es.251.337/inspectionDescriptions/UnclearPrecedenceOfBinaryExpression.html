El informe contiene expresiones binarias con diferentes operadores y sin paréntesis.

Estas expresiones pueden ser difíciles de leer debido a las diferentes reglas de precedencia de los operadores.

Ejemplo:

  fun foo(b: Boolean?, i: Int?) {
      val x = b ?: i == null // se evalúa como `(b ?: i) == null`
      val y = i ?: 0 + 1 // se evalúa como `i ?: (0 + 1)`
  }
