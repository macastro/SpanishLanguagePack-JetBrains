Informa sobre cualquier instancia de sentencias for, while y do donde el cuerpo se ejecuta como máximo una vez. Esto generalmente indica la presencia de un error.

Utilice la opción Ignorar bucles for mejorados para excluir los bucles foreach. Estos a veces se utilizan como una forma compacta de realizar operaciones solo en el primer elemento de un iterable.

Ejemplo:


  for (String s : stringIterable) {
    doSomethingOnFirstString(s);
    break;
  }
