Informa sobre las llamadas de registro que están rodeadas por condiciones de protección. Esta inspección se puede utilizar para ajustar el estilo de código personalizado.

Ejemplo:


  public class TestObject {
    void test(Object object) {
      if(LOG.isDebugEnabled()){
        LOG.debug("some logging " + expensiveCalculation(1));
      }
    }
  }

Después de aplicar la corrección rápida:


  public class TestObject {
    void test(Object object) {
      LOG.debug("some logging " + expensiveCalculation(1));
    }
  }

Esta inspección es compatible con los frameworks de registro Log4j2 y SLF4J (excepto builders).
