Reporta referencias a métodos no resueltos en los parámetros initMethod y destroyMethod de la anotación @Bean.

Ejemplo:

Si la clase MyBean no contiene el método 'public void doInit(){...}', se reportará "Método no resuelto" en la expresión 'doInit' 

  public class MyBean {...}
  @Component
  public class MyComponent {
     @Bean(initMethod="doInit" )
     public MyBean myBean() {...}
  }

En este ejemplo, si MyBean no define el método doInit(), esta inspección reportará una referencia a método no resuelta.
