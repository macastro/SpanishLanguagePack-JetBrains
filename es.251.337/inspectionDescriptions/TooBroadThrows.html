Cláusula throws que declara excepciones más generales que las que realmente lanza el método.

Ejemplo:


  public void createFile() throws Exception { // Advertencia: 'throws Exception' es demasiado general, oculta la excepción 'IOException'
    File file = new File("pathToFile");
    file.createNewFile();
  }

Después de aplicar la corrección rápida:


  public void createFile() throws IOException {
    File file = new File("pathToFile");
    file.createNewFile();
  }

Configuración de la inspección:

  • Use el campo Número máximo de excepciones ocultas para advertir para ignorar las excepciones que ocultan más excepciones que el número especificado.
  • Use la opción Advertir solo para RuntimeException, Exception, Error o Throwable para que esta inspección advierta solo sobre las excepciones más comunes.
  • Use la opción Ignorar excepciones declaradas en métodos que sobrescriben métodos de biblioteca para ignorar las cláusulas throws demasiado generales en métodos que sobrescriben métodos de biblioteca.
  • Use la opción Ignorar excepciones que ocultan otras pero que son lanzadas para ignorar cualquier excepción que oculte otras pero que aún pueda ser lanzada desde el cuerpo del método y por lo tanto técnicamente no sea demasiado general.
