Informa de todos los objetos poseídos que no se han liberado, almacenados en variables de instancia.
Obtengras la posesión de un objeto cuando:
- Se crea usando un método cuyo nombre comienza con alloc o new o incluye copy
- Le envías un mensaje retain
- Estableces el valor de una propiedad que tiene el atributo retain o copy y se sintetiza con una variable de instancia
Para abandonar la posesión de un objeto, necesitas:
- Enviarle un mensaje release o autorelease
- Asignar nil a la propiedad sintetizada con la variable de instancia
Esta inspección se deshabilita si está habilitada la gestión automática de referencias (ARC). En caso de que la siguiente casilla de verificación esté desactivada, puedes liberar las variables de instancia en cualquier método de la clase. Si está desactivada, podrás liberar las variables de instancia en los métodos dealloc, tearDown, applicationWillTerminate:, didTurnIntoFault y en cualquier método que pueda invocarse desde ellos.
