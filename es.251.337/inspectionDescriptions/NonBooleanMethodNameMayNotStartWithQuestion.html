Reporta métodos no booleanos que comienzan con palabras interrogativas. Estos nombres de métodos pueden resultar confusos.

Esta inspección ignorará los métodos no booleanos que sobrescriben métodos de biblioteca.

Ejemplo: 


  public void hasName(String name) {
    assert names.contains(name);
  }

La corrección rápida para renombrar estos métodos solo está disponible en el editor.

Configuración de la inspección:

  • Utilice la lista de prefijos para nombres de métodos booleanos para especificar palabras interrogativas que deberían usarse solo para métodos booleanos.
  • Utilice la opción ignorar métodos con tipo de retorno 'java.lang.Boolean' para ignorar métodos que devuelven java.lang.Boolean.
  • Utilice la opción ignorar métodos que sobrescriben/implementan métodos superiores para ignorar métodos que tienen métodos superiores.
