Informa sobre usos de expresiones this con instanceof o getClass() == SomeClass.class.

Tales expresiones indican un fallo en el diseño orientado a objetos y deberían reemplazarse con construcciones polimórficas.

Ejemplo:


class Super {
  void process() {
    if (this instanceof Sub) { // advertencia
      doSomething();
    } else {
      doSomethingElse();
    }
  }
}
class Sub extends Super {}

Para corregir el problema, utilice métodos sobrescritos:


class Super {
  void process() {
    doSomethingElse();
  }
}
class Sub extends Super {
  @Override
  void process() {
    doSomething();
  }
}
