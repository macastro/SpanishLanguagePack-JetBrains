Se informa cuando las clases padre e hijo en la jerarquía de pruebas JUnit están anotadas con @RunWith. Esto puede provocar un comportamiento inesperado en las pruebas.

Ejemplo:


  @RunWith(Suite.class)
  @SuiteClasses(MySuiteClass.class)
  public abstract Parent {
  }
  @RunWith(Parameterized.class)
  public MyTest {
  }
