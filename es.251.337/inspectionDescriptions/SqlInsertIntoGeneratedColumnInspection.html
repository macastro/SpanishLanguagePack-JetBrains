Informa de sentencias INSERT que asignan valores a columnas generadas. Las columnas generadas se pueden leer, pero sus valores no pueden escribirse directamente.

Ejemplo (PostgreSQL):

CREATE TABLE foo
(
   col1 INT,
   col2 INT GENERATED ALWAYS AS (col1 + 1) STORED
);
INSERT INTO foo(col1, col2) VALUES (1, 2);

No puedes insertar 2 en la columna col2 porque esta columna es generada. Para que este script funcione, puedes cambiar 2 por DEFAULT.

INSERT INTO foo(col1, col2) VALUES (1, DEFAULT);
