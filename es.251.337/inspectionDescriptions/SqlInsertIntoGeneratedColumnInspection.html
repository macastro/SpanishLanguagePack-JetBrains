Reporta una sentencia INSERT que intenta asignar valores a columnas generadas. Las columnas generadas pueden ser le√≠das pero no se pueden escribir directamente.

Ejemplo (PostgreSQL):

CREATE TABLE foo
(
   col1 INT,
   col2 INT GENERATED ALWAYS AS (col1 + 1) STORED
);
INSERT INTO foo(col1, col2) VALUES (1, 2);

No puede insertar 2 en la columna col2 porque es una columna generada. Para que este script funcione, puede cambiar 2 por DEFAULT.

INSERT INTO foo(col1, col2) VALUES (1, DEFAULT);
