Informa clases que no tienen un registrador declarado.

Asegurarse de que cada clase tiene un registrador dedicado es un paso importante para proveer una implementación de registro unificada para una aplicación. Las interfaces, enumeraciones, anotaciones, clases internas y clases abstractas no son informadas por esta inspección.

Por ejemplo:


  public class NoLoggerDeclared {

    int calculateNthDigitOfPi(int n) {
      // todo
      return 1;
    }
  }

Usa la tabla en la sección Opciones para especificar nombres de clases de registrador. Las clases que no declaren un campo con el tipo de una de las clases especificadas serán informadas por esta inspección.
