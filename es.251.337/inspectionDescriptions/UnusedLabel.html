Informa sobre etiquetas que no son objetivo de ninguna sentencia break o continue.

Ejemplo:


  label: for (int i = 0; i < 10; i++) {
    if (i == 3) {
      break;
    }
  }

Después de aplicar la corrección rápida, la etiqueta será eliminada:


  for (int i = 0; i < 10; i++) {
    if (i == 3) {
      break;
    }
  }
