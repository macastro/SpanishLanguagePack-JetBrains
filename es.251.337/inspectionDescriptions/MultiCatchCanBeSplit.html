Informa sobre múltiples secciones catch y sugiere separarlas en bloques catch individuales.

Ejemplo:


  try {
    int i = getIndex();
  } catch (NullPointerException|IndexOutOfBoundsException e) {
    e.printStackTrace();
  }

Después de aplicar la corrección rápida:


  try {
    int i = getIndex();
  } catch (NullPointerException e) {
    e.printStackTrace();
  } catch (IndexOutOfBoundsException e) {
    e.printStackTrace();
  }
