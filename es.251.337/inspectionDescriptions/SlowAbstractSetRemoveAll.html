Informa sobre las llamadas a java.util.Set.removeAll() con un argumento java.util.List.

Una llamada de este tipo puede ser lenta cuando el tamaño del argumento es mayor o igual que el tamaño del conjunto, y el conjunto es una subclase de java.util.AbstractSet. En este caso, List.contains() se llama para cada elemento en el conjunto, lo que realizará una búsqueda lineal.

Ejemplo:

public void check(String... ss) {
    // posible complejidad O(n^2)
    mySet.removeAll(List.of(ss));
}

Después de aplicar la solución rápida:

public void check(String... ss) {
    // complejidad O(n)
    List.of(ss).forEach(mySet::remove);
}
