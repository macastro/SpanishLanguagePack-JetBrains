Informa sobre variables locales innecesarias que no mejoran la comprensión del método, incluyendo:
  • Variables locales que se devuelven inmediatamente.
  • Variables locales que se asignan inmediatamente a otra variable y no se vuelven a utilizar.
  • Variables locales que siempre tienen el mismo valor que otra variable local o parámetro.

Ejemplo:


  boolean yes() {
    boolean b = true;
    return b;
  }

Después de aplicar la corrección rápida:


  boolean yes() {
    return true;
  }

Configuración de la inspección:

  • Utilice la opción Ignorar variables que se devuelven o lanzan inmediatamente para ignorar variables que se devuelven o lanzan inmediatamente. Algunos estilos de codificación recomiendan usar estas variables para mayor claridad y facilidad de depuración.
  • Utilice la opción Ignorar variables con anotaciones para ignorar variables que tienen anotaciones.
