Informa sobre literales de cadena que contienen caracteres nacionales sin el prefijo N.

Sin el prefijo N, la cadena se convertirá a la página de códigos predeterminada de la base de datos. Esta página de códigos predeterminada podría no reconocer algunos caracteres. Para más información, consulte la página "nchar y nvarchar" (Transact-SQL) en docs.microsoft.com.

Ejemplo (Microsoft SQL Server):

SELECT 'abcde' AS a;
SELECT N'abcde' AS b;
SELECT 'абвгд' AS c;
SELECT N'абвгд' AS d;

SELECT 'абвгд' AS c; no tiene el prefijo N, la parte 'абвгд' será resaltada.
