Informa sobre declaraciones de import que no están ordenadas según el estilo de código actual (consulte "Configuración|Editor|Estilo de código").

Ejemplo:


  import java.util.List;
  import java.util.ArrayList;
  public class Example {
    List list = new ArrayList();
  }

Después de aplicar la corrección rápida "Optimizar imports": 


  import java.util.ArrayList;
  import java.util.List;
  public class Example {
    List list = new ArrayList();
  }
