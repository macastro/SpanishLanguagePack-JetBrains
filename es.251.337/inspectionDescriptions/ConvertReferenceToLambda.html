La referencia a función puede ser reemplazada por una función literal (lambda).

A veces, pasar un lambda es más directo y más consistente con el resto del código. Además, esta corrección puede ser útil cuando necesitas reemplazar una llamada simple por una más compleja.

Ejemplo:


  fun Int.isEven() = this % 2 == 0
  fun example() {
      val numbers = listOf(1, 2, 4, 7, 9, 10)
      val evenNumbers = numbers.filter(Int::isEven)
  }

Después de aplicar la corrección rápida:


  fun Int.isEven() = this % 2 == 0
  fun example() {
      val numbers = listOf(1, 2, 4, 7, 9, 10)
      val evenNumbers = numbers.filter { it.isEven() }
  }
