Informa sobre clases internas no estáticas declaradas en clases que implementan Serializable y en clases que no implementan Serializable.

Es poco probable que estas clases se serialicen correctamente debido a la referencia implícita a la clase externa.

Ejemplo:


  class A {
      class Main implements Serializable {
    }
  }

Configure la inspección con las siguientes opciones:

  • Lista de clases cuyos herederos no deberían ser reportados por esta inspección. Esto aplica a clases que heredan Serializable pero no se utilizan para serialización.
  • Si se deben ignorar las clases anónimas que implementan Serializable.
