Reporta las llamadas a los métodos `get` y `set` de Java que se pueden reemplazar con las propiedades sintéticas de Kotlin.

Se puede usar la corrección rápida "Usar sintaxis de acceso a propiedad" para modificar el código automáticamente.

Ejemplo:


  // Java:
  public class JavaClassWithGetter {
      private final String expr = "result";

      // ...

      public String getExpr() {
          return expr;
      }
  }


  // Kotlin:
  fun test(j: JavaClassWithGetter) {
      // ...
      j.getExpr() // <== La corrección rápida simplifica la expresión a 'j.expr'
  }
