Los métodos Java get y set pueden ser reemplazados por propiedades sintéticas de Kotlin.

La corrección rápida usando sintaxis de acceso a propiedades está disponible para corregir el código automáticamente.

Ejemplo:


  // Java:
  public class JavaClassWithGetter {
      private final String expr = "result";
      // ...
      public String getExpr() {
          return expr;
      }
  }


  // Kotlin:
  fun test(j: JavaClassWithGetter) {
      // ...
      j.getExpr() // <== la corrección rápida simplificará la expresión a 'j.expr'
  }
