Informa sobre variables de instancia que posiblemente no estén inicializadas durante la inicialización del objeto.

Ejemplo:


  class Foo {
    public int bar;
    static { }
  }

Tenga en cuenta que esta inspección utiliza un algoritmo de flujo de datos muy conservador y podría reportar incorrectamente variables de instancia como no inicializadas. Las variables reportadas como inicializadas siempre estarán inicializadas.

Use la opción Ignorar campos primitivos para ignorar los campos primitivos no inicializados.
