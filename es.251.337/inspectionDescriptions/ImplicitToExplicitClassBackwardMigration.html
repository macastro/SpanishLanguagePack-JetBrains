Inspecciona clases implícitamente declaradas y sugiere reemplazarlas por clases regulares.

Ejemplo (dentro del archivo Sample.java):


  public static void main() {
      System.out.println("Hello, world!");
  }

Después de aplicar el arreglo rápido:


public class Sample {
    public static void main() {
        System.out.println("Hello, world!");
    }
}

Esta inspección puede ser útil para degradar para garantizar la compatibilidad con versiones anteriores de Java.
