Informa de clases que implementan dos o más rasgos que contienen métodos con las mismas firmas.

Llamar a estos métodos puede tener resultados inesperados.

La corrección rápida agrega un método de anulación explícita.

Ejemplo:


  trait T1 {
      def foo() {}
  }

  trait T2 {
      def foo() {}
  }

  class X implements T1, T2 {}

  // Se llamará a T2.foo()
  new X().foo()

Después de aplicar la corrección rápida:


  class X implements T1, T2 {
      @Override
      Object foo() {
          return T2.super.foo()
      }
  }
