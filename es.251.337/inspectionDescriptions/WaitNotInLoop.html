Se informa de una llamada a wait() que no se realiza dentro de un bucle.

wait() se utiliza normalmente para suspender un hilo hasta que una condición se vuelva verdadera. El hilo puede despertarse por diferentes razones, por lo que la condición debe verificarse después de que la llamada a wait() retorne. Esto se logra fácilmente utilizando un bucle.

Ejemplo:


  class BoundedCounter {
    private int count;
    synchronized void inc() throws InterruptedException {
      if (count >= 10) wait();
      ++count;
    }
  }

El código correcto debería ser similar a:


  class BoundedCounter {
    private int count;
    synchronized void inc() throws InterruptedException {
      while (count >= 10) wait();
      ++count;
    }
  }
