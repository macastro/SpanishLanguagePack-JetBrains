Reporta llamadas a java.lang.Runtime.exec() que toman una cadena construida dinámicamente como el comando a ejecutar.

Las cadenas de ejecución construidas son una fuente común de brechas de seguridad. Por defecto, esta inspección ignora las constantes en tiempo de compilación.

Ejemplo:


  String i = getUserInput();
  Runtime runtime = Runtime.getRuntime();
  runtime.exec("foo" + i); // reporta una advertencia

Utiliza las configuraciones de inspección para considerar cualquier campo static final como constante. Ten cuidado, porque cadenas como la siguiente serán ignoradas cuando se habilite la opción: 


  static final String COMMAND = "ping " + getDomainFromUserInput() + "'";
