Reporta llamadas a java.lang.Runtime.exec() que utilizan cadenas construidas dinámicamente como comandos a ejecutar.

Las cadenas de ejecución construidas dinámicamente son una fuente común de vulnerabilidades de seguridad. Por defecto, esta inspección ignora las constantes en tiempo de compilación.

Ejemplo:


  String i = getUserInput();
  Runtime runtime = Runtime.getRuntime();
  runtime.exec("foo" + i); // Genera una advertencia

La configuración de la inspección permite tratar cualquier campo static final como una constante. Tenga cuidado, ya que cuando esta opción está habilitada, se ignorarán cadenas como esta: 


  static final String COMMAND = "ping " + getDomainFromUserInput() + "'";
