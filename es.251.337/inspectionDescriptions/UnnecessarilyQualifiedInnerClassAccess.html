Informa sobre cualquier referencia a clases internas que esté innecesariamente calificada con el nombre de la clase contenedora.

Esta calificación puede eliminarse de forma segura, lo que a veces requiere añadir importaciones para las clases internas.

Ejemplo:


  class X {
     X.Y foo;
     class Y{}
  }

Después de aplicar la corrección rápida:


  class X {
     Y foo;
     class Y{}
  }

Utilice la opción Ignorar referencias que requieran importaciones para ignorar las referencias a clases internas donde la eliminación de la calificación requeriría añadir importaciones.
