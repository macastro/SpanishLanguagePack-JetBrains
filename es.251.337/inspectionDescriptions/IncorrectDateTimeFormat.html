Informa sobre patrones de formato de fecha y hora incorrectos.

Informa sobre los siguientes errores:

  • Letras de patrón no soportadas, como "TT"
  • Uso de caracteres reservados, como "#"
  • Uso incorrecto del relleno
  • Corchetes no balanceados
  • Cantidad incorrecta de letras de patrón consecutivas

Ejemplos:


  DateTimeFormatter.ofPattern("[][]]"); // No hay '[' izquierdo antes del ']' derecho
  DateTimeFormatter.ofPattern("TT"); // Letra de patrón ilegal 'T'
  DateTimeFormatter.ofPattern("{"); // Uso del carácter reservado '{'
  DateTimeFormatter.ofPattern("MMMMMM"); // Demasiadas letras de patrón 'M' consecutivas
