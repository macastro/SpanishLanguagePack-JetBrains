Informa de constructores public de clases abstract.

Los constructores de clases abstract solo se pueden llamar desde los constructores de sus subclases, por lo que declararlos public puede resultar confuso.

La corrección rápida convierte dichos constructores en protegidos.

Ejemplo:


  public abstract class Foo {
    public Foo () { // warning: has 'public' modifier
        /* ... */
    }
  }

Después de aplicar la corrección rápida:


  public abstract class Foo {
    protected Foo () {
        /* ... */
    }
  }

Configurar la inspección:

Utiliza la opción Ignorar para clases no públicas siguiente para ignorar los constructores public en clases no públicas.
