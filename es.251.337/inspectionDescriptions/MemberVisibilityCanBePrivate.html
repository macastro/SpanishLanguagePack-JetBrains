El informe puede establecerse como private para cumplir con el principio de encapsulamiento.

Ejemplo:


class Service(val url: String) {
    fun connect(): URLConnection = URL(url).openConnection()
}

Después de aplicar la corrección rápida (considerando que url no se utiliza fuera de la clase Service):


class Service(private val url: String) {
    fun connect(): URLConnection = URL(url).openConnection()
}
