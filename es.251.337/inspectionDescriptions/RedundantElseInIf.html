Reporta un if redundante con return dentro de otro if

Ejemplo:


  fun foo(arg: Boolean): Int {
      if (arg) return 0
      else { // este else es redundante, el código entre llaves puede moverse a la izquierda
          someCode()
      }
  }

Después de aplicar la corrección rápida:


  fun foo(arg: Boolean): Int {
      if (arg) return 0
      someCode()
  }
