Inspecciona las expresiones pasadas como argumentos para los parámetros `@Pattern` y devueltas por los métodos anotados por `@Pattern` que no concuerdan con el patrón especificado.

Ejemplo:


  @Pattern("\\d\\d-\\d\\d\\d") String getId() {
    return "1";
  }

Utiliza la opción Marcar expresiones de constantes no compilables en tiempo para permitir que la inspección informe expresiones con un valor desconocido y ofrezca agregar una anotación de sustitución (@Subst).
