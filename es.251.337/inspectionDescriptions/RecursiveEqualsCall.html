Reporta llamadas recursivas de equals(==).

En Kotlin, == compara valores de objetos llamando al método equals en segundo plano. Por otro lado, === compara objetos por referencia.

=== se usa típicamente en implementaciones del método equals. Sin embargo, === puede confundirse erróneamente con ==, lo que lleva a una recursión infinita.

Ejemplo:


  class X {
      override fun equals(other: Any?): Boolean {
          if (this == other) return true
          return false
      }
  }

Después de aplicar la corrección rápida:


  class X {
      override fun equals(other: Any?): Boolean {
          if (this === other) return true
          return false
      }
  }
