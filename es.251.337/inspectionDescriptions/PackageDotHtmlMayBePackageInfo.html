El informe registra cualquier archivo package.html del paquete.

Desde JDK 1.5, se recomienda usar el archivo package-info.java en su lugar, ya que este tipo de archivo también puede contener anotaciones del paquete. De esta manera, package-info.java se convierte en el único repositorio para anotaciones y documentación a nivel de paquete.

Ejemplo: package.html


<html>
  <body>
    Documentation example.
  </body>
</html>

Después de aplicar la corrección rápida: package-info.java


/**
 * Documentation example.
 */
package com.sample;
