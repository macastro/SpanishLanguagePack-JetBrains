Informa sobre cláusulas except que no están listadas en el orden correcto de más específica a más general, o cuando una clase de excepción se captura dos veces.

Si no se corrige el orden, el manejador más específico podría no capturar algunas excepciones.

Ejemplo:


try:
    call()
except ValueError:
    pass
except UnicodeError:
    pass

El IDE sugiere mover la cláusula hacia arriba. Después de aplicar la corrección rápida, el código se convierte en:


try:
    call()
except UnicodeError:
    pass
except ValueError:
    pass
