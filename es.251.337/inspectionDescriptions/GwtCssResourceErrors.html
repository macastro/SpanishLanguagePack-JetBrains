Informa sobre inconsistencias entre la interfaz que extiende CssResource y su archivo CSS correspondiente.

Los métodos de la interfaz que extiende com.google.gwt.resources.client.CssResource deben corresponder a las clases CSS o declaraciones @def en el archivo CSS vinculado.

La corrección rápida crea los métodos faltantes para las clases CSS y declaraciones @def, y viceversa.
