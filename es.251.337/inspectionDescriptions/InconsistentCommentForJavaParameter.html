Informa sobre nombres de parámetros inconsistentes en llamadas a métodos de Java especificados en bloques de comentarios.

Ejemplo:


  // Java
  public class JavaService {
      public void invoke(String command) {}
  }


  // Kotlin
  fun main() {
      JavaService().invoke(/* name = */ "fix")
  }

La corrección rápida ajusta el nombre del parámetro en el bloque de comentarios:


  fun main() {
      JavaService().invoke(/* command = */ "fix")
  }
