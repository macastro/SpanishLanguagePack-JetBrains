Informa sobre expresiones switch que no cubren todos los resultados posibles de la expresión coincidente.

Groovy no requiere que las expresiones switch sean exhaustivas. Funciona como si tuviera una rama implícita default -> null. Si el desarrollador olvida insertar las ramas case necesarias, esto puede resultar en valores null inesperados.

Ejemplo:


  enum A { X, Y }
  def foo(A a) {
    def x = switch (a) { // reports switch
      case A.X -> ...
    }
  }
