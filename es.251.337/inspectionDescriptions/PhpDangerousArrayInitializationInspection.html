Informa sobre inicializaciones peligrosas de arrays (como $arr[] = value).

Esta práctica no es recomendable ya que si $arr ya contiene algún valor (por ejemplo, una cadena de texto proveniente de variables de solicitud), este valor se mantendrá y [] podría actuar como un operador de acceso a cadenas. Es mejor inicializar siempre las variables mediante asignación directa.

Para más información, consulta Creating/modifying with square bracket syntax (php.net).
