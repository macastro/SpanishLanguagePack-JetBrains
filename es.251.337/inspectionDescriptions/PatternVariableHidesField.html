Informa sobre variables de patrón que tienen el mismo nombre que los campos de las clases circundantes. Debido a esta nomenclatura, puede usar incorrectamente la variable de patrón cuando en realidad pretendía usar el campo del mismo nombre.

Se recomienda renombrar la variable mediante la corrección rápida.

Ejemplo:


  class Pointless {
    Point p = new Point();
    public void test(Object a) {
      if (a instanceof Point p) {
        System.out.print("a is a point (" + p.x + ", " + p.y + ')');
      } else {
        System.out.print("p is a point (" + p.x + ", " + p.y + ')');
      }
    }
  }
