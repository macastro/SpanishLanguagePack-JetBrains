Informa sobre accesos a campos que no están calificados con this u otros calificadores.

Algunos estilos de codificación requieren que todos los accesos a campos estén calificados para evitar confusiones con variables locales o parámetros.

Ejemplo:


  class Foo {
    int foo;
    void bar() {
      foo += 1;
    }
  }

Después de aplicar la corrección rápida:


  class Foo {
    int foo;
    void bar() {
      this.foo += 1;
    }
  }
