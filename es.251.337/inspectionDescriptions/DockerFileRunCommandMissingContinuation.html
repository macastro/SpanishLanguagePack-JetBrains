Informa sobre la falta de caracteres de continuación en el comando RUN.

En los comandos RUN con formato shell, se debe usar '\'(barra invertida) para continuar una instrucción RUN en la siguiente línea. De lo contrario, la construcción de Docker fallará.

Ejemplo:


  # the command below will fail
  RUN /bin/bash -c 'source $HOME/.bashrc;
      echo $HOME'

Después de aplicar la corrección rápida:


  RUN /bin/bash -c 'source $HOME/.bashrc; \
      echo $HOME'
