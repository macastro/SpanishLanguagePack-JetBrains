Informa sobre métodos que devuelven tipo boolean y siempre son negados cuando se invocan.

Se proporciona una corrección rápida para invertir y opcionalmente renombrar el método. Por razones de rendimiento, no todos los métodos problemáticos se resaltan en el editor.

Ejemplo:


  class C {
    boolean alwaysTrue() {
      return true;
    }
    void f() {
      if (!alwaysTrue()) {
        return;
      }
    }
    boolean member = !alwaysTrue();
  }

Después de aplicar la corrección rápida:


  class C {
    boolean alwaysFalse() {
      return false;
    }
    void f() {
      if (alwaysFalse()) {
        return;
      }
    }
    boolean member = alwaysFalse();
  }
