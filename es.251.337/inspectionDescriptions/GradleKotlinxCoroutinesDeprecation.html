Informa sobre las dependencias de la biblioteca kotlinx.coroutines en Gradle que deben actualizarse para ser compatibles con Kotlin 1.3+.

Ejemplo:


  dependencies {
      implementation 'org.jetbrains.kotlinx:kotlinx-coroutines-core:0.0.1'
  }

La corrección rápida cambiará la versión de la biblioteca kotlinx.coroutines a una versión compatible con Kotlin 1.3:


  dependencies {
      implementation 'org.jetbrains.kotlinx:kotlinx-coroutines-core:0.27.0-eap13'
  }
