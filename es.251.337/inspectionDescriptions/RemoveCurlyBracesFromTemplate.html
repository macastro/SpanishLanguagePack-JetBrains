Reporta el uso de llaves en plantillas de cadena alrededor de identificadores simples.

Usa la acción rápida de "Eliminar llaves" para eliminar las llaves redundantes.

Ejemplos:


  fun redundante() {
     val x = 4
     val y = "${x}" // <== redundante
  }

  fun uso_correcto() {
      val x = "x"
      val y = "${x.length}" // <== Ok
  }

Después de aplicar la acción rápida:


  fun redundante() {
     val x = 4
     val y = "$x"
  }

  fun uso_correcto() {
      val x = "x" <== Actualizado
      val y = "${x.length}"
  }
