Informa sobre variables de plantilla Go duplicadas declaradas en el mismo ámbito.

Las variables duplicadas reasignan variables existentes con el mismo nombre. Esta operación puede causar diferentes problemas impredecibles.

Ejemplo:

{{$v := 0}}{{$v := 1}}{{$v}} es 0. {{/* se evalúa como '1 es 0' */}}
{{$v := 0}}{{$w := 1}}{{$v}} es 0. {{/* funciona según lo esperado */}}
