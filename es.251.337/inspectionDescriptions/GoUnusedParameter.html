Informa sobre parámetros de función no utilizados. 
func main() {
 printAll(
  42,
  "bird",
 )
}
func printAll(
 i int,
 s string,
) {
 fmt.Println(i)
}

Llamamos a la función printAll pasando 42 y bird como argumentos. La función printAll acepta dos parámetros, int y string, pero solo utiliza el primer parámetro. Por lo tanto, s string se mostrará en gris.
