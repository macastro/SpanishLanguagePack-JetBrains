Informa sobre módulos utilizados en cláusulas de superclase. Las cláusulas de superclase solo pueden hacer referencia a clases.

La cláusula de superclase no válida se puede eliminar mediante una corrección rápida.

Ejemplo:

# 'Kernel' 模块用作超类
class Example < Kernel
end

Después de aplicar la corrección rápida:

class Example
end
