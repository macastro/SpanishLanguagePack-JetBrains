Informa sobre cualquier expresión List.indexOf() que pueda ser reemplazada por el método List.contains().

Ejemplo:


  boolean hasEmptyString(List<String> list) {
    // Advertencia: puede simplificarse
    return list.indexOf("") >= 0;
  }

La corrección rápida proporcionada reemplaza la llamada indexOf por una llamada contains:


  boolean hasEmptyString(List<String> list) {
    // Corrección rápida aplicada
    return list.contains("");
  }
