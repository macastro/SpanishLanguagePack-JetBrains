Regla de inspección que detecta declaraciones o expresiones switch con bloques de código redundantes.

Ejemplo:


  String s = switch (n) {
    case 1 -> { yield Integer.toString(n); }
    default -> "default";
  };

Después de aplicar la corrección rápida:


  String s = switch (n) {
    case 1 -> Integer.toString(n);
    default -> "default";
  };
