Informa sobre reglas con etiqueta sobre declaraciones `switch` o expresiones `switch` que tienen un bloque de código redundante.

Ejemplo:


  String s = switch (n) {
    case 1 -> { yield Integer.toString(n); }
    default -> "default";
  };

Después de aplicar la solución rápida:


  String s = switch (n) {
    case 1 -> Integer.toString(n);
    default -> "default";
  };
