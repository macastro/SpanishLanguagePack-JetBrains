Informa sobre paréntesis redundantes en llamadas a funciones donde el único parámetro es un lambda fuera de los paréntesis.

Utilice la corrección rápida "Eliminar paréntesis innecesarios de la llamada a función con lambda" para limpiar el código.

Ejemplo:


  fun foo() {
      listOf(1).forEach() {  }
  }

Después de aplicar la corrección rápida:


  fun foo() {
      listOf(1).forEach {  }
  }
