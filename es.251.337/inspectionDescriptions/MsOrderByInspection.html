Informa sobre el uso de la cláusula ORDER BY en vistas, funciones inline, tablas derivadas, subconsultas y expresiones de tabla comunes sin el uso de TOP, OFFSET o FOR XML.

Para obtener información detallada sobre el uso de ORDER BY, consulte SELECT - Cláusula ORDER BY (Transact-SQL) en docs.microsoft.com.

Ejemplo (Microsoft SQL Server):

CREATE TABLE foo (a INT NOT NULL, b  INT NOT NULL);
SELECT *
FROM (SELECT a, b
FROM foo A
WHERE a < 89
ORDER BY b) ALIAS;

ORDER BY se resaltará como un error en la subconsulta. Puede agregar TOP, OFFSET o FOR XML a la subconsulta. Alternativamente, use la corrección rápida Eliminar elemento para eliminar la parte ORDER BY.

Después de aplicar la corrección rápida:

SELECT *
FROM (SELECT a, b
FROM foo A
WHERE a < 89) ALIAS;
