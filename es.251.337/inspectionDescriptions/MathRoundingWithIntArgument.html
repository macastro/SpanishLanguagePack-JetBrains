Informa sobre las llamadas a los métodos round(), ceil(), floor(), rint() de Math y StrictMath con int como argumento.

Estos métodos podrían ser llamados en caso de que se espere que el argumento sea long o double, y podría tener resultados inesperados.

La inspección proporciona una corrección que simplifica dichas expresiones (excepto round) a castear a double.

Ejemplo:


  int i = 2;
  double d1 = Math.floor(i);

Después de aplicar la corrección rápida:


  int i = 2;
  double d1 = i;
