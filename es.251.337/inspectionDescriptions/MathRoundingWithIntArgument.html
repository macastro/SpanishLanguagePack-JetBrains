Reporta las llamadas a los métodos round(), ceil(), floor(), rint() de Math y StrictMath cuando se utilizan con argumentos de tipo int.

Estos métodos pueden ser llamados cuando se esperan argumentos de tipo long o double, lo que podría producir resultados inesperados.

Esta inspección proporciona una corrección que simplifica tales expresiones (excepto round) a una conversión a double.

Ejemplo:


  int i = 2;
  double d1 = Math.floor(i);

Después de aplicar la corrección rápida:


  int i = 2;
  double d1 = i;
