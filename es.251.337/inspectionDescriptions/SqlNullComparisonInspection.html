Informa las comparaciones con NULL que pueden ser reemplazadas con operadores IS NULL o IS NOT NULL.

Ejemplo (Microsoft SQL Server):

CREATE TABLE foo ( id int );

SELECT * FROM foo WHERE NULL = NULL;
SELECT * FROM foo WHERE NULL != NULL;
