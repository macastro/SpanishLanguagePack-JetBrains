Informa parámetros o variables locales que pueden tener el modificador final añadido a su declaración.

Ejemplo:


  ArrayList<Integer> list = new ArrayList();
  fill(list);
  return list;

Después de que se aplica la corrección rápida:


  final ArrayList<Integer> list = new ArrayList();
  fill(list);
  return list;

Utilice las opciones de la inspección para definir si los parámetros o las variables locales se deben informar.
