Informa sobre archivos module-info.java vacíos, indicando dependencias de módulos no resueltas. Añade automáticamente las declaraciones requires necesarias mediante el análisis de los import. Para suprimir esta advertencia, puede escribir cualquier comentario en el cuerpo de la declaración del módulo, como se muestra a continuación: 

module module.name {
    // Sin dependencias
}
Corrección rápida: Completar dependencias del módulo rellena los requires faltantes basándose en los import del código fuente. Novedades en 2024.1
