Informa sobre la llamada a containsAll() en java.util.List.

La complejidad temporal de esta llamada al método es O(n·m), donde n es el número de elementos en la lista que llama al método y m es el número de elementos en la colección pasada como parámetro formal al método. Esta puede ser una operación costosa cuando la lista es grande.

La corrección rápida envuelve la lista en un new java.util.HashSet<>(), ya que el tiempo necesario para crear un java.util.HashSet desde un java.util.List y ejecutar containsAll() en el java.util.HashSet es O(n+m).

Ejemplo:

  public boolean check(List<String> list, Collection<String> collection) {
    // Complejidad O(n·m)
    return list.containsAll(collection);
  }

Después de aplicar la corrección rápida:

  public boolean check(List<String> list, Collection<String> collection) {
    // Complejidad O(n+m)
    return new HashSet<>(list).containsAll(collection);
  }
