El informe puede reemplazar las llamadas a funciones con operadores binarios (especialmente los operadores de comparación).

Ejemplo:

  fun test(): Boolean {
      return 2.compareTo(1) > 0 // 'compareTo()' reemplazable
  }

Después de aplicar la corrección rápida:

  fun test(): Boolean {
      return 2 > 1
  }
