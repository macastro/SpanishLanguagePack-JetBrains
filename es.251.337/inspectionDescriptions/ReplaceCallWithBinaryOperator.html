Inspecciona llamadas a funciones que se pueden reemplazar con operadores binarios, en concreto con los relacionados con comparaciones.

Ejemplo:

  fun test(): Boolean {
      return 2.compareTo(1) > 0 // 'compareTo()' reemplazable
  }

Después de aplicar la corrección rápida:

  fun test(): Boolean {
      return 2 > 1
  }
