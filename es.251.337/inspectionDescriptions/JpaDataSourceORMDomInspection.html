Informa sobre los descriptores XML de Persistence ORM que no se pueden resolver en el origen de datos configurado y sugiere actualizar el origen de datos o asignar uno diferente.

Ejemplo de tabla de base de datos:


  CREATE TABLE DATA_TABLE (
    ID INTEGER,
    DATA VARCHAR(100)
  );

Ejemplo de archivo JSP:


  <?xml version="1.0" encoding="UTF-8"?>
  <entity-mappings xmlns="http://java.sun.com/xml/ns/persistence/orm" version="1.0">
    <entity name="JavaEntity" class="JavaEntity">
        <table name="UNKNOWN_TABLE"/> // Error: tabla no resuelta
        <attributes>
          <id name="id">
            <column name="ID" nullable="false"/>
          </id>
          <basic name="UNKNOWN_COLUMN"> // Error: columna no resuelta
            <column name="UNKNOWN_COLUMN" length="12"/>
          </basic>
        </attributes>
    </entity>
  </entity-mappings>
