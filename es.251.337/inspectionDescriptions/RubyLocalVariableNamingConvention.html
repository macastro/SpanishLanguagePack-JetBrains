Reporta variables locales cuyos nombres son demasiado cortos, demasiado largos o no coinciden con el patrón de expresión regular especificado.

Utiliza el campo patrón para especificar la expresión regular que se usará para validar los nombres de las variables locales.

Utiliza los campos longitud mínima y longitud máxima para especificar la longitud mínima y máxima permitida para los nombres, respectivamente.
