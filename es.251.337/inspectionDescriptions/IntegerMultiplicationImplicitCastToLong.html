Reporta multiplicaciones de enteros y desplazamientos a la izquierda con conversión implícita a long.

Ejemplo:


  void f(int i) {
    long val = 65536 * i;
  }

Después de aplicar la corrección rápida, el código se convierte en:


  void x(int i) {
    long val = 65536L * i;
  }

Ejemplo:


  void f(int i) {
    long value = i << 24;
  }

Después de aplicar la corrección rápida, el código se convierte en:


  void f(int i) {
    long value = (long) i << 24;
  }

Este tipo de multiplicaciones generalmente indica un error, ya que puede ocurrir un desbordamiento accidental con truncamiento. Convertir el literal int a un literal long (65536L) corrige el problema.
