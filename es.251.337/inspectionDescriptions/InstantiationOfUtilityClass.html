Informa sobre la instanciación de clases de utilidad usando la palabra clave new.

En las clases de utilidad, todos los campos y métodos son static. La instanciación de este tipo de clases probablemente sea innecesaria e indica un error.

Ejemplo:


  class MyUtils {
    public static double cube(double x) {
      return x * x * x;
    }
  }
  class Main {
    public static void main(String[] args) {
      // Instanciación de una clase de utilidad
      MyUtils utils = new MyUtils();
    }
  }

Para evitar la instanciación de clases de utilidad, se recomienda usar un constructor private.
