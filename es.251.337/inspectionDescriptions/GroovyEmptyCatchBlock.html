Informa sobre bloques catch vacíos. Aunque a veces son intencionales, los bloques catch vacíos pueden hacer que la depuración sea extremadamente difícil.

Ejemplo:


try {
  throw new Exception()
}
catch (Exception e) {
}

Después de aplicar la corrección rápida:


try {
  throw new Exception()
}
catch (Exception ignored) {
}
