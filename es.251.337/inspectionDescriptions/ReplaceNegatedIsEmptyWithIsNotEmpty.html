Informa sobre las negaciones de isEmpty() y isNotEmpty() para colecciones y String, o isBlank() y isNotBlank() para String.

El código puede simplificarse utilizando las funciones correspondientes.

Esta corrección rápida reemplazará las llamadas negadas por las llamadas correspondientes de la biblioteca estándar.

Ejemplo:


  fun main() {
      val list = listOf(1,2,3)
      if (!list.isEmpty()) {
          // ejecutar algo
      }
  }

Después de aplicar la corrección rápida:


  fun main() {
      val list = listOf(1,2,3)
      if (list.isNotEmpty()) {
          // ejecutar algo
      }
  }
