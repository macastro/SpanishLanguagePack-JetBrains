Reporta las declaraciones import static.

Las JVM de Java 1.4 o anteriores no admiten este tipo de declaraciones import.

Configuración de la inspección:

  • Utiliza la tabla siguiente para especificar las clases que esta inspección ignorará cuando se usen en declaraciones import static.
  • Utiliza la casilla de verificación Ignorar import static de campo único para ignorar las declaraciones import static de un solo campo.
  • Utiliza la casilla de verificación Ignorar import static de método único para ignorar las declaraciones import static de un solo método.
