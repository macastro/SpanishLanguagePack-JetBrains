Reporta sentencias import static.

Dichas sentencias import no son compatibles con JVMs de Java 1.4 o anteriores.

Configurar la inspección:

  • Utilice la siguiente tabla para especificar las clases que la inspección ignorará si se utiliza en una sentencia import static.
  • Utilice la casilla de verificación Ignorar imports static para un solo campo para ignorar sentencias import static de campos únicos.
  • Utilice la casilla de verificación Ignorar imports static para un solo método para ignorar sentencias import static de métodos únicos.
