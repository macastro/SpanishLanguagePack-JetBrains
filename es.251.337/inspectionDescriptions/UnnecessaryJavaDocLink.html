Reporta referencias Javadoc con etiquetas @see, {@link} y {@linkplain} que apuntan a métodos con comentarios, métodos super con comentarios o clases que contienen comentarios.

Estos enlaces son innecesarios y pueden eliminarse de forma segura mediante la corrección rápida de esta inspección. Si la etiqueta es el único contenido, la corrección rápida eliminará todo el comentario Javadoc.

Ejemplo:


  class Example {
    /**
     * @see Example#method
     */
    public void method() { }
  }

Después de aplicar la corrección rápida:


class Example {
  public void method() { }
}

Utilice la casilla de verificación a continuación para ignorar los enlaces en línea ({@link} y {@linkplain}) que apuntan a métodos super. Aunque la herramienta Javadoc agrega automáticamente enlaces a todos los métodos super, a veces puede ser necesario tener enlaces en línea a métodos super dentro del texto de un comentario Javadoc.
