Informa sobre bucles for que pueden reemplazarse por una secuencia de operaciones stdlib (como map, filter, etc.).

Ejemplo:


fun foo(list: List<String>): List<Int> {
  val result = ArrayList<Int>()
  for (s in list) {
     if (s.length > 0)
       result.add(s.hashCode())
     }
  return result
}

Después de aplicar la corrección rápida:


fun foo(list: List<String>): List<Int> {
  val result = list
    .filter { it.length > 0 }
    .map { it.hashCode() }
  return result
}
