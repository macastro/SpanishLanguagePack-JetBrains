Informa sobre clases EJB que contienen métodos que devuelven this o lo utilizan como parámetro.

Ejemplo:


@Stateless
public class ExampleBean {
  private AnotherBean anotherBean;
  public ExampleBean setAnotherBean(AnotherBean anotherBean) {
    this.anotherBean = anotherBean;
    return this; // Error: no se debe devolver `this` aquí
  }
}
