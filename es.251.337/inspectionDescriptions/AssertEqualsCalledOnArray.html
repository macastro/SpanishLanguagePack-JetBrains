Informa sobre llamadas a JUnit assertEquals() con argumentos de tipo array. Este método compara las referencias de los arrays, no su contenido. Se debe usar el método assertArrayEquals() para verificar el contenido de los arrays.

Ejemplo:


  @Test
  public void testSort() {
    int[] actual = {248, 496, 0, 56};
    Arrays.sort(actual);
    Assert.assertEquals(new int[] {0, 56, 248, 496}, actual);
  }

Después de aplicar la corrección rápida:


  @Test
  public void testSort() {
    int[] actual = {248, 496, 0, 56};
    Arrays.sort(actual);
    Assert.assertArrayEquals(new int[] {0, 56, 248, 496}, actual);
  }
