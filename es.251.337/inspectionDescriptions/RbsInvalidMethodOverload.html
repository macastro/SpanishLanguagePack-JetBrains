Informa sobre sobrecargas de métodos inválidas (...). La sobrecarga de métodos solo es válida cuando existe exactamente una definición no sobrecargada del método.

La sobrecarga inválida puede eliminarse mediante una corrección rápida.

Ejemplo:

class Example
  # No se encuentra la declaración no sobrecargada del método 'call' para la clase 'Example'
  def call: -> untyped | ...
end

Después de aplicar la corrección rápida:

class Example
  def call: -> untyped
end
