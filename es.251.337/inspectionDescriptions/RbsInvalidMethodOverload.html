Informa sobre sobrecargas de métodos no válidas (...). Una sobrecarga de métodos solo es válida si existe exactamente otra definición no sobrecargada para el método.

Hay disponible una corrección rápida para eliminar la sobrecarga de métodos no válida.

Ejemplo:

class Example
  # No se puede encontrar una declaración no sobrecargada para el método 'call' de la clase 'Example'
  def call: -> untyped | ...
end

Tras la corrección rápida:

class Example
  def call: -> untyped
end
