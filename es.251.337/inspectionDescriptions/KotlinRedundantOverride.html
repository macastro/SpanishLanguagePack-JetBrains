Informa sobre declaraciones de sobreescritura redundantes.

La sobreescritura puede omitirse si no modifica la semántica de la firma heredada, por ejemplo, al cambiar la visibilidad.

Ejemplo:


  open class Foo {
      open fun singleExpression() {
      }
  }
  class Bar : Foo() {
      override fun singleExpression() = super.singleExpression()
  }

Después de aplicar la corrección rápida:


  class Bar : Foo() {
  }
