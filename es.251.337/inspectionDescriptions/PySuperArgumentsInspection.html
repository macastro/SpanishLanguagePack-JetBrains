Se reportan los casos cuando alguna llamada a super(A, B) no cumple los siguientes requisitos:

  • B es una instancia de A
  • B es una subclase de A

Ejemplo:


class Figure:
    def color(self):
        pass


class Rectangle(Figure):
    def color(self):
        pass


class Square(Figure):
    def color(self):
        return super(Rectangle, self).color() # Square no es una instancia o subclase de Rectangle
