Reporta nombres de receptores que son diferentes a me, this, self u otros nombres de receptores similares para este tipo.

Ejemplo:

func (self *MeterSnapshot) Rate5() float64 { return math.Float64frombits(self.rate5) }

Según Receiver Names en github.com/golang, no deberías usar nombres genéricos como "me", "this" o "self". Estos identificadores son típicos en lenguajes orientados a objetos y podrían implicar un significado especial para el método.
