Reporta referencias a variables que son comprobadas por nulidad en la condición de una declaración if o expresión condicional pero no usadas dentro de esa declaración if.

Normalmente esto significa que la comprobación es innecesaria o que la variable no es referenciada dentro de la declaración if por error.

Ejemplo:


  void test(Integer i) {
    if (i != null) { // aquí 'i' no se usa dentro de la declaración 'if'
      System.out.println();
    }
  }
