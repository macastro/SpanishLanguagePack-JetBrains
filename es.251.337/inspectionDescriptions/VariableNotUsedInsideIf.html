Informa sobre referencias a variables que se comprueban por nulidad en condiciones de declaraciones if o expresiones condicionales, pero que no se utilizan dentro de dicha declaración if.

Esto generalmente indica que la comprobación es innecesaria o que hay una referencia incorrecta a la variable dentro de la declaración if.

Ejemplo:


  void test(Integer i) {
    if (i != null) { // Aquí 'i' no se utiliza dentro de la declaración 'if'
      System.out.println();
    }
  }
