Informa sobre funciones que no tienen una sentencia RETURN.

Ejemplo (Oracle):

CREATE FUNCTION foo RETURN int AS
BEGIN
END;

La función foo debe devolver un valor entero, pero el cuerpo de la función no devuelve nada. Para corregir el error, añada una sentencia RETURN (por ejemplo, RETURN 1;).

CREATE FUNCTION foo RETURN int AS
BEGIN
 RETURN 1;
END;
