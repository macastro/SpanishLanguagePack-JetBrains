Reporta llamadas a métodos de fábrica de MethodHandle y VarHandle que no coinciden con ningún método o campo.

También reporta argumentos que no coinciden con la firma de MethodHandle en MethodHandle.invoke() y métodos similares, así como argumentos de VarHandle.set() que no coinciden con el tipo de VarHandle

Ejemplo:

  MethodHandle mh = MethodHandles.lookup().findVirtual(
      MyClass.class, "foo", MethodType.methodType(void.class, int.class));
  // Este argumento debería ser un valor int
  mh.invoke(myObj, "abc");


  // Este argumento debería ser String.class
  VarHandle vh = MethodHandles.lookup().findVarHandle(
      MyClass.class, "text", int.class);


  VarHandle vh = MethodHandles.lookup().findVarHandle(
      MyClass.class, "text", String.class);
  // Este argumento debería ser un valor String
  vh.set(myObj, 42);
