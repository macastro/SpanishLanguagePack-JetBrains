Informa sobre incompatibilidades con la versión especificada de Python. Si necesitas que tu código sea compatible con múltiples versiones de Python (por ejemplo, al desarrollar una biblioteca), puedes activar esta comprobación.

Para definir el rango de versiones de Python a verificar, selecciona las casillas correspondientes en la sección de Opciones.
