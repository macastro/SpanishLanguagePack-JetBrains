Informa los parámetros de anotación que están asignados a su valor predeterminado default.

Ejemplo:


  @interface Test {
      Class<?> expected() default Throwable.class;
  }

  @Test(expected = Throwable.class)
  void testSmth() {}

Después de aplicar la corrección rápida:


  @Test()
  void testSmth() {}
