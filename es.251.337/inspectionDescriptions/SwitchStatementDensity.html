Informa sobre declaraciones o expresiones switch que tienen una proporción demasiado baja entre etiquetas switch y declaraciones ejecutables.

Estas declaraciones switch pueden resultar confusas y podrían requerir una refactorización.

Ejemplo:


    switch (i) { // 1 case, 5 declaraciones ejecutables -> 20% de densidad
        case 1:
            System.out.println("1");
            System.out.println("2");
            System.out.println("3");
            System.out.println("4");
            System.out.println("5");
            break;
    }

Utilice el campo Densidad mínima de ramas para especificar la proporción permitida entre etiquetas switch y declaraciones ejecutables.
