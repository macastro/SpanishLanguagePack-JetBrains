Informa sobre asignaciones o modificaciones del parámetro del bucle en el cuerpo de un bucle for.

Aunque a veces es intencional, esta estructura puede ser confusa y generalmente es el resultado de errores tipográficos o el uso de variables incorrectas.

La corrección rápida añade una declaración para la nueva variable.

Ejemplo:


  for (String s : list) {
    // Advertencia: s se modifica dentro del bucle
    s = s.trim();
    System.out.println("String: " + s);
  }

Después de aplicar la corrección rápida:


  for (String s : list) {
    String trimmed = s.trim();
    System.out.println("String: " + trimmed);
  }

No se informan las asignaciones en bucles for básicos sin declaración de actualización. En estos casos, la asignación es probablemente intencionada y no se puede mover fácilmente a la parte de actualización del bucle for.

Ejemplo:


  for (int i = 0; i < list.size(); ) {
    if (element.equals(list.get(i))) {
      list.remove(i);
    } else {
      // No se informa la modificación del parámetro del bucle for
      // porque no hay declaración de actualización
      i++;
    }
  }

Utilice la opción Verificar parámetros de bucle 'for' mejorado para especificar si también se deben informar las modificaciones de los parámetros en bucles for mejorados.
