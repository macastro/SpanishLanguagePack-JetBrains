Detecta casos de prueba (Test Cases) en ejecuciones de pruebas (Test Run) que podrían marcarse como "automatizados"

Un caso de prueba que tenga autotest podría marcarse con el estado "automatizado" y no requiere verificación manual.

Se informa una advertencia si hay un caso de prueba con el estado "desconocido" en una ejecución de prueba y hay una referencia a este caso de prueba en el código. De ser así, este caso de prueba podría considerarse como automatizado.
