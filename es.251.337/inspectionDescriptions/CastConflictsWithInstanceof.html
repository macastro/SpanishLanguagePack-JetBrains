El informe contiene expresiones de conversión de tipo que están precedidas por comprobaciones instanceof de diferentes tipos.

Aunque esto podría ser intencional, tales estructuras probablemente sean erróneas y resulten en una java.lang.ClassCastException durante la ejecución.

Ejemplo:


  class Main {
    int whenCharSequenceCastToNumber(Object o){
      if (o instanceof CharSequence) {
        return ((Number) o).intValue();
      }
      return 0;
    }
    int earlyReturnWhenNotCharSequence(Object o){
      if (!(o instanceof CharSequence)) return 0;
      return ((Number)o).intValue();
    }
  }
