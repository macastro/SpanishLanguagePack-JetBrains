Informa sobre llamadas a list.remove(index) en bucles de conteo ascendente.

Esto es sospechoso porque después de eliminar un elemento, la lista se acorta y se salta el siguiente elemento. Una solución simple es decrementar la variable del índice después de la eliminación, pero un enfoque más robusto sería eliminar usando un iterador o el método removeIf() (Java 8 y versiones posteriores). Si no desea que remove() se llame múltiples veces en el bucle, considere agregar un break después.

Ejemplo:

  public static void main(String[] args) {
    process(new ArrayList<>(
      Arrays.asList("1", "2", "|", "3", "4")));
  }
  static void process(List<String> list) {
    for (int i = 0; i < list.size(); i++) {
      if (list.get(i).equals("|")) {
        list.remove(i);
        continue;
      }
      System.out.println(list.get(i));
    }
  }

El código parece que debería imprimir 1 2 3 4, pero en realidad saltará 3 en la salida.
