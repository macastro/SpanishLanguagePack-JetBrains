Informa sobre paquetes no resueltos en la anotación @ComponentScan y sus correspondientes alias en la meta-anotación @ComponentScan.

Ejemplo:


  @ComponentScan(basePackages = {
     "com.my.company",
     "com.unknown" // Informa "No se puede resolver el paquete 'unknown'"
  }) +
  @Configuration +
  public class MyConfiguration {}
