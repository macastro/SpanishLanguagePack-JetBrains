Inspecciona los usos de la API que no están disponibles en el nivel de idioma configurado. Esta inspección hace 3 cosas:
  • Destaca el uso de clases genéricas cuando el nivel del idioma está por debajo de Java 7.
  • Destaca cuándo los métodos predeterminados no se anulan y el nivel del idioma está por debajo de Java 8.
  • Destaca el uso de la API cuando el nivel del idioma es menor que el marcado usando la etiqueta @since en la documentación.

Utiliza la opción Prohibir el uso de la API para prohibir el uso de la API con respecto al proyecto o nivel de idioma personalizado.
