Informa sobre el uso de API no disponible en el nivel de lenguaje configurado. Esta inspección tiene 3 propósitos:
  • Resalta el uso de clases genéricas cuando el nivel de lenguaje es inferior a Java 7.
  • Resalta cuando los métodos por defecto no están sobrescritos y el nivel de lenguaje es inferior a Java 8.
  • Resalta el uso de API cuando el nivel de lenguaje es inferior al nivel marcado con la etiqueta @since en la documentación.

Utilice la opción API prohibida para prohibir el uso de API en el proyecto o en un nivel de lenguaje personalizado.
