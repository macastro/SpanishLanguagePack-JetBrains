La declaración de variable puede moverse a la expresión when.

Ejemplo:


fun someCalc(x: Int) = x * 42
fun foo(x: Int): Int {
  val a = someCalc(x)
  return when (a) {
    1 -> a
    2 -> 2 * a
    else -> 24
  }
}

Después de aplicar la corrección rápida:


fun foo(x: Int): Int {
  return when (val a = someCalc(x)) {
    1 -> a
    2 -> 2 * a
    else -> 24
  }
}
