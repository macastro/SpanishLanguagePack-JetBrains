Informa sobre declaraciones break con etiquetas innecesarias. Estas etiquetas no alteran el flujo de control pero dificultan el seguimiento del código.

Ejemplo:


  label:
  for(int i = 0; i < 10; i++) {
    if (shouldBreak()) break label;
    //doSmth
  }

Después de aplicar la corrección rápida:


  label:
  for(int i = 0; i < 10; i++) {
    if (shouldBreak()) break;
    //doSmth
  }
