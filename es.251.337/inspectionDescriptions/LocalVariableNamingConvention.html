Informa variables locales cuyos nombres son demasiado cortos, demasiado largos o no siguen el patrón especificado de expresión regular.

Ejemplo: int X = 42; ese indicaría que se debe informar si la inspección se habilita con los ajustes predeterminados en los que un nombre de variable debe comenzar con una letra minúscula.

Configurar la inspección:

Utiliza los campos en la sección de Opciones para especificar la longitud mínima, la longitud máxima y una expresión regular que se espera para los nombres de variables locales. Especifica 0 para que no se verifique la longitud de los nombres. Las expresiones regulares deben especificarse en el formato estándar de java.util.regex.

Utiliza las casillas de verificación para ignorar los parámetros de secciones de for-loop y catch.
