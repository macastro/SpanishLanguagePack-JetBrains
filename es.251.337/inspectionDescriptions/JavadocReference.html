Informa sobre referencias no resueltas en comentarios de Javadoc.

En el siguiente ejemplo, falta el parámetro someParam, por lo que se resaltará:


class A {
    /**
     * @param someParam descripción
     **/
    void foo() {
    }
}

Deshabilita la opción Informar símbolos inaccesibles para ignorar las etiquetas que hacen referencia a parámetros de métodos, clases, campos y métodos no encontrados.
