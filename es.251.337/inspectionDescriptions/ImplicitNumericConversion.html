Informa sobre conversiones implícitas entre tipos numéricos.

Las conversiones numéricas implícitas no son un problema en sí mismas, pero pueden dificultar el seguimiento de errores si son accidentales.

Ejemplo:


  double m(int i) {
    return i * 10;
  }

Después de aplicar la corrección rápida:


  double m(int i) {
    return (double) (i * 10);
  }

Configuración de la inspección:

  • Use la opción Ignorar conversiones de ampliación para ignorar las conversiones implícitas que no resultan en pérdida de datos (por ejemplo, int->long).
  • Use la opción Ignorar conversiones entre 'char' y otros tipos para ignorar las conversiones entre char y otros tipos. La inspección seguirá informando sobre conversiones entre números de punto flotante y otros tipos.
  • Use Ignorar conversiones desde constantes y literales para que la inspección ignore las conversiones desde literales y constantes en tiempo de compilación.
