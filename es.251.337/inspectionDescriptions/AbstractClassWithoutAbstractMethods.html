Informa sobre clases abstract que no tienen ningún método abstract. En la mayoría de los casos, una clase abstract sin métodos abstract no tiene sentido y el modificador abstract puede eliminarse de la clase. Si una clase se declara como abstract para prevenir su instanciación, generalmente es mejor usar un constructor private para evitar la instanciación.

Ejemplo:


  abstract class Example {
    public String getName() {
      return "IntelliJ IDEA";
    }
  }

Use esta opción para ignorar las clases de utilidad.
