La llamada a String.format puede ser reemplazada por una plantilla de cadena.

El uso de plantillas de cadena hace que el código sea más simple.

Esta corrección rápida reemplazará la llamada con una plantilla de cadena.

Ejemplo:


  fun main() {
      val id = "abc"
      val date = "123"
      val s = String.format("%s_%s_%s", id, date, id)
  }

Después de aplicar la corrección rápida:


  fun main() {
      val id = "abc"
      val date = "123"
      val s = "${id}_${date}_$id"
  }
