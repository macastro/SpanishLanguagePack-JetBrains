Reporta llamadas String.format que se pueden reemplazar con plantillas de cadenas.

El uso de plantillas de cadenas hace que tu código sea más sencillo.

La solución rápida reemplaza la llamada con una plantilla de cadena.

Ejemplo:


  fun main() {
      val id = "abc"
      val date = "123"
      val s = String.format("%s_%s_%s", id, date, id)
  }

Después de aplicar la solución rápida:


  fun main() {
      val id = "abc"
      val date = "123"
      val s = "${id}_${date}_$id"
  }
