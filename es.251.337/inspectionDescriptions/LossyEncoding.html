Reportar caracteres que no se pueden mostrar debido a la codificación actual del documento.

Ejemplos:

  • Si se escriben caracteres internacionales en un documento con conjunto de caracteres US-ASCII, algunos caracteres se perderán al guardar.
  • Si se carga un archivo codificado en UTF-8 con un conjunto de caracteres de un byte ISO-8859-1, algunos caracteres no se mostrarán correctamente.

Para resolver este problema, puede cambiar la codificación del archivo especificándola directamente en el archivo, por ejemplo, editando el atributo encoding= en el prólogo XML para archivos XML, o cambiando las opciones correspondientes en Configuración | Editor | Codificación de archivos.
