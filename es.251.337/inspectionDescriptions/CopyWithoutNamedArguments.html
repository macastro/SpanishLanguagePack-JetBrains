Informa sobre llamadas al método copy() de clases de datos que no tienen argumentos nombrados.

Dado que todos los argumentos de la función copy() son opcionales, puede ser difícil entender qué propiedades se están modificando. Proporcionar nombres de parámetros explícitos hace que el código sea más fácil de entender sin necesidad de navegar a la declaración de la data class.

Ejemplo:


  data class User(val name: String, val age: Int)
  fun copyUser(user: User): User {
      return user.copy("John")
  }

La corrección rápida proporcionará nombres de parámetros para todos los argumentos de copy():


  data class User(val name: String, val age: Int)
  fun copyUser(user: User): User {
      return user.copy(name = "John")
  }
