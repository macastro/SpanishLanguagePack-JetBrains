Informa sobre la concatenación de String utilizada como argumento para StringBuffer.append(), StringBuilder.append() o Appendable.append().

Tales llamadas pueden transformarse ventajosamente en llamadas encadenadas de append en el StringBuffer/Builder/Appendable existente, ahorrando así el costo de asignación adicional de StringBuffer/Builder. Esta inspección ignorará las concatenaciones de String que se evalúan en tiempo de compilación, casos en los que la conversión solo reduciría el rendimiento.

Ejemplo:


  void bar(StringBuilder builder, String name) {
    builder.append("Hello," + name);    //advertencia
    builder.append("Hello," + "world"); //sin advertencia
  }
