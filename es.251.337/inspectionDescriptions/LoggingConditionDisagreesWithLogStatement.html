Informe if con la condición is log enabled for que no coincide con el nivel de registro de las llamadas al registro incluidas.

Ejemplo: 


  if (LOG.isTraceEnabled()) {
    // El nivel de depuración se registra, pero se comprueba el nivel de seguimiento
    LOG.debug("some log message");
  }

Esta inspección entiende los marcos de registro java.util.logging, log4j, Log4j 2, Apache Commons Logging y SLF4J.
