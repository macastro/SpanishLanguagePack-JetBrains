Informa sobre bucles while que iteran sobre colecciones y pueden ser reemplazados por bucles for mejorados (sintaxis de iteraci√≥n foreach).

Ejemplo:


  Iterator it = c.iterator();
  while(it.hasNext()) {
    Object obj = it.next();
    System.out.println(obj);
  }

Puede ser reemplazado por:


  for (Object obj : c) {
    System.out.println(obj);
  }
