Informa sobre las llamadas a Thread.yield().

El comportamiento de yield() es indeterminado y depende de la plataforma, por lo que rara vez es apropiado usar este método. Su uso debe combinarse con análisis detallados y pruebas de rendimiento para asegurar que realmente se logre el efecto deseado.

Ejemplo:


  public static void main(String[] args) {
    Runnable r = () -> {
      for (int i = 0; i < 10; i++) {
        System.out.println(i);
        Thread.yield();
      }
    };
    new Thread(r).start();
    new Thread(r).start();
  }
