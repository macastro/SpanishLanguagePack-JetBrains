Informa sobre campos en clases anónimas que tienen el mismo nombre que variables locales o parámetros del método contenedor o expresión lambda.

Este tipo de nomenclatura puede llevar a un uso incorrecto del campo de la clase anónima cuando se intenta usar una variable o parámetro con el mismo nombre en el método contenedor.

Se recomienda renombrar el campo usando la corrección rápida.

Ejemplo:


  class Test {
    public Test(String value) {
      Object foo = new Object() {
        private String value = "TEST";
        public void foo() {
          System.out.println(value); //accede al campo, no al parámetro
        }
      };
    }
  }
