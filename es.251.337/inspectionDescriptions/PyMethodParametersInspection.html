Informa sobre métodos que carecen del primer parámetro normalmente denominado self.

Ejemplo:


class Movie:
   def show():
       pass

Después de aplicar la corrección rápida, el código se convierte en:


class Movie:
   def show(self):
       pass

La inspección también informa sobre problemas de nomenclatura en los métodos de clase.

Ejemplo:


class Movie:
    @classmethod
    def show(abc):
        pass

Dado que el primer parámetro de un método de clase debe ser cls, el IDE proporciona una corrección rápida para renombrarlo.
