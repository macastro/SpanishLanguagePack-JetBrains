Informa sobre tablas derivadas sin alias.

Ejemplo (MySQL):

CREATE TABLE table1 (id INT, name VARCHAR(20), cats FLOAT);
CREATE TABLE table2 (id INT, age INTEGER);
SELECT id AS ID, name, cats, age
FROM (SELECT table1.id, name, cats, age
FROM table1
JOIN table2 ON table1.id = table2.id);

Según las Tablas Derivadas en dev.mysql.com, los alias son obligatorios. Puede utilizar la corrección rápida Introducir alias para añadir el alias.

Después de aplicar la corrección rápida:

SELECT id AS ID, name, cats, age
FROM (SELECT table1.id, name, cats, age
FROM table1
JOIN table2 ON table1.id = table2.id);
