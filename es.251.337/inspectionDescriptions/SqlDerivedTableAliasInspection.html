Informa de tablas derivadas sin alias.

Ejemplo (MySQL):

CREATE TABLE table1 (id INT, name VARCHAR(20), cats FLOAT);
CREATE TABLE table2 (id INT, age INTEGER);

SELECT id AS ID, name, cats, age
FROM (SELECT table1.id, name, cats, age
FROM table1
JOIN table2 ON table1.id = table2.id);

Según Tablas derivadas en dev.mysql.com, el alias es obligatorio. Puede añadir el alias usando el quick-fix Introducir alias.

Después de aplicar el quick-fix:

SELECT id AS ID, name, cats, age
FROM (SELECT table1.id, name, cats, age
FROM table1
JOIN table2 ON table1.id = table2.id);
