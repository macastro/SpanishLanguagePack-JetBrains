El informe puede reemplazarse por una operación a nivel de bits de la llamada al método Long.hashCode() o Double.hashCode(). Detectará la construcción (int)(x ^ (x >>> 32)), donde x es una variable de tipo long, o el resultado de una llamada Double.doubleToLongBits() anterior. Este reemplazo puede acortar el código y mejorar su legibilidad.

Ejemplo:


  int result = (int)(var ^ (var >>> 32));

Después de aplicar la corrección rápida:


  int result = Long.hashCode(var);

Esta inspección solo informará si el nivel de lenguaje del proyecto o módulo es 8 o superior.
