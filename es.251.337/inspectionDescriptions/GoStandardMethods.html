Reportar métodos con nombres específicos cuando:
  • El nombre del método coincide con el nombre de varios métodos de interfaz conocidos en la biblioteca estándar
  • La firma no coincide con la firma del método de interfaz correspondiente

Estos métodos pueden indicar que el tipo receptor pretende satisfacer una interfaz de la biblioteca estándar, pero no puede hacerlo debido a errores en la firma del método.

Ejemplo:

type MyReader []byte
func (r MyReader) ReadByte(data []byte) (byte, error) {
}

Este uso es sospechoso porque parece intentar implementar io.ByteReader, pero la firma es incorrecta. La forma correcta sería:

type MyReader []byte
func (r MyReader) ReadByte() (byte, error) {
}
