Informa de propiedades que están asignadas explícitamente a los parámetros primarios del constructor.

Las propiedades pueden ser declaradas directamente en el constructor primario, reduciendo la cantidad de código y mejorando la legibilidad del mismo.

Ejemplo:


  class User(nombre: String) {
      val nombre = nombre
  }

La corrección rápida une la declaración del parámetro y la propiedad en un parámetro del constructor primario:


  class User(val nombre: String) {
  }
