Reporta los métodos __toString que no devuelven string.

Antes de PHP 7.4, esto provocaba un error fatal de nivel E_RECOVERABLE_ERROR. En PHP 7.4, se lanza un error fatal irrecuperable ERROR. A partir de PHP 8.0, el valor devuelto sigue la semántica estándar de tipos de PHP, realizando una conversión a cadena cuando es posible y el modo estricto de tipos está desactivado.

Consulte __toString (php.net) para más detalles.
