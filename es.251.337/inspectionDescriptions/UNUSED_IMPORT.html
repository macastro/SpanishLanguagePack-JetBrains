Reporta declaraciones import redundantes.

Las declaraciones regulares de import no son necesarias si las clases y paquetes importados no se utilizan en el archivo fuente. Esto también se aplica a los campos y métodos static que están importados pero no se utilizan en el archivo fuente.

Ejemplo:


  import java.util.ArrayList;
  public class Example {
    public static void main(String[] args) {
      System.out.println("Hello World!");
    }
  }

Después de aplicar la corrección rápida: 


  public class Example {
    public static void main(String[] args) {
      System.out.println("Hello World!");
    }
  }
