Informa sobre conversiones seguras con return que pueden ser reemplazadas por comprobaciones de tipo if.

El código se puede simplificar usando la función correspondiente.

Esta corrección rápida reemplazará la conversión segura por una comprobación de tipo if.

Ejemplo:


  fun test(x: Any) {
      x as? String ?: return
  }

Después de aplicar la corrección rápida:


  fun test(x: Any) {
      if (x !is String) return
  }
