**Español** Informa de conversiones de tipo que pueden omitirse.

Ejemplo:

var s = string("hola")

El valor "hola" ya es del tipo string, la conversión adicional a string es redundante. Para eliminar la conversión, considera utilizar el quick-fix Eliminar conversión.

Una vez aplicado el quick-fix:

var s = "hola"

A veces la conversión de una expresión con coma flotante a un tipo con coma flotante puede ser intencionada (para consultar un ejemplo, consulta este problema). En estos casos, el IDE emite una advertencia sobre una conversión posiblemente redundante.
