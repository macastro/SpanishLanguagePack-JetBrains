Reporta referencias no resueltas a cualquier objeto de base de datos dentro de los archivos de configuración `orm.xml`:
  • Tablas
  • Columnas
  • Esquemas
  • Catálogos
  • Secuencias

Ejemplo de tabla de base de datos:



CREATE TABLE DATA_TABLE (
  ID INTEGER,
  DATA VARCHAR(100)
);

Ejemplo de archivo orm.xml:



<entity-mappings xmlns="http://java.sun.com/xml/ns/persistence/orm" version="1.0">
    <package>ejemplo</package>
    <entity name="EntidadJava" class="EntidadJava">
        <table name="DATA_TABLE"/>
        <attributes>
          <id name="id">
            <column name="ID_DESCONOCIDO" nullable="false" length="20"/> // Error: nombre de columna no resuelto
          </id>
          <basic name="datos">
            <column name="DATOS" length="12"/> // Correcto
          </basic>
        </attributes>
    </entity>
</entity-mappings>
