Reporta expresiones when que tienen ramas constantes con true o false.

Simplificar "when" está disponible como corrección rápida para arreglar el código automáticamente.

Ejemplo:


  fun redundant() {
      when { // <== Redundante, la corrección rápida simplificará la expresión when a "println("true")"
          true -> println("true")
          else -> println("false")
      }
  }
