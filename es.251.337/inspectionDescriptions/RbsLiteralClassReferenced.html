Informa sobre referencias a las clases NilClass, TrueClass y FalseClass en firmas de tipos. El uso de estas clases es innecesariamente detallado.

El uso de NilClass, TrueClass o FalseClass puede ser reemplazado por sus literales respectivos mediante una corrección rápida.

Ejemplo:

# Referencia a 'NilClass' en lugar de 'nil', ...
type example = NilClass | TrueClass | FalseClass

Después de aplicar la corrección rápida:

type example = nil | true | false
