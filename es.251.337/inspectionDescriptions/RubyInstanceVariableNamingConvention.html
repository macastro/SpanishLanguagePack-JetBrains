Reporta variables de instancia con nombres que son demasiado cortos, demasiado largos o que no coinciden con el patrón de expresión regular especificado.

Usa el campo Patrón para especificar una expresión regular para validar los nombres de las variables de instancia.

Usa los campos Longitud mínima y Longitud máxima para especificar la longitud mínima y máxima de un nombre respectivamente.

Inspirado por 'Ruby Style Guide'
