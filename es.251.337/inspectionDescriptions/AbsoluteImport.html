Informa de importaciones totalmente calificadas que pueden ser relativas al paquete base (Configuración | Idiomas y frameworks | Scala | Paquetes base).

Solo es aplicable cuando la opción Excepto para el paquete base está habilitada en Configuración | Editor | Estilo de código | Scala | Importaciones.

Ejemplo:


  package org.example.application
  package ui

  import org.example.application.model.Data

Después de aplicar la solución rápida:


  package org.example.application
  package ui

  import model.Data
