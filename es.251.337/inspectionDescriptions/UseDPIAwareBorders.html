Informa sobre usos simplificables de javax.swing.border.EmptyBorder y JBUI.Borders.emptyXyz().

Las instancias de EmptyBorder no son conscientes del DPI y pueden causar problemas de diseño en la interfaz de usuario.

La corrección rápida realizará el reemplazo o simplificará la expresión usando JBUI.Borders.empty().

Ejemplos:


// Incorrecto:
Border border1 = new EmptyBorder(1, 2, 3, 4);
Border border2 = new EmptyBorder(1, 2, 1, 2);
Border border3 = new EmptyBorder(1, 0, 0, 0);
// Correcto:
Border border1 = JBUI.Borders.empty(1, 2, 3, 4);
Border border2 = JBUI.Borders.empty(1, 2);
Border border3 = JBUI.Borders.emptyTop(1);
