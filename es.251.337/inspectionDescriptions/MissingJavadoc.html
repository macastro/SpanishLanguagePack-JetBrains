Informa sobre comentarios y etiquetas Javadoc faltantes.

Ejemplo:


  /**
   * Informa sobre "@param" faltante (si está configurado).
   */
  public void sample(int param){
  }

La corrección rápida añadirá las etiquetas faltantes o los comentarios Javadoc faltantes:


  /**
   * Informa sobre "@param" faltante (si está configurado).
   * @param param
   */
  public void sample(int param){
  }

La inspección se puede configurar para ignorar elementos obsoletos o métodos de acceso simples como getField() o setField().

También puede configurar las etiquetas requeridas y la visibilidad mínima requerida para elementos de código específicos como métodos, campos, clases, paquetes, módulos, etc.
