Informa sobre expresiones aritméticas con demasiados operandos. Tales expresiones pueden ser difíciles de entender y propensas a errores.

Los parámetros, referencias a campos y otras expresiones primarias cuentan como un operando cada uno.

Ejemplo: 

int calc(int a, int b) {
        return a + a + a + b + b + b + b; // Esta línea contiene 7 operandos y será reportada.
    }

Utilice el campo siguiente para especificar el número permitido de operandos en una expresión aritmética.
