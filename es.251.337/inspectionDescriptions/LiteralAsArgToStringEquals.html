Reporta las llamadas String.equals() o String.equalsIgnoreCase() con un argumento de literal de cadena.

Algunos estándares de codificación especifican que los literales de cadena deberían ser el calificador de equals(), en lugar de argumento, minimizando así NullPointerException-s.

Hay disponible una corrección rápida para intercambiar el literal y la expresión.

Ejemplo:


  boolean isFoo(String value) {
    return value.equals("foo");
  }

Después de aplicar la corrección rápida:


  boolean isFoo(String value) {
    return "foo".equals(value);
  }
