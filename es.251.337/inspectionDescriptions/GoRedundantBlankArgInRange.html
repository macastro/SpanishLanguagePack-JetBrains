Informa sobre variables en blanco opcionales en bucles range.

Cuando se itera sobre un slice usando un bucle range, se devuelven dos valores para cada iteración. El primer valor es el número de índice, y el segundo valor es una copia del elemento en ese índice. Si no necesita el segundo valor, puede omitirlo usando un identificador en blanco.

Ejemplo:

for a, _ = range v {} // `for a, _ =` es igual que `for a =`

Para eliminar el identificador en blanco, puede usar la corrección rápida Eliminar argumento en blanco. Después de aplicar la corrección rápida, el código se verá así:

for a = range v {}
