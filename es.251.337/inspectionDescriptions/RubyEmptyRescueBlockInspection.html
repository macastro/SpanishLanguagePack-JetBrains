Reporta bloques rescue vacíos. Aunque ocasionalmente son intencionales, los bloques rescue vacíos generalmente indican errores de codificación y pueden dificultar la depuración.

Utiliza la casilla de verificación Ignorar bloques 'rescue' con comentarios para definir si se deben contar los bloques rescue que solo contienen comentarios.

Utiliza la casilla de verificación Ignorar parámetros 'rescue' nombrados como '_' para definir si se deben contar los bloques rescue con parámetros nombrados como _.
