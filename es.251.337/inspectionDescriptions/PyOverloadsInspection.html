Informa sobre casos en archivos Python regulares donde las sobrecargas se colocan después de la implementación o donde sus firmas son incompatibles con la implementación.

Ejemplo:


from typing import overload
@overload
def foo(p1, p2): # firma de sobrecarga incompatible con la implementación
    pass
@overload
def foo(p1): # firma de sobrecarga incompatible con la implementación
    pass
def foo(p1, p2, p3):
    print(p1, p2, p3)
