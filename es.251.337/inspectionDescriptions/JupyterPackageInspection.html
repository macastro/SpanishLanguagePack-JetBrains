Informa cuando el paquete de jupyter no está instalado en el intérprete de Python seleccionado Sin un paquete de jupyter correctamente instalado, no puedes ejecutar notebooks Jupyter.

Haz clic en el enlace correspondiente en el banner de advertencia para instalar el paquete que falta. También puedes instalar el paquete en las Preferencias/Configuraciones del proyecto o en la Herramienta de ventana de los Paquetes Python.

Consulta Instalación del paquete Python para obtener más información.

Esta inspección advertirá cuando el paquete jupyter no esté instalado cuando se abra el archivo del notebook. 이 검사는 jupyter 패키지가 설치되어 있지 않을 때 Notebook 파일이 열리면 경고합니다. 如果不安装 jupyter 软件包，则在打开 Notebook 文件时，此检查会发出警告。
