Informa sobre las diferentes formas de comprobar si las colecciones y las Options están vacías y sugiere reemplazarlas por .isEmpty、.nonEmpty o .isDefined.

Ejemplo:


  !Seq(1).isEmpty
  Seq(1).size == 0
  Seq(1).size != 0
  !(Seq(1).length > 0)
  Seq(1).exists(_ => true)
  import scala.Function.const
  Seq(1).exists(const(true))
  !Option(0).isEmpty
  Option(0) == None
  Option(0) != None

Después de aplicar la solución rápida:


  Seq(1).nonEmpty
  Seq(1).isEmpty
  Seq(1).nonEmpty
  Seq(1).isEmpty
  Seq(1).nonEmpty
  import scala.Function.const
  Seq(1).nonEmpty
  Option(0).isDefined
  Option(0).isEmpty
  Option(0).isDefined
