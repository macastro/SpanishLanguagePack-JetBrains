La propiedad data no es una función del componente Vue. Se recomienda envolver el objeto literal en una función.

Al definir un componente, data debe declararse como una función que devuelve el objeto de datos inicial, ya que se crearán múltiples instancias usando la misma definición. Si data sigue siendo un objeto simple, ¡ese objeto se compartirá por referencia en todas las instancias creadas! Con una función data, cada vez que se crea una nueva instancia, se puede llamar directamente para devolver una nueva copia de los datos iniciales.
