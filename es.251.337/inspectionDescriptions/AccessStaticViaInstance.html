Informa sobre referencias a métodos y campos static a través de instancias de clase en lugar de la clase misma.

Aunque la especificación del lenguaje Java permite referenciar miembros static mediante variables de instancia, esto puede confundir a los lectores, quienes podrían pensar erróneamente que el método depende de la instancia.

Esta corrección rápida reemplazará la variable de instancia por el nombre de la clase.

Ejemplo:


  String s1 = s.valueOf(0);

Después de aplicar la corrección rápida:


  String s = String.valueOf(0);
