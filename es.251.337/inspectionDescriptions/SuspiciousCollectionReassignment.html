Informa de expresiones de asignación aumentada (+=) en una Collection de solo lectura.

La expresión de asignación aumentada (+=) en una Collection de solo lectura asigna temporalmente una nueva colección, lo que puede perjudicar el rendimiento.

El quickfix "Cambiar tipo a mutable" se puede usar para modificar el código automáticamente.

Ejemplo:


  fun test() {
      var list = listOf(0)
      list += 42 // Aquí se asigna una nueva lista, equivalente a list = list + 42
  }

Después de aplicar el quickfix:


  fun test() {
      val list = mutableListOf(0)
      list += 42
  }
