Informa sobre expresiones de asignación aumentada (+=) en Collection de solo lectura.

Las expresiones de asignación aumentada (+=) en Collection de solo lectura asignan temporalmente una nueva colección, lo que puede afectar al rendimiento.

La corrección rápida Cambiar tipo a mutable está disponible para corregir el código automáticamente.

Ejemplo:


  fun test() {
      var list = listOf(0)
      list += 42 // Aquí se asigna una nueva lista, equivalente a list = list + 42
  }

Después de aplicar la corrección rápida:


  fun test() {
      val list = mutableListOf(0)
      list += 42
  }
