Se informa sobre conversiones de tipo que pueden eliminarse si se reduce el alcance del tipo de variable al tipo de conversión.

Ejemplo:


  Object x = "  string  ";
  System.out.println(((String)x).trim());

En este caso, cambiar el tipo de x a String hace que la conversión sea redundante. La corrección rápida sugerida actualizará el tipo de variable y eliminará todas las conversiones redundantes en esa variable: 


  String x = "  string  ";
  System.out.println(x.trim());
