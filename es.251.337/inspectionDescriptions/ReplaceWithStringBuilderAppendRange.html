Informa sobre las llamadas a la función StringBuilder.append(CharArray, offset, len) que deberían ser reemplazadas por llamadas a StringBuilder.appendRange(CharArray, startIndex, endIndex) en la plataforma JVM.

La función append se comporta de manera diferente en las plataformas JVM, JS y Native, por lo que se recomienda usar la función appendRange.

Ejemplo:


  fun f(charArray: CharArray, offset: Int, len: Int): String {
      return buildString {
          append(charArray, offset, len)
      }
  }

Después de aplicar la corrección rápida:


  fun f(charArray: CharArray, offset: Int, len: Int): String {
      return buildString {
          appendRange(charArray, offset, offset + len)
      }
  }
