Reporta tipos de parámetros de método del repositorio incorrectos.

Ejemplo:


  public class Person {
    private int id;
    @Column
    private String lastname;

    @Column
    private int age;
  }

  public interface CustomerRepository extends Repository<Person, Integer> {
    List<Person> findByAddress(int lastname);  // Debe ser de tipo 'String'
    List<Person> deleteByAge(String str);   // Debe ser de tipo 'int'
  }
