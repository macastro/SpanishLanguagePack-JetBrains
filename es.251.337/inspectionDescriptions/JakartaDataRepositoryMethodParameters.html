Informa sobre tipos de parámetros incorrectos en los métodos del repositorio.

Ejemplo:


  public class Person {
    private int id;
    @Column
    private String lastname;
    @Column
    private int age;
  }
  public interface CustomerRepository extends Repository<Person, Integer> {
    List<Person> findByAddress(int lastname);  // debería ser tipo 'String'
    List<Person> deleteByAge(String str);   // debería ser tipo 'int'
  }
