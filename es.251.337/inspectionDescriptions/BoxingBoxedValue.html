Informa sobre el boxing redundante de valores ya boxed.

Esta es una operación redundante, ya que cualquier valor boxed primero se desempaquetará automáticamente y luego se volverá a boxear. Este tipo de código puede causar problemas de rendimiento si se realiza dentro de bucles internos.

Ejemplo:


  Integer value = 1;
  method(Integer.valueOf(value));

Después de aplicar la corrección rápida:


  Integer value = 1;
  method(value);
