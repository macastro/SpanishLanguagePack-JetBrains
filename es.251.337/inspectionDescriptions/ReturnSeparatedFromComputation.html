Informa sobre las declaraciones return que devuelven variables locales cuyos valores se calculan en otro lugar dentro del mismo método.

La corrección rápida permite integrar la variable devuelta moviendo la declaración return al lugar donde se calcula el valor de la variable. Cuando el valor devuelto no se puede integrar en la declaración return, la corrección rápida intenta mover la declaración return lo más cerca posible de donde se calcula el valor devuelto.

Ejemplo:


  int n = -1;
  for (int i = 0; i < a.length; i++) {
    if (a[i] == b) {
      n = i;
      break;
    }
  }
  return n;

Después de aplicar la corrección rápida:


  int n = -1;
  for (int i = 0; i < a.length; i++) {
    if (a[i] == b) {
      return i;
    }
  }
  return n;
