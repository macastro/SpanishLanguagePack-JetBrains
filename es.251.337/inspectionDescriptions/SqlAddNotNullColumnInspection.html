Informa sobre intentos de crear columnas NOT NULL sin un valor DEFAULT.

Ejemplo (Microsoft SQL Server):

CREATE TABLE foo (a INT, b  INT)
ALTER TABLE foo ADD c INT NOT NULL;

Por defecto, las columnas contienen valores NULL. En el ejemplo, usamos la restricción NOT NULL para forzar que la columna no acepte valores NULL. Si prohibimos el uso de valores NULL, debemos establecer un valor DEFAULT que SQL pueda usar al crear nuevos registros. 

ALTER TABLE foo ADD c INT NOT NULL DEFAULT 42;

Puede usar la corrección rápida Agregar valor DEFAULT para añadir rápidamente un valor DEFAULT.
