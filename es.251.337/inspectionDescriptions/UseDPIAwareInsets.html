Informa sobre usos simplificables de java.awt.Insets y JBUI.insetsXyz().

Las instancias de Insets no son conscientes del DPI y pueden causar problemas en el diseño de la UI.

La corrección rápida realizará el reemplazo o simplificará la expresión usando JBUI.insets().

Ejemplos:


// Incorrecto:
Insets insets1 = new Insets(1, 2, 3, 4);
Insets insets2 = new Insets(1, 2, 1, 2);
Insets insets3 = new Insets(1, 0, 0, 0);
// Correcto:
Insets insets1 = JBUI.insets(1, 2, 3, 4);
Insets insets2 = JBUI.insets(1, 2);
Insets insets3 = JBUI.insetsTop(1);
