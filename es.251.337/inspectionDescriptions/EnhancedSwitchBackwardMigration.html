Reporta sentencias y expresiones switch mejoradas. Sugiere reemplazarlas por sentencias switch regulares.

Ejemplo:


  boolean even = switch (condition) {
    case 1, 3, 5, 7, 9 -> false;
    default -> true;
  };

Después de aplicar la corrección rápida:


  boolean even;
  switch (condition) {
    case 1:
    case 3:
    case 5:
    case 7:
    case 9:
      even = false;
      break;
    default:
      even = true;
      break;
}

El switch mejorado apareció en Java 14. Esta inspección ayuda a degradar el código para mantener la compatibilidad con versiones anteriores de Java.
