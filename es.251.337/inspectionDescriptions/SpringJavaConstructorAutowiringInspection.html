Informa sobre el uso incorrecto de @Autowired en constructores de Spring Bean: varios constructores @Autowired, ningún constructor @Autowired coincidente, etc.

Ejemplo:


@Component
public class MyComponent {
    @Autowired
    public MyComponent(BarBean bean) {...} // informa 'Sólo se permite un constructor @Autowired'

    @Autowired
    public MyComponent(FooBean bean) {...} // informa 'Sólo se permite un constructor @Autowired'
}

@Component
public class MyFactory {  // informa 'No se ha encontrado ningún constructor @Autowired coincidente'
    public MyFactory(String str) {...}
    public MyFactory(int count) {...}
}
