Informa sobre cualquier implementación del método Object.finalize() que esté declarado como public.

Por convención de Object.finalize(), solo el recolector de basura debe llamar a este método. Declararlo como public puede ser confuso, ya que sugiere que el método puede ser utilizado desde otro código.

Se puede utilizar una corrección rápida para establecer el método como protected, evitando así que sea llamado por otras clases.

Ejemplo:


  class X {
    public void finalize() {
      /* ... */
    }
  }

Después de aplicar la corrección rápida:


  class X {
    protected void finalize() {
      /* ... */
    }
  }
