Informa sobre el uso del operador unario +. El operador unario más generalmente es una operación nula y su presencia puede indicar un error de codificación. Por ejemplo, cuando se combina con operadores de incremento (como en +++) u operadores de igualdad (como en =+).

Ejemplo:


  void unaryPlus(int i) {
    int x = + +i;
  }

Se sugieren las siguientes correcciones rápidas:

  • Eliminar el operador + antes de la variable i:

    
  void unaryPlus(int i) {
    int x = i;
  }

  • Reemplazar el operador + con el operador de incremento prefijo:

    
  void unaryPlus(int i) {
    int x = ++i;
  }

Utilice la casilla de verificación a continuación para informar sobre el operador unario más cuando se usa junto con expresiones binarias u otras expresiones unarias. Esto significa que la inspección no informará casos donde la expresión del operador unario más se use para inicializadores de arrays o como argumentos de método.
