Informa sobre todos los constructores public en clases que no son public.

Normalmente no hay razón para crear constructores public en clases con un nivel de acceso más bajo. Sin embargo, tenga en cuenta que esta inspección puede cambiar el comportamiento de ciertas llamadas de reflexión. En particular, Class.getConstructor() no podrá encontrar el constructor actualizado (se debe usar Class.getDeclaredConstructor() en su lugar). No utilice esta inspección si su código o el código de algún framework que utilice depende de la accesibilidad del constructor a través de getConstructor().

Ejemplo:


  class House {
    public House() {}
  }

Después de aplicar la corrección rápida:


  class House {
    House() {}
  }
