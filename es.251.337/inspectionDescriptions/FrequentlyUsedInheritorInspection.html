El informe puede ser reemplazado por clases o interfaces más específicas que implementen o extiendan las clases o interfaces comúnmente utilizadas.

Para realizar esta verificación, la superclase debe estar ubicada en los archivos fuente del proyecto y el proyecto debe usar el sistema de construcción de IntelliJ IDEA.

Ejemplo:


  class MyInheritor implements A {} // B proporciona sugerencias para la referencia A
  interface A {}
  abstract class B implements A {}
  abstract class C1 extends B {}
  abstract class C2 extends B {}
  abstract class C3 extends B {}
  abstract class C4 extends B {}
  abstract class C5 extends B {}

Por defecto, esta inspección no resalta problemas en el editor, sino que solo proporciona correcciones rápidas.
