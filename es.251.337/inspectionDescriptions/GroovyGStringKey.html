Informa sobre declaraciones que utilizan objetos groovy.lang.GString como claves de mapa. Normalmente, los objetos GString son mutables y no deberían usarse como claves. Además, no se puede acceder a las entradas GString usando objetos java.lang.String con el mismo valor.

Ejemplo:


def map = [:]
def key = 'foo'
map << ["${key}": 'bar']
assert map[key] == null // resultado de comparación confuso 'true'
