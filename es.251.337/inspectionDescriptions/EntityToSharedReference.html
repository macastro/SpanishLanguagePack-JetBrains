Reporta una referencia inválida desde Entity a SharedEntity.

No se puede referenciar un Entity desde un SharedEntity. Solo se puede referenciar un SharedEntity desde un Entity cuando la propiedad es nullable o tiene la anotación @CascadeDeleteBy. De lo contrario, puede causar una excepción en tiempo de ejecución.

Ejemplo:


  interface A : LegacyEntity {
    var x: SharedEntity
    var y: SharedEntity
  }

Se puede aplicar una corrección rápida para generar la anotación o hacer que el tipo sea nullable


  interface A : LegacyEntity {
    var x: SharedEntity?
    @CascadeDeleteBy
    var y: SharedEntity
  }
