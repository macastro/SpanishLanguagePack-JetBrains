El informe puede reemplazarse con una llamada al método forEach de Java Map para Kotlin.forEach.

Ejemplo:


  fun test(map: HashMap<Int, String>) {
      map.forEach { key, value ->
          foo(key, value)
      }
  }
  fun foo(i: Int, s: String) {}

La corrección rápida añadirá paréntesis:


  fun test(map: HashMap<Int, String>) {
      map.forEach { (key, value) ->
          foo(key, value)
      }
  }
  fun foo(i: Int, s: String) {}
