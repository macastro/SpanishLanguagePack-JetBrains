Informa sobre campos de array declarados como volatile. Tales declaraciones pueden ser confusas, ya que el acceso al array en sí sigue las reglas de los campos volatile, pero no así el acceso a los contenidos del array.

Ejemplo:


  class Data {
    private volatile int[] idx = new int[0];
  }

Si se necesita este tipo de acceso volátil al contenido del array, considere usar las clases de java.util.concurrent.atomic: 


  class Data {
    private final AtomicIntegerArray idx = new AtomicIntegerArray(new int[0]);
  }
