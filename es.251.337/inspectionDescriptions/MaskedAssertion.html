Informa sobre las declaraciones assert y las aserciones del marco de pruebas que están suprimidas por los bloques catch circundantes. Estas aserciones nunca fallarán porque el AssertionError lanzado será capturado e ignorado silenciosamente.

Ejemplo 1:


  void javaAssertion() {
    try {
      ...
      assert 1 == 2;
    } catch (AssertionError e) {
      // Esta aserción se ignora silenciosamente
    }
  }

Ejemplo 2:


  @Test
  void testWithAssertJ() {
    try {
      ...
      assertThat(1).as("test").isEqualTo(2);
    } catch (AssertionError e) {
      // Esta aserción se ignora silenciosamente
    }
  }

Ejemplo 3:


  @Test
  void testWithJunit() {
    try {
      ...
      assertEquals(1, 2);
    } catch (AssertionError e) {
      // Esta aserción se ignora silenciosamente
    }
  }
