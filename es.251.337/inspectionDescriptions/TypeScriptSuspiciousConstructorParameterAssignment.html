Informa un error común en el código de TypeScript cuando se declara un campo de clase como parámetro de constructor y luego se le asigna este parámetro.
En este caso, no se asignará el campo correspondiente, solo se modifica el valor del parámetro local. 

class Foo {
  constructor(private p: number) {
    p = 1; //debe ser this.p = 1;
  }
}
