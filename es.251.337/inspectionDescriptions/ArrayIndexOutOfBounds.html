Informa sobre expresiones de acceso a arrays o punteros donde el índice podría exceder los límites del array o del buffer asignado.

Ejemplo:


  void foo() {
      int buffer[100];
      for (int i = 0; i <= 100; i ++)
          buffer[i] = 0; // buffer overflow when i is equal to 100
  }
