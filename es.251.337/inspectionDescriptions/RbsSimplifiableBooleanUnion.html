Se informa que el uso de true | false puede ser reemplazado por bool. La palabra clave bool es una forma abreviada que ayuda a mejorar la legibilidad.

Cualquier combinación de true | false puede convertirse a bool mediante una corrección rápida.

Ejemplo:

# 'true | false' puede simplificarse
type example = true | false

Después de aplicar la corrección rápida:

type example = bool
