Informa sobre la instanciación de la clase java.lang.ClassLoader.

Aunque generalmente es benigna, cualquier instanciación de ClassLoader debe examinarse cuidadosamente en cualquier auditoría de seguridad.

Ejemplo:

    Class<?> loadExtraClass(String name) throws Exception {
      try(URLClassLoader loader =
          new URLClassLoader(new URL[]{new URL("extraClasses/")})) {
        return loader.loadClass(name);
      }
    }
