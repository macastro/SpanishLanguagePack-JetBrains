Informa sobre implementaciones de Iterator.next() que no lanzan java.util.NoSuchElementException.

Tales implementaciones violan el contrato de java.util.iterator y pueden causar errores difíciles de rastrear cuando el iterador se usa de manera no estándar.

Ejemplo:


  class Numbers implements Iterator<Integer> {
    @Override
    public Integer next() { //warning
        if (hasNext()) {
            return generateNext();
        } else {
            return null; //debería lanzar NoSuchElementException en su lugar
        }
    }
    ...
  }
