Informa las llamadas innecesarias a close() al final de un bloque try-with-resources y sugiere eliminarlas.

Ejemplo:


  try(MyAutoCloseable ac = new MyAutoCloseable()) {
    foo();
    ac.close();
  }

Después de aplicar la corrección rápida:


  try(MyAutoCloseable ac = new MyAutoCloseable()) {
    foo();
  }
