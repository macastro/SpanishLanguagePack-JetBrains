Reporta llamadas a métodos que usan java.nio.file.Files para leer o escribir un String como bytes Estas llamadas pueden ser reemplazadas por llamadas a los métodos Files.readString() o Files.writeString() introducidos en Java 11.

Ejemplo:


  String s = "example";
  Files.write(Paths.get("out.txt"), s.getBytes(StandardCharsets.UTF_8), StandardOpenOption.WRITE);
  s = new String(Files.readAllBytes(Paths.get("in.txt")), StandardCharsets.ISO_8859_1);

Después de aplicar la corrección rápida:


  String s = "example";
  Files.writeString(Paths.get("out.txt"), s, StandardOpenOption.WRITE);
  s = Files.readString(Paths.get("in.txt"), StandardCharsets.ISO_8859_1);

Novedades de la versión 2018.3
