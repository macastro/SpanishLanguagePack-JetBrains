Informa sobre llamadas a start() en java.lang.Thread o cualquiera de sus subclases durante la construcción del objeto.

Aunque estas estructuras pueden ser ocasionalmente útiles, deben evitarse debido a problemas de herencia. Las subclases de una clase que inicia un hilo durante la construcción del objeto no completarán ninguna de sus lógicas de inicialización antes de que el hilo se inicie.

Esta inspección no informa si la clase que inicia el hilo está declarada como final.

Ejemplo:


  class MyThread extends Thread {
    MyThread() {
      start();
    }
  }
