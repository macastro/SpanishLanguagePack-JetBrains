El informe puede reemplazar las llamadas a métodos estáticos de Java Collections por sus equivalentes en Kotlin stdlib.

Ejemplo:


  import java.util.Collections
  fun test() {
      val mutableList = mutableListOf(1, 2)
      Collections.fill(mutableList, 3)
  }

Esta corrección rápida reemplazará la llamada al método estático de Java Collections por su llamada equivalente en Kotlin stdlib:


  import java.util.Collections
  fun test() {
      val mutableList = mutableListOf(1, 2)
      mutableList.fill(3)
  }
