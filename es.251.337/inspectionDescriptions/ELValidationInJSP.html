Informa sobre posibles problemas de EL, como referencias no resueltas y ubicaciones EL inválidas.

Ejemplo:


Considera la función `exampleFun(String)` definida en el espacio de nombres actual. 

<jsp:root xmlns:example="urn:jsptld:exampleNamespace.tld">
  ${ example:exampleFun("1") + example:unknownFun("2") } // Error: no se puede resolver la función 'unknownFun'
  ${ example:exampleFun("1", "2") } // Error: el número de parámetros formales y reales no coincide
</jsp:root>
