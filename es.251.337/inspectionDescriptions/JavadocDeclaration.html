Informa los siguientes problemas en los comentarios y etiquetas Javadoc:
  • nombres de etiqueta no válidos
  • descripciones de etiqueta incompletas
  • etiquetas duplicadas
  • descripciones Javadoc faltantes

Ejemplo:


  /**
   * Nombre de etiqueta no válido
   * @poram param description
   */
  public void sample(int param){
  }

Ejemplo:


  /**
   * Señalando a sí mismo {@link #sample(int)}
   */
  public void sample(int param){
  }

La corrección rápida agrega la etiqueta Javadoc desconocida a la lista de etiquetas adicionales definidas por el usuario.

Usa el campo de texto que aparece a continuación para definir etiquetas Javadoc adicionales.

Utiliza la primera casilla de verificación para ignorar la etiqueta 'throws' duplicada.

Usa la segunda casilla de verificación para ignorar el problema con la primera oración faltante o incompleta en la descripción.

Usa la tercera casilla de verificación para ignorar las referencias que señalan a sí mismas.
