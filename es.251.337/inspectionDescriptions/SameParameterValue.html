Informa sobre parámetros de método que siempre tienen el mismo valor constante.

Ejemplo:


  static void printPoint(int x, int y) { // x siempre es 0
    System.out.println(x + ", " + y);
  }
  public static void main(String[] args) {
    printPoint(0, 1);
    printPoint(0, 2);
  }

La corrección rápida incorporará el valor constante. Esto puede simplificar la implementación del método.

Utilice la opción Ignorar cuando no se pueda proporcionar una corrección rápida para desactivar la inspección en los siguientes casos:

  • El parámetro se modifica dentro del método
  • El valor del parámetro pasado es una referencia a un campo inaccesible (solo Java)
  • El parámetro es un vararg (solo Java)

Utilice la opción Visibilidad máxima del método para controlar la visibilidad máxima de los métodos a informar.

Utilice el campo Número mínimo de usos del método para informar parámetros para especificar el número mínimo de usos del método con el mismo valor de parámetro.
