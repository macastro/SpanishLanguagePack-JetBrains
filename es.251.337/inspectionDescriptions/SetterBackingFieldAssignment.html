Reportar setter de propiedades que no actualiza el campo de respaldo.

Esta corrección rápida añade una asignación al campo de respaldo.

Ejemplo:


  class Test {
      var foo: Int = 1
          set(value) {
          }
  }

Después de aplicar la corrección rápida:


  class Test {
      var foo: Int = 1
          set(value) {
              field = value
          }
  }
