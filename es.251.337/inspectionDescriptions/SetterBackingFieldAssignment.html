Informa de los setters de propiedades que no actualizan el campo de respaldo.

La solución rápida añade una asignación al campo de respaldo.

Ejemplo:


  class Test {
      var foo: Int = 1
          set(value) {
          }
  }

Después de aplicar la solución rápida:


  class Test {
      var foo: Int = 1
          set(value) {
              field = value
          }
  }
