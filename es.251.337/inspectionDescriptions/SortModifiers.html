El informe detecta modificadores que no siguen el orden recomendado en la guía de estilo.

La corrección rápida de ordenación de modificadores está disponible para corregir el código automáticamente.

Ejemplo:


  private inline fun correctOrder(f: () -> Unit) {} // <== Ok
  infix private fun Int.wrongOrder(expr: Int) {} // <== Orden incorrecto, la corrección rápida lo cambiará a "private infix"
