Reporta elementos incorrectos en el archivo Java Server Faces faces-config.xml:

  • Referencias a clases no instanciables
  • Referencias a clases que no extienden las clases requeridas
  • Referencias a clases con ámbito inadecuado
  • Etiquetas y valores de atributos vacíos
  • Etiquetas y valores de atributos que no coinciden con el esquema requerido (por ejemplo, identificadores Java)
  • Etiquetas que no contienen las subetiquetas o atributos requeridos
  • Etiquetas que definen objetos con nombres duplicados

Ejemplo:


  <faces-config >
     <application>
        <action-listener>java.lang.String</action-listener>  <!-- 'java.lang.String' no es asignable a 'javax.faces.event.ActionListener'  -->
    </application>
    <navigation-rule>
        <from-view-id>myView.xhtml</from-view-id>
        <navigation-case>
            <to-view-id>unknown.xhtml</to-view-id>  <!-- No se puede resolver 'unknown.xhtml'  -->
        </navigation-case>
    </navigation-rule>
  </faces-config>
