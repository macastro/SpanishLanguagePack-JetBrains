Reporta una declaración de desestructuración incompleta.

Ejemplo:


  data class Person(val name: String, val age: Int)
  val person = Person("", 0)
  val (name) = person

La acción rápida completa la declaración de desestructuración con las nuevas variables:


  data class Person(val name: String, val age: Int)
  val person = Person("", 0)
  val (name, age) = person
