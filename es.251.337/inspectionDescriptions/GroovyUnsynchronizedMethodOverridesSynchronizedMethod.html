Informa sobre métodos no synchronized que sobrescriben métodos synchronized.

Ejemplo:


  abstract class Base {
      synchronized void foo() {
          // ...
      }
  }
  class Derived extends Base {
      @Override
      void foo() {
          super.foo()
            // ...
        }
    }

Aquí, el método no synchronized foo() en la clase Bar sobrescribe un método synchronized.
