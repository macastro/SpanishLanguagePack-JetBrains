Informa sobre la declaración de equal() con un solo parámetro. Normalmente, esto es un error tipográfico, cuando en realidad se pretendía usar equals().

Se sugiere renombrar el método a equals mediante una corrección rápida.

Ejemplo:


  class Main {
    public boolean equal(Object obj) {
      return true;
    }
  }

Después de aplicar la corrección rápida:


  class Main {
    public boolean equals(Object obj) {
      return true;
    }
  }
