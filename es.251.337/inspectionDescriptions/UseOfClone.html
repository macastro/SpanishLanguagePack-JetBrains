Reporta implementaciones del método clone(), sus invocaciones y el uso de la interfaz java.lang.Cloneable.

Algunos estándares de codificación prohíben el uso de clone() y sugieren usar constructores de copia o métodos de fábrica static en su lugar.

Esta inspección ignora las llamadas a clone() en arrays, ya que es una forma correcta y concisa de copiar arrays.

Ejemplo:


  class Copy implements Cloneable /*advertencia*/ {
    public Copy clone() /*advertencia*/ {
      try {
        return (Copy) super.clone(); // advertencia
      } catch (CloneNotSupportedException e) {
        throw new AssertionError();
      }
    }
  }
