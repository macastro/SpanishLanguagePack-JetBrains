Informa casos en los que la llamada al constructor dinámico podría reemplazarse de forma segura con una llamada al método de fábrica.

Ejemplo:


  import akka.actor._
  
  class Foo extends AbstractActor
  
  object Foo {
    def props(): Props = Props(new Foo())
  }
  
  Props(classOf[Foo])

Tras aplicar la solución rápida:


  import akka.actor._
  
  class Foo extends AbstractActor
  
  object Foo {
    def props(): Props = Props(new Foo())
  }
  
  Foo.props()
