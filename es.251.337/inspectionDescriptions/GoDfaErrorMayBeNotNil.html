Informa sobre instancias donde las variables pueden tener valores nil o inesperados porque no se verificó si el error asociado era distinto de nil, como en v, err := foo().

Esta corrección rápida eliminará la función del alcance de esta verificación.

Ejemplo:


func _() {
    file, err := os.Open("file.txt")
    // Se omite la verificación de error aquí
    name := file.Name()
    print(name, err)
}

En el ejemplo proporcionado, file podría ser nil o tener un valor inesperado cuando err no es nil.
