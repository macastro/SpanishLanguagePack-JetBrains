Informa sobre código que utiliza bucles para copiar contenido de arrays a colecciones.

Una forma más corta y potencialmente más rápida (dependiendo de la implementación de la colección) es usar Collection.addAll(Arrays.asList()) o Collections.addAll().

Solo se informan los bucles que no contienen declaraciones adicionales en su interior.

Ejemplo:


  void addAll(List<String> list, String[] arr) {
    for (int i = 0; i < arr.length; i++) {
      String s = arr[i];
      list.add(s);
    }
  }

Después de aplicar la corrección rápida:


  void addAll(List<String> list, String[] arr) {
    Collections.addAll(list, arr);
  }
