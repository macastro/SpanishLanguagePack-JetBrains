Propiedad desconocida en el bloque de informe

Ejemplo del problema:


resource "awscc_appflow_flow" "aws_flow1" {
  flow_name          = ""
  source_flow_config = ""
  trigger_config     = ""
  destination_flow_config_list {}
  tasks {}
  test = ""
}

En este ejemplo, la propiedad test en el bloque es desconocida.

Ejemplo corregido:


resource "awscc_appflow_flow" "aws_flow1" {
  flow_name          = ""
  source_flow_config = ""
  trigger_config     = ""
  destination_flow_config_list {}
  tasks {}
}
