Informa sobre propiedades que no siguen las convenciones de nomenclatura.

Se informan las siguientes propiedades:

  • Propiedades private en objetos y objetos companion

Puede especificar los patrones deseados en las opciones de inspección.

Convenciones de nomenclatura recomendadas: debe comenzar con guion bajo o letra mayúscula, usando camelCase.

Ejemplo:


  class Person {
    companion object {
      // Propiedad en objeto companion
      private val NO_NAME = Person()
    }
  }
