Reporte las variables de instancia "no frágiles", las cuales brindan soporte para propiedades que se sintetizarán en tiempo de ejecución, sin necesidad de declararlas en la interfaz de la clase. A pesar de que esta es la manera correcta de declarar variables de instancia, es posible que se le olvide declararlas explícitamente. Debido a que las variables de instancia "no frágiles" se implementan mediante una capa de indirección adicional, pueden generar gastos generales no deseados en tiempo de ejecución.
