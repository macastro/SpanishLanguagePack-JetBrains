Informa sobre expresiones condicionales que tienen ramas then y else lo suficientemente similares como para que la expresión pueda moverse hacia adentro. Esta operación acorta el código.

Ejemplo:


  double g(int a, int b) {
    return a == b ? Math.cos(0) : Math.cos(1);
  }

Después de aplicar la corrección rápida:


  double g(int a, int b) {
    return Math.cos(a == b ? 0 : 1);
  }
