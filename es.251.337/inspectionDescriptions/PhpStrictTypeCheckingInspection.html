Informa sobre violaciones de las reglas de comprobación estricta de tipos.

En el modo estricto, solo se aceptan variables que coincidan exactamente con el tipo declarado. El modo de comprobación estricta de tipos se puede habilitar de las siguientes maneras:

  • Para archivos individuales, si se especifica la directiva declare(strict_types=1). Esta directiva debe ser la primera declaración en el archivo, y el modo de bloque no está permitido explícitamente.
  • Para todos los archivos, si la opción de inspección Habilitar para todos los archivos está activada.

Para más información, consulte Strict typing (php.net).
