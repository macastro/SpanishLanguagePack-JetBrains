Informa sobre métodos que sobrescriben métodos anotados con @Pattern sin incluir la anotación @Pattern. Esto no impide que se resalten los errores en el editor, pero el método sobrescrito no será verificado en tiempo de ejecución.

Se proporciona una corrección rápida para añadir la anotación @Pattern que coincida con la anotación del método de la superclase. Esto asegura que la verificación en tiempo de ejecución funcione correctamente.

Ejemplo:


  abstract class Parent {
    abstract @Pattern("\\d\\d-\\d\\d\\d") String getId();
  }
  class Child extends Parent {
    @Override String getId() { // advertencia aquí
      return "12-345";
    }
  }
