Informa sobre el uso del operador ternario condicional y sugiere convertirlos en declaraciones if/else.

Algunos estándares de código prohíben el uso del operador condicional.

Ejemplo:


  Object result = (condition) ? foo() : bar();

Después de aplicar la corrección rápida:


  Object result;
  if (condition) {
    comp = foo();
  }
  else {
    comp = bar();
  }

Configuración de la inspección:

Utilice la opción Ignorar asignaciones y retornos simples para ignorar asignaciones y retornos simples y permitir estructuras como:


  String s = (foo == null) ? "" : foo.toString();

Use la opción Ignorar ubicaciones donde no se puede usar if para ignorar expresiones condicionales en contextos donde no es posible reemplazar automáticamente con una declaración if (por ejemplo, cuando la expresión condicional se usa como llamada al constructor super()).
