Informa sobre caracteres de escape innecesarios en literales de String y char (opcional).

No se informará sobre el carácter de escape de tabulación \t, ya que el carácter de tabulación no es visible.

Ejemplo:


  String s = "\'Scare\' quotes";
  String t = """
    All you need is\n\tLove\n""";

Después de aplicar la corrección rápida:


  String s = "'Scare' quotes";
  String t = """
    All you need is
    \tLove
  """;

Cambios en la versión 2019.3
