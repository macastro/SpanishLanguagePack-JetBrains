Reporta incompatibilidades de tipos de propiedades en las características de JPA.

Ejemplo:



  @Entity
  public class JavaEntity {
    @OneToOne
    Map<Integer, AnotherEntity> incorrectRelationship; // Error: El tipo de característica 'One To One' debe ser Entity, no Map
    @ManyToMany
    Map<Integer, AnotherEntity> correctRelationship;
  }
