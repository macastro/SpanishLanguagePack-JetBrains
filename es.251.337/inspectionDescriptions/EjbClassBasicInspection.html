Reporta las siguientes violaciones de la especificación EJB:
  • La clase no implementa las interfaces esperadas (EJBLocalObject, EJBObject, etc.)
  • La clase no define un constructor público sin parámetros
  • Uso de clase en lugar de interfaz o uso de interfaz en lugar de clase
  • El método EJB es static
  • La clase Bean de sesión sin estado define múltiples métodos `ejbCreate`
  • El tipo de retorno del método de la clase Bean de sesión es incorrecto
  • El EJB define un método `finalize`

Ejemplo:


public class SessionEJB implements java.io.Serializable { // Error: la clase debería extender `SessionBean`
  private SessionEJB() {} // Error: la clase debería definir un constructor público sin parámetros
}
