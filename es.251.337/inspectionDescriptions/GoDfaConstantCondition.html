Informa sobre condiciones en el código que siempre se evalúan como true o false.

Ejemplo:


func _(file *os.File) {
    if file == nil {
        return
    }
    if file == nil {
    }
}

En el ejemplo dado, la segunda condición if file == nil es redundante. Siempre se evaluará como false porque repite una comprobación que ya se ha ejecutado en el código.
