Informa sobre el uso del operador rangeTo o .. en lugar del uso de until.

El código puede simplificarse usando la función correspondiente.

Esta corrección rápida reemplazará la llamada a rangeTo o .. por until.

Ejemplo:


  fun foo(a: Int) {
      for (i in 0..a - 1) {
      }
  }

Después de aplicar la corrección rápida:


  fun foo(a: Int) {
      for (i in 0 until a) {
      }
  }
