Reporta configuraciones incorrectas del contexto de Spring.

  • Archivos y directorios no resueltos en el atributo locations y sus alias correspondientes en la meta-anotación @ContextConfiguration
  • Archivo de contexto de aplicación predeterminado faltante

Para más detalles, consulte @ContextConfiguration.

Ejemplo:


  @ContextConfiguration(locations = "classpath:META-INF/unknown-context.xml") // Reporta "No se puede resolver el archivo 'unknown-context.xml'"
  class MyTests {...}
