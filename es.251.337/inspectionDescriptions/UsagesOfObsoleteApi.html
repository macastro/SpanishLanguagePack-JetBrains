Reporta declaraciones (clases, métodos, campos) anotadas como @ApiStatus.Obsolete.

Algunas veces es imposible eliminar el API actual, aunque pudiera no funcionar correctamente, hay un API más nuevo, o mejor, o más genérico. De esta forma, es una variante más débil de la anotación @Deprecated. Se supone que el API anotado no debe ser usado en el código nuevo, pero se permite posponer la migración del código existente, por lo tanto el uso no es considerado una advertencia.
