Informa sobre declaraciones (clases, métodos, campos) anotadas con @ApiStatus.Obsolete.

A veces no es posible eliminar una API actual, aunque puede que no funcione correctamente, pero existe una API más nueva, mejor o más general. En este caso, es una variante más débil de la anotación @Deprecated. La API anotada no debe utilizarse en código nuevo, pero se permite posponer la migración del código existente, por lo que este uso no se considera una advertencia.
