Informa sobre dos comparaciones consecutivas que pueden convertirse en una comprobación de rango.

La comprobación de rangos simplifica el código al eliminar la repetición en la condición de prueba.

Ejemplo:


  fun checkMonth(month: Int): Boolean {
      return month >= 1 && month <= 12
  }

Esta corrección rápida reemplaza la comprobación basada en comparaciones por una comprobación de rango:


  fun checkMonth(month: Int): Boolean {
      return month in 1..12
  }
