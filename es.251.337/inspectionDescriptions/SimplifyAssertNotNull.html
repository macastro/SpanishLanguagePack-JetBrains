Reporta llamadas a assert que verifican que las variables declaradas no son null.

El código puede simplificarse usando !! o ?:.

Esta corrección rápida reemplaza assert en los inicializadores de variables con los operadores !! o ?:.

Ejemplo:


  fun foo(p: Array<String?>) {
      val v = p[0]
      assert(v != null, { "Should be not null" })
  }

Después de aplicar la corrección rápida:


  fun foo(p: Array<String?>) {
      val v = p[0] ?: error("Should be not null")
  }
