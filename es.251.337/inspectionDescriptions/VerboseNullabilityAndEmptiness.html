El informe se puede simplificar a una única combinación de comprobaciones de null y vacío.

Esta corrección rápida reemplazará las comprobaciones resaltadas con una llamada de comprobación combinada, como isNullOrEmpty().

Ejemplo:


  fun test(list: List<Int>?) {
      if (list == null || list.isEmpty()) {
          println("List is empty!")
      } else {
          println(list.joinToString())
      }
  }

Después de aplicar la corrección rápida:


  fun test(list: List<Int>?) {
      if (list.isNullOrEmpty()) {
          println("List is empty!")
      } else {
          println(list.joinToString())
      }
  }
