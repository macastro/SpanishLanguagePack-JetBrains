Informa sobre el uso de constantes de alineación absoluta en AWT y Swing. Las aplicaciones internacionalizadas deben usar alineación relativa, ya que esta sigue la configuración de dirección del componente según la configuración regional.

Ejemplo:


  JPanel panel = new JPanel(new BorderLayout(2, 2));
  JLabel label = new JLabel("Hello World");
  panel.add(label, BorderLayout.NORTH);

Después de aplicar la corrección rápida:


  JPanel panel = new JPanel(new BorderLayout(2, 2));
  JLabel label = new JLabel("Hello World");
  panel.add(label, BorderLayout.PAGE_START);
