Informa los usos de constantes de alineación absolutas de AWT y Swing. Las aplicaciones internacionalizadas utilizan alineación relativa porque respeta las configuraciones de orientación de componentes locales.

Ejemplo:


  JPanel panel = new JPanel(new BorderLayout(2, 2));
  JLabel label = new JLabel("Hola Mundo");
  panel.add(label, BorderLayout.NORTH);

Después de aplicar la corrección rápida:


  JPanel panel = new JPanel(new BorderLayout(2, 2));
  JLabel label = new JLabel("Hola Mundo");
  panel.add(label, BorderLayout.PAGE_START);
