Informa sobre clases, métodos y campos no utilizados.

Ejemplo:


  public class Department {
      private Organization myOrganization;
  }

Aquí, Department hace referencia explícita a Organization, pero si la clase Department no se utiliza, esta inspección informará sobre ambas clases.

La inspección también informa sobre métodos, parámetros cuyos métodos de implementación/anulación no se utilizan, así como variables locales que se han declarado pero no se utilizan.

Para más detalles, consulte la misma inspección en Java.
