Informa sobre valores de varibles que no se utilizan tras ser asignados. Esto puede ocurrir en los siguientes casos:
  • La variable no es leída en ningún momento después de la asignación.
  • El valor se sobrescribe mediante otra asignación antes de que se haya accedido por lectura.
  • El inicializador de la variable es redundante (por una de las dos razones anteriores).
  • La variable en sí no se utiliza.

Ejemplo:


  class C {
    int field;

    void foo() {
      field = unused_value;
    }

    void bar() {
      use(field);
      foo();
    }
  };
