Reporta expresiones if que tienen una condición literal constante de true o false y pueden simplificarse.

Aunque ocasionalmente sea intencional, esta construcción confunde y a menudo resulta de un error tipográfico o de una refactorización previa.

Ejemplo:


  fun example() {
      if (true) {
          throw IllegalStateException("Unexpected state")
      }
  }

La corrección rápida elimina la condición if:


  fun example() {
      throw IllegalStateException("Unexpected state")
  }
