Las llamadas a la función map.put pueden reemplazarse por el operador de indexación ([]).

El código puede simplificarse usando azúcar sintáctico.

Esta corrección rápida reemplazará la llamada a put por una asignación.

Ejemplo:


  fun foo(map: MutableMap<Int, String>) {
      map.put(42, "foo")
  }

Después de aplicar la corrección rápida:


  fun foo(map: MutableMap<Int, String>) {
      map[42] = "foo"
  }
