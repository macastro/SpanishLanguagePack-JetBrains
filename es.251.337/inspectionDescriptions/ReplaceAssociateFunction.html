Informa sobre llamadas a associate() y associateTo() que pueden ser reemplazadas por associateBy() o associateWith().

Ambas funciones aceptan una función transformadora que se aplica a los elementos de una secuencia o colección dada (como receptor). Luego, esta función de pares se utiliza para construir el Map resultante.

Dado que el transformador hace referencia a it, las llamadas a associate[To]() pueden reemplazarse por las alternativas más eficientes associateBy() o associateWith().

Ejemplo:

  fun getKey(i: Int) = 1L
  fun getValue(i: Int) = 1L
  fun test() {
      arrayOf(1).associate { getKey(it) to it }  // 'associate()' reemplazable
      listOf(1).associate { it to getValue(it) } // 'associate()' reemplazable
  }

Después de aplicar la corrección rápida:

  fun getKey(i: Int) = 1L
  fun getValue(i: Int) = 1L
  fun test() {
      arrayOf(1).associateBy { getKey(it) }
      listOf(1).associateWith { getValue(it) }
  }
