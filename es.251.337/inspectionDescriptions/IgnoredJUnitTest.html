Informa sobre el uso de las anotaciones @Ignore de JUnit 4 o @Disabled de JUnit 5. Se considera un mal olor de código tener pruebas anotadas con estas anotaciones por mucho tiempo, especialmente cuando no se especifica ningún motivo.

Ejemplo:


  @Ignore
  public class PruebaUrgente {

    @Test
    public void probarlo() {
      Assert.assertEquals("esperado", "real");
    }
  }

Configurar la inspección:

  • Utiliza la opción Solo informar anotaciones sin motivo para informar solamente los casos en que no se especifica ningún motivo como el atributo value de la anotación.
