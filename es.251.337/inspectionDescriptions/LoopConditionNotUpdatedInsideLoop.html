Informa sobre cualquier variable y parámetro que se use en una condición de bucle y no se actualice dentro del mismo.

Estas variables y parámetros suelen utilizarse incorrectamente, ya que su ejecución puede resultar en bucles infinitos.

Ejemplo:


  void loopDoesNotLoop(boolean b) {
    while (b) {
      System.out.println();
      break;
    }
  }

Configuración de la inspección:

Use la opción Ignorar posibles modificaciones no locales para desactivar esta inspección si la condición puede actualizarse indirectamente (por ejemplo, a través de un método invocado o mediante actualizaciones concurrentes desde otro hilo).
