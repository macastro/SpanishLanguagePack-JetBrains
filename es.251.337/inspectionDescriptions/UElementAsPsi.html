Reporta el uso de UAST UElement como PsiElement.

El PsiElement obtenido de esta manera es ambiguo.

Para obtener el PsiElement "físico", utilice UElementKt.getSourcePsiElement(), y para el PsiElement que emula el comportamiento de elementos Java "simulados" (PsiClass, PsiMethod, etc.), utilice UElementKt.getAsJavaPsiElement().

Consulte UAST - Árbol de Sintaxis Abstracta Unificado en la documentación del SDK.
