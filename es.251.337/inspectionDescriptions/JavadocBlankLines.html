Informa sobre líneas vacías en comentarios Javadoc.

Las líneas vacías en Javadoc pueden indicar la intención de dividir el texto en párrafos diferentes. Sin embargo, tanto la herramienta Javadoc como IntelliJ IDEA las ignoran al renderizar los comentarios de documentación.

La corrección rápida sugiere reemplazar las líneas vacías con etiquetas de párrafo (<p>).

Ejemplo:


  class Main {
    /**
     * No funciona.
     *
     * No funciona en absoluto
     */
    void foo() {}
  }

Después de aplicar la corrección rápida:


  class Main {
    /**
     * No funciona.
     * <p>
     * No funciona en absoluto
     */
    void foo() {}
  }
