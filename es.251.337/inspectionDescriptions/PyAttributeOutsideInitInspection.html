Informa de un problema cuando la definición del atributo de la instancia está fuera del método __init__.

Ejemplo:


    class Book:
    def __init__(self):
        self.author = 'Mark Twain'

    def release(self):
        self.year = '1889'

Cuando se aplica la corrección rápida, el ejemplo de código cambia a: 


    class Book:
    def __init__(self):
        self.year = '1889'
        self.author = 'Mark Twain'

    def release(self):
        pass
