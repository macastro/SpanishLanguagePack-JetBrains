Reporta las llamadas a la función errors.As cuando el segundo argumento no es un puntero a una interfaz o a un tipo que implemente error.

Estas llamadas provocarán un panic en tiempo de ejecución.

Para más información sobre la función As, consulta func As en go.dev.

Ejemplo:

_, err := os.Open("non-existing")
var pathError *fs.PathError
if errors.As(err, pathError) { // necesita un puntero a *fs.PathError
}

Después de aplicar la corrección rápida preponer '&':

_, err := os.Open("non-existing")
var pathError *fs.PathError
if errors.As(err, &pathError) {
}

Esta verificación solo se reporta cuando la versión del lenguaje es 1.13 o superior.
