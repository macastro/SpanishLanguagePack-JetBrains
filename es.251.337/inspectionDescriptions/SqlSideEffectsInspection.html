Informa sobre sentencias que podrían modificar la base de datos durante una conexión de solo lectura.

Para habilitar el modo de solo lectura para una conexión, haz clic derecho en el origen de datos en la ventana de herramientas Base de datos (Ver | Ventanas de herramientas | Base de datos) y selecciona Propiedades. En el diálogo Orígenes de datos y controladores, haz clic en la pestaña Opciones y marca la casilla Solo lectura.

Ejemplo (MySQL):

CREATE TABLE foo(a INT);
INSERT INTO foo VALUES (1);

Como las sentencias CREATE TABLE y INSERT INTO provocan modificaciones en la base de datos, estas sentencias se resaltarán en el modo de conexión de solo lectura.
