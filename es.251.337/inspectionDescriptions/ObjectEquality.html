Reporta código que usa == o != en lugar de equals() para probar la igualdad de objetos.

Usar == o != para comparar objetos suele ser un error, ya que compara la identidad de los objetos en lugar de su igualdad. No se reportan las comparaciones con null.

Las comparaciones de arrays, String y Number se reportan mediante inspecciones separadas.

Ejemplo:

  if (list1 == list2) {
    return;
  }

Después de aplicar la corrección rápida:

  if (Objects.equals(list1, list2)) {
    return;
  }

Utiliza la configuración de la inspección para configurar excepciones para esta comprobación.
