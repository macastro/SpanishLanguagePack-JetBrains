Informa sobre variables locales que se declaran en una rama de la sentencia switch y se utilizan en otra rama. Este tipo de declaraciones pueden resultar especialmente confusas.

Ejemplo:


    switch(i) {
      case 2:
          int x = 0;
          break;
      case 3:
          x = 3;
          System.out.println(x);
          break;
    }
