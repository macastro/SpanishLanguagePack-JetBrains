Informar sobre imports desordenados.

Todos los programas Go deben estar formateados de la misma manera, con reglas de formato definidas por la herramienta gofmt. Estas reglas requieren que los imports estén ordenados.

Ejemplo de ordenamiento incorrecto:

import (
 "net"
 "errors"
 "fmt"
)

Puede aplicar la corrección rápida Ordenar imports para solucionar el problema de ordenamiento. Después de aplicar la corrección rápida, el ordenamiento se verá así:

import (
 "errors"
 "fmt"
 "net"
)
