Informa sobre las declaraciones if, when y try que pueden convertirse en expresiones extrayendo las declaraciones return o asignaciones.

Ejemplo:


  fun foo(arg: Int): String {
      when (arg) {
          0 -> return "Zero"
          1 -> return "One"
          else -> return "Multiple"
      }
  }
Después de aplicar la corrección rápida: 

  fun foo(arg: Int): String {
      return when (arg) {
          0 -> "Zero"
          1 -> "One"
          else -> "Multiple"
      }
  }

Si desea que esta inspección resalte código más complejo con ramas de múltiples declaraciones, desmarque la opción "Informar solo cuando cada rama sea una única declaración".
