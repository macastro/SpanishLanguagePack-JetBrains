Informa sobre llamadas a String.equals() con CharSequence como argumento.

Para argumentos de tipo String, String.equals() solo puede devolver true. Para comparar el contenido de un String con argumentos CharSequence que no sean String, utilice el método contentEquals().

Ejemplo:


  boolean equals(String s, CharSequence ch) {
    return s.equals(ch);
  }

Después de aplicar la corrección rápida:


  boolean equals(String s, CharSequence ch) {
    return s.contentEquals(ch);
  }
