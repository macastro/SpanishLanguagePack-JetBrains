Informa sobre casos de prueba no abstract que no tienen métodos de prueba. Estos casos de prueba generalmente indican código incompleto o posibles residuos de refactorización que deberían eliminarse.

Ejemplo:


  public class CrucialTest {
    @Before
    public void setUp() {
      System.out.println("setting up");
    }
  }

Utilice la opción Ignorar casos de prueba con superclases que contienen métodos de prueba para excluir los casos de prueba que tienen superclases con métodos de prueba.
