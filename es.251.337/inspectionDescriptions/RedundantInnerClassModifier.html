Informa cuando el modificador inner en una clase es redundante si la clase no hace referencia a los miembros de su clase externa.

Ejemplo:


  class Foo {
      inner class InnerClass { // Modificador `inner` redundante
          fun hello() {
              println("Hi!")
          }
      }
  }
  class List {
      val objects = Array<Any>(42) { Any() }
      inner class Iterator { // Modificador `inner` no redundante
          fun next(): Any {
              return objects[0]
          }
      }
  }

Después de aplicar la corrección rápida:


  class Foo {
      class InnerClass { // Modificador `inner` redundante
          fun hello() {
              println("Hi!")
          }
      }
  }
  class List {
      val objects = Array<Any>(42) { Any() }
      inner class Iterator { // Modificador `inner` no redundante
          fun next(): Any {
              return objects[0]
          }
      }
  }
