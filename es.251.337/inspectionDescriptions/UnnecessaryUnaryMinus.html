Informa sobre operadores unarios de resta innecesarios. Estas expresiones pueden ser difíciles de entender y podrían contener errores.

Por ejemplo:

void unaryMinus(int i) {
    int x = - -i;
  }

Aquí se sugieren las siguientes correcciones rápidas:

  • Eliminar el operador - antes de la variable i:

    void unaryMinus(int i) {
    int x = i;
  }

  • Reemplazar el operador - con el operador de decremento prefijo:

    void unaryMinus(int i) {
    int x = --i;
  }

Otro ejemplo:

void unaryMinus(int i) {
    i += - 8;
  }

Después de aplicar la corrección rápida:

void unaryMinus(int i) {
    i -= 8;
  }
