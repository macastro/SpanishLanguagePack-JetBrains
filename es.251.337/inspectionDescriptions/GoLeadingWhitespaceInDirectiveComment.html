Informa sobre espacios en blancos líderes antes de las directivas Go en comentarios.

Las directivas Go no se reconocen si hay un espacio en blanco entre // y la directiva Go.

La corrección rápida elimina el espacio en blanco inicial antes de la directiva Go.

Ejemplo:


//  go:embed file.txt
var File string

Después de aplicar la corrección rápida:


//go:embed file.txt
var File string
