Reporta espacios en blanco iniciales antes de las directivas Go en los comentarios.

La directiva Go no será reconocida si hay espacios entre // y la directiva Go.

Esta corrección rápida elimina los espacios en blanco iniciales antes de la directiva Go.

Ejemplo:


//  go:embed file.txt
var File string

Después de aplicar la corrección rápida:


//go:embed file.txt
var File string
