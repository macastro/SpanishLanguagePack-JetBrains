Informa de declaraciones que se pueden desestructurar.

Ejemplo:


  data class My(val first: String, val second: Int, val third: Boolean)

  fun foo(list: List<My>) {
      list.forEach { my ->
          println(my.second)
          println(my.third)
      }
  }

La corrección rápida desestructura la declaración e introduce nuevas variables con nombres de la clase correspondiente:


  data class My(val first: String, val second: Int, val third: Boolean)

  fun foo(list: List<My>) {
      list.forEach { (_, second, third) ->
          println(second)
          println(third)
      }
  }
