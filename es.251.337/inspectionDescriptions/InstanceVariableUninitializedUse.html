Informa sobre variables de instancia que se leen antes de su inicialización.

Esta inspección ignora las comprobaciones de igualdad con null.

Ejemplo:


  class Foo {
    int bar;
    Foo() {
      System.out.println(bar);
    }
  }

Tenga en cuenta que esta inspección utiliza un algoritmo de flujo de datos muy conservador y podría reportar incorrectamente variables de instancia como no inicializadas. Las variables reportadas como inicializadas siempre estarán inicializadas.

Use la opción Ignorar cuando hay anotaciones para especificar anotaciones especiales. La inspección ignorará los campos anotados con cualquiera de estas anotaciones.

Use la opción Ignorar campos primitivos para ignorar campos primitivos no inicializados.
