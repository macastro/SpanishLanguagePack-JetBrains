Informa sobre arrays utilizados en concatenación de String o pasados como parámetros a métodos de java.io.PrintStream, como System.out.println().

Por lo general, se debe utilizar el contenido del array en lugar del objeto array en sí.

Ejemplo:


  void print(Object[] objects) {
    System.out.println(objects);
  }

Después de aplicar la corrección rápida:


  void print(Object[] objects) {
    System.out.println(Arrays.toString(objects));
  }
