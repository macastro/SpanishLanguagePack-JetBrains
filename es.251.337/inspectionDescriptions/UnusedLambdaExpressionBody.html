Informa sobre llamadas a funciones donde el valor devuelto no se utiliza cuando la función llamada devuelve una lambda con cuerpo de expresión.

Si existe un = entre el encabezado de la función y el bloque del cuerpo, el código dentro de la función no se evaluará, lo que puede resultar en un comportamiento incorrecto.

Eliminar el token = de la declaración de la función está disponible como corrección rápida.

Ejemplo:


  fun printHello() = { println("Hello") }
  fun main() {
      printHello() // Esta función no imprime nada
  }

Después de aplicar la corrección rápida:


  fun printHello() { println("Hello") }
  fun main() {
      printHello()
  }
