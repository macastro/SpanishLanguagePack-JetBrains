Informa sobre problemas en los archivos de propiedades contenidos en el paquete de recursos.
  • Informar sobre traducciones faltantes

    Utiliza esta opción para informar sobre propiedades que están presentes en el archivo de propiedades padre pero faltan en las propiedades heredadas (a menos que sea un dialecto del idioma).

    Ejemplo:

    
    # messages.properties
    abc=xxx
    # messages_fr.properties
    # archivo vacío
    La propiedad vacía abc se informará como no traducida.


  • Informar sobre propiedades inconsistentes

    Utiliza esta opción para informar sobre propiedades que están presentes en los archivos de propiedades heredados pero faltan en las propiedades padre (o en las propiedades del mismo nivel si no hay padre).

    Ejemplo:

    
    # messages.properties
    # archivo vacío
    # messages_fr.properties
    abc=xxx
    Aquí, la propiedad abc traducida no está disponible para ningún idioma excepto francés, por lo que se informará como faltante en el archivo de propiedades (predeterminado) messages.properties.


  • Informar sobre propiedades sobrescritas con el mismo valor

    Utiliza esta opción para informar sobre propiedades que se han copiado y pegado literalmente en múltiples archivos de propiedades.


    Ejemplo:

    
    # messages.properties
    abc=xxx
    # messages_fr.properties
    abc=xxx
    La propiedad abc se informará como heredada innecesariamente en el archivo messages_fr.properties.


  • Informar sobre propiedades sobrescritas con marcadores de posición diferentes

    Utiliza esta opción para verificar la consistencia de los marcadores de posición en las propiedades sobrescritas.

    Ejemplo:

    
    # messages.properties
    qwe={0}xxx{1}
    abc={0}yyy{1}
    # messages_fr.properties
    qwe={0}xxx{0}xxx{1}
    abc={0}yyy
    La propiedad abc se informará como una propiedad que contiene marcadores de formato de mensaje que no corresponden con messages.properties.


  • Informar sobre propiedades sobrescritas con diferentes terminaciones

    Utiliza esta opción para verificar la consistencia de las terminaciones en las propiedades sobrescritas.

    Ejemplo:

    
    # messages.properties
    abc=xxxzzz
    # messages_fr.properties
    abc=xxx;
    La propiedad abc se informará como terminada con un carácter especial (! / ? / . / : / ;) mientras que el valor padre en messages.properties no lo hace.
