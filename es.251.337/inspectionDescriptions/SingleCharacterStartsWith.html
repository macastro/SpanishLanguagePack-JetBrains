Informa sobre las llamadas a String.startsWith() y String.endsWith() donde se pasan literales de cadena de un solo carácter como argumentos.

Se sugiere reemplazar estas llamadas mediante una corrección rápida utilizando String.charAt(), que tiene una implementación más eficiente.

Sin embargo, debido a la verificación adicional de longitud no nula, la ganancia de rendimiento de tales cambios es mínima y reduce la legibilidad del código, por lo que se recomienda aplicar esta corrección rápida solo dentro de bucles ajustados.

Esta inspección es relevante para Java ME y otros entornos con recursos muy limitados. La aplicación indiscriminada de esta inspección puede tener un impacto negativo en la claridad y el diseño del código.

Ejemplo:


  boolean startsWithX(String s) {
    return s.startsWith("x");
  }

Después de aplicar la corrección rápida:


  boolean startsWithX(String s) {
    return !s.isEmpty() && s.charAt(0) == 'x';
  }
