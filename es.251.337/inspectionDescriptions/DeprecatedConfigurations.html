Detecta el uso de métodos de configuración obsoletos.

Los métodos de configuración pueden estar obsoletos por una de estas dos razones posibles:
  • La API de Gradle ha evolucionado, por lo que se debe evitar el uso de métodos antiguos;
  • Un plugin proporciona una mejor versión del método de configuración obsoleto.
Lo más probable es que exista un método alternativo para reemplazar el método obsoleto.

Ejemplo:


plugins {
  id 'java'
}
dependencies {
  archive 'org.gradle.api:gradle:1.0' // reports 'archive'
}
