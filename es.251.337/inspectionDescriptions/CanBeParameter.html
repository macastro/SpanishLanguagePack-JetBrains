Informa sobre parámetros del constructor principal donde val o var pueden ser eliminados.

Las propiedades de clase declaradas en el constructor aumentan el consumo de memoria. Si el valor del parámetro solo se utiliza en el constructor, se puede omitir.

Ten en cuenta que los objetos referenciados pueden haber sido recolectados por el garbage collector anteriormente.

Ejemplo:


  class Task(val name: String) {
      init {
          print("Task created: $name")
      }
  }

La corrección rápida eliminará las palabras clave val o var innecesarias:


  class Task(name: String) {
      init {
          print("Task created: $name")
      }
  }
