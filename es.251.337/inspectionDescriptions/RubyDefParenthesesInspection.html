Informa sobre definiciones de métodos que utilizan paréntesis de manera incorrecta. A diferencia de las definiciones de métodos sin parámetros, las definiciones de métodos con parámetros deben usar paréntesis.

Ejemplo:

# 不良做法
def example_method(); end
def example_method_with_parameters param1, param2; end
Los paréntesis se pueden eliminar o agregar mediante una corrección rápida. Después de aplicar la corrección rápida, el resultado se verá así: 
# 优良做法
def example_method; end
def example_method_with_parameters(param1, param2); end
