Informa sobre métodos setter estándar que pueden ser reemplazados por la anotación lombok @Setter.

Ejemplo:


  import java.util.Date;
  public class MyClass {
    /**
     * Fecha de procesamiento.
     */
    private Date processDate;
    /**
     * Establece la fecha.
     *
     * @param The date
     */
    public void setProcessDate(Date param) {
      processDate = param;
    }
  }

Después de aplicar la corrección rápida/limpieza:


  import lombok.Setter;
  import java.util.Date;
  @Setter
  public class MyClass {
    /**
     * Fecha de procesamiento.
     * -- SETTER --
     * Establece la fecha.
     *
     * @param The date
     */
    private Date processDate;
  }

Solo se informa cuando la biblioteca lombok está configurada. Para eliminar lombok, consulte el sitio web de lombok.
