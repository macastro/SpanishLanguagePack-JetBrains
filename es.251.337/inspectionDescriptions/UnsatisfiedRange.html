Informa sobre valores devueltos por métodos que no cumplen con el rango de retorno declarado. Puedes declarar el rango de retorno de un método usando varias anotaciones:
  • org.jetbrains.annotations.Range del paquete de anotaciones de JetBrains (especifica 'from' y 'to')
  • org.checkerframework.common.value.qual.IntRange del paquete de anotaciones de Checker Framework (especifica 'from' y 'to')
  • org.checkerframework.checker.index.qual.GTENegativeOne del paquete de anotaciones de Checker Framework (rango '>= -1')
  • org.checkerframework.checker.index.qual.NonNegative del paquete de anotaciones de Checker Framework (rango '>= 0')
  • org.checkerframework.checker.index.qual.Positive del paquete de anotaciones de Checker Framework (rango '> 0')
  • javax.annotation.Nonnegative del paquete de anotaciones JSR 305 (rango '>= 0')
  • javax.validation.constraints.Min (especifica el valor mínimo)
  • javax.validation.constraints.Max (especifica el valor máximo)

Ejemplo:


  @Range(from = 0, to = Integer.MAX_VALUE) int getValue() {
    // Advertencia: -1 está fuera del rango declarado
    return -1;
  }
