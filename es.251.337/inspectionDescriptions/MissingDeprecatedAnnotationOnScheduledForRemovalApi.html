**Inspección: Declaraciones deprecadas marcadas con `@ApiStatus.ScheduledForRemoval` sin `@Deprecated`** Informa sobre las declaraciones marcadas con `@ApiStatus.ScheduledForRemoval` sin `@Deprecated`.

Ejemplo:


  @ApiStatus.ScheduledForRemoval(inVersion = "2017.3")
  public void myLegacyMethod() { }

Después de aplicar la corrección rápida, el resultado es:


  @Deprecated
  @ApiStatus.ScheduledForRemoval(inVersion = "2017.3")
  public void myLegacyMethod() { }
