Informa sobre nombres duplicados en los alias de columnas en la lista SELECT.

Ejemplo (Sybase ASE):

CREATE TABLE t1 (a TEXT, b INT, c INT);
SELECT a AS x, b AS x FROM t1;

El alias x se usa para las columnas a y b. Estas asignaciones se resaltan como errores porque no se puede usar el mismo alias para columnas en Sybase ASE.
