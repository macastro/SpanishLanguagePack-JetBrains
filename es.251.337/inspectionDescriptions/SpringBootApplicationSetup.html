Informa sobre anotaciones redundantes de @SpringBootApplication y @EnableAutoConfiguration o @ComponentScan en el paquete predeterminado.

Esta corrección rápida elimina las anotaciones redundantes.

Ejemplo:


@SpringBootApplication
@ComponentScan // Informa 'Declaración redundante: @SpringBootApplication ya implica @ComponentScan'
public class DemoApplication {
    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
}

Después de aplicar la corrección rápida:


  @SpringBootApplication
  public class DemoApplication {
    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
  }
