Informa sobre objetos anónimos literales que implementan una interfaz Java con un único método abstract, que puede convertirse en una expresión lambda.

Ejemplo:


class SomeService {
  val threadPool = Executors.newCachedThreadPool()
  fun foo() {
    threadPool.submit(object : Runnable {
      override fun run() {
        println("hello")
      }
    })
  }
}

Después de aplicar la corrección rápida:


  fun foo() {
    threadPool.submit { println("hello") }
  }
