Informa sobre posibles escapes de this durante la inicialización de objetos. El escape ocurre cuando this se utiliza como argumento de método o como objeto de asignación en un constructor o inicializador. Estos escapes pueden provocar errores difíciles de detectar, ya que el objeto queda disponible en un contexto donde no se garantiza su inicialización.

Ejemplo:


  class Foo {
    {
      System.out.println(this);
    }
  }
