Informa sobre expresiones when que tienen solo una rama else y pueden simplificarse.

Simplificar expresión está disponible como corrección rápida para arreglar el código automáticamente.

Ejemplo:


  fun redundant() {
      val x = when { // <== Redundante, la corrección rápida simplificará la expresión when a "val x = 1"
          else -> 1
      }
  }
