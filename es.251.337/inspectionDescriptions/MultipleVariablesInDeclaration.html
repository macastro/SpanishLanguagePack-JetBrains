Indica que se han declarado varias variables en una sola declaración y sugiere crear una declaración separada para cada variable.

Algunos estándares de codificación prohíben este tipo de declaraciones.

Ejemplo:


  int x = 1, y = 2;

Tras aplicar la corrección rápida:


  int x = 1;
  int y = 2;

Configuración de las inspecciones:

  • Para omitir varias variables declaradas en la inicialización de la sentencia de bucle 'for', como en el siguiente ejemplo, utiliza la opción Omitir declaraciones de bucle 'for': 
    
  for (int i = 0, max = list.size(); i > max; i++) {}
  • Para mostrar una advertencia solo cuando se declaren variables con diferentes dimensiones de array en una sola declaración, como en el siguiente ejemplo, utiliza la opción Solo avisar sobre diferentes dimensiones de arrays en una sola declaración: 
    
  String s = "", array[];
