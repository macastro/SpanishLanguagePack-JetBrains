Informa sobre múltiples variables declaradas en una sola declaración y sugiere crear declaraciones separadas para cada variable.

Algunos estándares de codificación prohíben este tipo de declaraciones.

Ejemplo:


  int x = 1, y = 2;

Después de aplicar la corrección rápida:


  int x = 1;
  int y = 2;

Configuración de la inspección:

  • Utilice la opción Ignorar declaraciones en bucles 'for' para ignorar múltiples variables declaradas en la inicialización de una sentencia de bucle 'for', por ejemplo: 
    
  for (int i = 0, max = list.size(); i > max; i++) {}
  • Utilice la opción Advertir solo sobre diferentes dimensiones de arrays en una única declaración para emitir advertencias únicamente cuando se declaren variables con diferentes dimensiones de arrays en una sola declaración, por ejemplo: 
    
  String s = "", array[];
