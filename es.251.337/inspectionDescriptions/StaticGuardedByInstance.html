Reporta anotaciones @GuardedBy en campos o métodos static donde el lock especificado es un campo no static o this.

Proteger elementos static con elementos no static puede llevar a una concurrencia excesiva, ya que múltiples hilos pueden bloquear en diferentes contextos de objeto para acceder simultáneamente al campo protegido.

Ejemplo:


    private ReadWriteLock lock = new ReentrantReadWriteLock();
    @GuardedBy("lock")
    public static void bar() {
        // ...
    }

Las anotaciones @GuardedBy soportadas son:

  • net.jcip.annotations.GuardedBy
  • javax.annotation.concurrent.GuardedBy
  • org.apache.http.annotation.GuardedBy
  • com.android.annotations.concurrency.GuardedBy
  • androidx.annotation.GuardedBy
  • com.google.errorprone.annotations.concurrent.GuardedBy
