Informa sobre propiedades var que tienen getter y setter predeterminados y no hacen referencia a un campo de respaldo.

Estas propiedades no afectan a las llamadas a su setter; por lo tanto, sería más claro convertirlas a val y eliminar el inicializador.

Cambiar a val y eliminar el inicializador está disponible como corrección rápida para arreglar el código automáticamente.

Ejemplo:


  // Esta propiedad siempre devuelve '1', no importa que sea 'var'
  var foo: Int = 0
      get() = 1
