La inspección puede reemplazar automáticamente las sentencias switch con sentencias o expresiones switch mejoradas.

Ejemplo:


  double getPrice(String fruit) {
    // La sentencia Switch puede ser reemplazada por un 'switch' mejorado
    switch (fruit) {
      case "Apple":
        return 1.0;
      case "Orange":
        return 1.5;
      case "Mango":
        return 2.0;
      default:
        throw new IllegalArgumentException();
    }
  }

Después de aplicar la corrección rápida:


  double getPrice(String fruit) {
    return switch (fruit) {
      case "Apple" -> 1.0;
      case "Orange" -> 1.5;
      case "Mango" -> 2.0;
      default -> throw new IllegalArgumentException();
    };
  }

  • Use la opción Mostrar advertencia solo cuando sea posible convertir a expresión para no mostrar advertencias sobre la conversión a sentencias switch.
  • Use la opción Número máximo de sentencias en una rama para convertir a expresión switch para mostrar advertencias sobre la conversión a expresiones solo cuando el número de sentencias en cada rama sea menor que el número especificado.
