Reporta sentencias for mejoradas.

Ejemplo:


  for (int x: Arrays.asList(1, 2, 3)) {
    System.out.println(x);
  }

Después de aplicar la corrección rápida:


  for (Iterator<Integer> iterator = Arrays.asList(1, 2, 3).iterator(); iterator.hasNext(); ) {
    final int x = iterator.next();
    System.out.println(x);
  }

La sentencia mejorada for fue introducida en Java 5. Esta inspección ayuda a degradar el código para mantener la compatibilidad con versiones anteriores de Java.
