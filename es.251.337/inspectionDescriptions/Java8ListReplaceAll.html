El bucle puede ser colapsado en una única llamada a List.replaceAll().

Ejemplo:


  for (int i = 0; i < strings.size(); i++) {
    String str = strings.get(i).toLowerCase();
    strings.set(i, str);
  }

Después de aplicar la corrección rápida:


  strings.replaceAll(String::toLowerCase);
