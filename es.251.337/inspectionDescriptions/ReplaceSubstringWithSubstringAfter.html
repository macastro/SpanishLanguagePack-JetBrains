El informe sugiere reemplazar la llamada s.substring(s.indexOf(x)) por s.substringAfter(x).

Usar s.substringAfter(x) hace que el código sea más simple.

Esta corrección rápida reemplazará la llamada substring por substringAfter.

Ejemplo:


  fun foo(s: String) {
      s.substring(s.indexOf('x'))
  }

Después de aplicar la corrección rápida:


  fun foo(s: String) {
      s.substringAfter('x')
  }
