Informa sobre las sentencias u expresiones switch que contienen el mismo código en diferentes ramas y sugiere fusionar las ramas duplicadas.

Ejemplo:


  switch (n) {
    case 1:
      System.out.println(n);
      break;
    case 2:
      System.out.println(n);
      break;
    default:
      System.out.println("default");
  }

Después de aplicar la corrección rápida:


  switch (n) {
    case 1:
    case 2:
      System.out.println(n);
      break;
    default:
      System.out.println("default");
  }
