Reporta el uso de operadores == o != en arrays donde deberían reemplazarse por contentEquals().

Los operadores == y != comparan las referencias de los arrays en lugar de su contenido.

Ejemplo:

  fun test() {
      val a = arrayOf(1, 2, 3)
      val b = arrayOf(1, 2, 3)
      println(a == b) // comparación por referencia
  }

Después de aplicar la corrección rápida:

  fun test() {
      val a = arrayOf(1, 2, 3)
      val b = arrayOf(1, 2, 3)
      println(a.contentEquals(b))
  }
