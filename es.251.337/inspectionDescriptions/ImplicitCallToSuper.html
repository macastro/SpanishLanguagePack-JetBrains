Informa sobre constructores que no comienzan con una llamada al constructor "super" o a otro constructor de la misma clase.

Se puede asumir que estos constructores comienzan implícitamente con una llamada a super(). Algunos estándares de codificación prefieren llamar a super() explícitamente.

Ejemplo:


  class Foo {
    Foo() {}
  }

Después de aplicar la corrección rápida:


  class Foo {
    Foo() {
      super();
    }
  }

La configuración de la inspección permite ignorar las clases que extienden directamente de Object. Por ejemplo: 


  class Foo {
    Foo() {} // No informado
  }
  class Bar extends Foo {
    Bar() {} // Llamada implícita a 'super()'
  }
