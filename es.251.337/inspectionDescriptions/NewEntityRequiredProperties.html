Reporta propiedades no asignadas de Entity cuando se crea usando las funciones new o getOrCreate en un ChangeScope.

Todas las propiedades no nulables de Entity deben ser asignadas o marcadas con la anotación @Many. De lo contrario, puede producirse una excepción en tiempo de ejecución.

Ejemplo:


  interface A : LegacyEntity {
    var x: Int
    var y: Int?
    @Many var z: Int
    var p: Int
  }
  fun ChangeScope.foo() {
    new(A::class) {
      this.p = 0
    }
  }

Las asignaciones para las propiedades requeridas faltantes pueden generarse mediante una corrección rápida


  fun ChangeScope.foo() {
    new(A::class) {
      this.x = x
      this.p = 0
    }
  }
