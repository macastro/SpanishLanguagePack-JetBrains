La función hashMapOf o la llamada al constructor de HashMap puede reemplazarse por una llamada al constructor de EnumMap.

Usar el constructor de EnumMap hace que el código sea más simple.

Esta corrección rápida reemplazará la llamada a la función por una llamada al constructor de EnumMap.

Ejemplo:


  enum class E {
      A, B
  }
  fun getMap(): Map<E, String> = hashMapOf()

Después de aplicar la corrección rápida:


  enum class E {
      A, B
  }
  fun getMap(): Map<E, String> = EnumMap(E::class.java)
