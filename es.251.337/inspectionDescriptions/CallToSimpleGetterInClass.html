Informa sobre llamadas a getters simples de propiedades desde dentro de la clase que define la propiedad.

Un getter simple de propiedad se define como aquel que solo devuelve el valor del campo sin realizar otros cálculos. Esta corrección rápida permite integrar de forma segura las llamadas a estos getters simples. Algunos estándares de codificación también recomiendan evitar el uso de getters simples para mantener el código más claro.

Ejemplo:


  public class Salient {
    private String name;
    public String getName() {
      return name;
    }
    @Override
    public String toString() {
      return getName();
    }
  }

Después de aplicar la corrección rápida:


  public class Salient {
    private String name;
    public String getName() {
      return name;
    }
    @Override
    public String toString() {
      return name;
    }
  }

Configure la inspección con las siguientes opciones:

  • Si se debe informar solo sobre llamadas a getters en this y no en objetos del mismo tipo pasados como parámetros formales.
  • Si se deben ignorar los getters no private.
