Informa sobre comparaciones encadenadas que pueden simplificarse.

Ejemplo:


  def do_comparison(x):
      xmin = 10
      xmax = 100
      if x >= xmin and x <= xmax:
          pass

El IDE sugiere simplificar if x >= xmin and x <= xmax. Después de aplicar la corrección rápida, el código se convierte en:


  def do_comparison(x):
      xmin = 10
      xmax = 100
      if xmin <= x <= xmax:
          pass
