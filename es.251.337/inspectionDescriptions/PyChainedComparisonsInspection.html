Informa de comparaciones encadenadas que pueden simplificarse.

Ejemplo:


  def do_comparison(x):
      xmin = 10
      xmax = 100
      if x >= xmin and x <= xmax:
          pass

El IDE propone simplificar if x >= xmin and x <= xmax. Al aplicar la corrección rápida, el código cambia a:


  def do_comparison(x):
      xmin = 10
      xmax = 100
      if xmin <= x <= xmax:
          pass
