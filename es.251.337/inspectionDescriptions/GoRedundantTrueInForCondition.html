Reporta el literal true redundante en la condición de bucle for.

En Go, el bucle for tiene una forma con una condición implícita. Esto significa que puedes crear un bucle infinito escribiendo for { }. Por lo tanto, en casos con la condición true (for true { }), true es considerado redundante y se puede omitir.

Para más información sobre el bucle for en Go, consulta For en go.dev.

Ejemplo:

for true {
  println("ping")
}
