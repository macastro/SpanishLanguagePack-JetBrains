Informa de los siguientes errores de especificación JPA:
  • Clases de escucha de entidades sin constructor público sin argumentos
  • Métodos de escucha de entidades con firma incorrecta

Ejemplo:


  @Entity
  @EntityListeners(value = JavaEntityListener.class)
  class JavaEntity { ... }
  class JavaEntityListener {
    @PostLoad
    void postLoad(String parameter) { ... } // Error: el método 'postLoad' debe tener un parámetro de tipo 'Object' aquí
  }
