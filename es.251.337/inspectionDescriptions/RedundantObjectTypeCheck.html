Reporta comprobaciones de tipo is no idiomáticas para objetos.

Sugiere reemplazar tales comprobaciones con comparaciones de referencia.

Ejemplo:


  object Foo
  fun foo(arg: Any) = when {
      arg is Foo -> ...
      arg !is Foo -> ...
  }

Después de aplicar la corrección rápida:


  object Foo
  fun foo(arg: Any) = when {
      arg === Foo -> ...
      arg !== Foo -> ...
  }
