Informa sobre campos de colección public static que son modificables.

Aunque estos campos se utilizan frecuentemente para almacenar colecciones de valores constantes, siguen representando un riesgo de seguridad ya que su contenido puede ser modificado incluso cuando el campo está declarado como final.

Ejemplo:


  public static final List<String> EVENTS = new ArrayList<>();

Utilice la tabla en la sección de Opciones para especificar métodos que devuelvan colecciones inmutables. Los campos de colección public static inicializados usando estos métodos no serán reportados.
