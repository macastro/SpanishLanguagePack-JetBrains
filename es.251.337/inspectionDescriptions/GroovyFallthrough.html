Informa sobre fallthrough en las declaraciones switch. Aunque ocasionalmente útil, el fallthrough suele ser involuntario y puede provocar errores inesperados.

Ejemplo:


switch(n) {
  case 1:
    print 1
  case 2: // "case 1" cae directamente a "case 2". Las declaraciones que comienzan en "case 2" se ejecutarán inmediatamente después de "case 1".
    print 2
    break
  default:
    print "Default"
}
