El constructor de Pair puede ser reemplazado por la función infija to().

Las llamadas explícitas al constructor pueden añadir redundancia, especialmente cuando se usan múltiples veces. Reemplazar las llamadas al constructor con to() hace que el código sea más fácil de leer y mantener.

Ejemplo:


  val countries = mapOf(
      Pair("France", "Paris"),
      Pair("Spain", "Madrid"),
      Pair("Germany", "Berlin")
  )

Después de aplicar la corrección rápida:


  val countries = mapOf(
      "France" to "Paris",
      "Spain" to "Madrid",
      "Germany" to "Berlin"
  )
