Configuración incorrecta del modo 'mode' en la anotación @DirtiesContext.

Ejemplo:


  @ContextConfiguration
  @DirtiesContext(methodMode = MethodMode.AFTER_METHOD, // No tiene sentido establecer el methodMode en una clase de prueba anotada. Para control a nivel de clase, use classMode en su lugar.
                  hierarchyMode = DirtiesContext.HierarchyMode.CURRENT_LEVEL) // hierarchyMode debe usarse cuando el contexto está configurado como parte de una jerarquía mediante @ContextHierarchy
  public class MyTest {
    @DirtiesContext(classMode = DirtiesContext.ClassMode.AFTER_CLASS, // No tiene sentido establecer el classMode en un método de prueba anotado. Para control a nivel de método, use methodMode en su lugar.
                     hierarchyMode = DirtiesContext.HierarchyMode.CURRENT_LEVEL) // hierarchyMode debe usarse cuando el contexto está configurado como parte de una jerarquía mediante @ContextHierarchy
    public void testFoo() {...}
  }
