Informa sobre propiedades lateinit var que sobrescriben otras propiedades lateinit var.

Una sola propiedad en una instancia de subclase tendrá dos campos, mientras que el campo heredado de la superclase permanecerá sin usar.

Ejemplo:


    open class BaseClass {
      open lateinit var name: String
    }
    class RealClass : BaseClass() {
      override lateinit var name: String
    }
