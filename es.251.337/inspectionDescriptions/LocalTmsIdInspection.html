Informa sobre IDs faltantes y duplicados en las entradas locales de TMS declaradas en Markdown.

No es posible vincular elementos TMS al código (pruebas unitarias) cuando faltan IDs o estos son inválidos.

La corrección rápida generará un ID único.

Se muestra una advertencia cuando falta el ID del caso de prueba y un error cuando no es único. Se muestra un error cuando el ID de la suite/ejecución de pruebas no es único.
