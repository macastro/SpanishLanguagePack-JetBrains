Informa sobre parámetros @PathParam que están declarados en la firma del método pero faltan en la ruta URL, o que están declarados en la ruta URL pero faltan en la firma del método.

Ejemplo:


  @Get("/path/{myVariable}/")
  public String handler(@PathParam("name_is_not_equal_to_myVariable") String myVariable) { // error
    return "...";
  }
