Informa sobre el uso de elementos obsoletos.

Ejemplo:

// Obsoleto: usar io.SeekStart, io.SeekCurrent y io.SeekEnd.
const (
 SEEK_SET int = 0 // Buscar relativo al origen del archivo
 SEEK_CUR int = 1 // Buscar relativo al desplazamiento actual
 SEEK_END int = 2 // Buscar relativo al final
)

Según las Constantes en go.dev, SEEK_SET, SEEK_CUR y SEEK_END están obsoletos.
