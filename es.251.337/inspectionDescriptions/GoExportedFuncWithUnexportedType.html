Informa sobre funciones exportadas que tienen tipos de retorno no exportados.

Los tipos no exportados pueden ser difíciles de utilizar cuando se consulta la documentación en go doc.

Ejemplo:

type hidden struct{}
func Exported() hidden { // función exportada con tipo de retorno no exportado 'hidden'
 return hidden{}
}

Puede aplicar la corrección rápida Exportar para exportar el tipo. Después de aplicar la corrección rápida, el nombre del tipo se convertirá en mayúscula:

type Hidden struct{}
func Exported() Hidden { // corregido
 return Hidden{}
}
