Informa sobre llamadas a métodos de colecciones parametrizadas donde el tipo real del argumento no coincide con el tipo de elemento de la colección.

Ejemplo:


  List<Integer> list = getListOfElements();
  list.remove(""); //remove será resaltado

En la configuración de la inspección, se puede deshabilitar las advertencias para código potencialmente correcto, como se muestra a continuación: 


  public boolean accept(Map<Integer, String> map, Object key) {
    return map.containsKey(key);
  }
