Informa sobre parámetros de palabras clave duplicados en métodos, bloques y tipos proc. Solo se reconoce la primera declaración de cada parámetro de palabra clave; todas las demás declaraciones serán ignoradas.

Ejemplo:

class Example
  # 重复的关键字形参
  def call: (keyword: untyped, keyword: untyped) -> untyped
end
