El informe muestra las clases de utilidad que no tienen un constructor private.

Las clases de utilidad declaran todos sus campos y métodos como static. Crear un constructor private en una clase de utilidad evita que sea instanciada accidentalmente.

Utilice la opción Ignorar cuando tiene anotaciones para especificar anotaciones especiales. La inspección ignorará las clases marcadas con cualquiera de estas anotaciones.

Use la opción Ignorar clases que solo tienen método main para ignorar las clases que no tienen otros métodos además del método main.
