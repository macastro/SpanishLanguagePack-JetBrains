Informa sobre campos anotados como no nulos que no están inicializados en el constructor.

Ejemplo:

public class MyClass {
  private @NotNull String value;
  public void setValue(@NotNull String value) {
    this.value = value;
  }
  public @NotNull String getValue() {
    return value;
  }
}

Estos campos pueden violar la restricción de no nulidad. En el ejemplo anterior, el parámetro de setValue está anotado como no nulo, pero si el setter no se llama, getValue podría devolver null.
