Informa sobre las llamadas al método java.util.Map.get(). Estas llamadas pueden ser reemplazadas por una forma de acceso mediante clave más corta y explícita.

Ejemplo:


def map = ["foo": "bar"]
def str = map.get("foo") // map.get("foo") puede reemplazarse por map["foo"]

Después de aplicar la corrección rápida:


def map = ["foo": "bar"]
def str = map["foo"]
