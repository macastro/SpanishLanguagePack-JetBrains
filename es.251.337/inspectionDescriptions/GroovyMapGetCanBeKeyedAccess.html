Se reportan llamadas a métodos java.util.Map.get(). Dichas llamadas podrían reemplazarse por la forma más corta y clara de acceso con clave.

Ejemplo:


def map = ["foo": "bar"]
def str = map.get("foo") // map.get("foo") podría reemplazarse por map["foo"]

Después de aplicar la corrección rápida:


def map = ["foo": "bar"]
def str = map["foo"]
