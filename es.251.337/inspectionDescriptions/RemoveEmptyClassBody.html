Informa sobre declaraciones de clases y objetos con cuerpos vacíos.

Utiliza la corrección rápida "Eliminar cuerpo de clase redundante vacío" para limpiar el código.

Ejemplo:


  class EmptyA() {} // <== cuerpo vacío
  class EmptyB {
      companion object {} // <== cuerpo vacío
  }
  fun emptyC() {
     object {} // <== objeto anónimo, permitido (no reportado)
  }

Después de aplicar la corrección rápida:


  class EmptyA()
  class EmptyB {
      companion object
  }
  fun emptyC() {
     object {}
  }
