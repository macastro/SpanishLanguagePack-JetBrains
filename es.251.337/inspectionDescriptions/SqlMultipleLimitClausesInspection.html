Inspecciona los usos de múltiples cláusulas limitantes de filas en una sola consulta.

Ejemplo (Microsoft SQL Server):

create table foo(a int);
select top 1 * from foo order by a offset 10 rows fetch next 20 rows only;

La cláusula SELECT TOP se usa para especificar que solo se debe devolver 1 registro. La cláusula FETCH especifica el número de filas a devolver después de que la cláusula OFFSET haya sido procesada. Pero como ya tenemos la cláusula limitante SELECT TOP, la cláusula FETCH podría ser redundante.
