Informa sobre directivas requires redundantes en el archivo module-info.java del Sistema de Módulos de Java Platform. Cuando el módulo A requiere el módulo B, pero el código en el módulo A no importa ningún paquete o clase de B, la directiva requires es redundante. Además, todos los módulos tienen una dependencia implícita declarada del módulo java.base, por lo que la directiva requires java.base; siempre es redundante.

La corrección rápida elimina las directivas requires redundantes. Si la dependencia eliminada reexportaba módulos que se utilizan actualmente, la corrección añadirá directivas requires para esos módulos.

Esta inspección solo informa cuando el nivel de lenguaje del proyecto o módulo es 9 o superior.
