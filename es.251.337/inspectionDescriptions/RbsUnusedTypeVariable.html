Informa los contenedores, métodos y las variables de tipos alias que no se usan en el proyecto.

Las variables de tipo de contenedor solo se verifican si el contenedor tiene exactamente una declaración.

Se ofrece una corrección rápida para eliminar la variable de tipo no utilizada.

Ejemplo:

# La variable de tipo 'A' de la clase 'Example' no se usa
class Example[A]
end

Después de la corrección rápida:

class Example
end
