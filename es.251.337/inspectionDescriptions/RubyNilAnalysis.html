Informa sobre posibles desreferencias de nil. La desreferencia de nil puede provocar un NoMethodError.

Usando la lista de receptores ignorados, puedes especificar las llamadas a métodos que deben ignorarse como receptores (por ejemplo, Foo#call o Foo::call para métodos de instancia o singleton respectivamente).

Ejemplo:

def foo
  rand(2) == 1 ? nil : 1
end
a = foo
a.nomethod # La llamada al método puede producir NoMethodError
