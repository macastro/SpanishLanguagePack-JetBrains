Informa sobre llamadas a métodos synchronized dentro de bloques o métodos native.

Cuando sea posible, es mejor mantener las llamadas a métodos native fuera del contexto de sincronización, ya que estas llamadas pueden causar cambios de contexto costosos y potenciales problemas de rendimiento.

Ejemplo:


  native void nativeMethod();
  void example(){
    synchronized (lock){
      nativeMethod();//advertencia
    }
  }
