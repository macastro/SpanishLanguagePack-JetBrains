Reporta llamadas sospechosas a System.arraycopy().

Este tipo de llamadas son sospechosas en los siguientes casos:

  • El origen o destino no son de tipo array
  • Los tipos de origen y destino son diferentes
  • La longitud del bloque copiado es mayor que src.length - srcPos
  • La longitud del bloque copiado es mayor que dest.length - destPos
  • Cuando el origen y destino son el mismo array, los rangos siempre se superponen

Ejemplo:


  void foo() {
    int[] src = new int[] { 1, 2, 3, 4 };
    System.arraycopy(src, 0, src, 1, 2); // Advertencia: copia al mismo array con rangos superpuestos
  }
