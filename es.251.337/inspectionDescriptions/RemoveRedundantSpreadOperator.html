Informa sobre el uso de un operador de propagación redundante para una familia de llamadas de funciones arrayOfUsa el arreglo rápido "Eliminar operador de propagación redundante" para limpiar el código.

Ejemplos:


  fun foo(vararg s: String) { }

  fun bar(ss: Array<String>) {
      foo(*arrayOf("abc"))       // para las dos llamadas de 'foo', la creación del array
      foo(*arrayOf(*ss, "zzz"))  // y su "propagación" posterior es redundante    
  }

Después de aplicar el arreglo rápido:


  fun foo(vararg s: String) { }

  fun bar(ss: Array<String>) {
      foo("abc")
      foo(*ss, "zzz")
  }
