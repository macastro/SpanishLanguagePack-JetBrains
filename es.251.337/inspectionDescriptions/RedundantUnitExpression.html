Reporta expresiones redundantes de Unit.

En Kotlin, Unit puede usarse como tipo de retorno para funciones que no devuelven ningún valor significativo. El tipo Unit solo tiene un valor posible, que es el objeto Unit.

Ejemplo:


  fun redundantA(): Unit {
      return Unit // Redundante, 'Unit' se devuelve por defecto y coincide con el tipo de retorno esperado
  }
  fun requiredA(condition: Boolean): Any {
      if (condition) return "hello"
      return Unit // 'Unit' explícito es necesario porque el tipo esperado es 'Any'
  }
  fun redundantB(condition: Boolean): Any = if (condition) {
      fun ancillary(): Int = 1
      println("${ancillary()}")
      Unit // Redundante porque la expresión anterior ya es de tipo 'Unit'
  } else {
      println("else")
  }
  fun requiredB(condition: Boolean): Any = if (condition) {
      1024
      Unit // Necesario, de lo contrario '1024' (Int) sería el valor de retorno
  } else {
      println("else")
  }
