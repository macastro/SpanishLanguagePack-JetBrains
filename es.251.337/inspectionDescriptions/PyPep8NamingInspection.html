Informa sobre violaciones de las convenciones de nomenclatura de PEP8.

Ejemplo:


class mammalia(object):
    extremities = 4
    def feeds(self):
        print("milk")

En este código, el IDE sugiere renombrar mammalia a Mammalia. Después de aplicar la corrección rápida, el código queda así:


class Mammalia(object):
    extremities = 4
    def feeds(self):
        print("milk")
