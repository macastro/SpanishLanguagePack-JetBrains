Reporta problemas de estilo de código detectados por PHP Mess Detector.

Esta inspección requiere que PHP Mess Detector esté correctamente instalado y configurado en Configuración | PHP | Herramientas de calidad | Mess Detector.

Para más información sobre la instalación de PHP Mess Detector, consulte PHP Mess Detector installation (phpmd.org).

Puede utilizar conjuntos de reglas predefinidos o agregar sus propias reglas especificando un archivo ruleset.xml válido. Para más detalles, consulte rules index (phpmd.org) y How to create a custom rule set (phpmd.org).
