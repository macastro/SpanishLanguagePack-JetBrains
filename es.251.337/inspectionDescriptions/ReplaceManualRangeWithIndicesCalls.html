La notificación puede reemplazarse con Collection.indices o los operadores until y rangeTo para la iteración de colecciones en bucles for.

El uso de azúcar sintáctico puede hacer que el código sea más simple.

Esta corrección rápida reemplazará el rango manual con la construcción correspondiente.

Ejemplo:


  fun main(args: Array<String>) {
      for (index in 0..args.size - 1) {
          println(args[index])
      }
  }

Después de aplicar la corrección rápida:


  fun main(args: Array<String>) {
      for (element in args) {
          println(element)
      }
  }
