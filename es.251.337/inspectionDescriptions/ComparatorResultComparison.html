Informa sobre comparaciones del resultado de llamadas a Comparator.compare() o Comparable.compareTo() con constantes distintas de cero. Por convención, estos métodos pueden devolver cualquier número entero (no solo -1, 0 o 1), por lo que comparar con números específicos es una mala práctica. Algunos métodos de comparación ampliamente utilizados (como String.compareTo()) en realidad devuelven valores fuera del rango [-1..1], y tales comparaciones pueden llevar a un comportamiento incorrecto del programa.

Ejemplo:


  void validate(String s1, String s2) {
    // Comparación incorrecta con 1
    if (s1.compareTo(s2) == 1) {
      throw new IllegalArgumentException("Incorrect order");
    }
  }

Después de aplicar la corrección rápida:


  void validate(String s1, String s2) {
    if (s1.compareTo(s2) > 0) {
      throw new IllegalArgumentException("Incorrect order");
    }
  }
