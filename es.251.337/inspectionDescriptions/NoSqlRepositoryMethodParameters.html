Reporta tipos de parámetros de método de repositorio incorrectos.

Ejemplo:


  public class Person {
    private int id;
    @Column
    private String lastname;

    @Column
    private int age;
  }

  public interface CustomerRepository extends Repository<Person, Integer> {
    List<Person> findByAddress(int lastname);  // Se necesita tipo 'String'
    List<Person> deleteByAge(String str);   // Se necesita tipo 'int'
  }
