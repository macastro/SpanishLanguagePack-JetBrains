Reporta creación de nuevas instancias de clases anotadas con @groovy.lang.Singleton. Tales construcciones pueden conducir a excepciones en tiempo de ejecución No se puede instanciar singleton.

Ejemplo:


  @Singleton
  class Foo{
  }
  
  Foo foo = new Foo()

Después de aplicar la corrección rápida:


  @Singleton
  class Foo{
  }
  
  Foo foo = Foo.instance
