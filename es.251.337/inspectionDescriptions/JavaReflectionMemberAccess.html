Informa sobre accesos por reflexión a campos y métodos inexistentes o no visibles.

Ejemplo:


  Field stringHashField() throws NoSuchFieldException {
    return String.class.getField("hash");
  }

Después de aplicar la corrección rápida:


  Field stringHashField() throws NoSuchFieldException {
    return String.class.getDeclaredField("hash");
  }

Para clases final, es evidente si existe un campo o método con el nombre especificado en la clase.

Para clases no final, pueden producirse falsos positivos ya que los campos o métodos con ese nombre podrían existir en las subclases. La configuración de esta inspección permite suprimir estos falsos positivos en cualquier lugar o para clases específicas.
