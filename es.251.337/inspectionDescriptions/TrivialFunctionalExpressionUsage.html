Informa sobre llamadas directas a métodos de interfaces funcionales en definiciones de lambdas, referencias a métodos o clases anónimas. Estas llamadas a métodos pueden reemplazarse por la implementación de la interfaz funcional.

Ejemplo:


  boolean contains(List<String> names, String name) {
    return ((Predicate<String>)x -> {
      return names.contains(x);
    }).test(name);
  }

Después de aplicar la corrección rápida, la llamada al método se convierte en:


  boolean contains(List<String> names, String name) {
    return names.contains(name);
  }
