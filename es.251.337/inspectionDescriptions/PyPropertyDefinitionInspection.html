Informa sobre problemas con los argumentos de property() y funciones decoradas con @property.


class C:
    @property
    def abc(self):  # El Getter debe retornar o generar algún contenido
        pass
    @abc.setter
    def foo(self, value):  # El nombre de la función no coincide con el decorador
        pass
    @abc.setter
    def abc(self, v1, v2):  # La firma del Setter debe ser (self, value)
        pass
    @abc.deleter
    def abc(self, v1):  # La firma del Deleter debe ser (self)
        pass

La corrección rápida sugiere actualizar los parámetros.
