Informa sobre código que utiliza interfaces de escucha de eventos obsoletas en GWT 1.6.

Las interfaces *Listener del paquete com.google.gwt.user.client.ui (por ejemplo, com.google.gwt.user.client.ui.ClickListener) están obsoletas desde GWT 1.6.

La corrección rápida migra el código para utilizar las interfaces *Handler correspondientes.
