Reporta un object (incluyendo data object) que implementa java.io.Serializable pero no implementa readResolve

Ejemplo:


  import java.io.Serializable
  object Foo : Serializable

La corrección rápida implementará el método readResolve:


  import java.io.Serializable
  object Foo : Serializable {
      private fun readResolve() = Foo
  }
