Informa sobre llamadas a métodos modificadores de Java (como fill, reverse, shuffle, sort) en colecciones inmutables de Kotlin.

Esto puede provocar una UnsupportedOperationException en tiempo de ejecución.

Ejemplo:


  import java.util.Collections
  fun test() {
      val immutableList = listOf(1, 2)
      Collections.reverse(immutableList)
  }

Para corregir este problema, haga que la lista sea mutable.
