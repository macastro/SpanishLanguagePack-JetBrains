Informa sobre métodos de interfaz que entran en conflicto con los métodos protected clone() y finalize() de la clase java.lang.Object.

En las interfaces, estos métodos pueden declararse con tipos de retorno incompatibles con los métodos de java.lang.Object. Las clases que implementan dichas interfaces no se pueden compilar. Aunque aún es posible crear lambdas a partir de la interfaz cuando funciona, no se recomienda hacerlo.

Ejemplo:


  // // Advertencia: esta interfaz no puede ser implementada por ninguna clase,
  // solo puede ser implementada por lambdas o referencias a métodos
  interface MyInterface {
    double clone();
  }
