Informa sobre llamadas a toString() que pueden eliminarse de forma segura en plantillas de cadenas.

Ejemplo:

  fun foo(a: Int, b: Int) = a + b
  fun test(): String {
      return "Foo: ${foo(0, 4).toString()}" // 'toString()' redundante
  }

Después de aplicar la corrección rápida:

  fun foo(a: Int, b: Int) = a + b
  fun test(): String {
      return "Foo: ${foo(0, 4)}"
  }
