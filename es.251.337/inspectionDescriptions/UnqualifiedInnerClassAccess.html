Informa sobre referencias a clases internas que no utilizan nombres de clase envolvente calificados.

Ejemplo:


  import foo.Foo.Bar;
  class Foo {
    class Bar {}
  }
  class Baz {
    void f(Bar bar) {}
  }

Después de aplicar la corrección rápida:


  class Foo {
    class Bar {}
  }
  class Baz {
    void f(Foo.Bar bar) {}
  }

La configuración de la inspección permite ignorar las referencias a clases internas dentro de la misma clase, por lo que estas clases no requieren importación.
