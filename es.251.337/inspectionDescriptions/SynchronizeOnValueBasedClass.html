Informa sobre intentos de sincronización en instancias de clases basadas en valores, lo que genera advertencias de compilación y excepciones en tiempo de ejecución a partir de Java 16.

Por ejemplo, java.lang.Double está anotado con jdk.internal.ValueBased, por lo que el siguiente código genera una advertencia durante la compilación: 


Double d = 20.0;
synchronized (d) { ... } // advertencia de javac
