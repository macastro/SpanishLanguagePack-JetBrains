Informa sobre llamadas a get() en Optional sin verificar si contiene un valor.

Llamar a Optional.get() en una instancia vacía de Optional lanzará una excepción.

Ejemplo:


  void x(List<Integer> list) {
    final Optional<Integer> optional =
      list.stream().filter(x -> x > 10).findFirst();
    final Integer result = optional.get(); // Esto es problemático
  }
