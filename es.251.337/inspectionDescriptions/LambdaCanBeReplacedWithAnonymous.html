El informe puede ser reemplazado por una expresión lambda de clase anónima.

La expansión de una expresión lambda a una clase anónima es útil cuando se necesita implementar métodos adicionales en la clase anónima.

Ejemplo:


  s -> System.out.println(s)

Después de aplicar la corrección rápida:

new Consumer<String>() {
  @Override
  public void accept(String s) {
    System.out.println(s);
  }
}
