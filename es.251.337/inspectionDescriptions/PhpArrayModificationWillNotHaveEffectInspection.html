Informa sobre llamadas a funciones que devuelven arrays por valor (lo que hace que las asignaciones sean inefectivas). Se debe cambiar para devolver por referencia.

Por ejemplo:


<?php
class A {
    private $values = [];
    public function getValues() {
        return $this->values;
    }
}
$config = new A();
$config->getValues()['a'] = 'a';
