Reporta un constructor principal private en una clase de datos.

Las clases data tienen un método de fábrica copy() que puede usarse como un constructor. Para proporcionar suficiente seguridad, el constructor no debería marcarse como private.

Ejemplo:


  data class User private constructor(val name: String)

Esta corrección rápida cambiará el modificador de visibilidad del constructor a public:


  data class User(val name: String)
