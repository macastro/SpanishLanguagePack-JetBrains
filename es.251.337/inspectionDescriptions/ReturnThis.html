Métodos que devuelven this.

Aunque este tipo de retorno es válido, raramente es necesario y generalmente indica que el método está diseñado para ser parte de una cadena de llamadas de método (por ejemplo, buffer.append("foo").append("bar").append("baz")). Muchos estándares de codificación desaconsejan este tipo de encadenamiento.

Ejemplo:


  public Builder append(String str) {
    // [...]
    return this;
  }
