Se informa que la operación de conversión de tipo puede ser reemplazada por una variable local o de patrón existente con el mismo valor.

Ejemplo:


  void foo(Object obj) {
    String s = (String) obj;
    System.out.println(((String) obj).trim());
  }

Después de aplicar la corrección rápida:


  void foo(Object obj) {
    String s = (String) obj;
    System.out.println(s.trim());
  }
