Informa sobre columnas en la cláusula USING que no existen en ninguna de las dos tablas.

Ejemplo (MySQL):

CREATE TABLE t1 (i INT, j INT);
CREATE TABLE t2 (k INT, l INT);
SELECT * FROM t1 JOIN t2 USING (j);

En la cláusula USING, los nombres de las columnas deben existir en ambas tablas, ya que la consulta SELECT unirá automáticamente las tablas usando el nombre de columna proporcionado. Como no tenemos una columna j en t2, podemos reescribir la consulta usando ON. La cláusula ON permite unir tablas donde los nombres de las columnas no coinciden.

SELECT * FROM t1 JOIN t2 ON t1.j = t2.l;
