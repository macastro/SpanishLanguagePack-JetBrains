La expresión lambda puede ser reemplazada por una referencia a función.

Reemplazar lambdas con referencias a funciones generalmente hace que el código sea más conciso y legible.

Ejemplo:


  fun Int.isEven() = this % 2 == 0
  fun example() {
      val numbers = listOf(1, 2, 4, 7, 9, 10)
      val evenNumbers = numbers.filter { it.isEven() }
  }

Después de aplicar la corrección rápida:


  fun Int.isEven() = this % 2 == 0
  fun example() {
      val numbers = listOf(1, 2, 4, 7, 9, 10)
      val evenNumbers = numbers.filter(Int::isEven)
  }
