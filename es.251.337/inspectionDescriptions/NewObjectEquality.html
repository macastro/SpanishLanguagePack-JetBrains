Reporta código que aplica == o != a objetos recién asignados en lugar de llamar a equals().

Una referencia a un objeto recién asignado no puede apuntar a un objeto existente, por lo que la comparación siempre será false. La inspección también puede reportar objetos recién creados que se devuelven desde métodos simples.

Ejemplo:


  void test(Object obj) {
    if (new Object() == obj) {...}
  }

Después de aplicar la corrección rápida:


  void test(Object obj) {
    if (new Object().equals(obj)) {...}
  }
