Reporta valores no manejados en la declaración switch cuando el identificador iota aparece en una declaración const. Para manejar estos valores, considere usar cláusulas case o default.

La palabra clave iota en Go crea una secuencia de enteros dentro de un bloque const. Aunque el compilador de Go no requiere manejar todos los valores de iota, la ausencia de algunos puede indicar un bug.

Para más información sobre iota, consulte Iota en go.dev.

Ejemplo:

const (
  a = iota
  b
  c = -5
)
func main() {
  v := 5
  switch v {
  case a:
    break;
  case b:
    break;
  }
}

El IDE activa la inspección porque c no está manejado por la declaración switch. Observe que c está en el mismo bloque const, incluso si no utiliza iota.

Las opciones de corrección rápida son crear cláusula 'case' para el valor y crear cláusula 'default'. Después de aplicar la corrección rápida crear cláusula 'default':

const (
  a = iota
  b
  c
)
func main() {
  v := 5
  switch v {
  case a:
    break;
  default:
  }
}
