Informa sobre métodos getter estándar que pueden ser reemplazados por la anotación lombok @Getter.

Ejemplo:


  import java.util.Date;
  public class MyClass {
    /**
     * Fecha de procesamiento.
     */
    private Date processDate;
    /**
     * Devuelve la fecha.
     *
     * @return The date
     */
    public Date getProcessDate() {
      return processDate;
    }
  }

Después de aplicar la corrección rápida/limpieza:


  import lombok.Getter;
  import java.util.Date;
  @Getter
  public class MyClass {
    /**
     * Fecha de procesamiento.
     * -- GETTER --
     * Devuelve la fecha.
     *
     * @return The date
     */
    private Date processDate;
  }

Solo se informa cuando la biblioteca lombok está configurada. Para eliminar lombok, consulte el sitio web de lombok.
