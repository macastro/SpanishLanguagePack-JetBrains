Informa sobre accesos a campos static no constantes que están sincronizados sobre this o sobre campos de instancia de this.

Sincronizar campos static sobre datos de instancia no evita que otras instancias modifiquen el campo, lo que puede llevar a condiciones de carrera inesperadas.

Ejemplo:


  static String test;
  public void foo() {
      synchronized (this) {
          System.out.println(test); // advertencia
      }
  }

Los campos static de tipos específicos pueden ser ignorados mediante una corrección rápida. Los tipos ignorados pueden gestionarse en las opciones de inspección.

Las clases que deben ignorarse para campos static pueden especificarse usando las opciones de inspección.
