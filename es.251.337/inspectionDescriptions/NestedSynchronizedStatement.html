Informa sobre declaraciones synchronized anidadas. Se recomienda evitar la sincronización anidada ya que, en algunos casos, puede provocar deadlocks.

Ejemplo:


  synchronized (lockA){
    //El hilo 1 está esperando lockB
    synchronized (lockB){ //advertencia
    }
  }
  ...
  synchronized (lockB) {
    //El hilo 2 está esperando lockA
    synchronized (lockA) { //advertencia
    }
  }
