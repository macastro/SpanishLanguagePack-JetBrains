Informa sobre métodos dentro de una clase que solo difieren en mayúsculas y minúsculas. Tales nombres pueden resultar muy confusos.

Ejemplo: 


  public int hashcode() { // reportado, ¿debería ser hashCode?
    return 0;
  }

La corrección rápida para renombrar estos métodos solo está disponible en el editor.

Use la opción Ignorar métodos que sobrescriben/implementan métodos super para ignorar los métodos que sobrescriben o implementan métodos de la superclase.
