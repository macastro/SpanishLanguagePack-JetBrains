Informa sobre objetos no Serializable utilizados como argumentos en java.io.ObjectOutputStream.write(). Estas llamadas provocarán una excepción en tiempo de ejecución.

Esta inspección asume que los objetos de tipo java.util.Collection y java.util.Map son Serializable, a menos que sus tipos declarados no sean Serializable.

Ejemplo:


  public class IWantToSerializeThis {
    public static void main(String[] args) throws IOException {
      try(var stream = new ObjectOutputStream(Files.newOutputStream(Paths.get("output")))) {
        // Advertencia -- fallará y lanzará NotSerializableException
        stream.writeObject(new IWantToSerializeThis());
      }
    }
  }
