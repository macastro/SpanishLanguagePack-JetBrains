Informa sobre clases declaradas como final. No informa sobre clases final que extienden clases o interfaces sealed. Estas clases no pueden ser heredadas y pueden indicar una falta de diseño orientado a objetos. Algunos estándares de codificación desaconsejan el uso de clases final.

Ejemplo:


  public final class Main {
  }

Después de aplicar la corrección rápida:


  public class Main {
  }
