Las excepciones reportadas siempre pueden ser capturadas por una sentencia throw incluida en una sentencia try.

Usar la sentencia throw como un "goto" para cambiar el flujo de control local puede ser confuso y resultar en un rendimiento deficiente.

Ejemplo:


  try {
    if (!Files.isDirectory(PROJECTS)) {
        throw new IllegalStateException("Directory not found."); // Advertencia: 'throw' es capturado por la sentencia 'try' que lo contiene
    }
    ...
  } catch (Exception e) {
      LOG.error("run failed");
  }
