Informa sobre los accesos a la propiedad .length de arrays en las condiciones de bucles. En entornos con recursos muy limitados, estas llamadas pueden tener un impacto negativo en el rendimiento.

Esta inspección es relevante para Java ME y otros entornos con recursos muy limitados. La aplicación indiscriminada de esta inspección puede tener un impacto negativo en la claridad y el diseño del código.

Ejemplo:


  void foo(Object[] x) {
    for (int i = 0; i < x.length; i++) { /**/ }
  }
