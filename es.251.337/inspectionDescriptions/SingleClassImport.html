Informa sobre declaraciones import que importan una clase individual (en lugar de todo el paquete).

Algunos estándares de codificación prohíben este tipo de declaraciones import.

Puede configurar IntelliJ IDEA para detectar y corregir estas declaraciones usando el comando Optimizar importaciones. Vaya a Configuración | Editor | Estilo de código | Java | Importaciones y desmarque la casilla Usar importación de clase única. Por lo tanto, esta inspección es útil para informes fuera de línea de bases de código que no se pretenden modificar.
