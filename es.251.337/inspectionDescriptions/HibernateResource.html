Reporta las llamadas al método openSession() cuando el recurso org.hibernate.Session devuelto no se cierra de manera segura.

Por defecto, esta inspección asume que el recurso puede ser cerrado por cualquier método que contenga 'close' o 'cleanup' en su nombre.

Ejemplo:


  void doHibernateQuery(SessionFactory factory) {
    Session session = factory.openSession(); //advertencia
    session.createQuery("...");
  }

Configure la inspección con las siguientes opciones:

  • Si se permite abrir recursos org.hibernate.Session dentro de un bloque try. Este estilo es menos preferible ya que es más verboso que abrir el recurso antes del bloque try.
  • Si el recurso puede ser cerrado mediante cualquier llamada a método que tome el recurso como argumento.
