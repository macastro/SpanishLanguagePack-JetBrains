Informa sobre expresiones aritméticas y de bits con operandos no soportados.

A partir de PHP 8.0, el uso de operadores aritméticos y de bits +, -, *, /, **, %, < <, >, &, |, ^, ~, ++, - resultará en un TypeError cuando uno de los operandos sea un array, recurso u objeto no sobrecargado. La única excepción es la operación de unión array + array, que sigue siendo compatible.

Para ver la lista completa de operaciones, consulte el código fuente de pruebas internas de PHP.
