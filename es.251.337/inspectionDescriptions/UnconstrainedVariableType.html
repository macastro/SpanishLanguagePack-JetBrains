Informa cuando una expresión restringida por un concepto se asigna a una variable declarada como auto y sugiere agregar restricciones de tipo.

Ejemplo:


  template<class T>
  concept Foo = true;

  Foo auto func() { return true; }

  void bar() {
      auto f = func(); // Aquí se advertirá para agregar la restricción 'Foo'
  }
