Informa sobre cualquier llamada a métodos de las clases junit.framework.Assert, org.junit.Assert o org.junit.Assume en pruebas JUnit 5.

Aunque estas pruebas funcionan correctamente, migrar a org.junit.jupiter.api.Assertions/org.junit.jupiter.api.Assumptions ayuda a evitar dependencias de versiones anteriores de JUnit.

Ejemplo:


  import org.junit.Assert;
  import org.junit.jupiter.api.Test;
  public class MyTest {
    @Test
    public void simpleTest() {
      Assert.assertEquals(4, 2 + 2);
    }
  }

Después de aplicar la corrección rápida:


  import org.junit.jupiter.api.Assertions;
  import org.junit.jupiter.api.Test;
  public class MyTest {
    @Test
    public void simpleTest() {
      Assertions.assertEquals(4, 2 + 2);
    }
  }
