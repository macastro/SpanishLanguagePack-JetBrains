Reporta expresiones if que comprueban si una variable es null inmediatamente después de su inicialización, las cuales pueden convertirse en un operador elvis en el inicializador.

Ejemplo:


  fun test(foo: Int?, bar: Int): Int {
      var i = foo
      if (i == null) {
          return bar
      }
      return i
  }

La corrección rápida convierte la expresión if con el inicializador en una expresión elvis:


  fun test(foo: Int?, bar: Int): Int {
      var i = foo ?: return bar
      return i
  }
