Informa una sentencia bloque que no se utiliza como el cuerpo de sentencias if, for, while, do, with o try, o como el cuerpo de una declaración de función. A partir de ECMAScript 6, los bloques JavaScript introducen nuevos ámbitos para las variables let y const, pero las sentencias de bloque independientes aún pueden ser confusas y dar lugar a errores sutiles cuando se utilizan con variables var.
