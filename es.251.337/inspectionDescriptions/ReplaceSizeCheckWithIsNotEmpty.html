La comprobación de tamaño de Collections/Array/String debe reemplazarse por isNotEmpty().

Usar isNotEmpty() hace que el código sea más simple.

Esta corrección rápida reemplazará la comprobación de tamaño por isNotEmpty().

Ejemplo:


  fun foo() {
      val arrayOf = arrayOf(1, 2, 3)
      arrayOf.size > 0
  }

Después de aplicar la corrección rápida:


  fun foo() {
      val arrayOf = arrayOf(1, 2, 3)
      arrayOf.isNotEmpty()
  }
