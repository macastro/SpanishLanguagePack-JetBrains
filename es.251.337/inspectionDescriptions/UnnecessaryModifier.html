Informa sobre modificadores redundantes y sugiere eliminarlos. El código resultante será más corto, pero su comportamiento y significado permanecerán sin cambios.

Ejemplo 1:


  // En Java 17 y versiones posteriores, todo el código es implícitamente strictfp
  strictfp class X {
    // Los enums internos son implícitamente static
    static enum Inner {
        A, B, C
    }
    // Los records internos son implícitamente static
    static record R() {
    }
  }

Ejemplo 2:


  final record R() {
    // Todos los records son implícitamente final
  }

Ejemplo 3:


  // Todas las interfaces son implícitamente abstract
  abstract interface Printer {
    // Todos los miembros de la interfaz son implícitamente public
    public int size();
    // Todas las clases internas de la interfaz son implícitamente static
    static class Inner {}
  }
