Informa sobre clases que sobrescriben el método clone() pero no implementan la interfaz Cloneable. Esto generalmente indica un error de programación.

Use la opción Advertir solo para métodos clone 'public' para ignorar los métodos que no son public.

Para clases diseñadas para ser heredadas, puede optar por no implementar la interfaz Cloneable, sino sobrescribir clone() y declararlo como protected, dejando la decisión de implementar la interfaz Cloneable a las subclases.
