Reportar excepciones lanzadas desde JSP Scriptlets de nivel superior.

Ejemplo:


  <%!
  String getFormattedDate() {
    ...
    throw new IllegalArgumentException(...); // Error
  }
  %>
  <html>
  <head>
    <meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
    <title>Hello, JSP!</title>
  </head>
  <body>
  <i>Today is <%= getFormattedDate() %></i>
  </body>
  </html>

Nota: Tales páginas JSP se compilarán, ya que el contenedor servlet envuelve todo el código JSP en bloques try {} catch() {}, pero generarán excepciones en tiempo de ejecución cuando se desplieguen en el servidor.
