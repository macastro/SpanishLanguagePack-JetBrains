Reporta puntos de inyección incorrectos: campos, métodos y parámetros que usan la anotación @Inject y violan la especificación CDI.

Requisitos para campos:

  • No pueden ser static ni final
  • No pueden usar la anotación @Produces

Requisitos para métodos con inyección:

  • No pueden ser static ni abstract
  • No pueden tener parámetros con las anotaciones @Disposes y @Observes
