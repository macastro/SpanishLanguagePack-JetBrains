Informa sobre llamadas a Object.notify() u Object.notifyAll() donde no se detectan cambios de estado.

Object.notify() y Object.notifyAll() se utilizan normalmente para notificar a otros hilos sobre un cambio de estado. Este cambio de estado debería ocurrir en el contexto sincronizado que contiene la llamada a Object.notify() o Object.notifyAll(), y antes de la llamada. Aunque la ausencia de dicho cambio de estado no es necesariamente incorrecta, definitivamente merece ser revisada.

Ejemplo:


  synchronized (this) {
    notify();
  }
  // Sin cambios de estado
  synchronized (this) {
    notify(); // Notificación potencialmente redundante
  }
