Reporta las llamadas a hashCode() y equals() en objetos java.net.URL, así como las llamadas para agregar objetos URL a mapas y colecciones.

Los métodos equals() y hashCode() de URL pueden realizar búsquedas DNS para resolver nombres de host. Esto puede causar retrasos significativos, dependiendo de la disponibilidad y velocidad de la red y los servidores DNS. Usar java.net.URI en lugar de java.net.URL evitará las búsquedas DNS.

Ejemplo:


  boolean urlEquals(URL url1, URL url2) {
      return url1.equals(url2);
  }
