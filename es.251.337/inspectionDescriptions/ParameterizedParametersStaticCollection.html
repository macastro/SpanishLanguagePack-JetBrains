Clase de pruebas parametrizadas de JUnit 4 que tiene la anotación @RunWith(Parameterized.class) pero que no incluye un método proveedor de datos con la anotación @Parameterized.Parameters, o tiene una firma incorrecta. Esta clase de prueba no se puede ejecutar. El método proveedor de datos debe ser public y static y debe tener un tipo de retorno Iterable u Object[].

Se sugiere crear un método proveedor de parámetros vacío o modificar la firma del método proveedor de datos incorrecto.

Ejemplo:


  @RunWith(Parameterized.class)
  public class ImportantTest {
    private int input;
    private int expected;
    ImportantTest(int input, int expected) {
      this.input = input;
      this.expected = expected;
    }
    // ...casos de prueba
  }

Después de aplicar la corrección rápida:


  @RunWith(Parameterized.class)
  public class ImportantTest {
    private int input;
    private int expected;
    ImportantTest(int input, int expected) {
      this.input = input;
      this.expected = expected;
    }
    @Parameters
    public static Iterable<Object[]> parameters() {
      return null;
    }
    // ... test cases
  }
