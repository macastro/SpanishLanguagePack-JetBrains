Reporta el uso de funciones o métodos de Python que están marcados como obsoletos y generan advertencias DeprecationWarning o PendingDeprecationWarning.

Además, esta inspección resalta el uso de los decoradores abc.abstractstaticmethod, abc.abstractproperty y abc.abstractclassmethod.

Ejemplo:


class Foo:
    @property
    def bar(self):
        import warnings
        warnings.warn("this is deprecated", DeprecationWarning, 2)
        return 5
foo = Foo()
print(foo.bar)
