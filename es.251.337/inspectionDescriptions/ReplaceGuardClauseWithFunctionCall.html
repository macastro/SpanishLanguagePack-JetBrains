Informa de cláusulas de protección que pueden sustituirse por una llamada a una función.

Ejemplo:

  fun test(foo: Int?) {
      if (foo == null) throw IllegalArgumentException("foo") // cláusula reemplazable
  }

Después de aplicar la corrección rápida:

  fun test(foo: Int?) {
      checkNotNull(foo)
  }
