Informa sobre campos no inicializados después de la deserialización de objetos en el método readObject().

Esta inspección no informa sobre campos transitorios.

Nota: Esta inspección utiliza un algoritmo de flujo de control muy conservador y podría informar erróneamente campos como no inicializados.

Ejemplo:


class DataObject implements Serializable {
  String s; // s no está inicializado en readObject
  int i;
  private void readObject(ObjectInputStream stream) throws IOException {
    i = stream.readInt();
  }
}
