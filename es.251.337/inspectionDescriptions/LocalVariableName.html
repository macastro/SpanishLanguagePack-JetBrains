Informa sobre variables locales que no siguen las convenciones de nomenclatura.

Puede especificar los patrones deseados en las opciones de inspección.

Convenciones de nomenclatura recomendadas: deben comenzar con letra minúscula, usar camelCase y no contener guiones bajos.

Ejemplo:


  fun fibonacciNumber(index: Int): Long = when(index) {
      0 -> 0
      else -> {
          // No sigue la convención de nomenclatura: contiene guión bajo (`_`)
          var number_one: Long = 0
          // No sigue la convención de nomenclatura: comienza con mayúscula
          var NUMBER_TWO: Long = 1
          // Sigue la convención de nomenclatura: comienza con minúscula, usa camelCase y no tiene guiones bajos.
          var numberThree: Long = number_one + NUMBER_TWO
          for(currentIndex in 2..index) {
              numberThree = number_one + NUMBER_TWO
              number_one = NUMBER_TWO
              NUMBER_TWO = numberThree
          }
          numberThree
      }
  }
