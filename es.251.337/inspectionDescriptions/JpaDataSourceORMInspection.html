Se informan las referencias en las siguientes anotaciones de Persistence ORM si no se pueden resolver en las fuentes de datos configuradas:
  • Table
  • Column
  • [Any]To[Any]
  • JoinColumn / JoinTable
  • NamedQuery / NativeQuery
  • Embeddable

Se recomienda actualizar la fuente de datos o asignar una fuente de datos diferente.

Ejemplo de tabla de base de datos:


  CREATE TABLE DATA_TABLE (
    ID INTEGER,
    DATA VARCHAR(100)
);

Ejemplo de clase de entidad JPA:


  @Entity
  @Table(name = "UNKNOWN_TABLE") // Referencia no resuelta
  public class JavaEntity {
    @Column(name = "ID") // No se puede resolver la columna en una tabla no definida
    Integer id;
  }

Nota: La fuente de datos se puede configurar mediante la acción `Asignar fuente de datos` o mediante el icono del área de conexión correspondiente a la clase de entidad
