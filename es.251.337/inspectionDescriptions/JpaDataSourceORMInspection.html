Reporta referencias dentro de las siguientes anotaciones de ORM de persistencia si no pueden resolverse en el origen de datos configurado:
  • Tabla
  • Columna
  • [Any]To[Any]
  • JoinColumn / JoinTable
  • NamedQuery / NativeQuery
  • Empotrable

Sugiere actualizar el origen de datos o asignar uno diferente.

Ejemplo de tabla de base de datos:


  CREATE TABLE DATA_TABLE (
    ID INTEGER,
    DATA VARCHAR(100)
);

Ejemplo de clase de entidad JPA:


  @Entity
  @Table(name = "UNKNOWN_TABLE") // referencia no resuelta
  public class JavaEntity {
    @Column(name = "ID") // no se pueden resolver las columnas en una tabla que no está definida
    Integer id;
  }

NOTA: El origen de datos se puede configurar mediante la acción `Asignar origen de datos` o el icono de canal correspondiente para la clase de entidad
