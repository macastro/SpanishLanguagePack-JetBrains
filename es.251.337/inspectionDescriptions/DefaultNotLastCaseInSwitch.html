Reporta declaraciones o expresiones switch donde la rama default está ubicada antes de otro case. Esta estructura puede causar confusión innecesaria. Si es posible, se puede mover la rama default a la última posición mediante una corrección rápida.

Ejemplo:


  switch (n) {
      default:
          System.out.println();
          break;
      case 1:
          break;
  }

Después de aplicar la corrección rápida:


  switch (n) {
    case 1:
        break;
    default:
        System.out.println();
        break;
  }
