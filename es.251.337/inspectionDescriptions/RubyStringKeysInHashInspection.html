Reporta hashes que usan cadenas de texto como claves. Es mejor usar símbolos en lugar de cadenas como claves de hash.

Ejemplo:

# 不良做法
{ 'foo' => 1, 'bar' => 2 }
Todas las claves de cadena pueden convertirse a símbolos mediante una corrección rápida. Después de aplicar la corrección rápida, el resultado se verá así: 
# 优良做法
{ foo: 1, bar: 2 }
