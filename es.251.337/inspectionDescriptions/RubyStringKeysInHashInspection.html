Reporta hashes con strings como claves. Es preferible utilizar símbolos en vez de strings como claves de hash.

Ejemplo:

# Mala Práctica
{ 'foo' => 1, 'bar' => 2 }
Se encuentra disponible una corrección rápida para convertir todas las claves de string a símbolos. Después de aplicar la corrección rápida, el resultado se verá así: 
# Buena Práctica
{ foo: 1, bar: 2 }
