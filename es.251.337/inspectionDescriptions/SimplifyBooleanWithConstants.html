Informa las partes de la expresión booleana que se pueden reducir a constantes.

La corrección rápida simplifica la condición.

Ejemplo:


  fun use(arg: Boolean) {
      if (false == arg) {

      }
  }

Después de aplicar la corrección rápida:


  fun use(arg: Boolean) {
      if (!arg) {

      }
  }
