El bucle puede ser reemplazado por un único método String.repeat() (disponible desde Java 11).

Ejemplo:


  void append(StringBuilder sb, int count, Object obj) {
    for (int i = 0; i < count; i++) {
      sb.append(obj);
    }
  }

Después de aplicar la corrección rápida:


  void append(StringBuilder sb, int count, Object obj) {
    sb.append(String.valueOf(obj).repeat(Math.max(0, count)));
  }

Por defecto, si no se puede demostrar estáticamente que count no es negativo, la comprobación puede envolver count con Math.max(0, count). Esto se hace para prevenir posibles cambios semánticos, ya que String.repeat() rechaza números negativos. Este comportamiento puede desactivarse si es necesario usando la opción Agregar Math.max(0, count) para evitar posibles cambios semánticos.

De manera similar, si no se sabe si la cadena a repetir puede ser null, se puede envolver en String.valueOf para prevenir posibles NullPointerException.

Esta inspección solo se informa cuando el nivel de lenguaje del proyecto o módulo es 11 o superior.
