El bucle puede reemplazarse por una llamada a Arrays.setAll() o Arrays.fill().

Si el nivel de lenguaje del proyecto o módulo es 8 o superior, esta inspección sugiere reemplazar el bucle con Arrays.setAll(). Puede reemplazar el bucle con Arrays.fill() en cualquier nivel de lenguaje.

Ejemplo:


  for (int i=0; i<array.length; i++) {
     array[i] = calc(i);
  }

Después de aplicar la corrección rápida:


  Arrays.setAll(array, this::calc);
