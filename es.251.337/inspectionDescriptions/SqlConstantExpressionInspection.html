Reporta las condiciones y expresiones que siempre son verdaderas, falsas o nulas.

Ejemplo (MySQL):

CREATE TABLE t1 (a TEXT, b INT, c BOOLEAN);
SELECT a FROM t1 WHERE 'Cat' = 'Cat';
SELECT a FROM t1 WHERE 'Cat' = null;

El 'Cat' = 'Cat' siempre es verdadero y será reportado.

El 'Cat' = null siempre es nulo y será reportado.
