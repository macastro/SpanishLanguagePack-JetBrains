Una plantilla integrada para una clase Ruby. Esta plantilla se usa cada vez que crea una nueva clase Ruby. Seleccione Nuevo | Archivo/Clase Ruby en el menú emergente de una de las vistas del proyecto. Además de las sentencias, expresiones y comentarios de Ruby, también puede usar las variables predefinidas que se enumeran a continuación. Cada variable se expande como una macro al valor correspondiente. Mediante la directiva #parse, se pueden incluir plantillas de la pestaña Inclusiones especificando el nombre completo de la plantilla requerida como un parámetro entre comillas.
Las variables predefinidas reciben los siguientes valores:
${FILE_NAME}   Nombre del archivo actual
${USER}   Nombre de inicio de sesión del sistema del usuario actual
${DATE}   Fecha actual del sistema
${TIME}   Hora actual del sistema
${YEAR}   Año actual
${MONTH}   Mes actual
${DAY}   Día del mes actual
${HOUR}   Hora actual
${MINUTE}   Minuto actual
${PRODUCT_NAME}   Nombre del IDE actual
${PROJECT_NAME}   Nombre del proyecto actual
