Plantilla incorporada para enumeraciones PHP. Esta plantilla se utiliza cada vez que crea una nueva enumeración PHP. Seleccione Nuevo | Clase PHP desde el menú emergente en una de las vistas del proyecto.
La plantilla es editable. Además de las declaraciones, expresiones y comentarios de PHP, puede usar las variables predefinidas que se enumeran a continuación. Cada variable se expande como una macro a su valor correspondiente.
Las plantillas de la pestaña Includes se pueden incluir mediante la directiva #parse especificando el nombre completo de la plantilla deseada como un parámetro entre comillas.
Las variables predefinidas aceptan los siguientes valores:
${FILE_NAME}   Nombre del archivo actual
${USER}   Nombre de inicio de sesión del usuario actual
${DATE}   Fecha actual del sistema
${TIME}   Hora actual del sistema
${YEAR}   Año actual
${MONTH}   Mes actual
${DAY}   Día del mes actual
${HOUR}   Hora actual
${MINUTE}   Minuto actual
${PRODUCT_NAME}   Nombre del IDE actual
${DS}   Símbolo del dólar, evaluado como el carácter "$" simple.
${PROJECT_NAME}   Nombre del proyecto actual
