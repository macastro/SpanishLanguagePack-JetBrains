Una plantilla incorporada de un módulo RBS. Esta plantilla se utiliza cada vez que crea un nuevo módulo RBS. Seleccione Nuevo | Archivo/Clase RBS desde el menú emergente en una de las vistas del proyecto. Además de las declaraciones, expresiones y comentarios de RBS, puede utilizar las variables predefinidas que se enumeran a continuación. Cada variable se expande como un macro a su valor correspondiente. Con la directiva #parse, puede incluir plantillas de la pestaña Include especificando el nombre completo de la plantilla deseada como un parámetro entre comillas.
Las variables predefinidas aceptan los siguientes valores:
${FILE_NAME}   nombre del archivo actual
${USER}   nombre de inicio de sesión del usuario actual
${DATE}   fecha actual del sistema
${TIME}   hora actual del sistema
${YEAR}   año actual
${MONTH}   mes actual
${DAY}   día del mes actual
${HOUR}   hora actual
${MINUTE}   minuto actual
${PRODUCT_NAME}   nombre del IDE actual
${PROJECT_NAME}   nombre del proyecto actual
