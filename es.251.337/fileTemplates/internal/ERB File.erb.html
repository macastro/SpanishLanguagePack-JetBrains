Una plantilla incorporada para archivos ERB. Esta plantilla se utiliza cada vez que crea un nuevo archivo ERB. Seleccione Nuevo | Archivo ERB desde el menú emergente en una de las vistas del proyecto. Además de las declaraciones, expresiones y comentarios de Ruby y HTML, puede usar las variables predefinidas que se enumeran a continuación. Cada variable se expande como un macro a su valor correspondiente. A través de la directiva #parse, puede incluir plantillas de la pestaña Includes especificando el nombre completo de la plantilla deseada como un parámetro entre comillas.
Las variables predefinidas aceptan los siguientes valores:
${FILE_NAME}   Nombre del archivo actual
${USER}   Nombre de inicio de sesión del usuario actual
${DATE}   Fecha actual del sistema
${TIME}   Hora actual del sistema
${YEAR}   Año actual
${MONTH}   Mes actual
${DAY}   Día del mes actual
${HOUR}   Hora actual
${MINUTE}   Minuto actual
${PRODUCT_NAME}   Nombre del IDE actual
${PROJECT_NAME}   Nombre del proyecto actual
