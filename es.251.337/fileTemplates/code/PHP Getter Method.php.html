Plantilla integrada para un método getter de PHP. Se usa, por ejemplo, cuando se invoca la corrección rápida "Añadir getter y setter".
Las variables predefinidas toman los siguientes valores:
${STATIC}   Toma el valor 'static' si la propiedad es estática; de lo contrario, una cadena vacía.
${GET_OR_IS}   Prefijo 'get/is'. Si una propiedad tiene un tipo booleano (pista de tipo), se evalúa como 'is'; de lo contrario, se evalúa como la cadena 'get'.
${NAME}   Nombre del método en estilo CamelCase sin el prefijo "get", con el primer carácter en mayúscula. Por ejemplo, el nombre de propiedad '_my_property' se convierte en 'MyProperty'.
${CLASS_NAME}   Contiene el nombre de la clase.
${RETURN_TYPE}   Contiene la pista de tipo de retorno si el nivel de lenguaje es PHP 7.0 o superior.
${FIELD_NAME}   Nombre de la propiedad original, sin modificar.
${TYPE_HINT}   Pista de tipo de la propiedad o una cadena vacía si no se especifica.
${DS}   Signo de dólar, se evalúa como el carácter '$' normal.
