Plantilla integrada para un método setter de PHP. Se usa, por ejemplo, cuando se invoca la corrección rápida "Añadir getter y setter".
Variables predefinidas que reciben los siguientes valores:
${STATIC}   Si la propiedad es estática, toma el valor 'static', de lo contrario, una cadena vacía.
${NAME}   Nombre del método en estilo CamelCase sin el prefijo "set", con el primer carácter en mayúscula. Por ejemplo, el nombre de la propiedad '_my_property' se convierte en 'MyProperty'.
${CLASS_NAME}   Contiene el nombre de la clase.
${SCALAR_TYPE_HINT}   Contiene la pista de tipo escalar si el nivel de lenguaje es PHP 7.0 o superior.
${FIELD_NAME}   Nombre original de la propiedad, sin modificaciones.
${PARAM_NAME}   Nombre del parámetro del setter. Casi idéntico a ${FIELD_NAME}, pero se elimina el primer guion bajo, si existe. Por ejemplo, el nombre de la propiedad '_myProperty' se convierte en 'myProperty'.
${TYPE_HINT}   Pista de tipo de la propiedad. Si no se especifica, es una cadena vacía.
${VOID_RETURN_TYPE}   Indica si el tipo de retorno void está disponible para el nivel de lenguaje PHP del proyecto.
${DS}   Signo de dólar, se evalúa como el carácter '$'.
