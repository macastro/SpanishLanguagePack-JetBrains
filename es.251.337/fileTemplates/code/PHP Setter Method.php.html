Una plantilla incorporada para un método setter de PHP. Se utiliza, por ejemplo, cuando se invoca la corrección rápida "Agregar getter y setter".
Las variables predefinidas aceptan los siguientes valores:
${STATIC}   Toma el valor 'static' si la propiedad es estática, o una cadena vacía en caso contrario.
${NAME}   El nombre del método en estilo camelCase, sin el prefijo "set" y con la primera letra en mayúscula. Por ejemplo, el nombre de propiedad '_my_property' se convierte en 'MyProperty'.
${CLASS_NAME}   Contiene el nombre de la clase.
${SCALAR_TYPE_HINT}   Incluye el type hint escalar si el nivel de lenguaje es PHP 7.0 o superior
${FIELD_NAME}   El nombre original de la propiedad sin modificaciones.
${PARAM_NAME}   El nombre del parámetro para el setter. Es casi idéntico a ${FIELD_NAME}, pero se elimina el primer guion bajo si existe. Por ejemplo, el nombre de propiedad '_myProperty' se convierte en 'myProperty'.
${TYPE_HINT}   El type hint de la propiedad, o una cadena vacía si no está especificado.
${VOID_RETURN_TYPE}   Indica si el tipo de retorno void está disponible para el nivel de lenguaje PHP en el proyecto.
${DS}   El símbolo del dólar, evaluado como el carácter "$" simple.
