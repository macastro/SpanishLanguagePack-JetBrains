Rellena el cuerpo del método cada vez que se genera un método sobrescrito por el programa (por ejemplo, al invocar la función Código | Sobrescribir métodos...).
Esta plantilla incorporada es editable. Además de las expresiones Java y comentarios, puede usar variables predefinidas que luego se expandirán a sus valores correspondientes.
Las variables predefinidas aceptan los siguientes valores:
${CALL_SUPER}   Llamada al método super, para métodos void - super.method_name();, con o sin parámetros formales; para otros métodos - return super.method_name();, con o sin parámetros formales
${PLAIN_CALL_SUPER}   Llamada al método super, super.method_name();, con o sin parámetros formales
${RETURN_TYPE}   Tipo de retorno del método creado
${DEFAULT_RETURN_VALUE}   Valor de retorno predeterminado del método
${METHOD_NAME}   Nombre del método sobrescrito
${CLASS_NAME}   Nombre cualificado de la clase donde se sobrescribe el método
${SIMPLE_CLASS_NAME}   Nombre no cualificado de la clase que implementa el método
También están disponibles todas las variables predefinidas de la plantilla Encabezado de archivo (pestaña Include). (Por ejemplo: ${USER}, ${DATE}, etc.)
