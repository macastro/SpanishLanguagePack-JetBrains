Rellena el cuerpo del método cada vez que el programa genera un método anulado (por ejemplo, al invocar la función Código | Anular Métodos…).
Esta plantilla integrada es editable. Además de expresiones y comentarios Java, puede usar variables predefinidas que se expandirán a sus valores correspondientes.
Las variables predefinidas reciben los siguientes valores:
${CALL_SUPER}   Llamada al método super, para métodos void - super.nombre_método();, con o sin parámetros; para otros métodos - return super.nombre_método();, con o sin parámetros
${PLAIN_CALL_SUPER}   Llamada al método super, super.nombre_método();, con o sin parámetros
${RETURN_TYPE}   Tipo de retorno del método creado
${DEFAULT_RETURN_VALUE}   Valor de retorno predeterminado del método
${METHOD_NAME}   Nombre del método anulado
${CLASS_NAME}   Nombre completo de la clase donde se anula el método
${SIMPLE_CLASS_NAME}   Nombre simple de la clase que implementa el método
Todas las variables predefinidas de la plantilla Encabezado de archivo (pestaña Incluir) también están disponibles. (Por ejemplo: ${USER}, ${DATE} etc.)
