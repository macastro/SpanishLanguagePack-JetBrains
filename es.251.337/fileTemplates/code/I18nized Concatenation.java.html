Reemplazar la concatenación de cadenas (como String s = "Hello " + name;) por una expresión Java internacionalizada (como String s = MessageFormat.format(​myResourceBundle.getString(​"hello.0"), name);).
Esta plantilla incorporada es editable. Además de las expresiones Java y comentarios, puede utilizar variables predefinidas que luego se expandirán a sus valores correspondientes.
Las variables predefinidas aceptan los siguientes valores:
${RESOURCE_BUNDLE}   Una expresión de tipo java.util.ResourceBundle cuya instancia está disponible en este contexto.
${PARAMETERS}   Variables utilizadas en la concatenación de cadenas, generalmente pasadas como argumentos a la expresión internacionalizada.
${PROPERTY_KEY}   El nombre de la clave de propiedad definida en el archivo de propiedades correspondiente. Normalmente, es el valor utilizado como parámetro para el método ResourceBundle.getString().
${PROPERTY_VALUE}   El valor de la propiedad definida en el archivo de propiedades correspondiente. Este es el valor literal de la cadena Java original.
