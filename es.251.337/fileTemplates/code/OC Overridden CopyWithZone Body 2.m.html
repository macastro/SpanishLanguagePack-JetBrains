El cuerpo del método -copyWithZone: que se implementa cuando se genera (por ejemplo, al invocar la acción Código | Generar | -copyWithZone:).
La plantilla es editable. Además de las expresiones y comentarios de Objective-C, puede usar variables predefinidas que luego se expandirán a sus valores correspondientes.
Las variables predefinidas tomarán los siguientes valores:
${RETURN_TYPE}   El tipo de retorno del método generado
${DEFAULT_RETURN_VALUE}   El valor predeterminado que retorna el método
${CONTAINING_CLASS}   El nombre de la clase donde se implementa el método
${CUSTOM_CODE}   El cuerpo del método generado
Todas las variables predefinidas del encabezado de archivo C (C file header) en la pestaña Include también están disponibles (${USER}, ${DATE}, etc.)
