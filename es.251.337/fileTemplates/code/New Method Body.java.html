Rellena el cuerpo del método cuando se genera, por ejemplo, al aplicar la acción de intención Crear método a partir del uso.
Esta plantilla incorporada es editable. Además de expresiones y comentarios Java, también puede usar variables predefinidas que luego se expandirán a los valores correspondientes.
Las variables predefinidas reciben los siguientes valores:
${RETURN_TYPE}   Tipo de retorno del método que se crea.
${DEFAULT_RETURN_VALUE}   Valor de retorno predeterminado del método.
${METHOD_NAME}   Nombre del método que se crea.
${CLASS_NAME}   Nombre calificado de la clase en la que se crea el método.
${SIMPLE_CLASS_NAME}   Nombre no calificado de la clase que implementa el método.
Todas las variables predefinidas de la plantilla Encabezado de archivo (pestaña Incluir) también están disponibles. (Por ejemplo: ${USER}, ${DATE}, etc.)
