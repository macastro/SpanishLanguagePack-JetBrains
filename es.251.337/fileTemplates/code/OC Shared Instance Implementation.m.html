Genera el método +sharedInstance en la implementación de la clase, por ejemplo, cuando se invoca la acción Código | Generar | +sharedInstance.
La plantilla es editable. Además de las expresiones y comentarios de Objective-C, puede utilizar variables predefinidas que luego se expandirán a sus valores correspondientes.
Las variables predefinidas tomarán los siguientes valores:
${$CLASS_TYPE}   El nombre de la clase en la que se implementa el método
Todas las variables predefinidas disponibles en la plantilla de Encabezado C (pestaña Include) también están disponibles (${USER}, ${DATE}, etc.)
