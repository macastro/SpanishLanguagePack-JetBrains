Genera el método +sharedInstance en la sección de implementación de la clase, por ejemplo, al llamar a la acción Código | Generar | +sharedInstance.
La plantilla es editable. Junto con las expresiones y comentarios de Objective-C, puede usar variables predefinidas que luego se expandirán a los valores correspondientes.
Las variables predefinidas tomarán los siguientes valores:
${$CLASS_TYPE}   el nombre de la clase donde se implementa el método
Todas las variables predefinidas de la plantilla Encabezado de Archivo C (la pestaña Incluye) también están disponibles (${USER}, ${DATE}, etc.)
