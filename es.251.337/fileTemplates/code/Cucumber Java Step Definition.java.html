Crear definiciones de pasos para archivos Java en Cucumber.
Las variables predefinidas aceptan los siguientes valores:
${STEP_KEYWORD}   La anotación para la definición del paso. Generalmente, la palabra clave utilizada para invocar el paso (Given, Then, And, etc.).
${STEP_REGEXP}   La expresión regular generada para aceptar el paso.
${METHOD_NAME}   El nombre del método generado.
${PARAMETERS}   Los parámetros formales del método generado.
${BODY}   El cuerpo del método generado.
