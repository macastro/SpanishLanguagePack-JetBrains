Plantilla integrada para un método de prueba PHPUnit. Se usa, por ejemplo, cuando se selecciona la casilla para generar métodos de prueba en el diálogo "Nueva prueba".
Variables predefinidas que reciben los siguientes valores:
${NAME}   Nombre del método.
${CAPITALIZED_NAME}   Nombre del método con la primera letra en mayúscula.
${TESTED_NAME}   Nombre de la clase probada.
