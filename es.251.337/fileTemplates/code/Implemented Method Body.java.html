Rellena el cuerpo del método al generar métodos implementados (por ejemplo, al aplicar las intenciones Implementar clase abstracta o Implementar método abstracto o al invocar la funcionalidad Código | Implementar métodos...).
Esta plantilla incorporada es editable. Además de expresiones Java y comentarios, puedes usar variables predefinidas que luego se expandirán a sus valores correspondientes.
Las variables predefinidas aceptan los siguientes valores:
${RETURN_TYPE}   El tipo de retorno del método creado
${DEFAULT_RETURN_VALUE}   El valor de retorno predeterminado del método
${METHOD_NAME}   El nombre del método implementado
${CLASS_NAME}   El nombre cualificado de la clase que implementa el método
${SIMPLE_CLASS_NAME}   El nombre no cualificado de la clase que implementa el método
Todas las variables predefinidas disponibles en la plantilla de Encabezado de archivo (pestaña Includes) también están disponibles (por ejemplo, ${USER}, ${DATE}, etc.).
