Reemplaza las expresiones literales de cadenas (como String s = "Hello";) con expresiones de código Java internacionalizadas (como String s = myResourceBundle.getString("hello");).
Esta plantilla integrada es editable. Además de las expresiones y comentarios de Java, también puede usar variables predefinidas que luego se expandirán a sus valores correspondientes.
Las variables predefinidas reciben los siguientes valores:
${RESOURCE_BUNDLE}   Una expresión de tipo java.util.ResourceBundle, cuya instancia está disponible en este contexto.
${PROPERTY_KEY}   El nombre de la clave de propiedad especificada en el archivo de propiedades correspondiente. Generalmente, es el valor usado para el parámetro del método ResourceBundle.getString().
${PROPERTY_VALUE}   El valor de la propiedad definido en el archivo de propiedades correspondiente. Este es el valor literal de la cadena Java original.
