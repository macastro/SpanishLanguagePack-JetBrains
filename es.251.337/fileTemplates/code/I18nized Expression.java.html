Reemplazar expresiones literales de cadena (como String s = "Hello";) por expresiones de código Java internacionalizadas (como String s = myResourceBundle.getString("hello");).
Esta plantilla incorporada es editable. Además de expresiones Java y comentarios, puede usar variables predefinidas que luego se expandirán a sus valores correspondientes.
Las variables predefinidas aceptan los siguientes valores:
${RESOURCE_BUNDLE}   Una expresión de tipo java.util.ResourceBundle cuya instancia está disponible en este contexto.
${PROPERTY_KEY}   El nombre de la clave de propiedad especificada en el archivo de propiedades correspondiente. Normalmente, es el valor utilizado como parámetro para el método ResourceBundle.getString().
${PROPERTY_VALUE}   El valor de la propiedad definido en el archivo de propiedades correspondiente. Este es el valor literal original de la cadena Java.
