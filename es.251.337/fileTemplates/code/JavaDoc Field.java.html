Define la plantilla para generar JavaDoc para campos, por ejemplo, cuando se aplica la intención Añadir Javadoc.

Además de las expresiones y comentarios de Java, puedes usar variables predefinidas que se expandirán como macros a sus valores correspondientes.

Las plantillas de la pestaña Includes pueden incluirse usando la directiva #parse. Para incluir una plantilla, especifica su nombre completo como un parámetro entre comillas (por ejemplo, #parse("File Header.java")).
Las variables predefinidas aceptan los siguientes valores:
${ELEMENT_NAME}   Nombre del miembro de la clase para el que se genera el JavaDoc
${CONTAINING_CLASS}   Nombre de la clase contenedora
${PACKAGE_NAME}   Nombre del paquete donde se crea el nuevo enum
${NAME}   Nombre del nuevo enum que especificaste en el diálogo Crear Nueva Clase
${USER}   Nombre de inicio de sesión del usuario actual en el sistema
${DATE}   Fecha actual del sistema
${TIME}   Hora actual del sistema
${YEAR}   Año actual
${MONTH}   Mes actual
${MONTH_NAME_SHORT}   Primeras 3 letras del nombre del mes actual (Jan, Feb, etc.)
${MONTH_NAME_FULL}   Nombre completo del mes actual (January, February, etc.)
${DAY}   Día del mes actual
${HOUR}   Hora actual
${MINUTE}   Minuto actual
${PROJECT_NAME}   Nombre del proyecto actual
