Se rellena cada vez que el programa genera un constructor secundario de Kotlin, por ejemplo, al usar la intención Crear constructor secundario desde uso.
La plantilla es editable. Además de las expresiones y comentarios de Kotlin, puede usar variables predefinidas que luego se expandirán a sus valores correspondientes.
Las variables predefinidas tomarán los siguientes valores:
${CLASS_NAME}   Nombre cualificado de la clase donde se crea la función
${SIMPLE_CLASS_NAME}   Nombre no cualificado de la clase donde se implementa la función
