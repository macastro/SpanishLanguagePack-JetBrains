Rellena el cuerpo del método al generar métodos implementados (por ejemplo, usando la acción Código | Generar | Implementar Métodos).
Las plantillas son editables. Además de expresiones y comentarios Swift, puedes usar variables predefinidas que luego se expandirán a sus valores correspondientes.
Las variables predefinidas aceptarán los siguientes valores:
${RETURN_TYPE}   El tipo de retorno del método creado
${DEFAULT_RETURN_VALUE}   El valor de retorno predeterminado sugerido (puede estar vacío)
${SIGNATURE}   La firma del método
Todas las variables predefinidas disponibles en la plantilla de Encabezado de archivo Swift (pestaña Include) también están disponibles (por ejemplo, ${USER}, ${DATE}, etc.).
