Define la plantilla para generar JavaDoc para métodos sobrescritos, por ejemplo, cuando se aplica la intención Añadir Javadoc.

Además de las expresiones Java y comentarios, puede usar variables predefinidas que se expandirán como macros a sus valores correspondientes.

Usando la directiva #parse, puede incluir plantillas de la pestaña Includes. Para incluir una plantilla, especifique su nombre completo como parámetro entre comillas (por ejemplo, #parse("File Header.java")).
Las variables predefinidas aceptan los siguientes valores:
${PARAMS}   Lista de nombres de parámetros
${TYPE_PARAMS}   Lista de nombres de parámetros de tipo
${THROWS}   Lista de excepciones lanzadas
${RETURN_TYPE}   Tipo devuelto por el método
${PARAMS_INHERITED}   Lista de parámetros con descripción del método super
${ELEMENT_NAME}   Nombre del miembro de clase propietario del JavaDoc generado
${CONTAINING_CLASS}   Nombre de la clase padre
${SUPER_METHOD}   Nombre completamente calificado del método super
${PACKAGE_NAME}   Nombre del paquete donde se crea el nuevo enum
${NAME}   Nombre del nuevo enum que especificó en el diálogo Crear Nueva Clase
${USER}   Nombre de inicio de sesión del sistema del usuario actual
${DATE}   Fecha actual del sistema
${TIME}   Hora actual del sistema
${YEAR}   Año actual
${MONTH}   Mes actual
${MONTH_NAME_SHORT}   Primeras 3 letras del nombre del mes actual (Jan, Feb, etc.)
${MONTH_NAME_FULL}   Nombre completo del mes actual (January, February, etc.)
${DAY}   Día del mes actual
${HOUR}   Hora actual
${MINUTE}   Minuto actual
${PROJECT_NAME}   Nombre del proyecto actual
