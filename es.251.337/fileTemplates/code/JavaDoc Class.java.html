Define plantillas para generar JavaDoc para clases, interfaces o registros, por ejemplo, al aplicar la intención Añadir Javadoc.

Además de las expresiones y comentarios de Java, puede utilizar variables predefinidas que se expandirán como macros a sus valores correspondientes.

Las plantillas de la pestaña Includes se pueden incluir mediante la directiva #parse. Para incluir una plantilla, especifique su nombre completo como parámetro entre comillas (por ejemplo, #parse("File Header.java")).
Las variables predefinidas aceptan los siguientes valores:
${RECORD_COMPONENTS}   Lista de parámetros del registro
${TYPE_PARAMS}   Lista de nombres de parámetros de tipo
${CONTAINING_CLASS}   Nombre de la clase padre
${INTERFACE}   Devuelve true cuando el propietario del JavaDoc es una interfaz
${ENUM}   Devuelve true cuando el propietario del JavaDoc es una enumeración
${RECORD}   Devuelve true cuando el propietario del JavaDoc es un registro
${PACKAGE_NAME}   Nombre del paquete donde se crea la nueva enumeración
${NAME}   Nombre de la nueva enumeración que especificó en el diálogo Crear nueva clase
${USER}   Nombre de inicio de sesión del sistema del usuario actual
${DATE}   Fecha actual del sistema
${TIME}   Hora actual del sistema
${YEAR}   Año actual
${MONTH}   Mes actual
${MONTH_NAME_SHORT}   Primeras 3 letras del nombre del mes actual (Jan, Feb, etc.)
${MONTH_NAME_FULL}   Nombre completo del mes actual (January, February, etc.)
${DAY}   Día del mes actual
${HOUR}   Hora actual
${MINUTE}   Minuto actual
${PROJECT_NAME}   Nombre del proyecto actual
