Define la plantilla utilizada para generar Javadoc para una clase, interfaz o registro, por ejemplo, cuando se aplica la acción de intención Añadir Javadoc.

Además de las expresiones y comentarios Java, puede utilizar variables predefinidas que se expandirán a sus valores correspondientes como macros.

Mediante la directiva #parse, puede incluir plantillas de la pestaña Incluye. Para incluir una plantilla, especifique el nombre completo de la plantilla entre comillas como parámetro (por ejemplo, #parse("Encabezado de archivo.java")).
Las variables predefinidas toman los siguientes valores:
${RECORD_COMPONENTS}   Lista de parámetros de registro
${TYPE_PARAMS}   Lista de nombres de parámetros de tipo
${CONTAINING_CLASS}   Nombre de la clase contenedora
${INTERFACE}   Devuelve true si el propietario de Javadoc es una interfaz
${ENUM}   Devuelve true si el propietario de Javadoc es una enumeración
${RECORD}   Devuelve true si el propietario de Javadoc es un registro
${PACKAGE_NAME}   Nombre del paquete en el que se crea la nueva enumeración
${NAME}   Nombre de la nueva enumeración especificado en el cuadro de diálogo Crear nueva clase
${USER}   Nombre de inicio de sesión del sistema del usuario actual
${DATE}   Fecha actual del sistema
${TIME}   Hora actual del sistema
${YEAR}   Año actual
${MONTH}   Mes actual
${MONTH_NAME_SHORT}   Primeras 3 letras del nombre del mes actual (ene, feb, etc.)
${MONTH_NAME_FULL}   Nombre completo del mes actual (enero, febrero, etc.)
${DAY}   Día del mes actual
${HOUR}   Hora actual
${MINUTE}   Minuto actual
${PROJECT_NAME}   Nombre del proyecto actual
