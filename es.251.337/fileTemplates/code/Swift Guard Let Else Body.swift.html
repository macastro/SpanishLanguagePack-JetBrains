Después de desenvolver un valor opcional usando la declaración guard, complete el bloque else. Esta plantilla es editable. Además de expresiones Swift y comentarios, puede usar variables predefinidas que luego se expandirán a sus valores correspondientes.
Las variables predefinidas aceptarán los siguientes valores:
${RETURN_TYPE}   El tipo de retorno de la función circundante (si existe)
${DEFAULT_RETURN_VALUE}   Valor de retorno predeterminado sugerido (puede estar vacío)
Todas las variables predefinidas disponibles en la plantilla Encabezado de archivo Swift (pestaña Include) también están disponibles (por ejemplo, ${USER}, ${DATE}, etc.).
