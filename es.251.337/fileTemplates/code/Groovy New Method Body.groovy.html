Esta es la plantilla incorporada que se utiliza para rellenar el cuerpo del método cada vez que se genera un método Groovy (por ejemplo, usando la intención Crear método desde uso).
La plantilla es editable. Además de las expresiones y comentarios de Groovy, puede usar variables predefinidas que luego se expandirán a sus valores correspondientes.
Las variables predefinidas tomarán los siguientes valores:
${RETURN_TYPE}   El tipo de valor de retorno del método creado
${DEFAULT_RETURN_VALUE}   El valor predeterminado que retorna el método
${METHOD_NAME}   El nombre del método creado
${CLASS_NAME}   El nombre cualificado de la clase donde se crea el método
${SIMPLE_CLASS_NAME}   El nombre no cualificado de la clase que implementa el método
