Rellena el cuerpo del método al generar métodos sobrescritos (por ejemplo, usando la acción Código | Generar | Sobrescribir métodos).
Las plantillas son editables. Además de expresiones y comentarios Swift, puedes usar variables predefinidas que luego se expandirán a sus valores correspondientes.
Las variables predefinidas tomarán los siguientes valores:
${CALL_SUPER}   Llamada al método super, para métodos Void: super.methodName() con o sin parámetros; para otros métodos: return super.methodName() con o sin parámetros
${RETURN_TYPE}   Tipo de retorno del método creado
${DEFAULT_RETURN_VALUE}   Valor de retorno predeterminado del método
${SIGNATURE}   Firma del método
Todas las variables predefinidas disponibles en la plantilla de Encabezado de archivo Swift (pestaña Include) también están disponibles (por ejemplo, ${USER}, ${DATE}, etc.).
