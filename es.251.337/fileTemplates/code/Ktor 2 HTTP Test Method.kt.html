Esta es una plantilla para crear métodos de prueba HTTP para Ktor 2.x.x en clases de prueba JUnit 4.
Las variables predefinidas aceptarán los siguientes valores:
${NAME}   Nombre del método creado
${HTTP_METHOD}   Método HTTP utilizado (GET, POST, etc.)
${ROUTE}   Ruta relativa al servicio actual que se va a probar
${CONFIGURE_SERVER}   Método de extensión de ruta para configurar el árbol de rutas para la parte del servidor que se va a probar
${BODY}   Cuerpo del método generado
