Esta plantilla incorporada se utiliza para completar el cuerpo del método cada vez que se genera un método sobrescrito (por ejemplo, al invocar la funcionalidad Código | Sobrescribir métodos...).
La plantilla es editable. Además de las expresiones Javascript y comentarios, puede utilizar variables predefinidas. Cuando se aplica la plantilla, estas variables se reemplazan con sus valores correspondientes.
Las variables predefinidas aceptarán los siguientes valores:
${CALL_SUPER}   Llamada al método super, para métodos void - super.method_name(); con o sin parámetros formales; para otros métodos - return super.method_name(); con o sin parámetros formales
${PLAIN_CALL_SUPER}   Llamada al método super, super.method_name(); con o sin parámetros formales;
${RETURN_TYPE}   Tipo de retorno del método creado
${DEFAULT_RETURN_VALUE}   Valor de retorno predeterminado del método
${METHOD_NAME}   Nombre del método sobrescrito
Todas las variables predefinidas de la plantilla de encabezado de archivo (pestaña Include) también están disponibles. (Por ejemplo: ${USER}, ${DATE}, etc.)
