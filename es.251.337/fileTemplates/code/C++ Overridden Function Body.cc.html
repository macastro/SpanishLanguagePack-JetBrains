Crea el cuerpo de la función sobrescrita cuando se genera (por ejemplo, usando la acción Código | Sobrescribir métodos).
Al editar la plantilla, puede usar expresiones C++, comentarios y las siguientes variables predefinidas:
${CALL_SUPER}   Llamada a la función base, para funciones void — BaseClass::functionName(), con o sin parámetros formales; para otras funciones - return BaseClass::functionName(), con o sin parámetros formales.
${RETURN_TYPE}   Tipo de retorno de la función.
${DEFAULT_RETURN_VALUE}   Valor de retorno predeterminado de la función.
Todas las variables predefinidas de la plantilla Encabezado de archivo C (pestaña Incluir), como ${USER} y ${DATE}, también están disponibles.
