Esta plantilla integrada se utiliza para crear componentes de clase React, por ejemplo, a través de la refactorización Extraer Componente. Puede personalizar la plantilla, por ejemplo, usar Component en lugar de PureComponent o cambiar las definiciones de los tipos de prop. También puede usar variables predefinidas. Estas variables serán reemplazadas con los valores correspondientes cuando se aplique la plantilla.
Las variables predefinidas tomarán los siguientes valores:
${NAME}   nombre del componente
${COMPONENT_BODY}   cuerpo del componente
${PROP_TYPES}   definición de tipos de prop (usando el módulo prop-types)
${PROP_TYPES}   valores de prop predeterminados que React usará cuando no se proporcione el valor de prop
