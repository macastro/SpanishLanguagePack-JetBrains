Incluye un fragmento de código que puede ser incluido en la plantilla de archivo (pestaña Templates) con ayuda de la directiva #parse.
Esta plantilla incorporada es editable. Además del texto estático, código y comentarios, puede usar variables predefinidas que luego se expandirán como macros a sus valores correspondientes.
Las variables predefinidas aceptan los siguientes valores:
${PACKAGE_NAME}   Nombre del paquete donde se crea el nuevo archivo
${USER}   Nombre de inicio de sesión del usuario actual del sistema
${DATE}   Fecha actual del sistema
${TIME}   Hora actual del sistema
${YEAR}   Año actual
${MONTH}   Mes actual
${MONTH_NAME_SHORT}   Primeras 3 letras del nombre del mes actual. Por ejemplo: Jan, Feb, etc.
${MONTH_NAME_FULL}   Nombre completo del mes actual. Por ejemplo: January, February, etc.
${DAY}   Día del mes actual
${DAY_NAME_SHORT}   Primeras 3 letras del nombre del día actual. Por ejemplo: Mon, Tue, etc.
${DAY_NAME_FULL}   Nombre completo del día actual. Por ejemplo: Monday, Tuesday, etc.
${HOUR}   Hora actual
${MINUTE}   Minuto actual
${PROJECT_NAME}   Nombre del proyecto actual
