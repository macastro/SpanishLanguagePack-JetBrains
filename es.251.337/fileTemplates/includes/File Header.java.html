Contiene un fragmento de código que se puede incluir en las plantillas de archivos (pestaña Plantillas) con la ayuda de la directiva #parse.
Esta plantilla incorporada es editable. Además del texto estático, código y comentarios, también puede usar variables predefinidas que luego se expandirán a sus valores correspondientes, como macros.
Las variables predefinidas toman los siguientes valores:
${PACKAGE_NAME}   Nombre del paquete en el que se crea el nuevo archivo.
${USER}   Nombre de inicio de sesión del sistema del usuario actual.
${DATE}   Fecha actual del sistema.
${TIME}   Hora actual del sistema.
${YEAR}   Año actual.
${MONTH}   Mes actual.
${MONTH_NAME_SHORT}   Primeras 3 letras del nombre del mes actual. Ej.: Ene, Feb, etc.
${MONTH_NAME_FULL}   Nombre completo del mes actual. Ej.: Enero, Febrero, etc.
${DAY}   Día del mes actual.
${DAY_NAME_SHORT}   Primeras 3 letras del nombre del día actual. Ej.: Lun, Mar, etc.
${DAY_NAME_FULL}   Nombre completo del día actual. Ej.: Lunes, Martes, etc.
${HOUR}   Hora actual.
${MINUTE}   Minuto actual.
${PROJECT_NAME}   Nombre del proyecto actual.
