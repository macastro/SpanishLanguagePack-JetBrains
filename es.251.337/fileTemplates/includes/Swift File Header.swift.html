Incluye un fragmento de código que puede incluirse en una plantilla de archivo (pestaña Plantillas) usando la directiva #parse.
La plantilla es editable. Además del texto estático, código y comentarios, puede usar variables predefinidas que luego se expandirán como macros a sus valores correspondientes.
Las variables predefinidas tomarán los siguientes valores:
${FILE_NAME}   Nombre del archivo actual
${USER}   Nombre de inicio de sesión del sistema del usuario actual
${USER_NAME}   Nombre completo del usuario actual
${ORGANIZATION_NAME}   Nombre de la organización del proyecto Xcode
${DATE}   Fecha actual del sistema
${TIME}   Hora actual del sistema
${YEAR}   Año actual
${MONTH}   Mes actual
${MONTH_NAME_SHORT}   Primeras 3 letras del nombre del mes actual. Por ejemplo: Jan, Feb, etc.
${MONTH_NAME_FULL}   Nombre completo del mes actual. Por ejemplo: January, February, etc.
${DAY}   Día actual
${DAY_NAME_SHORT}   Primeras 3 letras del nombre del día actual. Por ejemplo: Mon, Tue, etc.
${DAY_NAME_FULL}   Nombre completo del día actual. Por ejemplo: Monday, Tuesday, etc.
${HOUR}   Hora actual
${MINUTE}   Minuto actual
${PROJECT_NAME}   Nombre del proyecto actual
${COMPILER_VERSION_MAJOR}   Versión principal del compilador Swift
${COMPILER_VERSION_MINOR}   Versión secundaria del compilador Swift
${PREFER_EXPLICIT_RETURN}   Valor de la opción Preferir retorno explícito (Preferencias | Editor | Estilo de código | Swift | Generación de código)
