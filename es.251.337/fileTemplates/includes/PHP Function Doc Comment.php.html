Una plantilla incorporada para comentarios PHPDoc de una función PHP. Contiene un fragmento de código que puede incluirse en plantillas de archivo (pestaña Templates) con ayuda de la directiva #parse.
La plantilla es editable. Además del texto estático, código y comentarios, puede usar variables predefinidas que se expandirán como macros a sus valores correspondientes.
Las variables predefinidas aceptan los siguientes valores:
${NAME}   Nombre de la función.
${TYPE_HINT}   Sugerencia de tipo para el valor de retorno de la función. Se evalúa como 'void' si no se infiere ningún tipo de retorno.
${PARAM_DOC}  

Comentarios PHPDoc para los parámetros.

Se genera como una serie de líneas '* @param type name'. Se evalúa como contenido vacío si no hay parámetros.

${THROWS_DOC}  

Comentarios de documentación para excepciones.

Se genera como una serie de líneas '* @throws type'. Se evalúa como contenido vacío si no se lanzan excepciones.

${STATIC}   Evalúa a 'static' si la función/método es estático, de lo contrario es una cadena vacía. Por ejemplo: 
      #if (${STATIC} == "static") * @static
      #end
${DS}   Símbolo del dólar, se evalúa como el carácter "$" normal.
${DATE}   Fecha actual del sistema
${TIME}   Hora actual del sistema
${YEAR}   Año actual
${MONTH}   Mes actual
${MONTH_NAME_SHORT}   Primeras 3 letras del nombre del mes actual. Por ejemplo: Jan, Feb, etc.
${MONTH_NAME_FULL}   Nombre completo del mes actual. Por ejemplo: January, February, etc.
${DAY}   Día del mes actual
${HOUR}   Hora actual
${MINUTE}   Minuto actual
${SECOND}   Segundo actual
${PROJECT_NAME}   Nombre del proyecto actual
${USER}   Usuario actual
${CARET}   Marca la posición donde debe moverse el cursor después de agregar el comentario.
