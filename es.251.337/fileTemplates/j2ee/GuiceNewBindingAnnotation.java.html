Ésta es la plantilla integrada que se usa cada vez que se crea una nueva anotación de enlace Guice.
Usando la directiva #parse, se pueden incluir plantillas que están en la pestaña Incluidos al especificar el nombre completo de la plantilla deseada como un parámetro entre comillas. Ejemplo:
#parse("File Header.java")
Las variables predefinidas reciben los siguientes valores:
${PACKAGE_NAME} El nombre del paquete en el cual se está creando la nueva clase
${NAME} El nombre de la nueva clase, el cual es especificado por el usuario en el diálogo de Crear nueva clase
${USER} El nombre de inicio de sesión del sistema del usuario actual
${DATE} La fecha actual del sistema
${TIME} La hora actual del sistema
${YEAR} El año actual
${MONTH} El mes actual
${MONTH_NAME_SHORT} Las primeras 3 letras del nombre del mes actual. Por ejemplo: Jan, Feb, etc.
${MONTH_NAME_FULL} El nombre completo del mes actual. Por ejemplo: January, February, etc.
${DAY} El día del mes actual
${HOUR} La hora actual
${MINUTE} El minuto actual
${PROJECT_NAME} El nombre del proyecto actual
