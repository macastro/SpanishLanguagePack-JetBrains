Informa los componentes que no tienen ningún texto estático y que no tienen ninguna etiqueta marcada con setLabelFor para este componente.

Los componentes que no tienen texto estático incluyen campos de edición y cuadros combinados. Dichos componentes no se pueden activar con un atajo de teclado. El arreglo rápido para esta inspección te permite asociar automáticamente una etiqueta adyacente con el componente problemático.
