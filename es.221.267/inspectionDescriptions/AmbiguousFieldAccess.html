Informa sobre el acceso a un campo de superclase desde una clase interna anónima o local, si hay disponible una variable local, parámetro o campo con el mismo nombre en el código que rodea a la clase. En este caso, puede parecer que se accede a un elemento del código circundante, cuando en realidad se trata de un acceso a un campo de la superclase.

Para aclarar la intención del código, se recomienda agregar un calificador explícito super al acceso de campo.

Ejemplo:


    class Primero {
      protected String ambiguo;
    }
    class Segundo {
      void foo(String ambiguo) {
        new Primero() {
          {
            System.out.println(ambiguo); // se accede al campo, no al parámetro
          }
        };
      }
    }

Después de aplicar la corrección rápida:


    class Primero {
      protected String ambiguo;
    }
    class Segundo {
      void foo(String ambiguo) {
        new Primero() {
          {
            System.out.println(super.ambiguo);
          }
        };
      }
    }
