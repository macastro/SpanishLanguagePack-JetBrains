Reporta sentencias match que pueden convertirse a funciones anónimas con coincidencia de patrones.

Ejemplo:


  iterable.map(value => value match {
      case Some(value) => whenValue(value)
      case None => whenNothing()
  })

Después de aplicar la solución rápida:


  iterable.map {
      case Some(value) => whenValue(value)
      case None => whenNothing()
  }
