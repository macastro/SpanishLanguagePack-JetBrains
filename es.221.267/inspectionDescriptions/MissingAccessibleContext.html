Se reporta un componente Swing que no proporciona un contexto de accesibilidad.

Esta información es utilizada por los lectores de pantallas. Si no se proporciona, los usuarios con discapacidad visual no podrán acceder al componente.

Ejemplo:


  ListCellRenderer<String> renderer = (list, val, index, sel, cell) -> {
    JPanel panel = new JPanel();
    return panel;
  };

Para corregir este problema, se debe llamar a setAccessibleName() en el JPanel devuelto o sobrescribir el método getAccessibleContext().

El texto devuelto debe reflejar el propósito del componente. Por ejemplo, para un ListCellRenderer, este sería el texto del elemento del menú.
