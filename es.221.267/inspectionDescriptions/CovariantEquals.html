Reporta métodos equals() que tomen un tipo de argumento distinto de java.lang.Object si la clase contenedora no tiene otras sobrecargas de equals() que tomen java.lang.Object como su tipo de argumento.

Una versión covariante de equals() no sobrescribe el método Object.equals(Object). Puede causar un comportamiento inesperado en tiempo de ejecución. Por ejemplo, si la clase se usa para construir una de las clases de colección estándar, que espera que el método Object.equals(Object) sea sobrescrito.

Ejemplo:


  class Foo {
      public boolean equals(Foo foo) {  // advertencia
          return false;
      }
  }
  class Bar {
      public boolean equals(Bar bar) {  // no advertencia aquí
          return false;
      }
      @Override
      public boolean equals(Object obj) {
          return false;
      }
  }
