Reporta incompatibilidades con las versiones especificadas de Python. Active esta inspección si necesita que su código sea compatible con un rango de versiones de Python, por ejemplo, si está creando una librería.

Para definir el rango de versiones de Python inspeccionadas, seleccione las casillas de verificación correspondientes en la sección Opciones.

Para más información sobre las versiones de Python soportadas por el IDE, vea la ayuda web.
