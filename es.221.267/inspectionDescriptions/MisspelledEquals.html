Informa sobre las declaraciones de equal() con un único parámetro. Normalmente, se trata de un error tipográfico y, en realidad, el propósito es equals().

Se sugiere una solución rápida para renombrar el método a equals.

Ejemplo:


  class Main {
    public boolean equal(Object obj) {
      return true;
    }
  }

Una vez aplicada la solución rápida:


  class Main {
    public boolean equals(Object obj) {
      return true;
    }
  }
