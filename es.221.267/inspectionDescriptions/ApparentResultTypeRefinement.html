Informa de que falta = entre el tipo de resultado y el cuerpo de la función.

No se puede refinar un tipo cuando se utiliza como tipo de resultado de una función. Sin embargo, cuando falta una = entre el tipo de resultado y el cuerpo de la función, puede parecer que está refinado.

Ejemplo:


  def test(): T {}

Después de aplicar la corrección rápida


  def test(): T = {}
