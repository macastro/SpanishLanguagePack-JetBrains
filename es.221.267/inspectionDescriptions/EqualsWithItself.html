Informa de las llamadas a equals(), compareTo() o similares que comparan un objeto para igualdad con él mismo. Los contratos de método de estos métodos especifican que tales llamadas siempre devolverán true para equals() o 0 para compareTo(). La inspección también comprueba las llamadas a Objects.equals(), Objects.deepEquals(), Arrays.equals(), Comparator.compare(), métodos assertEquals() de marcos de prueba (JUnit, TestNG, AssertJ), Integer.compare(), Integer.compareUnsigned() y métodos similares.

Ejemplo:


  class Foo {
    boolean foo(Object o) {
        return o.equals(o); // advertencia
    }

    boolean bar(String[] ss) {
        return Arrays.equals(ss, ss); // advertencia
    }
}

Utilice la opción para informar de aserciones de prueba solo en clases de biblioteca no extensibles (como String) y tipos primitivos. Esta opción puede ser útil al probar métodos equals().
