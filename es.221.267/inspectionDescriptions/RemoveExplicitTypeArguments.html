Informa de llamadas a funciones con argumentos de tipo que pueden ser inferidos automáticamente. Dichos argumentos de tipo son redundantes y pueden omitirse de forma segura.

Utiliza la acción rápida "Eliminar argumentos de tipo explícitos" para limpiar el código.

Ejemplos:


  // El tipo 'String' puede inferirse aquí
  fun foo(): MutableList<String> = mutableListOf<String>()

  // Aquí no se puede inferir 'String', se necesita el argumento de tipo.
  fun bar() = mutableListOf<String>()

Tras aplicar la acción rápida:


  fun foo(): MutableList<String> = mutableListOf() <== Actualizado

  fun bar() = mutableListOf<String>()
