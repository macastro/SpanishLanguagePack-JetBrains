Reporte llamadas a funciones y métodos que no gestionan el resultado de la llamada de tipo error.

Esta API de funciones implica que su ejecución podría finalizar sin éxito y devolverían un error. Las llamadas que no gestionan el resultado del error podrían ser un indicio de un mal uso de la API.

Ejemplo:


os.Remove("non-existing") // se ignora el error

Después de aplicar la corrección rápida Gestionar errores:

err := os.Remove("non-existing") // se gestiona el error
if err != nil {
  return err
}
