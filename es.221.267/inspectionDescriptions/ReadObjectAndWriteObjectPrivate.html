Inspecciona clases Serializable donde los métodos readObject o writeObject no están declarados como privados. No hay razones para que estos métodos tengan una visibilidad superior a private.

Se sugiere una corrección rápida para hacer que el método correspondiente sea private.

Ejemplo:


  public class Test implements Serializable {
    public void readObject(ObjectInputStream stream) {
      /* ... */
    }
  }

Después de aplicar la corrección rápida:


  public class Test implements Serializable {
    private void readObject(ObjectInputStream stream) {
      /* ... */
    }
  }
