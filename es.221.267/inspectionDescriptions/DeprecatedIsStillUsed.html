Reporta clases, métodos y campos obsoletos que igualmente se utilizan en tu código.

Ejemplo:


  class MyCode {
    @Deprecated
    void oldMethod() {}// advertencia: "Todavía se utiliza un miembro obsoleto"

    void newMethod() {
      oldMethod(); // uso olvidado
    }
  }

Se ignoran los usos dentro de los elementos obsoletos.

NOTA: Debido a razones de rendimiento, se revisa a un miembro no privado solo cuando su nombre casi nunca aparece en el proyecto.
