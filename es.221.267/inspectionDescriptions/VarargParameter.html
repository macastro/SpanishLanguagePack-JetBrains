Informa sobre métodos que aceptan un número arbitrario de parámetros (también conocidos como métodos varargs).

Ejemplo:


  enum EnumConstants {
  A(null), B, C;

  EnumConstants(String... ss) {}
}

Hay una corrección rápida disponible para reemplazar un parámetro de argumento variable con un parámetro de matriz equivalente. Los argumentos relevantes en las llamadas al método están envueltos en una expresión de inicializador de matriz. Después de aplicar la corrección rápida:


  enum EnumConstants {
  A(null), B(new String[]{}), C(new String[]{});

  EnumConstants(String[] ss) {}
}

El método varargs apareció en Java 5. Esta inspección puede ayudar a degradar para la compatibilidad con versiones anteriores con versiones anteriores de Java.
