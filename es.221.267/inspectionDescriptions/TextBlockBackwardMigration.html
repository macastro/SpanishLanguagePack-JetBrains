Informa bloques de texto que pueden ser reemplazados con literales de cadena normales.

Ejemplo:


  Object obj = engine.eval("""
                           function hello() {
                             print('"Hello, world"');
                           }

                           hello();
                           """);

Después de aplicar corrección rápida:


  Object obj = engine.eval("function hello() {\n" +
                           "  print('\"Hello, world\"');\n" +
                           "}\n" +
                           "\n" +
                           "hello();\n");
