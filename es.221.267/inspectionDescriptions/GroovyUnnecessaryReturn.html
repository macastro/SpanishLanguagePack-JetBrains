Informa sobre las sentencias return al final de los constructores y métodos que devuelven void. Son innecesarias, por lo que pueden eliminarse de forma segura.

Ejemplo: 


  void foo (String s){
      print(s)
      return
  }

Después de aplicar la corrección rápida:


  void foo (String s){
      print(s)
  }

Para obtener más información, consulta la misma inspección en Java.
