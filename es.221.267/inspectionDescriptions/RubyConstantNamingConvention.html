Informa cualquier constante cuyos nombres sean demasiado cortos, demasiado largos o no sigan el patrón de expresión regular especificado.

Utiliza el campo Patrón para especificar una expresión regular para validar nombres de constantes.

Utiliza los campos Longitud mínima y Longitud máxima para especificar la longitud mínima y máxima de un nombre correspondientemente.
