Informa sobre anotaciones utilizadas en componentes de registros que no tienen efecto.

Esto puede suceder en dos casos:

  • La anotación informada tiene el objetivo METHOD, pero el accesor correspondiente está definido explícitamente.
  • La anotación informada tiene el objetivo PARAMETER, pero el constructor canónico está definido explícitamente.

Ejemplo:


@Target(ElementType.METHOD)
@interface A { }

// La anotación no aparecerá en el bytecode en absoluto,
// ya que debería propagarse al accesor pero el accesor está definido explícitamente
record R(@A int x) {
  public int x() { return x; }
}
