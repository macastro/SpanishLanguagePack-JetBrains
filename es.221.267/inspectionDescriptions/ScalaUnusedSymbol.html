Informa declaraciones que no se usan.

La corrección rápida elimina la declaración sin uso.

En las Opciones de abajo, puedes controlar si la inspección deberá funcionar para declaraciones públicas y/o locales. En el caso de declaraciones locales, además de habilitar y deshabilitar la inspección, también puedes hacerla depender de la existencia de una de tres opciones del compilador: -Wunused:locals, -Wunused:linted o -Xlint:unused.

Ejemplo:


  def test(): Int = {
    val unused = 3
    3
  }

Después de aplicar la corrección rápida:


  def test(): Int = {
    3
  }
