Reporta los bucles for que carecen de cláusulas de inicialización, condición o actualización. Algunos estilos de codificación prohíben tales bucles.

Ejemplo:


  for (int i = 0;;i++) {
    // cuerpo
  }

Utilice la opción Ignorar iteraciones de colección para ignorar los bucles que utilizan un iterador. Esta es una forma estándar de iterar sobre una colección en la que el bucle for no tiene una cláusula de actualización.
