Reporta sentencias if que se pueden simplificar a una sola asignación, return, o sentencia assert.

Ejemplo:


  if (foo()) {
    return true;
  } else {
    return false;
  }

Después de aplicar la solución rápida:


  return foo();

Configurar la inspección:

Usa la opción Ignorar sentencias 'if' encadenadas si deseas ocultar una advertencia para sentencias if encadenadas.

Por ejemplo, en el siguiente código la advertencia se ocultará, pero la solución rápida aún estará disponible:


  if (condition1) return true;
  if (condition2) return false;
  return true;

Nota que reemplazar if (isTrue()) assert false; con assert isTrue(); puede cambiar la semántica del programa cuando las aserciones están deshabilitadas si la condición tiene efectos secundarios. Usa la opción Ignorar sentencias 'if' con 'assert' triviales si deseas ocultar una advertencia para sentencias if que solo contienen una sentencia assert en sus cuerpos.
