Informar de constructores innecesarios.

Si el constructor es el único constructor de la clase, no tiene parámetros, tiene el mismo modificador de acceso que su clase contenedora y no realiza ninguna inicialización excepto invocar explícita o implícitamente al constructor de la superclase sin argumentos, entonces es innecesario. Tales constructores pueden eliminarse de forma segura, ya que serán generados por el compilador incluso si no se especifican.

Ejemplo:


  public class Foo {
    public Foo() {}
  }

Después de aplicar la corrección rápida:


  public class Foo {}

Utilice la configuración de la inspección para ignorar los constructores innecesarios anotados.
