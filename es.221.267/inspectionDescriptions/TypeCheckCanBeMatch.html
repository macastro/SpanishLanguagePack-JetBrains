Informa sobre comprobaciones de tipos y transformaciones de tipos en una serie de sentencias if-else y las sustituye por coincidencia de patrones.

Ejemplo:


  val value = 0
  if (value.isInstanceOf[Int]) {
    val x = value.asInstanceOf[Int] + 1
    val y = value.asInstanceOf[Int] + 2
  } else if (value.isInstanceOf[Long]) {
    val x = value.asInstanceOf[Long]
    println(x)
  }

Después de aplicar la corrección rápida:


  val value = 0
  value match {
    case i: Int =>
      val x = i + 1
      val y = i + 2
    case x: Long =>
      println(x)
    case _ =>
  }
