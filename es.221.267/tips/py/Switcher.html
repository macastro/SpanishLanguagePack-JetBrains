Switcher de navegación

Usa el Switcher &shortcut:Switcher; para navegar entre archivos y ventanas de herramientas abiertas. Mantén presionado Control para no cerrar el popup del Switcher. Usa Tab y Shift+Tab para navegar.

Presiona Delete o BackSpace para cerrar una pestaña del editor u ocultar una ventana de herramientas.
