Barra de navegación

La barra de navegación es una alternativa a la herramienta de ventana Proyecto. Permite comprobar la estructura del proyecto, abrir archivos específicos y desplazarse por elementos de código específicos en el archivo actual.

Pulse &shortcut:ShowNavBar; para mostrar la barra de navegación y usar las teclas de dirección para recorrer archivos y carpetas.
