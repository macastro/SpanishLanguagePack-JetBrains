Volcado en una fuente de datos DDL

Una fuente de datos DDL es una fuente de datos virtual que tiene un esquema basado en varios ficheros de SQL. Estos ficheros se pueden almacenar en un sistema de control de versiones (VCS) y así gestionar la base de datos mediante el VCS.

Para crear una fuente de datos DDL, haz clic con el botón derecho del ratón en la fuente de datos en "Explorador de bases de datos" > "Importar/Exportar | Volcado en una fuente de datos DDL".
