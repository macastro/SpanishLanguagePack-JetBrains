Evaluación de expresiones

Mientras el script esta detenido en un punto de interrupción, se puede ejecutar expresiones o reasignar valores a las variables. La función "Evaluación de expresiones" ayuda a obtener detalles adicionales sobre el estado del programa, o probar diferentes escenarios durante el tiempo de ejecución.

  • Si existe una expresión dentro del código, presiona Alt (Windows y Linux) o Alt (macOS) y haz clic sobre la expresión. Si quieres evaluar un segmento de código específico, selecciónalo antes de hacer clic.

  • Para evaluar cualquier expresión, selecciona "Ejecutar | Acciones de depuración | Evaluación de expresiones".
