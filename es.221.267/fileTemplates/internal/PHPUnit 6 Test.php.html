Plantilla predefinida para pruebas PHPUnit (versión 6 y posteriores). Se utiliza cada vez que se crea una nueva prueba PHPUnit (en una de las vistas del proyecto, seleccione Nuevo | Prueba PHP en el menú emergente y luego seleccione "PHPUnit" como biblioteca de pruebas).
La plantilla es editable. Además de las expresiones, sentencias y comentarios de PHP, también puede utilizar las variables predefinidas que se indican a continuación. Cada variable se expande al valor correspondiente, como una macro.
Puede incluir plantillas de la pestaña Incluir especificando el nombre completo de la plantilla deseada como parámetro entre comillas mediante la directiva #parse.

Las variables predefinidas toman los siguientes valores:
${TESTED_NAME} Nombre de la clase en prueba
${TESTED_NAMESPACE} Espacio de nombres de la clase en prueba
${NAME} Nombre de la clase de prueba
${NAMESPACE} Espacio de nombres de la clase de prueba
${FILE_NAME} Nombre del archivo actual
${USER} Nombre de usuario actual de inicio de sesión del sistema
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${DAY} Día del mes actual
${HOUR} Hora actual
${MINUTE} Minuto actual
${PRODUCT_NAME} Nombre del IDE actual
${DS} El signo de dólar evalúa el carácter '$' normal.
${PROJECT_NAME} Nombre del proyecto actual
