Especifica un fragmento de código que deseas incluir en las plantillas de archivos (pestaña Plantillas) con la ayuda de la directiva #parse.
Varias plantillas de archivos pueden compartir una sola plantilla.

Además del texto estático, código y comentarios, también puedes utilizar las variables predefinidas que luego se expandirán, como macros, en los valores correspondientes.
Las variables predefinidas toman los siguientes valores:
${PACKAGE_NAME} Nombre del paquete en el que se crea un archivo nuevo
${USER} Nombre de inicio de sesión del usuario actual
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${MONTH_NAME_SHORT} Primeras 3 letras del nombre del mes actual (ene, feb, etc.)
${MONTH_NAME_FULL} Nombre completo del mes actual (enero, febrero, etc.)
${DAY} Día actual del mes
${DAY_NAME_SHORT} Primeras 3 letras del nombre del día actual (lun, mar, etc.)
${DAY_NAME_FULL} Nombre completo del día actual (lunes, martes, etc.)
${HOUR} Hora actual
${MINUTE} Minuto actual
${PROJECT_NAME} Nombre del proyecto actual
