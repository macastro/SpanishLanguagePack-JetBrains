Esta plantilla interna se utiliza para rellenar el cuerpo de un método cuando este se genera, por ejemplo, a través de la acción intención Crear un método desde el uso.

La plantilla es editable. Junto con las expresiones de TypeScript y los comentarios, también puede utilizar variables predefinidas. Estas variables se sustituirán por los valores correspondientes cuando se aplique la plantilla.
Las variables predefinidas tomarán los siguientes valores:
${RETURN_TYPE} tipo de retorno del método creado
${DEFAULT_RETURN_VALUE} valor devuelto por el método por defecto
${METHOD_NAME} nombre del método creado
${SEMICOLON} punto y coma o vacío, según las opciones de estilo de código
Todas las variables predefinidas de la plantilla Cabecera de archivo (pestaña Incluye) también están disponibles (p. ej., ${USER}, ${DATE}, etc.)
