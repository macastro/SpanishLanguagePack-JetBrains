Reemplaza una expresión literal de cadena (por ejemplo, String s = "Hello";) por una expresión de código Java internacionalizada (por ejemplo, String s = myResourceBundle.getString("hello");).
Esta plantilla incorporada se puede editar. Junto con las expresiones de Java y los comentarios, puedes utilizar también las variables predefinidas que luego se expandirán en los valores correspondientes.
Las variables predefinidas toman los siguientes valores:
${RESOURCE_BUNDLE} Expresión del tipo java.util.ResourceBundle, cuya instancia está disponible en este contexto.
${PROPERTY_KEY} Nombre de la clave de la propiedad que se especifica en el archivo de propiedades correspondiente. Normalmente es el valor que se utiliza para el parámetro del método ResourceBundle.getString().
${PROPERTY_VALUE} Valor de la propiedad que se define en el archivo de propiedades correspondiente. Este es el valor del literal de cadena Java original.
