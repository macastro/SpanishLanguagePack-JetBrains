Rellena el cuerpo de un método cuando se genera, por ejemplo, cuando se aplica la acción de intención Crear método desde el uso.
Esta plantilla integrada se puede editar. Junto con las expresiones y comentarios de Java, también puedes utilizar las variables predefinidas que luego se ampliarán a los valores correspondientes.
Las variables predefinidas toman los siguientes valores:
${RETURN_TYPE} Tipo de retorno del método creado
${DEFAULT_RETURN_VALUE} Valor devuelto por el método de forma predeterminada
${METHOD_NAME} Nombre del método creado
${CLASS_NAME} Nombre calificado de la clase en la que se crea el método
${SIMPLE_CLASS_NAME} Nombre no calificado de la clase en la que se implementa el método
Todas las variables predefinidas de la plantilla Encabezado de archivo (pestaña Incluye) también están disponibles (por ejemplo: ${USER}, ${DATE}, etc.)
