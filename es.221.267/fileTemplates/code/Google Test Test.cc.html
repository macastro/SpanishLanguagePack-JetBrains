## Plantilla de archivo en IntelliJ IDE
Crea un método de prueba en la suite de pruebas Google Test.
Puedes utilizar las siguientes variables predefinidas, además de expresiones y comentarios C++, para editar la plantilla:
${SuiteName} El nombre de la suite.
${TestName} El nombre del método que se creará.
${END} Mueve el cursor a esta ubicación.
