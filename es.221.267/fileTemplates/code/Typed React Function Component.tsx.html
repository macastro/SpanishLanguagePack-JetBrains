Esta plantilla incorporada se utiliza para crear componentes de función React, por ejemplo, a través del refactorizado de Extraer componente. Puedes personalizar esta plantilla, por ejemplo, definiendo componentes como funciones de flecha o cambiar las definiciones de tipos de prop. También puedes utilizar variables predefinidas. Estas variables se reemplazarán con los valores correspondientes cuando se aplique la plantilla. Esta plantilla se utiliza en TypeScript y Flow.
Las variables predefinidas tomarán los siguientes valores:
${NAME} nombre del componente
${COMPONENT_BODY} cuerpo del componente
${PROP_TYPES} definición del tipo props
