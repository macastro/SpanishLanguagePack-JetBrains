Define la plantilla que se usa para generar JavaDoc para las clases, interfaces o registros, por ejemplo cuando se aplica la acción de intención Agregar Javadoc.

Además de las expresiones y comentarios de Java, también puedes usar las variables predefinidas que luego se expandirán como macros en valores correspondientes.

Al utilizar la directiva #parse, puedes incluir plantillas de la pestaña Includes. Para incluir una plantilla, especifica el nombre completo de la plantilla como un parámetro entre comillas (por ejemplo, #parse("File Header.java").
Las variables predefinidas toman los siguientes valores:
${RECORD_COMPONENTS} Lista de los parámetros de registro
${TYPE_PARAMS} Lista de los nombres de los parámetros de tipo
${CONTAINING_CLASS} Nombre de la clase padre
${INTERFACE} Devuelve true cuando el propietario de JavaDoc es una interfaz
${ENUM} Devuelve true cuando el propietario de JavaDoc es una enumeración
${RECORD} Devuelve true cuando el propietario de JavaDoc es un registro
${PACKAGE_NAME} Nombre del paquete en el que se crea una nueva enumeración
${NAME} Nombre de la nueva enumeración especificado por ti en el cuadro de diálogo Crear nueva clase
${USER} Nombre de inicio de sesión del sistema del usuario actual
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${MONTH_NAME_SHORT} Primeras 3 letras del nombre del mes actual (ene., feb., etc.)
${MONTH_NAME_FULL} Nombre completo del mes actual (enero, febrero, etc.)
${DAY} Día actual del mes
${HOUR} Hora actual
${MINUTE} Minuto actual
${PROJECT_NAME} Nombre del proyecto actual
