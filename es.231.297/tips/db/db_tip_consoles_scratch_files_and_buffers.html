Consolas de base de datos, archivos de borrador y buffers de borrador

Consolas de base de datos son archivos SQL preasignados para la base de datos. Se crean automáticamente al crear un origen de datos. Para abrir una consola, haz clic con el botón secundario en el origen de datos y selecciona "Navegación | Ir a consola de consulta | Abrir consola predeterminada". Para crear una nueva consola, haz clic con el botón secundario en el origen de datos y selecciona "Nuevo | Consola de consulta".

Archivos de borrador son archivos SQL que no pertenecen a ningún origen de datos. Los archivos de borrador se utilizan como notas temporales o borradores para ideas de código fuera del contexto del proyecto. Para crear un archivo de borrador, ve a "Archivo | Nuevo | Archivo de borrador" (o presiona &shortcut:NewScratchFile;).

Buffers de borrador son archivos de texto simples que no tienen funciones de asistencia para la codificación. Se pueden utilizar para listas de tareas simples y notas. No hay un elemento de menú exclusivo para la acción de creación de un nuevo búfer de borrador, pero se puede utilizar el cuadro emergente "Buscar acciones" (&shortcut:GotoAction;) para ejecutar la acción "Nuevo búfer de borrador".
