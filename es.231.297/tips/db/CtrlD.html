Duplicar bloques de código o líneas

Presioni &shortcut:EditorDuplicate; en el editor para duplicar el bloque de código seleccionado o, si no hay ningún bloque seleccionado, para duplicar la línea actual.

Si desea agregar una nueva línea entre el duplicado y el texto original, presione &shortcut:GotoAction; y escriba "Duplicate Entire Lines". Se puede asignar un atajo a esta acción mediante la configuración del elemento del menú "Keymap" &shortcut:ShowSettings;.
