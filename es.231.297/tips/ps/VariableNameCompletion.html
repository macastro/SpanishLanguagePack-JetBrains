Finalización de nombres de variables

Para declarar una variable, puede utilizar la finalización de código para sugerir un nombre para esta. Por ejemplo, cuando declare un parámetro de método, escriba stdClass $ y presione &shortcut:CodeCompletion;.

Puede personalizar el estilo de nombramiento de variables en la página Settings (&shortcut:ShowSettings;) > Editor | Estilo de código | PHP | Generación de código
