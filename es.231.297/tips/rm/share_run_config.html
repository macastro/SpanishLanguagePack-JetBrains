Comparte la configuración de ejecución/depuración

Puedes guardar la configuración de ejecución/depuración como un archivo XML y compartirla con tus compañeros de equipo mediante un sistema de control de versiones. Cuando edites la configuración de ejecución/depuración necesaria, habilita la opción "Guardar como archivo de proyecto" y especifica dónde guardar el archivo de configuración.
