Cuando se genera el subíndice _Implement_ (por ejemplo, mediante la acción **Código | Generar | Implementar métodos**), se completa el cuerpo del subíndice.
La plantilla es editable. Además de las expresiones y comentarios de Swift, puedes utilizar variables predefinidas que se expanden con el valor correspondiente.
Las variables predefinidas reciben los siguientes valores:
${RETURN_TYPE} El tipo de retorno del subíndice creado
${DEFAULT_RETURN_VALUE} El valor predeterminado sugerido para el valor de retorno (puede estar vacío)
${SIGNATURE} La firma del subíndice
${HAS_SET} "true" si el subíndice tiene un setter
${GETTER_EFFECTS} Los efectos del getter para el getter de la propiedad (async, throws)
${GET_ATTRIBUTES} Los atributos del getter del subíndice
${SET_ATTRIBUTES} Los atributos del setter del subíndice
También están disponibles todas las variables predefinidas en la plantilla **Encabezado de archivo Swift** (pestaña **Incluye**) (por ejemplo, ${USER}, ${DATE}).
