Esta es una plantilla incorporada para los comentarios PHPDoc usados en las interfaces PHP. Contiene fragmentos de código que pueden ser incluidas en plantillas de archivos (pestaña Plantilla) con la ayuda de directivas #parse. La plantilla es editable. Además del texto estático, código y comentarios, también se pueden utilizar variables predefinidas, las cuales se expandirán a su valor correspondiente como en las macros.
Las variables predefinidas utilizan los siguientes valores:
${NAME} Nombre de la interface.
${NAMESPACE} Nombre completamente calificado del namespace de la interfaz, sin la barra inclinada inicial.
${DS} El signo de pesos, evaluado como el caracter normal '$'.
${DATE} La fecha actual del sistema
${TIME} La hora actual del sistema
${YEAR} El año actual
${MONTH} El mes actual
${MONTH_NAME_SHORT} Las primeras 3 letras del nombre del mes actual. Ej: Jan, Feb, etc.
${MONTH_NAME_FULL} El nombre completo del mes actual. Ej: January, February, etc.
${DAY} El día del mes actual
${HOUR} La hora actual
${MINUTE} El minuto actual
${SECOND} El segundo actual
${PROJECT_NAME} El nombre del proyecto actual
${USER} El usuario actual
${CARET} Tiene el desplazamiento en el que se deberá mover el caret después de que se haya añadido el comentario.
