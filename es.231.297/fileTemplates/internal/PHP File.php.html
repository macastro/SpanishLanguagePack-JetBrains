Plantilla predeterminada para los archivos PHP. Se utiliza cada vez que se crea un nuevo archivo PHP seleccionando Nuevo | Archivo PHP en el menú emergente de una de las vistas de proyecto.
La plantilla es editable. Además de las expresiones, las declaraciones y los comentarios de PHP, también puedes utilizar las variables predefinidas que se indican a continuación. Cada variable se ampliará al valor correspondiente, tal y como ocurre con las macros.
Mediante la directiva #parse, puedes incluir plantillas en la pestaña Incluye, especificando el nombre completo de la plantilla deseada como parámetro entre comillas.
Las variables predefinidas toman los siguientes valores:
${FILE_NAME} el nombre actual del archivo
${USER} el nombre del usuario que ha iniciado sesión en el sistema
${DATE} la fecha actual del sistema
${TIME} la hora actual del sistema
${YEAR} el año actual
${MONTH} el mes actual
${DAY} el día actual del mes
${HOUR} la hora actual
${MINUTE} el minuto actual
${PRODUCT_NAME} el nombre del IDE actual
${PROJECT_NAME} el nombre del proyecto actual
