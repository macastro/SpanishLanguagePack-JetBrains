Esta es una plantilla integrada para una nueva interfaz PHP. Se utiliza cada vez que crea una nueva interfaz PHP mediante la selección en el menú emergente Nuevo | Clase de PHP en una de las vistas de proyecto.
La plantilla es editable. Además de sentencias, expresiones y comentarios de PHP, también puede usar las variables predefinidas que se enumeran a continuación. Cada variable se expande a su valor correspondiente, como una macro.
Mediante la directiva #parse, puede incluir plantillas en la pestaña Includes especificando el nombre completo de la plantilla que desea como parámetro entre comillas.
Las variables predefinidas toman los siguientes valores:
${FILE_NAME} Nombre de archivo actual
${USER} Nombre de inicio de sesión del sistema del usuario actual
${DATE} Fecha del sistema actual
${TIME} Hora del sistema actual
${YEAR} Año actual
${MONTH} Mes actual
${DAY} Día del mes actual
${HOUR} Hora actual
${MINUTE} Minuto actual
${PRODUCT_NAME} Nombre del IDE actual
${DS} El símbolo de dólar se evalúa en el carácter '$' normal.
${PROJECT_NAME} Nombre del proyecto actual
