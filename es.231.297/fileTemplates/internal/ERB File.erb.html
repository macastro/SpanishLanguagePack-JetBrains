Una plantilla integrada para un archivo ERB. Se utiliza cada vez que se crea un nuevo archivo ERB seleccionando Nuevo | Archivo ERB en el menú emergente de una de las vistas del proyecto. Junto con las instrucciones, expresiones, comentarios, de Ruby y HTML, también puede utilizar las variables predefinidas que se muestran a continuación. Se expande cada variable como macros en el valor correspondiente. Por medio de la directiva #parse, se puede incluir plantillas desde la pestaña Incluye especificando el nombre completo de la deseada plantilla en forma de parámetro entre comillas.
Las variables predefinidas toman los siguientes valores:
${FILE_NAME} El nombre del archivo actual
${USER} El nombre de inicio de sesión del sistema actual
${DATE} La fecha actual del sistema
${TIME} La hora actual del sistema
${YEAR} El año actual
${MONTH} El mes actual
${DAY} El día actual del mes
${HOUR} La hora actual
${MINUTE} El minuto actual
${PRODUCT_NAME} El nombre actual del IDE
${PROJECT_NAME} El nombre del proyecto actual
