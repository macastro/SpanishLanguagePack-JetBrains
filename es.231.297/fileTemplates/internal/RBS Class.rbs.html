Una plantilla incorporada para una clase RBS. Se utiliza cada vez que creas una nueva clase RBS seleccionando Nuevo | Archivo/Clase RBS en el menú emergente en una de las vistas del proyecto. Junto con las sentencias RBS, las expresiones y los comentarios, también puedes utilizar las variables predefinidas que se indican a continuación. Cada variable se expande como macros en el valor correspondiente. Por medio de la directiva #parse, puedes incluir plantillas de la ficha Incluye especificando el nombre completo de la plantilla deseada como parámetro entre comillas.
Las variables predefinidas toman los siguientes valores:
${FILE_NAME} Nombre de archivo actual
${USER} Nombre de inicio de sesión del sistema del usuario actual
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${DAY} Día actual del mes
${HOUR} Hora actual
${MINUTE} Minuto actual
${PRODUCT_NAME} Nombre actual del IDE
${PROJECT_NAME} Nombre actual del proyecto
