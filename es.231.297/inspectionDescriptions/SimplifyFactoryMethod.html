Informa los casos en los que la invocación dinámica podría reemplazarse con una invocación de constructor.

Ejemplo:


  import akka.actor._
  
  class Foo(foo: String, bar: Int) extends AbstractActor
  
  object Foo {
    def props(foo: String, bar: Int): Props = Props(classOf[Foo], foo, bar)
  }
  
  Foo.props("foo", 42)

Después de aplicar la corrección rápida:


  import akka.actor._
  
  class Foo(foo: String, bar: Int) extends AbstractActor
  
  object Foo {
    def props(foo: String, bar: Int): Props = Props(new Foo(foo, bar))
  }
  
  Foo.props("foo", 42)
