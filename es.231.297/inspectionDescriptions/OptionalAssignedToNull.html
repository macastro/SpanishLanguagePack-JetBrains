Informa los valores null asignados a las variables Optional o devueltos de un método que devuelve Optional.

Se recomienda que uses Optional.empty() (o Optional.absent() para Guava) para indicar un valor vacío.

Ejemplo:


  Optional<Integer> foo(boolean flag) {
    return flag ? Optional.of(42) : null;
  }

Después de aplicar la corrección rápida:


  Optional<Integer> foo(boolean flag) {
    return flag ? Optional.of(42) : Optional.empty();
  }

Configura la inspección:

Usa la opción Informar la comparación de Optional con null para también informar comparaciones como optional == null. Si bien en casos poco comunes (por ejemplo, campo opcional inicializado de forma diferida) esto podría ser correcto, la variable opcional por lo general nunca es nula, y probablemente se pretendía optional.isPresent().
