Informa declaraciones switch sobre tipos enumerados que no son exhaustivos.

Ejemplo:


  enum AlphaBetaGamma {
    A, B, C;

    void x(AlphaBetaGamma e) {
      switch (e) {

      }
    }
  }

Después de aplicar la solución rápida:


  enum AlphaBetaGamma {
    A, B, C;

    void x(AlphaBetaGamma e) {
      switch (e) {
        case A -> {}
        case B -> {}
        case C -> {}
      }
    }
  }

Use la opción **Ignorar sentencias switch con una rama predeterminada** para ignorar switch sentencias que tienen una rama default .
