Los informes de métodos con nombres similares a mutadores que no tienen cláusulas de parámetro.

Para un método con efectos secundarios, es convencional tener una cláusula de parámetro vacía para indicar que no es sólo un método de acceso.

La solución rápida agrega una cláusula de parámetro vacía.

Ejemplo:


  trait Test {
    def updateAll: Boolean
  }

Después de aplicar la solución rápida:


  trait Test {
    def updateAll(): Boolean
  }
