Reporta una sincronización sobre una variable local o un parámetro.

Es muy difícil garantizar una correcta operación cuando se utiliza dicha sincronización. Es posible mejorar dicho código, por ejemplo, controlando el acceso utilizando una clase sincronizada envolvente o sincronizando sobre un campo.

Ejemplo:


  void bar() {
    final Object lock = new Object();
    synchronized (lock) { }
  }
