Reporta todos los usos de variables conocidas como constantes.

Esto se aplica cuando las ubicaciones donde se usa (lee) la variable están encerradas en instrucciones if, while o for que contienen una condición == que compara la variable con una constante. En este caso, los usos de la variable conocida como constante se pueden reemplazar por la constante real.

Ejemplo:


    private static void foo(double number) {
        if (number == 1.0) {
            f(number);
        }
    }
    private static void f(double number) {}

Después de aplicar la corrección rápida:


    private static void foo(double number) {
        if (number == 1.0) {
            f(1.0);
        }
    }
    private static void f(double number) {}
