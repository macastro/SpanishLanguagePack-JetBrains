Informa de llamadas a start() en java.lang.Thread o cualquiera de sus subclases durante la construcción del objeto.

Aunque a veces útiles, estas construcciones deben evitarse debido a problemas de herencia. Las subclases de una clase que lanza un hilo durante la construcción del objeto no habrán terminado ninguna lógica de inicialización propia antes de que el hilo se haya lanzado.

Esta inspección no informa si la clase que inicia un hilo está declarada final.

Ejemplo:


  class MyThread extends Thread {
    MyThread() {
      start();
    }
  }
