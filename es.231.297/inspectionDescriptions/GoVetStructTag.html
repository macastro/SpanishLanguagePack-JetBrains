**Inspección de Etiqueta de Estructura Go** Informa sobre etiquetas de estructura que no se ajustan a los convenios de Go para etiquetas de estructura.

Según estas convenciones, las cadenas de etiqueta son una concatenación de pares `clave:"valor"` separados por espacios, opcionalmente. Cada clave es una cadena no vacía que consta de caracteres que no sean de control que no sean espacio (U+0020 ' '), comilla (U+0022 '"'), y dos puntos (U+003A ':'). Cada valor va entre comillas utilizando caracteres U+0022 '"' y sintaxis literal de cadena de Go.

Además, la inspección comprueba que los campos con etiquetas se exportan.

Ejemplo de una etiqueta válida:

type Ejemplo struct {
  Campo int `json:"campo" xml:"demo"`
}
