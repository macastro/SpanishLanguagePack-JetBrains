Informa las sentencias continue si son las últimas sentencias alcanzables en el bucle. Estas sentencias continue son innecesarias y se pueden eliminar de forma segura.

Ejemplo:


  para(int i in arreglo) {
      imprime(i)
      continúa
  }

Después de aplicar la corrección rápida:


  para(int i in arreglo) {
      imprime(i)
  }

Para más información, mira la misma inspección en Java.
