Informa casos de falta de llamada al constructor super en una clase.

Ejemplo:


class Fruit:
    def __init__(self):
        pass


class Pear(Fruit):
    def __init__(self):
        pass

La clase Pear debe tener una llamada super en el método __init__.

Al aplicar la corrección rápida, el código cambia a:


class Fruit:
    def __init__(self):
        pass


class Pear(Fruit):
    def __init__(self):
        super().__init__()
