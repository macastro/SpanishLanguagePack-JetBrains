Informan tipos de argumentos genéricos y comodines que se declaran explícitamente como que extienden `java.lang.Object`.

Esta cláusula `extends` es redundante porque `java.lang.Object` es el supertipo de todas las clases.

Ejemplo:

  class ClassA<T extends Object> {}

Deshabilite la opción Ignorar cuando java.lang.Object está anotado si necesita conservar la cláusula `extends Object` para las anotaciones. Esto puede resultar útil, por ejemplo, cuando se utiliza el analizador de nulidad y la cláusula `extends Object` contiene anotaciones `@Nullable`/`@NotNull`.

Ejemplo:

  class MyClass<T extends @NotNull Object> {}
