Reporta llamadas a Objects.equals(a, b) en las que se sabe estáticamente que el primer argumento no es nulo.

Esta llamada puede reemplazarse de forma segura por a.equals(b) o a == b si ambos argumentos son primitivos.

Ejemplo:


  String defaultName = "default";
  boolean isDefault = Objects.equals(defaultName, name);

Después de aplicar la corrección rápida:


  String defaultName = "default";
  boolean isDefault = defaultName.equals(name);
