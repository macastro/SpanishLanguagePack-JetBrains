Inspecciona nombres de métodos ilegales pasados al atributo зависиendeMétodos en la anotación @Prueba.

Un nombre de método se considera ilegal si no puede resolverse en un método anotado @Prueba válido y accesible en la clase actual o en cualquiera de sus clases padre.

Ejemplo:


  public class PruebaDeMuestra {
    @Prueba(dependeDeMétodos = "pruebaErrorDeOrtografía")
    public void pruebaMuestra() {}
    @Prueba
    public void pruebaErrorDeOrtografía(){}
  }
