Informa sobre las declaraciones de variables, argumentos o funciones que coinciden con la palabra clave integrada o reservada.

Si se recibe este error, es posible que su código no sea lo más explícito posible y podría confundir a otros usuarios.

Ejemplo:

type byte struct{}
type string interface{}

Los tipos byte y string entran en conflicto con los alias de tipo integrados. Por tanto, se resaltarán. Considere renombrar dichas declaraciones.
