Reportea clases/módulos con demasiadas variables de instancia. Esto usualmente significa que se está tratando de 'realizar demasiado', y podrían necesitar ser refactorizados a múltiples componentes más pequeños.

Usa el campo Número de variables de instancia para especificar el número máximo de variables de instancia que se le permiten a una clase/módulo.
