Inspecciona variables locales que se usan únicamente en la siguiente sentencia return o que son copias exactas de otras variables.

Estas variables se pueden insertar en línea de forma segura para que el código sea más claro.

Ejemplo:


  fun sum(a: Int, b: Int): Int {
      val c = a + b
      return c
  }

Después de aplicar la corrección rápida:


  fun sum(a: Int, b: Int): Int {
      return a + b
  }

Configurar la inspección:

Usa la opción Informar sobre variables devueltas inmediatamente para informar sobre variables devueltas inmediatamente. Cuando se les da nombres descriptivos, dichas variables pueden mejorar la legibilidad del código en algunos casos, por eso esta opción está deshabilitada de forma predeterminada.
