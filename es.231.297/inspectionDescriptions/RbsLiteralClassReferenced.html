Informa sobre referencias a las clases NilClass, TrueClass y FalseClass en las signaturas de tipo. El uso de estas clases es innecesariamente verborrágico.

Hay una corrección rápida disponible para reemplazar el uso de NilClass, TrueClass o FalseClass por su literal respectivo.

Ejemplo:

# Se hace referencia a 'NilClass' en lugar de 'nil', ...
escribe ejemplo = NilClass | TrueClass | FalseClass

Después de la corrección rápida:

type example = nil | true | false
