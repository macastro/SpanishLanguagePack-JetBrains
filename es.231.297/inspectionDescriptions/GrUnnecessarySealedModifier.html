Informa sobre modificadores sealed innecesarios que se utilizan en métodos, campos o variables.

Este modificador sólo tiene efecto en clases, interfaces y rasgos.

Ejemplo:


  sealed boolean foo() {} // el modificador es innecesario
  sealed Object bar // el modificador es innecesario

  // el modificador es necesario y por tanto no está resaltado
  sealed class A {}
