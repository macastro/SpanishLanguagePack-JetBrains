Se informa de una conversión segura con return que se puede reemplazar por una comprobación de tipos con if.

El uso de las funciones correspondientes simplifica su código.

La corrección rápida reemplaza la conversión segura con una comprobación de tipos con if.

Ejemplo:


  fun test(x: Any) {
      x as? String ?: return
  }

Después de aplicar la corrección rápida:


  fun test(x: Any) {
      if (x !is String) return
  }
