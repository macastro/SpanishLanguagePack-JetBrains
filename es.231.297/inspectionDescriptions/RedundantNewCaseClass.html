Informa sobre new usado para instanciar clases de casos y sugiere eliminarlo.

El método apply se proporciona automáticamente para clases de casos.

Ejemplo:


  class Book(isbn: String)
  val guide = new Book("978-0345391803")

Después de aplicar la solución rápida:


  class Book(isbn: String)
  val guide = Book("978-0345391803")
