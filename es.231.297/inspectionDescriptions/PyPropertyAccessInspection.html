Informa casos en los que se accede a propiedades de forma inadecuada:
  • Se establecen propiedades de solo lectura
  • Se leen propiedades de solo escritura
  • Se eliminan propiedades no eliminables

Ejemplo:


class MyClass:
    @property
    def read_only(self): return None

    def __write_only_setter(self, value): pass

    write_only = property(None, __write_only_setter)


a = MyClass()
a.read_only = 10 # no se puede establecer propiedad
del a.read_only # no se puede eliminar propiedad
print(a.write_only) # no se puede leer propiedad
