Reporta propiedades de bean setter @Required que no se inyectan ni autowirean.

Ejemplo:


<beans>
  <bean class="beans.MyComponent" id="foo"/>
  <!-- reporta "Faltan propiedades requeridas: 'port'" -->
</beans>


public class MyComponent {
   @Required
   public void setPort(int port) // reporta "La propiedad requerida 'port' no est√° inyectada"
   {...}
}
