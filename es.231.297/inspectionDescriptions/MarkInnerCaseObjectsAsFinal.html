Informa de modificadores finales ausentes para objetos case.

Si bien los objetos case internos son finalmente efectivos en Scala, no se pueden extender y no siempre están marcados como tales en el bytecode compilado. Entonces, esta inspección te anima a hacerlo. Ver buenas prácticas.

Ejemplo:


  object Outer {
    case object Inner
  }

Después de aplicar la solución rápida:


  object Outer {
    final case object Inner
  }
