Reporta las expresiones de tabla comunes (CTE) no usadas dentro de la consulta.

Ejemplo (PostgreSQL):

CREATE TABLE foo(a INT);

WITH a AS (SELECT 1 AS x FROM foo)
SELECT 1 + 2 FROM foo;

Al utilizar WITH, creamos un conjunto de resultados nombrado temporal con el nombre a, también conocido como una expresión de tabla común (CTE). Pero no usamos este CTE posteriormente en el código. El CTE no usado está atenuado.
