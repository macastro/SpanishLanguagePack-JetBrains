Inspecciona las operaciones de acceso a los campos que no están calificadas con this u otro calificador.

Algunos de los estilos de programación requieren que todas las operaciones de acceso a campos estén calificadas para evitar la confusión con el acceso a variables o parámetros locales.

Ejemplo:


  class Foo {
    int foo;

    void bar() {
      foo += 1;
    }
  }

Después de que se aplique la corrección rápida:


  class Foo {
    int foo;

    void bar() {
      this.foo += 1;
    }
  }
