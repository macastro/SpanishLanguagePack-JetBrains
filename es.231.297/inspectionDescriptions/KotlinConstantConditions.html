Reporta condiciones y valores no triviales que estáticamente se sabe que siempre son verdad, falso, nulos o cero. Si bien a veces es intencional, a menudo es un signo de error lógico en el programa. Además, informa ramas when nunca alcanzables y algunas expresiones que estáticamente se sabe que siempre fallan.

Ejemplos:


fun process(x: Int?) {
  val isNull = x == null
  if (!isNull) {
    if (x != null) {} // La condición siempre es verdadera
    require(x!! < 0 && x > 10) // La condición siempre es falsa
  } else {
    println(x!!) // El operador !! siempre fallará
  }
}
fun process(v: Any) {
  when(v) {
    is CharSequence -> println(v as Int) // La conversión siempre fallará
    is String -> println(v) // La rama es inalcanzable
  }
}

Desmarque la opción "Avisar cuando la constante se almacena en una variable" para evitar informar sobre variables que tienen valores constantes que no están en condiciones.
