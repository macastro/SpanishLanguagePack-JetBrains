Reporta una llamada a la función StringBuilder.append(CharArray, offset, len) en la plataforma JVM que debería ser reemplazada por una llamada a la función StringBuilder.appendRange(CharArray, startIndex, endIndex).

La función append se comporta de manera diferente en las plataformas JVM, JS y Native, por lo que se recomienda utilizar la función appendRange.

Ejemplo:


  fun f(charArray: CharArray, offset: Int, len: Int): String {
      return buildString {
          append(charArray, offset, len)
      }
  }

Después de aplicar la corrección rápida:


  fun f(charArray: CharArray, offset: Int, len: Int): String {
      return buildString {
          appendRange(charArray, offset, offset + len)
      }
  }
