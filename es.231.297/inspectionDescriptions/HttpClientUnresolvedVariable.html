Reporta las variables sin declarar en el entorno actual del Cliente HTTP.

Ejecutar solicitudes con variables no declaradas probablemente fallará. Considera añadir una variable al entorno o selecciona un entorno con esta variable.

La inspección no reporta las variables en los cuerpos de las solicitudes, ya que pueden ser una sintaxis válida del cuerpo.

Es posible que algunas variables no se reporten como no resueltas, ya que se declaran en los scripts de respuesta o de controlador previo a la solicitud mediante la llamada a las funciones client.global.set o request.variables.set.
