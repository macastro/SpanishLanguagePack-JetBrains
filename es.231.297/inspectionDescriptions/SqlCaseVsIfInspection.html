Informa situaciones cuando CASE e IF son intercambiables.

Ejemplo (MySQL):

SELECT CASE
WHEN C1 IS NULL THEN 1
ELSE 0
END
FROM dual;

Para mantener tu código corto, puedes reemplazar la estructura CASE por IF. Puedes hacerlo aplicando la acción de intención "Reemplazar con llamada a 'IF'". El código de ejemplo se verá como a continuación:

SELECT IF(C1 IS NULL, 1, 0)
FROM dual;

Para revertir IF a CASE, haz clic en IF y aplica la acción de intención "Reemplazar con expresión CASE".
