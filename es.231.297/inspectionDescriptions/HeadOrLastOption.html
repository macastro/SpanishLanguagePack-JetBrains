Informa cuando headOption o lastOption se emula.

Ejemplo:


  var seq: Seq[Int]
  if (seq.nonEmpty) Some(seq.head) else None
  seq.lift(0)

  if (seq.size == 0) None else seq.lastOption
  seq.lift(seq.size - 1)

Después de aplicar la corrección rápida:


  var seq: Seq[Int]
  seq.headOption
  seq.headOption

  seq.lastOption
  seq.lastOption
