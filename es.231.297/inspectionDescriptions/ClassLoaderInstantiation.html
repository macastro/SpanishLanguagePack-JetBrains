Informa sobre las instanciaciones de la clase java.lang.ClassLoader.

Aunque normalmente son benignas, cualquier instanciación de ClassLoader debe examinarse minuciosamente en cualquier auditoría de seguridad.

Ejemplo:

    Class<?> loadExtraClass(String name) throws Exception {
      try(URLClassLoader loader =
          new URLClassLoader(new URL[]{new URL("extraClasses/")})) {
        return loader.loadClass(name);
      }
    }
