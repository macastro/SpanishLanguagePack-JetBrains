Reporta expresiones de conversión de tipo que están antecedidas por una verificación de instanceof para un tipo diferente.

Aunque esto podría ser intencional, tal construcción es muy probable que sea un error y dará como resultado una java.lang.ClassCastException en tiempo de ejecución.

Ejemplo:


  class Main {
    int whenCharSequenceCastToNumber(Object o){
      if (o instanceof CharSequence) {
        return ((Number) o).intValue();
      }
      return 0;
    }

    int earlyReturnWhenNotCharSequence(Object o){
      if (!(o instanceof CharSequence)) return 0;
      return ((Number)o).intValue();
    }
  }
