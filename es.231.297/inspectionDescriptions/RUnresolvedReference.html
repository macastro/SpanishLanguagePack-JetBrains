Reporta referencias de símbolos y funciones no resueltos.

Ejemplo:


my_fahrenheit_to_celsius(40)

La función my_fahrenheit_to_celsius no está definida, por lo que el IDE muestra una advertencia. Deberás agregar la declaración faltante.

Posible solución: 


my_fahrenheit_to_celsius  <- function(f_temp) {
  return((f_temp - 32) * 5 / 9)
}

my_fahrenheit_to_celsius(40)
