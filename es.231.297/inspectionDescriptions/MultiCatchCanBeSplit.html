Informa sobre secciones catch múltiples y sugiere dividirlas en bloques catch separados.

Ejemplo:


  try {
    int i = getIndex();
  } catch (NullPointerException|IndexOutOfBoundsException e) {
    e.printStackTrace();
  }

Después de aplicar la corrección rápida:


  try {
    int i = getIndex();
  } catch (NullPointerException e) {
    e.printStackTrace();
  } catch (IndexOutOfBoundsException e) {
    e.printStackTrace();
  }
