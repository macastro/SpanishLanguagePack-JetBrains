Informa de expresiones de alias que duplican nombres de columnas en tablas y que podrían ser redundantes.

Ejemplo (PostgreSQL):

CREATE TABLE foo(a INT, b INT);

SELECT * FROM foo foo(a, b);
SELECT * FROM foo foo(a);
SELECT * FROM foo foo(x);
SELECT * FROM foo foo(x, y);

Los dos primeros alias utilizan los mismos nombres de columna que la tabla foo. Se consideran redundantes porque los nombres de columna son idénticos.
