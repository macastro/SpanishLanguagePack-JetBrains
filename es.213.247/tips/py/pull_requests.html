Gestión de peticiones de incorporación de cambios (pull requests)

Puedes gestionar las peticiones de incorporación de cambios de GitHub directamente desde PyCharm. Para ello, selecciona "Git | GitHub | Mostrar peticiones de incorporación de cambios " en el menú principal. Con PyCharm, puedes asignar y combinar peticiones de incorporación de cambios, ver las líneas de tiempo y los comentarios en línea, enviar comentarios y revisiones, y aprobar cambios sin salir del IDE.
