Autocompletar código

El autocompletado de código sugiere las opciones que son alcanzables desde la posición actual del cursor mientras se escribe. Presiona Ctrl+Espacio dos veces y la lista de candidatos se expandirá.

Si usas el completado para resolver un nombre de clase pero la clase aún no se ha importado, IntelliJ IDEA generará automáticamente la declaración de importación.
