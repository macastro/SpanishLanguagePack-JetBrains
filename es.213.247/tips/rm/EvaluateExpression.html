¡Evaluación de expresiones!

Mientras el script se detiene en un punto de interrupción, se puede ejecutar una expresión o reasignar un valor a una variable. La función "¡Evaluación de expresiones!" sirve para obtener detalles adicionales sobre el estado del programa o probar diferentes escenarios en tiempo de ejecución.

Para evaluar y mostrar rápidamente el valor actual de una parte del código, coloca el cursor en el fragmento del código y pulsa &shortcut:QuickEvaluateExpression;. O navega hasta "¡Ejecutar | Acciones de depuración | ¡Evaluación rápida de expresiones!".
