Visualizar anotaciones de Git Blame

Las anotaciones de Git muestran información detallada sobre el origen de cada línea de código (haz clic derecho en el gutter (canalón) y selecciona "Annotate with Git Blame ").

Haz clic derecho en la anotación y selecciona "Show Diff " para ver las diferencias entre la versión actual del archivo y sus versiones anteriores.
