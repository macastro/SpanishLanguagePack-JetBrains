Formateo de código

Para formatear el código conforme a la configuración de estilo de código actual, selecciona en el menú principal la opción "Código | Formatear código " o pulsa &shortcut:ReformatCode;.

También puedes utilizar "Código | Optimizar importaciones " para ordenar las declaraciones import y eliminar las que no se utilicen, conforme a la configuración de estilo de código del proyecto.
