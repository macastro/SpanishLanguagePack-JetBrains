Es una plantilla incorporada para las especificaciones de PHPSpec. Se utiliza cada vez que creas una nueva especificación PHPSpec (elige Nuevo | Prueba PHP en el menú emergente de una de las vistas del proyecto y selecciona 'PHPSpec' como biblioteca de pruebas).
Esta plantilla se puede editar. Además de los enunciados, expresiones y comentarios de PHP, también puedes utilizar las siguientes variables predefinidas que figuran más abajo. Cada variable se ampliará al valor correspondiente como una macro.
Si utilizas la instrucción #parse, podrás incluir plantillas que se encuentran en la pestaña Incluye, especificando el nombre completo de la plantilla deseada como parámetro entre comillas.
Las variables predefinidas reciben los siguientes valores:
${TESTED_NAME} Nombre de la clase que se está probando
${TESTED_NAMESPACE} Espacio de nombres de la clase que se está probando
${NAME} Nombre de la clase de pruebas
${NAMESPACE} Espacio de nombres de la clase de pruebas
${FILE_NAME} Nombre del archivo actual
${USER} Nombre de inicio de sesión del sistema del usuario actual
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${DAY} Día del mes actual
${HOUR} Hora actual
${MINUTE} Minuto actual
${PRODUCT_NAME} Nombre del IDE actual
${DS} El signo de dólar se evalúa como el carácter "$" normal.
${PROJECT_NAME} Nombre del proyecto actual
