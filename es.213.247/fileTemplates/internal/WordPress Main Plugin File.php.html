Una plantilla incorporada para el archivo del plugin principal de WordPress, que se usará cada vez que se cree un nuevo proyecto de tipo de plugin de WordPress. La plantilla se puede editar.
Además de declaraciones, expresiones y comentarios de PHP, también se pueden utilizar variables predefinidas enumeradas a continuación. Cada variable se expandirá al valor correspondiente, similar a una macro.
Al usar la directiva #parse, se pueden incluir plantillas que aparecen en la pestaña Incluidos, especificando la ruta completa de la plantilla deseada como parámetro entre comillas.
Las variables predefinidas toman los siguientes valores:
${PLUGIN_NAME} Nombre del plugin actual derivado del nombre del directorio
${FILE_NAME} Nombre del archivo actual
${USER} Nombre de inicio de sesión del sistema del usuario actual
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${DAY} Día del mes actual
${HOUR} Hora actual
${MINUTE} Minuto actual
${PRODUCT_NAME} Nombre del IDE actual
${PROJECT_NAME} Nombre del proyecto actual
${DS} El signo de dólar evalúa a un carácter '$' normal.
