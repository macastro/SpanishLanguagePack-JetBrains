Incluye un fragmento de código utilizando la directiva #parse que puede incluirse en una plantilla de archivo (pestaña Plantillas).
Esta plantilla se puede editar. Además de código, comentarios y texto estático, también puede utilizar variables predefinidas que se expandirán al valor correspondiente, tal como lo hacen las macros.
Las variables predefinidas toman los siguientes valores:
${FILE_NAME} Nombre del archivo actual
${USER} Nombre de inicio de sesión del sistema del usuario actual
${USER_NAME} Nombre completo del usuario actual
${ORGANIZATION_NAME} Nombre de la organización del proyecto de Xcode
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${MONTH_NAME_SHORT} Las primeras 3 letras del nombre del mes actual. Por ejemplo, Ene, Feb, etc.
${MONTH_NAME_FULL} Nombre completo del mes actual. Por ejemplo, Enero, Febrero, etc.
${DAY} Día del mes actual
${DAY_NAME_SHORT} Las primeras 3 letras del día de la semana actual. Por ejemplo, Lun, Mar, etc.
${DAY_NAME_FULL} Nombre completo del día actual. Por ejemplo, Lunes, Martes, etc.
${HOUR} Hora actual
${MINUTE} Minuto actual
${PROJECT_NAME} Nombre del proyecto actual
${COMPILER_VERSION_MAJOR} Versión principal del compilador Swift
${COMPILER_VERSION_MINOR} Versión secundaria del compilador Swift
${PREFER_EXPLICIT_RETURN} Valor de la opción Preferir retorno explícito (Preferencias | Editor | Estilo de código | Swift | Generación de código)
