Crea el cuerpo de un método -description sobreescrito cuando se genera, por ejemplo, al llamar la acción**Código | Generar | -description**.
La plantilla se puede editar. Junto con expresiones y comentarios de Objective-C, puede utilizar variables predefinidas que se expanden después en sus valores correspondientes.
Las variables predefinidas tomarán los valores siguientes:
${CALL_SUPER} una llamada al supermétodo, para métodos **void** — [super nombreMétodo] con o sin parámetros; para otros métodos — return [super nombreMétodo] con o sin parámetros
${RETURN_TYPE} el tipo de retorno del método creado
${DEFAULT_RETURN_VALUE} el valor que devuelve el método por defecto
${CONTAINING_CLASS} el nombre de la clase en la que se sobreescribe el método
${CUSTOM_CODE} el cuerpo del método generado
Todas las variables predefinidas de la plantilla Encabezado del archivo C (la pestaña Incluye) también están disponibles (${USER}, ${DATE}, etc.)
