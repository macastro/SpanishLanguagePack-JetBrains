Crea el cuerpo de un método reemplazado cuando se genera, por ejemplo, al llamar a la acción Código | Reemplazar método.
La plantilla es editable. Junto con expresiones y comentarios Objective-C, también puede utilizar variables predefinidas que luego se expandirán a los valores correspondientes.
Las variables predefinidas tomarán los siguientes valores:
${CALL_SUPER} una llamada de método super; para los métodos void — [super nombreMétodo] con o sin parámetros; para otros métodos — return [super nombreMétodo] con o sin parámetros
${RETURN_TYPE} el tipo de retorno del método creado
${DEFAULT_RETURN_VALUE} el valor retornado por el método por defecto
${CONTAINING_CLASS} el nombre de la clase donde se reemplaza el método
Todas las variables predefinidas de la plantilla Encabezado de archivo C (la pestaña Include) también están disponibles (por ejemplo, ${USER}, ${DATE}, etc.)
