Esta plantilla incorporada se utiliza para rellenar el cuerpo de un método cada vez que se genera, por ejemplo, a través de la acción de intención Crear método desde uso.
La plantilla se puede editar. Junto con las expresiones JavaScript y los comentarios, también puedes utilizar variables predefinidas. Estas variables se sustituirán por los valores correspondientes cuando se aplique la plantilla.
Las variables predefinidas tomarán los siguientes valores:
${IS_ASSIGNMENT_TARGET} devuelve "true" si el método creado es un objetivo de asignación
${IS_NEW_CALL} devuelve "true" si el método creado está en la llamada "new"
${HAS_EXPECTED_TYPE} devuelve "true" si el método creado está en la ubicación con el tipo esperado
${DEFAULT_RETURN_VALUE} valor predeterminado del tipo esperado para el método creado ("indefinido" si no hay tipo esperado)
${METHOD_NAME} nombre del método creado
${SEMICOLON} punto y coma o vacío, según las opciones de estilo de código
Todas las variables predefinidas de la plantilla Encabezado de archivo (pestaña Incluye) también están disponibles. (p. ej. ${USER}, ${DATE}, etc.)
