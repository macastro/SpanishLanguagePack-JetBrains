Crea el cuerpo de una función sobreescrita cuando se genera, por ejemplo, usando la acción Código | Sobreescribir métodos.
Al editar la plantilla, puedes usar expresiones de C++, comentarios y las siguientes variables predefinidas:
${CALL_SUPER} la llamada a la función base, para funciones void — claseBase::nombreFuncion() con o sin parámetros; para otras funciones - return claseBase::nombreFuncion() con o sin parámetros.
${RETURN_TYPE} el tipo de retorno de la función.
${DEFAULT_RETURN_VALUE} el valor de retorno predeterminado de la función.
Todas las variables predefinidas de la plantilla Encabezado de archivo C (pestaña Incluye), como ${USER} y ${DATE}, están disponibles también.
