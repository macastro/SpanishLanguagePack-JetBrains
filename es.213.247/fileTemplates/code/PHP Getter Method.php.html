Es una plantilla creada para los métodos "getter" en PHP. Por ejemplo, cuando se invoca la solución rápida "Agregar getter y setter".
Las variables predefinidas son:
${STATIC} Si la propiedad es estática, toma un valor 'static', de lo contrario una cadena vacía.
${GET_OR_IS} Prefijo 'get/is'. Si una propiedad es de tipo booleano (sugerencia de tipo), se evalúa como la cadena 'is', de lo contrario se evalúa como la cadena 'get'.
${NAME} Un nombre de método con estilo "CamelCase", sin prefijo "get" y con su primer caracter en mayúsculas. Por ejemplo, el nombre de propiedad '_mi_propiedad' se convierte en 'MiPropiedad'.
${CLASS_NAME} El nombre de la clase que incluye.
${RETURN_TYPE} Contiene una sugerencia de tipo de valor devuelto si el nivel de lenguaje es PHP 7.0 o superior.
${FIELD_NAME} El nombre de propiedad original.
${TYPE_HINT} Sugerencia de tipo para la propiedad, o una cadena vacía si no se especifica.
${DS} El signo de dólar se evalúa como el caracter '$' común.
