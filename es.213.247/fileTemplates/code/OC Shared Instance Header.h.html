Genera una declaración del método +sharedInstance en la sección de cabecera de clase, por ejemplo, al invocar la acción Código | Generar | +sharedInstance.
La plantilla es editable. Junto con expresiones de Objective-C y comentarios, puedes usar variables predefinidas que luego se expandirán a los valores correspondientes.
Las variables predefinidas tomarán los siguientes valores:
${$CLASS_TYPE} el nombre de la clase donde el método está implementado
Todas las variables predefinidas de la plantilla Cabecera de archivo C (la pestaña Incluye) también están disponibles (${USER}, ${DATE}, etc.)
