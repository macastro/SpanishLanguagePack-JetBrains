Crea un cuerpo de un método getter anulado para una propiedad cuando se genera, por ejemplo, al llamar a la acción Código | Anular método.
La plantilla se puede editar. Junto con las expresiones y comentarios de Objective-C, puede usar variables predefinidas que luego se expandirán a los valores correspondientes.
Las variables predefinidas tomarán los siguientes valores:
${CALL_SUPER} una llamada al supermétodo, para métodos void — [super methodName] con o sin parámetros; para otros métodos — return [super methodName] con o sin parámetros
${RETURN_TYPE} el tipo de retorno del método creado
${DEFAULT_RETURN_VALUE} el valor devuelto por el método de forma predeterminada
${CONTAINING_CLASS} el nombre de la clase donde se anula el método
${IVAR_IS_AVAILABLE} "true" cuando la propiedad tiene una variable de instancia
${IVAR} el nombre de la variable de instancia
${SETTER_SEMANTICS} en modo ARC: "copyArc" cuando se establece el atributo copy, "assign" en caso contrario
en modo no ARC: "copy" cuando se establece el atributo copy, "retain" cuando se establece el atributo retain, "assign" en caso contrario
Todas las variables predefinidas de la plantilla Encabezado del archivo C (la pestaña Incluye) también están disponibles (${USER}, ${DATE}, etc.)
