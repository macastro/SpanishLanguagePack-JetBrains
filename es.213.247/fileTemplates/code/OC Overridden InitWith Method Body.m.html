Crea el cuerpo de un método -init anulado cuando se genera, por ejemplo al llamar a la acción Código | Anular método o **Código | Generar | -initWith.
La plantilla es editable. Junto con comentarios y expresiones de Objective-C, también puedes utilizar variables predefinidas que se expandirán en los valores correspondientes.
Las variables predefinidas tomarán los siguientes valores:
${CALL_SUPER} una llamada a un supermétodo, para métodos void — [super nombreMétodo] con o sin parámetros; para otros métodos — return [super nombreMétodo] con o sin parámetros
${RETURN_TYPE} el tipo de retorno del método creado
${DEFAULT_RETURN_VALUE} el valor retornado por el método por defecto
${CONTAINING_CLASS} el nombre de la clase en la que se anula el método
${CUSTOM_CODE} el cuerpo de lo generado
Todas las variables predefinidas de la plantilla Encabezado del Archivo C (la solapa Incluye) también están disponibles (${USER}, ${DATE}, etc.)
