Esta es una plantilla incorporada que se usa para reemplazar el texto dentro del GSP. Por ejemplo:
<html> <body> ¡Hola! </body> </html>
Cuando se utiliza sintaxis de etiqueta personalizada de GSP internacionalizado, Por ejemplo:
<html> <body> <g:message code="HelloKey"/> </body> </html>

La plantilla es editable. Además de las expresiones permitidas en el contexto GSP, puede usar variables predefinidas que se expandirán al valor correspondiente.

Las variables predefinidas asumen los siguientes valores:
${PROPERTY_KEY} El nombre de la clave de la propiedad especificada en el archivo de propiedades correspondiente.
${PROPERTY_VALUE} El valor de la propiedad especificada en el archivo de propiedades correspondiente. Este es el valor del texto GSP original seleccionado.
