JUnit 4 @BeforeClass 및 @AfterClass 어노테이션이 추가된 메서드와 JUnit 5 @BeforeAll 및 @AfterAll 어노테이션이 추가된 메서드 중 public이 아니거나 static이 아니거나 또는 반환 타입이 void가 아니거나 매개변수 해결자를 선언하지 않으면서 인수를 취하는 메서드를 보고합니다. 이러한 메서드는 실수로 생성되기 쉬우며, JUnit 테스트 러너를 통해 실행되지 않습니다.

예:


  public class MainTest {
    @BeforeClass
    String beforeClass(int i) { ... }
  }

빠른 수정 적용 후:


  class MainTest {
    @BeforeClass
    public static void beforeClass() { ... }
  }
