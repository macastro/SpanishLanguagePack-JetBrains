Reporta usos de coma final en literales de objetos.

La advertencia sólo se notifica cuando la versión del lenguaje JavaScript se ha configurado como ECMAScript 5.1.

Las comas finales en literales de objetos están permitidas por la especificación; sin embargo, algunos navegadores podrían lanzar un error cuando se usa una coma final.

Puede configurar opciones de formato para comas finales en Estilo de código | JavaScript o TypeScript | Puntuación.
