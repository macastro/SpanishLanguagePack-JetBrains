Informa sobre una definición no válida de una typing.NamedTuple.

Ejemplo:


import typing


class FullName(typing.NamedTuple):
    first: str
    last: str = ""
    middle: str

Para corregirlo, pon el campo con el valor por defecto después de los campos sin valores por defecto:


import typing


class FullName(typing.NamedTuple):
    first: str
    middle: str
    last: str = ""
