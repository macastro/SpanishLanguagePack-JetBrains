Informa las anotaciones @GuardedBy en campos o métodos static donde la protección es un campo no estático o this.

Proteger un elemento estático con un elemento no estático puede resultar en una concurrencia excesiva, donde múltiples hilos puedan acceder al campo protegido simultáneamente mediante bloqueo en diferentes contextos de objeto.

Ejemplo:


    private ReadWriteLock lock = new ReentrantReadWriteLock();

    @GuardedBy("lock")
    public static void bar() {
        // ...
    }

Las anotaciones @GuardedBy soportadas son:

  • net.jcip.annotations.GuardedBy
  • javax.annotation.concurrent.GuardedBy
  • org.apache.http.annotation.GuardedBy
  • com.android.annotations.concurrency.GuardedBy
  • androidx.annotation.GuardedBy
  • com.google.errorprone.annotations.concurrent.GuardedBy
