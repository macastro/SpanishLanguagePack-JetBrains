Reporta el tipo java.util.Optional o com.google.common.base.Optional que tiene el parámetro de tipo array o colección.

En esos casos, es más claro usar un array o colección vacía para indicar la ausencia de resultado.

Ejemplo:


  Optional<List<Integer>> foo() {
    return Optional.empty();
  }

Este código puede escribirse como:


  List<Integer> foo() {
    return List.of();
  }
