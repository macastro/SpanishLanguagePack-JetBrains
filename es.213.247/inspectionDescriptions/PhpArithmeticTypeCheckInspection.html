Reporta expresiones aritméticas y bit a bit con operandos no admitidos.

A partir de PHP 8.0, el uso de los operadores aritméticos y bit a bit +, -, *, /, **, %, <<, >>, &, |, ^, ~, ++, -- cuando uno de los operandos es un arreglo, recurso u objeto no sobrecargado, resultará en un TypeError. La única excepción es la operación de fusión array + array, que sigue estando admitida.

Ve el código de prueba interno de PHP para la lista completa de operaciones.
