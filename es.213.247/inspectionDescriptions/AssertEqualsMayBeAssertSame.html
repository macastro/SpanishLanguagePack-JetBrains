Reporta llamadas a JUnit assertEquals() que pueden ser reemplazadas con una llamada assertSame() equivalente. Esto es posible cuando los argumentos son instancias de una clase final que no sobreescribe el método `Object.equals()` y explicita que la identidad del objeto es comparada.

Sugiere reemplazar assertEquals() con assertSame().

Ejemplo:


  @Test
  public void testObjectType() {
    Object o = getObject();
    Assert.assertEquals(String.class, o.getClass());
  }

Después de que se aplica el arreglo rápido:


  @Test
  public void testSort() {
    Object o = getObject();
    Assert.assertSame(String.class, o.getClass());
  }
