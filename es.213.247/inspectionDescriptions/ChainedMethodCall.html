Informa las llamadas al método cuyo destino es otra llamada al método. La solución rápida sugiere introducir una variable local.

Ejemplo:


  class X {
    int foo(File f) {
      return f.getName().length();
    }
  }

Después de aplicar la solución rápida:


  class X {
    int foo(File f) {
      final String name = f.getName();
      return name.length();
    }
  }

Utilice las opciones de inspección para alternar las advertencias para los siguientes casos:

  • llamadas al método encadenado en inicializadores de campo, por ejemplo, private final int i = new Random().nextInt();

  • las llamadas al método encadenado que operan en el mismo tipo, por ejemplo, new StringBuilder().append("x: ").append(new X()).append("y: ").append(new Y()).toString();.
