Reporta el uso de UElement del UAST como un PsiElement.

Los PsiElement obtenidos de esta manera son ambiguos.

Para obtener un PsiElement "físico", utiliza UElementKt.getSourcePsiElement(). Para los PsiElement que "emulen" el comportamiento de elementos Java (PsiClass, PsiMethod, etc.), utiliza UElementKt.getAsJavaPsiElement().

Para más información, consulta UAST - Árbol de sintaxis abstracto unificado en la documentación del SDK.
