Reporta clases permitidas que no extienden la clase base **sealed**.

Groovy no requiere que todas las clases permitidas estén disponibles en tiempo de compilación y compiladas junto con la clase base. El compilador no advierte al usuario sobre el manejo de subclases permitidas no extendidas, pero contradice la naturaleza de las clases sealed.

Ejemplo:


  class A permits B {} // reporta B
  class B {}
