Inspecciona calificadores super innecesarios en referencias de campos y llamadas a métodos.

Un calificador super es innecesario cuando el campo o método de la superclase no está oculto/sobreescrito en la clase que lo llama.

Ejemplo:


  class Foo {
    void foo() {}
  }

  class Bar extends Foo {
    void bar() {
      super.foo();
    }
  }

Después de aplicar la corrección rápida:


  class Foo {
    void foo() {}
  }

  class Bar extends Foo {
    void bar() {
      foo();
    }
  }

Usa los ajustes de inspección para ignorar calificadores que ayudan a distinguir el acceso a miembros de la superclase de los miembros de la clase externa que tienen el mismo nombre.

Consulta también las siguientes inspecciones:

  • Java | Visibilidad | El acceso a un campo heredado parece un acceso a un elemento del código circundante
  • Java | Visibilidad | La llamada a un método heredado parece una llamada a un método local
