Informa de un bloque try-finally con resource.close() en finally que puede convertirse en una llamada a resource.use().

use() es más fácil de leer y menos propenso a errores, ya que no es necesario realizar una llamada explícita a close().

Ejemplo:


  fun ejemplo() {
      val reader = File("file.txt").bufferedReader()
      try {
          reader.lineSequence().forEach(::print)
      } finally {
          reader.close()
      }
  }

Tras la aplicación de la corrección rápida:


  fun ejemplo() {
      File("file.txt").bufferedReader().use { reader ->
          reader.lineSequence().forEach(::print)
      }
  }
