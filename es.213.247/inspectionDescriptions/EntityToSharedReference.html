Entity a SharedEntity referencia inválida.

No se puede referenciar Entity desde SharedEntity. Solo se puede referenciar SharedEntity desde Entity si la propiedad admite valores nulos o tiene la anotación @CascadeDeleteBy. De lo contrario, se producirá una excepción de tiempo de ejecución.

Ejemplo:


  interface A: Entity {
    var x: SharedEntity
    var y: SharedEntity
  }

La corrección rápida genera la anotación o hace que el tipo admita valores nulos


  interface A: Entity {
    var x: SharedEntity?
    @CascadeDeleteBy
    var y: SharedEntity
  }
