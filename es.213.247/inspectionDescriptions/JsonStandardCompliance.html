Informa las siguientes discrepancias de un archivo JSON con la especificación del idioma:
  • Un comentario de línea o bloque (configurable).
  • Múltiples valores de nivel superior (esperar archivos JSON Lines, configurable para otros).
  • Una coma final en un objeto o matriz (configurable).
  • Una cadena entre comillas simples.
  • Una clave de propiedad no es una cadena entre comillas dobles.
  • Un valor numérico NaN o Infinity/-Infinity como un literal de punto flotante (configurable).
