Informa de las llamadas JUnit assertEquals() con argumentos de tipo array. Tales métodos comparan las identidades de las matrices en lugar de los contenidos de las matrices. El contenido de las matrices debe comprobarse con el método assertArrayEquals().

Ejemplo:


  @Test
  public void testSort() {
    int[] actual = {248, 496, 0, 56};
    Arrays.sort(actual);
    Assert.assertEquals(new int[] {0, 56, 248, 496}, actual);
  }

Tras la aplicación de la corrección rápida:


  @Test
  public void testSort() {
    int[] actual = {248, 496, 0, 56};
    Arrays.sort(actual);
    Assert.assertArrayEquals(new int[] {0, 56, 248, 496}, actual);
  }
