Informa de las llamadas al método de una clase de datos copy() sin argumentos nombrados.

Como todos los argumentos de la función copy() son opcionales, puede ser difícil entender qué propiedades se modifican. Proporcionar nombres de parámetros explícitamente hace que el código sea fácil de entender sin navegar a la declaración de la data class.

Ejemplo:


  data class Usuario(val name: String, val age: Int)

  fun copyUser(usuario: Usuario): Usuario {
      return usuario.copy("John")
  }

La corrección rápida proporciona nombres de parámetros a todos los argumentos copy():


  data class Usuario(val name: String, val age: Int)

  fun copyUser(usuario: Usuario): Usuario {
      return usuario.copy(name = "John")
  }
