Informa doble negación en llamadas a exists o forall.

El quick-fix remueve la doble negación.

Ejemplo:


  def condition(x: Int): Boolean = ???
  !Seq(1, 2).exists(x => !condition(x))
  !Seq(1, 2).forall(x => !condition(x))

Después de que sea aplicado el quick-fix:


  def condition(x: Int): Boolean = ???
  Seq(1, 2).forall(x => condition(x))
  Seq(1, 2).exists(x => condition(x))
