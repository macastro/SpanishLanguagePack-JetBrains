Informa sobre las declaraciones que pueden definirse como private para seguir el principio de encapsulamiento.

Ejemplo:


class Service(val url: String) {
    fun connect(): URLConnection = URL(url).openConnection()
}

Después de aplicar la corrección rápida (considerando que no hay usos de url fuera de la clase Service):


class Service(private val url: String) {
    fun connect(): URLConnection = URL(url).openConnection()
}
