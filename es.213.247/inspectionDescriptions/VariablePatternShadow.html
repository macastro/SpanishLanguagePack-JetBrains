Informa patrones de variables que oscurecen identificadores estables definidos en el ámbito de cierre.

Para realizar una prueba de igualdad sobre ese valor, usar comillas invertidas.

Ejemplo:


  val foo = 0
  0 match {
    case foo =>
  }

Después de que se aplica la reparación rápida:


  val foo = 0
  0 match {
    case `foo` =>
  }
