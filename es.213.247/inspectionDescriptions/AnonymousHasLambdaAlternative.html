Reporta clases anónimas que podrían ser transformadas a un constructor o una llamada a método de fábrica con un argumento de expresión lambda.

Las siguientes clases son reportadas por esta inspección:

  • Clases anónimas extendiendo ThreadLocal que tienen un método initialValue() (pueden reemplazarse con ThreadLocal.withInitial)
  • Clases anónimas extendiendo Thread que tienen un método run() (pueden reemplazarse con new Thread(Runnable)

Ejemplo:


  new Thread() {
    @Override
    public void run() {
      System.out.println("Hello from thread!");
    }
  }.start();

Después de aplicar la corrección rápida:


  new Thread(() -> {
    System.out.println("Hello from thread!");
  }).start();
