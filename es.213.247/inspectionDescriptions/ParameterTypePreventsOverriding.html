Informa de los tipos de parámetros de un método de subclase que tienen el mismo nombre que el tipo de parámetro del método supercorrespondiente pero que pertenecen a un paquete diferente. En estos casos, el método de la subclase no puede anular el método super.

Ejemplo:


public class A {
 public void method(Object o) {}
}

public class B extends A {
 public void method(Object o) {} // advertencia sobre el tipo de parámetro
 class Object {}
}

Después de aplicar la solución rápida:


public class A {
 public void method(Object o) {}
}

public class B extends A {
 public void method(java.lang.Object o) {} // nuevo tipo de parámetro
 class Object {}
}
