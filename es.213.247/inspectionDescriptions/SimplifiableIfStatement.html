Informa sobre sentencias if que pueden ser reemplazadas por condiciones que utilizan los operadores &&, ||, ==, != o ?:.

El resultado es generalmente más corto, aunque no siempre más claro, por lo que no es recomendable aplicar esta corrección rápida en todos los casos.

Por ejemplo:


  if (condición) return true; else return foo;

Tras aplicar la corrección rápida:


  return condición || foo;

Configuración de la inspección:

  • Usa la opción No sugerir el operador '?' para deshabilitar las advertencias donde se sugiere el operador ?:. En este caso, solo se resaltarán las sugerencias &&, ||, == y !=. Las correcciones rápidas seguirán estando disponibles en el editor.
  • Usa la opción Ignorar las sentencias 'if' encadenadas para deshabilitar las advertencias sobre cadenas if-else. Las correcciones rápidas seguirán estando disponibles en el editor.
