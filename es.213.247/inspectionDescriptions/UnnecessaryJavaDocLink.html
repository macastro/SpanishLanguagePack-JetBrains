Informa los tags Javadoc @see, {@link} y {@linkplain} que hagan referencia al método al que corresponde el comentario, al supermétodo del método correspondiente al comentario o a la clase que contiene el comentario.

Tales enlaces son innecesarios y pueden ser eliminados con seguridad mediante la acción rápida de esta inspección. La acción rápida eliminará el comentario Javadoc completo si el tag es su único contenido.

Ejemplo:


  class Ejemplo {
    /**
     * @see Ejemplo#método
     */
    public void método() { }
  }

Luego de aplicar la acción rápida:


class Ejemplo {
  public void método() { }
}

Utilice el casillero de abajo para ignorar los enlaces en línea ({@link} y {@linkplain}) a los supermétodos. Si bien el enlace a todos los supermétodos es añadido automáticamente por la herramienta Javadoc, un enlace en línea al supermétodo puede requerirse a veces en los textos de los comentarios Javadoc.
