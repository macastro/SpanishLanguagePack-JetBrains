Inspecciona las asignaciones utilizadas como condición en if, while, for, do o expresiones condicionales.

Si bien a veces es intencional, este uso puede resultar confuso y podría indicar un error tipográfico (por ejemplo, = en lugar de ==).

Esta corrección rápida reemplaza = por ==.

Ejemplo:


  void update(String str, boolean empty) {
    // Advertencia: 'empty' se está reasignando,
    // no se compara con str.isEmpty()
    if (empty = str.isEmpty()) {
      ...
    }
  }

Después de aplicar la corrección rápida:


  void update(String str, boolean empty) {
    if (empty == str.isEmpty()) {
      ...
    }
  }
