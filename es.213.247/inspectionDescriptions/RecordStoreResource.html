Reporta los recursos javax.microedition.rms.RecordStore de Java ME que no están abiertos frente a un bloque try y cerrados en el bloque finally correspondiente.

Dichos recursos pueden perderse sin darse cuenta si se lanza una excepción antes de cerrar el recurso.

Esta inspección está pensada para Java ME y otros entornos con recursos muy limitados. Aplicar los resultados de esta inspección sin consideración puede tener efectos negativos en la claridad y el diseño del código.

Ejemplo:


  void foo1() throws RecordStoreException {
      RecordStore rs = RecordStore.openRecordStore("bar", true); // advertencia
  }
  void foo2() throws RecordStoreException {
      RecordStore rs = RecordStore.openRecordStore("bar", true); // no advertencia
      try {
          /* ... */
      } finally {
          rs.closeRecordStore();
      }
  }
