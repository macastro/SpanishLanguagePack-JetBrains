Reporta accesos a campos estáticos donde la llamada es calificada por una subclase de la clase que declara, en lugar de por la clase que declara en sí.

Java permite tal calificación, pero tales accesos pueden indicar una sutil confusión de herencia y sobrescritura.

Ejemplo:


  class Padre {
    static int foo = 0;
  }

  class Hijo extiende Padre { }

  void barra() {
    System.out.println(Hijo.foo);
  }

Después de que se aplica la solución rápida, el resultado se ve así:


  class Padre {
    static int foo = 0;
  }

  class Hijo extiende Padre { }

  void barra() {
    System.out.println(Padre.foo);
  }
