Informa de métodos no públicos referenciados en los atributos factory-method de los elementos <bean>.

Ejemplo:


public class MyFactory {
  private static FooBean createBean() {...}
  }
}


<beans>
  <bean class="MyFactory" factory-method="createBean"/>
  // informa de "El método de fábrica 'createBean' debe ser público"
</beans>
