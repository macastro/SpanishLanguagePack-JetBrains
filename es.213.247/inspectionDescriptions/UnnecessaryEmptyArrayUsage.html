Informa sobre las asignaciones de arreglos con longitudes conocidas de cero cuando hay una constante para eso en la clase del tipo de elemento del arreglo. Como los arreglos de longitud cero son inmutables, puedes ahorrar memoria reutilizando la misma instancia de arreglo.

Ejemplo:


  class Item {
    // Constante de matriz pública de longitud cero que se puede reutilizar
    public static final Item[] EMPTY_ARRAY = new Item[0];
  }
  class EmptyNode {
    Item[] getChildren() {
      // Creación innecesaria de matriz de longitud cero
      return new Item[0];
    }
  }

Después de aplicar la solución rápida:


  class EmptyNode {
    Item[] getChildren() {
      return Item.EMPTY_ARRAY;
    }
  }
