Reporta parámetros de constructores primarios que pueden remover val o var.

Las propiedades de clase declaradas en el constructor incrementan el consumo de memoria. Si el valor de parámetro solo se usa en el constructor, puedes omitirlo.

Nota que el objeto referenciado puede ser recolectado como basura más temprano.

Ejemplo:


  class Task(val name: String) {
      init {
          print("Tarea creada: $name")
      }
  }

La corrección rápida remueve las palabras claves val o var:


  class Task(name: String) {
      init {
          print("Tarea creada: $name")
      }
  }
