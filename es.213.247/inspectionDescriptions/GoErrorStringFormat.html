Informa los problemas de formato en cadenas de error.

Ejemplo:

err := fmt.Errorf("¡No se puede leer el archivo!")
log.Printf("Leyendo %s: %v", file, err)

De acuerdo con Error Strings en github.com/golang, las cadenas de error no deben escribirse con mayúsculas ni terminar con puntuación porque pueden aparecer en otro contexto.

Para corregir el formato, puedes aplicar la corrección rápida Solucionar formato de cadena de error. Una vez aplicada la corrección rápida, la cadena de error se verá así:

err := fmt.Errorf("no se puede leer el archivo")
log.Printf("Leyendo %s: %v", file, err)
