Informa de clases anidadas declaradas sin modificadores de acceso (también conocido como package-private).

Ejemplo:


  public class Outer {
    static class Nested {}  // advertencia
    class Inner {}          // advertencia
    enum Mode {}            // advertencia dependiendo de la configuración
    interface I {}          // advertencia dependiendo de la configuración
  }

Configuración de la inspección:

  • Utilice la opción Ignorar enumeraciones internas visibles en el paquete para ignorar enumeraciones internas de package-private.
  • Utilice la opción Ignorar interfaces internas visibles en el paquete para ignorar interfaces internas de package-private.
