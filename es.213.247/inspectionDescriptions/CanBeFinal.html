Informa de campos, métodos o clases que pueden tener el modificador final añadido a sus declaraciones.

Las clases finales no se pueden extender, los métodos finales no pueden ser sobreescritos y los campos finales no se pueden reasignar.

Ejemplo:


  public class Persona {
    private String nombre;

    Persona(String nombre) {
      this.nombre = nombre;
    }

    public String getNombre() {
      return nombre;
    }

    public String toString() {
      return getNombre();
    }
  }

Después de aplicar la corrección rápida:


  public final class Persona {
    private final String nombre;

    Persona(String nombre) {
      this.nombre = nombre;
    }

    public final String getNombre() {
      return nombre;
    }

    public final String toString() {
      return getNombre();
    }
  }

Utiliza las opciones Informar clases e Informar métodos para definir qué declaraciones se van a informar.
