Informa campos que se inicializan explícitamente con sus valores por defecto.

Ejemplo:


  clase Foo {
    int foo = 0;
    List bar = null;
  }

Después de que se ha aplicado la corrección rápida:


  class Foo {
    int foo;
    List bar;
  }

Usa los ajustes de la inspección para informar solo las inicializaciones explícitas con null, por ejemplo: 


  class Foo {
    int foo = 0;     // sin advertencia
    List bar = null; // advertencia de inicialización de campo redundante
  }
