Informa de los usos de los siguientes elementos que pueden eliminarse de forma segura porque ya no se aplica en este contexto la inspección a la que afectan:
  • anotación @SuppressWarning, o
  • comentario de línea // noinspection, o
  • comentario de JavaDoc /** noinspection */

Ejemplo:


public class C {
 // el símbolo ya es privado,
 // pero la anotación sigue existiendo
  @SuppressWarnings({"WeakerAccess"})
 private boolean CONST = true;
 void f() {
    CONST = false;
  }
}
