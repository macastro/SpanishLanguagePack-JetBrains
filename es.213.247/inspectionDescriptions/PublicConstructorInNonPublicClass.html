Reporta constructores públicos en clases no públicas.

Usualmente, no hay razón para crear un constructor público en una clase con un menor nivel de acceso. Sírvanse notar, sin embargo, que esta inspección cambia el comportamiento de algunas llamadas de reflexión. En particular, Class.getConstructor() no podrá encontrar el constructor actualizado (se debería usar Class.getDeclaredConstructor() en su lugar). No use la inspección si su código o el código de algunos frameworks usados se basa en la accesibilidad del constructor vía getConstructor().

Ejemplo:


  class House {
    public House() {}
  }

Después de que se aplica la solución rápida:


  class House {
    House() {}
  }
