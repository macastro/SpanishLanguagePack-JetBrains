Informa del uso del operador de navegación segura (&.) con un receptor no nulo, que puede omitirse.

Use la lista Receptores ignorados para especificar las llamadas de método utilizadas como receptores que deben ignorarse (por ejemplo, Foo#call o Foo::call para un método de instancia o singleton respectivamente).

Ejemplo:

value = 1
if value&.even?
  # ...
end
