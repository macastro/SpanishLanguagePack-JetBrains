Informa de las expresiones que son afectadas por conversión de boxing automático (envoltorio automático de valores primitivos como objetos). Intenta no usar objetos en lugar de primitivos. Podría afectar significativamente al rendimiento.

Ejemplo:

Integer x = 42;

La corrección rápida hace que la conversión sea explícita:

Integer x = Integer.valueOf(42);

AutoBoxing apareció en Java 5. Esta inspección puede ayudar a degradar para compatibilidad con versiones anteriores con versiones anteriores de Java.
