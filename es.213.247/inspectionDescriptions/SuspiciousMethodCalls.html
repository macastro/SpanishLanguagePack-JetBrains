Informa las llamadas a métodos de colecciones parametrizadas donde el tipo del argumento real no coincide con el tipo del elemento de la colección.

Ejemplo:


  List<Integer> list = getListOfElements();
  list.remove(""); // remove se resalta

En la configuración de la inspección, puede desactivar las advertencias para el código que puede ser correcto, como el siguiente: 


  public boolean accept(Map<Integer, String> map, Object key) {
    return map.containsKey(key);
  }
