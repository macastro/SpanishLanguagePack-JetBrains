Se encuentran usos de los operadores booleanos 'and' y 'or' no cortocircuitados (&, |, &=, |=). Si bien las versiones no cortocircuitadas pueden ser válidas en algunos casos, en la mayoría de los casos deben utilizarse las versiones cortocircuitadas (&& y ||) ya que el uso no intencional de las versiones no cortocircuitadas puede generar errores difíciles de detectar.

Se sugiere una corrección rápida para utilizar las versiones cortocircuitadas.

Ejemplo:


  void foo(boolean x, boolean y, boolean z) {
    if (x | y) { x |= z; }
  }

Después de aplicar la corrección rápida:


  void foo(boolean x, boolean y) {
    if (x || y) { x = x || z; }
  }
