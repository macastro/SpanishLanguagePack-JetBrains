Informa sobre situaciones en que se ha modificado el código fuente de un objeto de base de datos.

La inspección se activa cuando se realiza una introspección de la base de datos o del objeto. La introspección se ejecuta cuando se abre el código fuente de un objeto, se ejecutan sentencias y se realiza la refactorización del código. También puede ejecutar la introspección haciendo clic con el botón derecho en un objeto y seleccionando Actualizar.

La inspección cubre las siguientes situaciones:

  • El código fuente del objeto se modificó en la base de datos pero el código en el editor no se actualizó. Funciona en PostgreSQL, Microsoft SQL Server, Oracle y Sybase ASE.
  • Cambió el código fuente del objeto, realizó una introspección de la base de datos, pero el código fuente ya ha sido modificado por otra persona.
  • El inspector de la base de datos se actualizó en el IDE y debe descargar nuevas propiedades de objetos que faltaban en la versión anterior del inspector.
