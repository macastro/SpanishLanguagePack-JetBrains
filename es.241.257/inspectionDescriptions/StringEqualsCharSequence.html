CharSequence informa de la llamada a String.equals() que incluye un argumento.

String.equals() únicamente puede devolver true para un argumento de String. Para comparar el contenido de String con un argumento que no sea String de CharSequence, utilice el método contentEquals().

Ejemplo:


  boolean equals(String s, CharSequence ch) {
    return s.equals(ch);
  }

Después de aplicar la corrección rápida:


  boolean equals(String s, CharSequence ch) {
    return s.contentEquals(ch);
  }
