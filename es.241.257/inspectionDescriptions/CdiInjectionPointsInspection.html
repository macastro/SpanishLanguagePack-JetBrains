Informa sobre puntos de inyección ambiguos en clases de beans que tienen dependencias ambiguas.

Ejemplo:


  interface Motor {}

  @Singleton
  class MotorGasolina implements Motor {}

  @Singleton
  class MotorElectrico implements Motor {}

  @Singleton
  class Coche {
    @Inject Motor motor; // Dependencia ambigua: hay varios beans que coinciden con el punto de inyección
  }
