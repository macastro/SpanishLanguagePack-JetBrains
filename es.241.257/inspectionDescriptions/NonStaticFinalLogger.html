Informa de campos de registro que no se declaran static o final. Asegurarse de que cada registrador de clase es efectivamente constante y está vinculado a dicha clase simplifica la tarea de proporcionar una implementación de registro unificada para una aplicación.

Se proporciona una corrección rápida para cambiar los modificadores de registro a static final.

Ejemplo:


  public class Significant {
    private Logger LOG = Logger.getLogger(Critical.class);
  }

Después de aplicar la corrección rápida:


  public class Significant {
    private static final Logger LOG = Logger.getLogger(Critical.class);
  }

Configurar la inspección:

  • Usa la tabla Nombre de clase de registro para especificar los nombres de clase de registro. La inspección informará sobre los campos que no son static y final y que son de un tipo igual a uno de los nombres de clase especificados.
