Informa sobre posibles escapadas de this durante la inicialización del objeto. Las escapadas ocurren cuando this se utiliza como argumento de método o como objeto de asignación en un constructor o inicializador. Tales escapadas pueden dar lugar a fallos sutiles, ya que el objeto está ahora disponible en el contexto en el que no está garantizado que se inicialice.

Ejemplo:


  class Foo {
    {
      System.out.println(this);
    }
  }
