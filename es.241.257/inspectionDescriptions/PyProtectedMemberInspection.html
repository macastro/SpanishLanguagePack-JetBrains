Informa los casos en que se accede a un miembro protegido fuera de la clase, un descendiente de la clase donde se define o un módulo.

Ejemplo:


class Foo:
    def _protected_method(self):
        pass


class Bar(Foo):
    def public_method(self):
        self._protected_method()


foo = Foo()
foo._protected_method() # Acceso a un método protegido
