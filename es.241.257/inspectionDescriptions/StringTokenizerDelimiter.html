Informa llamadas al contructor StringTokenizer() o llamadas al método nextToken() que contienen caracteres duplicados en el argumento delimitador.

Ejemplo:


  void printTokens(String text) {
    StringTokenizer tokenizer = new StringTokenizer(text, "\n\n");
    while (tokenizer.hasMoreTokens()) {
      System.out.println(tokenizer.nextToken());
    }
  }

Después de aplicar la corrección rápida:


  void printTokens(String text) {
    StringTokenizer tokenizer = new StringTokenizer(text, "\n");
    while (tokenizer.hasMoreTokens()) {
      System.out.println(tokenizer.nextToken());
    }
  }
