Indica a las definiciones con una anotación @targetName que sobreescriben definiciones con la anotación @targetName pero tienen distintos nombres externos.

Ver La anotación @targetName.

El arreglo rápido reemplaza el nombre externo de la anotación @targetName con el nombre externo de la definición sobreescrita.

Ejemplo:


  trait A:
    @targetName("multiply")
    def *(i: Int): A

  class B extends A:
    @targetName("times")
    override def *(i: Int): A = ???

Después de aplicar el arreglo rápido:


  trait A:
    @targetName("multiply")
    def *(i: Int): A

  class B extends A:
    @targetName("multiply")
    override def *(i: Int): A = ???
