Siempre informa true, siempre false o parte de una expresión bool innecesaria. Estas expresiones booleanas se pueden simplificar para que el código sea más legible. En algunos casos, esto puede indicar otro problema.

예:

func isNonZero(x, y int) bool {
  // La segunda comparación siempre es verdadera o
  // no corre en absoluto
  return x > 0 && x > 0
}

Puede aplicar la solución rápida Simplificar expresión para la parte x > 0 && x > 0. Después de aplicar la solución rápida, la expresión queda: x > 0.
