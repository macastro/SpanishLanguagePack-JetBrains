Informa si una operación bit a bit se puede reemplazar por una llamada al método Long.hashCode o Double.hashCode(). Un constructor donde x es una variable de tipo long o el resultado de una llamada anterior a Double.doubleToLongBits(). (int)(x ^ (x >>> 32))Detecta. Este reemplazo mejora la legibilidad de su código.

Sí:


  int result = (int)(var ^ (var >>> 32));

Después de aplicar la solución rápida:


  int result = Long.hashCode(var);

Esta verificación solo informa si el nivel de idioma del proyecto o módulo es 8 o superior.
