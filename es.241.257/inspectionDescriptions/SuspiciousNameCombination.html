Informa asignaciones o llamadas a funciones donde el nombre de la variable de destino o el parámetro de función no coincide con el nombre del valor asignado.

Ejemplo 1:


  int x = 0;
  int y = x; // x se utiliza como coordenada y

예시 2:


  int x = 0, y = 0;
  // x se usa como coordenada y y y se usa como coordenada x
  Rectangle rc = new Rectangle(y, x, 20, 20);

Configuración de prueba:

Utilice el área grupo de nombres para especificar nombres mutuamente excluyentes. Informa de un error si el nombre del parámetro o el nombre del destino de la asignación contiene palabras de un grupo, pero el nombre de la variable asignada o la variable pasada contiene palabras de otro grupo.

Utilice el área Ignorar método para especificar métodos que no se comprobarán pero que tienen nombres potencialmente sospechosos. Por ejemplo, los parámetros Integer.compare() pueden denominarse x y y, pero pueden no estar relacionados con las coordenadas.
