Reporta aquellos literales no cadena que están siendo utilizados como clave de propiedad de paquete.

Esto es difícil de mantener. Es mejor pasar alrededor las cadenas de lenguajes naturales ya resueltas.

Ejemplo:


  val key =
    if (???) "property.key.1"
    else "property.key.2"

  Bundle.message(key)

Usa en su lugar:


  val string =
    if (???) Bundle.message("property.key.1")
    else Bundle.message("property.key.2")

  string
