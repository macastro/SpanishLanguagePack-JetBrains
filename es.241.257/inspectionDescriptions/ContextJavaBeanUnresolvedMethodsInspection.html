Inspecciona métodos no resueltos en los parámetros initMethod y destroyMethod de la anotación @Bean .

Ejemplo:

Se reportara "No se puede resolver el método" en la expresión 'doInit' si la clase MyBean no contiene el método 'public void 'doInit'(){...}' 

  public class MyBean {...}

  @Component
  public class MyComponent {
     @Bean(initMethod="doInit" )
     public MyBean myBean() {...}
  }

En este ejemplo, la inspección reportara una referencia de método no resuelto si MyBean no define el método doInit().
