Informa de casos en los que los alias de clases y módulos se declaran de forma incoherente. Las clases no pueden usar alias de módulo y los módulos no pueden usar alias de clase.

Establecer un alias de módulo en una clase o establecer un alias de clase en un módulo genera InconsistentClassModuleAliasError.

Puede utilizar una solución rápida para navegar a declaraciones contradictorias.

Sí: 


class User
end

# La clase 'Usuario' se utiliza como destino del alias del módulo
module Customer = User
