Informa de un inicializador de instancia que puede ser static.

Un inicializador de instancia puede ser estático si no hace referencia a ningún miembro no estático de la clase. Los inicializadores estáticos se ejecutan cuando se resuelve la clase y los inicializadores de instancia se ejecutan cada vez que se crea una instancia de la clase.

Esta verificación no informa inicializadores con instancias vacías ni inicializadores de clases anónimas.

Sí:


  class A {
     public static String CONSTANT;
     {
        CONSTANT = "Hello";
     }
  }

Después de aplicar la solución rápida:


  class A {
     public static String CONSTANT;
     static {
        CONSTANT = "Hello"; // Reiniciar solo una vez por clase
     }
  }
