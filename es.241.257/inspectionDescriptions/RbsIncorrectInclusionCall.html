Los informes incluyen llamadas (include, prepend y extend) que hacen referencia a clases. Las llamadas que contienen solo pueden hacer referencia a módulos o interfaces.

Puede utilizar una solución rápida para eliminar llamadas de inclusión incorrectas:

Sí:

class Example
  # Clase 'Objeto' utilizada en la llamada 'incluir'
  include Object
end

Después de aplicar la solución rápida:

class Example
end
