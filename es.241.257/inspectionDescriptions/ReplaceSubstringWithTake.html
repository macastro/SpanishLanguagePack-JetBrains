Informa de llamadas como s.substring(0, x) que se pueden reemplazar por s.take(x).

Usar take() simplifica tu código.

La corrección rápida reemplaza la llamada a substring por take().

Ejemplo:


  fun foo(s: String) {
      s.substring(0, 10)
  }

Tras aplicar la corrección rápida:


  fun foo(s: String) {
      s.take(10)
  }
