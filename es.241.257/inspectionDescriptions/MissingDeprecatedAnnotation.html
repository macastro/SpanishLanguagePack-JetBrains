@deprecated Reporta declaraciones de módulos, clases, campos o métodos que tienen una etiqueta Javadoc @deprecated, pero no una anotación @java.lang.Deprecated.

Ejemplo:


  /**
   * @deprecated use {@code example()} instead
   */
  void sample(){ }

Después de aplicar la corrección rápida:


  /**
   * @deprecated use {@code example()} instead
   */
  @Deprecated
  void sample(){ }

Esta inspección solo informa si el nivel de lenguaje del proyecto o módulo es 5 o superior.
