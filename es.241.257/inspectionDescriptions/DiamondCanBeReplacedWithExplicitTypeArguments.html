Informa la instanciación de clases genéricas en las cuales se utiliza el símbolo <> (rombo) en lugar de los parámetros de tipo.

La corrección rápida remplaza <> (rombo) con parámetros de tipo explícitos.

Ejemplo:

List <String> list = new ArrayList<>()

Después de aplicar la corrección rápida:

List <String> list = new ArrayList<String>()

La operación de rombo apareció en Java 7. Esta inspección puede ayudarte a degradar a una versión anterior de Java para compatibilidad hacia atrás.
