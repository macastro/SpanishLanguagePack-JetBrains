Informa sobre columnas en la cláusula USING que no existen en ambas tablas.

Ejemplo (MySQL):

CREATE TABLE t1 (i INT, j INT);
CREATE TABLE t2 (k INT, l INT);
SELECT * FROM t1 JOIN t2 USING (j);

En las cláusulas USING, el nombre de una columna debe estar presente en ambas tablas, y la consulta SELECT unirá automáticamente esas tablas utilizando el nombre de columna proporcionado. Como no tenemos la columna j en t2, podemos reescribir la consulta utilizando ON. La cláusula ON puede unir tablas en las que los nombres de columna no coinciden en ambas tablas.

SELECT * FROM t1 JOIN t2 ON t1.j = t2.l;
