Reporta campos public. Las constantes (campos marcados con static y final) no son reportadas.

Ejemplo:


  class Main {
    public String name;
  }

Después de aplicar la solución rápida:


  class Main {
    private String name;

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }
  }

Configurar la inspección:

  • Usa la lista Ignorar si cuenta con anotaciones para especificar anotaciones a ignorar. La inspección ignorará los campos con cualquiera de estas anotaciones.
  • Usa la opción Ignorar campos 'public final' de enums para ignorar los campos public final del tipo enum.
