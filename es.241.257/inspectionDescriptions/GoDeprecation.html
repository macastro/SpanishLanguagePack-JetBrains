Informa casos de uso para elementos que pronto quedarán obsoletos.

예:

// Pronto quedará obsoleto: utilice io.SeekStart, io.SeekCurrent y io.SeekEnd.
const (
 SEEK_SET int = 0 // Ir al principio del archivo
 SEEK_CUR int = 1 // Mover según el desplazamiento actual
 SEEK_END int = 2 // Mover al final del archivo
)

Según Constantes en go.dev SEEK_SET, SEEK_CUR y SEEK_END están en desuso.
