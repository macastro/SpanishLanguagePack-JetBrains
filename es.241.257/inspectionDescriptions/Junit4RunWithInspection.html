Informa cuando las clases padre e hijo en una jerarquía de pruebas JUnit están anotadas con @RunWith. Puede provocar un comportamiento de prueba inesperado.

Ejemplo:


  @RunWith(Suite.class)
  @SuiteClasses(MySuiteClass.class)
  public abstract Padre {
  }

  @RunWith(Parameterized.class)
  public MiPrueba {
  }
