Informa los objetos de Kotlin registrados como extensiones de complemento.

El ciclo de vida de la extensión lo gestiona la plataforma IntelliJ. El uso de objetos Kotlin para el registro de extensiones puede generar instancias de extensión innecesarias y puede impedir que se descargue el complemento.

Sí

Registro de extensión:


<annotator language="myLang" implementationClass="com.example.MyAnnotator"/>

Implementación de extensión:


// Malo:
object MyAnnotator : Annotator {
  ...
}

// Bueno:
class MyAnnotator : Annotator {
  ...
}
