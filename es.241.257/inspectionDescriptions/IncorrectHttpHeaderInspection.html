Informa encabezados HTTP desconocidos que no coinciden con ningún encabezados conocidos públicamente. La solución rápida sugiere agregar el encabezado a la lista de encabezados personalizados si la opción Usar encabezados HTTP personalizados está habilitada. Los encabezados HTTP incluidos en la lista de encabezados personalizados no activan esta verificación.
