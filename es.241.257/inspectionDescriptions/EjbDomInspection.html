Se informan los siguientes problemas que pueden ocurrir en ejb-jar.xml y otros descriptores XML que se utilizan para desplegar EJB en un servidor Java EE:
  • Referencias a clases que no se pueden instanciar
  • Referencias a clases que no extienden las clases necesarias
  • Referencias a clases cuyo ámbito es inadecuado
  • Etiquetas y valores de atributos vacíos
  • Etiquetas y valores de atributos que no coinciden con los patrones requeridos (por ejemplo, identificadores de Java)
  • Etiquetas que no contienen las subetiquetas o atributos requeridos
  • Etiquetas que definen objetos con nombres duplicados
