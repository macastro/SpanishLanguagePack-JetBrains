Reporta cualquier método público que no contenga una sentencia de registro. Esta inspección no informa sobre getters y setters simples.

Por ejemplo:


  public class Crucial {
    private static final Logger LOG = LoggerFactory.getLogger(Crucial.class);
    public void doImportantStuff() {
      // advertencia para este método
    }

    public void doOtherStuff() {
      LOG.info("do other stuff");
    }
  }
