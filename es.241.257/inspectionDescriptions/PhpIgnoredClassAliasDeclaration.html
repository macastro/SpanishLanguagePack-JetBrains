Informa de una declaración de alias de clase que el IDE ignoró porque existe una declaración de clase real con el mismo FQN.

Esta verificación tiene como objetivo aclarar el comportamiento del IDE cuando están presentes declaraciones de clase y de alias de clase. Si hay varias declaraciones con el mismo FQN, el IDE normalmente no elige una de ellas. Esto conduce a una finalización incorrecta del código, una inferencia de tipos incorrecta, etc. Debido a que las declaraciones de clase tienen prioridad sobre las declaraciones de alias de clase, las declaraciones de alias de clase se tratan como redundantes y el IDE las ignora.

Ignorar las declaraciones de alias de clase no afecta el comportamiento del intérprete PHP.
