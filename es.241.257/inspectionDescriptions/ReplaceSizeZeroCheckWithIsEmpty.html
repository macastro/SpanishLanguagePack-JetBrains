Informa que verifica size == 0 de Colecciones/Array/String, que debe reemplazarse con isEmpty().

Usar isEmpty() simplifica el código.

Una solución rápida es reemplazar la verificación de tamaño con isEmpty().

Sí:


  fun foo() {
      val arrayOf = arrayOf(1, 2, 3)
      arrayOf.size == 0
  }

Después de aplicar la solución rápida:


  fun foo() {
      val arrayOf = arrayOf(1, 2, 3)
      arrayOf.isEmpty()
  }
