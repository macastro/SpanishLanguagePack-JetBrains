Informa las llamadas associate() y associateTo(), que se pueden reemplazar con associateBy() o associateWith() .

Estas dos funciones aceptan una función de transformación que se aplica a los elementos de una secuencia o colección determinada (como receptor). Este par se utiliza para construir el Mapa resultante.

Teniendo en cuenta que el convertidor hace referencia a it, llamar a associate[To]() sería más adecuado para associateBy() o asociarCon().

Sí:

  fun getKey(i: Int) = 1L
  fun getValue(i: Int) = 1L

  fun test() {
      arrayOf(1).associate { getKey(it) to it }  // 'asociado()' reemplazable
      listOf(1).associate { it to getValue(it) } // 'asociado()' reemplazable
  }

Después de aplicar la solución rápida:

  fun getKey(i: Int) = 1L
  fun getValue(i: Int) = 1L

  fun test() {
      arrayOf(1).associateBy { getKey(it) }
      listOf(1).associateWith { getValue(it) }
  }
