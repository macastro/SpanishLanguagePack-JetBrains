Informa de llamadas a la función filterIsInstance de la biblioteca estándar de Kotlin con un argumento de la clase literal. Es más idiomático utilizar una versión de esta función con un parámetro de tipo concreto para evitar la sintaxis ::class.java.

Nota: la inspección no se informa para los literales de clase genéricos porque la sintaxis Class<*, *> en la lista de argumentos de tipo puede ser indeseable.

Ejemplo:


  fun foo(list: List<*>) {
      list.filterIsInstance(Int::class.java)
  }

Tras aplicar la corrección rápida:


  fun foo(list: List<*>) {
      list.filterIsInstance<Int>()
  }
