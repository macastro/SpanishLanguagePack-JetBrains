Informa de bucles que se pueden contraer en una única llamada List.replaceAll().

Ejemplo:


  for (int i = 0; i < strings.size(); i++) {
    String str = strings.get(i).toLowerCase();
    strings.set(i, str);
  }

Después de aplicar la corrección rápida:


  strings.replaceAll(String::toLowerCase);
