Informa anotaciones esperadas opcionalmente sin anotación real en algunos módulos de la plataforma.

Ejemplo:

// código común
@OptionalExpectation
expect annotation class JvmName(val name: String)

@JvmName(name = "JvmFoo")
fun foo() { }

// código jvm
actual annotation class JvmName(val name: String)

La inspección también informa casos cuando se omite actual annotation class JvmName para plataformas no JVM (por ejemplo, Native).
