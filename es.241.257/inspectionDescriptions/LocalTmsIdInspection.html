Reporta identificadores faltantes y duplicados para ítems de TMS locales declarados en Markdown.

No se puede enlazar un elemento TMS con identificador faltante o inválido con el código (pruebas unitarias).

La corrección rápida genera una identificación única.

Se reporta una advertencia si falta el identificador de caso de prueba y un error si no es único. Se reporta un error si el número de identificador de ejecución/suite de prueba no es único.
