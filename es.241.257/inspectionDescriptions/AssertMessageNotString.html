Informa de mensajes assert que no son del tipo java.lang.String.

Utilizar una cadena proporciona más información para ayudar a diagnosticar el fallo o el motivo de la aserción.

Ejemplo:


  void foo(List<Integer> myList) {
    assert myList.isEmpty() : false;
  }

Utiliza la opción Advertir solo cuando el tipo de mensaje assert sea 'boolean' o 'java.lang.Boolean' para advertir únicamente cuando el tipo de mensaje assert sea boolean o java.lang.Boolean. Es improbable que un mensaje de detalle boolean proporcione información adicional sobre un fallo de aserción y podría ser el resultado de haber introducido por error : en lugar de &.
