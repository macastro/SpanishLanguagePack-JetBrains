@com.intellij.openapi.components.Service Informa clases que están anotadas pero no son definitivas.

Si tu clase es una esfera, te sugiero que la hagas definitiva.

Sí:


  // MyService.kt
  @Service(Service.Level.APP)
  open class MyService

Después de aplicar la solución rápida:


  // MyService.kt
  @Service(Service.Level.APP)
  class MyService

Para clases o interfaces abstractas, sugiero eliminar la anotación Service.

Sí:


  // MyService.java
  @Service(Service.Level.APP)
  abstract class MyService {}

Después de aplicar la solución rápida:


  // MyService.java
  abstract class MyService {}
