Informa las llamadas de toString() en plantillas de cadenas que se pueden eliminar de manera segura.

Ejemplo:

  fun foo(a: Int, b: Int) = a + b

  fun test(): String {
      return "Foo: ${foo(0, 4).toString()}" // 'toString()' es redundante
  }

Después de aplicar la corrección rápida:

  fun foo(a: Int, b: Int) = a + b

  fun test(): String {
      return "Foo: ${foo(0, 4)}"
  }
