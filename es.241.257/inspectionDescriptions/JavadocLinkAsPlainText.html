Reporta enlaces de texto sin formato en los comentario Javadoc.

El arreglo rápido sugiere envolver el enlace en una etiqueta <a> .

Ejemplo:


  class Main {
   /**
    * https://en.wikipedia.org/
    */
    void foo() {}
  }

Después de aplicar el arreglo rápido:


  class Main {
   /**
    * <a href="https://en.wikipedia.org/">https://en.wikipedia.org/</a>
    */
    void foo() {}
  }
