Informa las llamadas a los métodos java.util.Map.put(). Esas llamadas pueden remplazarse por la forma de acceso con clave, más corta y clara.

Ejemplo:


def map = ["foo": "bar"]
map.put("foo", "baz") // map.put("foo", "baz") podría reemplazarse con map["foo"] = "baz"

Después de aplicar la corrección rápida:


def map = ["foo": "bar"]
map["foo"] = "baz"
