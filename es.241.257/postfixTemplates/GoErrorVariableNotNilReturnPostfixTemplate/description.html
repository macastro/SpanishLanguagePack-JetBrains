Genere código repetitivo para el manejo de errores idiomáticos.

Se puede invocar en variables y expresiones de tipo error.

  • Reemplace la variable err con la siguiente sintaxis:

    if err != nil {
    return value1, value2, ...
  }

  • Reemplace la expresión expr con la siguiente sintaxis:

    if err := expr; err != nil {
    return value1, value2, ...
  }

Cada value es una variable err o el valor predeterminado del tipo de retorno correspondiente de la función adjunta. Puede llamar a esta plantilla para expresiones que toman múltiples valores.
