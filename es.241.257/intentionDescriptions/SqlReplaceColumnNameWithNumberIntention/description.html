Reemplazar nombre de columna por referencia de posición.

Algunos DBMS pueden referenciar columnas en ORDER BY y GROUP BY con el número de secuencia de la columna. ORDER BY es soportado en todos los dialectos excepto Cassandra, ClickHouse, Hive, Spark y HSQLDB. GROUP BY es soportado en MySQL, Maria, PostgreSQL (y sus subdialectos), Exasol, SQLite y Snowflake.

Ejemplo (PostgreSQL):

CREATE TABLE tab(a INT, b INT);
SELECT * FROM tab ORDER BY 1;

Puede utilizar esta acción de intención para reemplazar 1 en la consulta SELECT por el nombre de la columna. De este modo, tras aplicar la acción de intención, la consulta tendrá el siguiente aspecto:

SELECT * FROM tab ORDER BY a;
