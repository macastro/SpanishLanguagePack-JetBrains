Búsqueda en todas partes

Presiona Shift 2 veces para buscar en todo el proyecto archivos, acciones, símbolos, elementos de la interfaz de usuario, ramas de Git y comentarios. Presiona Shift 2 veces de nuevo para buscar también elementos fuera del proyecto.

Utiliza las pestañas para filtrar los resultados de la búsqueda o utiliza directamente el atajo necesario (por ejemplo, &shortcut:GotoClass; para clases, &shortcut:GotoFile; para archivos, &shortcut:GotoSymbol; para símbolos, &shortcut:GotoAction; para acciones).
