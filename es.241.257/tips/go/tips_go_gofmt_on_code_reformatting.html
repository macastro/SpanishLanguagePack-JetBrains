Ejecutar 'gofmt' después de la acción de reformateado

Se puede ejecutar gofmt cada vez que se reformatea código (p. ej., presionando &shortcut:ReformatCode; ). Primero se realizará el reformateado con el formateador integrado de GoLand y después con gofmt.

Para habilitar esta función, presiona &shortcut:ShowSettings; para abrir la configuración navega hasta "Editor | Estilo de código | Go ". Haz clic en la pestaña "Otro " y selecciona la casilla de verificación "En la acción 'Reformat Code' ".
