Live Edit

El uso de Live Edit en sesiones de depuración permite supervisar cambios en el código sin necesidad de refrescar la página. ¿Sabías que Live Edit se puede utilizar ahora sin la extensión JetBrains IDE Support para Chrome? Solo debes abrir el cuadro de diálogo "Configuración" &shortcut:ShowSettings; y dirigirte a "Construcción, ejecución y despliegue | Depurador | Live Edit", y a continuación seleccionar la casilla de verificación "Intervalo de cambios para actualizar aplicaciones de Chrome".
