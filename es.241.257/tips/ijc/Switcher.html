Switcher de navegación

Para navegar entre archivos y ventanas de herramientas abiertos, use el switcher &shortcut:Switcher;. Mantenga presionada la tecla Control para salir del popup abierto del switcher. Para navegar, use Tab y Shift+Tab.

Para cerrar pestañas de edición u ocultar ventanas de herramientas, presione Delete o BackSpace.
