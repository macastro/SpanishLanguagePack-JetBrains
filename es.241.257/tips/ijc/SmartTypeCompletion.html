Finalización de código con coincidencia de tipos

La finalización de código con "coincidencia de tipos" analiza el tipo anticipado de toda la expresión y te ayuda a encontrar métodos y variables aplicables en el contexto actual. Esta finalización funciona después de la palabra clave return, en asignaciones, en la lista de argumentos de llamadas a métodos, etc. Puedes presionar &shortcut:SmartTypeCompletion; ( Código | Finalización de código | Coincidencia de tipos ) para acotar la lista de finalización.
