Docker Compose como intérprete remoto

Para configurar Docker Compose como intérprete remoto, abre el cuadro de diálogo "Ajustes" &shortcut:ShowSettings; y dirígete a la página "Idiomas y marcos | SDKs de Ruby y Gem". Haz clic en el botón "Agregar" y selecciona "Nuevo remoto". En el cuadro de diálogo que aparece, selecciona Docker Compose y especifica la configuración necesaria.
