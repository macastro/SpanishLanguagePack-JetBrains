Buscar en todas partes

Presiona Shift dos veces para buscar archivos, acciones, símbolos, elementos de IU, ramas de Git y comentarios en todo el proyecto. Presiona Shift dos veces de nuevo para buscar también elementos que no estén incluidos en el proyecto.

Para reducir los resultados de la búsqueda, usa las pestañas o directamente los atajos necesarios (para clases &shortcut:GotoClass;, para archivos &shortcut:GotoFile;, para símbolos &shortcut:GotoSymbol;, para acciones &shortcut:GotoAction;).
