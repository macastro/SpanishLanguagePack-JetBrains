Documentación externa

Para abrir la página de documentación del elemento que se encuentra en la posición del cursor, presione &shortcut:ExternalJavaDoc; o seleccione "Herramientas | Navegador web y vista previa" en el menú principal.

Revise la configuración para verificar que el navegador predeterminado esté configurado. Para verificar el navegador predeterminado, presione &shortcut:ShowSettings; para abrir la configuración, luego navegue hasta "Herramientas | Navegador web y vista previa".
