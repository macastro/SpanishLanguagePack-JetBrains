Ejecutar funciones del IDE desde la terminal

Si ves un comando resaltado en la terminal, no lo ejecutes inmediatamente. Presiona Control+Enter (en Windows y Linux)/Control+Enter (en macOS) para activar la función correspondiente del IDE.

Por ejemplo, puedes intentar usar git log para ver la pestaña Log en la ventana de herramientas de Git, o intentar usar "Mostrar diferencias" para abrir el archivo en el visor de diferencias.
