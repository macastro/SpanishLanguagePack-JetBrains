Autocompletado de código

Al utilizar el autocompletado de código, presiona &shortcut:EditorChooseLookupItemReplace; para aceptar el elemento seleccionado que aparece destacado en la lista emergente.

A diferencia de aceptar con &shortcut:EditorChooseLookupItem;, el resto del nombre a la derecha del cursor se sobrescribe con el nombre seleccionado. Esto es particularmente útil para reemplazar un nombre de método o variable por otro nombre de método o variable.
