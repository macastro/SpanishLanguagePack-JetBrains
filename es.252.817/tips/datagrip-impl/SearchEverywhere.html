Buscar en cualquier parte

Presione Shift dos veces para buscar archivos, acciones, símbolos, elementos de interfaz de usuario, ramas de Git y comentarios en su proyecto. Pulsando dos veces Shift de nuevo, se ampliará la búsqueda a elementos que no sean del proyecto.

Use pestañas o accesos directos directos &shortcut:GotoClass; para objetos de base de datos, &shortcut:GotoFile; para archivos, &shortcut:GotoSymbol; para símbolos y &shortcut:GotoAction; para acciones para limitar los resultados de búsqueda.
