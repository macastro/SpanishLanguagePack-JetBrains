Duplicar un bloque de código o una línea

Presiona &shortcut:EditorDuplicate; en el editor para duplicar el bloque de código seleccionado o la línea actual cuando no se selecciona ningún bloque.

Si quieres que haya un salto de línea entre la parte duplicada y el texto original, presiona &shortcut:GotoAction; y escribe Duplicar líneas completas. Puedes asignar un atajo a esta acción en la configuración &shortcut:ShowSettings; en el elemento de menú Keymap.
