Abrir una tabla en el editor de datos

Al hacer doble clic sobre una tabla en la ventana de herramientas Base de datos (Ver | Ventanas de herramientas | Base de datos), la tabla se abre en el editor de datos.

En el editor de datos, puede crear y eliminar nuevas entradas, modificar las existentes, filtrar datos por una columna, ver datos en el visor separado y comparar dos tablas.
