Visor de diferencias para objetos de base de datos

Tu IDE tiene un visor de diferencias para objetos de base de datos. Por ejemplo, en la ventana de herramientas Database, puedes seleccionar dos tablas, hacer clic con el botón derecho sobre la selección y elegir Compare. La diferencia entre estas tablas estará disponible en el cuadro de diálogo separado del visor de diferencias.
