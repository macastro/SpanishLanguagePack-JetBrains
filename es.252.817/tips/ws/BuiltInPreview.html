Vista previa integrada

La vista previa integrada de WebStorm muestra el resultado del código HTML en una pestaña separada del editor. WebStorm guarda cualquier cambio que realices en el archivo HTML o en los archivos vinculados de CSS y JavaScript, y recarga automáticamente la vista previa.

Para abrir la vista previa, coloca el cursor sobre el código HTML para mostrar el menú emergente del navegador y haz clic en el icono de WebStorm.
