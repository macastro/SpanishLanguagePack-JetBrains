Navegar entre errores

Presiona &shortcut:GotoNextError; / &shortcut:GotoPreviousError; para navegar entre errores de sintaxis resaltados.

Presiona &shortcut:PreviousOccurence; / &shortcut:NextOccurence; para navegar entre mensajes de error o resultados de búsqueda.

Para omitir las advertencias, haz clic derecho en la barra de validación/marcadores y selecciona Ir solo a problemas de alta prioridad.
