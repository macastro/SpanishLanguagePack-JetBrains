Búsqueda en cualquier lugar

Presiona Shift 2 veces para buscar archivos, acciones, símbolos, elementos de UI, ramas de Git y comentarios en todo el proyecto. Presiona Shift 2 veces de nuevo para incluir elementos no incluidos en la búsqueda.

Acotar los resultados de la búsqueda mediante las pestañas o utilizando el atajo directo requerido (por ejemplo, &shortcut:GotoClass; para clases, &shortcut:GotoFile; para archivos, &shortcut:GotoSymbol; para símbolos y &shortcut:GotoAction; para acciones).
