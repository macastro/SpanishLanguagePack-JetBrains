Rodear bloque de código

Para rodear rápidamente un bloque de código con un constructo válido, selecciona el bloque de código en cuestión en el editor y pulsa &shortcut:SurroundWith; (Código | Rodear).

La lista de opciones o contenedores disponibles depende del contexto y el lenguaje. Por ejemplo, puedes realizar acciones tales como envolver bloques html con etiquetas.
