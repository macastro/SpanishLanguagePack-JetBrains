File Watchers

La inspección "El observador de archivos está disponible " se activa para todos los archivos que se reconocen para ser aplicables a un observador de archivos predefinido (SASS, LESS, SCSS, TypeScript, Jade, Dart2JS, CoffeeScript). Si no hay un observador de archivos predefinido aplicable para el proyecto actual, WebStorm sugiere agregar uno de los observadores de archivos predefinidos.
