Desestructuración

La desestructuración te permite extraer fácilmente valores de arrays y objetos en variables. Esta característica tiene una sintaxis muy limpia y se usa comúnmente cuando necesitas pasar datos en tu aplicación.

En WebStorm, puedes invocar la desestructuración usando la acción de intención (&shortcut:ShowIntentionActions;). Usa la acción Reemplazar con desestructuración de objeto/array para eliminar la asignación original. Para mantener la asignación, usa Introducir desestructuración de objeto/array.
