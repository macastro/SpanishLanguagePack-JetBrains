Agregar declaración de copyright

En PyCharm Community, puedes configurar diferentes declaraciones de copyright para diferentes conjuntos de archivos. Para mayor comodidad, puedes utilizar variables. Por ejemplo, usa $today.year para mantener el año actualizado.

En el cuadro de diálogo Settings (&shortcut:ShowSettings;), selecciona Editor | Copyright | Copyright Profiles, crea un nuevo perfil y agrega el texto de copyright usando variables.
