Barra de navegación

La barra de navegación es un reemplazo para la ventana de herramientas "Explorador de proyectos"Proyecto. Puedes revisar la estructura del proyecto, abrir archivos específicos y navegar a elementos de código específicos en el archivo actual.

Presiona &shortcut:ShowNavBar; para mostrar la barra de navegación. Puedes navegar entre archivos y carpetas usando las teclas de dirección.
