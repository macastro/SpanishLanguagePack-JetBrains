Ver documentación rápida al pasar el cursor

No es necesario invocar explícitamente la documentación rápida &shortcut:QuickJavaDoc;. La ventana emergente de documentación rápida se mostrará automáticamente cuando pases el cursor sobre el símbolo deseado.

Para activar esta función, marca la casilla Mostrar documentación rápida al pasar el cursor en la página Editor | Edición de código del diálogo Configuración (&shortcut:ShowSettings;).
