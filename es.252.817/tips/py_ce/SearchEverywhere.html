Donde sea buscá

Shift presioná dos veces y hace una búsqueda en todo el proyecto de archivos, acciones, símbolos, elementos de la interfaz de usuario, ramas de Git y comentarios.

Presioná Shift otra vez para buscar en elementos que no están incluidos en el proyecto.

Para limitar los resultados de la búsqueda, usá las teclas de atajo (por ejemplo, para clases, usá IrAClase; para archivos, IrAArchivo; para símbolos, IrSímbolo; para acciones, IrAAcciones).
