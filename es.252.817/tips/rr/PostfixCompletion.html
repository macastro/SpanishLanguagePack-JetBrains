Completado de código mediante sufijos

El completado de código mediante sufijos se aplica de derecha a izquierda para evitar que el cursor de texto salte hacia adelante durante la codificación. Escribe un punto . después del fragmento de código que deseas modificar y selecciona la opción deseada.

Para ver la lista de sugerencias con plantillas de sufijos, también puedes presionar &shortcut:InsertLiveTemplate;.
