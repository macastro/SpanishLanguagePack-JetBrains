Gestionar solicitudes de extracción

Gestiona las solicitudes de extracción entrantes de GitHub directamente desde RustRover: selecciona Git | GitHub | Ver solicitudes de extracción desde el menú principal. RustRover te permite asignar y fusionar solicitudes de extracción, ver la línea de tiempo y los comentarios en línea, comentar y revisar, y aceptar cambios sin salir del IDE.
