Selector de navegación

Utiliza el selector &shortcut:Switcher; para navegar entre archivos abiertos y ventanas de herramientas. Mantén presionada la tecla Control para mantener abierta la ventana del selector. Para navegar, usa Tab y Shift+Tab.

Presiona Delete o BackSpace para cerrar las pestañas del editor u ocultar las ventanas de herramientas.
