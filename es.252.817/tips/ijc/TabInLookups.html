Auto completado de código

Cuando uses el auto completado de código presiona &shortcut:EditorChooseLookupItemReplace; para aceptar el elemento seleccionado que está resaltado en la lista desplegable.

Esto sobrescribirá el resto del nombre a la derecha del cursor con el nombre seleccionado, al contrario de aceptar con &shortcut:EditorChooseLookupItem;. Esto es particularmente útil cuando estás reemplazando un nombre de método o variable con otro nombre de método o variable.
