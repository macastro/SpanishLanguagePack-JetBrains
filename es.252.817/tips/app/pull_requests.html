Manejo de confirmaciones pull

Gestiona las confirmaciones pull de GitHub directamente desde IntelliJIDEACommunity: desde el menú principal selecciona Git | GitHub | Ver confirmaciones de pull. IntelliJIDEACommunity te permite asignar y fusionar confirmaciones de pull, ver la línea de tiempo y los comentarios integrados, enviar comentarios y revisiones, y aceptar cambios sin salir del IDE.
