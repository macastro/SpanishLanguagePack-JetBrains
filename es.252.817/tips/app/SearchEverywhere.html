Búsqueda global

Presiona dos veces Shift para buscar archivos, acciones, símbolos, elementos del interfaz de usuario, ramas de Git y comentarios en tu proyecto. Presionar dos veces Shift otra vez, extenderá la búsqueda a elementos no relacionados con el proyecto.

Usa pestañas o atajos directos &shortcut:GotoClass; para clases, &shortcut:GotoFile; para archivos, &shortcut:GotoSymbol; para símbolos, y &shortcut:GotoAction; para acciones que acoten los resultados de tu búsqueda.
