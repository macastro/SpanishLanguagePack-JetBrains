Solución rápida: Eliminar el nombre del parámetro

Para parámetros no utilizados, GoLand resaltará cuando tanto el nombre como el tipo del parámetro se puedan eliminar. Si el tipo se utiliza para implementar la especificación del método, resaltará solo el nombre.

En el último caso, al hacer clic en el nombre y presionar &shortcut:ShowIntentionActions;, podrás seleccionar una de las soluciones rápidas: "Eliminar nombre de parámetro" o "Cambiar nombre a _". Se proporciona una acción similar para las funciones que se utilizan como tipo de función.
