Ejecutar 'gofmt' después de la operación de reformateo

Es posible ejecutar gofmt cada vez que se reformatea el código (por ejemplo, al presionar &shortcut:ReformatCode;). El primer paso lo realiza el formateador incorporado de GoLand, seguido por gofmt.

Para habilitar esta función, presiona &shortcut:ShowSettings; para abrir la configuración, luego navega a Editor | Estilo de código | Go. Haz clic en la pestaña Otros y marca la casilla Al ejecutar la acción 'Reformatear código'.
