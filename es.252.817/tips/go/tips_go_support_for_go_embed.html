Archivos y directorios embebidos

A partir de Go 1.16, puedes embeber archivos y directorios en el binario de Go durante la compilación.

La operación de embebido funciona con string, []byte y embed.FS. Puedes declarar estas variables en el nivel superior del paquete, lo que significa que no pueden estar contenidas dentro del cuerpo de una función.

La refactorización Renombrar &shortcut:RenameElement;, las operaciones de Buscar usos, las inspecciones, el autocompletado de código y la búsqueda de usos &shortcut:FindUsages; están disponibles para las directivas embed.
