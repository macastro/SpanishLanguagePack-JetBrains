Buscar en todas partes

Presione Shift 2 veces para buscar archivos, acciones, símbolos, elementos de IU, ramas de Git y comentarios en todo el proyecto. Vuelva a presionar Shift 2 veces para buscar también elementos que no están incluidos en el proyecto.

Use las pestañas o directamente los atajos de teclado necesarios para acotar los resultados de la búsqueda (por ejemplo, &shortcut:GotoClass; para clases, &shortcut:GotoFile; para archivos, &shortcut:GotoSymbol; para símbolos y &shortcut:GotoAction; para acciones).
