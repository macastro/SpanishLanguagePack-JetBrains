Sugerencias integradas en rutas URL

Haga clic en las sugerencias integradas sobre cada acción del controlador para abrir una lista de acciones disponibles para la URL:

  • Ir a declaración o uso: Navega a la declaración en el archivo routes.rb.

  • Generar solicitud en cliente HTTP: Crea una nueva solicitud en el cliente HTTP.

  • Mostrar todos los endpoints del módulo: Proporciona una vista resumida de la API del módulo en la ventana de herramientas "Endpoints".
