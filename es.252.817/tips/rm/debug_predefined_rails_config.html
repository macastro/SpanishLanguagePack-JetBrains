Depurar usando la configuración predefinida de Rails

Para proyectos Rails, RubyMine crea automáticamente una configuración de ejecución/depuración development con el entorno Rails correspondiente. Para comenzar a depurar tu aplicación usando la configuración de ejecución/depuración development, presiona Control dos veces para invocar la ventana emergente Run Anything. Luego, comienza a escribir development, selecciónalo de la lista, mantén presionada la tecla Shift (el título del diálogo cambiará a Debug) y presiona &shortcut:EditorEnter;.
