Compartir configuraciones de ejecución/depuración

Puede almacenar las configuraciones de ejecución/depuración como archivos XML y compartirlas con los miembros de su equipo mediante un sistema de control de versiones. Active la opción Guardar como archivo de proyecto al editar la configuración de ejecución/depuración deseada y especifique la ubicación donde se guardará el archivo de configuración.
