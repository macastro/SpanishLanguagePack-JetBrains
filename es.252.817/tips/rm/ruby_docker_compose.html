Docker Compose como intérprete remoto

Para configurar Docker Compose como intérprete remoto, abra el cuadro de diálogo Configuración &shortcut:ShowSettings;, vaya a la página Lenguajes y Frameworks | Ruby SDK y Gems. Haga clic en y seleccione Nuevo remoto. En el cuadro de diálogo que aparece, seleccione Docker Compose y especifique la configuración requerida.
