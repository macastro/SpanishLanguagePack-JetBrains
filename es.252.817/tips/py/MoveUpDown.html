Mover la declaración hacia arriba / abajo

La acción "Código | Mover la declaración hacia arriba / abajo" es útil para reorganizar la línea de código, como acercar la declaración de variable a donde se utiliza la variable.

Selecciona el fragmento de código y presiona &shortcut:MoveStatementUp; o &shortcut:MoveStatementDown;. Si no se seleccionó nada en el editor, se moverá la línea en la que se encuentra el cursor.
