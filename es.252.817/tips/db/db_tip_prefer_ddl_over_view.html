DDL de objetos de base de datos, datos y ubicación en el Database Explorer

Presiona &shortcut:GotoDeclaration; para abrir el DDL del objeto seleccionado. Por ejemplo, si presionas este atajo sobre el nombre de una tabla en una sentencia SELECT, verás el DDL de esa tabla (sentencia CREATE TABLE).

Presiona &shortcut:EditSource; para abrir los datos del objeto. Por ejemplo, si presionas este atajo sobre el nombre de una tabla en una sentencia SELECT, verás el contenido de esa tabla.

Presiona Alt+Shift+B para navegar hasta el objeto en la ventana de herramientas Database Explorer.
