Visualizar varios archivos en paralelo

El editor permite visualizar varios archivos en paralelo. Para ello, haz clic con el botón secundario en la pestaña del editor de destino y selecciona la forma de dividir el editor (en "Dividir a la derecha" o "Dividir hacia abajo").

También puedes arrastrar la pestaña a cualquier área del editor para activar el modo de pantalla dividida.
