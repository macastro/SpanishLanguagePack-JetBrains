Consolas de base de datos, archivos temporales y búferes temporales

Una consola de base de datos es un archivo SQL vinculado a un origen de datos. Se crea automáticamente al crear un origen de datos. Para abrir la consola, haz clic derecho en el origen de datos y selecciona Navegar | Ir a la consola de consultas | Abrir consola predeterminada. Para crear una nueva consola, haz clic derecho en el origen de datos y selecciona Nuevo | Consola de consultas.

Los archivos temporales son archivos SQL no vinculados a ningún origen de datos. Son útiles para ideas de código fuera del contexto del proyecto, notas temporales o borradores. Para crear un archivo temporal, navega a Archivo | Nuevo | Archivo temporal (o presiona &shortcut:NewScratchFile;).

Los búferes temporales son archivos de texto simples sin ninguna función de asistencia de codificación. Los búferes temporales son útiles para listas de tareas sencillas y notas personales. No hay una opción de menú específica para crear nuevos búferes temporales, pero puedes usar la ventana emergente Buscar acción (&shortcut:GotoAction;) y ejecutar la acción Nuevo búfer temporal.
