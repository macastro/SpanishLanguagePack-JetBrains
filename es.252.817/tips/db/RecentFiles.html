Visualizar archivos usados recientemente

Si cierras por error una pestaña como la Consola de consulta o un script SQL, pulsa &shortcut:RecentFiles;. Se mostrará el cuadro de diálogo "Archivos recientes" con una lista de los archivos editados recientemente.

Pulsa &shortcut:RecentFiles; (Ver | Archivos usados recientemente) para ver una lista de los archivos abiertos recientemente.
