Evaluar una expresión

Puedes ejecutar expresiones o reasignar valores a variables mientras tu script se encuentra en pausa mediante un punto de ruptura. La funcionalidad Evaluar expresión te ayuda a obtener detalles adicionales acerca del estado del programa o para poner a prueba diferentes escenarios en tiempo de ejecución.

  • Si la expresión se encuentra en el código, presiona Alt (en Windows y Linux) o Alt (en macOS) y haz clic sobre la expresión. Si deseas evaluar un fragmento de código específico, selecciónalo antes de hacer clic.

  • Para evaluar una expresión arbitraria, selecciona Ejecutar | Acciones de depuración | Evaluar expresión. Se abrirá un cuadro de diálogo aparte que admite varias construcciones, como bucles, asignaciones, lambdas, entre otras.
