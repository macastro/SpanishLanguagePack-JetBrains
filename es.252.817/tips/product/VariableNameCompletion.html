Nombres de variables completos

La ayuda de código puede sugerirte el nombre de una variable al declararla. Por ejemplo, empieza a teclear private FileOutputStream. El IDE te mostrará una lista con sugerencias.

Puedes personalizar los prefijos para los nombres de las variables locales, los parámetros, las instancias y los campos estáticos en la pestaña Settings (&shortcut:ShowSettings;) under Code Style.
