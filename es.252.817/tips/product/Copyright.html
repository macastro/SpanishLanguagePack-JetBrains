Agregar avisos de copyright

En IntelliJIDEA, puedes configurar distintos avisos de copyright para distintos conjuntos de archivos. Por comodidad, puedes usar variables. Por ejemplo, usa $today.year para mantener el año actualizado.

En el cuadro de diálogo Configuración (&shortcut:ShowSettings;), selecciona Editor | Copyright | Perfiles de copyright, crea un perfil nuevo y agrega el texto para tu aviso de copyright usando variables.
