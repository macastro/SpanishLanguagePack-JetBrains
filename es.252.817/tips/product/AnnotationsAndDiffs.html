Visualizar anotaciones de blame de Git

Las anotaciones de Git muestran información detallada sobre el origen de cada línea de código (haz clic con el botón derecho en el gutter y selecciona Annotate with Git Blame).

Haz clic con el botón derecho en una anotación y selecciona Show Diff para revisar las diferencias entre la versión actual y la versión anterior del archivo.
