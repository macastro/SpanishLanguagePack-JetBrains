Compleción de código Postfix

La finalización de código Postfix se aplica de derecha a izquierda, evitando que el cursor se mueva hacia atrás mientras se codifica. Escriba un punto . después del fragmento de código que desee modificar y elija la opción deseada.

También puede presionar &shortcut:InsertLiveTemplate; para mostrar una lista de sugerencias que incluyen plantillas de Postfix.
