Agregar aviso de copyright

En IntelliJ IDEA, puedes configurar diferentes avisos de copyright para diferentes conjuntos de archivos. Para mayor comodidad, puedes utilizar variables. Por ejemplo, usa $today.year para mantener el año actualizado.

En el cuadro de diálogo Settings (&shortcut:ShowSettings;), selecciona Editor | Copyright | Copyright Profiles, crea un nuevo perfil y agrega el texto del aviso de copyright usando variables.
