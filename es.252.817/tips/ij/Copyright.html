Agregar avisos de Copyright

IntelliJIDEA te permite configurar distintos avisos de copyright para cada conjunto de archivos. Puedes utilizar variables para hacerlos más útiles. Por ejemplo, puedes utilizar $today.year para mantener actualizado el año.

Ve al cuadro de diálogo de "Configuraciones" (&shortcut:ShowSettings;) y selecciona "Editor | Copyright | Perfil de Copyright" para crear nuevos perfiles o agregar texto de copyright con variables.
