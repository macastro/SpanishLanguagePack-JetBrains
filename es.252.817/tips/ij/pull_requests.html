Gestión de peticiones de incorporación de cambios

Puedes gestionar las peticiones de incorporación de cambios de GitHub directamente desde IntelliJIDEA. Para ello, selecciona "Git | GitHub | Mostrar las peticiones de incorporación de cambios" en el menú principal. Con IntelliJIDEA, puedes asignar y fusionar peticiones de incorporación de cambios, ver el cronograma y los comentarios en línea, enviar comentarios y revisiones, y aprobar cambios sin salir del IDE.
