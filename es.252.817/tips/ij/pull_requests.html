Gestionar solicitudes de extracción

Gestiona las solicitudes de extracción entrantes de GitHub directamente desde IntelliJ IDEA: selecciona Git | GitHub | Ver solicitudes de extracción desde el menú principal. IntelliJ IDEA te permite asignar y fusionar solicitudes de extracción, ver la línea de tiempo y los comentarios en línea, comentarios de commit y verificaciones, y aceptar cambios sin salir del IDE.
