Completado de nombre de variable

Al declarar una variable puedes utilizar el completado de código para que te proponga nombres de variable. Por ejemplo, comienza a escribir private FileOutputStream y el IDE te mostrará una lista de candidatos.

Puedes personalizar los prefijos para los nombres de variables locales, parámetros, instancias y campos estáticos en el cuadro de diálogo Settings de Code Style (&shortcut:ShowSettings;).
