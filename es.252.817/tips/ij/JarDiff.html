Comparación de archivos

Para comparar dos archivos comprimidos .jar, .zip o .phar o los archivos contenidos en ellos, selecciona el archivo en la ventana de herramientas 'Proyecto' y pulsa &shortcut:CompareTwoFiles;.

La función 'Comparar archivos' funciona conjuntamente con un descompilador de Java Bytecode. Con esta función, puedes ver qué cambios exactos se hicieron entre dos versiones distintas de una biblioteca.
