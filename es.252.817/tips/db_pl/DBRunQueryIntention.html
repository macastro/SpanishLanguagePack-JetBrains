Manipulación de archivos SQL

Para abrir un archivo SQL en el IDE, haz doble clic. Para ejecutar una consulta de este archivo, invoca la acción de intención (Option+Intro en macOS, Alt+Intro en Windows y Linux) y selecciona "Ejecutar consulta en la consola". Selecciona una consola existente o crea una nueva en la lista "Sesiones".

Ten en cuenta que al abrir una nueva consola de consulta se genera una nueva conexión a la fuente de datos.
