Barra de navegación

La barra de navegación es un reemplazo para la herramienta de ventana Proyecto. Puede usarla para inspeccionar la estructura del proyecto, abrir archivos específicos o navegar a elementos de código específicos dentro del archivo actual.

Puede mostrar la barra de navegación presionando &shortcut:ShowNavBar; y usar las flechas de dirección para moverse por los archivos y carpetas.
