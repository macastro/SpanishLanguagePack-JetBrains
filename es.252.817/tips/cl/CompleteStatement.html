Creación de construcciones de código con autocompletado

Utilice la función de autocompletado de sentencias para crear construcciones de código. Comience introduciendo una declaración de método, una llamada de método o sentencias como if, for, do-while, try-catch, switch-case, return etc. Pulse &shortcut:EditorCompleteStatement; para completar la sentencia, esto produce una construcción sintácticamente correcta.
