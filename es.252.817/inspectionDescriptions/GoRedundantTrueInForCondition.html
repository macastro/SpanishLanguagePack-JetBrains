Reporta el uso redundante del literal true en la condición del bucle for.

En Go, el bucle for tiene una forma con una condición implícita. Esto significa que puedes crear un bucle infinito escribiendo for { }. Por lo tanto, en casos con una condición true (for true { }), el true se considera redundante y puede omitirse.

Para más información sobre los bucles for en Go, consulta For en go.dev.

Ejemplo:

for true {
  println("ping")
}
