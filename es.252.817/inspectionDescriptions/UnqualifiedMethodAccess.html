Informa sobre llamadas a métodos no static que no utilizan el calificador this en la misma instancia.

Ejemplo:


  class Foo {
    void bar() {}
    void foo() {
      bar();
    }
  }

Después de aplicar la corrección rápida:


  class Foo {
    void bar() {}
    void foo() {
      this.bar();
    }
  }
