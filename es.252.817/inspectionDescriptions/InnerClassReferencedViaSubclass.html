Informa sobre accesos a clases internas y anidadas donde la llamada está calificada por una subclase de la clase declarante en lugar de la propia clase declarante.

Java permite tales calificaciones, pero este tipo de acceso puede indicar una confusión sutil entre herencia y sobrescritura.

Ejemplo:


  class Super {
    static class Inner {}
  }
  class Sub extends Super {
    void test() {
      Sub.Inner s = new Sub.Inner(); // La clase 'Inner' está declarada en 'Super' pero se referencia a través de 'Sub'
    }
  }

Después de aplicar la corrección rápida:


  class Super {
    static class Inner {}
  }
  class Sub extends Super {
    void test() {
      Super.Inner s = new Super.Inner();
    }
  }
