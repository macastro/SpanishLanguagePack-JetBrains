Informa sobre paquetes y especificaciones de tipos de objetos que carecen de declaraciones de cuerpo.

Las especificaciones de paquetes y tipos de objetos que declaran rutinas y cursores deben tener declaraciones de cuerpo que implementen estas rutinas y cursores. La ausencia del cuerpo provocará errores en tiempo de ejecución cuando se llamen las rutinas o cursores en el código del programa.

Ejemplo (Oracle):

CREATE OR REPLACE PACKAGE ppp IS
    FUNCTION foo(a INT) RETURN INT;
END;
