Informa sobre expresiones de máscara de bits que evalúan a true o false de manera definitiva. La expresión tiene la forma (var & constant1) == constant2 o (var | constant1) == constant2, donde constant1 y constant2 son constantes de máscara de bits incompatibles.

Ejemplo:


// Máscaras incompatibles: debido a que el último byte en la máscara es cero,
// 0x1200 es posible, pero 0x1234 no lo es
if ((mask & 0xFF00) == 0x1234) {...}
