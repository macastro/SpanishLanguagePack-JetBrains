Informa sobre declaraciones case que no contienen un bloque else. El bloque else siempre debe estar presente para especificar el resultado por defecto cuando no se encuentra ninguna coincidencia.

Ejemplo:

# Mala práctica
case status
when :active
  perform_action
end
Se puede insertar un bloque else mediante una corrección rápida. Después de aplicar la corrección rápida, el resultado se muestra así: 
# Buena práctica
case status
when :active
  perform_action
else
  # Escriba el código aquí
end
