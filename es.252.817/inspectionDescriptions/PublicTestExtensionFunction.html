Reporta las funciones de extensión públicas para Driver/Finder/IdeaFrameUI en las clases de prueba del módulo intellij.driver.tests.

Las funciones de extensión públicas en el módulo intellij.driver.tests crean problemas de mantenibilidad porque:

  • son visibles para todas las pruebas en todos los contextos
  • pueden generar dependencias de prueba poco claras y un mantenimiento más difícil

Solución recomendada:

  • Mover la función a un companion object privado
  • Reubicarla en el page object adecuado:
    • Un page object genérico para funciones de aplicación general
    • Un page object específico para funciones que dependen del contexto
