Informa de clases que hacen referencia a sus subclases en inicializadores estáticos o campos estáticos.

Estas referencias pueden potencialmente causar bloqueos de nivel JVM cuando un hilo en un entorno multihilo intenta cargar la superclase mientras otro hilo intenta cargar la subclase simultáneamente.

Ejemplo:


  class Padre {
    static final Niño campo = new Niño();
  }
  class Niño se extiende Padre { }
