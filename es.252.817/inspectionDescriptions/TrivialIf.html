La declaración if se puede simplificar a una única asignación, declaración return o assert.

Ejemplo:


  if (foo()) {
    return true;
  } else {
    return false;
  }

Después de aplicar la corrección rápida:


  return foo();

Configuración de la inspección:

Para ocultar advertencias en declaraciones if encadenadas, utilice la opción Ignorar declaraciones 'if' encadenadas.

Por ejemplo, en el siguiente código, la advertencia se ocultará pero la corrección rápida seguirá disponible:


  if (condition1) return true;
  if (condition2) return false;
  return true;

Tenga en cuenta que si la condición tiene efectos secundarios, reemplazar if (isTrue()) assert false; por assert isTrue(); podría cambiar la semántica del programa cuando las aserciones están deshabilitadas. Para ocultar advertencias en declaraciones if que solo contienen una declaración assert en su cuerpo, utilice la opción Ignorar declaraciones 'if' con 'assert' simple.
