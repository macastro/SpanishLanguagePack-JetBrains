Informa de las variables de instancia que se pueden convertir en static de forma segura. Un campo puede ser estático si se declara final y se inicializa con una constante.

Ejemplo:


  public final String str = "sample";

La inspección no informa de los campos finales que se pueden escribir implícitamente. Utiliza el botón "Anotaciones" para modificar la lista de anotaciones que suponen la escritura implícita en el campo.
