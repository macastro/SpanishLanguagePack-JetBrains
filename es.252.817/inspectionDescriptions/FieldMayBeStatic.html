Informa sobre variables de instancia que pueden declararse como static de manera segura. Si un campo se declara como final y se inicializa con una constante, puede ser un campo static.

Ejemplo:


  public final String str = "sample";

Esta inspección no informa sobre campos final que pueden tener escrituras implícitas. Utilice el botón "Anotaciones" para modificar la lista de anotaciones que se asumen para escrituras implícitas de campos.
