Informa los valores en punto flotante que se están comparando usando el operador == o !=.

Los valores en punto flotante son inherentemente inexactos, y compararlos para que tengan una igualdad exacta raramente es la semántica deseada.

Esta inspección ignora las comparaciones con los literales cero e infinito.

Ejemplo: 


  void m(double d1, double d2) {
    if (d1 == d2) {}
  }
