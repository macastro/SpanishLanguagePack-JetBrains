Informa sobre campos no final en tipos enum. Los campos no final introducen un estado mutable global, lo cual generalmente se considera indeseable.

Ejemplo:


  enum Enum {
    FIRST("first"),
    SECOND("second");
    public String str;
    Enum(String str) {
        this.str = str;
    }
  }

Después de aplicar la corrección rápida:


  enum Enum {
    FIRST("first"),
    SECOND("second");
    public final String str;
    Enum(String str) {
        this.str = str;
    }
  }
La opción "Ignorar campos que no pueden ser 'final'" solo mostrará advertencias para campos que pueden convertirse en final mediante una corrección rápida.
