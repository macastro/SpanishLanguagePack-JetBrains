Informa de los campos no finales en tipos de enumeraciones. Los campos no finales introducen un estado mutable global, lo que generalmente no se considera deseable.

Ejemplo:


  enum Enum {
    FIRST("first"),
    SECOND("second");

    public String str;

    Enum(String str) {
        this.str = str;
    }
  }

Después de aplicar la corrección rápida:


  enum Enum {
    FIRST("first"),
    SECOND("second");

    public final String str;

    Enum(String str) {
        this.str = str;
    }
  }
Usa la opción "Ignorar campos que no se pueden hacer 'final'" para advertir solo sobre los campos que se pueden hacer finales mediante la corrección rápida.
