Informa sobre parámetros catch que no se utilizan en su bloque correspondiente. Esta inspección no informa sobre parámetros catch nombrados como "ignore" o "ignored".

Ejemplo:


  try {
      def arr = new int[3]
      arr[5] = 5
  } catch(Exception ex) {
      println('Catching the exception')
  }

El parámetro ex nunca se utiliza en el bloque catch.

Después de aplicar la corrección rápida:


  try {
      def arr = new int[3]
      arr[5] = 5
  } catch(Exception ignored) {
      println('Catching the exception')
  }
