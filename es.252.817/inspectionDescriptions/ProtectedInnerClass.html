Reporta clases anidadas `protected`.

Ejemplo:


  public class Outer {
    protected static class Nested {} // advertencia
    protected class Inner {}         // advertencia
    protected enum Mode {}           // advertencia depende de la configuración
    protected interface I {}         // advertencia depende de la configuración
  }

Configura la inspección:

  • Usa la opción Ignorar enumeraciones internas 'protected' para ignorar las enumeraciones internas `protected`.
  • Usa la opción Ignorar interfaces internas 'protected' para ignorar las interfaces internas `protected`.
