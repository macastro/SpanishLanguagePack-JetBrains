Reporta clases anidadas con modificador protected.

Ejemplo:


  public class Outer {
    protected static class Nested {} // advertencia
    protected class Inner {}         // advertencia
    protected enum Mode {}           // advertencia según configuración
    protected interface I {}         // advertencia según configuración
  }

Configuración de la inspección:

  • Utiliza la opción Ignorar enumeraciones internas 'protected' para ignorar enumeraciones internas con modificador protected.
  • Utiliza la opción Ignorar interfaces internas 'protected' para ignorar interfaces internas con modificador protected.
