Informa de los intentos de crear columnas NOT NULL sin valores DEFAULT.

Ejemplo (Microsoft SQL Server):

CREATE TABLE foo (a INT, b  INT)

ALTER TABLE foo ADD c INT NOT NULL;

De forma predeterminada, una columna contiene valores NULL. En el ejemplo, utilizamos la restricción NOT NULL que obliga a que una columna no acepte valores NULL. Si prohibimos utilizar valores NULL, debemos establecer el valor DEFAULT que SQL puede utilizar cuando creamos un nuevo registro. 

ALTER TABLE foo ADD c INT NOT NULL DEFAULT 42;

Puedes añadir rápidamente el valor DEFAULT utilizando el arreglo rápido Añadir valor DEFAULT.
