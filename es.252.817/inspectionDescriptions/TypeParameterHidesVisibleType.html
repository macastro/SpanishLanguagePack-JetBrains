Informa los parámetros de tipo que tienen los mismos nombres que los tipos visibles en el ámbito actual. Estos nombres de parámetros pueden ser confusos.

Ejemplo:


  abstract class MyList<T> extends AbstractList<T> {
    private List<T> elements;
    // el parámetro de tipo 'T' oculta el parámetro de tipo 'T'
    public <T> T[] toArray(T[] array) {
        return elements.toArray(array);
    }
}
