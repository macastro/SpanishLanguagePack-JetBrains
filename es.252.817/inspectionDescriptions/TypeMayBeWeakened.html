El informe puede cambiar a tipos de variables y tipos de retorno de métodos más abstractos (más débiles). Esto hace que el código sea más abstracto y, por lo tanto, más reutilizable.

Ejemplo:


  // 形参类型可以削弱为 java.util.List
  void processList(ArrayList<String> list) {
    if (list.isEmpty()) return;
    System.out.println("Processing");
    for (String s : list) {
      System.out.println("String: " + s);
    }
  }

Active la casilla Debilitar solo a interfaces para informar problemas solo cuando el tipo pueda debilitarse a un tipo de interfaz.

Active la casilla No sugerir debilitar variables declaradas como 'var' para evitar informes sobre variables locales declaradas con la palabra clave 'var' (Java 10+)

Clases de parada están destinadas a prevenir el debilitamiento de clases por debajo del nivel de la clase de parada, incluso cuando sea posible. En algunos casos, esto puede mejorar la legibilidad.
