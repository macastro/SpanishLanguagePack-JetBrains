Informa sobre la falta de archivos de descripción HTML, archivos before.template o after.template para la intención. Estos se muestran en Configuración | Editor | Intenciones.

La corrección rápida Crear archivo de descripción creará una plantilla de archivo de descripción HTML.

Si no necesitas los archivos before.template o after.template, agrega <skipBeforeAfter>true</skipBeforeAfter> en la sección de registro de intenciones en plugin.xml.

Para más información, consulta la documentación del SDK de plugins para la plataforma IntelliJ.
