Informa sobre el uso de la función withIndex() en lugar de una variable de índice en bucles for para iterar colecciones.

El código se puede limpiar usando la corrección rápida "Eliminar índice del bucle 'for'".

Ejemplo:


  fun foo(bar: List<String>) {
     for ((index : Int, value: String) in bar.withIndex()) { // <== 'index' no utilizado
         println(value)
     }
  }

Después de aplicar la corrección rápida:


  fun foo(bar: List<String>) {
      for (value: String in bar) { // <== '.withIndex()' e 'index' han sido eliminados
          println(value)
      }
  }
