Informa sobre declaraciones de variables de Collection que utilizan clases de colección como tipo en lugar de sus interfaces correspondientes. No se emite ninguna advertencia si debilitar el tipo de variable causaría errores de compilación.

Ejemplo:


  // Advertencia: se usa la clase concreta ArrayList.
  int getTotalLength(ArrayList<String> list) {
    return list.stream().mapToInt(String::length).sum();
  }
  // Sin advertencia, ya que el método trimToSize()
  // no está disponible en la interfaz List
  void addData(ArrayList<String> data) {
    data.add("Hello");
    data.add("World");
    data.trimToSize();
  }
Se sugiere utilizar la interfaz de colección correspondiente (por ejemplo, Collection, Set o List) mediante la corrección rápida.
