Informa sobre declaraciones o expresiones switch que contienen código idéntico en diferentes ramas y sugiere combinar las ramas duplicadas.

Ejemplo:


  switch (n) {
    case 1:
      System.out.println(n);
      break;
    case 2:
      System.out.println(n);
      break;
    default:
      System.out.println("default");
  }

Después de aplicar la corrección rápida:


  switch (n) {
    case 1:
    case 2:
      System.out.println(n);
      break;
    default:
      System.out.println("default");
  }
