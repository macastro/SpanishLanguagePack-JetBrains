Informa sobre comentarios Javadoc que no pertenecen a ninguna clase, método o campo. La herramienta Javadoc ignora los comentarios Javadoc flotantes y no los incluye en su documentación HTML generada.

Ejemplo:


  class A {
    /**
     * Dangling comment
     */
    /**
     * Method javadoc
     */
     public void m(){}
  }

Estos comentarios se pueden eliminar completamente o convertir en comentarios de bloque mediante una corrección rápida. Después de aplicar la corrección rápida:


  class A {
    /*
      Dangling comment
    */
    /**
     * Method javadoc
     */
    public void m(){}
  }

Use la opción Ignorar comentarios de encabezado con formato JavaDoc para ignorar los comentarios al inicio de los archivos Java. Estos suelen ser mensajes de copyright.
