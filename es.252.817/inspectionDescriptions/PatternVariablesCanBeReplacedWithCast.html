Informa de instanceof con patrones y sugiere convertirlos a instanceof ordinarios con conversiones.

Esta inspección permite mover instanceof con patrones a una base de código que utiliza una versión anterior de Java aplicando la corrección rápida.

Ten en cuenta que el resultado puede no ser completamente equivalente al instanceof original con patrones cuando se utiliza una expresión compleja antes de instanceof. En este caso, esta expresión se volverá a evaluar.

Ejemplo:


  if (object instanceof String txt && txt.length() == 1) {
      System.out.println(txt);
  } else {
      return;
  }
  System.out.println(txt);

Después de aplicar la corrección rápida:


  if (object instanceof String && ((String) object).length() ==1) {
      String txt = (String) object;
      System.out.println(txt);
  } else {
      return;
  }
  String txt = (String) object;
  System.out.println(txt);
