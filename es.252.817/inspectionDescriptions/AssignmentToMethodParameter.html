Informa sobre asignaciones o modificaciones a los parámetros de método.

Aunque a veces es intencional, esta estructura puede resultar confusa y está prohibida en algunos proyectos Java.

La corrección rápida añadirá una declaración de nueva variable.

Ejemplo:


  void printTrimmed(String s) {
    s = s.trim();
    System.out.println(s);
  }

Después de aplicar la corrección rápida:


  void printTrimmed(String s) {
    String trimmed = s.trim();
    System.out.println(trimmed);
  }

Use la opción Ignorar si la asignación es una transformación del parámetro original para ignorar las asignaciones que modifican el valor del parámetro basándose en su valor previo.
