Informa sobre métodos public en clases que no están expuestos en interfaces.

Exponer todos los métodos public a través de interfaces es importante para mantener un acoplamiento débil y puede ser necesario para ciertos estilos de programación basados en componentes.

Ejemplo:


interface Person {
  String getName();
}
class PersonImpl implements Person {
  private String name;
  // Normal: método expuesto en la interfaz
  @Override
  public String getName() {
    return name;
  }
  // Advertencia: método es public
  // pero no está expuesto en la interfaz
  public void setName() {
    this.name = name;
  }
}

Use la lista Ignorar si está anotado con para especificar anotaciones especiales. La inspección ignorará los métodos anotados con cualquiera de estas anotaciones.

Use la opción Ignorar si la clase contenedora no implementa interfaces no bibliotecarias para ignorar métodos en clases que no implementan ninguna interfaz del proyecto.
