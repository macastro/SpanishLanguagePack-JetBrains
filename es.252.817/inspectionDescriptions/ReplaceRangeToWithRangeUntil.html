Informa sobre llamadas al operador rangeTo o .., en lugar de llamadas a ...

El código puede simplificarse usando la función correspondiente.

Esta corrección rápida reemplazará las llamadas a rangeTo o .. por ...

Ejemplo:


  fun foo(a: Int) {
      for (i in 0..a - 1) {
      }
  }

Después de aplicar la corrección rápida:


  fun foo(a: Int) {
      for (i in 0..<a) {
      }
  }
