Informa las directivas requires redundantes en archivos module-info.java del sistema de módulos de la plataforma Java. Una directiva requires es redundante cuando un módulo A requiere un módulo B, pero el código del módulo A no importa ningún paquete o clases de B. Además, todos los módulos tienen una dependencia implícitamente declarada en el módulo java.base, por lo tanto, una directiva requires java.base; siempre es redundante.

La solución rápida elimina la directiva requires redundante. Si la dependencia eliminada volvió a exportar módulos que en realidad se usan, la solución agrega directivas requires para estos módulos.

Esta inspección solo informa si el nivel de lenguaje del proyecto o módulo es 9 o superior.
