Reporta las siguientes características no resueltas en las anotaciones relacionadas con EntityGraph:
  • Valores en @NamedAttributeNode
  • Subgráficos nombrados en @NamedAttributeNode
  • Nombres en @NamedEntityGraph

Ejemplo:



  @Entity
  @NamedEntityGraph(name = "entity-graph", ...)
  public class JavaEntity {
    public void foo() {
      Persistence.createEntityManagerFactory("")
        .createEntityManager()
        .getEntityGraph("unknown-entity-graph"); // Referencia no resuelta
    }
  }
