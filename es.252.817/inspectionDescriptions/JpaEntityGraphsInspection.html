Informa de los siguientes atributos no resueltos en las anotaciones relacionadas con EntityGraph:
  • Valor en @NamedAttributeNode
  • Subgrafo de nombre en @NamedAttributeNode
  • Nombre en @NamedEntityGraph

Ejemplo:



  @Entity
  @NamedEntityGraph(name = "entity-graph", ...)
  public class JavaEntity {
    public void foo() {
      Persistence.createEntityManagerFactory("")
        .createEntityManager()
        .getEntityGraph("unknown-entity-graph"); // referencia no resuelta
    }
  }
