Informa sobre llamadas a métodos de aridad variable (arity) donde el argumento en la posición del parámetro vararg es null o un array de un subtipo del parámetro vararg. Este tipo de argumentos puede resultar confuso, ya que no queda claro si se requiere una llamada vararg o no vararg.

Ejemplo:


  String[] ss = new String[]{"foo", "bar"};
  System.out.printf("%s", ss);

En este ejemplo, no se imprimirá todo el array, sino solo el primer elemento del array.
