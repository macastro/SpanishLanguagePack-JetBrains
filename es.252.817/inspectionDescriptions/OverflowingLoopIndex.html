Informa sobre ciclos que no pueden completarse sin un sobreflujo de índices o ciclos que no se ejecutan en absoluto. Habitualmente, esto sucede por un error en la operación de actualización.

Ejemplo:


  void foo(int s) {
    para (int i = s; i > 12; i++) { // Aquí debería ir i--
      System.out.println(i);
    }
  }
