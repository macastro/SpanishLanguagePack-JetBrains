Informa de expresiones del tipo char usadas en expresiones de suma o resta.

Este código no es necesariamente un problema, pero puede provocar errores (por ejemplo, si se espera una cadena).

Ejemplo: int a = 'a' + 42;

Después de aplicar la corrección rápida: int a = (int) 'a' + 42;

Para el contexto String: 

int i1 = 1;
int i2 = 2;
System.out.println(i2 + '-' + i1 + " = " + (i2 - i1));

Después de aplicar la corrección rápida:

System.out.println(i2 + "-" + i1 + " = " + (i2 - i1));
