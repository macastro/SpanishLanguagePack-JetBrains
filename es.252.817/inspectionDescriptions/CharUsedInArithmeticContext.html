Informa sobre expresiones de tipo char utilizadas en expresiones de suma o resta.

Este código no es necesariamente un problema, pero puede conducir a errores (por ejemplo, cuando se requiere una cadena de texto).

Ejemplo: int a = 'a' + 42;

Después de aplicar la corrección rápida: int a = (int) 'a' + 42;

Para el contexto de String: 

int i1 = 1;
int i2 = 2;
System.out.println(i2 + '-' + i1 + " = " + (i2 - i1));

Después de aplicar la corrección rápida:

System.out.println(i2 + "-" + i1 + " = " + (i2 - i1));
