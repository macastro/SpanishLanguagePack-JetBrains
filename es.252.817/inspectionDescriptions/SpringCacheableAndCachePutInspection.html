Informe los @CachePut y @Cacheable anotados declarados sobre el mismo método.

Mientras que @Cacheable hace que el método omita la ejecución usando el caché, @CachePut fuerza la ejecución para actualizar el caché. Esto lleva a un comportamiento inesperado y debería evitarse, excepto en casos muy específicos cuando las condiciones en las anotaciones las excluyen entre sí. Note también que tales condiciones no deberían depender del objeto resultado (la variable #result) ya que estas se validan de antemano para confirmar la exclusión.
