Informa sobre el desempaquetado (unboxing) innecesario, es decir, el desenvolvimiento explícito de valores primitivos que ya están envueltos.

Para Java 5 y versiones posteriores, el desempaquetado es innecesario y puede eliminarse de forma segura.

Ejemplo:

  • Integer i = Integer.valueOf(42).intValue(); → Integer i = Integer.valueOf(42);
  • int k = Integer.valueOf(42).intValue(); → int k = Integer.valueOf(42);

    (Solo se informa cuando la opción Informar solo sobre expresiones de desempaquetado verdaderamente redundantes no está seleccionada)

Con la opción Informar solo sobre expresiones de desempaquetado verdaderamente redundantes, solo se informan los casos de desempaquetado verdaderamente redundantes donde el valor desempaquetado se empaqueta inmediatamente de forma implícita o explícita. En este caso, se puede eliminar todo el paso de desempaquetado-empaquetado. La inspección no informa sobre casos simples de desempaquetado explícito.

Esta inspección solo informa cuando el nivel de lenguaje del proyecto o módulo es 5 o superior.
