Reporta el desempaquetado, que es el desenvuelto explícito de valores primitivos envueltos.

El desempaquetado es innecesario a partir de Java 5 en adelante y puede eliminarse de forma segura.

Ejemplos:

  • Integer i = Integer.valueOf(42).intValue(); → Integer i = Integer.valueOf(42);
  • int k = Integer.valueOf(42).intValue(); → int k = Integer.valueOf(42);

    (informa solo cuando la opción Informar solo expresiones realmente desempaquetadas superfluamente no está marcada)

Use la opción Informar solo expresiones realmente desempaquetadas superfluamente para informar solo el desempaquetado realmente superfluo, donde un valor desempaquetado se empaqueta inmediatamente de forma implícita o explícita. En este caso, se puede eliminar todo el paso desempaquetar-empaquetar. La inspección no informa desempaquetado explícito simple.

Esta inspección solo informa si el nivel de idioma del proyecto o módulo es 5 o superior.
