La expresión return puede ser reemplazada por la sintaxis de cuerpo de expresión (una línea de código o when).

Las recomendaciones para la sintaxis de cuerpo de expresión se proporcionan en la guía de estilo.

La corrección rápida Convertir a cuerpo de expresión está disponible para corregir el código automáticamente.

Ejemplo:


  fun sign(x: Int): Int {
      return when { // <== puede simplificarse
          x < 0 -> -1
          x > 0 -> 1
          else -> 0
      }
  }

Después de aplicar la corrección rápida:


  fun sign(x: Int): Int = when {
      x < 0 -> -1
      x > 0 -> 1
      else -> 0
  }
