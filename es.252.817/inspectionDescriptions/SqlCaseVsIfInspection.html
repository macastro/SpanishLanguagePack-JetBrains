Informa sobre situaciones donde CASE e IF son intercambiables.

Ejemplo (MySQL):

SELECT CASE
WHEN C1 IS NULL THEN 1
ELSE 0
END
FROM dual;

Puede acortar el código reemplazando la estructura CASE con IF. Para hacerlo, aplique la intención Reemplazar con llamada 'IF'. El código de ejemplo quedará así:

SELECT IF(C1 IS NULL, 1, 0)
FROM dual;

Para volver a convertir IF en CASE, haga clic en IF y aplique la intención Reemplazar con expresión CASE.
