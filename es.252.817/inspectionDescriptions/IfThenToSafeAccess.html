El informe se puede colapsar en una expresión if-then dentro de una expresión de acceso seguro (?.).

Ejemplo:


  fun bar(x: String) = ""
  fun foo(a: String?) {
     if (a != null) bar(a) else null
  }

La corrección rápida convertirá la expresión if-then en una expresión de acceso seguro (?.):


  fun bar(x: String) = ""
  fun foo(a: String?) {
     a?.let { bar(it) }
  }
