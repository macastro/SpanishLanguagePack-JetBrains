Informa sobre clases Serializable donde el campo serialPersistentFields no está declarado como private static final ObjectStreamField[].

Si el campo serialPersistentFields no se declara con estos modificadores, el comportamiento de serialización será el mismo que si el campo no se hubiera declarado en absoluto.

Ejemplo:


  class List implements Serializable {
    private List next;
    ObjectStreamField[] serialPersistentFields = {new ObjectStreamField("next", List.class)};
  }
