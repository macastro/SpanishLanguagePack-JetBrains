Informa las clases Serializable cuyo campo serialPersistentFields no esté declarado como private static final ObjectStreamField[].

Si un campo serialPersistentFields no se declara con esos modificadores, el comportamiento de serialización será como si el campo no estuviera declarado en absoluto.

Ejemplo:


  class List implements Serializable {
    private List next;

    ObjectStreamField[] serialPersistentFields = {new ObjectStreamField("next", List.class)};

  }
