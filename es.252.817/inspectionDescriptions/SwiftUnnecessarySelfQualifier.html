Reporta el uso de self como calificador cuando no es necesario en el closure de escape.

Se sugiere eliminar el self redundante.

Ejemplo: 


  struct URL {
      func loadAsync() {
          print(self.path)
          DispatchQueue.global().async {
              self.loadSync()
          }
      }
  }

Después de aplicar la corrección rápida: 


  struct URL {
      func loadAsync() {
          print(path)
          DispatchQueue.global().async {
              loadSync()
          }
      }
  }
