Informa sobre comprobaciones innecesarias de longitud de array después de una iteración. Cuando un array tiene longitud cero, la iteración se omite, por lo que no es necesario verificar explícitamente la longitud.

Ejemplo:


  void f(String[] array) {
    if (array.length != 0) { // Comprobación innecesaria
      for (String str : array) {
        System.out.println(str);
      }
    }
  }

Utilice la corrección rápida para desenvolver o eliminar la comprobación de longitud:


  void f(String[] array) {
    for (String str : array) {
      System.out.println(str);
    }
  }
