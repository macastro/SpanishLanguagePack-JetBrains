Reporta instrucciones inalcanzables dentro de rutinas SQL.

Ejemplo (Microsoft SQL Server):

CREATE FUNCTION foo() RETURNS INT AS
BEGIN
    THROW;
    RETURN 1;
END;

En Microsoft SQL Server, la instrucción THROW lanza una excepción y transfiere la ejecución al bloque CATCH de la construcción TRY...CATCH. Por lo tanto, la parte RETURN 1; nunca será ejecutada.
