Informa métodos no sincronizados que sobreescriben métodos sincronizados.

Ejemplo:


  clase abstracta Base {
      synchronized void foo() {
          // ...
      }
  }
  clase Derivada extends Base {
      @Override
      void foo() {
          super.foo()
            // ...
        }
    }

En este caso, el método no sincronizado foo() de la clase Bar sobreescribe un método sincronizado.
