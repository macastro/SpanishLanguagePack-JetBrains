Informa sobre variables sin tipo especificado

Cuando el código no puede resolver el tipo de una variable, se debe especificar manualmente.

Ejemplo:


  var int_val = 1
  val string_val = ""

Tras aplicar la corrección rápida:


  var int_val: int = 1
  val string_val: String = ""

O al usar la forma abreviada:


  var int_val := 1
  val string_val := ""
