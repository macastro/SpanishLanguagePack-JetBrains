Informa sobre clases de utilidad que no son final o abstract.

Las clases de utilidad declaran todos sus campos y métodos como static. Hacerlas final evita que sean heredadas accidentalmente.

Ejemplo:


  public class UtilityClass {
    public static void foo() {}
  }

Después de aplicar la corrección rápida:


  public final class UtilityClass {
    public static void foo() {}
  }
