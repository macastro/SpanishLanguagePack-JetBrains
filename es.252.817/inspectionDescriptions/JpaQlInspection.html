Reporta los siguientes errores en consultas Persistence QL:
  • Tipos de expresiones incompatibles
  • Parámetros incorrectos
  • Condiciones vacías o constantes
  • Símbolos no resueltos

Consultas de ejemplo:


SELECT OBJECT(e) FROM JavaEntity e WHERE TRUE // Advertencia: condición constante
SELECT OBJECT(e) FROM JavaEntity e WHERE e.id BETWEEN 0 AND '2' // Error: se esperaba un número pero se obtuvo una cadena
SELECT e from JavaEntity e where e.id=?1 and e.name=?3 // Error: solo hay 2 parámetros de consulta reales
