Reporta los problemas de firma para funciones integradas.

La inspección reportará un número incorrecto de argumentos, palabras clave no válidas, tipos de datos incorrectos y otros problemas.

Ejemplo (MySQL):

CREATE TABLE foo (a INT, b INT, c INT)

SELECT IFNULL() FROM foo; -- error
SELECT IFNULL(a) FROM foo; -- error
SELECT IFNULL(a, b) FROM foo; -- OK
SELECT IFNULL(a, b, c) FROM foo; -- error

En MySQL, la función IFNULL() acepta estrictamente dos argumentos. Entonces, solo la consulta SELECT IFNULL(a, b) FROM foo; es correcta.
