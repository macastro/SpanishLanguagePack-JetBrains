Informa sobre llamadas a equals() que comparan dos números java.math.BigDecimal. Esto suele ser un error, ya que dos números java.math.BigDecimal solo son iguales cuando tienen el mismo valor y escala.

Ejemplo:


  if (new BigDecimal("2.0").equals(
    new BigDecimal("2.00"))) {} // false

Después de aplicar la corrección rápida:


  if (new BigDecimal("2.0").compareTo(
    new BigDecimal("2.00")) == 0) {} // true
