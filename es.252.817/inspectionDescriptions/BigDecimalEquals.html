Se está llamando a .equals() para comparar dos números java.math.BigDecimal. Por lo general, esto es un error, dado que para que dos números java.math.BigDecimal sean iguales, deben coincidir tanto el valor como la escala.

Ejemplo:


  if (new BigDecimal("2.0").equals(
    new BigDecimal("2.00"))) {} // false

Después de aplicar el arreglo rápido:


  if (new BigDecimal("2.0").compareTo(
    new BigDecimal("2.00")) == 0) {} // true
