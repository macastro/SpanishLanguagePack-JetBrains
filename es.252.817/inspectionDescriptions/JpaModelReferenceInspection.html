Reportar símbolos no resueltos en las siguientes anotaciones JPA y métodos de API:
  • AttributeOverride
  • AssociationOverride
  • MapsId
  • MapKey
  • Métodos de Criteria API

Ejemplo:



  @Entity
  @AssociationOverrides({
    @AssociationOverride(name = "missingAttribute", joinColumns = { ... }), // Error: atributo no resuelto
    @AssociationOverride(name = "existingAttribute", joinColumns = { ... })
  })
  public class JavaEntity {
    @OneToOne
    AnotherEntity existingAttribute;
  }
