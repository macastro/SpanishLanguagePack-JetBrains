Reporta las propiedades lateinit var que sobrescriben otras propiedades lateinit var.

Una instancia de subclase tendrá dos campos para una única propiedad, y el de la superclase se mantendrá sin usar.

Ejemplo:


    open class BaseClass {
      open lateinit var name: String
    }

    class RealClass : BaseClass() {
      override lateinit var name: String
    }
