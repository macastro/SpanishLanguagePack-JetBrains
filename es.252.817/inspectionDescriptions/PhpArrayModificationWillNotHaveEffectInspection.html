Reporta el llamado a una función que devuelve un arreglo por valor, lo cual invalida la operación de asignación. Se debería usar retorno por referencia.

Por ejemplo:


<?php
class A {
    private $values = [];
    public function getValues() {
        return $this->values;
    }
}
$config = new A();
$config->getValues()['a'] = 'a';
