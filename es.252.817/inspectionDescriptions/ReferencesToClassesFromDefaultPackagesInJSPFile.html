Informa sobre cualquier referencia a clases en el paquete predeterminado dentro de archivos JSP.

Ejemplo:


  <%!
  Integer countSomeUsefulStuff() {
    ...
    new ClassFromDefaultPackage(...); // Error
    new com.example.GoodClass(...); // OK
  }
  %>
  <html>
    <head>
      <title>Hello, JSP!</title>
    </head>
  </html>

Nota: Estas referencias no se resolverán durante la compilación de JSP, ya que las clases compiladas de JSP generalmente se colocan en paquetes no predeterminados (por ejemplo, "org.apache.jsp" para Tomcat, "com.ibm._jsp" para WebSphere), y Java no permite importar desde el paquete predeterminado.
