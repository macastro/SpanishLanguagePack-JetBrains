Informa de cualquier referencia a clase del paquete por defecto en los archivos JSP.

Ejemplo:


  <%!
  Integer countSomeUsefulStuff() {
    ...
    new ClassFromDefaultPackage(...); // Error
    new com.example.GoodClass(...); //Correcto
  }
  %>
  <html>
    <head>
      <title>Hello, JSP!</title>
    </head>
  </html>

NOTA: Dichas referencias no se resolverán durante la compilación del JSP ya que una clase compilada para el JSP normalmente se colocará en un paquete no por defecto (por ejemplo, "org.apache.jsp" en caso de Tomcat, "com.ibm._jsp" en caso de WebSphere), pero la importación del paquete por defecto no está permitido en Java.
