Informa sobre llamadas a métodos de superclase desde clases anónimas, internas o locales cuando existe un método con la misma firma en el código circundante. En estos casos, aunque parezca que se está llamando al método del código circundante, en realidad se está llamando al método de la superclase.

Para clarificar la intención del código, se recomienda añadir el calificador explícito super a la llamada del método.

Ejemplo:


  class Parent {
    void ambiguous(){}
  }
  class Example {
    void ambiguous(){}
    class Inner extends Parent {
      void example(){
        ambiguous(); //advertencia
      }
    }
  }

Después de aplicar la corrección rápida:


  class Parent {
    void ambiguous(){}
  }
  class Example {
    void ambiguous(){}
    class Inner extends Parent {
      void example(){
        super.ambiguous();
      }
    }
  }
