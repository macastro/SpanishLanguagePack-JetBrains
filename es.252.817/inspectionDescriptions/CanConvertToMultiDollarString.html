Informa sobre los casos en los que resulta beneficioso convertir una cadena a una versión con un prefijo de interpolación de varios signos de dólar.

Un prefijo de interpolación puede hacer que una cadena sea más fácil de entender, ya que una cadena con prefijo puede usar literales de dólar sin escapes adicionales.

Ejemplo:


val jsonSchema: String = """
{
  "${'$'}id": "https://example.com/product.schema.json",
}"""
La corrección rápida convierte todos los caracteres de dólar escapados que se pueden reemplazar de forma segura en caracteres de dólar normales: 

val jsonSchema: String = $$"""
{
  "$id": "https://example.com/product.schema.json",
}"""
