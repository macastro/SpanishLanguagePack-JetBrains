Informa sobre la asignación de null a variables Optional o su retorno desde métodos que devuelven Optional.

Se recomienda usar Optional.empty() (o Optional.absent() para Guava) para representar valores nulos.

Ejemplo:


  Optional<Integer> foo(boolean flag) {
    return flag ? Optional.of(42) : null;
  }

Después de aplicar la corrección rápida:


  Optional<Integer> foo(boolean flag) {
    return flag ? Optional.of(42) : Optional.empty();
  }

Configuración de la inspección:

La opción Informar comparaciones de Optional con null también puede informar sobre comparaciones como optional == null. Aunque esto puede ser correcto en casos muy raros (por ejemplo, campos opcionales con inicialización diferida), las variables opcionales normalmente no deberían ser null y probablemente deberían usar optional.isPresent().
