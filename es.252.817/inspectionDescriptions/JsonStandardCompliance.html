Reporta las siguientes diferencias entre el archivo JSON y la especificación del lenguaje:
  • Comentarios de línea o bloque (configurable).
  • Múltiples valores de nivel superior (para archivos JSON Lines, configurable para otros archivos).
  • Comas finales en objetos o arrays (configurable).
  • Cadenas de texto entre comillas simples.
  • Claves de propiedades que no son cadenas entre comillas dobles.
  • Literales de punto flotante usando NaN o infinito positivo/negativo (configurable).
