Se informa que la copia manual del contenido del array puede ser reemplazada por una llamada a System.arraycopy().

Ejemplo:


  for (int i = 0; i < array.length; i++) {
    newArray[i] = array[i];
  }

Después de aplicar la corrección rápida:


  System.arraycopy(array, 0, newArray, 0, array.length);
