Reporta el atributo autowire en el elemento <bean> y sugiere la inyección explícita de propiedades Bean cuando sea posible.

Ejemplo:


  public class MyComponent {
    public void setOtherBean(OtherBean bean){...}
  }


  <beans>
    <bean class="beans.OtherBean" id="bar"/>
    <bean autowire="byType" class="beans.MyComponent"/>
    <!-- "Dependencia autowired innecesaria" -->
    <!-- Sugerencia: "Hacer explícita la dependencia autowired" -->
  </beans>

Después de aplicar la corrección rápida:


  <beans>
    <bean class="beans.OtherBean" id="bar"/>
    <bean class="beans.MyComponent" id="foo">
      <property name="otherBean" ref="otherBean"/>
    </bean>
  </beans>
