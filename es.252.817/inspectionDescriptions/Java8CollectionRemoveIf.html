El bucle puede ser colapsado en una única llamada a Collection.removeIf().

Ejemplo:


  for (Iterator<String> it = collection.iterator(); it.hasNext(); ) {
    String aValue = it.next();
    if(shouldBeRemoved(aValue)) {
      it.remove();
    }
  }

Después de aplicar la corrección rápida:


  collection.removeIf(aValue -> shouldBeRemoved(aValue));
