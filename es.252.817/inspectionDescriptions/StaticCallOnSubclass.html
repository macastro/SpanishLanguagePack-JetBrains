Informa sobre llamadas a métodos static que están calificadas por una subclase de la clase declarante en lugar de la clase declarante misma.

Java permite tales calificaciones de clase, pero dichas llamadas pueden indicar una confusión sutil entre herencia y sobrescritura.

Ejemplo:


  class Parent {
    public static void print(String str) {}
  }
  class Child extends Parent {}
  Child.print("Hello, world!");

Después de aplicar la corrección rápida:


  Parent.print("Hello, world!");
