La clase declara múltiples registradores (loggers). Un paso importante para proporcionar una implementación de registro unificada para la aplicación es asegurar que cada clase tenga su propio registrador dedicado.

Por ejemplo:


  public class Critical {
    protected static final Logger LOG = Logger.getLogger(Critical.class);
    protected static final Logger myLogger = Logger.getLogger(getClass());
  }
