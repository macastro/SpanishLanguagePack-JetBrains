Informa sobre contenedores, métodos y alias de tipo no utilizados en las variables de tipo del proyecto.

Las variables de tipo de contenedor solo se verifican cuando el contenedor tiene exactamente una declaración.

Las variables de tipo no utilizadas se pueden eliminar mediante una corrección rápida.

Ejemplo:

# La variable de tipo 'A' de la clase 'Example' no se utiliza
class Example[A]
end

Después de aplicar la corrección rápida:

class Example
end
