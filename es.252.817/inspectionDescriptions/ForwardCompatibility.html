Informa las construcciones de código Java que podrían fallar en compilaciones en futuras versiones de Java.

Se informan los siguientes problemas:

  • Uso de assert, enum o _ como identificador
  • Uso de identificadores restringidos var, yield o record como nombres de tipos
  • Llamadas al método yield sin calificadores
  • Modificadores en la declaración requires java.base dentro de module-info.java
  • Punto y coma redundante entre declaraciones de importación

Ejemplo:


  // Esta clase solía ser válida, pero no compila para Java 14 ya que
  // 'yield' se convirtió en un identificador restringido.
  public class yield {}

Solucionar estos problemas oportunamente puede simplificar la migración a futuras versiones de Java.
