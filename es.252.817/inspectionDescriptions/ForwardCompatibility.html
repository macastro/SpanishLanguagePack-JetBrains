Informa sobre estructuras de código Java que podrían no compilar en futuras versiones de Java.

Se informará sobre los siguientes problemas:

  • Uso de assert, enum o _ como identificadores
  • Uso de identificadores restringidos var, yield o record como nombres de tipo
  • Llamadas no calificadas al método yield
  • Modificadores en la declaración requires java.base dentro de module-info.java
  • Punto y coma redundante entre declaraciones import

Ejemplo:


  // Esta clase anteriormente válida no compilará en Java 14,
  // porque 'yield' se convirtió en un identificador restringido.
  public class yield {}

Resolver estos problemas con anticipación facilitará la migración a futuras versiones de Java.
