Reporta código que compara cadenas de texto usando == o !=.

Estos operadores determinan la igualdad de referencia en lugar de comparar el contenido. En la mayoría de los casos, se debe usar equals() para comparar cadenas, que realiza una comparación carácter por carácter cuando las cadenas son objetos diferentes.

Ejemplo:


  void foo(String s, String t) {
    final boolean b = t == s;
  }

Si se sabe que t no es null, se puede aplicar de manera segura la corrección rápida "insegura" y obtener un resultado similar al siguiente:


  void foo(String s, String t) {
    final boolean b = t.equals(s);
  }
