Reporta problemas con los campos AtomicLongFieldUpdater, AtomicIntegerFieldUpdater o AtomicReferenceFieldUpdater (paquete java.util.concurrent.atomic).

Los problemas reportados son los mismos que pueden ocurrir en tiempo de ejecución con los actualizadores de campos atómicos: campo especificado no encontrado, campo especificado inaccesible, tipo de campo especificado incorrecto, etc.

Ejemplos:

  • 
  class A {
    private static volatile int value = 0;
    private static final AtomicIntegerFieldUpdater updater =
    AtomicIntegerFieldUpdater.newUpdater((A.class), "value"); // Advertencia: el campo 'value' tiene el modificador 'static'
  }
  
  • 
  class B {
    private static final AtomicIntegerFieldUpdater updater =
      AtomicIntegerFieldUpdater.newUpdater(B.class, "value"); // Advertencia: no se encuentra el campo 'value' en la clase 'B'
  }
