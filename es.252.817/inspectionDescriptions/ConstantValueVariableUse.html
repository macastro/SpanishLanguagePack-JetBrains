Informa sobre todos los usos de variables que se sabe que son constantes.

Esto ocurre cuando el uso (lectura) de una variable está envuelto en una sentencia if, while o for, y existe una condición == que compara dicha variable con una constante. En estos casos, el uso de la variable que se sabe que es constante puede ser reemplazado por la constante real.

Ejemplo:


    private static void foo(double number) {
        if (number == 1.0) {
            f(number);
        }
    }
    private static void f(double number) {}

Después de aplicar la corrección rápida:


    private static void foo(double number) {
        if (number == 1.0) {
            f(1.0);
        }
    }
    private static void f(double number) {}
