Informa de una llamada al método estático Collections de Java que puede ser reemplazado por Kotlin stdlib.

Ejemplo:


  import java.util.Collections

  fun test() {
      val mutableList = mutableListOf(1, 2)
      Collections.fill(mutableList, 3)
  }

La corrección rápida reemplaza la llamada al método estático Collections de Java por la llamada al método stdlib de Kotlin correspondiente:


  import java.util.Collections

  fun test() {
      val mutableList = mutableListOf(1, 2)
      mutableList.fill(3)
  }
