Informa de patrones de formato de mensaje incorrectos o de índices incorrectos de marcadores de posición

Se informan de los siguientes errores:

  • Índice no analizado o negativo
  • Llave no cerrada
  • Comillas no emparejadas. En este caso, una parte del patrón puede no utilizarse
  • Probablemente número incorrecto de comillas
  • Límite inferior incorrecto de patrones de opciones anidadas
  • Índices incorrectos de marcadores de posición. En este caso, es posible que no se sustituya un marcador de posición o que no se utilice un argumento

Ejemplos:


  MessageFormat.format("{wrong}", 1); // índice incorrecto
  MessageFormat.format("{0", 1); // Llave sin cerrar
  MessageFormat.format("'{0}", 1); // Comillas sin emparejar
  MessageFormat.format("It''''s {0}", 1); // Se imprimirá "It''s" en lugar de "It's"
  MessageFormat.format("{0}", 1, 2); // El argumento con índice '1' no se utiliza en el patrón
