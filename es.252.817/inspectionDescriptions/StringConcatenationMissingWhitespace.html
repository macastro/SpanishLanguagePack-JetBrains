Reporta las concatenaciones de cadenas con espacios en blanco faltantes, es decir, cuando el lado izquierdo termina con una letra o dígito Unicode y el lado derecho comienza con una letra o dígito Unicode.

Ejemplo:


  String sql = "SELECT columna" +
               "DE tabla";

Utiliza la opción Ignorar concatenaciones con cadenas variables para reportar solo cuando tanto el lado izquierdo como el derecho de la concatenación son literales.
