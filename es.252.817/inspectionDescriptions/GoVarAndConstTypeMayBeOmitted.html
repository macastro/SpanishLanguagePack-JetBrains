Informa sobre tipos de variables y constantes que pueden omitirse porque el compilador puede inferirlos. Esta redundancia de tipos puede eliminarse para mejorar la legibilidad del código.

Ejemplo:

var s string = fmt.Sprintln("hi")

El tipo string en la declaración de variable puede omitirse. Para eliminar este tipo, utilice la corrección rápida Eliminar tipo. Después de aplicar la corrección rápida:

var s = fmt.Sprintln("hi")
