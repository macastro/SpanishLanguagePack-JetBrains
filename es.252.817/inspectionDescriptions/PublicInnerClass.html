Reporta clases anidadas public.

Ejemplo:


  public class Outer {
    public static class Nested {} // advertencia
    public class Inner {}         // advertencia
    public enum Mode {}           // advertencia según la configuración
    public interface I {}         // advertencia según la configuración
  }

Configuración de la inspección:

  • Utilice la opción Ignorar enums en 'public' para ignorar enums dentro de public.
  • Utilice la opción Ignorar interfaces en 'public' para ignorar interfaces dentro de public.
