Informa sobre columnas, alias y otros elementos de subconsultas no referenciados en expresiones de consultas externas.

Ejemplo (PostgreSQL):

CREATE TABLE for_subquery(id INT);
SELECT a, q FROM (SELECT 1 AS a, 10 AS b, 2 + 3 AS q, id
      FROM for_subquery) x;

Hacemos referencia a los alias a y q de la subconsulta. Sin embargo, el alias b y la columna id no se referencian en la sentencia SELECT externa. Por lo tanto, b e id se mostrar√°n en gris.
