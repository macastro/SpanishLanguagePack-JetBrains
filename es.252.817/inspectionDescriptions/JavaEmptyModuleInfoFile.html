Reporta un archivo module-info.java vacío, indicando dependencias de módulos no resueltas. Agrega automáticamente declaraciones necesarias de requires mediante la inspección de importaciones. Para suprimir esta advertencia, puedes escribir cualquier comentario dentro del cuerpo de la declaración del módulo, así: 

module module.name {
    // sin dependencias
}
Solución rápida: Rellenar dependencias de módulo rellena requires faltantes según las importaciones del código fuente. Novedad en 2024.1
