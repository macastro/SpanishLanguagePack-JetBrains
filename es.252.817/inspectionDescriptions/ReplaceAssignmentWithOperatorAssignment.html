El informe puede reemplazar la asignación con operadores de asignación compuesta.

El código que utiliza operadores de asignación compuesta es más corto y potencialmente más claro.

Ejemplo:

  x = x + 3;
  x = x / 3;

Después de aplicar la corrección rápida:

  x += 3;
  x /= 3;

La opción Ignorar operadores condicionales permite ignorar && y ||. Reemplazar operadores condicionales con operadores de asignación compuesta puede cambiar la evaluación perezosa a evaluación inmediata, alterando potencialmente la semántica de la expresión.

La opción Ignorar operadores ambiguos permite ignorar los operadores menos conocidos ^ y %.
