Informa de los métodos con un tipo de devolución boolean que siempre se niegan al ser llamados.

Se proporciona una corrección rápida para invertir y, opcionalmente, renombrar el método. Por razones de rendimiento, es posible que no todos los métodos problemáticos se resalten en el editor.

Ejemplo:


  class C {
    boolean alwaysTrue() {
      return true;
    }

    void f() {
      if (!alwaysTrue()) {
        return;
      }
    }
    boolean member = !alwaysTrue();
  }

Después de aplicar la corrección rápida:


  class C {
    boolean alwaysFalse() {
      return false;
    }

    void f() {
      if (alwaysFalse()) {
        return;
      }
    }
    boolean member = alwaysFalse();
  }
