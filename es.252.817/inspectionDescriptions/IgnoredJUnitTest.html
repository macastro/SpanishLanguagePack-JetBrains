Informa sobre el uso de la anotación @Ignore de JUnit 4 o @Disabled de JUnit 5. El uso prolongado de estas anotaciones en las pruebas se considera un code smell, especialmente cuando no se especifica un motivo.

Ejemplo:


  @Ignore
  public class UrgentTest {
    @Test
    public void testIt() {
      Assert.assertEquals("expected", "actual");
    }
  }

Configuración de la inspección:

  • La opción Informar solo anotaciones sin motivo permite reportar únicamente los casos donde no se especifica un motivo como atributo value de la anotación.
