Reporta variables que exceden la longitud de caracteres definida.

Ejemplo (Microsoft SQL Server):

CREATE PROCEDURE test() AS
BEGIN
DECLARE myVarOk VARCHAR(5) = 'abcde';
DECLARE myVarExceeded VARCHAR(5) = 'abcde12345';
SET myVarOk = 'xyz';
SET myVarExceeded = '123456789';
END;

La variable myVarExceeded está definida como VARCHAR(5) pero ambos valores asignados ('abcde12345' y '123456789') exceden este límite. Puede truncar los valores especificados o aumentar la longitud definida. Para aumentar la longitud, utilice la corrección rápida Aumentar longitud del tipo.

Después de aplicar la corrección rápida:

CREATE PROCEDURE test() AS
BEGIN
DECLARE myVarOk VARCHAR(5) = 'abcde';
DECLARE myVarExceeded VARCHAR(10) = 'abcde12345';
SET myVarOk = 'xyz';
SET myVarExceeded = '123456789';
END;
