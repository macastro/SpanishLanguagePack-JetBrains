Informa sobre variables que superan la longitud definida en caracteres.

Ejemplo (Microsoft SQL Server):

CREATE PROCEDURE test() AS
BEGIN
DECLARE myVarOk VARCHAR(5) = 'abcde';
DECLARE myVarExceeded VARCHAR(5) = 'abcde12345';

SET myVarOk = 'xyz';
SET myVarExceeded = '123456789';
END;

La variable myVarExceeded está definida como VARCHAR(5) pero ambos valores asignados ('abcde12345' y '123456789') exceden esta limitación. Puedes recortar los valores asignados o incrementar la longitud definida. Para incrementar la longitud, usa el quick-fix Incrementar longitud de tipo.

Después de que el quick-fix sea aplicado:

CREATE PROCEDURE test() AS
BEGIN
DECLARE myVarOk VARCHAR(5) = 'abcde';
DECLARE myVarExceeded VARCHAR(10) = 'abcde12345';

SET myVarOk = 'xyz';
SET myVarExceeded = '123456789';
END;
