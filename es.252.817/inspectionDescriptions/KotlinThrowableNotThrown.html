Informa cuando se crea una instancia de Throwable o sus subclases que nunca se lanza realmente.

El código reportado indica errores que son difíciles de capturar en las pruebas.

Además, esta inspección informa sobre las llamadas a métodos que devuelven una instancia de Throwable o sus subclases cuando la instancia resultante no se lanza.

Ejemplo:


  fun check(condition: Boolean) {
      if (!condition) /* throw is missing here */ IllegalArgumentException("condition is not met");
  }
  fun createError() = RuntimeException()
  fun foo() {
      /* throw is missing here */ createError()
  }
