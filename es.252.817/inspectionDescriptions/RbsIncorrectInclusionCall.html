Informa sobre las llamadas de inclusión de clases (include, prepend y extend). Las llamadas de inclusión solo pueden hacer referencia a módulos o interfaces.

Las llamadas de inclusión incorrectas se pueden eliminar mediante una corrección rápida.

Ejemplo:

class Example
  # En la llamada 'include' usando la clase 'Object'
  include Object
end

Después de aplicar la corrección rápida:

class Example
end
