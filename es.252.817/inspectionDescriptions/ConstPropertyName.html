Informa sobre nombres de propiedades const que no siguen las convenciones de nomenclatura recomendadas.

Una nomenclatura consistente facilita la lectura y comprensión del código. Según la guía de estilo oficial de Kotlin, las propiedades const deben usar nombres en mayúsculas separados por guiones bajos.

Ejemplo:


  const val Planck: Double = 6.62607015E-34

La corrección rápida renombrará la propiedad:


  const val PLANCK: Double = 6.62607015E-34
