Informa sobre la sincronización de variables locales o parámetros.

Es difícil garantizar operaciones correctas cuando se usa este tipo de sincronización. Este código se puede mejorar, por ejemplo, controlando el acceso mediante una clase contenedora sincronizada o sincronizando los campos.

Ejemplo:


  void bar() {
    final Object lock = new Object();
    synchronized (lock) { }
  }
