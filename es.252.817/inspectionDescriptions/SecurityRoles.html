Inspecciona nombres incorrectos de roles de seguridad

Ejemplo:


  <web-app>
    <security-role>
      <role-name>custom#role</role-name> // Error: nombre incorrecto
    </security-role>
    <security-role>
      <role-name>customRole</role-name> // Nombre correcto
    </security-role>
  </web-app>
