Informa sobre las llamadas a java.util.Set.removeAll() con un argumento java.util.List.

Tales llamadas pueden ser lentas cuando el tamaño del argumento es mayor o igual al tamaño del conjunto y el conjunto es una subclase de java.util.AbstractSet. En este caso, se llama a List.contains() para cada elemento del conjunto, lo que resulta en una búsqueda lineal.

Ejemplo:


  public void check(String... ss) {
    // Posible complejidad O(n^2)
    mySet.removeAll(List.of(ss));
  }

Después de aplicar la corrección rápida:


  public void check(String... ss) {
    // Complejidad O(n)
    List.of(ss).forEach(mySet::remove);
  }
