Informa sobre variables locales con parámetros de tipo.

Los parámetros de tipo en variables locales no tienen sentido ya que no pueden ser especializados.

Ejemplo:


  fun main() {
      val <T> x = ""
  }

Después de aplicar la corrección rápida:


  fun main() {
      val x = ""
  }

Esta inspección solo informa cuando el nivel del lenguaje Kotlin del proyecto o módulo es 1.4 o superior.
