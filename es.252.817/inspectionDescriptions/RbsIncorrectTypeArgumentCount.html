Informa sobre referencias a tipos genéricos con un número incorrecto de argumentos de tipo. Es un error usar un tipo genérico sin pasar el número correcto de argumentos de tipo.

El número de argumentos de tipo proporcionados puede corregirse mediante una corrección rápida.

Ejemplo:

class Example[A]
end
# Demasiados argumentos de tipo proporcionados
type example = Example[untyped, untyped]

Después de aplicar la corrección rápida:

class Example[A]
end
type example = Example[untyped]
