Reporta importaciones no ordenadas.

Todos los programas en Go deben ser formateados de la misma manera, las reglas de formato son fijas por la herramienta gofmt. Esas reglas requieren que las importaciones estén ordenadas.

Ejemplo de un orden incorrecto:

import (
 "net"
 "errors"
 "fmt"
)

Puedes aplicar la solución rápida Ordenar importaciones para arreglar el orden. Después de que se aplica la solución rápida, el orden luce como sigue:

import (
 "errors"
 "fmt"
 "net"
)
