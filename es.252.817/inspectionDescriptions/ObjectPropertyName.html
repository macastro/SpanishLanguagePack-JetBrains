Informa sobre propiedades que no siguen las convenciones de nomenclatura.

Se informan las siguientes propiedades:

  • Propiedades de nivel superior
  • Propiedades en objetos y objetos companion

Puede especificar los patrones deseados en las opciones de inspección.

Convenciones de nomenclatura recomendadas: deben comenzar con una letra mayúscula, usar camelCase y no contener guiones bajos.

Ejemplo:


  // Propiedad de nivel superior
  val USER_NAME_FIELD = "UserName"
  // Propiedad de nivel superior que contiene referencia a objeto singleton
  val PersonComparator: Comparator<Person> = /*...*/
  class Person {
    companion object {
      // Propiedad en objeto companion
      val NO_NAME = Person()
    }
  }
