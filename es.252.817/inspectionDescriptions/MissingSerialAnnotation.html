Informa sobre métodos y campos en clases Serializable y Externalizable que son candidatos para ser anotados con la anotación java.io.Serial. La corrección rápida añade la anotación.

Ejemplo:


class Main implements Serializable {
  private static final long serialVersionUID = 7874493593505141603L;
  private void writeObject(ObjectOutputStream out) throws IOException {
  }
}

Después de aplicar la corrección rápida:


  class Main implements Serializable {
    @Serial
    private static final long serialVersionUID = 7874493593505141603L;
    @Serial
    private void writeObject(ObjectOutputStream out) throws IOException {
    }
}

Ejemplo:


  class Main implements Externalizable {
    protected Object readResolve() throws ObjectStreamException {
      return "SomeObject";
    }
  }

Después de aplicar la corrección rápida:


  class Main implements Externalizable {
    @Serial
    protected Object readResolve() throws ObjectStreamException {
        return "SomeObject";
    }
  }

Para más información sobre todos los casos posibles, consulte el Javadoc de la clase java.io.Serial.
