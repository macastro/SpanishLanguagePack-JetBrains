Informa sobre el uso de comparaciones == y === en la misma variable dentro de una única expresión.

Debido a su similitud, == y === pueden mezclarse inadvertidamente, por lo que es necesario verificar cuidadosamente si se está usando == en lugar de ===

Ejemplo:


  if (type === FIELD || type == METHOD || type == ANNOTATION_METHOD || // Observe el uso incorrecto de "=="
      type === LAMBDA_EXPRESSION) return
