Informa sobre tipos de parámetros especificados explícitamente en los setters de propiedades.

El tipo de parámetro del setter siempre coincide con el tipo de la propiedad, por lo que no es necesario que sea explícito. La corrección rápida "Eliminar especificación de tipo explícito" permite modificar el código en consecuencia.

Ejemplo:


  fun process(x: Int) {}
  var x: Int = 0
      set(value: Int) = process(value) // <== La especificación 'Int' se puede omitir de forma segura

Después de aplicar la corrección rápida:


  fun process(x: Int) {}
  var x: Int = 0
      set(value) = process(value)
