Informa sobre los inicializadores de instancia que pueden ser declarados como static.

Un inicializador de instancia puede ser static si no hace referencia a ningún miembro no static de su clase. Los inicializadores static se ejecutan después de que se resuelve la clase, mientras que los inicializadores de instancia se ejecutan para cada instanciación de esta clase.

Esta inspección no informa sobre los inicializadores de instancia vacíos ni los inicializadores en clases anónimas.

Ejemplo:


  class A {
     public static String CONSTANT;
     {
        CONSTANT = "Hello";
     }
  }

Después de aplicar la corrección rápida:


  class A {
     public static String CONSTANT;
     static {
        CONSTANT = "Hello"; // ahora se instancia solo una vez por clase
     }
  }
