Reporta las siguientes violaciones de la especificación EJB:
  • Tipo de propiedad inyectada no válido
  • La referencia EJB no coincide con `ejb-jar.xml`

Ejemplo:


@LocalBean
@Stateless
public class ExampleBean {
  private AnotherBean anotherBean;
  @Inject
  public void getAnotherBean() { // Error: la inyección debe realizarse en un campo o método setter
    return anotherBean;
  }
  ...
}
