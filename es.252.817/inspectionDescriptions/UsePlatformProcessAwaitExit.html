Informa sobre el uso de Process.waitFor() y Process.onExit() en contextos de corrutinas. El uso de estos métodos está prohibido en contextos de corrutinas.

Use com.intellij.util.io.awaitExit() en su lugar.

Ejemplo:


suspend fun doSomething(process: Process) {
  val exitCode = process.waitFor() // Incorrecto
  // ...
}
suspend fun doSomething(process: Process) {
  val exitCode = process.awaitExit() // Correcto
  // ...
}
