Sincronizar campos static sobre datos de instancia no evita que otras instancias modifiquen el campo, lo que puede provocar condiciones de carrera inesperadas.

Ejemplo:


  static String s;
  def foo() {
      synchronized (this) {
          System.out.println(s); // advertencia
      }
  }
