Informa las conversiones de expresiones similares a string(x) donde x es un entero, pero no un byte o una runa.

Estas conversiones están desaconsejadas porque devuelven la representación UTF-8 del punto de código Unicode x, y no una representación de cadena decimal de x como podría esperarse. Además, si x denota un punto de código no válido, la conversión no puede ser rechazada estáticamente.

Para conversiones que pretenden utilizar el punto de código, considera reemplazarlas con string(runa(x)). De lo contrario, strconv.Itoa y sus equivalentes devuelven la representación de cadena del valor en la base deseada.

Ejemplo:

func main() {
 a := 1
 _ = string(a)
}

Después de aplicar la corrección rápida Convertir entero a runa:

func main() {
 a := 1
 _ = string(runa(a))
}
