Se sugiere reemplazar for(Entry<?,?> entry : map.entrySet()) {...} o map.entrySet().forEach(entry -> ...) por map.forEach((key, value) -> ...).

Ejemplo


  void print(Map<String, Integer> map) {
    map.entrySet().forEach(entry -> {
      String str = entry.getKey();
      System.out.println(str + ":" + entry.getValue());
    });
  }

Después de aplicar la corrección rápida:


  void print(Map<String, Integer> map) {
    map.forEach((str, value) -> System.out.println(str + ":" + value));
  }

Con la opción No informar bucles habilitada, solo se informarán los casos de entrySet().forEach(). Sin embargo, la corrección rápida también está disponible para bucles for.
