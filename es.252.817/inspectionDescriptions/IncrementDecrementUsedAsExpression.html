Informa sobre expresiones de incremento o decremento anidadas dentro de otras expresiones. Estas expresiones pueden resultar confusas y violan el principio general de diseño de que cada construcción debe tener un solo propósito.

Esta corrección rápida extrae la operación de incremento o decremento a una declaración de expresión independiente.

Ejemplo:


  int i = 10;
  while (i-- > 0) {
    System.out.println(i);
  }

Después de aplicar la corrección rápida:


  int i = 10;
  while (i > 0) {
    i--;
    System.out.println(i);
  }
  i--;
