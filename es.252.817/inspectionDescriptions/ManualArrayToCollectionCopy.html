Informa del código que usa un bucle para copiar el contenido de un array a una colección.

Una forma más corta y, potencialmente, más rápida (dependiendo de la implementación de la colección) de hacerlo es usar Collection.addAll(Arrays.asList()) o Collections.addAll().

Solo los bucles sin declaraciones adicionales en su interior son informados.

Ejemplo:


  void addAll(List<String> list, String[] arr) {
    for (int i = 0; i < arr.length; i++) {
      String s = arr[i];
      list.add(s);
    }
  }

Después de aplicar la corrección rápida:


  void addAll(List<String> list, String[] arr) {
    Collections.addAll(list, arr);
  }
