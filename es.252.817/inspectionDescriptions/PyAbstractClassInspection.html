Informa los casos cuando no todas las propiedades o métodos abstractos se definen en una subclase.

Ejemplo:


from abc import abstractmethod, ABC


class Figure(ABC):

    @abstractmethod
    def do_figure(self):
        pass


class Triangle(Figure):
    def do_triangle(self):
        pass

Cuando se aplica la corrección rápida, el IDE implementa un método abstracto para la clase Triangle:


from abc import abstractmethod, ABC


class Figure(ABC):

    @abstractmethod
    def do_figure(self):
        pass


class Triangle(Figure):
    def do_figure(self):
        pass

    def do_triangle(self):
        pass
