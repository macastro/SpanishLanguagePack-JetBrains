Informa sobre casos donde no todos los métodos o propiedades abstractos están definidos en las clases hijas.

Ejemplo:


from abc import abstractmethod, ABC
class Figure(ABC):
    @abstractmethod
    def do_figure(self):
        pass
class Triangle(Figure):
    def do_triangle(self):
        pass

Después de aplicar la corrección rápida, el IDE implementa el método abstracto para la clase Triangle:


from abc import abstractmethod, ABC
class Figure(ABC):
    @abstractmethod
    def do_figure(self):
        pass
class Triangle(Figure):
    def do_figure(self):
        pass
    def do_triangle(self):
        pass
