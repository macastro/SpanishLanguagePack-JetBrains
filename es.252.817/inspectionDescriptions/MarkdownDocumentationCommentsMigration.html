Informa sobre los comentarios Javadoc que se pueden convertir a comentarios de documentación de Markdown.

Ejemplo:


  /**
   * For blubbering of the florz.
   *
   * Similar to {@link com.deeps.Querp querping} class.
   */
  class FlorzBlubber {
  }

Tras aplicar la corrección rápida:


  /// For blubbering of the florz.
  /// Similar to [querping][com.deeps.Querp] class.
  class FlorzBlubber {
  }
