Informa sobre llamadas a métodos de operaciones de arrays no genéricos con tipos de argumentos incompatibles, como Arrays.fill(). Estas llamadas no realizan ningún trabajo útil y probablemente sean erróneas.

Ejemplo:


  int foo(String[] strings) {
    return Arrays.binarySearch(strings, 1);
  }

Novedades de la versión 2017.2
