Informa sobre clases Serializable que tienen métodos readObject o writeObject no declarados como private. No hay razón para que estos métodos tengan una visibilidad mayor que private.

Se recomienda establecer los métodos correspondientes como private mediante la corrección rápida.

Ejemplo:


  public class Test implements Serializable {
    public void readObject(ObjectInputStream stream) {
      /* ... */
    }
  }

Después de aplicar la corrección rápida:


  public class Test implements Serializable {
    private void readObject(ObjectInputStream stream) {
      /* ... */
    }
  }
