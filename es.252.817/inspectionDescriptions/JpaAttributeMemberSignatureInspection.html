Reporta los siguientes errores de especificación JPA:
  • Múltiples anotaciones configuradas para un método o campo
  • Uso de anotaciones no permitidas en objetos Embeddable (JPA 1.0)
  • Metadatos de persistencia no especificados en campos o getters de propiedades
  • Falta de setter o getter para el acceso basado en propiedades

Ejemplo:


  @Entity
  @Table(name = "TABLE")
  public class JavaEntity {
    private Integer id; // Error: deben existir tanto getter como setter
    public Integer getId() { ... }
    private String data;
    public String getData() { ... }
    @Basic // Error: la anotación de persistencia debe estar en el campo o método getter
    public void setData(String data) { ... }
  }
