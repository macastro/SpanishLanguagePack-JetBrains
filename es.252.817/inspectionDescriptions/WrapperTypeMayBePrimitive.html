El informe se utiliza principalmente para variables locales de tipos wrapper de tipos primitivos.

En algunos casos, el boxing puede causar una pérdida significativa de rendimiento, especialmente en bucles.

Se utiliza una estimación heurística del número de operaciones de boxing. Por ejemplo, las conversiones dentro de un bucle se consideran más numerosas.

Ejemplo:

  public void example() {
    Integer value = 12;
    needBox(value);
    for (int i = 0; i < 10; i++) {
      // El uso en el bucle se vuelve cada vez más frecuente
      needPrimitive(value);
    }
  }
  void needPrimitive(int value) {}
  void needBox(Integer value) {}

Después de aplicar la corrección rápida:

  public void example() {
    int value = 12;
    needBox(value);
    for (int i = 0; i < 10; i++) {
      // El uso en el bucle se vuelve cada vez más frecuente
      needPrimitive(value);
    }
  }
  void needPrimitive(int value) {}
  void needBox(Integer value) {}
