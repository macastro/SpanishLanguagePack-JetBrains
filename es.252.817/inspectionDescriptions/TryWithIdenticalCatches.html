Reporta secciones catch idénticas dentro de una misma declaración try.

Combinar estas secciones en un bloque multi-catch reduce la duplicación de código y evita situaciones donde una sección catch se actualiza mientras que otra queda sin actualizar.

Ejemplo:


    try {
        doSmth();
    }
    catch (IOException e) {
        LOG.error(e);
    }
    catch (URISyntaxException e) {
        LOG.error(e);
    }

El código se puede hacer más compacto mediante una corrección rápida:


    try {
        doSmth();
    }
    catch (IOException | URISyntaxException e) {
        LOG.error(e);
    }
