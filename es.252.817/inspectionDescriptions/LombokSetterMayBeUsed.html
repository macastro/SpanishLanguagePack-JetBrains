Informa del método setter estándar que puede ser reemplazado por la anotación lombok @Setter.

Ejemplo:


import java.util.Date;

public class MyClass {
    /**
     * La fecha del proceso.
     */
    private Date processDate;

    /**
     * Establece la fecha.
     *
     * @param La fecha
     */
    public void setProcessDate(Date param) {
      processDate = param;
    }
  }

Tras aplicar el arreglo/limpieza rápidos:


import lombok.Setter;
import java.util.Date;

@Setter
public class MyClass {
    /**
     * La fecha del proceso.
     * -- SETTER --
     * Establece la fecha.
     *
     * @param La fecha
     */
    private Date processDate;
  }

Sólo informa cuando la biblioteca lombok está configurada. Para eliminar el lombok, consulte el sitio de lombok.
