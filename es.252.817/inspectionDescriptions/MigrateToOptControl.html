El método createOptionsPanel() en la implementación del inspector puede convertirse automáticamente a getOptionsPane().

La creación de controles de opciones de inspección a través de createOptionsPanel() está obsoleta y debe reemplazarse por la descripción declarativa de controles getOptionsPane(). Esta inspección analiza las implementaciones de createOptionsPanel() y sugiere automáticamente la conversión a la nueva API si utilizan InspectionOptionsPanel o sus descendientes y son lo suficientemente simples.

Esta inspección actualmente solo es compatible con Java y Kotlin. No puede convertir código que utilice métodos de Kotlin como apply o also. Intente integrarlos directamente.
