Informa sobre métodos clone() que no están marcados como final.

El método clone() puede usarse para instanciar objetos sin usar constructores, por lo que permitir la sobrescritura del método clone() puede llevar a la corrupción de objetos e incluso a vulnerabilidades de seguridad. Esto se puede prevenir haciendo que el método clone() o la clase contenedora sea final.

Ejemplo:


  class Main implements Cloneable {
      @Override
      protected Object clone() throws CloneNotSupportedException {
        return super.clone();
      }
  }
