Inspecciona las conversiones implícitas entre tipos numéricos.

La conversión numérica implícita no es un problema en sí mismo, pero si no se espera, puede ocasionar dificultades al rastrear errores.

Ejemplo:


  double m(int i) {
    return i * 10;
  }

Luego de la aplicación de la solución rápida:


  double m(int i) {
    return (double) (i * 10);
  }

Configurar la inspección:

  • Usa la opción Ignorar conversiones de ampliación para ignorar la conversión implícita que no puede resultar en pérdida de datos (por ejemplo, int->long).
  • Usa la opción Ignorar conversiones desde y hacia 'char' para ignorar la conversión desde y hacia char. La inspección seguirá informando la conversión desde y hacia números de punto flotante.
  • Usa la opción Ignorar conversión desde constantes y literales para hacer que la inspección ignore la conversión desde literales y constantes de tiempo de compilación.
