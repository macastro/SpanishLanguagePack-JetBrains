Informa sobre las funciones forEach en las funciones de ámbito tales como also o apply que se pueden simplificar.

La corrección rápida "Convertir llamada de forEach a onEach" se puede utilizar para modificar el código automáticamente.

Ejemplos:


  fun test(list: List<Int>) {
      val x = list.also { it.forEach { it + 4 } }.toString()
      val y = list.apply { forEach { println(it) } }
  }

Tras aplicar la corrección rápida:


  fun test(list: List<Int>) {
      val x = list.onEach { it + 4 }.toString()
      val y = list.onEach { println(it) }
  }
