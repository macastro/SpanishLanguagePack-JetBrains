Reporta funciones forEach que pueden simplificarse en funciones de alcance como also o apply.

La corrección rápida Convertir forEach a onEach está disponible para corregir el código automáticamente.

Ejemplo:


  fun test(list: List<Int>) {
      val x = list.also { it.forEach { it + 4 } }.toString()
      val y = list.apply { forEach { println(it) } }
  }

Después de aplicar la corrección rápida:


  fun test(list: List<Int>) {
      val x = list.onEach { it + 4 }.toString()
      val y = list.onEach { println(it) }
  }
