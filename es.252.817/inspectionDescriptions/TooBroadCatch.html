Bloque catch que reporta parámetros formales más generales que las excepciones lanzadas por el bloque try correspondiente.

Ejemplo:


  try  {
    File file = new File(pathToFile);
    return file.getAbsolutePath();
  } catch (Exception ex) { // warning: 'catch' of 'Exception' is too broad, masking exceptions 'RuntimeException'
    return defaultFilePath;
  }

Después de aplicar la corrección rápida:


  try  {
    File file = new File(pathToFile);
    return file.getAbsolutePath();
  } catch (RuntimeException ex) {
    return defaultFilePath;
  }

Configuración de la inspección:

  • Use la opción Advertir solo para RuntimeException, Exception, Error o Throwable para que esta inspección emita advertencias solo para las excepciones más comunes.
  • Use la opción Ignorar excepciones que ocultan otras pero que son lanzadas para ignorar cualquier excepción que oculte otras pero que aún pueda ser lanzada, por lo que técnicamente no es demasiado general.
