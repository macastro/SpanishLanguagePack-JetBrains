Informa sobre problemas con decoradores anidados. Esta inspección resalta los casos donde classmethod o staticmethod se aplican después de otro decorador.

Ejemplo:


def innocent(f):
    return f
class A:
    @innocent  # El decorador no recibirá el callable esperado
    @classmethod
    def f2(cls):
        pass
    @innocent  # El decorador no recibirá el callable esperado
    @staticmethod
    def f1():
        pass

La corrección rápida propuesta por el IDE es eliminar este decorador.
