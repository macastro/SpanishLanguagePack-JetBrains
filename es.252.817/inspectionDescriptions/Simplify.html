Informa sobre declaraciones que pueden simplificarse, como condiciones constantes, ramas if idénticas, expresiones booleanas redundantes, etc. Utilice las siguientes casillas de verificación para controlar las opciones de análisis:
  • Simplificar "if (true)", "while(false)": sugiere reemplazar declaraciones if/while con literales booleanos.
  • Simplificar "condition == true": sugiere reemplazar declaraciones de igualdad con literales booleanos.
  • Simplificar "condition1 ? true : condition2;": sugiere simplificar operadores ternarios donde una de las expresiones es un literal booleano.
  • Simplificar "if (condition1) return true; return condition2;": sugiere simplificar declaraciones if que devuelven un literal booleano si la condición es verdadera, o devuelven otra condición en caso contrario.
