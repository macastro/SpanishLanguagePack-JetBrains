Reporta declaraciones que pueden ser simplificadas, por ejemplo condiciones constantes, ramas if idénticas, expresiones booleanas redundantes, etc. Usa las siguientes casillas para controlar las opciones de análisis:
  • Simplificar "if (true)", "while(false)": Sugiere reemplazar las sentencias if/while por un literal booleano.
  • Simplificar "condition == true": Sugiere reemplazar las sentencias de comparación de igualdad por un literal booleano.
  • Simplificar "condition1 ? true : condition2;": Sugiere simplificar los operadores ternarios en los que una de las expresiones es un booleano.
  • Simplificar "if (condition1) return true; return condition2;": Sugiere simplificar los if que devuelven un literal booleano si la condición es verdadera y una condición diferente si es falsa.
