record informa del método de serialización o campos definidos en una clase. Los métodos de serialización que se notifican incluyen, writeObject(), readObject(), readObjectNoData(), writeExternal(), readExternal() y el campo serialPersistentFields. Estos miembros no son necesarios porque no se usan para serializar o deserializar registros.

Ejemplo:


  record R1() implements Serializable {
    // Este campo se ignora durante la serialización del registro
    @Serial
    private static final ObjectStreamField[] serialPersistentFields = new ObjectStreamField[0];

    // Este método se ignora durante la serialización del registro
    @Serial
    private void writeObject(ObjectOutputStream out) throws IOException {
    }
  }


  record R2() implements Externalizable {
    // Este método se ignora durante la serialización del registro
    @Override
    public void writeExternal(ObjectOutput out) throws IOException {
    }

    // Este método se ignora durante la serialización del registro
    @Override
    public void readExternal(ObjectInput in) throws IOException, ClassNotFoundException {
    }
  }
