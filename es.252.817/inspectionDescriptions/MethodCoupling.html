Informa sobre métodos que hacen referencia a demasiadas clases externas. Los métodos con alto acoplamiento pueden ser muy frágiles y deberían dividirse en métodos más pequeños.

Cada clase referenciada se cuenta una sola vez, independientemente del número de referencias.

Configuración de la inspección:

  • Use el campo Límite de acoplamiento del método para especificar el acoplamiento máximo permitido para un método.
  • Use la opción Incluir acoplamiento con clases del sistema Java para contar referencias a clases en los paquetes java o javax.
  • Use la opción Incluir acoplamiento con clases de biblioteca para contar referencias a clases de bibliotecas de terceros.
