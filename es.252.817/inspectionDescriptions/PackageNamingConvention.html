Reporta paquetes de software cuyo nombre es demasiado corto, demasiado largo o no sigue el patrón de expresión regular especificado.

Ejemplo:


  package io;

Las opciones permiten especificar la longitud mínima y máxima del nombre del paquete, así como una expresión regular para validar nombres de paquetes (la expresión regular utiliza el formato estándar de java.util.regex).
