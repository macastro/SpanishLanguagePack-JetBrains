Informa sobre clases que no tienen un logger declarado.

Asegurar que cada clase tenga su propio logger dedicado es un paso importante para proporcionar una implementación de logging uniforme en la aplicación. Esta inspección no informa sobre interfaces, enumeraciones, anotaciones, clases internas ni clases abstractas.

Por ejemplo:


  public class NoLoggerDeclared {
    int calculateNthDigitOfPi(int n) {
      // todo
      return 1;
    }
  }

Se pueden especificar los nombres de las clases logger usando la tabla en la sección de Options. Esta inspección informará sobre las clases que no tengan declarado un campo de alguno de los tipos especificados.
