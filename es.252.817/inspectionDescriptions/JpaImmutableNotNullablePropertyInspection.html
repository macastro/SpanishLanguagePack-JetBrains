Informa sobre los usos de propiedades inmutables en entidades JPA validadas.

Detecta propiedades inmutables anotadas con @NotNull. Sugiere hacerlas mutables para garantizar la compatibilidad con Hibernate.

Fragmento de código incrustado: 


@Entity
open class User(
  @Id
  val id: Long = 0L,
  @NotNull
  val name: String = "",
)
Después de aplicar la corrección rápida: 

@Entity
open class User(
  @Id
  val id: Long = 0L,
  @NotNull
  var name: String = "",
)
