Reporta cualquier cadena de declaraciones if-else donde todas las condiciones son expresiones instanceof o expresiones de igualdad de clase (por ejemplo, comparaciones con String.class). Tales estructuras generalmente indican un fallo en el diseño orientado a objetos, que requiere que este tipo de distribución basada en tipos se realice a través de llamadas a métodos polimórficos en lugar de cadenas explícitas de comprobaciones de tipo.

Ejemplo:


  double getArea(Shape shape) {
    // Advertencia: Fallo de abstracción.
    // Es mejor declarar getArea() como método abstracto
    // en la interfaz shape e implementarlo
    // en cada clase heredada.
    if (shape instanceof Point) {
      return 0;
    }
    if (shape instanceof Circle) {
      return Math.PI *
        Math.pow(((Circle) shape).radius(), 2);
    }
    if (shape instanceof Rectangle) {
      return ((Rectangle) shape).width() *
        ((Rectangle) shape).height();
    }
    throw new IllegalArgumentException();
  }
