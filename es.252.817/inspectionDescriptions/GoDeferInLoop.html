Reporta las declaraciones defer dentro de bucles.

El uso de defer dentro de bucles puede provocar fugas de memoria o un orden de ejecución impredecible de las declaraciones.

Ejemplo:

func main() {
 for {
  field, err := db.Query("SELECT 1")
  if err != nil {
   // ...
  }
  defer field.Close()
  // ...
 }
}

La llamada a defer row.Close() dentro del bucle no se ejecutará hasta que la función termine su ejecución, no al final de cada iteración del bucle for. Esta implementación puede causar desbordamiento de pila de la función y otros problemas.
