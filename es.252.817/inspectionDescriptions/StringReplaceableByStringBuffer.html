Informa sobre variables declaradas como java.lang.String que se concatenan repetidamente. Estas variables pueden declararse de manera más eficiente como java.lang.StringBuffer o java.lang.StringBuilder.

Ejemplo:


  String s = "";
  for (int i = 0; i < names.length; i++) {
    String name = names[i] + (i == names.length - 1 ? "" : " ");
    s = s + name;
  }

Este tipo de bucle puede reemplazarse por:


  StringBuilder s = new StringBuilder();
  for (int i = 0; i < names.length; i++) {
    String name = names[i] + (i == names.length - 1 ? "" : " ");
    s.append(name);
  }

O incluso puede reemplazarse por:


  String s = String.join(" ", names);

Utilice esta opción para que la inspección solo informe cuando la variable se concatene dentro de un bucle.
