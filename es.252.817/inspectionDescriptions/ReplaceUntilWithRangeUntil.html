El informe puede ser reemplazado por el operador ..< en lugar de until.

Cada reemplazo de until por ..< no altera la semántica de ninguna manera.

Los estudios de experiencia de usuario muestran que los desarrolladores cometen aproximadamente 20-30% menos errores al leer código con ..< en comparación con until.

Ejemplo:


  fun main(args: Array<String>) {
      for (index in 0 until args.size) {
          println(index)
      }
  }

Después de aplicar la corrección rápida:


  fun main(args: Array<String>) {
      for (index in 0..<args.size) {
          println(index)
      }
  }

Esta inspección solo se informa cuando el nivel del lenguaje Kotlin del proyecto o módulo es 1.7 o superior.
