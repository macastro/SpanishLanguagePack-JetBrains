Reporta campos que están explícitamente inicializados con sus valores predeterminados.

Ejemplo:


  class Foo {
    int foo = 0;
    List bar = null;
  }

Después de aplicar la corrección rápida:


  class Foo {
    int foo;
    List bar;
  }

Usando la configuración de inspección, se pueden reportar solo las inicializaciones explícitas a null, por ejemplo: 


  class Foo {
    int foo = 0;     // sin advertencia
    List bar = null; // advertencia de inicialización redundante de campo
  }
