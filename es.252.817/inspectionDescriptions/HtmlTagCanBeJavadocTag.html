Informa sobre el uso de la etiqueta <code> en comentarios Javadoc. Desde Java 5, estas etiquetas pueden ser reemplazadas por la estructura {@code ...}. Esto permite usar los símbolos menor que < y mayor que > en lugar de entidades HTML en los comentarios.

Ejemplo:


  /**
   * @return empty <code>ArrayList&lt;Integer&gt;</code>
   */
  List<Integer> getList(){ ... }

Después de aplicar la corrección rápida:


  /**
   * @return empty {@code ArrayList<Integer>}
   */
  List<Integer> getList(){ ... }
