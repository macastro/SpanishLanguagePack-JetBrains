Informa de literales de cadena que usan caracteres nacionales sin el prefijo N.

Sin el prefijo N, la cadena se convierte a la página de códigos predeterminada de la base de datos. Puede que esta página de códigos predeterminada no reconozca determinados caracteres. Para obtener más información, consulte nchar y nvarchar (Transact-SQL) en docs.microsoft.com.

Ejemplo (Microsoft SQL Server):

SELECT 'abcde' AS a;
SELECT N'abcde' AS b;
SELECT 'абвгд' AS c;
SELECT N'абвгд' AS d;

SELECT 'абвгд' AS c; no tiene el prefijo N, se resaltará la parte 'абвгд'.
