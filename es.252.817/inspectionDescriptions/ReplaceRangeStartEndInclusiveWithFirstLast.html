Reporta las propiedades empaquetadas Range.start y Range.endInclusive.

Estas propiedades pueden ser reemplazadas por las propiedades no empaquetadas first y last para evitar llamadas redundantes.

Esta corrección rápida reemplazará las propiedades start y endInclusive por sus equivalentes first y last.

Ejemplo:


  fun foo(range: CharRange) {
      val lastElement = range.endInclusive
  }

Después de aplicar la corrección rápida:


  fun foo(range: CharRange) {
      val lastElement = range.last
  }
