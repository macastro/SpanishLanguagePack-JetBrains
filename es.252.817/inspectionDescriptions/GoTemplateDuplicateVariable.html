Informa sobre variables duplicadas de Go Template declaradas en el mismo ámbito.

Duplicar una variable reasigna con el mismo nombre la variable existente. Esta operación puede dar lugar a distintos problemas impredecibles.

Ejemplo:

{{$v := 0}}{{$v := 1}}{{$v}} es 0. {{/* se evalúa como '1 es 0' */}}
{{$v := 0}}{{$w := 1}}{{$v}} es 0. {{/* funciona como se esperaba */}}
