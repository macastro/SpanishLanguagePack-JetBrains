Informa sobre llamadas a toString() en objetos de clases que extienden Number. Estas llamadas generalmente son incorrectas en un entorno internacionalizado y deberían utilizar un formato específico según la configuración regional.

Ejemplo:


  void print(Double d) {
    System.out.println(d.toString());
  }
Una posible forma de corregir este problema es: 

  void print(Double d) {
    System.out.printf("%f%n", d);
  }
Esto utilizará la configuración regional predeterminada para formatear el número, la cual se establece durante el inicio de la JVM y se basa en el entorno del host.
