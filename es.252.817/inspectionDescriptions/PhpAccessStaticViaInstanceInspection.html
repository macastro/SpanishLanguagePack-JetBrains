Reporta referencias a métodos y campos estáticos por medio de una instancia de clase en vez de la clase misma.

A pesar de que PHP permite referencias a miembros estáticos a través de variables de instancia, esto vuelve confuso el código, ya que el lector podría pensar que el resultado del método depende de la instancia.

El arreglo rápido reemplaza la variable de instancia con el nombre de clase.

Ejemplo:


  class StaticClass {
    public static function someStaticFunction(): int
    {
        return 3;
    }

  }

  $var = new StaticClass
  $var::someStaticFunction();

Después de que se aplica el arreglo rápido:


  StaticClass::someStaticFunction()
