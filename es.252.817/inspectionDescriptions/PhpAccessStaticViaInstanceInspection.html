Reporta referencias a métodos y campos estáticos a través de instancias de clase en lugar de la clase misma.

Aunque la especificación de PHP permite referenciar miembros estáticos a través de variables de instancia, esto puede confundir a los lectores, quienes podrían asumir erróneamente que el método depende de la instancia.

Esta corrección rápida reemplazará la variable de instancia con el nombre de la clase.

Ejemplo:


  class StaticClass {
    public static function someStaticFunction(): int
    {
        return 3;
    }
  }
  $var = new StaticClass
  $var::someStaticFunction();

Después de aplicar la corrección rápida:


  StaticClass::someStaticFunction()
