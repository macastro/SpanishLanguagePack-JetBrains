El valor del atributo parameters en la anotación de Micronaut Cache contiene nombres de parámetros de método no definidos.

Se admiten las siguientes anotaciones:

  • io.micronaut.cache.annotation.Cacheable
  • io.micronaut.cache.annotation.CachePut
  • io.micronaut.cache.annotation.CacheInvalidate

Ejemplo:


  @CachePut(parameters = "myParameter") // Error: no se puede resolver el símbolo 'myParameter'
  public String cachedMethod(String name_is_not_equal_to_myParameter) {
    return "...";
  }
