Detecta referencias a métodos de cifrado inexistentes en la configuración del bloque de cifrado de OpenTofu. Para el siguiente código: 

    
      terraform {
        encryption {
          key_provider "pbkdf2" "foo" {
            # Configuration parameters
          }
          method "aes_gcm" "yourname2" {
            keys = key_provider.pbkdf2.foo
          }
        }
      }
Comprueba si el nombre del método es correcto.

Para más detalles, consulte https://opentofu.org/docs/language/state/encryption/#methods.
