Detecta referencias a métodos de cifrado inexistentes en la configuración de bloques de cifrado de OpenTofu. Para el siguiente código: 

    
      terraform {
        encryption {
          key_provider "pbkdf2" "foo" {
            # Configuration parameters
          }
          method "aes_gcm" "yourname2" {
            keys = key_provider.pbkdf2.foo
          }
        }
      }
Debemos verificar que los nombres de los métodos sean correctos.

Para más detalles, consulta https://opentofu.org/docs/language/state/encryption/#methods.
