Informa sobre expresiones return sin etiquetar dentro de lambdas en línea.

Estas expresiones pueden resultar confusas ya que puede no estar claro a qué ámbito pertenece el return.

La corrección rápida Cambiar a return@... está disponible para arreglar el código automáticamente.

Ejemplo:


  fun test(list: List<Int>) {
      list.forEach {
          // Esta expresión return retorna desde la función test
          // Se puede cambiar a return@forEach para modificar el ámbito
          if (it == 10) return
      }
  }

Después de aplicar la corrección rápida:


  fun test(list: List<Int>) {
      list.forEach {
          if (it == 10) return@test
      }
  }
