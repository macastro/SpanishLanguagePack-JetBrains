Informa sobre los siguientes elementos que pueden eliminarse de forma segura porque las inspecciones que afectan ya no son aplicables en este contexto:
  • anotaciones @SuppressWarning o
  • comentarios de línea // noinspection, o
  • comentarios JavaDoc /** noinspection */

Ejemplo:


public class C {
 // El símbolo ya es private,
 // pero la anotación aún permanece
  @SuppressWarnings({"WeakerAccess"})
 private boolean CONST = true;
 void f() {
    CONST = false;
  }
}
