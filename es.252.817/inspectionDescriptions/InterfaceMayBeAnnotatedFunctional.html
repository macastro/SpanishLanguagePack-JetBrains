Informa sobre interfaces que pueden ser anotadas con @FunctionalInterface.

Anotar una interfaz con @FunctionalInterface indica que es una interfaz funcional y no se pueden añadir más métodos abstract.

Ejemplo:


  interface FileProcessor {
    void execute(File file);
  }

Después de aplicar la corrección rápida:


  @FunctionalInterface
  interface FileProcessor {
    void execute(File file);
  }
