Informa de interfaces que pueden anotarse con @FunctionalInterface (disponible desde JDK 1.8).

Anotar una interfaz con @FunctionalInterface indica que la interfaz es funcional y que no se pueden añadir métodos abstract.

Ejemplo:


  interface FileProcessor {
    void execute(File file);
  }

Después de aplicar la corrección rápida:


  @FunctionalInterface
  interface FileProcessor {
    void execute(File file);
  }

Esta inspección solo informa si el nivel de lenguaje del proyecto o módulo es 8 o superior.
