Informa de parámetros de método redundantes que pueden reemplazarse con variables locales.

Si todos los usos locales de un parámetro han sido precedidos por asignaciones a ese parámetro, el parámetro puede eliminarse y sus usos pueden reemplazarse con variables locales. No tiene sentido tener dicho parámetro, ya que los valores que se le pasan se sobrescriben. Normalmente, el problema aparece como resultado de la refactorización.

Ejemplo:


  void test(int p) {
    p = 1;
    System.out.print(p);
  }

Tras aplicar la corrección rápida:


  void test() {
    int p = 1;
    System.out.print(p);
  }
