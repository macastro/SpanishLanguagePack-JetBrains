Informa sobre casos donde se intentan insertar valores NULL en columnas que solo aceptan valores NOT NULL.

Ejemplo (Microsoft SQL Server):

CREATE TABLE br2 (
id INT NOT NULL,
col1 NVARCHAR (20) NOT NULL,
col2 NVARCHAR (20) NOT NULL,
);
--
INSERT INTO br2 (id, col1, col2)
VALUES (1, NULL, NULL);

No se pueden insertar valores NULL en col1 y col2 porque están definidas como NOT NULL. Si ejecuta el script tal como está, recibirá un mensaje de error. Para corregir este código, reemplace los NULL en la sección VALUES con valores (por ejemplo, 42 y 'bird').

INSERT INTO br2 (id, col1, col2)
VALUES (1, 42, 'bird');
