Informa la creación de objetos Boolean.

Rara vez es necesario construir nuevos objetos Boolean, y puede ocasionar problemas de desempeño si se hace lo suficiente seguido. Además, los constructores de Boolean están obsoletos desde Java 9 y podrían eliminarse o volverse inaccecibles en futuras versiones de Java.

Ejemplo:


  Boolean b1 = new Boolean(true);
  Boolean b2 = new Boolean(str);

Después de aplicar la reparación rápida:


  Boolean b1 = Boolean.TRUE;
  Boolean b2 = Boolean.valueOf(str);
