Informa sobre la creación de objetos Boolean.

La creación de nuevos objetos Boolean es generalmente innecesaria y puede causar problemas de rendimiento si se hace con frecuencia. Además, el constructor Boolean está obsoleto desde Java 9 y podría ser eliminado o volverse inaccesible en futuras versiones de Java.

Ejemplo:


  Boolean b1 = new Boolean(true);
  Boolean b2 = new Boolean(str);

Después de aplicar la corrección rápida:


  Boolean b1 = Boolean.TRUE;
  Boolean b2 = Boolean.valueOf(str);
