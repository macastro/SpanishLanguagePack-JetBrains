Se reportan las siguientes violaciones a la especificación EJB:
  • El método no declara que lanza java.rmi.RemoteException
  • El tipo o los argumentos de retorno del método no son compatibles con RMI/IIOP
  • La interfaz de punto final del servicio EJB tiene declaraciones constantes

Ejemplo:


public interface EntityRemote extends javax.ejb.EJBObject{
  public int businessExample(String value) throws java.lang.RuntimeException; // Error: debería lanzar 'RemoteException'
}
