Reporta expresiones switch que no cubren todos los resultados posibles de la expresión coincidente.

Groovy no requiere que la expresión switch sea exhaustiva. Actúa como si se insertara una rama implícita default -> null. Puede causar nulos inesperados si un desarrollador olvida insertar ramas case necesarias.

Ejemplo:


  enum A { X, Y }

  def foo(A a) {
    def x = switch (a) { // reporta switch
      case A.X -> ...
    }
  }
