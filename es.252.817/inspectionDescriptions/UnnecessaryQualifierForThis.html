Informa sobre calificadores innecesarios de this o super.

Los calificadores en this o super utilizados para desambiguar referencias en el código pueden volverse innecesarios mediante refactorización automática y deberían eliminarse para mayor claridad.

Ejemplo:


  class Foo {
    void foo() {}
  }
  class Bar extends Foo {
    void bar() {
      Bar.super.foo();
    }
  }

Después de aplicar la corrección rápida:


  class Foo {
    void foo() {}
  }
  class Bar extends Foo {
    void bar() {
      super.foo();
    }
  }
