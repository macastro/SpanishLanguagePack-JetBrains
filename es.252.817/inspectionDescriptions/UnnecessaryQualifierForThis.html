Reporta calificaciones innecesarias de this o super.

El uso de un calificador en this o super para desambiguar una referencia de código puede fácilmente volverse innecesario a través de refactorizaciones automáticas y debería eliminarse para mayor claridad.

Ejemplo:


  class Foo {
    void foo() {}
  }

  class Bar extends Foo {
    void bar() {
      Bar.super.foo();
    }
  }

Después de aplicar la solución rápida:


  class Foo {
    void foo() {}
  }

  class Bar extends Foo {
    void bar() {
      super.foo();
    }
  }
