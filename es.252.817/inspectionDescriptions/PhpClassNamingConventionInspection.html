Informa de los nombres de las clases que son demasiado cortos, demasiado largos o no siguen el patrón de expresiones regulares especificado. Algunos estilos de codificación tienen una convención de nomenclatura especial para las clases.

Utilice las opciones de inspección para especificar la longitud máxima/mínima y la expresión regular esperada para los nombres de las clases. Para ignorar la longitud de los nombres, especifique 0.

Para obtener más información sobre expresiones regulares, consulte la Quick Start guide (regular-expressions.info).
