Reporta el uso de Hooks.onOperatorDebug() cuando se llama a ReactorDebugAgent.init().

Si ReactorDebugAgent está en el classpath y spring.reactor.debug-agent.enabled es true (valor por defecto), ReactorDebugAgent.init() se llama automáticamente al inicio. Usar Hooks.onOperatorDebug() y ReactorDebugAgent.init() simultáneamente hace que Reactor genere el doble de frames de pila de depuración. Además, Hooks.onOperatorDebug() causa sobrecarga de rendimiento.

Esta corrección rápida elimina la llamada a Hooks.onOperatorDebug().

Ejemplo:


  public void hook() {
    Hooks.onOperatorDebug(); // Reporta 'Llamada a Hooks.onOperatorDebug() durante la inicialización de ReactorDebugAgent'
  }

Después de aplicar la corrección rápida:


  public void hook() {
  }

Esta inspección solo se activa cuando la versión de Spring Boot del proyecto o módulo es 2.2.0 o superior.
