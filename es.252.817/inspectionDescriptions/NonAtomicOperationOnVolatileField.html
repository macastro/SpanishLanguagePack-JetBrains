Informa sobre operaciones no atómicas en campos volátiles.

Un ejemplo de una operación no atómica es la actualización del campo utilizando el operador de incremento. Como la operación implica lectura y escritura, y pueden ocurrir otras modificaciones entre medias, los datos pueden corromperse. La operación puede hacerse atómica rodeándola con un bloque synchronized o utilizando una de las clases del paquete java.util.concurrent.atomic.

Ejemplo:


  private volatile int v = 1;

  void foo() {
    v = 2 * v;
  }
