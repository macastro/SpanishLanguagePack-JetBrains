Informa sobre operaciones no atómicas en campos volatile.

Un ejemplo de operación no atómica es actualizar un campo usando operadores de incremento. Como la operación involucra lectura y escritura, y otras modificaciones pueden ocurrir entre ellas, los datos pueden corromperse. La operación se puede hacer atómica envolviendo el código en un bloque synchronized o usando una de las clases del paquete java.util.concurrent.atomic.

Ejemplo:


  private volatile int v = 1;
  void foo() {
    v = 2 * v;
  }
