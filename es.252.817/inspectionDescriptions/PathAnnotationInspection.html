Informa sobre el uso incorrecto de las anotaciones de ruta.

Esta inspección detecta los casos en los que una cadena con una anotación de ruta se utiliza en un contexto que requiere otra anotación de ruta. Ayuda a garantizar el uso correcto de las anotaciones @MultiRoutingFileSystemPath y @NativePath.

La inspección resalta los siguientes problemas:

  • Cuando una cadena anotada con @NativePath se utiliza en un constructor Path o en un método de fábrica.
  • Cuando una cadena anotada con @NativePath se pasa a un parámetro de método anotado con @MultiRoutingFileSystemPath.
  • Cuando una cadena anotada con @MultiRoutingFileSystemPath se pasa a un parámetro de método anotado con @NativePath.
  • Cuando se utiliza un literal de cadena en un contexto que requiere cualquiera de las dos anotaciones.

Se proporciona una solución rápida para añadir la anotación adecuada donde sea necesario.

Esta inspección ayuda a prevenir errores en tiempo de ejecución que pueden producirse cuando se utilizan incorrectamente rutas de diferentes formatos. La anotación @MultiRoutingFileSystemPath se utiliza para las rutas que deben funcionar en diferentes sistemas de archivos, mientras que la anotación @NativePath se utiliza para las rutas específicas del sistema de archivos nativo.
