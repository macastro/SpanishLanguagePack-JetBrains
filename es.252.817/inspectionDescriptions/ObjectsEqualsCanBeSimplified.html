Informa sobre llamadas a Objects.equals(a, b) donde se sabe estáticamente que el primer argumento no es null.

Si ambos argumentos son primitivos, estas llamadas se pueden reemplazar de forma segura por a.equals(b) o a == b.

Ejemplo:


  String defaultName = "default";
  boolean isDefault = Objects.equals(defaultName, name);

Después de aplicar la corrección rápida:


  String defaultName = "default";
  boolean isDefault = defaultName.equals(name);

Novedades de la versión 2018.3
