Informa sobre llamadas a métodos de java.util.Collection o java.util.Map donde el argumento es la propia colección o mapa. Esto puede ocurrir en código con tipos sin procesar debido a operaciones de copiar y pegar.

Ejemplo:


  ArrayList list = new ArrayList<>();
  list.add(list); // Aquí hay una advertencia
  return list.hashCode(); // Lanza StackOverflowError
