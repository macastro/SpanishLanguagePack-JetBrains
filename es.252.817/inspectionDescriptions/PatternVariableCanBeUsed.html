Reporta declaraciones de variables locales que se pueden reemplazar con variables de patrón, que usualmente son más compactas.

Ejemplo:


  if (obj instanceof String) {
    String str = (String) obj;
    System.out.println(str);
  }

Se puede reemplazar con:


  if (obj instanceof String str) {
    System.out.println(str);
  }
