El informe indica que las declaraciones de variables locales pueden reemplazarse por variables de patrón, que suelen ser más concisas.

Ejemplo:


  if (obj instanceof String) {
    String str = (String) obj;
    System.out.println(str);
  }

Puede reemplazarse por:


  if (obj instanceof String str) {
    System.out.println(str);
  }
