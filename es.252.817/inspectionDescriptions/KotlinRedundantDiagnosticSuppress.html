Informa sobre las anotaciones @Suppress que pueden eliminarse de manera segura porque los diagnósticos del compilador que afectaban ya no son aplicables en este contexto.

Ejemplo:


  fun doSmth(@Suppress("UNUSED_PARAMETER") used: Int) {
    println(used)
  }

Después de aplicar la corrección rápida:


  fun doSmth(used: Int) {
    println(used)
  }
