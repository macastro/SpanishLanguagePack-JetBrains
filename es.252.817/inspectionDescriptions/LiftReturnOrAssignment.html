Reporta sentencias `if`, `when` y `try` que se pueden convertir en expresiones extrayendo la sentencia `return` o una asignación.

Ejemplo:


  fun foo(arg: Int): String {
      when (arg) {
          0 -> return "Zero"
          1 -> return "One"
          else -> return "Multiple"
      }
  }
Después de aplicar la corrección rápida: 

  fun foo(arg: Int): String {
      return when (arg) {
          0 -> "Zero"
          1 -> "One"
          else -> "Multiple"
      }
  }

Si desea que esta inspección resalte código más complejo con ramas de varias sentencias, deseleccione la opción "Reportar solo si cada rama es una sola sentencia".
