Informa sobre condiciones y expresiones que siempre devuelven true, false o null.

Ejemplo (MySQL):

CREATE TABLE t1 (a TEXT, b INT, c BOOLEAN);
SELECT a FROM t1 WHERE 'Cat' = 'Cat';
SELECT a FROM t1 WHERE 'Cat' = null;

'Cat' = 'Cat' siempre devuelve true, será reportado.

'Cat' = null siempre devuelve null, será reportado.
