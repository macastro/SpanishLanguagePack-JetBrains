Informa sobre variables static que se leen antes de su inicialización.

Esta inspección ignora las comprobaciones de igualdad con null.

Ejemplo:


  class Foo {
    public static int bar;
    public static void main(String[] args) {
      System.out.println(bar);
    }
  }

Tenga en cuenta que esta inspección utiliza un algoritmo de flujo de datos muy conservador y podría informar incorrectamente variables static como no inicializadas. Las variables informadas como inicializadas siempre estarán inicializadas.

Use la opción Ignorar campos primitivos para ignorar los campos primitivos no inicializados.
