Informa sobre llamadas a String.equals() o String.equalsIgnoreCase() que contienen literales de cadena como argumentos.

Algunos estándares de codificación especifican que los literales de cadena deben ser el calificador de equals() en lugar del argumento, para minimizar la posibilidad de NullPointerException.

Es posible intercambiar el literal y la expresión mediante una corrección rápida.

Ejemplo:


  boolean isFoo(String value) {
    return value.equals("foo");
  }

Después de aplicar la corrección rápida:


  boolean isFoo(String value) {
    return "foo".equals(value);
  }
