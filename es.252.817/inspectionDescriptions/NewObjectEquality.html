Informa el código que aplica == o != a un objeto recién asignado en lugar de llamar a equals().

Las referencias a objetos recién asignados no pueden apuntar a objetos existentes, por lo que la comparación siempre se evaluará como false. La inspección también puede informar objetos nuevos creados retornados desde métodos simples.

Ejemplo:


  void test(Object obj) {
    if (new Object() == obj) {...}
  }

Después de que se aplica la corrección rápida:


  void test(Object obj) {
    if (new Object().equals(obj)) {...}
  }
