Informa sobre intentos de acceder a la columna 0 de java.sql.ResultSet o java.sql.PreparedStatement. Por razones históricas, las columnas en java.sql.ResultSet y java.sql.PreparedStatement no comienzan en 0, sino en 1, y el acceso a la columna 0 es un error común en la programación JDBC.

Ejemplo:


  String getName(ResultSet rs) {
    return rs.getString(0);
  }
