Informa sobre llamadas a java.util.concurrent.locks.Condition.await() que no se realizan dentro de un bucle.

await() y métodos relacionados se utilizan típicamente para suspender un hilo hasta que una condición se vuelva verdadera. El hilo puede despertarse por diferentes razones, por lo que la condición debe verificarse después de que la llamada await() retorne. Esto se logra fácilmente usando un bucle.

Ejemplo:


  void acquire(Condition released) throws InterruptedException {
    released.await();
  }

El código correcto debería ser similar a:


  void acquire(Condition released) throws InterruptedException {
    while (acquired) {
      released.await();
    }
  }
