Reporta los m√©todos del repositorio CRUD de Spring Data para los cuales el Spring Data Query builder no puede generar la consulta correspondiente.

Ejemplo:


interface PersonRepository extends Repository<Person, Long> {
  List<Person> findByEmailAddressAndLastname(EmailAddress emailAddress, String lastname);
  List<Person> findByEmailDesconocidoAndLastname(EmailAddress emailAddress, String lastname); // No se puede resolver la propiedad 'Desconocido'
  List<Person> findByAndLastname(EmailAddress emailAddress, String lastname); // Falta el nombre de la propiedad

  List<Person> findDistinctPeopleByLastnameOrFirstname(String lastname, String firstname);
  List<Person> findPeopleDistinctByDesconocidoOrFirstname(String lastname, String firstname); // No se puede resolver la propiedad 'Desconocido'

  List<Person> findByLastnameOrderByFirstnameAsc(String lastname);
  List<Person> findByLastnameOrderByDesconocidoDesc(String lastname);  // No se puede resolver la propiedad 'Desconocido'
}
