Informa los casos en los que el intérprete de Python no está configurado como un núcleo de notebook. Esta inspección previene discrepancias entre Código Insight y el comportamiento de tiempo de ejecución de las celdas de código del cuaderno Jupyter.

Si ves la siguiente advertencia, el núcleo del servidor recién configurado no coincide con tu intérprete de Python actual. Por ejemplo, podría tener un conjunto diferente de paquetes instalados.

Puedes registrar tu intérprete de Python dando clic en el correspondiente enlace en el banner de advertencia. Después de eso, puedes ver el intérprete en la lista de núcleos de servidor configurados.
