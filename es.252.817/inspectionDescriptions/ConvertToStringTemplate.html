Informa sobre la concatenación de cadenas que pueden convertirse en plantillas de cadena.

Se recomienda usar plantillas de cadena ya que hacen que el código sea más legible.

Ejemplo:


  fun example() {
      val capitals = mapOf("France" to "Paris", "Spain" to "Madrid")
      for ((country, capital) in capitals) {
          print(capital + " is a capital of " + country)
      }
  }

Después de aplicar la corrección rápida:


  fun example() {
      val capitals = mapOf("France" to "Paris", "Spain" to "Madrid")
      for ((country, capital) in capitals) {
          print("$capital is a capital of $country")
      }
  }
