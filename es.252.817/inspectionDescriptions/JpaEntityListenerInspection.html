Reporta los siguientes errores de especificación JPA:
  • Clases de listeners de entidades sin constructores públicos sin argumentos
  • Métodos de listeners de entidades con firmas incorrectas

Ejemplo:


  @Entity
  @EntityListeners(value = JavaEntityListener.class)
  class JavaEntity { ... }

  class JavaEntityListener {

    @PostLoad
    void postLoad(String parameter) { ... } // Error: el método 'postLoad' debería tomar un parámetro de tipo 'Object' aquí
  }
