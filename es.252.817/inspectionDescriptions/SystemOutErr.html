Reporta el uso de System.out o System.err.

Estas declaraciones se utilizan típicamente para depuración temporal y deberían eliminarse del código de producción o reemplazarse por herramientas de registro más robustas.

La corrección rápida proporcionada es compatible con SLF4J y Log4j 2. Reemplaza las llamadas a System.out y System.err por llamadas de registro

Ejemplo:


  public static void test(Object o) {
    System.out.println("Test: "+ o);
  }

Después de aplicar la corrección rápida:


  private static final Logger log = LoggerFactory.getLogger(Test.class);
  public static void test(Object o) {
      log.info("Test: {}", o);;
  }

La opción Método de registro para 'System.err' permite especificar el método utilizado para registrar mensajes para llamadas a 'System.err'. La opción Método de registro para 'System.out' permite especificar el método utilizado para registrar mensajes para llamadas a 'System.out'.
