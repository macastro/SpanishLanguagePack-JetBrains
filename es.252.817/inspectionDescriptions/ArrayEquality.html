Informa sobre el uso de los operadores == y != para comparar la igualdad de arrays. En la mayoría de los casos, cuando se necesita comparar el contenido de los arrays, esto se puede lograr usando el método java.util.Arrays.equals().

Se recomienda usar la corrección rápida para reemplazar == por java.util.Arrays.equals().

Ejemplo:


  void foo(Object[] x, Object[] y) {
    boolean comparison = x == y;
  }

Después de aplicar la corrección rápida:


  void foo(Object[] x, Object[] y) {
    boolean comparison = Arrays.equals(x, y);
  }
