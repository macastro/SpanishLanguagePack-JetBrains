Son detectadas las siguientes violaciones de la especificación EJB a nivel de error:
  • Finder y los métodos selectores no están definidos correctamente
  • Los accesadores de campo CMP y CMR no están definidos correctamente

Ejemplo:


public class SessionEJB implements javax.ejb.SessionBean {
  // Error: El método 'ejbCreateXXX' debe ser público
  protected void ejbCreateExample() { ... }

  // Error: El método 'ejbCreateAnotherExample' debe devolver 'void'
  public int ejbCreateAnotherExample() { ... }
}
