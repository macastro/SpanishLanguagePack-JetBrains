Detecta las siguientes violaciones de la especificación EJB de nivel de error:
  • Métodos Finder y Selector definidos incorrectamente
  • Accesores de campos CMP y CMR definidos incorrectamente

Ejemplo:


public class SessionEJB implements javax.ejb.SessionBean {
  // Error: el método 'ejbCreateXXX' debe ser public
  protected void ejbCreateExample() { ... }
  // Error: el método 'ejbCreateAnotherExample' debe devolver 'void'
  public int ejbCreateAnotherExample() { ... }
}
