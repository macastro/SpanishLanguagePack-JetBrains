Informa sobre beans gestionados que tienen campos de instancia públicos con un ámbito diferente al predeterminado @Dependent. Según la especificación CDI, los ámbitos contextuales (@RequestScoped, @ApplicationScoped, @SessionScoped y ConversationScoped) no están permitidos para beans gestionados que tienen campos de instancia públicos.

Ejemplo:


  @ApplicationScoped // Advertencia: Un bean con ámbito no predeterminado no debe tener campos públicos
  class Car {
    public String engine;
  }
