Informa sobre beans manejados con campos de instancia públicos que no están en el ámbito @Dependent predeterminado. Según la especificación de CDI, los ámbitos contextuales (@RequestScoped, @ApplicationScoped, @SessionScoped y ConversationScoped) no están permitidos para beans manejados con campos de instancia públicos.

Ejemplo:


  @ApplicationScoped // Advertencia: El bean con ámbito no predeterminado no debe tener campos públicos
  class Car {
    public String engine;
  }
