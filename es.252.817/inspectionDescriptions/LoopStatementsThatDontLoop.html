Informa cualquier instancia de las sentencias for, while y do cuyos cuerpos se ejecutarán una sola vez a lo sumo. Normalmente, esto es un indicio de un error.

Use la opción Ignorar bucles for mejorados para ignorar los bucles foreach. A veces se usan para realizar una acción únicamente en el primer elemento de un elemento iterable de forma compacta.

Ejemplo:


  for (String s : stringIterable) {
    doSomethingOnFirstString(s);
    break;
  }
