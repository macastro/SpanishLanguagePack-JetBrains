Informa los modificadores que no siguen el orden recomendado en la guía de estilos.

El arreglo rápido Ordenar modificadores puede utilizarse para modificar el código automáticamente.

Ejemplos:


  private inline fun correctOrder(f: () -> Unit) {} // <== Ok

  infix private fun Int.wrongOrder(expr: Int) {} // <== Orden incorrecto, el arreglo rápido modifica los modificadores a "private infix"
