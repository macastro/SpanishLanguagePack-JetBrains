Informa sobre las directivas include que no se utilizan o no son necesarias y que se pueden eliminar de forma segura.
Detectar directivas include completamente sin usar
Una directiva include se considera sin usar si ninguna de sus declaraciones se utiliza en la unidad de traducción.
Detectar directivas include no utilizadas directamente
Una directiva include se considera sin usar si sus declaraciones no se utilizan directamente en el archivo. Funciona mejor con la política "Incluir lo que usas".
Detectar directivas include innecesarias
Detecta el conjunto mínimo de directivas include necesarias para compilar el archivo. Todas las demás directivas #include se consideran sin usar.

Utilice las casillas de verificación a continuación para ejecutar la inspección en archivos de encabezado.
