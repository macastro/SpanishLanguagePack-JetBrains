Las comprobaciones de null pueden ser reemplazadas por llamadas a métodos static de Objects o Stream.

Ejemplo:


  if (message == null) {
    application.messageStorage().save(new EmptyMessage());
  } else {
    application.messageStorage().save(message);
  }

Después de aplicar la corrección rápida:


  application.messageStorage()
    .save(Objects.requireNonNullElseGet(message, () -> new EmptyMessage()));

Utilice la opción No alertar cuando el reemplazo sea más largo que el original para ignorar los casos donde el reemplazo excede la longitud del código original.

Novedades de la versión 2017.3
