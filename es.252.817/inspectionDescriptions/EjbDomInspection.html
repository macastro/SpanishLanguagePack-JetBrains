Informa sobre los siguientes problemas que pueden aparecer en ejb-jar.xml y otros descriptores XML utilizados para desplegar EJB en servidores Java EE:
  • Referencias a clases no instanciables
  • Referencias a clases que no extienden las clases requeridas
  • Referencias a clases con ámbito inadecuado
  • Etiquetas y valores de atributos vacíos
  • Etiquetas y valores de atributos que no coinciden con el esquema requerido (por ejemplo, identificadores Java)
  • Etiquetas que no contienen las subetiquetas o atributos requeridos
  • Etiquetas que definen objetos con nombres duplicados
