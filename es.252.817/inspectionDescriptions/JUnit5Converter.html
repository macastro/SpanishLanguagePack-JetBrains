Informa sobre las pruebas de JUnit 4 que pueden migrarse automáticamente a JUnit 5. Aunque el ejecutor predeterminado puede convertirse automáticamente, los ejecutores personalizados, las reglas de método y las reglas de campo no pueden convertirse automáticamente y requieren cambios manuales.

Ejemplo:


  import org.junit.Assert;
  import org.junit.Test;
  public class RelevantTest {
    @Test
    public void testIt() {
      Assert.assertEquals("expected", "actual");
    }
  }

Después de aplicar la corrección rápida:


  import org.junit.jupiter.api.Assertions;
  import org.junit.jupiter.api.Test;
  public class RelevantTest {
    @Test
    public void testIt() {
      Assertions.assertEquals("expected", "actual");
    }
  }

Esta inspección requiere que la biblioteca JUnit 5 esté disponible en el classpath y que el proyecto esté configurado con JDK 1.8 o superior.
