Informa sobre llamadas a wait() realizadas en objetos java.util.concurrent.locks.Condition. Esto podría ser un error de programación, ya que se debería usar alguna variante del método await().

Ejemplo:


  void acquire(Condition released) throws InterruptedException {
    while (acquired) {
      released.wait();
    }
  }

El código correcto debería ser similar a:


  void acquire(Condition released) throws InterruptedException {
    while (acquired) {
      released.await();
    }
  }
