Informa sobre literales de cadena que están duplicados en todo el proyecto sin modificaciones. Proporciona dos correcciones rápidas. Una introduce una constante para la cadena duplicada y la utiliza en todo el proyecto, y la otra muestra las ubicaciones de todas las ocurrencias de un literal de cadena específico.

Ejemplo:


  class C1 { String CONST1 = "duplicate string"; }
  class C2 { String CONST2 = "duplicate string"; }

Configuración de la inspección:

  • Utilice el campo Longitud mínima de cadena para establecer la longitud mínima de cadena requerida para detectar duplicados.
  • Utilice la opción Ignorar expresiones @PropertyKey para ignorar las cadenas que se pasan como argumentos a métodos anotados con org.jetbrains.annotations.PropertyKey.
