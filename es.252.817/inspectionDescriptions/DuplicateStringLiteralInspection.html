Informa de literales de cadena que se repiten sin cambios a lo largo del proyecto. Se proporcionan dos arreglos rápidos. Uno, para introducir una constante para una cadena duplicada y utilizarla a lo largo del proyecto, y otro, para mostrar la ubicación de todos los duplicados de un literal de cadena determinado.

Ejemplo:


  class C1 { String CONST1 = "duplicate string"; }
  class C2 { String CONST2 = "duplicate string"; }

Configurar la inspección:

  • Utilizar el campo "Longitud mínima de la cadena" para establecer la longitud mínima de cadena necesaria para detectar los duplicados.
  • Utilizar la opción "Ignorar expresiones @PropertyKey" para ignorar las cadenas pasadas como argumentos a los métodos anotados con org.jetbrains.annotations.PropertyKey.
