Reporta claves obsoletas en archivos YAML.

La obsolescencia se comprueba sólo si existe un esquema JSON asociado con el archivo YAML correspondiente.

Ten en cuenta que el mecanismo de obsolescencia no está definido aún en la especificación de esquema JSON, y esta inspección usa una extensión no estándar deprecationMessage.

Ejemplo de esquema de obsolescencia:


  {
    "properties": {
      "SomeDeprecatedProperty": {
        "deprecationMessage": "Baz",
        "description": "Foo bar"
      }
    }
  }

El siguiente es un ejemplo con el mensaje de advertencia correspondiente:


  SomeDeprecatedProperty: some value
