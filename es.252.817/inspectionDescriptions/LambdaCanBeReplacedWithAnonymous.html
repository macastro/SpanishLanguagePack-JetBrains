Reporta expresiones lambda que pueden ser reemplazadas con clases anónimas.

Expandir expresiones lambda en clases anónimas puede ser útil si necesitas implementar otros métodos dentro de una clase anónima.

Ejemplo:


  s -> System.out.println(s)

Después de aplicar el arreglo rápido:

new Consumer<String>() {
  @Override
  public void accept(String s) {
    System.out.println(s);
  }
}
