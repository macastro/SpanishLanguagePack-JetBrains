Reportar enlaces TMS potencialmente inválidos.

Indica errores ortográficos en enlaces a elementos TMS con IDs que posiblemente hayan sido eliminados.

Posibles causas:

  • La anotación del enlace TMS no proporciona el atributo value o está vacío
  • Los datos del proyecto TMS no se han cargado.
  • El ID especificado no se encuentra entre los IDs de elementos TMS cargados
