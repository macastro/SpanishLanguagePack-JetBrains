Reporta las llamadas a java.util.Random.nextDouble() que se utilizan para crear números enteros positivos mediante la multiplicación de la llamada por un factor y la conversión a un número entero.

Usar java.util.Random.nextInt(int) para generar números enteros positivos aleatorios dentro de un rango es más simple y eficiente.

Ejemplo:


  int getRandomInt() {
    return (int) ((new Random()).nextDouble() * SIZE);
  }

Después de aplicar el arreglo rápido:


  int getRandomInt() {
    return (new Random()).nextInt(SIZE);
  }
