Informa sobre las llamadas a java.util.Random.nextDouble() que se utilizan para crear números enteros positivos multiplicando la llamada por un coeficiente y convirtiéndola a entero.

Para generar números enteros positivos aleatorios dentro de un rango, java.util.Random.nextInt(int) es más simple y eficiente.

Ejemplo:


  int getRandomInt() {
    return (int) ((new Random()).nextDouble() * SIZE);
  }

Después de aplicar la corrección rápida:


  int getRandomInt() {
    return (new Random()).nextInt(SIZE);
  }
