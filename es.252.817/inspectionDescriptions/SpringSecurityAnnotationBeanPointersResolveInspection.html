Reporta los Spring Beans no resueltos que se referencian en las anotaciones de Spring Security.

Ejemplos:

  • @WithUserDetails(value = "admin", userDetailsServiceBeanName = "unknownUserDetailsService")
      // Se reportará "No se puede resolver el Bean 'unknownUserDetailsService'" si 'unknownUserDetailsService' no está definido en el modelo Spring
    
  • @WithUserDetails(value = "admin", userDetailsServiceBeanName = "jpaUserDetailsService")
      // "El Bean debe ser de tipo 'org.springframework.security.core.userdetails.UserDetailsService'"
      // Se reportará si el Bean "jpaUserDetailsService" no es de tipo 'UserDetailsService'
