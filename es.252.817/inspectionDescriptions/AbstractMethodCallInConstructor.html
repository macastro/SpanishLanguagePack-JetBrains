Informa sobre llamadas a métodos abstract de la clase actual durante la construcción del objeto.

Un método se llama durante la construcción del objeto si está ubicado en:

  • Un constructor
  • Un inicializador de instancia no static
  • Un inicializador de campo no static
  • El método clone()
  • El método readObject()
  • El método readObjectNoData()

Este tipo de llamadas pueden causar errores difíciles de rastrear, ya que la inicialización del objeto puede ocurrir después de la llamada al método.

Ejemplo:


  abstract class Parent {
    abstract void abstractMethod();
  }
  class Child extends Parent {
    Child() {
      abstractMethod();
    }
  }

Esta inspección comparte funcionalidad con las siguientes inspecciones:

  • Método sobreescribible llamado durante la construcción del objeto
  • Método sobreescrito llamado durante la construcción del objeto

Solo se puede habilitar una inspección a la vez para evitar advertencias duplicadas.
