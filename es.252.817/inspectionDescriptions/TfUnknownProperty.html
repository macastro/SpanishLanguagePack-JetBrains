Inspección de propiedades desconocidas en bloques

Ejemplo de problema:


resource "awscc_appflow_flow" "aws_flow1" {
  flow_name          = ""
  source_flow_config = ""
  trigger_config     = ""
  destination_flow_config_list {}
  tasks {}
  test = ""
}

En este ejemplo, la propiedad 'test' en el bloque es desconocida.

Ejemplo de corrección:


resource "awscc_appflow_flow" "aws_flow1" {
  flow_name          = ""
  source_flow_config = ""
  trigger_config     = ""
  destination_flow_config_list {}
  tasks {}
}
