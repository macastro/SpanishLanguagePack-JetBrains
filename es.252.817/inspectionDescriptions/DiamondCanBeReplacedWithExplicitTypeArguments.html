Reporta la instanciación de clases genéricas donde se usa el símbolo <> (diamante) en lugar de parámetros de tipo explícitos.

La corrección rápida reemplazará el <> (diamante) con parámetros de tipo explícitos.

Ejemplo:

List <String> list = new ArrayList<>()

Después de aplicar la corrección rápida:

List <String> list = new ArrayList<String>()

El operador diamante fue introducido en Java 7. Esta inspección ayuda a degradar la compatibilidad con versiones anteriores de Java.
