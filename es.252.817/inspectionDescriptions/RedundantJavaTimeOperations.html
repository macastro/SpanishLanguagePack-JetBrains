Informe sobre operaciones redundantes en objetos 'java.time':
  • Creación de objetos de fecha/hora del paquete java.time de JDK cuando se puede usar una llamada a método más simple o evitar la creación.
  • Llamadas a métodos de java.time con java.time.temporal.ChronoField y java.time.temporal.ChronoUnit como argumentos que pueden ser reemplazadas por métodos más específicos.
  • Comparaciones de java.time con llamadas a compareTo() que pueden ser reemplazadas por llamadas a isAfter(), isBefore() o isEqual().

Ejemplos:

- Antes: 

LocalDateTime now = LocalDateTime.now();
return LocalDateTime.from(now);

Después de aplicar la corrección rápida:


LocalDateTime now = LocalDateTime.now();
return now;
- Antes: 

LocalTime localTime = LocalTime.now();
int minute = localTime.get(ChronoField.MINUTE_OF_HOUR);

Después de aplicar la corrección rápida:


LocalTime localTime = LocalTime.now();
int minute = localTime.getMinute();
- Antes: 

  LocalDate date1 = LocalDate.now();
  LocalDate date2 = LocalDate.now();
  boolean t = date1.compareTo(date2) > 0;

Después de aplicar la corrección rápida:


  LocalDate date1 = LocalDate.now();
  LocalDate date2 = LocalDate.now();
  boolean t = date1.isAfter(date2);
