Informa de operaciones redundantes en objetos 'java.time':
  • Creación de objetos de fecha/hora del paquete JDK java.time cuando se puede utilizar una llamada a un método más simple o se puede evitar su creación.
  • Llamadas a métodos de java.time que toman java.time.temporal.ChronoField y java.time.temporal.ChronoUnit como argumentos cuando se pueden reemplazar por llamadas a métodos más específicos.
  • Comparaciones de java.time mediante llamadas a compareTo() que pueden reemplazarse por llamadas a isAfter(), isBefore() o isEqual().

Ejemplo:

- Antes: 

LocalDateTime now = LocalDateTime.now();
return LocalDateTime.from(now);

Después de aplicar la corrección rápida:


LocalDateTime now = LocalDateTime.now();
return now;
- Antes: 

LocalTime localTime = LocalTime.now();
int minute = localTime.get(ChronoField.MINUTE_OF_HOUR);

Después de aplicar la corrección rápida:


LocalTime localTime = LocalTime.now();
int minute = localTime.getMinute();
- Antes: 

  LocalDate date1 = LocalDate.now();
  LocalDate date2 = LocalDate.now();
  boolean t = date1.compareTo(date2) > 0;

Después de aplicar la corrección rápida:


  LocalDate date1 = LocalDate.now();
  LocalDate date2 = LocalDate.now();
  boolean t = date1.isAfter(date2);
