Informa sobre llamadas a las funciones eval(), setTimeout() o setInterval() o la asignación de objetos Function. Estas funciones se utilizan para ejecutar cadenas de texto JavaScript arbitrarias que generalmente se generan de forma dinámica. Esto puede resultar muy confuso y puede presentar riesgos de seguridad.

Ignora los casos en los que se proporcionan funciones de callback a estos métodos de forma estática sin generar código.
