Reporta campos anotados como no-nulos que no están inicializados en el constructor.

Ejemplo:

public class MyClass {
  private @NotNull String value;

  public void setValue(@NotNull String value) {
    this.value = value;
  }

  public @NotNull String getValue() {
    return value;
  }
}

Estos campos pueden violar la restricción no-nula. En el ejemplo anterior, el parámetro setValue está anotado como no-nulo, pero getValue puede devolver nulo si no se llamó al setter.
