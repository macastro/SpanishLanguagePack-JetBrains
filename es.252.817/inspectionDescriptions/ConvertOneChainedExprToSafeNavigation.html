El informe sugiere simplificar las llamadas en cadena utilizando el operador de navegación segura (&.). Una secuencia de llamadas en cadena es un patrón común para verificar nil en cada componente de la llamada.

Ejemplo:

# 不良做法
if user && user.profile && user.profile.thumbnails
end
Se puede utilizar la corrección rápida para simplificar la expresión usando &.. Después de aplicar la corrección rápida, el resultado se verá así: 
# 优良做法
if user&.profile&.thumbnails
end
