Informa sobre los casos en que se colocan sobrecargas en archivos Python regulares después de la implementación o cuando sus firmas no son compatibles con la implementación.

Ejemplo:


from typing import overload


@overload
def foo(p1, p2): # La firma de sobrecarga no es compatible con la implementación
    pass


@overload
def foo(p1): # La firma de sobrecarga no es compatible con la implementación
    pass


def foo(p1, p2, p3):
    print(p1, p2, p3)
