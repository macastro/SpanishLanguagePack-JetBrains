Informa referencias incorrectas de beans en el parámetro scheduler de la anotación @Scheduled.

Ejemplo:


  @Component public class UserController {...}
  @Component public class UserServiceAsyncTaskScheduler extends SimpleAsyncTaskScheduler {...}

  @Component public class UserService {
    @Scheduled(scheduler="userServiceAsyncTaskScheduler", cron="1 * * * * * ?")
    public void scheduleTask() {...}

    @Scheduled(scheduler="userController", cron="1 * * * * * ?") // informa "El bean debe ser uno de estos tipos: TaskScheduler,ScheduledExecutorService "
    public void scheduleTask_2() {...}

    @Scheduled(scheduler="unknown", cron="1 * * * * * ?")   // informa "No se puede resolver el bean 'unknown'"
    public void scheduleTask_3() {...}
  }
