Informa sobre el uso de variables static no final durante la inicialización de clases.

En tales casos, la semántica del código puede depender del orden de creación de la clase. También puede provocar el uso de variables no inicializadas, lo que a menudo provoca errores difíciles de detectar y confusos.

Ejemplo:


  clase Foo {
    public static int bar = 0;

    static {
      System.out.println(bar);
    }
  }
