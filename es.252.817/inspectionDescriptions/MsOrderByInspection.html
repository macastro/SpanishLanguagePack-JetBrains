Informa usos cuando la cláusula ORDER BY se utiliza sin TOP, OFFSET o FOR XML en vistas, funciones en línea, tablas derivadas, subconsultas y expresiones de tabla comunes.

Para más información sobre usos de ORDER BY, ver SELECT - Cláusula ORDER BY (Transact-SQL) en docs.microsoft.com.

Ejemplo (Microsoft SQL Server):

CREATE TABLE foo (a INT NOT NULL, b  INT NOT NULL);

SELECT *
FROM (SELECT a, b
FROM foo A
WHERE a < 89
ORDER BY b) ALIAS;

En una subconsulta, ORDER BY se resaltará como un error. Se puede añadir TOP, OFFSET o FOR XML a una subconsulta. Alternativamente, utilice la corrección rápida Eliminar elemento para eliminar la sección ORDER BY.

Tras aplicar la corrección rápida:

SELECT *
FROM (SELECT a, b
FROM foo A
WHERE a < 89) ALIAS;
