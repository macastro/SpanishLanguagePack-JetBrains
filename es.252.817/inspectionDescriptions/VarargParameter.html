Reporta métodos que aceptan un número variable de parámetros (también conocidos como métodos vararg).

Ejemplo:


  enum EnumConstants {
  A(null), B, C;
  EnumConstants(String... ss) {}
}

El parámetro vararg puede ser reemplazado por un parámetro de array equivalente mediante una corrección rápida. Los argumentos correspondientes en las llamadas al método se envuelven en expresiones inicializadoras de array. Después de aplicar la corrección rápida:


  enum EnumConstants {
  A(null), B(new String[]{}), C(new String[]{});
  EnumConstants(String[] ss) {}
}

Los métodos varargs fueron introducidos en Java 5. Esta inspección ayuda a la degradación para mantener compatibilidad con versiones anteriores de Java.
