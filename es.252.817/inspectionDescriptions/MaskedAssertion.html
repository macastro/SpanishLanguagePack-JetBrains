Reporta sentencias assert y aserciones del marco de pruebas que son suprimidas por un bloque de captura circundante. Dichas aserciones nunca fallarán, ya que el AssertionError arrojado será capturado e ignorado silenciosamente.

Ejemplo 1:


  void javaAssertion() {
    try {
      ...
      assert 1 == 2;
    } catch (AssertionError e) {
      // la aserción es ignorada silenciosamente
    }
  }

Ejemplo 2:


  @Test
  void testWithAssertJ() {
    try {
      ...
      assertThat(1).as("test").isEqualTo(2);
    } catch (AssertionError e) {
      // la aserción es ignorada silenciosamente
    }
  }

Ejemplo 3:


  @Test
  void testWithJunit() {
    try {
      ...
      assertEquals(1, 2);
    } catch (AssertionError e) {
      // la aserción es ignorada silenciosamente
    }
  }
