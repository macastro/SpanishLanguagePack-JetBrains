La concatenación de String puede simplificarse reemplazándola con plantillas de cadenas.

Ejemplo:


  String name = "Bob";
  String greeting = "Hello, " + name + ". You are " + 29 + " years old.";

Después de aplicar la corrección rápida:


  String name = "Bob";
  String greeting = STR."Hello, \{name}. You are 29 years old.";
