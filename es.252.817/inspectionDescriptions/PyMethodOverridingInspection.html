Informa sobre inconsistencias en la firma de métodos sobreescritos.

Ejemplo:


class Book:
    def add_title(self):
        pass
class Novel(Book):
    def add_title(self, text):
        pass

El método add_title en la clase Novel tiene parámetros que no coinciden con la firma del método especificado en la clase Book. El IDE sugiere aplicar la refactorización "Cambiar firma" como solución.
