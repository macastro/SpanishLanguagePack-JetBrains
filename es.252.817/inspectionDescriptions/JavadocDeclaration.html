Reporta los siguientes problemas en comentarios y etiquetas Javadoc:
  • Nombre de etiqueta inválido
  • Descripción de etiqueta incompleta
  • Etiqueta duplicada
  • Descripción Javadoc faltante

Ejemplo:


  /**
   * Nombre de etiqueta inválido
   * @poram param description
   */
  public void sample(int param){
  }

Ejemplo:


  /**
   * Referencia a sí mismo {@link #sample(int)}
   */
  public void sample(int param){
  }

Esta corrección rápida agregará etiquetas Javadoc desconocidas a la lista de etiquetas adicionales definidas por el usuario.

Use el campo de texto a continuación para definir etiquetas Javadoc adicionales.

Use la primera casilla de verificación para ignorar las etiquetas 'throws' duplicadas.

Use la segunda casilla de verificación para ignorar problemas con la primera oración faltante o incompleta en la descripción.

Use la tercera casilla de verificación para ignorar las referencias a sí mismo.
