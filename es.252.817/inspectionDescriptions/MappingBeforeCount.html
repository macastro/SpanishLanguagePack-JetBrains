Informa sobre llamadas redundantes a la API de Stream, como map() o boxed() antes de una llamada a count().

Estas llamadas no alteran el conteo final y por lo tanto pueden eliminarse. El código podría depender de efectos secundarios en los lambda de estas llamadas map. Sin embargo, depender de efectos secundarios dentro de una cadena Stream es una práctica muy desaconsejable. No hay garantía de que estas llamadas no sean optimizadas en futuras versiones de Java.

Ejemplo:


  // Llamada map() redundante
  long count = list.stream().filter(s -> !s.isEmpty()).map(s -> s.trim()).count();
