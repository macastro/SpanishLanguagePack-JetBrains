Informa sobre declaraciones o expresiones switch de enumeración donde la rama default nunca será alcanzada porque todas las posibles valores están cubiertos por las ramas case.

Tales elementos son redundantes, especialmente para expresiones switch, ya que no se compilarán cuando las ramas case no cubran todas las constantes de enumeración.

El nivel de lenguaje debe configurarse en 14 para que se informen las expresiones switch.

La corrección rápida proporcionada puede eliminar la rama default.

Ejemplo:


  enum E { A, B }
  int foo(E e) {
    return switch (e) {
      case A -> 1;
      case B -> 2;
      default -> 3;
    };
  }

Después de aplicar la corrección rápida:


  enum E { A, B }
  int foo(E e) {
    return switch (e) {
      case A -> 1;
      case B -> 2;
    };
  }
