Informa sobre clases que implementan Serializable y no declaran un campo serialVersionUID.

Sin un campo serialVersionUID, cualquier cambio realizado en la clase provocará que las versiones previamente serializadas sean ilegibles.

Ejemplo:


  class Main implements Serializable {
  }

Después de aplicar la corrección rápida:


  class Main implements Serializable {
      private static final long serialVersionUID = -1446398935944895849L;
  }
Cuando se usa un nivel de lenguaje de JDK 14 o superior, la corrección rápida también agregará la anotación java.io.Serial.

Configure la inspección con las siguientes opciones:

  • Lista de clases cuyos herederos no deberían ser reportados por esta inspección. Esto es aplicable a clases que heredan Serializable de una superclase pero no se utilizan para serialización.
  • Si se deben ignorar las clases anónimas Serializable.
