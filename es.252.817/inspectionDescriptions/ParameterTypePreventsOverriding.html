Informa sobre métodos en subclases cuyos parámetros tienen el mismo nombre que los del método super pero pertenecen a diferentes paquetes. En estos casos, el método de la subclase no puede sobrescribir el método super.

Ejemplo:


public class A {
 public void method(Object o) {}
}
public class B extends A {
 public void method(Object o) {} // advertencia sobre el tipo de parámetro
 class Object {}
}

Después de aplicar la corrección rápida:


public class A {
 public void method(Object o) {}
}
public class B extends A {
 public void method(java.lang.Object o) {} // nuevo tipo de parámetro
 class Object {}
}
