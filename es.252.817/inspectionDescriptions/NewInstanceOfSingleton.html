Informa sobre la creación de nuevas instancias de clases anotadas con @groovy.lang.Singleton. Esta construcción puede provocar una excepción en tiempo de ejecución imposible instanciar el singleton.

Ejemplo:


  @Singleton
  class Foo{
  }
  Foo foo = new Foo()

Después de aplicar la corrección rápida:


  @Singleton
  class Foo{
  }
  Foo foo = Foo.instance
