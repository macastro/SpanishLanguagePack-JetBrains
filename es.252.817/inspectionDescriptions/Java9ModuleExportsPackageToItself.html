Los exports se dirigen al mismo módulo Java 9 que los define o a paquetes abiertos en ese módulo. Esta corrección rápida elimina tales directivas del archivo module-info.java.

Ejemplo:


  module com.mycomp {
    exports com.mycomp.main to com.mycomp;
  }

Después de aplicar la corrección rápida:


  module main {
  }
