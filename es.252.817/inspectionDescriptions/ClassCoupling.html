Reporte de clases que hacen referencia a demasiadas otras clases.

Las clases con alto acoplamiento pueden ser muy frágiles y deberían dividirse en clases más pequeñas.

Configuración de la inspección:

  • Utilice el campo Límite de acoplamiento de clase para especificar el máximo acoplamiento permitido para una clase.
  • Utilice la opción Incluir acoplamiento con clases del sistema Java para especificar si se deben contar las referencias a clases del sistema (aquellas en los paquetes java. o javax.).
  • Utilice la opción Incluir acoplamiento con clases de biblioteca para especificar si se deben contar las referencias a cualquier clase de biblioteca.
