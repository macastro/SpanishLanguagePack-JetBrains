Informa sobre declaraciones de arrays con sintaxis estilo C, donde los corchetes del array se colocan después del nombre de la variable o después de la lista de parámetros del método. La mayoría de los estilos de código prefieren las declaraciones de arrays estilo Java, donde los corchetes se colocan después del nombre del tipo.

Ejemplo:


  public String process(String value[])[] {
    return value;
  }

Después de aplicar la corrección rápida:


  public String[] process(String[] value) {
    return value;
  }

Configuración de la inspección:

Utilice la opción Ignorar declaraciones estilo C en variables para informar solo sobre declaraciones de arrays estilo C en tipos de retorno de métodos.
