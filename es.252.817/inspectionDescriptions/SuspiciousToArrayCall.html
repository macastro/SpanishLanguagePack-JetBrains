Informa sobre llamadas sospechosas a Collection.toArray().

Los siguientes tipos de llamadas se consideran sospechosas:

  • El tipo del argumento array no coincide con el tipo del array al que se convierte el resultado.
  • El tipo del argumento array no coincide con el parámetro de tipo en la declaración de la colección.

Ejemplo:


void m1(List list) {
  Number[] ns = (Number[]) list.toArray(new String[0]);
}
void m2(List<Number> list) {
  Number[] ns = list.toArray(new String[0]);
}
