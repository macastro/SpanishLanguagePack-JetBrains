Informa las asignaciones cuyo lado derecho sea una división que no debería truncarse como entero.

Aunque a veces es intencional, esta construcción es, con frecuencia, defectuosa.

Ejemplo:


  int x = 18;
  x *= 3/2; // no cambia x por el resultado de la división entera

Este código debería reemplazarse por: 


  int x = 18;
  x *= 3.0/2;

En las opciones de inspección, puede deshabilitar las advertencias para divisiones sospechosas, pero posiblemente correctas; por ejemplo, cuando el dividendo no puede calcularse estáticamente. 


  void calc(int d) {
     int x = 18;
     x *= d/2;
  }
