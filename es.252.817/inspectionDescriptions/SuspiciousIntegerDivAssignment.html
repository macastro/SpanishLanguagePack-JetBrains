Informa sobre asignaciones donde la división en el lado derecho no debería truncarse a enteros.

Aunque a veces es intencional, esta estructura generalmente indica un error.

Ejemplo:


  int x = 18;
  x *= 3/2; // no cambiará x debido al resultado de la división de enteros

Este código debería reemplazarse por: 


  int x = 18;
  x *= 3.0/2;

En las opciones de inspección, se pueden desactivar las advertencias para divisiones sospechosas pero potencialmente correctas, como cuando el dividendo no se puede calcular estáticamente. 


  void calc(int d) {
     int x = 18;
     x *= d/2;
  }
