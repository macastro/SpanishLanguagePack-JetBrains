Informa sobre herederos directos de clases `sealed` que no tienen estado ni método `equals()` anulado.

Se recomienda encarecidamente anular `equals()` para proporcionar estabilidad de comparación, o bien convertir la `clase` en un `objeto` para alcanzar el mismo efecto.

Ejemplo:


  sealed class Receptor {
      class Todos : Receptor()
      class Usuario(val id: Int) : Receptor()
  }

La corrección rápida convierte una `clase` en un `objeto`:


  sealed class Receptor {
      object Todos : Receptor()
      class Usuario(val id: Int) : Receptor()
  }
