Informa sobre herederos directos de clases sealed que no tienen estado y carecen del método equals() sobrescrito.

Se recomienda encarecidamente sobrescribir equals() para proporcionar estabilidad en las comparaciones, o convertir la class en un object para lograr el mismo efecto.

Ejemplo:


  sealed class Receiver {
      class Everyone : Receiver()
      class User(val id: Int) : Receiver()
  }

La corrección rápida convertirá la class en un object:


  sealed class Receiver {
      object Everyone : Receiver()
      class User(val id: Int) : Receiver()
  }
