Informa sobre llamadas a java.lang.Thread.sleep() que ocurren dentro de bucles.

Tales llamadas indican una "espera activa". La espera activa suele ser ineficiente y puede provocar deadlocks inesperados, ya que el hilo en espera activa no libera los recursos bloqueados.

Ejemplo:


  class X {
     volatile int x;
     public void waitX() throws Exception {
      while (x > 0) {
        Thread.sleep(10);// Advertencia: llamada a 'Thread.sleep()' en un bucle, posible espera activa
      }
    }
  }
