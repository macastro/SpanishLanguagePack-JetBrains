Informa las llamadas `async` con `await` inmediatamente después. Esas llamadas se pueden reemplazar por llamadas bloqueantes.

Ejemplo:


  suspend fun test(ctx: CoroutineContext, scope: CoroutineScope) {
      scope.async(ctx) { doSomeJob() }.await()
  }

Después de aplicar la reparación rápida:


  suspend fun test(ctx: CoroutineContext, scope: CoroutineScope) {
      withContext(scope.coroutineContext + ctx) { doSomeJob() }
  }
