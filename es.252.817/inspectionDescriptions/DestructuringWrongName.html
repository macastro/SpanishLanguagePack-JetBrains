Informa sobre las declaraciones de desestructuración donde los nombres de las propiedades no coinciden con los de la clase de datos.

Ejemplo:


  data class Foo(val a: String, val b: Int, val c: String)
  fun bar(f: Foo) {
      val (a, c) = f
  }

La corrección rápida cambia los nombres de las variables para que coincidan con los nombres de los campos de la clase:


  data class Foo(val a: String, val b: Int, val c: String)
  fun bar(f: Foo) {
      val (a, b) = f
  }
