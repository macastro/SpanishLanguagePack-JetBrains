Informa sobre el boxing explícito, es decir, el empaquetado de valores primitivos en objetos.

Para Java 5 y versiones posteriores, el boxing manual explícito es innecesario y puede eliminarse de forma segura.

Ejemplo:

  • Integer i = new Integer(1); → Integer i = Integer.valueOf(1);
  • Integer i = Integer.valueOf(1); → int i = 1;

La opción Informar solo sobre expresiones de boxing verdaderamente redundantes permite informar únicamente sobre los casos de boxing realmente redundantes, donde el valor empaquetado se desempaqueta inmediatamente de forma implícita o explícita. En este caso, se puede eliminar todo el proceso de boxing-unboxing. Esta inspección no informa sobre el simple boxing explícito.

Esta inspección solo informa cuando el nivel de lenguaje del proyecto o módulo es 5 o superior.
