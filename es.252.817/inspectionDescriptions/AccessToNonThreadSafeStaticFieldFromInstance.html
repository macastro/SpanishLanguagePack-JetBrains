Informa sobre accesos a campos static de tipos no seguros para hilos.

Cuando se accede a campos static desde métodos de instancia o bloques no sincronizados, múltiples hilos pueden acceder al campo. Esto puede provocar efectos secundarios no especificados, como excepciones y resultados incorrectos.

Ejemplo:


  class Sample {
    private static final SimpleDateFormat df = new SimpleDateFormat("yyyy-MM-dd");
    String method() {
      return df.format("");
    }
  }

Puede especificar qué tipos deben considerarse no seguros para hilos. Solo se informará sobre campos que tengan exactamente estos tipos o se inicialicen con estos tipos exactos, ya que pueden existir subclases seguras para hilos de estos tipos.
