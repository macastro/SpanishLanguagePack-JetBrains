Informa constantes con nombres que no coinciden con la convención especificada.

Las constantes son campos de un tipo inmutable declarados con los modificadores static y final.

Informa constantes cuyos nombres son demasiado cortos, demasiado largos o que no siguen el patrón de expresión regular especificado.

Configurar la inspección:

  • Utiliza el campo patrón para especificar java.util.regex.Pattern que se espera que coincida con un nombre de constante.
  • Utiliza el campo longitud mínima para especificar la longitud mínima de un nombre de constante.
  • Utiliza el campo longitud máxima para especificar la longitud máxima de un nombre de constante.
