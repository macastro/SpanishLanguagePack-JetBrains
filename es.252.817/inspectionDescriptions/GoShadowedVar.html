Informa sobre declaraciones de variables que se superponen con declaraciones en un ámbito externo.

Esta situación puede causar confusión y consecuencias inesperadas, ya que el significado de las variables depende del ámbito.

Ejemplo:

for i := 0; i < len(nums); i++ {
    for i := 0; i < len(nums); i++ {
    }
  }

La variable i en el bucle anidado está oculta. Para eliminar el ocultamiento, considere renombrar la variable en el bucle anidado.

for i := 0; i < len(nums); i++ {
    for j := 0; j < len(nums); j++ {
    }
  }
