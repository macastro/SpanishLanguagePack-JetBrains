Informa sobre métodos equals() que tienen un tipo de parámetro diferente a java.lang.Object si la clase contenedora no tiene otras sobrecargas de equals() que usen java.lang.Object como tipo de parámetro.

Las versiones covariantes de equals() no sobrescriben el método Object.equals(Object). Esto puede llevar a comportamientos inesperados en tiempo de ejecución. Por ejemplo, cuando la clase se utiliza para construir una clase de colección estándar que espera que se sobrescriba el método Object.equals(Object).

Ejemplo:


  class Foo {
      public boolean equals(Foo foo) {  // advertencia
          return false;
      }
  }
  class Bar {
      public boolean equals(Bar bar) {  // sin advertencia aquí
          return false;
      }
      @Override
      public boolean equals(Object obj) {
          return false;
      }
  }
