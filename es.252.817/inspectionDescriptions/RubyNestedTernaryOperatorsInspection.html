Reporta operadores ternarios anidados. Los operadores ternarios anidados pueden ser difíciles de leer, y generalmente es mejor reescribirlos usando al menos una declaración if.

Ejemplo:

min = x < y ? (x < z ? x : z) : (y < z ? y : z)

Después de aplicar la corrección rápida, el resultado se verá así: 
min = if x < y
  x < z ? x : z
else
  y < z ? y : z
end
