Informa sobre métodos cuyo cuerpo contiene una única sentencia synchronized. Para métodos de instancia, la expresión de bloqueo de esta sentencia synchronized debe ser igual a this; para métodos static, debe ser igual a [ClassName].class.

Para mejorar la legibilidad de estos métodos, se puede eliminar el contenedor synchronized y marcar el método como synchronized.

Ejemplo:


  public int generateInt(int x) {
    synchronized (this) {
      return 1;
    }
  }

Después de aplicar la corrección rápida:


  public synchronized int generateInt(int x) {
    return 1;
  }
