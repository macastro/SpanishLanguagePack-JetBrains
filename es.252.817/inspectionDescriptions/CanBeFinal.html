Informa sobre campos, métodos o clases que podrían tener el modificador final añadido a sus declaraciones.

Las clases final no pueden ser extendidas, los métodos final no pueden ser sobrescritos y los campos final no pueden ser reasignados.

Ejemplo:


  public class Person {
    private String name;
    Person(String name) {
      this.name = name;
    }
    public String getName() {
      return name;
    }
    public String toString() {
      return getName();
    }
  }

Después de aplicar la corrección rápida:


  public final class Person {
    private final String name;
    Person(String name) {
      this.name = name;
    }
    public final String getName() {
      return name;
    }
    public final String toString() {
      return getName();
    }
  }

Use las opciones Informar clases e Informar métodos para definir qué declaraciones deben ser informadas.
