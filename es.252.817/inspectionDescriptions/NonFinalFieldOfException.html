Reporta campos no declarados como final en subclases de java.lang.Exception.

Los datos en objetos de excepción no deberían modificarse, ya que esto podría resultar en la pérdida del contexto de error necesario para la depuración y el registro posterior.

Ejemplo:


  public class EditorException extends Exception {
    private String message; // Advertencia: campo no final 'message' en clase de excepción
  }
