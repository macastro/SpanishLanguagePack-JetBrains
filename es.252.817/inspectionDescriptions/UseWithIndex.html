Informa sobre bucles for que utilizan variables de índice con incremento manual.

Los bucles for que utilizan variables de índice con incremento manual se pueden simplificar usando la función withIndex().

La corrección rápida Reemplazar incremento manual de índice con withIndex() está disponible para corregir el código automáticamente.

Ejemplo:


  fun foo(list: List<String>): Int? {
      var index = 0
      for (s in list) { <== can be simplified
          val x = s.length * index
          index++
          if (x > 0) return x
      }
      return null
  }

Después de aplicar la corrección rápida:


  fun foo(list: List<String>): Int? {
      for ((index, s) in list.withIndex()) {
          val x = s.length * index
          if (x > 0) return x
      }
      return null
  }
