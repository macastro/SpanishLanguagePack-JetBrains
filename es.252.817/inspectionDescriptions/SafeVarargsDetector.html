Informa sobre todos los métodos con aridad variable que pueden ser anotados con @SafeVarargs. La anotación @SafeVarargs suprime las advertencias no verificadas sobre la creación de arrays parametrizados en el punto de invocación.

Ejemplo:


  public class Foo<T> {
    private List<T> list = new ArrayList<>();
    public final void safeVarargs(T... elements) {
      Collections.addAll(list, elements);
    }
  }

Después de aplicar la corrección rápida:


  public class Foo<T> {
    private List<T> list = new ArrayList<>();
    @SafeVarargs
    public final void safeVarargs(T... elements) {
      Collections.addAll(list, elements);
    }
  }

Esta anotación no es compatible con JVM versión 1.6 o anteriores.
