Informa sobre los métodos del repositorio CRUD de Spring Data con tipos de retorno incorrectos.

Se admiten los siguientes tipos de retorno:

  • void
  • Tipos primitivos
  • Tipos envolventes
  • T cuando se espera que el método de consulta devuelva un resultado como máximo. Devuelve null si no hay resultados y lanza IncorrectResultSizeDataAccessException si hay más de un resultado.
  • java.util.List o cualquier tipo Iterable común
  • java.util.stream.Stream
  • java.util.Optional o scala.Option cuando se espera que el método de consulta devuelva un resultado como máximo. Devuelve Optional.empty() o Optional.absent() si no hay resultados y lanza IncorrectResultSizeDataAccessException si hay más de un resultado.
  • io.micronaut.data.model.Page
  • io.micronaut.data.model.Slice
  • java.util.concurrent.Future, java.util.concurrent.CompletableFuture, o ListenableFuture si el método está anotado con @Async y la ejecución asincrónica del método Spring está habilitada
  • GeoResults<T>, GeoResult<T>, o GeoPage<T>
