Informa sobre métodos de repositorio CRUD de Spring Data con tipos de retorno incorrectos.

Se admiten los siguientes tipos de retorno:

  • void
  • Tipos primitivos
  • Tipos wrapper
  • T (cuando el método de consulta debe devolver como máximo un resultado). Devuelve null si no hay resultados, o lanza IncorrectResultSizeDataAccessException si hay múltiples resultados.
  • java.util.List o cualquier tipo común de Iterable
  • java.util.stream.Stream
  • java.util.Optional o scala.Option (cuando el método de consulta debe devolver como máximo un resultado). Devuelve Optional.empty() o Optional.absent() si no hay resultados, o lanza IncorrectResultSizeDataAccessException si hay múltiples resultados.
  • io.micronaut.data.model.Page
  • io.micronaut.data.model.Slice
  • java.util.concurrent.Future, java.util.concurrent.CompletableFuture o ListenableFuture (si el método está anotado con @Async y la ejecución de métodos asíncronos de Spring está habilitada)
  • GeoResults<T>, GeoResult<T> o GeoPage<T>
