Informa sobre campos logger que no están declarados como static y/o final. Asegurar que cada logger de clase sea una constante válida y esté vinculado a esa clase simplifica la tarea de proporcionar una implementación de registro uniforme para la aplicación.

Se proporciona una corrección rápida para cambiar los modificadores del logger a static final.

Ejemplo:


  public class Significant {
    private Logger LOG = Logger.getLogger(Critical.class);
  }

Después de aplicar la corrección rápida:


  public class Significant {
    private static final Logger LOG = Logger.getLogger(Critical.class);
  }

Configuración de la inspección:

  • La tabla Nombres de clase logger permite especificar nombres de clases logger. La inspección informará sobre campos que no son static y final y cuyo tipo coincide con uno de los nombres de clase especificados.
