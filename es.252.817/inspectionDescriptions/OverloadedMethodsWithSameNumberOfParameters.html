Informa sobre métodos declarados en la misma clase que tienen el mismo nombre y el mismo número de parámetros. Este tipo de sobrecarga puede ser muy confusa, ya que puede no estar claro qué sobrecarga se está invocando.

Ejemplo:


  class Main {
    public static void execute(Runnable r) {}
    public static <T> void execute(RunnableFuture<T> c) {}
  }

Use esta opción para ignorar las sobrecargas de métodos donde los tipos de parámetros son absolutamente incompatibles.
