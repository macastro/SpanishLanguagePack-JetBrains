Informa sobre declaraciones if que solo lanzan java.lang.Throwable desde la rama then sin tener una rama else. Estas declaraciones se pueden convertir en declaraciones assert más compactas.

La inspección también informa sobre Preconditions.checkNotNull() de Guava. Estos pueden reemplazarse por llamadas a Objects.requireNonNull() que no requieren biblioteca.

Ejemplo:


  if (x == 2) throw new RuntimeException("fail");
  if (y == null) throw new AssertionError();
  Preconditions.checkNotNull(z, "z");

Después de aplicar la corrección rápida:


  assert x != 2 : "fail";
  Objects.requireNonNull(y);
  Objects.requireNonNull(z, "z");

Por defecto, esta inspección proporciona correcciones rápidas en el editor sin resaltar el código.
