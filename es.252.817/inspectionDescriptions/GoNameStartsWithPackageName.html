Reporta los nombres de exportación que comienzan con el nombre del paquete. Esta inspección no reporta dichos nombres en el paquete main.

Ejemplo:

package myPackage
func MyPackageGetIP() {
}

El nombre MyPackageGetIP será resaltado porque comienza con el nombre del paquete.

Según Package Names en github.com/golang, todas las referencias a nombres en un paquete se realizarán con el nombre del paquete, por lo que se puede omitir ese nombre en el identificador. Por ejemplo, si estás en el paquete foo, no necesitas un tipo FooFile porque los clientes lo escribirán como foo.FooFile. En su lugar, nombra el tipo como File y los clientes lo escribirán como foo.File.
