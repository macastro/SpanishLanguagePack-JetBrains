Reporta campos static de tipo Collection. Aunque no siempre es un problema, las colecciones estáticas suelen provocar fugas de memoria y, por lo tanto, están prohibidas por algunos estándares de codificación.

Ejemplo:


  public class Example {
    static List<String> list = new ArrayList<>();
  }

Configuración de la inspección:

  • Use la opción Ignorar colecciones o mapas estáticos débiles para ignorar campos de tipo java.util.WeakHashMap.
