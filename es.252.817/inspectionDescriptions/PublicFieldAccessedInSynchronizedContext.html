Informa sobre campos no final ni private a los que se accede en un contexto sincronizado.

No se puede garantizar que a un campo no private se acceda siempre de forma sincronizada, y dicho acceso "parcialmente sincronizado" puede dar lugar a estructuras de datos inesperadamente inconsistentes.

Ejemplo:


  class Bar {
    public String field1;
  }
  public Bar myBar;

  synchronized public void sample() {
   myBar.field1 = "bar";
  }
