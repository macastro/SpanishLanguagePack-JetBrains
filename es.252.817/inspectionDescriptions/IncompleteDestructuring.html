Informa sobre declaraciones de desestructuración incompletas.

Ejemplo:


  data class Person(val name: String, val age: Int)
  val person = Person("", 0)
  val (name) = person

La corrección rápida completa la declaración de desestructuración con nuevas variables:


  data class Person(val name: String, val age: Int)
  val person = Person("", 0)
  val (name, age) = person
