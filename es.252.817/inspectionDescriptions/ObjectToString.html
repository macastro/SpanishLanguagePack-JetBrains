Reporta las llamadas a toString() que usan la implementación por defecto de java.lang.Object.

La implementación por defecto rara vez es intencional pero se puede utilizar por accidente.

Las llamadas a toString() en objetos con tipo java.lang.Object, interface o clase abstracta son ignoradas por esta inspección.

Ejemplo:


  class Bar {
      void foo1(Bar bar) {
          String s = bar.toString(); // warning
          /* ... */
      }

      void foo2(Object obj) {
          String s = obj.toString(); // no warning here
          /* ... */
      }
  }
