Reporta la invocación del método openSession() cuando el recurso org.hibernate.Session devuelto no se cierra de forma segura.

Por defecto, esta inspección asume que un recurso puede ser cerrado por cualquier método cuyo nombre contenga 'close' o 'cleanup'.

Ejemplo:


  void doHibernateQuery(SessionFactory factory) {
    Session session = factory.openSession(); //Advertencia
    session.createQuery("...");
  }

Esta inspección se puede configurar con las siguientes opciones:

  • Si permitir abrir el recurso org.hibernate.Session dentro de un bloque try. Este estilo no es deseable ya que es más redundante que abrir el recurso antes del bloque try.
  • Si el recurso puede ser cerrado por cualquier invocación de método que pase el recurso como argumento.
