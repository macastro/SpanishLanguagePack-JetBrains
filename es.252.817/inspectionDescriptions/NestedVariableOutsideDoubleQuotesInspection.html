Detecta las instancias en las que las variables anidadas se utilizan fuera de las comillas dobles en los archivos .env y podrían no analizarse correctamente. Esta inspección asegura que las variables anidadas estén correctamente encerradas para evitar problemas de configuración.

Esta inspección ayuda a mantener la sintaxis correcta en los archivos .env al forzar el uso de comillas dobles para las variables anidadas, lo que previene comportamientos inesperados durante la evaluación de variables.

Corrección rápida: La inspección proporciona una corrección rápida para encerrar automáticamente la variable anidada entre comillas dobles. Haga clic en la corrección rápida proporcionada para resolver el problema automáticamente.
