Reporta constantes cuyo nombre es demasiado corto, demasiado largo o no coincide con el patrón de expresión regular especificado. Algunos estilos de codificación tienen convenciones de nomenclatura específicas para las constantes.

Utilice las opciones de inspección para especificar la longitud mínima/máxima y la expresión regular esperada para los nombres de constantes. Para ignorar la longitud del nombre, especifique 0.

Para obtener más información sobre expresiones regulares, consulte la Quick Start guide (regular-expressions.info).
