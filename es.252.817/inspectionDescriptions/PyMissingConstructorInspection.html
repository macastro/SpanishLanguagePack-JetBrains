Falta la llamada al constructor super en la clase derivada.

Ejemplo:


class Fruit:
    def __init__(self):
        pass
class Pear(Fruit):
    def __init__(self):
        pass

La clase Pear debería tener una llamada a super en el método __init__.

Después de aplicar la corrección rápida, el código queda así:


class Fruit:
    def __init__(self):
        pass
class Pear(Fruit):
    def __init__(self):
        super().__init__()
