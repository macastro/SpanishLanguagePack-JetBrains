Reporta operaciones de conversión lanzadas entre tipos numéricos primitivos que pueden resultar en pérdida de precisión.

Tales conversiones no son necesariamente un problema, pero pueden resultar en errores difíciles de rastrear si la pérdida de precisión no es esperada.

Ejemplo:


  int a = 420;
  byte b = (byte) a;

Usa la opción Ignorar conversiones de int a char para ignorar conversiones de int a char. Este tipo de conversión se utiliza comúnmente al implementar operaciones de E/S porque el método read() de la clase java.io.Reader devuelve un int.

Usa la opción Ignorar conversiones de int 128-255 a byte para ignorar conversiones de valores constantes (128-255) de int a byte. Dichos valores se desbordarán a números negativos que aún caben dentro de un byte.
