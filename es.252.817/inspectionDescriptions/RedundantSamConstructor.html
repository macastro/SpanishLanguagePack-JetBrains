Informa sobre los usos del constructor SAM (método abstracto único) que se pueden reemplazar con lambdas.

Ejemplo:


  fun main() {
      foo(Runnable { println("Hola!") })
  }

  fun foo(other: Runnable) {}

Después de aplicar la corrección rápida:


  fun main() {
      foo( { println("Hola!") })
  }

  fun foo(other: Runnable) {}
