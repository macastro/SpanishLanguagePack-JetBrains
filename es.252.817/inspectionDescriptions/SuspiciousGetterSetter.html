Informa sobre métodos getter o setter que acceden a campos con nombres diferentes a los esperados. Por ejemplo, cuando getY() devuelve el campo x. Esto generalmente puede indicar un error de copiar y pegar.

Ejemplo:

class Point {
    private int x;
    private int y;
    public void setX(int x) { // Advertencia: el setter 'setX()' asigna el campo 'y'
      this.y = x;
    }
    public int getY() { // Advertencia: el getter 'getY()' devuelve el campo 'x'
      return x;
    }
  }

Utilice la casilla de verificación a continuación para informar casos donde los nombres de los campos en la clase coinciden con los nombres de los getters o setters.
