Inspecciona las funciones que tienen un tipo de retorno `kotlinx.coroutines.Deferred`.

Las funciones que usan `Deferred` como tipo de retorno, deberían tener un sufijo `Async` en su nombre. De lo contrario, se recomienda marcar la función como `suspend` y desenvolver el `Deferred` en ella.

Ejemplo:


  fun calcEverything(): Deferred<Int> {
      return CompletableDeferred(42)
  }

Después de aplicar el arreglo rápido para agregar el sufijo `Async`:


  fun calcEverythingAsync(): Deferred<Int> {
      return CompletableDeferred(42)
  }

Después de aplicar el arreglo rápido para convertir la función a `suspend`:


  suspend fun calcEverything(): Int {
      return CompletableDeferred(42).await()
  }
