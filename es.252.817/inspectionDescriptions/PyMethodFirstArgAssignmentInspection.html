Informa sobre casos donde el primer parámetro (como self o cls) se reasigna dentro del método. El IDE indica un error ya que, en la mayoría de los casos, esta reasignación no tiene un propósito válido.

Ejemplo:


class Account:
    def calc(self, balance):
        if balance == 0:
            self = balance
        return self

La solución puede ser revisar y modificar el algoritmo para asegurar que tal reasignación sea necesaria. Si todo está correcto, puede invocar esta acción de intención y elegir ignorar la advertencia.
