Reporta llamadas a constructores en tipos de función padre que causarán errores de compilación a partir de la versión 1.9.

Tipo de motivación:

  • La implementación no cumple con la especificación o documentación publicada

Más detalles: KT-46344: No hay error al llamar al constructor de superclase en una interfaz de función en la lista de tipos padre

La corrección rápida elimina la llamada al constructor.

Ejemplo:


  abstract class A : () -> Int()

Después de aplicar la corrección rápida:


  abstract class A : () -> Int

Esta inspección solo reporta cuando el nivel del lenguaje Kotlin del proyecto o módulo es 1.7 o superior.
