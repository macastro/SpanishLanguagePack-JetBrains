Informa una llamada al método Java Map.forEach que puede ser reemplazada con forEach de Kotlin.

Ejemplo:


  fun test(map: HashMap<Int, String>) {
      map.forEach { key, value ->
          foo(key, value)
      }
  }

  fun foo(i: Int, s: String) {}

El arreglo rápido agrega paréntesis:


  fun test(map: HashMap<Int, String>) {
      map.forEach { (key, value) ->
          foo(key, value)
      }
  }

  fun foo(i: Int, s: String) {}
