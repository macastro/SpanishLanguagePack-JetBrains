Las clases de prueba JUnit 3 pueden convertirse a clases de prueba JUnit 4.

Ejemplo:


  public class MainTestCase extends junit.framework.TestCase {
    public void test() {
      Assert.assertTrue(true);
    }
  }

Después de aplicar la corrección rápida:


  public class MainTestCase {
    @org.junit.Test
    public void test() {
      Assert.assertTrue(true);
    }
  }

Esta inspección solo informa cuando el nivel de lenguaje del proyecto o módulo es 5 o superior y JUnit 4 está disponible en el classpath.
