El procesador STR informa sobre las expresiones de plantillas de cadena y proporciona una corrección rápida para migrar de nuevo a la concatenación de cadenas simple.

Ejemplo:


  String name = "Bob";
  String greeting = STR."Hola, \{name}. Tienes 29 años.";

Después de aplicar la corrección rápida:


  String name = "Bob";
  String greeting = "Hola, " + name + ". Tienes 29 años.";
