Informa bloques de código semánticamente redundantes en el programa, que pueden reemplazarse por su contenido.

Los bloques de código que son el cuerpo de una sentencia if, do, while o for no se informan en esta inspección.

Ejemplo:


  void foo() {
    { // innecesario
      int result = call();
      analyze(result);
    } // innecesario
  }

Configuración de la inspección:

Utiliza la opción Ignorar ramas de sentencia 'switch' para ignorar los bloques de código usados como ramas de una sentencia switch.
