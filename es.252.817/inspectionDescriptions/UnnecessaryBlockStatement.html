Informa sobre bloques de código semánticamente redundantes en el programa que pueden ser reemplazados por su contenido.

Los bloques de código que funcionan como cuerpo principal de las declaraciones if, do, while o for no serán reportados por esta inspección.

Ejemplo:


  void foo() {
    { // Innecesario
      int result = call();
      analyze(result);
    } // Innecesario
  }

Configuración de la inspección:

Utilice la opción Ignorar ramas de declaraciones 'switch' para ignorar los bloques de código que actúan como ramas de declaraciones switch.
