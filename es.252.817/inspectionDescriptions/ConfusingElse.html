Reporta palabras clave else redundantes en declaraciones if—else y cadenas de declaraciones.

La palabra clave else es redundante cuando todas las ramas anteriores terminan con declaraciones return, throw, break o continue. En estos casos, las declaraciones de la rama else pueden colocarse después de la declaración if y la palabra clave else puede eliminarse.

Ejemplo:


  if (name == null) {
      throw new IllegalArgumentException();
  } else {
      System.out.println(name);
  }

Después de aplicar la corrección rápida:


  if (name == null) {
      throw new IllegalArgumentException();
  }
  System.out.println(name);

Deshabilita la opción Reportar cuando no hay declaraciones después del 'if' para ignorar los casos donde la declaración if—else es la última declaración en un bloque de código.
