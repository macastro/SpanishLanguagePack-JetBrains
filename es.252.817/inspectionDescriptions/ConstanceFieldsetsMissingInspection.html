Detecta claves faltantes en CONSTANCE_CONFIG_FIELDSETS (utilizado para configurar constantes en Django Constance).

Ejemplo:


CONSTANCE_CONFIG = {
    'SITE_NAME': ('My Title', 'Website title'),
    'SITE_DESCRIPTION': ('', 'Website description'),
}
CONSTANCE_CONFIG_FIELDSETS = {
    'General Options': ('SITE_NAME'),
}
