Detecta las claves ausentes en CONSTANCE_CONFIG_FIELDSETS (la constante utilizada para configurar Django Constance).

Ejemplo:


CONSTANCE_CONFIG = {
    'SITE_NAME': ('My Title', 'Website title'),
    'SITE_DESCRIPTION': ('', 'Website description'),
}

CONSTANCE_CONFIG_FIELDSETS = {
    'General Options': ('SITE_NAME'),
}
