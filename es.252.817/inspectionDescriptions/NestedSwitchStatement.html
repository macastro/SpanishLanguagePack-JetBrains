Informa declaraciones o expresiones anidadas switch.

Las declaraciones switch anidadas pueden resultar en un código extremadamente confuso. Las mismas se pueden extraer a un método separado.

Ejemplo:


  int res = switch (i) {
            case 0 -> 0;
            default -> switch (i) {
                case 100 -> 0;
                default -> i;
            };
        };
