Reporta declaraciones o expresiones anidadas de switch.

Las declaraciones anidadas de switch pueden hacer que el código sea especialmente confuso. Estas declaraciones se pueden extraer a un método separado.

Ejemplo:


  int res = switch (i) {
            case 0 -> 0;
            default -> switch (i) {
                case 100 -> 0;
                default -> i;
            };
        };
