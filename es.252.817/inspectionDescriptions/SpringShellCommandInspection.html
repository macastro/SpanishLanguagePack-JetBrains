Reporta declaraciones incorrectas de comandos Spring Shell:

  • Descripción del comando faltante
  • Comando no está dentro de org.springframework.shell.standard.@ShellComponent

Ejemplo:


// Se reportarán todos los métodos debido a la falta de @ShellComponent
class MathCommands {
  @ShellMethod("")
  public int sum(int a, int b) { // Se reportará el método debido a la descripción vacía
    return a + b;
  }
  @ShellMethod
  public int mul(int a, int b) { // Se reportará el método debido a la falta de descripción
    return a * b;
  }
}
