Reporta el uso de java.awt.Color.

Esto podría no detectar temas "oscuros" (por ejemplo, el tema integrado "Darcula"), lo que podría resultar en una experiencia de UI deficiente.

La corrección rápida reemplazará el uso por JBColor, que permite definir variantes de color para temas oscuros.

Ejemplos:


// Incorrecto:
Color darkGreen = new Color(12, 58, 27);
Color blue = Color.BLUE;
// Correcto:
Color darkGreen = new JBColor(12, 58, 27);
Color blue = JBColor.BLUE;
Color green = new JBColor(new Color(12, 58, 27), new Color(27, 112, 39));
