Reporta asignaciones o modificaciones a un parámetro de ciclo for dentro del cuerpo del ciclo.

Aunque a veces es intencional, esta construcción puede ser confusa y, a menudo, es el resultado de un error tipográfico o una variable incorrecta que se está utilizando.

La corrección rápida agrega una declaración de una nueva variable.

Ejemplo:


  for (String s : list) {
    // Advertencia: s se modifica dentro del ciclo
    s = s.trim();
    System.out.println("String: " + s);
  }

Después de que se aplica la corrección rápida:


  for (String s : list) {
    String trimmed = s.trim();
    System.out.println("String: " + trimmed);
  }

No se informan las asignaciones en ciclos for básicos sin una declaración de actualización. En tales casos, la asignación probablemente sea intencional y no se puede mover fácilmente a la parte de actualización del ciclo for.

Ejemplo:


  for (int i = 0; i < list.size(); ) {
    if (element.equals(list.get(i))) {
      list.remove(i);
    } else {
      // no se informa la modificación del parámetro de ciclo for
      // ya que no hay declaración de actualización
      i++;
    }
  }

Use la opción Comprobar parámetros de ciclo 'for' mejorado para especificar si las modificaciones de los parámetros de ciclo for mejorado también deben informarse.
