Informa sobre campos, métodos o clases que podrían tener un alcance de modificador de acceso más restrictivo.

Ejemplo:


  class Sample {
    void foo() {
        bar("foo", "foo");
    }
    void bar(String x, String y) { } // podría ser private
  }

Después de aplicar la corrección rápida:


  class Sample {
    void foo() {
        bar("foo", "foo");
    }
    private void bar(String x, String y) { }
  }

Las opciones de inspección permiten definir las reglas para sugerir cambios de modificadores.
