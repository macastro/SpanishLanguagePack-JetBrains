Informa sobre campos, métodos o clases que pueden tener su modificador de acceso restringido.

Ejemplo:


  class Sample {
    void foo() {
        bar("foo", "foo");
    }
    void bar(String x, String y) { } // puede ser privado
  }

Después de aplicar la solución rápida:


  class Sample {
    void foo() {
        bar("foo", "foo");
    }
    private void bar(String x, String y) { }
  }

Utilice las opciones de inspección para definir las reglas para las sugerencias de cambio de modificador.
