Reporta condiciones de desigualdad que, según el análisis del flujo de datos, se pueden cumplir solo para un único valor de operando. Tales condiciones se pueden reemplazar con condiciones de igualdad para mejorar la claridad del código.

Ejemplo:


  if (x >= 10) {
    ...
    if (x <= 10) { // se puede reemplazar con 'x == 10'
    }
  }
