Informa sobre constantes que están definidas pero nunca se utilizan en el código. 
func main() {
 const i = 100
}

A diferencia de las variables y las importaciones no utilizadas, este código se compilará. Las constantes no utilizadas pueden aumentar la base del código y ralentizar la compilación del programa. Para eliminar la constante no utilizada, se puede usar la corrección rápida Eliminar constante.
