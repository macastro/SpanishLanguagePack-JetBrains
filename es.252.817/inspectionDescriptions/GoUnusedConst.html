Informa sobre constantes que están definidas pero nunca se utilizan en el código. 
func main() {
 const i = 100
}

A diferencia de las variables e imports no utilizados, este código se compilará. Las constantes no utilizadas pueden aumentar el tamaño de la base de código y ralentizar la compilación del programa. Para eliminar las constantes no utilizadas, considere usar la corrección rápida Eliminar constante.
