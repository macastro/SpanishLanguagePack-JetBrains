Informar sobre dependencias obsoletas en los scripts de construcción de Gradle.

Ejemplo:


  dependencies {
      compile "org.jetbrains.kotlin:kotlin-stdlib-jre7:1.2.0"
  }

Después de aplicar la corrección rápida:


  dependencies {
      compile "org.jetbrains.kotlin:kotlin-stdlib-jdk7:1.2.0"
  }
