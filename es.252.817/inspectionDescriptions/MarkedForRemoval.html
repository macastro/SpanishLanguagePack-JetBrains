Informa del uso de las API obsoletas (clases, campos y métodos) marcadas para su eliminación con @Deprecated(forRemoval=true).

El código que utiliza una API marcada para su eliminación puede provocar un error en tiempo de ejecución con una versión futura de la API. Por eso, la gravedad recomendada para esta inspección es Error.

Puedes cambiar la gravedad a Advertencia si quieres usar el mismo resaltado de código que en la depreciación normal.
