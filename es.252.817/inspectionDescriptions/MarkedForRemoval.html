Informa sobre el uso de API obsoletas (clases, campos y métodos) marcadas para eliminación con @Deprecated(forRemoval=true).

El código que utiliza API marcadas para eliminación puede provocar errores en tiempo de ejecución en futuras versiones de la API. Por esta razón, la gravedad recomendada para esta inspección es Error.

Si desea utilizar el mismo resaltado de código que para las obsolescencias normales, puede cambiar la gravedad a Advertencia.
