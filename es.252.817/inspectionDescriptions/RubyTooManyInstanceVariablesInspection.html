Informa sobre clases/módulos que tienen demasiadas variables de instancia. Esto generalmente significa que están intentando "hacer demasiado" y podrían necesitar ser refactorizados en múltiples componentes más pequeños.

Utilice el campo número de variables de instancia para especificar el número máximo de variables de instancia permitidas para una clase/módulo.
