La clase interna puede declararse como static.

Las clases internas static no mantienen una referencia implícita a su instancia contenedora. Cuando se usa Java 17 o versiones anteriores, esto previene fugas de memoria comunes y reduce la memoria utilizada por cada instancia de la clase.

Ejemplo:


  public class Outer {
    class Inner { // no static
      public void foo() {
        bar("x");
      }
      private void bar(String string) {}
    }
  }

Después de aplicar la corrección rápida:


  public class Outer {
    static class Inner {
      public void foo() {
        bar("x");
      }
      private void bar(String string) {}
    }
  }
