Reporta expresiones de conversión de tipo donde la expresión convertida tiene un tipo de clase/interfaz que no extiende/implementa el tipo de clase/interfaz de conversión, ni tiene subclases que lo hagan.

Es probable que estas estructuras sean erróneas y lancen una java.lang.ClassCastException en tiempo de ejecución.

Ejemplo:


  interface A {}
  interface Z {}
  static class C {}
  void x(C c) {
    if (c instanceof Z) {
      A a = ((A)c); // Conversión a interfaz incompatible 'A'
    }
  }
