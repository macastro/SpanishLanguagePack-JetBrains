Reporta expresiones de conversión de tipo donde la expresión convertida tiene un tipo clase/interfaz que no extiende/implementa el tipo clase/interfaz convertido, ni tiene subclases que lo hagan.

Tal construcción es probablemente errónea, y arrojará un java.lang.ClassCastException en tiempo de ejecución.

Ejemplo:


  interface A {}
  interface Z {}
  static class C {}

  void x(C c) {
    if (c instanceof Z) {
      A a = ((A)c); // conversión a interfaz incompatible 'A'
    }
  }
