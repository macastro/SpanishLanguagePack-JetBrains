Reporta bucles for-in sin filtrar.

Esta estructura no solo procesa las propiedades del objeto mismo, sino también las propiedades del prototipo del objeto. Esto puede ser inesperado en ciertos casos específicos, por ejemplo, en métodos utilitarios que copian o modifican todas las propiedades, o cuando el prototipo de Object puede haber sido modificado incorrectamente. Por ejemplo, el siguiente código imprimirá 42 y myMethod: 


Object.prototype.myMethod = function myMethod() {};
let a = { foo: 42 };
for (let i in a) {
  console.log(a[i]);
}

Se recomienda reemplazar todo el bucle con el método Object.keys(), o agregar una comprobación hasOwnProperty(). Después de aplicar la corrección rápida, el código se verá así:


for (let i in a) {
  if (a.hasOwnProperty(i)) {
    console.log(a[i]);
  }
}
