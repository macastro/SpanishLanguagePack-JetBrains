Reporta métodos static que tienen la misma firma que los métodos static de la superclase. Estos métodos pueden parecer que sobrescriben, pero en realidad ocultan el método de la superclase, ya que los métodos static en Java no pueden ser sobrescritos.

Ejemplo:


  class Parent {
    static void method(){}
  }
  class Example extends Parent {
    static void method(){} //advertencia
  }
