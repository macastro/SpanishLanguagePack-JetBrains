Informa sobre comentarios Javadoc y etiquetas ausentes.

Ejemplo:


  /**
   * El "@param" ausente se informa (si se configura).
   */
  public void sample(int param){
  }

Las correcciones rápidas agregan la etiqueta o el comentario Javadoc ausente:


  /**
   * El "@param" ausente se informa (si se configura).
   * @param param
   */
  public void sample(int param){
  }

La inspección se puede configurar para ignorar elementos obsoletos o métodos de acceso simples como getField() o setField().
