Reporta verificaciones de null seguidas de una llamada de método que definitivamente regresará false cuando se pasa null (ej. Class.isInstance).

Tal verificación se ve excesiva ya que la llamada de método siempre regresará false en este caso.

Ejemplo:


  if (x != null && myClass.isInstance(x)) { ... }

Después de la solución rápida es aplicada:


  if (myClass.isInstance(x)) { ... }
