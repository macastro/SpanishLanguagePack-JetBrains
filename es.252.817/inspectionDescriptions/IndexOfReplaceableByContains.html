Informa sobre comparaciones con llamadas a String.indexOf() que pueden ser reemplazadas por llamadas al método String.contains().

Ejemplo:


  boolean b = "abcd".indexOf('e') >= 0;

Después de aplicar la corrección rápida:


  boolean b = "abcd".contains('e');

Esta inspección solo se informa cuando el nivel de lenguaje del proyecto o módulo es 5 o superior.
