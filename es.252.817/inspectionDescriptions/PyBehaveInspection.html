Se informa sobre un uso incorrecto del framework Python Behave BDD:

  • El archivo de definición de pasos no contiene from behave import *.
  • El directorio que contiene las definiciones de pasos no se llama steps.

Si se produce alguno de los casos, el IDE resaltará el paso correspondiente en el archivo .feature. Se puede aplicar una corrección rápida para crear una definición de paso o cambiar el nombre del directorio que contiene el paso.

Para más información, consulte Creando una definición de paso.
