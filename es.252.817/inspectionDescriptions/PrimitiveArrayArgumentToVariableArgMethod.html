Reporta cualquier invocación de un método de aridad variable, en la que la invocación tiene un array primitivo en la posición del parámetro de aridad variable (por ejemplo, System.out.printf("%s", new int[]{1, 2, 3})). Este tipo de argumentos de array primitivo puede llevar a confusión, ya que se envolverá como un array de un solo elemento, en lugar de boxear cada elemento individual como se esperaría.

Ejemplo:


  String.format("%s", new int[]{1, 2, 3});

Tras aplicar la corrección rápida:


  String.format("%s", (Object) new int[]{1, 2, 3});
