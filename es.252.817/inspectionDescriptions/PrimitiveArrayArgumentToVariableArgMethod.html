Informa sobre cualquier llamada a métodos de aridad variable donde la llamada tiene un array primitivo en la posición del parámetro de aridad (por ejemplo, System.out.printf("%s", new int[]{1, 2, 3})). Estos argumentos de array primitivo pueden ser confusos, ya que se envolverán como un array de un solo elemento en lugar de convertir cada elemento individual como se esperaría.

Ejemplo:


  String.format("%s", new int[]{1, 2, 3});

Después de aplicar la corrección rápida:


  String.format("%s", (Object) new int[]{1, 2, 3});
