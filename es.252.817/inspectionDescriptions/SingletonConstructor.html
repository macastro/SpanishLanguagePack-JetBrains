Se informan los constructores de clases anotadas por @Singleton a menos que se declare no estricta.

Ejemplo:


  @Singleton
  class Foo{
      Foo(){
      }
  }

Hay dos posibles correcciones rápidas: eliminar el constructor o declarar que @Singleton no es estricta.

Después de aplicar la corrección rápida:


  @Singleton
  class Foo{
  }

o:


  @Singleton(strict = false)
  class Foo{
      Foo(){
      }
  }
