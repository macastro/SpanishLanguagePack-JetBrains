Se sugiere reemplazar la comprobación size == 0 de Collections/Array/String por isEmpty().

Usar isEmpty() hace que el código sea más simple.

Esta corrección rápida reemplazará la comprobación de tamaño por isEmpty().

Ejemplo:


  fun foo() {
      val arrayOf = arrayOf(1, 2, 3)
      arrayOf.size == 0
  }

Después de aplicar la corrección rápida:


  fun foo() {
      val arrayOf = arrayOf(1, 2, 3)
      arrayOf.isEmpty()
  }
