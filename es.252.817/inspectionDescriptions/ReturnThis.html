Informa sobre los métodos que devuelven this.

Si bien dicha devolución es válida, rara vez es necesaria y generalmente indica que el método está destinado a usarse como parte de una cadena de llamadas a métodos similares (por ejemplo, buffer.append("foo").append("bar").append("baz")). Muchas normas de codificación desaprueban dichas cadenas.

Ejemplo:


  public Builder append(String str) {
    // [...]
    return this;
  }
