Informa sobre métodos que tienen el mismo nombre que la superclase de la clase que los contiene, ya que estos nombres de métodos pueden resultar confusos.

Esta inspección no verifica interfaces o superclases en niveles más profundos de la jerarquía.

Ejemplo: 


  class Parent {}
  class Child extends Parent {
     public Parent Parent() {
       return null;
     }
  }

La corrección rápida para renombrar estos métodos solo está disponible en el editor.
