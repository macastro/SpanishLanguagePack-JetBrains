Informa sobre las llamadas a Collection<T>.count().

Esta llamada a función puede ser reemplazada por .size.

La forma .size garantiza que la operación es O(1) y no asigna objetos adicionales, mientras que count() puede confundirse con Iterable<T>.count(), que es O(n) y realiza asignaciones.

Ejemplo:

  fun foo() {
      var list = listOf(1,2,3)
      list.count() // 'count()' reemplazable
  }

Después de aplicar la corrección rápida:

  fun foo() {
      var list = listOf(1,2,3)
      list.size
  }
