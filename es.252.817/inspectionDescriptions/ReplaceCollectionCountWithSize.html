Informa llamadas a Collection<T>.count().

Esta llamada de función puede ser reemplazada por .size.

La estructura .size asegura que la operación sea de O(1) y que no se asignen objetos extra, mientras que count() puede ser confundido con Iterable<T>.count(), que es O(n) y de asignación.

Ejemplo:

  fun foo() {
      var list = listOf(1,2,3)
      list.count() // 'count()' reemplazable
  }

Después de aplicar la corrección rápida:

  fun foo() {
      var list = listOf(1,2,3)
      list.size
  }
