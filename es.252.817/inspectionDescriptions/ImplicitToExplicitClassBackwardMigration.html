Informa sobre clases declaradas implícitamente y sugiere reemplazarlas por clases regulares.

Ejemplo (en el archivo Sample.java):


  public static void main() {
      System.out.println("Hello, world!");
  }

Después de aplicar la corrección rápida:


public class Sample {
    public static void main() {
        System.out.println("Hello, world!");
    }
}

Esta inspección ayuda con la compatibilidad hacia atrás con versiones anteriores de Java.
