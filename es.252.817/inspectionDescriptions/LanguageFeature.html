Informa de los usos de funciones avanzadas de lenguaje que no están habilitadas en el archivo actual.

Las funciones avanzadas de lenguaje deben habilitarse mediante la opción de compilador correspondiente o mediante la importación de un indicador de función explícito en el archivo donde se utilizan.

Notación de operador de sufijo:
val s = 123 toString

Llamada reflexiva:
def foo(v: {def bar()}) = v.bar()

Selección dinámica de miembros:
class Foo extends Dynamic

Conversión implícita:
implicit def toInt(s: String) = s.toInt

Tipo de orden superior(habilitado por defecto desde Scala 2.13.1):
class Bar[M[A]]

Tipo existencial:
def foo(v: Seq[T] forSome { type T })

Definición de macro:
def assert(s: String) = macro Asserts.assertImpl

El compilador de Scala también puede emitir estas advertencias cuando se proporciona la opción "-feature".

Para las opciones del compilador, consulte: Archivo | Configuración | Compilación, ejecución, implementación | Compilador | Compilador de Scala | Funciones
