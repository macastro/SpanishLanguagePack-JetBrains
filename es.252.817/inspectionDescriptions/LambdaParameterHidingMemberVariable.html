Informa de parámetros lambda que comparten exactamente el mismo campo y nombre que una clase externa. Estas denominaciones pueden provocar que se utilicen erróneamente parámetros lambda cuando la intención es utilizar el campo con exactamente el mismo nombre.

Se sugiere una corrección rápida para cambiar el nombre del parámetro lambda.

Ejemplo:


  public class MyClass {
    public Object foo;

    void sort(List<Integer> list) {
      list.sort((foo, bar) -> foo - bar);
    }
  }

Usa la opción para seleccionar si se ignoran los campos que no son visibles para la expresión lambda. Por ejemplo, los campos privados de la superclase.
