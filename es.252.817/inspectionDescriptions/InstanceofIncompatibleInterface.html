Reporta expresiones instanceof donde el tipo de clase/interfaz que se está comprobando no extiende/implementa el tipo de clase/interfaz del lado derecho de la expresión instanceof, ni tiene subclases que lo hagan.

Aunque puede usarse en código de bibliotecas, estas construcciones probablemente sean erróneas, ya que ninguna instancia de ninguna clase declarada en el proyecto podrá pasar esta prueba instanceof.

Ejemplo:


  class Foo { }
  interface Bar { }
  class Main {
      void test(Foo f, Bar b) {
          if (f instanceof Bar) { // Problema
              System.out.println("fail");
          }
          if (b instanceof Foo) { // Problema
              System.out.println("fail");
          }
      }
  }
