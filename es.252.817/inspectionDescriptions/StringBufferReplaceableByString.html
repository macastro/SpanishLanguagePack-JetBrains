Informa sobre usos de StringBuffer, StringBuilder o StringJoiner que pueden ser reemplazados por concatenación de String.

El uso de concatenación de String hace que el código sea más corto y simple.

Esta inspección solo informa cuando el reemplazo sugerido no causará una degradación significativa del rendimiento en JVMs modernas. En muchos casos, la concatenación de String puede funcionar mejor.

Ejemplo:


  StringBuilder result = new StringBuilder();
  result.append("i = ");
  result.append(i);
  result.append(";");
  return result.toString();

Después de aplicar la corrección rápida:


  String result = "i = " + i + ";";
  return result;
