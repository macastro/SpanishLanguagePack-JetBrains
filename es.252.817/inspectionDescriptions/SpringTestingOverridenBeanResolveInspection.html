Informa sobre referencias incorrectas a beans o métodos en los parámetros de las anotaciones @MockitoBean, @MockitoSpyBean y @TestBean.

Ejemplo:


@SpringBootTest
class TestBeanApplicationTests {
   @TestBean(value = "unknownBean", methodName="createUserService")  // Informa de 1. "No se puede resolver el bean 'unknownBean'" y 2. "No se puede resolver el símbolo 'createUserService'"
   UserService userService;

  @TestBean  // Informa de "No se ha encontrado el método de fábrica estático"
  UserService userService;
}
