Informa sobre las declaraciones return que devuelven valores en el método __init__ de una clase.

Ejemplo:


class Sum:
    def __init__(self, a, b):
        self.a = a
        self.b = b
        self.sum = a + b
        return self.sum

El constructor no debe devolver ningún valor. El método __init__ solo debe inicializar los valores de los miembros de la nueva instancia del objeto.

La corrección rápida propuesta por el IDE es eliminar la declaración return.
