Reporta ocurrencias de sentencias return con un valor de retorno dentro de los métodos __init__ de clases.

Ejemplo:


class Sum:
    def __init__(self, a, b):
        self.a = a
        self.b = b
        self.sum = a + b
        return self.sum

Un constructor no debe regresar ningún valor. El método __init__ sólo debe inicializar los valores de los miembros de instancia para objetos nuevos.

Como solución rápida, el IDE se ofrece a eliminar la sentencia return.
