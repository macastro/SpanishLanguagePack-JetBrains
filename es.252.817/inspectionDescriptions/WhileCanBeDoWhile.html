El bucle while se puede escribir de manera más efectiva como un bucle do-while. Existe un bucle while donde el código antes del bucle es idéntico al código dentro del cuerpo del bucle. Reemplazarlo con un bucle do-while eliminará el código duplicado. Para los bucles while sin dicha duplicación de código, la corrección rápida también está disponible en el editor, pero sin resaltado.

Ejemplo:


  foo();
  while (x) {
      foo();
  }

Se puede reemplazar por:


  do {
    foo();
  } while (x);
