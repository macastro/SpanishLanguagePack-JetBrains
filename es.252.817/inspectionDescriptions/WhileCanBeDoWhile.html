Reporta ciclos while que podrían escribirse de manera más efectiva como ciclos do-while. Hay ciclos while en los que el código justo antes del ciclo es idéntico al código en el cuerpo del ciclo. El reemplazo con un ciclo do-while elimina el código duplicado. Para ciclos while sin dicho código duplicado, la solución rápida también se ofrece en el editor, pero sin resaltar.

Ejemplo:


  foo();
  while (x) {
      foo();
  }

Puede reemplazarse con:


  do {
    foo();
  } while (x);
