El informe puede comparar las llamadas a equals() entre dos arrays.

Llamar a equals() en arrays compara la identidad, equivalente a usar ==. Use Arrays.equals() para comparar el contenido de dos arrays, o Arrays.deepEquals() para comparar el contenido de arrays multidimensionales.

Ejemplo:


  void sample(int[] first, int[] second){
    if (first.equals(second)) return;
  }

Después de aplicar la corrección rápida:


  void sample(int[] first, int[] second){
    if (Arrays.equals(first, second)) return;
  }
