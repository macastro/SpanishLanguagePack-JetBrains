Informa de cadenas entre dobles comillas que se pueden convertir a comillas simples. Cualquier cadena que no contenga sustituciones de expresiones o símbolos especiales puede convertirse.

Ejemplo:

# Mala práctica
"Hello world"
"Hello \"John Doe\""
Hay disponible una corrección rápida para convertir una cadena entre comillas dobles. Después de aplicar la corrección rápida, el resultado es el siguiente: 
# Buena práctica
'Hello world'
'Hello "John Doe"'
"Hello 'John Doe'"

Utiliza la casilla de verificación Ignorar cadenas que contengan comillas simples para ignorar las cadenas que requieren escape.

Inspirado en 'RuboCop'
