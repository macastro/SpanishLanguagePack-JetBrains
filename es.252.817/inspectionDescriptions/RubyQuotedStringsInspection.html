El informe puede convertir cadenas de texto entrecomilladas dobles en cadenas entrecomilladas simples. Se pueden convertir las cadenas que no contengan sustituciones de expresiones ni símbolos especiales.

Ejemplo:

# 不良做法
"Hello world"
"Hello \"John Doe\""
Las cadenas con comillas dobles se pueden convertir mediante una corrección rápida. Después de aplicar la corrección rápida, el resultado se muestra así: 
# 优良做法
'Hello world'
'Hello "John Doe"'
"Hello 'John Doe'"

Use la casilla de verificación Ignorar cadenas que contienen comillas simples para ignorar las cadenas que requieren escape.
