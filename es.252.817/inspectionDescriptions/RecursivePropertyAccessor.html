Informa sobre llamadas a accesores de propiedades recursivas que pueden terminar en StackOverflowError. Este tipo de llamadas se confunden frecuentemente con el acceso a campos de respaldo.

Ejemplo:


  var counter: Int = 0
      set(value) {
          counter = if (value < 0) 0 else value
      }

Después de aplicar la corrección rápida:


  var counter: Int = 0
      set(value) {
          field = if (value < 0) 0 else value
      }
