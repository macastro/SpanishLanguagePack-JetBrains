Informa sobre paréntesis innecesarios en tipos de funciones. Las funciones sin parámetros no requieren paréntesis.

Los paréntesis innecesarios se pueden eliminar mediante una corrección rápida.

Ejemplo:

class Example
  # Paréntesis innecesarios en el tipo de función
  def call: () -> untyped
end

Después de aplicar la corrección rápida:

class Example
  def call: -> untyped
end
