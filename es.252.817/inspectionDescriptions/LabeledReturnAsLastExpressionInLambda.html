Inspeccionar el uso de `return` etiquetado en la última expresión en lambdas.

Dicho `return` es innecesario y puede eliminarse con seguridad.

Ejemplo:


fun foo() {
  listOf(1,2,3).find {
    return@find true
  }
}

Después de aplicar la corrección rápida:


fun foo() {
  listOf(1,2,3).find {
    true
  }
}
