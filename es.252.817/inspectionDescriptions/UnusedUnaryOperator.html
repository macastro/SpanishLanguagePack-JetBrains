Informa de operadores unarios en expresiones no utilizadas para tipos numéricos.

Los operadores unarios interrumpen la expresión anterior si se utilizan sin llaves. Como resultado, las expresiones matemáticas que abarcan varias líneas pueden llevar a confusión.

Ejemplo:


  fun main() {
      val result = 1 + 2 * 3
                  + 3              // <== note que '+ 3' no pertenece a la variable 'result', no se utiliza
      println("Resultado = $result")  // El resultado es '7' y no '10' como cabría esperar
  }
