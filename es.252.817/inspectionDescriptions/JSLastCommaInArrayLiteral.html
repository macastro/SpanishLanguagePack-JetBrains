Reporta el uso de comas finales en literales de array.

Esta advertencia solo se muestra cuando la versión de JavaScript está configurada como ECMAScript 5.1.

Aunque la especificación permite las comas finales en arrays, algunos navegadores pueden generar errores cuando se utilizan.

Puede configurar las opciones de formato para comas finales en Estilo de código | JavaScript o TypeScript | Puntuación.
