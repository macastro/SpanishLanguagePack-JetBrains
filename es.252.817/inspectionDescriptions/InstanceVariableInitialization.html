Reporta variables de instancia que podrían no estar inicializadas con la inicialización del objeto.

Ejemplo:


  class Foo {
    public int bar;

    static { }
  }

Nota que esta inspección usa un algoritmo de flujo de datos muy conservador y podría reportar incorrectamente variables de instancia como no inicializadas. Las variables reportadas como inicializadas siempre estarán inicializadas.

Use la opción Ignorar campos primitivos para ignorar campos primitivos no inicializados.
