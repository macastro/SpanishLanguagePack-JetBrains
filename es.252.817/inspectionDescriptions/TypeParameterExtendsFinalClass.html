Informa sobre parámetros de tipo declarados que extienden una clase final.

Dado que no es posible extender una clase final, se sugiere reemplazar el parámetro de tipo por el tipo de la clase final especificada.

Ejemplo:


  void foo() {
    List<? extends Integer> list; // advertencia: la clase Integer es final
  }

Después de aplicar la corrección rápida:


  void foo() {
    List<Integer> list;
  }
