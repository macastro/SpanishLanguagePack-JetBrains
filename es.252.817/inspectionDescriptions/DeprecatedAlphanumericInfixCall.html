Informa de los usos de las definiciones alfanuméricas sin el modificador `infix` como operador infijo.

  • La corrección rápida "Encerrar entre comillas invertidas" agrega símbolos ``` alrededor de las llamadas.
  • La corrección rápida "Convertir de expresión infija" convierte la llamada infija a la sintaxis del método `método(...)`.
  • La corrección rápida "Convertir de tipo infijo" convierte el tipo infijo a la sintaxis del prefijo "`Tipo[...]]`.
  • La corrección rápida "Convertir de patrón infijo" convierte el patrón infijo a la sintaxis del prefijo "patrón(...)".

Ejemplo:


  class C:
    def op(x: Int): Int = ???
  type Or[X, Y]
  case class Pair[T](x: T, y: T)

  val c = C()
  val p = Pair(1, 2)

  c op 2
  val or: Int Or String = ???
  val _ Pair _ = p
