Informa las propiedades públicas de un objeto con tipos específicos. Sugiere especificar la forma del objeto devuelto a través del atributo #[ObjectShape] o una forma de objeto en el comentario PHPDoc.

Ejemplo:


  $a = new stdClass();
  $a->nombre = "nombre";
  $a->edad = 2;

Después de aplicar el arreglo rápido:


  /** @var $a object{nombre: string, edad: int} */
  $a = new stdClass();
  $a->nombre = "nombre";
  $a->edad = 2;
