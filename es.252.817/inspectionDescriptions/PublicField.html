Reporta campos public. No reporta constantes (campos marcados con static y final).

Ejemplo:


  class Main {
    public String name;
  }

Después de aplicar la corrección rápida:


  class Main {
    private String name;
    public String getName() {
        return name;
    }
    public void setName(String name) {
        this.name = name;
    }
  }

Configuración de la inspección:

  • Utilice la lista Ignorar cuando tiene anotaciones para especificar las anotaciones que se deben ignorar. La inspección ignorará los campos que contengan cualquiera de estas anotaciones.
  • Utilice la opción Ignorar campos 'public final' en enumeraciones para ignorar campos public final de tipo enum.
