Informa sobre el uso de funciones o métodos de Python marcados como obsoletos y que generan la advertencia DeprecationWarning o PendingDeprecationWarning.

También este análisis destaca los usos de los decoradores abc.abstractstaticmethod, abc.abstractproperty y abc.abstractclassmethod.

Ejemplo:


class Foo:
    @property
    def bar(self):
        import warnings
        warnings.warn("esto ha quedado obsoleto", DeprecationWarning, 2)
        return 5


foo = Foo()
print(foo.bar)
