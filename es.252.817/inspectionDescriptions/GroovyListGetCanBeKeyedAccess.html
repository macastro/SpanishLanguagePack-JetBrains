Informa sobre las llamadas al método java.util.List.get(). Estas llamadas pueden ser reemplazadas por una forma de acceso indexado más corta y explícita.

Ejemplo:


def list = ["foo"]
def str = list.get(0) // list.get(0) puede reemplazarse por list[0]

Después de aplicar la corrección rápida:


def list = ["foo"]
def str = list[0]
