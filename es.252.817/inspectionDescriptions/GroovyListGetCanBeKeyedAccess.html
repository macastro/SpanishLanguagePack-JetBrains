Reporta llamadas a métodos de java.util.List.get(). Dichas llamadas podrían ser reemplazadas por el formato de acceso con clave, más corto y claro.

Ejemplo:


def list = ["foo"]
def str = list.get(0) // list.get(0) podría ser reemplazado con list[0]

Después de aplicar la corrección rápida:


def list = ["foo"]
def str = list[0]
