Se informa sobre las extensiones que se instancian mediante la plataforma IntelliJ pero que se declaran como objetos Kotlin.

El ciclo de vida de la extensión es administrado por la plataforma IntelliJ. El uso de objetos Kotlin para el registro de extensión puede crear instancias de extensión innecesarias y puede impedir la descarga del complemento.

Ejemplo

Registro de extensión:


<annotator language="myLang" implementationClass="com.example.MyAnnotator"/>

Implementación de la extensión:


// No apropiado:
object MyAnnotator : Annotator {
  ...
}

// Apropiado:
class MyAnnotator : Annotator {
  ...
}
