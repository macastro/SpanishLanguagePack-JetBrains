Reportes de la declaración if que puede reemplazar la declaración de case.

Utilice el campo Número mínimo de ramas para especificar el número mínimo de ramas when para la declaración de case resultante.

Ejemplo:

fruit = gets

if fruit == 'apple' # Declaración de "if" simplificable
  puts '¡Es una manzana!'
elsif fruit == 'peach'
  puts '¡Es un melocotón!'
elsif fruit == 'orange'
  puts '¡Es una naranja!'
elsif fruit == 'banana'
  puts '¡Es un plátano!'
end
