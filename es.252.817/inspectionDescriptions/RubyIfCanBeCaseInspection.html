Informa sobre declaraciones if que pueden ser reemplazadas por declaraciones case.

Utiliza el campo número mínimo de ramas para especificar el número mínimo de ramas when para la declaración case resultante.

Ejemplo:

fruit = gets
if fruit == 'apple' # Simplifiable 'if' statement
  puts 'This is apple!'
elsif fruit == 'peach'
  puts 'This is peach!'
elsif fruit == 'orange'
  puts 'This is orange!'
elsif fruit == 'banana'
  puts 'This is banana!'
end
