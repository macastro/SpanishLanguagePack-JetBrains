Reporta las clases anidadas que se declaran sin ningún modificador de acceso (también conocidas como package-private).

Ejemplo:


  public class Outer {
    static class Nested {}  // advertencia
    class Inner {}          // advertencia
    enum Mode {}            // advertencia según la configuración
    interface I {}          // advertencia según la configuración
  }

Configuración de la inspección:

  • Use la opción Ignorar enums internos con visibilidad de paquete para ignorar los enums internos package-private.
  • Use la opción Ignorar interfaces internas con visibilidad de paquete para ignorar las interfaces internas package-private.
