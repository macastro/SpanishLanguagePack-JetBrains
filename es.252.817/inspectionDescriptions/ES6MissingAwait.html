Informa sobre llamadas a funciones async sin el prefijo await esperado dentro de funciones async. Tales llamadas devuelven una Promise y el flujo de control continúa inmediatamente.

Ejemplo:


async function bar() { /* ... */ }
async function foo() {
    bar(); // Incorrecto
}

Después de aplicar la corrección rápida, se añadirá el prefijo await:


async function bar() { /* ... */ }
async function foo() {
    await bar(); // Correcto
}

Cuando la casilla "Informar sobre promesas en declaraciones return" está marcada, también se sugiere añadir await en las declaraciones return.
Aunque generalmente no es necesario, tiene dos ventajas principales.
  • No olvidará añadir await cuando envuelva el código con try-catch.
  • El await explícito ayuda al tiempo de ejecución V8 a proporcionar trazas de pila asíncronas.
