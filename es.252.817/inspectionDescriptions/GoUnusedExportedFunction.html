Informa sobre funciones exportadas no utilizadas.

En Go, una función se exporta si comienza con una letra mayúscula. Los nombres de las funciones exportadas que están definidas pero nunca se utilizan se mostrarán en gris.

// 未使用的导出函数
func ExportedUnusedFunc()  {
}
func main() {
 fmt.Println("Hello")
}
