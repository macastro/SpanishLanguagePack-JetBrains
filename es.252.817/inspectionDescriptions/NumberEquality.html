equals() en lugar de usar == o != para probar la equivalencia de Number.

Cuando se usa el autoboxing, es un error común comparar dos instancias del tipo contenedor en lugar de dos primitivos. Por ejemplo, usando Integer en lugar de int.

Ejemplo:

  void foo(Integer a, Integer b) {
    final boolean bool = a == b;
  }

Si se sabe que a no es null, es seguro aplicar una corrección rápida "insegura" para obtener un resultado como:

  void foo(Integer a, Integer b) {
    final boolean bool = a.equals(b);
  }
