Informa sobre nombres de bloque duplicados en plantillas de Django.

Ejemplo:


<!DOCTYPE html>
<html lang="en">
<head>
    <link rel="stylesheet" href="style.css">
    <title>{% block title %}Mi maravilloso sitio{% endblock %}</title>
</head>

<body>
    <div id="sidebar">
        {% block title %}
        <ul>
            <li><a href="/">Inicio</a></li>
            <li><a href="/blog/">Blog</a></li>
        </ul>
        {% endblock %}
    </div>

    <div id="content">
        {% block content %}{% endblock %}
    </div>
</body>
</html>
