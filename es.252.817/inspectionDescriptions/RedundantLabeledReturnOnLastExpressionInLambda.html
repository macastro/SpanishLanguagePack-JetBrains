Inspecciona el uso de retorno etiquetado en la última expresión en lambdas.

Dicho retorno es redundante y puede eliminarse de forma segura.

Ejemplo:


fun foo() {
  listOf(1,2,3).find {
    return@find true
  }
}

Después de aplicar la reparación rápida:


fun foo() {
  listOf(1,2,3).find {
    true
  }
}
