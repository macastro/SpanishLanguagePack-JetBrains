Informa sobre el uso de return etiquetado en la última expresión de una lambda.

Este tipo de return es redundante y puede eliminarse de forma segura.

Ejemplo:


fun foo() {
  listOf(1,2,3).find {
    return@find true
  }
}

Después de aplicar la corrección rápida:


fun foo() {
  listOf(1,2,3).find {
    true
  }
}
