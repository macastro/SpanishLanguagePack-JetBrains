Informa sobre constructores primarios vacíos cuando el constructor primario está implícitamente disponible.

El constructor primario es redundante y se puede omitir de forma segura cuando no tiene anotaciones ni modificadores de visibilidad. Utiliza la corrección rápida "Eliminar constructor primario vacío" para limpiar el código.

Ejemplo:


  class MyClassA constructor() //  redundante, puede reemplazarse por 'class MyClassA'
  annotation class MyAnnotation
  class MyClassB @MyAnnotation constructor() //  requerido debido a la anotación
  class MyClassC private constructor() // requerido debido al modificador de visibilidad
