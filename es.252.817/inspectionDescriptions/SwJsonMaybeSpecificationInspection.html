Detecta archivos JSON que se pueden interpretar como parte de la especificación OpenAPI/Swagger.

Dichos archivos no contienen atributos de especificación explícitos, pero se hace referencia a ellos desde archivos de especificación primarios ubicados cerca.

El siguiente ejemplo contiene dos archivos ubicados en el mismo directorio. El primero es un archivo de especificación primaria simple. El segundo se referencia desde el primero y, por lo tanto, se sugiere que también sea considerado una especificación.

Archivo de especificación primaria openapi.yaml:


openapi: 3.1.0
components:
  schemas:
    CustomSchema:
      description: Objeto de esquema personalizado
      properties:
        foo:
          $ref: 'common.components.json#/components/schemas/CommonSchema'

Candidato a archivo de especificación common.components.json:


{
  "components": { // Resaltado de 'Marcar archivo como especificación OpenAPI'
    "schemas": {
      "CommonSchema": {
        "description": "Objeto de esquema común reutilizado en varias especificaciones"
      }
    }
  }
}
