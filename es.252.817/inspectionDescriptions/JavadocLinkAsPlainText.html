Reporta enlaces de texto sin formato en comentarios Javadoc.

La corrección rápida sugiere envolver el enlace con la etiqueta <a>.

Ejemplo:


  class Main {
   /**
    * https://en.wikipedia.org/
    */
    void foo() {}
  }

Después de aplicar la corrección rápida:


  class Main {
   /**
    * <a href="https://en.wikipedia.org/">https://en.wikipedia.org/</a>
    */
    void foo() {}
  }
