Informa sobre declaraciones throw que lanzan excepciones inapropiadas. Por ejemplo, una excepción puede ser inapropiada por ser demasiado general, como java.lang.Exception o java.io.IOException.

Ejemplo:


  void setup(Mode mode) {
      if (mode == null)
          throw new RuntimeException("Problem during setup"); // Advertencia: Se lanzó la excepción prohibida 'RuntimeException'
      ...
  }

Use la lista de excepciones prohibidas para especificar qué excepciones deben ser informadas.
