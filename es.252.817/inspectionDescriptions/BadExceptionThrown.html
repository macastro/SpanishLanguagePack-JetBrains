Informa las declaraciones throw que lanzan una excepción inapropiada. Por ejemplo, una excepción puede ser inapropiada porque es demasiado genérica, como java.lang.Exception o java.io.IOException.

Ejemplo:


  void setup(Mode mode) {
      if (mode == null)
          throw new RuntimeException("Problema durante la configuración"); // advertencia: se lanzó la excepción prohibida 'RuntimeException'
      ...
  }

Usa la lista Excepciones prohibidas para especificar qué excepciones deberían informarse.
