Informa sobre llamadas a notify() o notifyAll() en objetos java.util.concurrent.locks.Condition.

Esto podría ser un error de programación donde se deberían haber usado variantes de los métodos signal() o signalAll(), de lo contrario podría producirse una IllegalMonitorStateException.

Ejemplo:


  class C {
    final Lock l = new ReentrantLock();
    final Condition c = l.newCondition();
    void release() {
      l.lock();
      try {
        c.notifyAll(); // Aquí probablemente debería usarse 'signalAll()'
      } finally {
        l.unlock();
      }
    }
  }
