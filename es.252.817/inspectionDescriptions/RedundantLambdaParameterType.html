Informa sobre tipos de parámetros lambda redundantes que pueden ser inferidos del contexto.

Ejemplo:


  Map<String, Integer> map = ...
  map.forEach((String s, Integer i) -> log.info(s + "=" + i));

Esta corrección rápida eliminará estos tipos de parámetros del lambda.


  Map<String, Integer> map = ...
  map.forEach((s, i) -> log.info(s + "=" + i));
