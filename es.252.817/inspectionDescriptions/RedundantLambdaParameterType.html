Informa sobre los tipos de parámetros formales lambda que son redundantes porque se pueden inferir desde el contexto.

Ejemplo:


  Map<String, Integer> map = ...
  map.forEach((String s, Integer i) -> log.info(s + "=" + i));

El arreglo rápido elimina los tipos de parámetros de la lambda.


  Map<String, Integer> map = ...
  map.forEach((s, i) -> log.info(s + "=" + i));
