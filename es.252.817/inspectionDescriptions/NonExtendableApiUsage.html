Reporta las clases, interfaces y métodos que extienden, implementan o sobrescriben elementos de API marcados con @ApiStatus.NonExtendable.

La anotación @ApiStatus.NonExtendable indica que una clase, interfaz o método no debe ser extendido, implementado o sobrescrito. Es común que estas interfaces y clases se conviertan en implementaciones de biblioteca interna, por lo que si el cliente proporciona una implementación diferente, se obtendrá una ClassCastException. Agregar nuevos métodos abstractos a este tipo de clases e interfaces romperá la compatibilidad con las implementaciones del cliente.
