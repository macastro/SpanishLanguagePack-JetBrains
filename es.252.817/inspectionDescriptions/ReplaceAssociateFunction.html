Se informa de llamadas a associate() y associateTo() que se pueden sustituir por associateBy() o associateWith().

Ambas funciones aceptan una función de conversión que se aplica a los elementos de una secuencia o colección determinada (como receptor). Luego, dicho par de funciones se usa para construir el resultado de Map.

Teniendo en cuenta que el convertidor hace referencia a it, se puede reemplazar la llamada a associate[To]() por associateBy() o associateWith(), que ofrecen un mayor rendimiento.

Ejemplo:

  fun getKey(i: Int) = 1L
  fun getValue(i: Int) = 1L

  fun test() {
      arrayOf(1).associate { getKey(it) to it }  // 'associate()' reemplazable
      listOf(1).associate { it to getValue(it) } // 'associate()' reemplazable
  }

Después de aplicar la corrección rápida:

  fun getKey(i: Int) = 1L
  fun getValue(i: Int) = 1L

  fun test() {
      arrayOf(1).associateBy { getKey(it) }
      listOf(1).associateWith { getValue(it) }
  }
