Informa sobre casos donde las llamadas CASE y COALESCE son intercambiables. Esta inspección tiene las siguientes acciones de intención: Reemplazar con llamada 'COALESCE' y su opuesto Reemplazar con expresión CASE.

Ejemplo (MySQL):

SELECT
  -- this CASE may be replaced by COALESCE
	CASE
		WHEN C1 IS NOT NULL THEN C1
		ELSE 0
		END
FROM dual;

En este ejemplo, la sentencia CASE puede reemplazarse por SELECT COALESCE(C1, 0), lo que producirá el mismo resultado.

Si prefieres usar expresiones CASE, selecciona la opción Preferir expresiones CASE sobre funciones COALESCE en la página de inspección.
