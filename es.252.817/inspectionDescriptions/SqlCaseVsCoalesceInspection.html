Informa sobre las situaciones en las que las llamadas a CASE y COALESCE son intercambiables. Esta inspección tiene las siguientes acciones de intención: Sustituir por llamada 'COALESCE' y la opuesta Sustituir por expresión CASE.

Ejemplo (MySQL):

SELECT
  -- this CASE may be replaced by COALESCE
	CASE
		WHEN C1 IS NOT NULL THEN C1
		ELSE 0
		END
FROM dual;

En este ejemplo, la instrucción CASE se puede sustituir por SELECT COALESCE(C1, 0), que produce la misma salida.

Si prefiere utilizar expresiones CASE, seleccione la opción Preferir expresiones CASE sobre la función COALESCE en la página de inspección.
