Informa sobre las llamadas a cadenas de llamadas Optional que pueden ser reemplazadas por secuencias de instrucciones if.

Ejemplo:


  return Optional.ofNullable(name)
    .map(this::extractInitials)
    .map(initials -> initials.toUpperCase(Locale.ENGLISH))
    .orElseGet(this::getDefault);

Una vez aplicada la corrección rápida:


  if (name != null) {
    String initials = extractInitials(name);
    if (initials != null) return initials.toUpperCase(Locale.ENGLISH);
  }
  return getDefault();

Esta inspección puede resultar útil para hacer un downgrade para mantener la compatibilidad con versiones anteriores de Java.
