Reporta llamadas a métodos java.util.List.set(). Estas llamadas pueden ser reemplazadas por un formato de acceso con clave más corto y claro.

Ejemplo:


def list = ["foo"]
list.set(0, "bar") // list.set(0, "bar") puede ser reemplazado por list[0] = "bar"

Luego de aplicar la corrección rápida:


def list = ["foo"]
list[0] = "bar"
