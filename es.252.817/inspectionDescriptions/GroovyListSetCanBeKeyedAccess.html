Informa sobre llamadas al método java.util.List.set(). Estas llamadas pueden ser reemplazadas por una forma de acceso indexado más corta y explícita.

Ejemplo:


def list = ["foo"]
list.set(0, "bar") // list.set(0, "bar") puede reemplazarse por list[0] = "bar"

Después de aplicar la corrección rápida:


def list = ["foo"]
list[0] = "bar"
