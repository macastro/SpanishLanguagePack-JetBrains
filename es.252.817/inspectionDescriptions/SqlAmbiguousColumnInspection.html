Columnas con el mismo nombre que pertenecen a diferentes tablas.

Ejemplo (MySQL):

CREATE TABLE foo(id INT PRIMARY KEY);
CREATE TABLE bar(id INT PRIMARY KEY);
SELECT foo.id, bar.id FROM foo, bar WHERE id > 0;

La columna id aparece en las tablas foo y bar. Necesitas calificar el nombre de la columna para que la consulta sea correcta.

SELECT foo.id, bar.id FROM foo, bar WHERE foo.id > 0;
