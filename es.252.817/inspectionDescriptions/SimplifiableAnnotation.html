Los informes pueden simplificarse a anotaciones en forma abreviada de elemento único o marcador.

Problemas reportados:

  • Redundancia de value= en pares nombre-valor de anotaciones
  • Llaves redundantes que encierran valores de array que contienen un solo valor
  • Espacio redundante entre el símbolo @ y el nombre de la anotación
  • Espacio redundante entre el nombre de la anotación y la lista de parámetros
  • Paréntesis redundantes en anotaciones sin parámetros

Ejemplo:


  @interface Foo { String[] value(); }
  @ Foo({"foo"})
  public String name;

Después de aplicar la corrección rápida:


  @interface Foo { String[] value(); }
  @Foo("foo")
  public String name;
