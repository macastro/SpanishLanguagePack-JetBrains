Reporta anotaciones que se pueden simplificar a su forma abreviada de un solo elemento o marcador.

Problemas reportados:

  • value= redundante en pares nombre-valor de anotaciones
  • Llaves redundantes alrededor de valores de arreglo que contienen solo un valor
  • Espacios en blanco redundantes entre el signo @ y el nombre de las anotaciones
  • Espacios en blanco redundantes entre los nombres de las anotaciones y las listas de parámetros
  • Paréntesis redundantes en anotaciones sin ningún parámetro

Ejemplo:


  @interface Foo { String[] value(); }

  @ Foo({"foo"})
  public String name;

Después de aplicar la corrección rápida:


  @interface Foo { String[] value(); }

  @Foo("foo")
  public String name;
