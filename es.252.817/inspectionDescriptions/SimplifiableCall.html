Informa que la llamada puede reemplazarse por una forma simplificada de llamada a función de biblioteca.

El código puede simplificarse usando la función correspondiente.

Esta corrección rápida permite intercambiar entre diferentes llamadas a funciones.

Ejemplo:


  fun test(list: List<Any>) {
      list.filter { it is String }
  }

Después de aplicar la corrección rápida:


  fun test(list: List<Any>) {
      list.filterIsInstance<String>()
  }
