Reporta comparaciones directas incorrectas de errores, como el uso de ==, y sugiere usar errors.Is en su lugar.

Desde Go 1.13, es posible envolver errores usando la función fmt.Errorf con el verbo %w. Por lo tanto, la comparación directa de errores mediante el operador de igualdad fallará para errores envueltos. La forma recomendada de comprobar un error específico es usar la función errors.Is de la biblioteca estándar, ya que esta función recorre la cadena de errores envueltos al verificar un error específico.

Para más información sobre errors.Is, consulta working with Errors in Go 1.13.

Esta corrección rápida reemplaza la comparación directa de errores con una llamada a errors.Is.

Ejemplo:


if err == MyError {
  fmt.Println("MyErr")
}

Después de aplicar la corrección rápida:


if errors.Is(err, MyError) {
  fmt.Println("MyErr")
}
