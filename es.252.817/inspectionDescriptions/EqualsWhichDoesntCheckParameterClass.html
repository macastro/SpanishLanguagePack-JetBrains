Informa sobre métodos equals() que no verifican el tipo de sus parámetros.

Cuando no se verifica el tipo del parámetro en el método equals(), pueden producirse errores potenciales si el objeto se utiliza en colecciones sin tipo.

Ejemplo:


  class MyClass {
    int x;
    @Override
    public boolean equals(Object obj) {
      // Si obj no es de tipo MyClass, el método equals debería devolver false
      return ((MyClass)obj).x == x;
    }
  }
