Informa sobre métodos y llamadas a constructores que utilizan implícitamente el conjunto de caracteres predeterminado de la plataforma. Estas llamadas pueden producir resultados diferentes en sistemas con distintos conjuntos de caracteres predeterminados y pueden provocar comportamientos inesperados.

Ejemplo:

void foo(byte[] bytes) {
  String s = new String(bytes);
}

Si está disponible la sobrecarga de método correspondiente, se puede aplicar una corrección rápida que especifica explícitamente el conjunto de caracteres UTF-8. Después de aplicar la corrección rápida:

void foo(byte[] bytes) {
  String s = new String(bytes, StandardCharsets.UTF_8);
}
