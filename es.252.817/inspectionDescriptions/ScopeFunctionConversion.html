Informa de las funciones de ámbito (let, run, apply, also) que se pueden convertir entre sí.

El uso de las funciones correspondientes simplifica tu código.

La corrección rápida reemplaza la función de ámbito con otra.

Ejemplo:


  val x = "".let {
      it.length
  }

Tras aplicar la corrección rápida:


  val x = "".run {
      length
  }
