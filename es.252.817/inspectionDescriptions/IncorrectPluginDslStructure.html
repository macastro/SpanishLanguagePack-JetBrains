Detecta declaraciones no permitidas antes del bloque 'plugins {}'.

Debido a las limitaciones del DSL de plugins de Gradle, solo un conjunto restringido de declaraciones Groovy está disponible antes del bloque 'plugins {}'. Las únicas opciones son 'buildscript {}', 'pluginManagement {}' y otros 'plugins {}'.
Consulte la documentación de Gradle

Ejemplo:


import foo.bar.Baz
plugins {} // reporta 'plugins'
plugins {
  foo() // reporta 'foo'
  id 'java'
}
