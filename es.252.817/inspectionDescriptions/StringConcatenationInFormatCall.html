Informa sobre la concatenación de cadenas no constantes utilizadas como argumentos de formato.

Aunque a veces es intencional, esto generalmente es un uso incorrecto de los métodos de formato y puede incluso provocar problemas de seguridad cuando las variables utilizadas en las cadenas concatenadas contienen caracteres especiales como %.

Además, a veces esto puede ser el resultado de concatenar incorrectamente argumentos de formato de cadena al escribir + cuando se debería haber escrito ,.

Ejemplo:


  static String formatGreeting(String userName) {
    return String.format("Hello, " + userName);
  }

Aquí, userName se interpretará como parte de la cadena de formato, lo que podría resultar en una IllegalFormatException (por ejemplo, si userName es "%") o en un uso excesivo de memoria (por ejemplo, si userName es "%2000000000%"). La llamada probablemente debería reemplazarse por String.format("Hello, %s", userName);.

Esta inspección verifica las llamadas a métodos de formato en java.util.Formatter, java.lang.String, java.io.PrintWriter o java.io.PrintStream.
