Informa sobre expresiones que serán del tipo Int, provocando errores de compilación en Kotlin 1.9 y versiones posteriores.

Ejemplo:


  fun takeByte(x: Byte) {}

  fun foo() {
      takeByte(1 + 1) // se resolverá a Int en 1.9
  }

Después de aplicar la corrección rápida:


  fun takeByte(x: Byte) {}

  fun foo() {
      takeByte((1 + 1).toByte()) // se resolverá a Int en 1.9
  }

La inspección está disponible para el lenguaje Kotlin a partir del nivel 1.7.
