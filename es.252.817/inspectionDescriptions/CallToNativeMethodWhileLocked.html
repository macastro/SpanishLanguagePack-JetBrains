Informa de llamadas de métodos nativos dentro de un bloque o método synchronized.

Cuando sea posible, es mejor mantener las llamadas a métodos nativos fuera del contexto sincronizado porque tales llamadas provocan un costoso cambio de contexto y pueden provocar problemas de rendimiento.

Ejemplo:


  native void nativeMethod();

  void example(){
    synchronized (lock){
      nativeMethod();//advertencia
    }
  }
