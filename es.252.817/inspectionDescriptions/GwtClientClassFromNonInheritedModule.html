Informa sobre código del lado del cliente en un módulo GWT que utiliza código de cliente de otro módulo que no está heredado en el módulo actual.

El código del lado del cliente de un módulo GWT solo puede utilizar código de cliente de otro módulo GWT cuando existe la etiqueta inherits correspondiente en el archivo gwt.xml.

La corrección rápida añade la etiqueta inherits necesaria al archivo gwt.xml.
