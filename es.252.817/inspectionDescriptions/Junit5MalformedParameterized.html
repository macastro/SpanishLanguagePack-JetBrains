Reporta pruebas parametrizadas con parámetros de origen mal formados:

  • MethodSource tiene un objetivo desconocido o el método no es estático, no-arg.
  • El tipo ValueSource y EnumSource no se puede convertir a un parámetro de método.
  • Origen no definido.

Ejemplo:


  class Test {
    @MethodSource("parameters")
    @ParameterizedTest
    void foo(String param) {}
  }

Después de aplicar la corrección rápida:


  class Test {
    private static Stream<Arguments> parameters() {
        return Stream.empty();
    }

    @MethodSource("parameters")
    @ParameterizedTest
    void foo(String param) {}
  }
