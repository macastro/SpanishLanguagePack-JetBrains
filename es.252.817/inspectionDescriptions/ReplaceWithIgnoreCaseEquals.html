El informe puede reemplazarse con una comparación sin distinción entre mayúsculas y minúsculas usando equals(..., ignoreCase = true).

Con equals(), las cadenas se pueden comparar sin necesidad de asignar cadenas adicionales con toLowerCase() o toUpperCase().

Esta corrección rápida reemplazará las comparaciones sin distinción entre mayúsculas y minúsculas que usan toLowerCase() o toUpperCase() por equals(..., ignoreCase = true).

Nota: La semántica puede cambiar para algunas configuraciones regionales.

Ejemplo:


  fun main() {
      val a = "KoTliN"
      val b = "KOTLIN"
      println(a.toLowerCase() == b.toLowerCase())
  }

Después de aplicar la corrección rápida:


  fun main() {
      val a = "KoTliN"
      val b = "KOTLIN"
      println(a.equals(b, ignoreCase = true))
  }
