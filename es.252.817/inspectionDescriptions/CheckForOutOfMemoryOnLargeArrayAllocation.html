El informe no verifica las asignaciones de arrays grandes para java.lang.OutOfMemoryError. En entornos con memoria limitada, se debería verificar la asignación de objetos de datos grandes para evitar el agotamiento de memoria.

Esta verificación es aplicable a Java ME y otros entornos con recursos altamente limitados. La aplicación indiscriminada de esta verificación puede tener un impacto negativo en la claridad del código y el diseño.

Esta opción permite especificar el número máximo de elementos permitidos en asignaciones de arrays no verificadas.
