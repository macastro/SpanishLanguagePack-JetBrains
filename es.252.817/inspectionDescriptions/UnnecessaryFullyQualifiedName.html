Informa sobre nombres de clase completamente calificados que pueden acortarse.

La corrección rápida acortará el nombre completamente calificado y añadirá una declaración de importación cuando sea necesario.

Ejemplo:


  class ListWrapper {
    java.util.List<Integer> l;
  }

Después de aplicar la corrección rápida:


  import java.util.List;
  class ListWrapper {
    List<Integer> l;
  }

Configuración de la inspección:

Utilice la opción Ignorar en declaraciones de módulos Java 9 para ignorar los nombres completamente calificados en las declaraciones de módulos provides y uses de Java 9.

Configure la inspección en Configuración | Editor | Estilo de código | Java | Importaciones usando las siguientes opciones:

  • Use la opción Insertar importación para clases internas si las referencias a clases internas deben ser calificadas con la clase externa.
  • Use la opción Usar nombres de clase completamente calificados en JavaDoc para permitir nombres completamente calificados en Javadoc.
