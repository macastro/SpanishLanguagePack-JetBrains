Informa sobre parámetros o variables locales a los que posiblemente se les pueda añadir el modificador final en su declaración.

Ejemplo:


  ArrayList<Integer> list = new ArrayList();
  fill(list);
  return list;

Después de aplicar la corrección rápida:


  final ArrayList<Integer> list = new ArrayList();
  fill(list);
  return list;

Las opciones de esta inspección permiten definir si se deben informar los parámetros o las variables locales.
