Se informa de etiquetas que no apuntan a funciones.

No se permite declarar etiquetas de destino que no representen funciones.

Esta corrección rápida eliminará la etiqueta.

Ejemplo:


  fun testValLabelInReturn() {
      L@ val fn = { return@L }
      fn()
  }

Después de aplicar la corrección rápida:


  fun testValLabelInReturn() {
      L@ val fn = { return }
      fn()
  }

Esta inspección solo se informa cuando el nivel de lenguaje del proyecto o módulo es 1.4 o superior.
