Reporta métodos que tienen tres o más negaciones. Este tipo de métodos pueden resultar confusos.

Ejemplo:


  void doSmth(int a, int b, boolean flag1, boolean flag2) {
    if (!flag && !flag2) {
      if (a != b) {
        doOther();
      }
    }
  }

Sin negaciones, el método se vuelve más fácil de entender:


  void doSmth(int a, int b, boolean flag1, boolean flag2) {
    if (flag1 || flag2 || a == b) return;
    doOther();
  }

Configuración de la inspección:

  • Utilice la opción Ignorar negaciones en el método 'equals()' para deshabilitar la inspección en los métodos equals().
  • Utilice Ignorar negaciones en declaraciones 'assert' para deshabilitar la inspección en las declaraciones assert.
