Informa sobre las comparaciones de .size() o .length() con el literal 0 que pueden reemplazarse por una llamada a .isEmpty().

Ejemplo:


  boolean emptyList = list.size() == 0;

Después de aplicar la corrección rápida:


  boolean emptyList = list.isEmpty();

Usando la lista de clases ignoradas, se pueden añadir clases para las que no se deberían reemplazar las comparaciones de .size() o .length().

La opción ignorar expresiones que se reemplazarían por '!isEmpty()' permite ignorar cualquier expresión que se reemplazaría por !isEmpty().
