Reporta las llamadas a get() en un Optional sin verificar que tenga un valor.

Llamar a Optional.get() en una instancia de Optional vacía, lanzará una excepción.

Ejemplo:


  void x(List<Integer> list) {
    final Optional<Integer> optional =
      list.stream().filter(x -> x > 10).findFirst();
    final Integer result = optional.get(); // problema aquí
  }
