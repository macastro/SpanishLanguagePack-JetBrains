Reporta expresiones binarias, condicionales o instanceof que constan de diferentes operadores sin paréntesis. Estas expresiones pueden ser menos legibles debido a las diferentes reglas de precedencia de los operadores.

Ejemplo:


  int n = 3 + 9 * 8 + 1;

Después de aplicar la corrección rápida:


  int n = 3 + (9 * 8) + 1;
