Informa sobre expresiones binarias, condicionales o instanceof que contienen diferentes operadores sin paréntesis. La legibilidad de tales expresiones puede verse reducida debido a las diferentes reglas de precedencia de operadores.

Ejemplo:


  int n = 3 + 9 * 8 + 1;

Después de aplicar la corrección rápida:


  int n = 3 + (9 * 8) + 1;
