Informa de las llamadas de list.remove(index) dentro de un bucle de conteo ascendente.

Esto es sospechoso ya que la lista se vuelve más corta después de la eliminación y el siguiente elemento se omite. Una solución simple es disminuir la variable de índice después de la eliminación, pero probablemente sea una alternativa más sólida eliminar mediante un iterador o utilizando el método removeIf() (Java 8 y posteriores). Si no espera que remove() se llame más de una vez en un bucle, considere añadir un break después de él.

Ejemplo:

  public static void main(String[] args) {
    process(new ArrayList<>(
      Arrays.asList("1", "2", "|", "3", "4")));
  }

  static void process(List<String> list) {
    for (int i = 0; i < list.size(); i++) {
      if (list.get(i).equals("|")) {
        list.remove(i);
        continue;
      }
      System.out.println(list.get(i));
    }
  }

Parece que el código imprimirá 1 2 3 4 pero, en realidad, 3 se saltará en la salida.
