Informa sobre literales enteros en formato octal. Algunos estándares de codificación prohíben el uso de literales octales porque pueden confundirse fácilmente con literales decimales.

Ejemplo:


  int i = 015;
  int j = 0_777;

Esta inspección ofrece dos correcciones rápidas diferentes. Al aplicar la corrección rápida Convertir literal octal a decimal, el código se convierte en:


  int i = 13;
  int j = 511;

Al aplicar la corrección rápida Eliminar cero inicial para convertir a decimal, el código se convierte en:


  int i = 15;
  int j = 777;
