Informa sobre la instanciación de objetos java.util.Set para tipos enumerados. Estos objetos Set pueden ser reemplazados por objetos java.util.EnumSet.

La implementación de EnumSet es más eficiente que otras colecciones, ya que utiliza un vector de bits como estructura de datos subyacente. Esta corrección rápida puede reemplazar el inicializador con una llamada a EnumSet.noneOf(). Esta corrección rápida no está disponible cuando el tipo de variable es una subclase de Set.

Ejemplo:


  enum MyEnum { FOO, BAR; }
  Set<MyEnum> enums = new HashSet<MyEnum>();

Después de aplicar la corrección rápida:


  enum MyEnum { FOO, BAR; }
  Set<MyEnum> enums = EnumSet.noneOf(MyEnum.class);
