Informa sobre argumentos inválidos pasados a métodos con parámetros anotados como @PropertyKey.

Estos argumentos deben ser claves de propiedades válidas en el archivo de propiedades correspondiente. Además, esta inspección verifica que el argumento resourceBundle de la anotación @PropertyKey se refiera a un paquete de recursos existente.

Utilice la corrección rápida para crear una nueva propiedad o seleccionar una existente.

Ejemplo:


  @PropertyKey(resourceBundle = "myBundle") String value = "invalid.key";
