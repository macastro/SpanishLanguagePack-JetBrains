Reporta comprobaciones de igualdad usando Float.NaN o Double.NaN (deberían reemplazarse por comprobaciones isNaN()).

Según IEEE 754, las comprobaciones de igualdad contra NaN siempre devuelven false, incluso para NaN == NaN. Por lo tanto, es probable que estas comprobaciones sean erróneas.

Esta corrección rápida reemplaza la comparación por una comprobación isNaN() que utiliza una técnica de comparación diferente y maneja correctamente los valores NaN.

Ejemplo:


  fun check(value: Double): Boolean {
      return Double.NaN == value
  }

Después de aplicar la corrección:


  fun check(value: Double): Boolean {
      return value.isNaN()
  }
