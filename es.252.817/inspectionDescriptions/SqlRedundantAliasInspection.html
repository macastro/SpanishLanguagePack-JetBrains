El informe puede superponerse con los nombres de las columnas en la tabla y puede contener expresiones de alias redundantes.

Ejemplo (PostgreSQL):

CREATE TABLE foo(a INT, b INT);
SELECT * FROM foo foo(a, b);
SELECT * FROM foo foo(a);
SELECT * FROM foo foo(x);
SELECT * FROM foo foo(x, y);

Los dos primeros alias utilizan los mismos nombres de columnas que la tabla foo. Estos se consideran redundantes porque las columnas tienen los mismos nombres.
