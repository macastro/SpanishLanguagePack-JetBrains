Informa sobre valores de variables que nunca se utilizan después de su asignación. Esto puede ocurrir en los siguientes casos:
  • La variable nunca se lee después de ser asignada
  • El valor se sobrescribe con otra asignación antes de ser leído
  • El inicializador de la variable es redundante (debido a una de las dos razones anteriores)
  • La variable nunca se utiliza.

Ejemplo:


  class C {
    int field;
    void foo() {
      field = unused_value;
    }
    void bar() {
      use(field);
      foo();
    }
  };

Utilice la casilla de verificación a continuación para hacer que esta inspección ignore los inicializadores de variables.
