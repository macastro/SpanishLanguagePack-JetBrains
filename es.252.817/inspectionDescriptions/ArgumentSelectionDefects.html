Informa de discrepancias entre el nombre del identificador del argumento y el nombre del parámetro formal. Esto puede ayudar a detectar errores de programación, por ejemplo, cuando se intercambian accidentalmente argumentos del mismo tipo. El documento de investigación original se puede encontrar aquí.

Ejemplo:


  const User* get_User(int64_t company_id, int64_t user_id) {
    return new User{company_id, user_id, "foo"};
  }

  void call_User(int64_t company_id, int64_t user_id) {
    const User* user = get_User(user_id, company_id); // posible problema aquí
    //...
  }

Utilice la casilla de verificación siguiente para inspeccionar argumentos con nombres de menos de tres símbolos.
