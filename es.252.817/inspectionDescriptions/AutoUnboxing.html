Informa sobre expresiones afectadas por la conversión de unboxing (conversión automática de objetos a valores primitivos). Es preferible usar primitivos en lugar de objetos. Esto puede tener un impacto significativo en el rendimiento.

Ejemplo:

int x = new Integer(42);

La corrección rápida hará la conversión explícita:

int x = new Integer(42).intValue();

El auto-unboxing fue introducido en Java 5. Esta inspección ayuda con la compatibilidad hacia atrás con versiones anteriores de Java.
