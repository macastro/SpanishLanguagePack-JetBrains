Reporta referencias redundantes de Companion.

Ejemplo:


  class A {
      companion object {
          fun create() = A()
      }
  }
  fun test() {
      val s = A.Companion.create()
  }

Después de aplicar la corrección rápida:


  class A {
      companion object {
          fun create() = A()
      }
  }
  fun test() {
      val s = A.create()
  }
