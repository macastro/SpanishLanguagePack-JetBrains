Informa sobre bucles que pueden ser reemplazados por llamadas a la API de Stream usando expresiones lambda.

Este tipo de reemplazo cambia el estilo imperativo a funcional, haciendo que el código sea más conciso.

Ejemplo:


  boolean check(List<String> data) {
    for (String e : data) {
      String trimmed = e.trim();
      if (!trimmed.startsWith("xyz")) {
        return false;
      }
    }
    return true;
  }

Después de aplicar la corrección rápida:


  boolean check(List<String> data) {
    return data.stream().map(String::trim).allMatch(trimmed -> trimmed.startsWith("xyz"));
  }
