Informa sobre bucles que pueden reemplazarse por llamadas a la API de transmisión que usan expresiones lambda.

Este tipo de reemplazo cambia el estilo imperativo por uno funcional, y el código se vuelve más compacto.

Ejemplo:


  boolean check(List<String> data) {
    for (String e : data) {
      String trimmed = e.trim();
      if (!trimmed.startsWith("xyz")) {
        return false;
      }
    }
    return true;
  }

Después de aplicar la solución rápida:


  boolean check(List<String> data) {
    return data.stream().map(String::trim).allMatch(trimmed -> trimmed.startsWith("xyz"));
  }
