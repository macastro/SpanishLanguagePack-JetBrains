Informa sobre cualquier llamada a métodos sobrescritos de la clase actual durante la construcción del objeto. Esto ocurre dentro de:
  • Constructores
  • Inicializadores de instancia no static
  • Inicializadores de campos no static
  • clone()
  • readObject()
  • readObjectNoData()

Este tipo de llamadas pueden provocar errores difíciles de detectar, ya que no hay garantía de que el objeto esté inicializado antes de que se produzca la llamada al método.

Ejemplo:


  abstract class Parent {
    void someMethod() { }
  }
  class Child extends Parent {
    Child() {
      someMethod();
    }
    @Override
    void someMethod() { }
  }

Esta inspección comparte su funcionalidad con:

  • La inspección de llamadas a métodos abstract durante la construcción del objeto
  • La inspección de llamadas a métodos sobrescribibles durante la construcción del objeto

Solo se debe habilitar una inspección a la vez para evitar advertencias duplicadas.
