Informa sobre cualquier llamada a métodos sobrescritos de la clase actual durante la construcción del objeto. Esto sucede si la construcción de un objeto se encuentra dentro de:
  • Un constructor
  • Un inicializador de instancia no estático
  • Un inicializador de campo no estático
  • clone()
  • readObject()
  • readObjectNoData()

Estas llamadas pueden generar errores sutiles, ya que no se garantiza que el objeto se inicialice antes de que ocurra la llamada al método.

Ejemplo:


  abstract class Parent {
    void someMethod() { }
  }

  class Child extends Parent {
    Child() {
      someMethod();
    }

    @Override
    void someMethod() { }
  }

Esta inspección comparte su funcionalidad con:

  • La inspección Método abstracto llamado durante la construcción del objeto
  • La inspección Método reemplazable llamado durante la construcción del objeto

Solamente una inspección debería estar habilitada a la vez para prevenir advertencias duplicadas.
