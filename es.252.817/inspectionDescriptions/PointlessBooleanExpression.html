Informa de expresiones booleanas innecesarias o demasiado complicadas.

Estas expresiones incluyen && con true, || con false, comparación de igualdad con un literal booleano o negación de un literal booleano. Estas expresiones se pueden simplificar.

Ejemplo: 


  boolean a = !(x && false);
  boolean b = false || x;
  boolean c = x != true;

Después de aplicar la solución rápida:


  boolean a = true;
  boolean b = x;
  boolean c = !x;

Configura la inspección:

Utiliza la opción Ignorar constantes con nombre para determinar expresiones sin sentido para ignorar constantes con nombre al determinar si una expresión no tiene sentido.
