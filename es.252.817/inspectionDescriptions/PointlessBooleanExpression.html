Informa sobre expresiones booleanas innecesarias o excesivamente complejas.

Esto incluye combinaciones de: && con true, || con false, comparaciones de igualdad con literales booleanos o negación de literales booleanos. Estas expresiones se pueden simplificar.

Ejemplo: 


  boolean a = !(x && false);
  boolean b = false || x;
  boolean c = x != true;

Después de aplicar la corrección rápida:


  boolean a = true;
  boolean b = x;
  boolean c = !x;

Configuración de la inspección:

Utilice la opción Ignorar constantes con nombre al determinar expresiones sin sentido para ignorar las constantes con nombre al determinar si una expresión carece de sentido.
