Reporta modificaciones de variables con una asignación simple (tal como y = y + x) que puede ser reemplazada con una asignación de operador.

El arreglo rápido reemplaza la asignación con un operador de asignación.

Ejemplo:


  fun foo() {
      val list = mutableListOf(1, 2, 3)
      list = list + 4
  }

Después de que el arreglo rápido es aplicado:


  fun foo() {
      val list = mutableListOf(1, 2, 3)
      list += 4
  }
