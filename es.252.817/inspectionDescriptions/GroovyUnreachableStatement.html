Informa sobre declaraciones inalcanzables. Esto puede ocurrir cuando una declaración está después de un bucle infinito o después de las declaraciones return, break o continue.

Ejemplo: 


  void foo (int n) {
      if (n < 1) {
          return
          print('This statement is unreachable')
      }
      while (true){
          print ('Hello, world!')
      }
      print('This statement is unreachable too')
  }
