Ejemplo (MySQL):

CREATE TABLE foo(a INT, b INT, c INT);
SELECT * FROM foo ORDER BY a ASC, b DESC, c ASC;

La palabra clave ORDER BY ordena los registros en orden ascendente por defecto. Por lo tanto, la palabra clave ASC para las columnas a y c es redundante.
