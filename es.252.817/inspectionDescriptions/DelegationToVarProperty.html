Delegación de interfaz a la propiedad var

La delegación solo utiliza el valor inicial de la propiedad, y las asignaciones posteriores no la afectan.

Ejemplo:


  class Example(var text: CharSequence): CharSequence by text

Esta corrección rápida reemplazará la propiedad por una propiedad inmutable:


  class Example(val text: CharSequence): CharSequence by text

Si por alguna razón necesita mantener la mutabilidad, puede usar este enfoque alternativo:


  class Example(text: CharSequence): CharSequence by text {
      var text = text
  }
