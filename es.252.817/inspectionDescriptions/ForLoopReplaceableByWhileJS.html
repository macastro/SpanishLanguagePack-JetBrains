El bucle for no contiene ni inicialización ni actualización de componentes. Se recomienda reemplazar este bucle por una sentencia while más simple.

Ejemplo:


  for(; exitCondition(); ) {
    process();
  }

Después de aplicar la corrección rápida, el resultado se verá así:


  while(exitCondition()) {
    process();
  }

Si desea que esta inspección ignore los bucles for que contienen condiciones normales o sin condiciones, utilice la casilla de verificación a continuación.
