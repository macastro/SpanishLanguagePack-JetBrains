Reporta que el tipo de propiedad no coincide con los atributos JPA.

Ejemplo:



  @Entity
  public class JavaEntity {

    @OneToOne
    Map<Integer, AnotherEntity> incorrectRelationship; // Error: el tipo de atributo 'One To One' debe ser una entidad, no un mapa

    @ManyToMany
    Map<Integer, AnotherEntity> correctRelationship;
  }
