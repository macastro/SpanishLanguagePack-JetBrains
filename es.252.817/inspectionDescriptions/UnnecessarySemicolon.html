Informa sobre cualquier punto y coma innecesario, incluyendo los que se encuentran entre miembros de clase, dentro de bloques de declaración o después de definiciones de clase.

Aunque estos puntos y coma son válidos en Java, son redundantes y pueden eliminarse.

Ejemplo:


  class C {
    ;
    void m() throws Exception {
        try (AutoCloseable r1 = createAutoCloseable();) {
          ;
        }
    }
    ;
  }

Después de aplicar la corrección rápida:


  class C {
    void m() throws Exception {
      try (AutoCloseable r1 = createAutoCloseable()) {
      }
    }
  }
