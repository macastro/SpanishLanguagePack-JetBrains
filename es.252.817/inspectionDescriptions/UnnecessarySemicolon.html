Informa cualquier punto y coma innecesario, incluyendo los puntos y comas utilizados entre miembros de una clase, dentro de instrucciones de bloque o después de definiciones de clase.

Incluso si estos puntos y comas son válidos en Java, son redundantes y pueden eliminarse.

Ejemplo:


  clase C {
    ;
    void m() throws Exception {
        try (AutoCloseable r1 = createAutoCloseable();) {
          ;
        }
    }
    ;
  }

Después de aplicar la solución rápida:


  clase C {
    void m() throws Exception {
      try (AutoCloseable r1 = createAutoCloseable()) {
      }
    }
  }
