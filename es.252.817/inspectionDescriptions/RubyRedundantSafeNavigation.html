Informa sobre el uso del operador de navegación segura (&.) con receptores no nulos que pueden ignorarse.

La lista de receptores ignorados permite especificar las llamadas a métodos que se utilizan como receptores que deberían ignorarse (por ejemplo, Foo#call o Foo::call para métodos de instancia o singleton, respectivamente).

Ejemplo:

value = 1
if value&.even?
  # ...
end
