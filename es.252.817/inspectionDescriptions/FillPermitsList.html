Informa sobre clases sealed que no tienen una lista explícita de permits. Las clases sealed que solo tienen subclases en el mismo archivo no necesitan tener una lista explícita de permits. Por defecto, esta inspección no resalta en el editor y solo proporciona una corrección.

Ejemplo:


  sealed class A {}
  final class B extends A {}

Después de aplicar la corrección rápida:


  sealed class A permits B {}
  final class B extends A {}

Novedades en 2020.3
