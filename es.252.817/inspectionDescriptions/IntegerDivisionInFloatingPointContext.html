Se informa de la división de enteros cuando el resultado se utiliza como número decimal. Este tipo de división generalmente indica un error y puede producir resultados inesperados debido al truncamiento en la división de enteros.

Ejemplo:


  float x = 3.0F + 3 * 2 / 5;

Después de aplicar la corrección rápida:


  float x = 3.0F + ((float) (3 * 2)) /5;
