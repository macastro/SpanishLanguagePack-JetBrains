Reporta operaciones inseguras al usar this durante la construcción de objetos, incluyendo:
  • Acceso a propiedades no finales durante la inicialización de la clase: desde constructores o inicializadores de propiedades
  • Llamadas a funciones no finales durante la inicialización de la clase
  • Uso de this como argumento de función en constructores de clases no finales

Si otras clases heredan de la clase dada, es posible que no estén completamente inicializadas cuando se realizan las operaciones inseguras.

Ejemplo:


  abstract class Base {
      val code = calculate()
      abstract fun calculate(): Int
  }
  class Derived(private val x: Int) : Base() {
      override fun calculate() = x
  }
  fun testIt() {
      println(Derived(42).code) // Valor esperado: 42, valor real: 0
  }
