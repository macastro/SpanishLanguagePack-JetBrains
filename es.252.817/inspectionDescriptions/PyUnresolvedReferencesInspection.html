Reporta referencias no resueltas en el código.

En lenguajes de tipado dinámico, esto puede ocurrir en algunos casos.

Si el tipo de referencia es desconocido, sus propiedades no se resaltarán como no resueltas, incluso cuando deberían estarlo:


def print_string(s):
  print(s.abc())

En este código, s es siempre una cadena de texto, y abc debería resaltarse como no resuelto. Sin embargo, el tipo de s se infiere como Any y no se reporta ninguna advertencia.

El IDE proporciona correcciones rápidas para agregar inmediatamente las referencias faltantes.
