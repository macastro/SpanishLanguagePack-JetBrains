Reportar tareas no registradas en la tabla de programación de Celery Beat

Ejemplo:


  #settings.py
  CELERY_BEAT_SCHEDULE = {
      'task_name': {
          'task': 'example_app.tasks.task_name', # path to a task represented by a class or function
          'schedule': 3.0,
      },
  }
  #example_app/tasks.py
  @app.task()
  def taskName():
      pass
  class TaskName(Task):
      pass

Si una clase de tarea no hereda de Task o una función de tarea no tiene el decorador necesario, no se considerará como registrada
