Informa sobre literales enteros octales y decimales mezclados en un único inicializador de array. Esto puede ocurrir cuando se copian listas de números en inicializadores de arrays. Algunos números en el array pueden estar rellenados con ceros, lo que el compilador interpreta como octales.

Ejemplo:

int[] elapsed = {1, 13, 052};

Después de aplicar la corrección rápida para eliminar los ceros iniciales:

int[] elapsed = {1, 13, 52};

Si se trata de un número octal (por ejemplo, después de una variable en línea), puede usar otra corrección rápida para convertir el octal a decimal:

int[] elapsed = {1, 13, 42};
