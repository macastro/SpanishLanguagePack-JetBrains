Informa sobre las implementaciones y llamadas al método clone() y de los usos de la interfaz java.lang.Cloneable.

Algunos estándares de codificación prohíben el uso de clone(), y recomiendan el uso de un constructor de copia o un método fábrica estático en cambio.

La inspección ignora a las llamadas a clone() en arreglos porque es una manera correcta y compacta de copiar un arreglo.

Ejemplo:


  class Copy implements Cloneable /*advertencia*/ {

    public Copy clone() /*advertencia*/ {
      try {
        return (Copy) super.clone(); // advertencia
      } catch (CloneNotSupportedException e) {
        throw new AssertionError();
      }
    }
  }
