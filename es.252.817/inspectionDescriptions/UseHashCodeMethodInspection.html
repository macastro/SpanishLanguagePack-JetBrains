La inspección puede reemplazar operaciones a nivel de bits con llamadas a los métodos Long.hashCode() o Double.hashCode(). Detecta construcciones del tipo (int)(x ^ (x >>> 32)), donde x es una variable de tipo long o el resultado de una llamada previa a Double.doubleToLongBits(). Este reemplazo puede reducir el código y mejorar su legibilidad.

Ejemplo:


  int result = (int)(var ^ (var >>> 32));

Después de aplicar la corrección rápida:


  int result = Long.hashCode(var);

Esta inspección solo se informa cuando el nivel de lenguaje del proyecto o módulo es 8 o superior.
