Informa sobre constructores innecesarios.

Un constructor es innecesario si es el único constructor de la clase, no tiene parámetros, tiene el mismo modificador de acceso que su clase contenedora y no realiza ninguna inicialización excepto la llamada explícita o implícita al constructor de la superclase sin argumentos. Este tipo de constructor puede eliminarse de forma segura ya que será generado por el compilador incluso si no se especifica.

Ejemplo:


  public class Foo {
    public Foo() {}
  }

Después de aplicar la corrección rápida:


  public class Foo {}

Los constructores innecesarios con anotaciones pueden ignorarse usando la configuración de la inspección.
