Reporta los proveedores de datos TestNG con nombres iguales si ha ocurrido org.testng.TestNGException.

Ejemplo:


  public class DuplicatedDataProviders {
    @DataProvider
    public Object[][] intTestData() { // duplicado 1
      return new Integer[][]{
        new Integer[]{1, 1},
      };
    }

    @DataProvider(name = "intTestData")
    public Object[][] someTestData() { // duplicado 2
      return new Integer[][]{
        new Integer[]{1, 1},
      };
    }

    @Test(dataProvider = "intTestData")
    public void testIsOK(Integer key, Integer value) {
      assertEquals(key, value);
    }
  }
