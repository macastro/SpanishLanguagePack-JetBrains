Reporta el uso de miembros importados estáticamente calificados con su nombre de clase contenedor.

Tal calificación es innecesaria y puede ser eliminada ya que los miembros importados estáticamente pueden ser accedidos directamente por nombre de miembro.

Ejemplo:


  import static foo.Test.WIDTH;

  class Bar {
    void bar() {
      System.out.println(Test.WIDTH);
    }
  }

Después de aplicar la corrección rápida:


  import static foo.Test.WIDTH;

  class Bar {
    void bar() {
      System.out.println(WIDTH);
    }
  }
