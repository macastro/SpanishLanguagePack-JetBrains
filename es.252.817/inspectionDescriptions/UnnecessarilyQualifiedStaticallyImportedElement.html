Informa sobre el uso de miembros importados estáticamente con nombres de clase calificados.

Esta calificación es innecesaria y puede eliminarse, ya que los miembros importados estáticamente son accesibles directamente por su nombre.

Ejemplo:


  import static foo.Test.WIDTH;
  class Bar {
    void bar() {
      System.out.println(Test.WIDTH);
    }
  }

Después de aplicar la corrección rápida:


  import static foo.Test.WIDTH;
  class Bar {
    void bar() {
      System.out.println(WIDTH);
    }
  }
