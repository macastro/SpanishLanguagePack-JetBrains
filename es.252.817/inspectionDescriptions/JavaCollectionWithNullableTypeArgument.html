Informa sobre argumentos de tipo anulables en colecciones de Java que no admiten valores null.

Algunas colecciones de Java de la biblioteca estándar no admiten elementos, claves o valores nulos. Los argumentos de tipo de estas colecciones no deberían instanciarse con tipos anulables.

Ejemplo:


val map = ConcurrentHashMap<String, String?>()

Después de aplicar la solución rápida:


val map = ConcurrentHashMap<String, String>()
