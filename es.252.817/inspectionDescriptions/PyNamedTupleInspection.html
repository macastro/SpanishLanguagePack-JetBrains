Definición no válida de typing.NamedTuple.

Ejemplo:


import typing
class FullName(typing.NamedTuple):
    first: str
    last: str = ""
    middle: str

La solución es colocar los campos con valores predeterminados después de los campos sin valores predeterminados:


import typing
class FullName(typing.NamedTuple):
    first: str
    middle: str
    last: str = ""
