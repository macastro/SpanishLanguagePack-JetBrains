El informe puede reemplazar las llamadas a métodos Java con funciones Kotlin equivalentes, como System.out.println().

Reemplazar este código elimina las dependencias de Java y hace que el código Kotlin sea más idiomático.

Esta corrección rápida reemplaza las llamadas a métodos Java con sus equivalentes en Kotlin.

Ejemplo:


  import java.util.Arrays
  fun main() {
      val a = Arrays.asList(1, 3, null)
  }

Después de aplicar la corrección rápida:


  fun main() {
      val a = listOf(1, 3, null)
  }
