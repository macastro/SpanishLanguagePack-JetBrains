Informa de una cadena con comillas simples en un formato de arreglo JSON.

El formato de arreglo JSON debe usar comillas dobles (") alrededor de las palabras, no comillas simples ('). De lo contrario, la compilación de Docker fallará.

Ejemplos:


  # todos los siguientes comandos fallarán.
  RUN ['/bin/bash', '-c', 'echo hello']
  ADD ['binaryA.jar', 'binary2.jar', 'destination/']
  COPY ['binaryA.jar', 'binary2.jar', 'destination/']

Después de aplicar la solución rápida:


  RUN ["/bin/bash", "-c", "echo hello"]
  ADD ["binaryA.jar", "binary2.jar", "destination/"]
  COPY ["binaryA.jar", "binary2.jar", "destination/"]
