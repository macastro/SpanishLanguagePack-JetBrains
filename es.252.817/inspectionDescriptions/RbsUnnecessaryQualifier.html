Informa sobre calificadores innecesarios en las referencias. Algunos calificadores son innecesarios cuando existe una cláusula 'use' o un espacio de nombres.

Se pueden eliminar los calificadores innecesarios mediante una corrección rápida.

Ejemplo:

use Enumerator::Generator
class Example
  # 'Enumerator::Generator' puede acortarse
  def call: -> Enumerator::Generator
end

Después de aplicar la corrección rápida:

use Enumerator::Generator
class Example
  def call: -> Generator
end
