Reporta calificadores innecesarios dentro de las referencias. Algunos calificadores son innecesarios cuando hay cláusulas de uso o espacios de nombres presentes.

Hay una solución rápida disponible para eliminar el calificador innecesario.

Ejemplo:

use Enumerator::Generator
class Example
  # 'Enumerator::Generator' se puede acortar
  def call: -> Enumerator::Generator
end

Después de la solución rápida:

use Enumerator::Generator
class Example
  def call: -> Generator
end
