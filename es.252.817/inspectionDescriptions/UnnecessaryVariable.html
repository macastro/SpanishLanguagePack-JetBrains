Reporta variables locales que se utilizan únicamente en la siguiente instrucción return o en una copia exacta de otra variable.

Estas variables pueden ser inlineadas de forma segura para hacer el código más claro y comprensible.

Ejemplo:


  fun sum(a: Int, b: Int): Int {
      val c = a + b
      return c
  }

Después de aplicar la corrección rápida:


  fun sum(a: Int, b: Int): Int {
      return a + b
  }

Configuración de la inspección:

La opción Reportar variables que se retornan inmediatamente permite reportar variables que se retornan de inmediato. Estas variables pueden mejorar la legibilidad del código en algunos casos cuando se les da un nombre descriptivo, razón por la cual esta opción está desactivada por defecto.
