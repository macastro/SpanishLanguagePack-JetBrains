**Inspección de métodos de instancia con el mismo nombre y la misma cantidad de parámetros que un método en una superclase,** pero donde al menos uno de los parámetros es de un tipo incompatible diferente.

En este caso, el método en una subclase estará sobrecargando el método de la superclase en lugar de sobrescribirlo. Si no es intencional, puede dar lugar a errores latentes.

Ejemplo:


  public class Foo {
    void foo(int x) {}
  }

  public class Bar extends Foo {
    void foo(Number x) {} // El método 'foo()' sobrecarga un método compatible de una superclase,
                          // cuando se podría haber intentado sobrescribirlo
  }

Use la opción para elegir si la inspección también debe informar casos donde los tipos de parámetros no son compatibles.
