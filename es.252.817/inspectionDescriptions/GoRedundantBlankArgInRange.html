Informa las variables en blanco opcionales en los bucles de rango.

Cuando utilizas el bucle range para iterar sobre una porción, se devuelven dos valores para cada iteración. El primero es el número de índice y el segundo es una copia del elemento en ese índice. Si no necesitas el segundo valor, puedes omitir este elemento en lugar de utilizar un identificador en blanco.

Ejemplo:

for a, _ = range v {} // `for a, _ =` es lo mismo que `for a =`

Para eliminar el identificador en blanco, puedes usar la corrección rápida Eliminar argumento en blanco. Una vez aplicada la corrección rápida, el código se verá de la siguiente manera:

for a = range v {}
