Informa sobre problemas de atributos de instancia definidos fuera del método __init__.

Ejemplo:


    class Book:
    def __init__(self):
        self.author = 'Mark Twain'
    def release(self):
        self.year = '1889'

Después de aplicar la corrección rápida, el código de ejemplo se convierte en: 


    class Book:
    def __init__(self):
        self.year = '1889'
        self.author = 'Mark Twain'
    def release(self):
        pass
