Informa sobre retornos o asignaciones desde parámetros de método a arrays o tipos mutables (como Collection, Date, Map, Calendar, etc.)

Debido a que estos tipos son mutables, esta estructura puede llevar a modificaciones inesperadas del estado del objeto desde fuera de su clase. Aunque esta estructura puede ser útil por razones de rendimiento, es inherentemente propensa a errores.

Se informan los siguientes tipos mutables:

  • java.util.Date
  • java.util.Calendar
  • java.util.Collection
  • java.util.Map
  • com.google.common.collect.Multimap
  • com.google.common.collect.Table

La corrección rápida añade una llamada al método .clone() del campo.

Ejemplo:


  class Log {
    String[] messages;
    ...
    String[] getMessages() {
      return messages; // Advertencia: devolviendo campo String[] 'messages'
    }
  }

Después de aplicar la corrección rápida:


  class Log {
    String[] messages;
    ...
    String[] getMessages() {
      return messages.clone();
    }
  }

Use la opción Ignorar asignaciones y retornos en métodos private para ignorar las asignaciones y retornos en métodos private.
