Reporta cadenas largas de transformación de Reactive Streams.

Cada llamada a un método operador, como map() o filter(), crea algunos objetos para estos operadores. Una cadena larga de operadores llamada en cada suscripción para cada elemento del flujo puede resultar en sobrecarga de rendimiento. Para evitar esto, combine cadenas largas de operadores en una sola llamada de operador cuando sea posible.

Ejemplo:


Flux.just(1, 2, 3)
        .map(it -> it + 1)
        .map(it -> it + 2)
        .map(it -> it + 3)

Después de aplicar la corrección rápida:


Flux.just(1, 2, 3)
        .map(it -> it + 1 + 2 + 3)
