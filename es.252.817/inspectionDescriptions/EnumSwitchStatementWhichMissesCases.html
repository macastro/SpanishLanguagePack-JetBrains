Informa sobre declaraciones switch que no manejan todos los valores de un tipo enumerado.

Ejemplo:


  enum AlphaBetaGamma {
    A, B, C;
    void x(AlphaBetaGamma e) {
      switch (e) {
      }
    }
  }

Después de aplicar la corrección rápida:


  enum AlphaBetaGamma {
    A, B, C;
    void x(AlphaBetaGamma e) {
      switch (e) {
        case A -> {}
        case B -> {}
        case C -> {}
      }
    }
  }

Utilice la opción Ignorar declaraciones switch con rama default para ignorar las declaraciones switch que tienen una rama default.
