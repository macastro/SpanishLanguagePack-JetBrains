Informa sobre los usos de primitivas de Guava que pueden migrarse a llamadas de API Java estándar.

Por ejemplo, la inspección informa sobre el uso de clases e interfaces (como FluentIterable, Optional, Function, Predicate o Supplier).

Ejemplo:


  ImmutableList<String> results = FluentIterable.from(List.of(1, 2, 3)).transform(Object::toString).toList();

Después de aplicar la corrección rápida:


  List<String> results = List.of(1, 2, 3).stream().map(Object::toString).collect(Collectors.toList());

La corrección rápida puede cambiar la semántica. Algunos iterables de evaluación perezosa de Guava pueden convertirse en iterables de evaluación anticipada.
