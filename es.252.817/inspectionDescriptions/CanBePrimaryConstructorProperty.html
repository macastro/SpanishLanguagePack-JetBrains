Informa sobre propiedades asignadas explícitamente a parámetros del constructor principal.

Las propiedades se pueden declarar directamente en el constructor principal, reduciendo la cantidad de código y mejorando su legibilidad.

Ejemplo:


  class User(name: String) {
      val name = name
  }

Esta corrección rápida combina la declaración del parámetro y la propiedad en el parámetro del constructor principal:


  class User(val name: String) {
  }
