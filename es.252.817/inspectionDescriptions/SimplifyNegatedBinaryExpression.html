Informa sobre expresiones booleanas negadas que pueden simplificarse.

Esta corrección rápida simplifica la expresión booleana.

Ejemplo:


    fun test(n: Int) {
        !(0 == 1)
    }

Después de aplicar la corrección rápida:


    fun test(n: Int) {
        0 != 1
    }

Tenga en cuenta que esta operación puede alterar la semántica del código cuando se involucran valores IEEE-754 NaN:


    fun main() {
        println(!(Double.NaN >= 0)) // true
    }

Después de aplicar la corrección rápida:


    fun main() {
        println(Double.NaN < 0) // false
    }
