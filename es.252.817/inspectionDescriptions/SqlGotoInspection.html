Informa sobre el uso de las sentencias GOTO hacia atrás y las sentencias GOTO utilizadas para salir de bucles.

Generalmente no se recomienda el uso extensivo de sentencias GOTO. Para más información, consulte GOTO statement in SQL procedures en ibm.com.

Considere usar bucles en lugar de usar GOTO para volver a una sentencia anterior.

Considere usar otra sentencia de control de flujo, como RETURN o BREAK, en lugar de usar GOTO para salir de un bucle WHILE.

Ejemplo (Oracle):

CREATE PROCEDURE test(n INT) AS
DECLARE
    x INT;
BEGIN
    x := 0;
    GOTO a;
    <<a>> x := 1;
    IF (n = 0) THEN
        GOTO a;
    END IF;
    WHILE TRUE
        LOOP
            GOTO b;
        END LOOP;
    <<b>> x := 3;
END;
