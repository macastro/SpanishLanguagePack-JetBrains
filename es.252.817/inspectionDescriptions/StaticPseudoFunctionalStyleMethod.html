API de Java Stream informa el uso de código con funciones simuladas de Guava cuando es posible usar

La API Iterable de Guava ofrece funciones similares a la API de Java Streams pero con algunas diferencias, y es posible que falten funciones. Especialmente, los elementos del flujo especializados en tipos primitivos como IntStream tienen un mejor rendimiento que los elementos genéricos.

Ejemplo:


List<String> transformedIterable = Iterables.transform(someList, someTransformFunction);//ADVERTENCIA: Código con estilo de función simulada

Después de aplicar la corrección rápida:

List<String> transformedIterable = someList.stream().map(someTransformFunction).collect(Collectors.toList());

Nota: Es posible que el significado del código cambie. Por ejemplo, Iterable.transform de Guava genera una iteración con evaluación retrasada pero el código de reemplazo genera una iteración cuya evaluación es anticipada.
