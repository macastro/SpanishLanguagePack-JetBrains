Reporta canales no resueltos en las anotaciones @StreamListener, @SendTo, @Output y @Input.

El uso de nombres de vinculación inválidos resulta en errores 'Bean not found' en tiempo de ejecución.

Ejemplo:


  @EnableBinding(Sink.class)
  public class LogSource {
    @StreamListener(Sink.class)
	  public void log1() {
    }
    @StreamListener("invalid")  // Reporta 'canal no encontrado'
	  public void log2() {
    }
  }
