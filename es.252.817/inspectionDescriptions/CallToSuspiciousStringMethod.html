Informa sobre las llamadas a:
  • equals()
  • equalsIgnoreCase()
  • compareTo()
  • compareToIgnoreCase() y
  • trim()

(realizadas en objetos String). Para la comparación de cadenas internacionalizadas, se debería considerar usar java.text.Collator. String.trim() solo elimina caracteres de control entre 0x00 y 0x20. El método String.strip(), introducido en Java 11, tiene mejor compatibilidad con Unicode y puede usarse como alternativa.
