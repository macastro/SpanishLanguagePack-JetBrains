Informa sobre cualquier método que no requiera crear una instancia de clase y que pueda establecerse como static.

Ejemplo:


class MyClass(object):
    def my_method(self, x):
        print(x)

Si se aplica la corrección rápida Convertir método a función, el código se convertirá en:


def my_method(x):
    print(x)
class MyClass(object):
    pass

Si se elige la corrección rápida Convertir método a static, se añadirá el decorador @staticmethod:


class MyClass(object):
    @staticmethod
    def my_method(x):
        print(x)
