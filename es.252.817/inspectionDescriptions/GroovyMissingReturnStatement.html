Informa sobre métodos con tipo de retorno no void que carecen de una sentencia return al final. El flujo de ejecución del método debe poder llegar hasta el final del mismo.

Ejemplo:


String foo(int a) {
  if (a > 0) {
    return "more than zero"
  }
} // foo(-1) devolverá 'null'
int bar(int a) {
  if (a > 0) {
    return a
  }
} // bar(-1) fallará y lanzará una excepción en tiempo de ejecución
