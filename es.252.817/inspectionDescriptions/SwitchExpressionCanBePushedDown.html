Se informa que cada rama tiene una subexpresión común, por lo que el switch se puede mover dentro de la expresión y declaración switch. Esta operación acorta el código. En muchos casos, es razonable extraer la expresión switch resultante a una variable o método separado.

Ejemplo:


  switch (value) {
    case 0 -> System.out.println("zero");
    case 1 -> System.out.println("one");
    case 2, 3, 4 -> System.out.println("few");
    default -> System.out.println("many");
  }

Después de aplicar la corrección rápida:


  System.out.println(switch (value) {
    case 0 -> "zero";
    case 1 -> "one";
    case 2, 3, 4 -> "few";
    default -> "many";
  });

Esta inspección solo se aplica a las expresiones switch mejoradas que utilizan la sintaxis de flecha.
