Inspecciona expresiones `switch` y sentencias donde se puede mover `switch` hacia adentro porque las ramas contienen subexpresiones comunes. Esta acción acorta el código. En muchos casos, tiene sentido extraer la expresión `switch` resultante en una variable o método separado.

Ejemplo:


  switch (value) {
    case 0 -> System.out.println("cero");
    case 1 -> System.out.println("uno");
    case 2, 3, 4 -> System.out.println("pocos");
    default -> System.out.println("muchos");
  }

Después de aplicar la corrección rápida:


  System.out.println((
    switch (value) {
      case 0 -> "cero";
      case 1 -> "uno";
      case 2, 3, 4 -> "pocos";
      default -> "muchos";
    }
  ));

Esta inspección solo se aplica a switch "mejorados" que utilizan la sintaxis de flecha.
