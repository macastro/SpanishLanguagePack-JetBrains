Reporta una asignación o modificación del parámetro lambda. Aunque a veces sea intencional, este tipo de construcción puede resultar confusa y a menudo ocurre por errores ortográficos o por el uso de una variable equivocada.

La solución rápida agregará una declaración de la nueva variable.

Ejemplo:


  list.forEach(s -> {
    s = s.trim();
    System.out.println("String: " + s);
  });

Tras aplicar la solución rápida:


  list.forEach(s -> {
    String trimmed = s.trim();
    System.out.println("String: " + trimmed);
  });

Utiliza la opción Ignorar si la asignación es una transformación del parámetro original para ignorar las asignaciones que modifican el valor del parámetro según el valor anterior.
