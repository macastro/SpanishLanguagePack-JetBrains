Informa sobre asignaciones o modificaciones a parámetros lambda. Aunque a veces es intencional, esta estructura puede ser confusa y generalmente es el resultado de errores tipográficos o el uso de variables incorrectas.

La corrección rápida añadirá una declaración de nueva variable.

Ejemplo:


  list.forEach(s -> {
    s = s.trim();
    System.out.println("String: " + s);
  });

Después de aplicar la corrección rápida:


  list.forEach(s -> {
    String trimmed = s.trim();
    System.out.println("String: " + trimmed);
  });

Utilice la opción Ignorar si la asignación es una transformación del parámetro original para ignorar las asignaciones que modifican el valor del parámetro basándose en su valor anterior.
