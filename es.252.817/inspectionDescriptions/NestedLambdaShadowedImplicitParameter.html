Reporta lambdas anidadas que tienen parámetros implícitos ocultos.

Ejemplo:


fun foo(listOfLists: List<List<String>>) {
  listOfLists.forEach {
    it.forEach {
      println(it)
    }
  }
}

Después de aplicar la corrección rápida:


fun foo(listOfLists: List<List<String>>) {
  listOfLists.forEach {
    it.forEach { it1 ->
      println(it1)
    }
  }
}
