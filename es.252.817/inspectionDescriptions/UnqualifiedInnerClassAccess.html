Informa de las referencias a clases internas que no están cualificadas con el nombre de la clase que las abarca.

Ejemplo:


  import foo.Foo.Bar;

  class Foo {
    class Bar {}
  }

  class Baz {
    void f(Bar bar) {}
  }

Después de aplicar la solución rápida:


  class Foo {
    class Bar {}
  }

  class Baz {
    void f(Foo.Bar bar) {}
  }

Usa la configuración de la inspección para ignorar referencias a clases internas dentro de la misma clase, que por tanto no requieren una importación.
