Informa sobre clases inline que están obsoletas y generan advertencias de compilación en Kotlin 1.5 y versiones posteriores. Consulta Novedades de Kotlin 1.5.0

Ejemplo:


  inline class Password(val s: String)

Después de aplicar la corrección rápida:


  @JvmInline
  value class Password(val s: String)

La comprobación está disponible para el nivel de lenguaje Kotlin desde la versión 1.5.
