Informa sobre los modificadores de modalidad que coinciden con la modalidad predeterminada del elemento (que es final para la mayoría de los elementos y open para miembros con override).

Ejemplo:


  final class Foo
  open class Bar : Comparable<Bar> {
      open override fun compareTo(other: Bar): Int = 0
  }

Después de aplicar la corrección rápida:


  class Foo
  open class Bar : Comparable<Bar> {
      override fun compareTo(other: Bar): Int = 0
  }
