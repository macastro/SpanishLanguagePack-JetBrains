Reporta declaraciones duplicadas de clases, módulos, interfaces, constantes, variables globales, alias de tipos y métodos.
  • Las declaraciones duplicadas generan un DuplicatedDeclarationError.
  • Las definiciones de métodos duplicados generan un DuplicatedMethodDefinitionError.
  • Las definiciones de métodos de interfaz duplicados generan un DuplicatedInterfaceMethodDefinitionError.

Puedes navegar a las declaraciones duplicadas mediante la corrección rápida.

Ejemplo:

# Declaración de interfaz duplicada
interface _Example
end
# Declaración de interfaz duplicada
interface _Example
end
