El método reportado llama a otro método que a su vez realiza otra llamada. Esta corrección rápida sugiere introducir una variable local.

Ejemplo:


  class X {
    int foo(File f) {
      return f.getName().length();
    }
  }

Después de aplicar la corrección rápida:


  class X {
    int foo(File f) {
      final String name = f.getName();
      return name.length();
    }
  }

Utilice las opciones de inspección para alternar las advertencias para:

  • Llamadas a métodos en cadena en inicializadores de campos, por ejemplo private final int i = new Random().nextInt();

  • Llamadas a métodos en cadena que operan sobre el mismo tipo, por ejemplo, new StringBuilder().append("x: ").append(new X()).append("y: ").append(new Y()).toString();.
