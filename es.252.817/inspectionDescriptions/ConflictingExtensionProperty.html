Informa sobre propiedades de extensión que entran en conflicto con propiedades sintéticas generadas automáticamente desde métodos Java get o set.

Estas propiedades deben eliminarse o renombrarse para evitar que el código se rompa en futuras modificaciones del compilador.

La corrección rápida elimina la propiedad de extensión.

Ejemplo:


  val File.name: String
      get() = getName()

La corrección rápida añade la anotación @Deprecated:


  @Deprecated("Is replaced with automatic synthetic extension", ReplaceWith("name"), level = DeprecationLevel.HIDDEN)
  val File.name: String
      get() = getName()
