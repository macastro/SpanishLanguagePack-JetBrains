Informa sobre paréntesis redundantes en expresiones y tipos.

Ejemplo:

func _(x (int), y ((string))) {
}
func _() {
 _ = (1 + 1)
 _ = (((1 + 1)))
 _ = (((1 + 1))) + (((2 + 2)))
}

Después de aplicar la corrección rápida Eliminar paréntesis:

func _(x int, y string) {
}
func _() {
 _ = 1 + 1
 _ = 1 + 1
 _ = (1 + 1) + (2 + 2)
}
