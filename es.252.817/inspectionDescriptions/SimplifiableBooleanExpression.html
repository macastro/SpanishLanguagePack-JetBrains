Informa sobre expresiones booleanas que pueden simplificarse.

Ejemplo:


  void f(boolean foo, boolean bar) {
    boolean b = !(foo ^ bar);
  }

Después de aplicar la corrección rápida:


  void f(boolean foo, boolean bar) {
    boolean b = foo == bar;
  }

Ejemplo:


  void f(boolean foo, boolean bar) {
    boolean b = (foo && bar) || !foo;
  }

Después de aplicar la corrección rápida:


  void f(boolean foo, boolean bar) {
    boolean b = !foo || bar;
  }
