Reporta referencias no resueltas en comentarios Javadoc.

En el siguiente ejemplo, el parámetro someParam se resaltará debido a que falta:


class A {
    /**
     * @param someParam description
     **/
    void foo() {
    }
}

Deshabilita la opción Reportar símbolos inaccesibles para ignorar las etiquetas que hacen referencia a parámetros de método, clases, campos y métodos faltantes.
