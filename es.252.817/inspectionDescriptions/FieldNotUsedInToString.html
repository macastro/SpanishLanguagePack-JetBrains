Informa sobre campos no utilizados en el método toString() de la clase.

Ayuda a descubrir campos que se han añadido después de la última actualización del método toString(). La corrección rápida regenerará el método toString().

Los campos pueden ser excluidos de esta inspección en el diálogo Generar | toString(). Si la opción habilitar getters en la generación de código está activada, esta inspección también verificará si existen problemas con los métodos getter.

Ejemplo:


  public class Relevant {
    private String name; // No utilizado en toString()
    private int index;
    private int length;
    @Override
    public String toString() {
        return "Relevant{" + "index=" + index +
          ", length=" + length + '}';
    }
  }

Después de aplicar la corrección rápida:


  public class Relevant {
    private String name;
    private int index;
    private int length;
    @Override
    public String toString() {
        return "Relevant{" + "name='" + name + '\'' +
          ", index=" + index + ", length=" + length + '}';
    }
  }
