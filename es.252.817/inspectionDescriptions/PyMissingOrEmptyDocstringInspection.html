Informa sobre docstrings que faltan o vacíos.

Ejemplo de un docstring faltante


def demo(a):
    c = a ** 2

Ejemplo de un docstring vacío


def demo(a):
    """
    """
    c = a ** 2

Cuando se aplica el arreglo rápido, los fragmentos de código cambian a:


def demo(a):
    """

    :param a:
    """
    c = a ** 2

Necesita proporcionar algunos detalles acerca del parámetro en la plantilla generada.
