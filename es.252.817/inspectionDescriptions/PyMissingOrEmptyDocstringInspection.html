Informa sobre docstrings faltantes y vacíos.

Ejemplo de docstring faltante


def demo(a):
    c = a ** 2

Ejemplo de docstring vacío


def demo(a):
    """
    """
    c = a ** 2

Después de aplicar la corrección rápida, el código se convierte en:


def demo(a):
    """
    :param a:
    """
    c = a ** 2

Necesita proporcionar algunos detalles sobre los parámetros en la plantilla generada.
