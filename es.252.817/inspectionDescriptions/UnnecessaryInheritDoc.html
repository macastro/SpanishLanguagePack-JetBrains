El informe solo incluye comentarios Javadoc con la etiqueta {@inheritDoc}. Dado que Javadoc copia los comentarios de la superclase cuando no hay comentarios, los comentarios que solo contienen {@inheritDoc} no aportan ningún valor.

Además, informa sobre el uso de {@inheritDoc} en ubicaciones no válidas (por ejemplo, campos).

Se recomienda eliminar los comentarios Javadoc innecesarios.

Ejemplo:


  class Example implements Comparable<Example> {
    /**
     * {@inheritDoc}
     */
    @Override
    public int compareTo(Example o) {
      return 0;
    }
  }

Después de aplicar la corrección rápida:


  class Example implements Comparable<Example> {
    @Override
    public int compareTo(Example o) {
      return 0;
    }
  }
