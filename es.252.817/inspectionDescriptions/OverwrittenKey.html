Informa sobre código que sobrescribe elementos de un Map, elementos de un Set, elementos de array en una secuencia de llamadas add/put o el uso de métodos de fábrica de Java 9 como Set.of (lo que provocará una excepción en tiempo de ejecución).

Esto suele ser el resultado de errores de copiar y pegar.

Ejemplo:


  map.put("A", 1);
  map.put("B", 2);
  map.put("C", 3);
  map.put("D", 4);
  map.put("A", 5); // Clave "A" duplicada, sobrescribe la entrada anterior
