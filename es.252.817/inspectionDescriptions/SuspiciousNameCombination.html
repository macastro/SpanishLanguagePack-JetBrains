Reporta asignaciones y llamadas a funciones donde el nombre de la variable destino o el parámetro de la función no coincide con el nombre del valor asignado.

Ejemplo 1:


  int x = 0;
  int y = x; // x se usa como coordenada y

Ejemplo 2:


  int x = 0, y = 0;
  // x se usa como coordenada y, y se usa como coordenada x
  Rectangle rc = new Rectangle(y, x, 20, 20);

Configuración de la inspección:

Puedes especificar nombres que no deberían usarse juntos usando el área de Grupos de nombres: si el nombre del parámetro o el nombre del destino de la asignación contiene una palabra de un grupo, y el nombre de la variable asignada o pasada contiene una palabra de otro grupo, se reportará un error.

Puedes especificar métodos que no deberían ser inspeccionados pero que podrían tener nombres sospechosos usando el área de Métodos a ignorar. Por ejemplo, los parámetros de Integer.compare() se llaman x y y, pero no tienen nada que ver con coordenadas.
