Informa sobre variables locales innecesarias que no mejoran la comprensión de la función:
  • Variables locales que se retornan inmediatamente
  • Variables locales que se asignan inmediatamente a otra variable y no se vuelven a utilizar
  • Variables locales que siempre tienen el mismo valor que otra variable local o parámetro.

Utiliza la casilla de verificación a continuación para ignorar las variables que se retornan o lanzan inmediatamente. Algunos estilos de codificación recomiendan usar tales variables para mayor claridad y facilidad de depuración.
