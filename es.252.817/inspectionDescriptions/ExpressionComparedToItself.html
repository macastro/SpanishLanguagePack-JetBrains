Inspecciona comparaciones donde el operando izquierdo y el operando derecho representan la misma expresión. Aunque a veces la comparación de expresiones con ellas mismas puede ser intencionada, normalmente se debe a un descuido.

Ejemplo:


  // Puede que sea intencionado que left.getLength() == right.getLength()
  boolean result = left.getLength() == left.getLength();

Para ignorar comparaciones que pueden tener efectos secundarios, utiliza la opción Ignorar condiciones con efectos secundarios. Deshabilitar esta opción puede provocar falsos positivos, como cuando el mismo método devuelve valores diferentes en llamadas posteriores.

Ejemplo:


  native int unknownMethod();
  
  ...
  
  if (unknownMethod() > unknownMethod()) {
    System.out.println("Lo tengo");
  }

Como `unknownMethod()` puede tener efectos secundarios (en el ejemplo), solo se activará un aviso cuando la opción Ignorar condiciones con efectos secundarios esté deshabilitada.
