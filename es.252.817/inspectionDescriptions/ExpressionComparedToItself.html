Informa sobre comparaciones donde los operandos izquierdo y derecho representan la misma expresión. Aunque a veces puede ser intencional comparar una expresión consigo misma, generalmente es resultado de un descuido.

Ejemplo:


  // Posiblemente left.getLength() == right.getLength() sea intencional
  boolean result = left.getLength() == left.getLength();

Para ignorar las comparaciones que puedan tener efectos secundarios, utilice la opción Ignorar condiciones con efectos secundarios. Deshabilitar esta opción puede resultar en falsos positivos, por ejemplo, cuando el mismo método devuelve diferentes valores en llamadas sucesivas.

Ejemplo:


  native int unknownMethod();
  ...
  if (unknownMethod() > unknownMethod()) {
    System.out.println("Got it");
  }

Dado que unknownMethod() puede tener efectos secundarios (en el ejemplo), la advertencia solo se activará cuando la opción Ignorar condiciones con efectos secundarios esté deshabilitada.
