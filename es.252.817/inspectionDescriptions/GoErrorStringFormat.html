Reporta problemas de formato en las cadenas de error.

Ejemplo:

err := fmt.Errorf("Cannot read the file!")
log.Printf("Reading %s: %v", file, err)

Según Error Strings en github.com/golang, las cadenas de error no deben comenzar con mayúscula ni terminar con signos de puntuación, ya que pueden aparecer en otros contextos.

Para corregir el formato, puedes aplicar la corrección rápida Corregir formato de cadena de error. Después de aplicar la corrección rápida, la cadena de error se verá así:

err := fmt.Errorf("cannot read the file")
log.Printf("Reading %s: %v", file, err)
