Reporta métodos booleanos cuyo nombre no comienza con una palabra interrogativa.

Esta inspección ignorará los métodos booleanos que sobrescriben métodos de biblioteca.

Ejemplo:

boolean empty(List<String> list) {
  return list.isEmpty();
}

La corrección rápida para renombrar tales métodos solo está disponible en el editor.

Configuración de la inspección:

  • Utilice la lista de prefijos para nombres de métodos booleanos para especificar las palabras interrogativas aceptables al inicio de nombres de métodos booleanos.
  • Utilice la opción ignorar métodos con tipo de retorno 'java.lang.Boolean' para ignorar métodos que devuelven java.lang.Boolean.
  • Utilice la opción ignorar métodos booleanos en @interface para ignorar métodos booleanos en tipos de anotación (@interface).
  • Utilice la opción ignorar métodos que sobrescriben/implementan métodos super para ignorar métodos que tienen un método super.
