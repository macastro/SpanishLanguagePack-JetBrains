**Español:** Informa de partes de las expresiones booleanas que siempre son true, siempre son false o redundantes. Este tipo de expresiones booleanas se pueden simplificar para que sea más fácil leer tu código. En algunos casos, también indica que hay otros problemas.

Ejemplo:

func isNonZero(x, y int) bool {
  // La segunda comparación o bien es siempre verdadera,
  // o bien no se ejecuta nunca
  return x > 0 && x > 0
}

Puedes aplicar la corrección rápida Simplificar expresión a la parte x > 0 && x > 0. Después de aplicar la corrección rápida, la expresión será: x > 0.
