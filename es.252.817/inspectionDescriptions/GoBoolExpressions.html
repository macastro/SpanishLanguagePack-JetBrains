Informa sobre partes de expresiones booleanas que son siempre true, siempre false o redundantes. Estas expresiones booleanas pueden simplificarse para mejorar la legibilidad del código. En algunos casos, esto también puede indicar otros problemas.

Ejemplo:

func isNonZero(x, y int) bool {
  // La segunda comparación es siempre true,
  // o nunca se ejecuta
  return x > 0 && x > 0
}

Puede aplicar la corrección rápida Simplificar expresión a la parte x > 0 && x > 0. Después de aplicar la corrección rápida, la expresión se verá así: x > 0.
