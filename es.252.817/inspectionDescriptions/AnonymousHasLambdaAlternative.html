El informe puede convertir clases anónimas en llamadas a constructores o métodos de fábrica con expresiones lambda como argumentos.

Esta inspección reporta las siguientes categorías:

  • Clases anónimas que extienden ThreadLocal con el método initialValue() (pueden reemplazarse con ThreadLocal.withInitial)
  • Clases anónimas que extienden Thread con el método run() (pueden reemplazarse con new Thread(Runnable))

Ejemplo:


  new Thread() {
    @Override
    public void run() {
      System.out.println("Hello from thread!");
    }
  }.start();

Después de aplicar la corrección rápida:


  new Thread(() -> {
    System.out.println("Hello from thread!");
  }).start();
