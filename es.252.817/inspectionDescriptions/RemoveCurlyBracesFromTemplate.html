Informa sobre el uso de llaves que encierran identificadores simples en plantillas de cadenas.

Use la corrección rápida "Eliminar llaves" para eliminar las llaves redundantes.

Ejemplo:


  fun redundant() {
     val x = 4
     val y = "${x}" // <== redundante
  }
  fun correctUsage() {
      val x = "x"
      val y = "${x.length}" // <== correcto
  }

Después de aplicar la corrección rápida:


  fun redundant() {
     val x = 4
     val y = "$x"
  }
  fun correctUsage() {
      val x = "x" <== actualizado
      val y = "${x.length}"
  }
