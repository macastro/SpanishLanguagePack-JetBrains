Reporta paréntesis redondos redundantes en las entradas de anotaciones.

Use la corrección rápida "Eliminar paréntesis innecesarios" para limpiar el código.

Ejemplo:


  annotation class MyAnnotationA
  annotation class MyAnnotationB(val x: Int)
  annotation class MyAnnotationC(val x: Int = 10) // tiene valor predeterminado
  @MyAnnotationA() // <== paréntesis redundantes
  fun testA() {
  }
  @MyAnnotationB() // <== faltan argumentos, paréntesis obligatorios
  fun testB() {
  }
  @MyAnnotationC() // <== paréntesis redundantes
  fun testC() {
  }
