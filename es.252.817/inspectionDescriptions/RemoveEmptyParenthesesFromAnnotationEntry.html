Informa sobre paréntesis vacíos redundantes en entradas de anotación.

Usa la corrección rápida "Eliminar paréntesis innecesarios" para limpiar el código.

Ejemplos:


  annotation class MyAnnotationA
  annotation class MyAnnotationB(val x: Int)
  annotation class MyAnnotationC(val x: Int = 10) // valor predeterminado presente

  @MyAnnotationA() // <== los paréntesis son redundantes
  fun testA() {
  }

  @MyAnnotationB() // <== argumento faltante, los paréntesis son necesarios
  fun testB() {
  }

  @MyAnnotationC() // <== los paréntesis son redundantes
  fun testC() {
  }
