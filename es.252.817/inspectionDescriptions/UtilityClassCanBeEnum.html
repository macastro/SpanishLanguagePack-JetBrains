Informa sobre clases de utilidad que pueden convertirse a enumeraciones.

Algunas guías de estilo de codificación requieren que las clases de utilidad se implementen como enumeraciones para evitar problemas de cobertura de código en constructores private.

Ejemplo:


  class StringUtils {
    public static final String EMPTY = "";
  }

Después de aplicar la corrección rápida:


  enum StringUtils {
    ;
    public static final String EMPTY = "";
  }
