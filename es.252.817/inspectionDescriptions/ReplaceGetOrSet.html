Informa sobre llamadas explícitas a funciones get o set que pueden ser reemplazadas por el operador de índice [].

Kotlin permite implementaciones personalizadas de un conjunto predefinido de operadores para tipos. Para sobrecargar un operador, puede marcar la función correspondiente con el modificador operator: 


  operator fun get(index: Int) {}
  operator fun set(index: Int, value: Int) {}
Estas funciones corresponden al operador de índice.

Ejemplo:

  class Test {
      operator fun get(i: Int): Int = 0
  }
  fun test() {
      Test().get(0) // 'get()' reemplazable
  }

Después de aplicar la corrección rápida:

  class Test {
      operator fun get(i: Int): Int = 0
  }
  fun test() {
      Test()[0]
  }
