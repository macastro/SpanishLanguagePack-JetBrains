Informa sobre llamadas a métodos de interfaces funcionales que se invocan directamente en la definición de lambda, referencia de método o clase anónima. Estas llamadas a métodos se pueden reemplazar con el cuerpo de la implementación de interfaz funcional.

Ejemplo:


  boolean contains(List<String> nombres, String nombre) {
    return ((Predicate<String>)x -> {
      return nombres.contains(x);
    }).test(nombre);
  }

Cuando se aplica el arreglo rápido, la llamada a método cambia a:


  boolean contains(List<String> nombres, String nombre) {
    return nombres.contains(nombre);
  }
