Informa sobre llamadas a funciones con argumentos de tipo que pueden ser inferidos automáticamente. Estos argumentos de tipo son redundantes y pueden omitirse de forma segura.

Utilice la corrección rápida "Eliminar argumentos de tipo explícitos" para limpiar el código.

Ejemplo:


  // Aquí se puede inferir el tipo 'String'
  fun foo(): MutableList<String> = mutableListOf<String>()
  // Aquí no se puede inferir 'String', el argumento de tipo es obligatorio
  fun bar() = mutableListOf<String>()

Después de aplicar la corrección rápida:


  fun foo(): MutableList<String> = mutableListOf() <== actualizado
  fun bar() = mutableListOf<String>()
