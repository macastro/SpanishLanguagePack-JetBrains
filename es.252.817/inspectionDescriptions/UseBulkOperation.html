Informa sobre operaciones individuales dentro de un bucle que pueden ser reemplazadas por métodos por lotes.

Los métodos por lotes no solo son más concisos, sino que a veces también tienen mejor rendimiento.

Ejemplo:

  void test(Collection<Integer> numbers) {
    List<Integer> result = new ArrayList<>();
    for (Integer i : numbers) {
      result.add(i);
    }
  }

Después de aplicar la corrección:


  void test(Collection<Integer> numbers) {
    List<Integer> result = new ArrayList<>();
    result.addAll(numbers);
  }

Aunque los métodos por lotes requieren colecciones, la opción envolver arrays con Arrays.asList() permite informar sobre arrays. En este caso, la corrección rápida envolverá automáticamente el array en una llamada a Arrays.asList().
