Informa sobre módulos que contienen demasiadas clases. Un módulo demasiado grande puede indicar falta de claridad en el diseño. Se contabilizan las clases de Java, Kotlin y Groovy.

Solo se aplica durante Código | Inspeccionar código o Código | Analizar código | Ejecutar inspección por nombre, y no se informa en el editor.

Utilice el campo Número máximo de clases para especificar el número máximo de clases que puede tener un módulo.
