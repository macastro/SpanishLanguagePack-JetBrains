Informa sobre alias de paquetes importados que pueden omitirse.

Normalmente, estos alias son iguales al nombre del paquete importado, por lo que el alias no tiene ninguna función y se puede usar directamente el nombre del paquete.

Ejemplo:

import fmt "fmt"

El alias fmt es redundante con el nombre del paquete "fmt". Para eliminar el alias, utilice la corrección rápida Eliminar alias de import.

Después de aplicar la corrección rápida:

import "fmt"
