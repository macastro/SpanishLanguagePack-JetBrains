Detecta referencias a proveedores de claves de cifrado inexistentes en la configuración de bloques de cifrado de OpenTofu. Por ejemplo, para el código 

  
    terraform {
      encryption {
        key_provider "pbkdf2" "foo" {
          # Configuration parameters
        }
        method "aes_gcm" "yourname" {
          keys = key_provider.pbkdf2.unknown
        }
        method "aes_gcm" "yourname2" {
          keys = key_provider.pbkdf2.foo
        }
      }
    }
debemos verificar si las referencias al proveedor de claves son correctas. En este caso, se debe resaltar la referencia unknown.

Para más detalles, consulte https://opentofu.org/docs/language/state/encryption/#methods.
