Informa sobre la iteración de keySet() en una instancia de java.util.Map, donde las claves iteradas se utilizan para recuperar valores del mapa.

Este tipo de iteración puede ser más eficiente cuando se reemplaza por una iteración de entrySet() o values() (si las claves no se utilizan realmente).

De manera similar, si se recuperan valores en un lambda, keySet().forEach(key -> ...) puede reemplazarse por forEach((key, value) -> ...).

Ejemplo:


  for (Object key : map.keySet()) {
   Object val = map.get(key);
  }

Después de aplicar la corrección rápida:


  for (Object val : map.values()) {}
