Informa de las expresiones literales de función que se pueden sustituir por referencias de función.

Sustituir las lambdas por referencias de función suele hacer que el código parezca más conciso y comprensible.

Ejemplo:


  fun Int.esPar() = this % 2 == 0

  fun example() {
      val numbers = listOf(1, 2, 4, 7, 9, 10)
      val evenNumbers = numbers.filter { it.esPar() }
  }

Después de aplicar la corrección rápida:


  fun Int.esPar() = this % 2 == 0

  fun example() {
      val numbers = listOf(1, 2, 4, 7, 9, 10)
      val evenNumbers = numbers.filter(Int::esPar)
  }
