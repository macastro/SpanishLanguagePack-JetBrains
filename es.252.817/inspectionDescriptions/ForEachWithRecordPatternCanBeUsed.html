Informa sobre declaraciones de variables locales y accesores que pueden ser reemplazados por variables de patrón en declaraciones 'for' mejoradas (que suelen ser más concisas).

Ejemplo:


record Record(Integer x, String y) {
}
public static void test(List<Record> records) {
    for (Record record : records) {
        System.out.println(record.y());
        Integer x = record.x;
        System.out.println(x);
    }
}

Puede ser reemplazado por:


record Record(Integer x, String y) {
}
public static void test(List<Record> records) {
    for (Record(Integer x, String y) : records) {
        System.out.println(y);
        System.out.println(x);
    }
}
  • Utilice la opción Límite de profundidad de anidamiento para especificar el número máximo de patrones de desestructuración anidados a informar
  • Utilice la opción Número máximo de componentes a desestructurar para especificar el número máximo de componentes que puede contener un registro para ser utilizado en un patrón de desestructuración
  • Utilice la opción Número máximo de componentes no utilizados para especificar el número máximo de componentes no utilizados en la declaración for
