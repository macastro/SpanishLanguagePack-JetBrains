Informa declaraciones de variable local y accesores para componentes de registro que pueden reemplazarse con variables de patrón en sentencias `for` mejoradas, las cuales son usualmente más compactas.

Ejemplo:


registro Registro(Entero x, Cadena y) {
}

público estático vacío probar(Lista<Registro> registros) {
    para (Registro registro : registros) {
        Sistema.imprimir(registro.y());
        Entero x = registro.x;
        Sistema.imprimir(x);
    }
}

Puede remplazarse con:


registro Registro(Entero x, Cadena y) {
}

público estático vacío probar(Lista<Registro> registros) {
    para (Registro(Entero x, Cadena y) : registros) {
        Sistema.imprimir(y);
        Sistema.imprimir(x);
    }
}
  • Utiliza la opción Límite de profundidad de anidamiento para especificar el máximo número de patrones de deconstrucción anidados a informar
  • Utiliza la opción Número máximo de componentes de registro para deconstruir para especificar el número máximo de componentes que un registro puede contener para ser usado en patrones de deconstrucción
  • Utiliza la opción Número máximo de componentes de registro no utilizados para especificar el número máximo de componentes que no se utilizan en una sentencia `for`
