Informa sobre las sentencias continue que son las últimas alcanzables en un bucle. Estas sentencias continue son innecesarias y pueden eliminarse de forma segura.

Ejemplo:


  for (String element: elements) {
    System.out.println();
    continue;
  }

Después de aplicar la corrección rápida:


  for (String element: elements) {
    System.out.println();
  }

Esta inspección no analiza archivos JSP.

Utilice la opción Ignorar en la rama 'then' de las sentencias 'if' con rama 'else' para ignorar las sentencias continue que se encuentran en la rama then de una sentencia completa if-else.

Ejemplo:


  for (String element: elements) {
    if(element.isEmpty()) {
      continue;
    } else {
      //...
    }
  }
