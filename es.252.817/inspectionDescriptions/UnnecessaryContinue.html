Informa sobre declaraciones `continue` que son la última sentencia alcanzable en un bucle. Estas declaraciones `continue` son innecesarias y se pueden eliminar de forma segura.

Ejemplo:


  for (String element: elements) {
    System.out.println();
    continue;
  }

Después de aplicar la corrección rápida:


  for (String element: elements) {
    System.out.println();
  }

Esta inspección no analiza archivos JSP.

Usa la opción **Ignorar en la rama then de instrucciones 'if' con rama 'else'** para ignorar instrucciones `continue` que se encuentren en la rama `then` de instrucciones `if`-`else` completas.

Ejemplo:


  for (String element: elements) {
    if(element.isEmpty()) {
      continue;
    } else {
      //...
    }
  }
