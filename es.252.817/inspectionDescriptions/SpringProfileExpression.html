Reporta expresiones @Profile incorrectas:

  • Los perfiles de Spring no pueden estar vacíos
  • Los operadores '&' y '|' no pueden mezclarse sin paréntesis en las expresiones de perfil de Spring

Ejemplos:


  // Configuración correcta
  @Configuration
  @Profile("a & (b | c)")
  public class MyConfiguration {...}
  // Vacío 
  @Configuration
  @Profile() // Reporta "La expresión del perfil debe contener texto"
  public class MyConfiguration {...}
  // Operadores mixtos sin paréntesis 
  @Configuration
  @Profile("a & b | c") // Reporta "Expresión de perfil con formato incorrecto"
  public class MyConfiguration {...}
