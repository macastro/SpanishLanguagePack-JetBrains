Informa sobre las llamadas de aserción de no nulo (!!) que se pueden reemplazar con el operador elvis y return (?: return).

Una aserción de no nulo puede provocar un NPE (NullPointerException) que no se espera. Es una buena práctica evitar el uso de !!.

El arreglo rápido reemplaza la aserción de no nulo con return o return null.

Ejemplo:


  fun test(number: Int?) {
      val x = number!!
  }

Después de aplicar el arreglo rápido:


  fun test(number: Int?) {
      val x = number ?: return
  }
