Informa de las funciones exportadas con tipos de retorno no exportados.

Los tipos no exportados pueden ser difíciles de usar al ver la documentación en go doc.

Ejemplo:

type hidden struct{}

func Exported() hidden { // Función exportada con tipo de retorno no exportado `hidden`
 return hidden{}
}

Se puede aplicar un arreglo rápido Exportar para exportar el tipo. Una vez aplicado el arreglo rápido, el nombre del tipo se escribirá en mayúscula:

type Hidden struct{}

func Exported() Hidden { // Arreglado
 return Hidden{}
}
