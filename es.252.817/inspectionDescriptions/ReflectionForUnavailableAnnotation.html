Informa sobre intentos de realizar comprobaciones reflexivas de la presencia de anotaciones no disponibles en tiempo de ejecución.

El uso de Class.isAnnotationPresent() para comprobar anotaciones con política de retención establecida en SOURCE o CLASS (valor predeterminado) siempre dará como resultado falso. Este error es fácil de pasar por alto.

Ejemplo:


  {
    getClass().isAnnotationPresent(SourceAnnotation.class); //siempre es false
  }
  @Retention(RetentionPolicy.SOURCE)
  @interface SourceAnnotation {}
