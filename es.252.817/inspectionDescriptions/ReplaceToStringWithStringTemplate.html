Reporta llamadas a funciones toString que pueden reemplazarse con una plantilla de cadena.

Usar plantillas de cadena simplifica el código.

La corrección rápida reemplaza toString con una plantilla de cadena.

Ejemplo:


  fun test(): String {
      val x = 1
      return x.toString()
  }

Después de aplicar la corrección rápida:


  fun test(): String {
      val x = 1
      return "$x"
  }
