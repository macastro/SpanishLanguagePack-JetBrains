El informe puede reemplazarse con una llamada a la función toString usando plantillas de cadena.

El uso de plantillas de cadena hace que el código sea más simple.

Esta corrección rápida reemplazará toString con una plantilla de cadena.

Ejemplo:


  fun test(): String {
      val x = 1
      return x.toString()
  }

Después de aplicar la corrección rápida:


  fun test(): String {
      val x = 1
      return "$x"
  }
