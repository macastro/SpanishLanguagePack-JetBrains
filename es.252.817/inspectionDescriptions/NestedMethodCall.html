Informa sobre llamadas a métodos que se utilizan como parámetros en otras llamadas a métodos.

Esta corrección rápida introduce una variable para hacer el código más simple y más fácil de depurar.

Ejemplo:


  public int y() { return 1; }
  public int f(int x) { return 2 * x; }
  public void foo() {
      int x = f(y());
  }

Después de aplicar la corrección rápida:


  public int y() { return 1; }
  public int f(int x) { return 2 * x; }
  public void foo() {
      int y = y();
      int x = f(y);
  }

Las opciones de inspección permiten alternar los informes para:

  • Llamadas a métodos en inicializadores de campos
  • Llamadas a métodos static
  • Llamadas a getters simples
