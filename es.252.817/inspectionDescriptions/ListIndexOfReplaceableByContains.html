Informa sobre cualquier expresión List.indexOf() que se pueda reemplazar con el método List.contains().

Ejemplo:


  boolean hasEmptyString(List<String> list) {
    // Advertencia: se puede simplificar
    return list.indexOf("") >= 0;
  }

El quick-fix proporcionado reemplaza la llamada a indexOf con la llamada a contains:


  boolean hasEmptyString(List<String> list) {
    // Se aplica el quick-fix
    return list.contains("");
  }
