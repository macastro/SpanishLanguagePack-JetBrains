El informe se puede contraer a una expresión if-then dentro de una expresión elvis (?:).

Ejemplo:


  fun maybeFoo(): String? = "foo"
  var foo = maybeFoo()
  val bar = if (foo == null) "hello" else foo

Esta corrección rápida convertirá la expresión if-then en una expresión elvis (?:):


  fun maybeFoo(): String? = "foo"
  var foo = maybeFoo()
  val bar = foo ?: "hello"
