Reporta tipos de parámetros incorrectos en los métodos del repositorio de Micronaut Data.

Ejemplo:


  public class Person {
    private int id;
    private String lastname;
    private Address address;
  }
  public class Address {
    private int zipCode;
  }
  public interface CustomerRepository extends CrudRepository<Person, Integer> {
    List<Person> findByAddress(String lastname);  // debería ser tipo 'Address'
    List<Person> findByAddressZipCode(String str);   // debería ser tipo 'int'
  }
