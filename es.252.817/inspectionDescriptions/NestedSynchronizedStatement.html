Reporta sentencias synchronized anidadas. Es recomendable evitar la sincronización anidada si es posible, porque en algunos casos puede provocar un interbloqueo.

Ejemplo:


  synchronized (lockA){
    // el hilo 1 está esperando por lockB
    synchronized (lockB){ //advertencia
    }
  }
  ...
  synchronized (lockB) {
    // el hilo 2 está esperando por lockA
    synchronized (lockA) { //advertencia
    }
  }
