Informa sobre casos donde se usa el operador spread redundante en una serie de llamadas a la función arrayOf.

El código se puede limpiar usando la corrección rápida "Eliminar operador spread redundante".

Ejemplo:


  fun foo(vararg s: String) { }
  fun bar(ss: Array<String>) {
      foo(*arrayOf("abc"))       // Para ambas llamadas a 'foo', la creación del array
      foo(*arrayOf(*ss, "zzz"))  // y su posterior asignación son redundantes
  }

Después de aplicar la corrección rápida:


  fun foo(vararg s: String) { }
  fun bar(ss: Array<String>) {
      foo("abc")
      foo(*ss, "zzz")
  }
