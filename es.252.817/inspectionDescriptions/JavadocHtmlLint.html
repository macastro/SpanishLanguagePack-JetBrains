Informa sobre los mismos problemas HTML en comentarios Javadoc que reporta DocLint desde Java 8.

Esta inspección detecta los siguientes problemas:

  • Etiquetas autocerrantes, sin cerrar, desconocidas, mal ubicadas o vacías
  • Atributos desconocidos o incorrectos
  • Texto mal ubicado

Ejemplo:


/**
 * Etiqueta desconocida: List<String>
 * Etiqueta sin cerrar: error</b>
 * Texto o etiquetas mal ubicadas: <ul><li>one</li>, <li>two</li></ul>
 * Atributos incorrectos o vacíos: <a href=""></a>
 * Etiqueta autocerrante: <br/>
 * …
 */
void sample(){ }
