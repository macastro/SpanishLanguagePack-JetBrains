El informe sugiere reemplazar la llamada s.substring(0, s.indexOf(x)) por s.substringBefore(x).

Usar substringBefore() hace que el código sea más simple.

Esta corrección rápida reemplazará la llamada substring por substringBefore.

Ejemplo:


  fun foo(s: String) {
      s.substring(0, s.indexOf('x'))
  }

Después de aplicar la corrección rápida:


  fun foo(s: String) {
      s.substringBefore('x')
  }
