Reporta sentencias try que capturan java.lang.Error o cualquiera de sus subclases y no vuelven a lanzar el error.

No se reportan las sentencias que capturan java.lang.ThreadDeath.

Ejemplo:


  try {
    executeTests(request);
  }
  catch (OutOfMemoryError ex) { // Advertencia: el error 'ex' no se vuelve a lanzar
    return false;
  }
