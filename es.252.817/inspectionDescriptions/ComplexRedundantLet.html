Informa sobre llamadas redundantes a let basadas en argumentos.

La función let es redundante cuando el parámetro lambda solo se usa como calificador en una expresión de llamada.

Si necesitas nombrar la expresión calificadora, declara una variable local.

Ejemplo:


  fun splitNumbers() {
      "1,2,3".let { it.split(',') }
  }

La corrección rápida eliminará la llamada let() adicional:


  fun example() {
      "1,2,3".split(',')
  }

O bien:


  fun splitNumbers() {
      val numbers = "1,2,3"
      numbers.split(',')
  }
