Reporta llamadas a containsAll() en java.util.List.

La complejidad temporal de esta llamada a un método es O(n·m), donde n es el número de elementos en la lista en la que se llama al método y m es el número de elementos en la colección pasada para el método como parámetro. Cuando la lista es grande, esta puede ser una operación costosa.

La solución rápida envuelve la lista en new java.util.HashSet<>() ya que el tiempo requerido para crear java.util.HashSet desde java.util.List y ejecutar containsAll() en java.util.HashSet es O(n+m).

Ejemplo:

  public boolean check(List<String> list, Collection<String> collection) {
    // Complejidad O(n·m)
    return list.containsAll(collection);
  }

Después de aplicar la solución rápida:

  public boolean check(List<String> list, Collection<String> collection) {
    // Complejidad O(n+m)
    return new HashSet<>(list).containsAll(collection);
  }
