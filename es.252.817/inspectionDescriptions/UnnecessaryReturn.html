Informa sobre las sentencias return al final de los constructores y métodos que devuelven void. Estas sentencias son redundantes y pueden eliminarse de forma segura.

Esta inspección no informa en archivos JSP.

Ejemplo:


  void message() {
    System.out.println("Hola Mundo");
    return;
  }

Después de aplicar la corrección rápida: 


  void message() {
    System.out.println("Hola Mundo");
  }

Utilice la opción Ignorar en el bloque then de la sentencia 'if' con un bloque 'else' para ignorar las sentencias return en el bloque then de las sentencias if que también tienen un bloque else.
