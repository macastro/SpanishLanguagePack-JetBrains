Informa sobre las declaraciones return al final de constructores y métodos que devuelven void. Estas declaraciones son redundantes y se pueden eliminar de forma segura.

Esta inspección no se informa en archivos JSP.

Ejemplo:


  void message() {
    System.out.println("Hello World");
    return;
  }

Después de aplicar la corrección rápida: 


  void message() {
    System.out.println("Hello World");
  }

La opción Ignorar en la rama 'then' de las declaraciones 'if' con rama 'else' permite ignorar las declaraciones return en la rama then de las declaraciones if que también tienen una rama else.
