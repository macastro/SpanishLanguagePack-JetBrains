Informa sobre un comentario regular que pertenece a un campo, método o clase que puede ser reemplazado por un comentario Javadoc.

Ejemplo:


  public class Main {
    /*
    * Hola,
    */
    // Mundo!
    void f() {
    }
  }

Después de aplicar la corrección rápida:


  public class Main {
    /**
     * Hola,
     * Mundo!
     */
    void f() {
    }
  }
