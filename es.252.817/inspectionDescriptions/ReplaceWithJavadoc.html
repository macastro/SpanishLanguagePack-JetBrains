Informa sobre comentarios regulares que pueden ser reemplazados por comentarios Javadoc en campos, métodos o clases.

Ejemplo:


  public class Main {
    /*
    * Hello,
    */
    // World!
    void f() {
    }
  }

Después de aplicar la corrección rápida:


  public class Main {
    /**
     * Hello,
     * World!
     */
    void f() {
    }
  }
