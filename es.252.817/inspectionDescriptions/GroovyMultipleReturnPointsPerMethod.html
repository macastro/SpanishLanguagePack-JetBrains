Informa sobre métodos con demasiados puntos de retorno. Los métodos con múltiples puntos de retorno pueden ser confusos y difíciles de refactorizar.

Ejemplo:


  int foo(int a) {
    if (a > 0) {
      return a
    }
    if (a < 0) return -a
    return 0
  }

Utilice el campo proporcionado a continuación para especificar el número máximo aceptable de puntos de retorno para un método.
