Informa sobre los modificadores de visibilidad public que se pueden omitir en las propiedades con una visibilidad set reducida. Si no se especifica lo contrario, las propiedades de PHP son implícitamente públicas y de lectura.

La visibilidad asimétrica es compatible con PHP 8.4 y versiones posteriores. Para obtener más información, consulte el PHP RFC: Visibilidad asimétrica v2.
