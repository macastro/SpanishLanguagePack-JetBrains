Reporta tipos explícitos de variables locales que son obvios y redundantes, como en val f: Foo = Foo().

Ejemplo:


  class Point(val x: Int, val y: Int)
  fun foo() {
      val t: Boolean = true
      val p: Point = Point(1, 2)
      val i: Int = 42
  }

Después de aplicar la corrección rápida:


  class Point(val x: Int, val y: Int)
  fun foo() {
      val t = true
      val p = Point(1, 2)
      val i = 42
  }
