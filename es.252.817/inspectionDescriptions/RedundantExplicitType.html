Informa acerca de variables locales con tipos explícitamente indicados que son obvios y por tanto redundantes, como val f: Foo = Foo().

Ejemplo:


class Point(val x: Int, val y: Int)

fun foo() {
    val t: Boolean = true
    val p: Point = Point(1, 2)
    val i: Int = 42
}

Tras aplicar la corrección rápida:


class Point(val x: Int, val y: Int)

fun foo() {
    val t = true
    val p = Point(1, 2)
    val i = 42
}
