Informa sobre cualquier llamada a super(A, B) que no cumpla con los siguientes requisitos:

  • B es una instancia de A
  • B es una subclase de A

Ejemplo:


class Figure:
    def color(self):
        pass
class Rectangle(Figure):
    def color(self):
        pass
class Square(Figure):
    def color(self):
        return super(Rectangle, self).color() # Square no es una subclase de Rectangle
