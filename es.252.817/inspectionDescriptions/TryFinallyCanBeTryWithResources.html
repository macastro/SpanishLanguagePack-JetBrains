Informa sobre sentencias try-finally que podrían utilizar la Gestión Automática de Recursos de Java 7 (lo cual es menos propenso a errores).

La sentencia try-finally puede convertirse en una sentencia try-with-resources mediante una corrección rápida.

Ejemplo:


  PrintStream printStream = new PrintStream(fileName);
  try {
    printStream.print(true);
  } finally {
    printStream.close();
  }

Después de aplicar la corrección rápida:


  try (PrintStream printStream = new PrintStream(fileName)) {
    printStream.print(true);
  }
