Reporta instrucciones `try`-`finally` donde se podría estar usando Automatic Resource Management (Manejo Automático de Recursos) de Java 7, que es menos propenso a errores.

Se puede utilizar el Quick Fix para transformar la instrucción `try`-`finally` en una instrucción `try`-with-resources.

Ejemplo:


  PrintStream printStream = new PrintStream(fileName);
  try {
    printStream.print(true);
  } finally {
    printStream.close();
  }

Se provee un Quick Fix para pasar argumentos al constructor:


  try (PrintStream printStream = new PrintStream(fileName)) {
    printStream.print(true);
  }
