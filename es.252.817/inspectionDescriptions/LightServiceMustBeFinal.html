Informa sobre clases que utilizan la anotación @com.intellij.openapi.components.Service y no son final.

Se recomienda que una clase concreta sea declarada como final.

Ejemplo:


  // MyService.kt
  @Service(Service.Level.APP)
  open class MyService

Después de aplicar la corrección rápida:


  // MyService.kt
  @Service(Service.Level.APP)
  class MyService

Para clases abstractas o interfaces, se recomienda eliminar la anotación @Service.

Ejemplo:


  // MyService.java
  @Service(Service.Level.APP)
  abstract class MyService {}

Después de aplicar la corrección rápida:


  // MyService.java
  abstract class MyService {}
