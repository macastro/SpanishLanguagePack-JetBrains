Se han detectado los siguientes problemas al recuperar el servicio:
  • Intento de recuperar un servicio no registrado.
  • Discrepancia al recuperar el servicio: intento de obtener un servicio a nivel de proyecto como servicio a nivel de aplicación, o viceversa.

Ejemplo (Kotlin):


@Service
class MyAppService
@Service(Service.Level.PROJECT)
class MyProjectService(private val project: Project)


// Error:
val projectService = service<MyProjectService>()         // Servicio a nivel de proyecto recuperado como servicio a nivel de aplicación
val applicationService = project.service<MyAppService>() // Servicio a nivel de aplicación recuperado como servicio a nivel de proyecto


// Correcto:
val projectService = project.service<MyProjectService>()
val applicationService = service<MyAppService>();
