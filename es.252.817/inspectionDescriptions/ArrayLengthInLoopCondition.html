Reporta los accesos a la propiedad .length de un array en la parte de condición de una sentencia de bucle. En entornos muy restringidos en recursos, dichas llamadas pueden tener implicaciones adversas en el rendimiento.

Esta inspección está pensada para Java ME y otros entornos muy restringidos en recursos. Aplicar los resultados de esta inspección sin consideración podría tener efectos negativos en la claridad del código y el diseño.

Ejemplo:


  void foo(Object[] x) {
    for (int i = 0; i < x.length; i++) { /**/ }
  }
