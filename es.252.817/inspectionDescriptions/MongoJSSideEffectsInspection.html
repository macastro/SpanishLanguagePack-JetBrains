Informa sobre sentencias que podrían causar efectos secundarios cuando la fuente de datos está en modo de solo lectura.

Para obtener más información sobre cómo habilitar el modo de solo lectura, consulte "Habilitar modo de solo lectura para una conexión" en la documentación del IDE.

La corrección rápida Deshabilitar modo de solo lectura desactivará el modo de solo lectura para la fuente de datos correspondiente.

Ejemplo:


db.my_collection.insertOne()
