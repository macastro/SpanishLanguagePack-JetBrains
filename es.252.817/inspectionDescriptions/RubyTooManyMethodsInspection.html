Inspecciona clases/módulos con demasiados métodos. Esto usualmente significa que intentan 'hacer demasiado', y pueden necesitar ser refactorizados en componentes más pequeños.

Utiliza el campo Número de métodos para especificar el número máximo de métodos que una clase/módulo puede tener.

Utiliza la caja de verificación Ignorar clases de prueba para ignorar clases de prueba.

Inspirado 'Reek Code Smells'
