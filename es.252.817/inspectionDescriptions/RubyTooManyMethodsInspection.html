Reporta clases/módulos con demasiados métodos. Esto generalmente indica que están intentando "hacer demasiado" y podrían necesitar ser refactorizados en varios componentes más pequeños.

Utilice el campo número de métodos para especificar el número máximo de métodos permitidos en una clase/módulo.

Utilice la casilla de verificación ignorar clases de prueba para excluir las clases de prueba.

Inspirado por 'Reek Code Smells'
