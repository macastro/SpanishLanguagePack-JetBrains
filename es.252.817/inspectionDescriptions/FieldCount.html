Reporta las clases que exceden el número máximo especificado de campos.

Las clases con muchos campos generalmente intentan hacer demasiadas cosas. Considera dividir esta clase en varias clases más pequeñas.

Configuración de la inspección:

  • Usa el campo Límite de recuento de campos para especificar el número máximo permitido de campos en una clase.
  • Usa la opción Incluir campos constantes en el recuento para indicar si se deben contar los campos constantes.
  • Por defecto, solo los objetos static final inmutables se cuentan como constantes. Usa la opción Contar campos 'static final' como constantes para contar cualquier campo static final como constante.
  • Usa la opción Incluir constantes enum en el recuento para especificar si se deben contar las constantes enum en las clases enum.
