Informa situaciones cuando usas palabras clave reservadas SQL como nombres de identificador en tu consulta.

Ejemplo (Microsoft SQL Server):

CREATE TABLE select (identity INT IDENTITY NOT NULL, order INT NOT NULL);

Usamos select, identity y order como nombres de tabla y columna. Pero también son palabras clave reservadas en Microsoft SQL Server. Por lo tanto, para usarlos como nombres de objeto en la consulta, debes citar estos identificadores. Para citarlos, puedes usar el quick-fix Citar identificador.

Después de aplicar el quick-fix:

CREATE TABLE [select] ([identity] INT IDENTITY NOT NULL, [order] INT NOT NULL);
