Informa sobre object que se pueden convertir a data object

Los data object generan automáticamente toString, equals y hashCode

Esta inspección sugiere convertir un object a data object en los siguientes 2 casos:

  • Cuando un toString personalizado devuelve el nombre de la clase
  • Cuando el object hereda de una sealed class/interface

Ejemplo:


    object Foo {
        override fun toString(): String = "Foo"
    }

Después de aplicar la corrección rápida:


    data object Foo

Esta inspección solo se activa cuando el nivel del lenguaje Kotlin del proyecto o módulo es 1.9 o superior
