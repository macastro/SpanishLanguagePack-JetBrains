Informa sobre referencias inversas que no se pueden resolver en tiempo de ejecución. Esto significa que la referencia inversa nunca podrá coincidir con ningún contenido. Una referencia inversa no será resoluble si el grupo se define después de la referencia inversa, o si el grupo se define en una rama diferente de una alternativa.

Ejemplo de un grupo definido después de su referencia inversa:


  \1(abc)

Ejemplo de un grupo y referencia inversa en diferentes ramas:


  a(b)c|(xy)\1z
