Inspecciona las llamadas de `String.startsWith()` y `String.endsWith()` que reciben un literal de cadena de un único carácter como argumento.

Se sugiere una corrección rápida para reemplazar estas llamadas por `String.charAt()`, una implementación más eficiente.

No obstante, la mejora del rendimiento que aporta este cambio es mínima y añade una comprobación de longitud distinta de cero, lo que reduce la legibilidad. Por lo tanto, se recomienda aplicar esta corrección rápida únicamente en bucles estrictos.

Esta inspección se ha diseñado para entornos como Java ME o para otros casos donde el uso de recursos está muy restringido. Aplicar los resultados de esta inspección sin pensarlo puede tener un impacto negativo en la claridad y en el diseño del código.

Ejemplo:


  boolean empiezaPorX(String s) {
    return s.startsWith("x");
  }

Después de aplicar la corrección rápida:


  boolean empiezaPorX(String s) {
    return !s.isEmpty() && s.charAt(0) == 'x';
  }
