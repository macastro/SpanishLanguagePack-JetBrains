Informa sobre la instanciación de objetos java.util.Map donde el tipo de clave es una clase enum. Estos objetos java.util.Map pueden ser reemplazados por objetos java.util.EnumMap.

La implementación de java.util.EnumMap puede ser más eficiente ya que utiliza un simple array como estructura de datos subyacente.

Ejemplo:


  Map<MyEnum, String> myEnums = new HashMap<>();

Después de aplicar la corrección rápida:


  Map<MyEnum, String> myEnums = new EnumMap<>(MyEnum.class);
