Informa sobre concatenaciones de String en bucles.

Dado que cada concatenación de String copia toda la cadena, generalmente es mejor reemplazarla con llamadas explícitas a StringBuilder.append() o StringBuffer.append().

Ejemplo:


  String str = "";
  for(int i=0; i<10; i++) {
    str += i;
  }

Después de aplicar la corrección rápida:


  String str = "";
  StringBuilder strBuilder = new StringBuilder(str);
  for(int i = 0; i<10; i++) {
    strBuilder.append(i);
  }
  str = strBuilder.toString();

En ocasiones, esta corrección rápida puede convertir una variable String en StringBuilder o introducir un nuevo StringBuilder. Se debe tener cuidado cuando el código original maneja específicamente valores null, ya que el reemplazo podría cambiar la semántica. Si los valores nulos son posibles, se recomienda realizar una corrección segura para nulos, generando las comprobaciones necesarias. Además, no hay garantía de que el reemplazo automático siempre resulte en un mejor rendimiento.
