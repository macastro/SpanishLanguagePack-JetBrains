Informa sobre llamadas a métodos sobrescribibles de la clase actual durante la construcción del objeto.

Se considera que un método es llamado durante la construcción del objeto si se encuentra en:

  • Un constructor
  • Un inicializador de instancia no static
  • Un inicializador de campo no static
  • El método clone()
  • El método readObject()
  • El método readObjectNoData()

    • Un método es sobrescribible si no está declarado como final, static o private. Los métodos package-local se consideran seguros incluso si son sobrescribibles. Tales llamadas pueden provocar errores difíciles de detectar, ya que la inicialización del objeto puede ocurrir después de la llamada al método.

    Ejemplo:

    
  class Parent {
    void someMethod() { }
  }
  class Child extends Parent {
    Child() {
      someMethod();
    }
  }

    Esta inspección comparte funcionalidad con las siguientes inspecciones:

    • Método abstract llamado durante la construcción del objeto
    • Método sobrescrito llamado durante la construcción del objeto

    Solo se puede habilitar una inspección a la vez para evitar advertencias duplicadas.
