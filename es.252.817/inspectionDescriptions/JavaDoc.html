Ofrece los siguientes defectos en los comentarios Javadoc:
  • No hay Javadoc donde se necesita
  • Faltan etiquetas requeridas
  • Las etiquetas son inválidas o incompletas
  • La descripción de Javadoc falta o está incompleta

Ejemplo:


  class Main {
  }

Después de aplicar la reparación rápida:


  /**
   *
   */
  class Main {
  }

Ejemplo:


  String id(String value) {
    return value;
  }

Después de aplicar la reparación rápida:


  /**
   * @param value
   * @return
   */
  String id(String value) {
    return value;
  }
Utiliza pestañas y la opción de inspección correspondiente para definir el alcance de visibilidad necesario para Javadoc, así como las etiquetas Javadoc necesarias para clases, métodos, campos y clases internas.
