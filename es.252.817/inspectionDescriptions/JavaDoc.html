Reporta las siguientes deficiencias en los comentarios Javadoc:
  • Falta de Javadoc donde se requiere
  • Ausencia de etiquetas requeridas
  • Etiquetas inválidas o incompletas
  • Descripción Javadoc faltante o incompleta

Ejemplo:


  class Main {
  }

Después de aplicar la corrección rápida:


  /**
   *
   */
  class Main {
  }

Ejemplo:


  String id(String value) {
    return value;
  }

Después de aplicar la corrección rápida:


  /**
   * @param value
   * @return
   */
  String id(String value) {
    return value;
  }
Use la pestaña y las opciones de inspección correspondientes para definir el nivel de visibilidad que requiere Javadoc, así como las etiquetas Javadoc necesarias para clases, métodos, campos y clases internas.
