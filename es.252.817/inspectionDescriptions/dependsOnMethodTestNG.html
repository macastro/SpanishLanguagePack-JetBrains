Se ha proporcionado un nombre de método no válido al atributo dependsOnMethods de la anotación @Test.

Un nombre de método se considera no válido si no puede resolverse como un método accesible y válido anotado con @Test en la clase actual o en cualquiera de sus clases padre.

Ejemplo:


  public class SampleTest {
    @Test(dependsOnMethods = "testSpellignError")
    public void testSample() {}
    @Test
    public void testSpellingError(){}
  }
