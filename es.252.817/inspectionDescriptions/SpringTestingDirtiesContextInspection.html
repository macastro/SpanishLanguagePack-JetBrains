Informa de una configuración incorrecta del "modo" en la @DirtiesContext anotación.

Ejemplo:


  @ContextConfiguration
  @DirtiesContext(methodMode = MethodMode.AFTER_METHOD, // Establecer el modo método en una clase de prueba anotada no tiene sentido. Para el control a nivel de clase, utilice en su lugar classMode.
                  hierarchyMode = DirtiesContext.HierarchyMode.CURRENT_LEVEL) // hierarchyMode debe usarse cuando el contexto se configura como parte de una jerarquía a través de @ContextHierarchy
  public class MyTest {
    @DirtiesContext(classMode = DirtiesContext.ClassMode.AFTER_CLASS, // Establecer el modo clase en un método de prueba anotado no tiene sentido. Para el control a nivel de método, utilice en su lugar methodMode.
                     hierarchyMode = DirtiesContext.HierarchyMode.CURRENT_LEVEL) // hierarchyMode debe usarse cuando el contexto se configura como parte de una jerarquía a través de @ContextHierarchy
    public void testFoo() {...}
  }
