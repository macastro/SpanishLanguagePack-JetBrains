Reporta referencias incorrectas de Beans en el parámetro value de la anotación @Lookup.

Ejemplo:


  @Component public class FooBean {...}
  @Component public class OtherBean {...}
  @Component public class MyComponent {
  @Lookup("fooBean")
  public FooBean fooBean() {...}
  @Lookup("fooBean") // Reporta "El Bean debe ser de tipo 'OtherBean'"
  public OtherBean otherBean() {...}
  @Lookup("unknown") // Reporta "No se puede resolver el Bean 'unknown'"
  public OtherBean fooBean() {...}
  }
