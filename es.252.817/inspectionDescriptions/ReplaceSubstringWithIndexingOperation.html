La llamada "abc".substring(0, 1) puede reemplazarse por "abc"[0].

Obtener elementos por índice hace que el código sea más simple.

Esta corrección rápida reemplazará la llamada substring por el operador de índice.

Ejemplo:


  fun foo() {
      "abc".substring(0, 1)
  }

Después de aplicar la corrección rápida:


  fun foo() {
      "abc"[0]
  }
