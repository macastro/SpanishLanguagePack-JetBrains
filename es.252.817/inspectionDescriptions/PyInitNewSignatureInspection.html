Informa sobre firmas incompatibles entre los métodos __new__ y __init__.

Ejemplo:


class MyClass(object):
    def __new__(cls, arg1):
        return super().__new__(cls)
    def __init__(self):
        pass

No es posible instanciar MyClass si __new__ y __init__ tienen diferentes argumentos.

El IDE sugiere aplicar la refactorización "Cambiar firma" como solución.
