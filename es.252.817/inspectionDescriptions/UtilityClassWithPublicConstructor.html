Informa sobre clases de utilidad que contienen constructores public.

Las clases de utilidad declaran todos sus campos y métodos como static. Crear constructores public en estas clases es confuso y podría llevar a una instanciación accidental de la clase.

Ejemplo:


  public final class UtilityClass {
    public UtilityClass(){
    }
    public static void foo() {}
  }

Después de aplicar la corrección rápida:


  public final class UtilityClass {
    private UtilityClass(){
    }
    public static void foo() {}
  }
