Informa sobre bucles que contienen una sentencia if y terminan con break sin alterar la semántica. Esto puede evitar iteraciones redundantes en el bucle.

Ejemplo:


  boolean found = false;
  for (int i = 0; i < arr.length; i++) {
    if (Objects.equals(value, arr[i])) {
      found = true;
    }
  }

Después de aplicar la corrección rápida:


  boolean found = false;
  for (int i = 0; i < arr.length; i++) {
    if (Objects.equals(value, arr[i])) {
      found = true;
      break;
    }
  }
