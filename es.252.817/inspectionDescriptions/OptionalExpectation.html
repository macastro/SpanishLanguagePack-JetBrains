Informa sobre anotaciones esperadas opcionales en algunos módulos de plataforma donde no hay anotaciones reales.

Ejemplo:

// 通用代码
@OptionalExpectation
expect annotation class JvmName(val name: String)
@JvmName(name = "JvmFoo")
fun foo() { }
// jvm 代码
actual annotation class JvmName(val name: String)

Esta inspección también informa sobre la omisión de actual annotation class JvmName para plataformas que no son JVM (por ejemplo, Native).
