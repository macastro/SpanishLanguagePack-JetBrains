Informe sobre concatenaciones de String que se pueden simplificar usando bloques de texto.

Requisitos:

  • \n aparece dos o más veces.
  • El bloque de texto no está concatenado.

La opción Informar literales de cadena individuales permite resaltar literales individuales que contienen saltos de línea. La corrección rápida sigue estando disponible incluso cuando esta opción está desactivada.

Ejemplo:


  String html = "<html>\n" +
                "    <body>\n" +
                "        <p>Hello, world</p>\n" +
                "    </body>\n" +
                "</html>\n";

Después de aplicar la corrección rápida:


  String html = """
                <html>
                    <body>
                        <p>Hello, world</p>
                    </body>
                </html>
                """;
