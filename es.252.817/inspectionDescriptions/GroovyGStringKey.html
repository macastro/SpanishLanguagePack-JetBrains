Informa sobre las declaraciones que utilizan un objeto groovy.lang.GString como clave para el mapa. En general, los objetos GString son mutables y probablemente no deberían utilizarse como claves. Además, no se puede acceder a una entrada de GString con un objeto java.lang.String con el mismo valor.

Ejemplo:


def mapa = [:]
def clave = 'foo'
mapa << ["${clave}": 'bar']
assert mapa[clave] == null // resultado 'true' confuso de la comparación
