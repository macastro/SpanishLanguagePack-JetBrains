La expresión puede ser reemplazada por una llamada a la función to en forma infija.

El uso de la forma infija hace que el código sea más simple.

Esta corrección rápida reemplazará to con su forma infija.

Ejemplo:


  fun foo(a: Int, b: Int) {
      val pair = a.to(b)
  }

Después de aplicar la corrección rápida:


  fun foo(a: Int, b: Int) {
      val pair = a to b
  }
