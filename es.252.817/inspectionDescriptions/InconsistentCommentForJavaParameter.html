Informa de los nombres de parámetros incoherentes para llamadas a métodos de Java especificados en un bloque de comentarios.

Ejemplos:


  // Java
  public class JavaService {
      public void invoke(String command) {}
  }


  // Kotlin
  fun main() {
      JavaService().invoke(/* name = */ "fix")
  }

La corrección rápida modifica el nombre del parámetro en el bloque de comentarios:


  fun main() {
      JavaService().invoke(/* command = */ "fix")
  }
