Reporta sincronización sobre expresiones this o class. Las construcciones reportadas incluyen bloques synchronized y llamadas a métodos wait(), notify() o notifyAll().

Existen varias razones por las que la sincronización sobre las expresiones this o class pueden ser contraproducentes:

  1. hace que la sincronización sea parte de la interfaz externa de la clase, lo que dificulta un cambio futuro a un mecanismo de bloqueo diferente,
  2. se vuelve difícil rastrear quién está bloqueando un objeto determinado,
  3. hace posible un ataque de denegación de servicio, ya sea de forma intencional o puede suceder fácilmente por accidente cuando se realiza la subclasificación.

Como alternativa, considere la posibilidad de sincronizar sobre un objeto de bloqueo private final, cuyo acceso puede controlarse por completo.

Ejemplo:


  public void print() {
    synchronized(this) { // Advertencia: las operaciones de bloqueo sobre 'this' pueden tener efectos colaterales imprevistos
      System.out.println("synchronized");
    }
  }
