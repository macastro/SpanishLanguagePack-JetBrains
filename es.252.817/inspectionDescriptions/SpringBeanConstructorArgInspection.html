Informa sobre los Spring <bean> que utilizan la inyección de dependencias basada en constructores.

La inspección se activa cuando no puede encontrar el constructor o método de fábrica apropiados para <bean> con las etiquetas <constructor-arg> configuradas y la política de autowire definida.

Ejemplo:


    public class MyComponent {
    // constructor
    public MyComponent(MyBean bean) {}
    // método de fábrica
    public static MyComponent getInstance(String name, int port) {
    }


    <beans>
      <bean class="beans.MyComponent" id="foo"> <!-- informa "No se encontró un constructor coincidente en la clase 'MyComponent" -->
         <constructor-arg ref="myBean"/>
         <constructor-arg value="123"/>
      </bean>

      <bean class="beans.MyComponent" factory-method="getInstance"> <!-- informa "No se encontró un método de fábrica coincidente en la clase" -->
         <constructor-arg value="123"/>
      </bean>
    </beans>
