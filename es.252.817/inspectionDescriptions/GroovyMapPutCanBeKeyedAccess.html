Informa sobre las llamadas al método java.util.Map.put(). Estas llamadas pueden ser reemplazadas por una forma de acceso con clave más corta y explícita.

Ejemplo:


def map = ["foo": "bar"]
map.put("foo", "baz") // map.put("foo", "baz") puede reemplazarse por map["foo"] = "baz"

Después de aplicar la corrección rápida:


def map = ["foo": "bar"]
map["foo"] = "baz"
