Se informa una asignación compuesta cuando el tipo del operando derecho es incompatible con el tipo de la variable.

Durante este tipo de asignación compuesta, ocurre una conversión implícita que puede resultar en una conversión con pérdida de datos.

Ejemplo:


  long c = 1;
  c += 1.2;

Después de aplicar la corrección rápida:


  long c = 1;
  c += (long) 1.2;
