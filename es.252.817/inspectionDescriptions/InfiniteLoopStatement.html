Reporta declaraciones for, while o do que solo pueden terminar al lanzar una excepción. Si bien estas declaraciones pueden ser correctas, a menudo ocurren debido a errores de codificación.

Ejemplo:


  for (;;) {
  }

Utilice la opción Ignorar cuando se coloca en Thread.run para ignorar las declaraciones de bucle infinito dentro de Thread.run. Puede resultar útil para los subprocesos de demonio.

Ejemplo:


  new Thread(() -> {
    while (true) {
    }
  }).start();
