Informa sobre sentencias for, while o do que solo pueden salir mediante una excepción. Aunque estas sentencias pueden ser correctas, frecuentemente son el resultado de errores de programación.

Ejemplo:


  for (;;) {
  }

Utiliza la opción Ignorar cuando está en Thread.run para ignorar bucles infinitos en Thread.run. Esto puede ser útil para hilos daemon.

Ejemplo:


  new Thread(() -> {
    while (true) {
    }
  }).start();
