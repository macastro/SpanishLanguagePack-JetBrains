Reporta funciones que coinciden con una de las convenciones de operadores pero carecen de la palabra clave operator.

Al agregar el modificador operator, puede permitir que los usuarios de la función escriban código Kotlin idiomático.

Ejemplo:


  class Complex(val real: Double, val imaginary: Double) {
      fun plus(other: Complex) =
          Complex(real + other.real, imaginary + other.imaginary)
  }
  fun usage(a: Complex, b: Complex) {
      a.plus(b)
  }

La corrección rápida agregará la palabra clave del modificador operator:


  class Complex(val real: Double, val imaginary: Double) {
      operator fun plus(other: Complex) =
          Complex(real + other.real, imaginary + other.imaginary)
  }
  fun usage(a: Complex, b: Complex) {
      a + b
  }
