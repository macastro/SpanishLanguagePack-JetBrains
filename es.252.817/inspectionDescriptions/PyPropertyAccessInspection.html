Informa sobre accesos inadecuados a propiedades:
  • Establecer propiedades de solo lectura
  • Leer propiedades de solo escritura
  • Eliminar propiedades no eliminables

Ejemplo:


class MyClass:
    @property
    def read_only(self): return None
    def __write_only_setter(self, value): pass
    write_only = property(None, __write_only_setter)
a = MyClass()
a.read_only = 10 # no se puede establecer la propiedad
del a.read_only # no se puede eliminar la propiedad
print(a.write_only) # no se puede leer la propiedad
