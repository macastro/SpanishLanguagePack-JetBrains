Informar sobre cambios en el código fuente de objetos de base de datos.

La verificación se activa durante la introspección de la base de datos o del objeto. La introspección se ejecuta cuando abre el código fuente de un objeto, ejecuta sentencias y realiza refactorizaciones de código. Además, puede ejecutar la introspección haciendo clic derecho en un objeto y seleccionando Actualizar.

La verificación incluye los siguientes casos:

  • El código fuente del objeto en la base de datos ha cambiado, pero el código en el editor no está actualizado. Aplicable para PostgreSQL, Microsoft SQL Server, Oracle y Sybase ASE.
  • Ha modificado el código fuente del objeto, ha realizado la introspección de la base de datos, pero el código fuente ha sido modificado por otra persona.
  • El introspector de base de datos en el IDE se ha actualizado y necesita descargar nuevas propiedades de objetos que faltaban en la versión anterior del introspector.
