Informa de valores no manejados en las sentencias `switch` cuando el identificador `iota` está en la declaración `const`. Para manejar estos valores, se debe considerar el uso de cláusulas `case` o `default`.

La palabra clave `iota` en Go crea una secuencia de enteros en un bloque `const`. Aunque el compilador de Go no requiere manejar todos los valores `iota`, la omisión de alguno de ellos podría indicar un error.

Para obtener más información sobre `iota`, consulta Iota en go.dev.

Ejemplos:

const (
  a = iota
  b
  c = -5
)

func main() {
  v := 5
  switch v {
  case a:
    break;
  case b:
    break;
  }
}

El IDE activa una inspección porque c no es manejado por la sentencia `switch`. Téngase en cuenta que c está en el mismo bloque const aunque no utilice iota.

Las opciones de corrección rápida son Crear cláusula 'case' para valores y Crear cláusula 'default'. Después de aplicar la corrección rápida Crear cláusula 'default':

const (
  a = iota
  b
  c
)

func main() {
  v := 5
  switch v {
  case a:
    break;
  default:
  }
}
