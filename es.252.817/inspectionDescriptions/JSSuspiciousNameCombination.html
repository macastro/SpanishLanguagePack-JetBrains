Informa una asignación o una llamada a función donde el nombre de la variable de destino o el parámetro de la función no coincide con el nombre del valor asignado a ella.

Ejemplo:


  var x = 0;
  var y = x;
o 

  var x = 0, y = 0;
  var rc = new Rectangle(y, x, 20, 20);
Aquí la inspección conjetura que x e y están mezcladas.

Especifica los nombres que no deben usarse juntos. Se informa un error si un nombre de parámetro o un nombre de destino de asignación contiene palabras de un grupo mientras que el nombre de la variable asignada o pasada contiene palabras de otro grupo.
