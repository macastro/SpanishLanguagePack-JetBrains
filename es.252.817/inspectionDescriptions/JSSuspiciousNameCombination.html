Informa sobre asignaciones o llamadas a funciones donde el nombre de la variable de destino o parámetro de función no coincide con el nombre del valor que se le asigna.

Ejemplo:


  var x = 0;
  var y = x;
o 

  var x = 0, y = 0;
  var rc = new Rectangle(y, x, 20, 20);
Aquí, la inspección sospecha que x y y están confundidos.

Especifica nombres que no deberían usarse juntos. Se informa un error si el nombre del parámetro o el nombre del objetivo de asignación contiene una palabra de un grupo, mientras que el nombre de la variable asignada o pasada contiene una palabra de otro grupo.
