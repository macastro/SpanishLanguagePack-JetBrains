Reporta las declaraciones return al final de constructores y métodos que devuelven void. Estas declaraciones son innecesarias y pueden eliminarse de forma segura.

Ejemplo: 


  void foo (String s){
      print(s)
      return
  }

Después de aplicar la corrección rápida:


  void foo (String s){
      print(s)
  }

Para más detalles, consulte la misma inspección en Java.
