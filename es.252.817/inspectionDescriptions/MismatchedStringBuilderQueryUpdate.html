Informa sobre objetos StringBuilder, StringBuffer o StringJoiner cuyo contenido ha sido leído pero no escrito, o escrito pero no leído.

Esta inconsistencia en la lectura y escritura no tiene sentido y puede indicar código muerto, código incompleto o errores.

Ejemplo:


  public void m1() {
    StringBuilder sb = new StringBuilder();
    sb.append("a");
  }
