Informa sobre cualquier declaración de import que cubra un paquete completo ('imports' con *).

Algunos estándares de codificación prohíben este tipo de declaraciones import.

Puede configurar IntelliJ IDEA para detectar y corregir estas declaraciones usando el comando Optimizar imports. Vaya a Configuración | Editor | Estilo de código | Java | Imports, asegúrese de que la opción Usar importación de clase única esté habilitada y especifique valores en los campos Número de clases para usar import con '*' y Número de nombres para usar static import con '*'. Por lo tanto, esta comprobación es útil para informes offline de bases de código que no se pretenden modificar.
