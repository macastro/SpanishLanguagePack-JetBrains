Informa métodos que no tienen primer parámetro generalmente llamado self.

Ejemplo:


class Movie:

   def show():
       pass

Al aplicar la corrección rápida, el código cambia a:


class Movie:

   def show(self):
       pass

La inspección también informa problemas de nombramiento en métodos de clase.

Ejemplo:


class Movie:
    @classmethod
    def show(abc):
        pass

Ya que el primer parámetro de un método de clase debería ser cls, el IDE proporciona una corrección rápida para renombrarlo.
