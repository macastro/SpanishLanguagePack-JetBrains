Reporta expresiones de asignación que están anidadas dentro de otras expresiones.

Dichas expresiones pueden ser confusas y violan el principio de diseño general, que establece que cualquier construcción debe hacer una sola cosa.

Ejemplo:


  String userName;
  // Advertencia: el resultado de la asignación a 'userName' se usa
  String message = "Hello " + (userName = "Alice") + "!"
  System.out.println(message);
  System.out.println("Goodbye " + userName);
