Informa sobre expresiones de asignación anidadas dentro de otras expresiones.

Estas expresiones pueden resultar confusas y violan el principio general de diseño de que cada construcción debe tener un solo propósito.

Ejemplo:


  String userName;
  // Advertencia: se utiliza el resultado de la asignación a 'userName'
  String message = "Hello " + (userName = "Alice") + "!"
  System.out.println(message);
  System.out.println("Goodbye " + userName);
