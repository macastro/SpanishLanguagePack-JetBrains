Informa sobre llamadas a la función de la biblioteca estándar de Kotlin filterIsInstance con argumentos de literal de clase. Es más idiomático usar la versión de esta función con parámetros de tipo reificados para evitar la sintaxis ::class.java.

Nota: No se informan las comprobaciones de literales de clase genéricos, ya que la sintaxis Class<*, *> en la lista de argumentos de tipo puede no ser deseable.

Ejemplo:


  fun foo(list: List<*>) {
      list.filterIsInstance(Int::class.java)
  }

Después de aplicar la corrección rápida:


  fun foo(list: List<*>) {
      list.filterIsInstance<Int>()
  }
