Admite declaraciones en for aumentadas.

Ejemplo:


  for (int x: Arrays.asList(1, 2, 3)) {
    System.out.println(x);
  }

Después de aplicar la solución rápida:


  for (Iterator<Integer> iterator = Arrays.asList(1, 2, 3).iterator(); iterator.hasNext(); ) {
    final int x = iterator.next();
    System.out.println(x);
  }

La declaración en for aumentada apareció en Java 5. Esta inspección puede ayudar a degradar para garantizar la compatibilidad con versiones anteriores de Java.
