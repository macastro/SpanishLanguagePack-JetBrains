Reporta referencias a métodos proveedores de datos que no existen o no están accesibles.

Ejemplo:


public class InstanceDataProviderFromForeignClass {
  // el método data() no existe en la clase A
  @Test(dataProvider = "data", dataProviderClass = A.class)
  public void test() {
  }
}
class A { }

Después de aplicar el arreglo rápido:


//el método data() necesario es creado en la clase A
class A {
  @DataProvider
  public Object[][] data() {
    return new Object[][]{};
  }
}
