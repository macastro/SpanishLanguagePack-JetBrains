Informa sobre clases que implementan dos o más traits (que contienen métodos con la misma firma).

La invocación de dichos métodos puede producir resultados inesperados.

Esta corrección rápida añade una sobrescritura explícita del método.

Ejemplo:


  trait T1 {
      def foo() {}
  }
  trait T2 {
      def foo() {}
  }
  class X implements T1, T2 {}
  // Llamará a T2.foo()
  new X().foo()

Después de aplicar la corrección rápida:


  class X implements T1, T2 {
      @Override
      Object foo() {
          return T2.super.foo()
      }
  }
