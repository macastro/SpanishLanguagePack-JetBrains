Informa de la declaración de una excepción no comprobada (java.lang.RuntimeException o una de sus subclases) en la cláusula throws de un método.

No se requieren declaraciones de excepciones no comprobadas y se pueden eliminar o mover a una etiqueta Javadoc @throws.

Ejemplo:


  public class InvalidDataException extends RuntimeException {}

  class TextEditor {
    void readSettings() throws InvalidDataException {} // advertencia: excepción no comprobada 'InvalidDataException' declarada en la cláusula 'throws'
  }
