Informa sobre las llamadas a java.lang.String.concat().

Si el método se llama en una constante con argumentos constantes, estas llamadas pueden reemplazarse por el operador + para mejorar la claridad y el rendimiento.

Ejemplo:


  String foo(String name) {
    return name.concat("foo");
  }

Después de aplicar la corrección rápida:


  String foo(String name) {
    return name + "foo";
  }
