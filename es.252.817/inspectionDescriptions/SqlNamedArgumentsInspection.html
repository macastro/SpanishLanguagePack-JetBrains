Informa sobre argumentos utilizados sin nombres en llamadas a rutinas. Por defecto, esta inspección está desactivada.

Para obtener más información sobre las diferencias entre parámetros nombrados y sin nombre, consulte Binding Parameters by Name (Named Parameters) en docs.microsoft.com.

Ejemplo (Microsoft SQL Server):

CREATE FUNCTION foo(n INT, m INT) RETURNS INT AS
BEGIN
    RETURN n + m;
END;
CREATE PROCEDURE test AS
BEGIN
    foo n = 1, m = 2;
--- La siguiente llamada carece de nombres de parámetros y será resaltada
    foo 1, 2;
END;

Los parámetros 1, 2 en la llamada foo 1, 2; están resaltados porque carecen de nombres.
