Reporta errores de tipo en expresiones de llamada de función, objetivos y valores de retorno. En un lenguaje de tipado dinámico, esto es posible en un número limitado de casos.

Los tipos de parámetros de función se pueden especificar en docstrings o en anotaciones de funciones de Python 3.

Ejemplo:


def foo() -> int:
    return "abc" # Se esperaba un int, se obtuvo un str


a: str
a = foo() # Se esperaba un str, se obtuvo un int

Con la corrección rápida, puedes modificar los tipos problemáticos:


def foo() -> str:
    return "abc"


a: str
a = foo()
