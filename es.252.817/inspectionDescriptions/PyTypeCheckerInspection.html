Informa sobre errores de tipo en expresiones de llamada a funciones, objetivos y valores de retorno. En lenguajes de tipado dinámico, esto puede ocurrir en algunos casos.

Los tipos de parámetros de función se pueden especificar en docstrings o anotaciones de funciones de Python 3.

Ejemplo:


def foo() -> int:
    return "abc" # Debería ser int, pero se obtiene str
a: str
a = foo() # Debería ser str, pero se obtiene int

La corrección rápida puede modificar el tipo problemático:


def foo() -> str:
    return "abc"
a: str
a = foo()
