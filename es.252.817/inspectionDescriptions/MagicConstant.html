Informa sobre expresiones que pueden ser reemplazadas por constantes "mágicas".

Ejemplo 1:


  // Uso de literal sin formato "2", advertencia:
  Font font = new Font("Arial", 2)

Ejemplo 2:


  // Uso de constante predefinida, correcto:
  Font font = new Font("Arial", Font.ITALIC)

Cuando sea posible, la corrección rápida insertará la constante predefinida correspondiente.

El comportamiento de esta inspección está controlado por la anotación org.intellij.lang.annotations.MagicConstant. Algunos métodos de bibliotecas Java estándar están pre-anotados, pero también puede usar esta anotación en su código.
