El informe puede reemplazarse con una sentencia if que utilice los operadores &&, ||, ==, != o ?:.

El resultado suele ser más corto, pero no siempre más claro, por lo que no se recomienda aplicar esta corrección en todos los casos.

Ejemplo:


  if (condition) return true; else return foo;

Después de aplicar la corrección rápida:


  return condition || foo;

Configuración de la inspección:

  • Use la opción No sugerir el operador '?:' para deshabilitar las advertencias cuando se sugiera el uso del operador ?:. En este caso, solo se resaltarán las sugerencias de &&, ||, == y !=. La corrección rápida seguirá disponible en el editor.
  • Use la opción Ignorar sentencias 'if' encadenadas para deshabilitar las advertencias en cadenas if-else. La corrección rápida seguirá disponible en el editor.
