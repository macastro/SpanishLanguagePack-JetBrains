Se informa una llamada inválida a ReadScope desde un contexto suspendido o de operación.

No se puede llamar a la función ReadScope desde una corrutina u operación. De lo contrario, podría producirse una excepción en tiempo de ejecución.

Ejemplo:


  fun ReadScope.kernel(/*...*/) { /*...*/ }
  fun ReadScope.foo() {
      readEditorAction { ac ->
          kernel() // Incorrecto
      }
  }
