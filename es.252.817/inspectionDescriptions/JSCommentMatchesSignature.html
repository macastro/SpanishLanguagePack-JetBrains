Informa sobre discrepancias entre los nombres y el número de parámetros en los comentarios JSDoc y los parámetros reales de la función. Se recomienda actualizar los parámetros en los comentarios JSDoc.

Ejemplo:


/**
 * @param height Height in pixels
 */
function sq(height, width) {} // ancho no documentado

Después de aplicar la corrección rápida:


/**
 * @param height Height in pixels
 * @param width
 */
function sq(height, width) {}
