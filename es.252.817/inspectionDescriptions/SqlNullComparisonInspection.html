Informa sobre comparaciones con NULL que pueden ser reemplazadas por los operadores IS NULL o IS NOT NULL.

Ejemplo (Microsoft SQL Server):

CREATE TABLE foo ( id int );
SELECT * FROM foo WHERE NULL = NULL;
SELECT * FROM foo WHERE NULL != NULL;

NULL = NULL puede ser reemplazado por IS NULL, y la comparación NULL != NULL puede ser reemplazada por IS NOT NULL. Para realizar esta sustitución, puede utilizar las correcciones rápidas Usar operador IS NULL o Usar operador IS NOT NULL.

SELECT * FROM foo WHERE NULL IS NULL;
SELECT * FROM foo WHERE NULL IS NOT NULL;
