Reporta una variable cuyo valor no se utiliza nunca después de su asignación.
Sugiere eliminar la variable no utilizada para acortar el código y evitar asignaciones redundantes.

Se reportan los siguientes casos:

  • Una variable nunca se lee después de su asignación.
  • El valor de una variable siempre se sobrescribe con otra asignación antes de que la variable se lea la próxima vez.
  • El inicializador de una variable es redundante (por una de las razones mencionadas anteriormente).
