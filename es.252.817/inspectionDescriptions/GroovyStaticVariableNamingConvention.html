Reporta variables static cuyo nombre es demasiado corto, demasiado largo o no sigue el patrón de expresión regular especificado.

Esta inspección no verifica las constantes (es decir, variables de tipos inmutables declaradas como static final).

Use los campos proporcionados a continuación para especificar la longitud mínima, máxima y la expresión regular esperada para los nombres de variables static.

La expresión regular debe especificarse en el formato estándar de java.util.regex.
