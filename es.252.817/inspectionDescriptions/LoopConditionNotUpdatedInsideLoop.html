Informa de variables y parámetros utilizados en la condición de un bucle y no actualizados en el interior del bucle.

Estas variables y parámetros son utilizados normalmente por error ya que pueden causar un bucle infinito si se ejecutan.

Ejemplo:


  void loopDoesNotLoop(boolean b) {
    while (b) {
      System.out.println();
      break;
    }
  }

Configurar la inspección:

Usa la opción Ignorar posibles cambios no locales para deshabilitar la inspección si la condición puede actualizarse indirectamente (p.e. mediante el método llamado o concurrentemente desde otro hilo).
