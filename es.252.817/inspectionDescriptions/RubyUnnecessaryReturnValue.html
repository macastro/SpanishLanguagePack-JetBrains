Inspecciona variables usadas como valores de retorno.

Ejemplo:

# Mala práctica
def foo(bar)
  resultado = bar + 1
  resultado
end
Se cuenta con una solución rápida para eliminar dichas variables. Luego de aplicar la solución rápida, el resultado se ve así: 
# Buena práctica
def foo(bar)
  bar + 1
end
