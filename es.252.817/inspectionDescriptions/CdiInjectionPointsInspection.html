Reporta los puntos de inyección en clases Bean que tienen dependencias ambiguas.

Ejemplo:


  interface Engine {}
  @Singleton
  class GasEngine implements Engine {}
  @Singleton
  class ElectricEngine implements Engine {}
  @Singleton
  class Car {
    @Inject Engine engine; // Dependencia ambigua: múltiples Beans coinciden con el punto de inyección
  }
