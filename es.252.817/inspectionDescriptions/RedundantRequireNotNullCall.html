Informa de llamadas redundantes a requireNotNull o checkNotNull en expresiones no nulas.

Ejemplo:


  fun foo(i: Int) {
      requireNotNull(i) // Esta 'i' nunca es nula, por lo que esta llamada a 'requireNotNull' es redundante.
      ...
  }

Después de aplicar la corrección rápida:


  fun foo(i: Int) {
      ...
  }
