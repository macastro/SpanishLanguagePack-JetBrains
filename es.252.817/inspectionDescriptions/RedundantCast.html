Informa sobre expresiones de conversión innecesarias.

Ejemplo:


  static Object toObject(String s) {
    return (Object) s;
  }

Utilice la casilla de verificación a continuación para ignorar las conversiones de aclaración, como las conversiones en llamadas a colecciones que deberían ser de tipo Object: 


  static void removeFromList(List<String> l, Object o) {
    l.remove((String)o);
  }
