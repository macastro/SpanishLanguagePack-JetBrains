Informa sobre beans abstractos referenciados.

Los beans abstractos sólo pueden usarse como definiciones de beans plantilla, es decir, padres para definiciones hijo. Intentar usar un bean padre abstracto de este tipo por sí mismo refiriéndose a él como una propiedad ref de otro bean, o llamando a getBean() con el ID del bean padre, dará como resultado un error.

Ejemplo:


  <bean abstract="true" id="parentBeanWithoutClass"/>
  <bean class="com.my.company.FooBean" id="childBean" parent="parentBeanWithoutClass">
  <!--  propiedades -->
  </bean>

En este ejemplo, el bean padre no se puede instanciar por sí mismo ya que está incompleto y marcado explícitamente como abstracto.
