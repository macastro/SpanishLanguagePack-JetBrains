Informa sobre instancias donde se desreferencia un puntero que podría ser nil.

Ejemplo:


type Person struct{ Name string }
func _(person *Person) {
	if person != nil || person.Name != "Peter" {
	}
}

En el ejemplo proporcionado, person es un puntero a la estructura Person, pero es nil porque no se le ha asignado ningún valor. Intentar desreferenciar person para acceder a Name sin verificar si person es nil resultará en un error en tiempo de ejecución.
