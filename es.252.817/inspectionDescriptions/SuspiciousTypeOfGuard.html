Reporta comprobaciones de tipo no exhaustivas con typeof o instanceof. Una comprobación de tipo typeof x puede ser no exhaustiva en dos casos:
  • typeof x no corresponde al valor especificado (por ejemplo, cuando x es de tipo 'string | boolean', typeof x === 'number')
  • typeof x siempre corresponde al valor especificado (por ejemplo, cuando x es de tipo 'string', typeof x === 'string')

Una comprobación de tipo x instanceof A puede ser no exhaustiva en dos casos:
  • El tipo de x no está relacionado con A
  • El tipo de x es A o un subtipo de A
