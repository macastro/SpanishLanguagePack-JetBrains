Informa sobre clases regulares que pueden convertirse en declaraciones implícitas.

Ejemplo:


  public class Sample {
      public static void main(String[] args) {
          System.out.println("Hello, world!");
      }
  }

Después de aplicar la corrección rápida:


  public static void main(String[] args) {
      System.out.println("Hello, world!");
  }
