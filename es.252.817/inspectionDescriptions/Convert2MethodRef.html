Se informa que el lambda puede reemplazarse con una referencia a método. Las referencias a métodos suelen ser más fáciles de entender y leer que las expresiones lambda, aunque esto generalmente depende de tus preferencias.

Ejemplo:


  Runnable r = () -> System.out.println();

Después de aplicar la corrección rápida:


  Runnable r = System.out::println;

Esta inspección puede sugerir referencias a métodos incluso cuando el lambda no invoca ningún método, por ejemplo, reemplazando obj -> obj != null por Objects::nonNull. Utiliza Configuración | Editor | Estilo de código | Java | Generación de código para configurar referencias a métodos especiales.
