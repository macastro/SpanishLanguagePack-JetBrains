Informa referencias a métodos y campos static mediante una instancia de clase en lugar de la propia clase.

Aunque la especificación del lenguaje Java permite hacer referencia a miembros estáticos mediante variables de instancia, esto hace que el código resulte confuso, ya que el lector podría pensar que el resultado del método depende de la instancia.

La corrección rápida reemplaza la variable de instancia con el nombre de la clase.

Ejemplo:


  String s1 = s.valueOf(0);

Tras aplicar la corrección rápida:


  String s = String.valueOf(0);
