Informa sobre cualquier método public que no contenga declaraciones de registro (logging). Esta inspección no informa sobre getters y setters simples.

Por ejemplo:


  public class Crucial {
    private static final Logger LOG = LoggerFactory.getLogger(Crucial.class);
    public void doImportantStuff() {
      // Se genera una advertencia para este método
    }
    public void doOtherStuff() {
      LOG.info("do other stuff");
    }
  }
