Se informa un método disposer sin el método productor correspondiente.

Cada método disposer debe tener exactamente un parámetro disposed del mismo tipo que el tipo de retorno del método productor o campo productor correspondiente. Al buscar un método disposer para un método productor o campo productor, el contenedor considera el tipo y los calificadores del parámetro disposed. Si el parámetro disposed se resuelve a un método productor o campo productor declarado en la misma clase Bean, el contenedor debe invocar este método cuando destruya cualquier instancia retornada por ese método productor o campo productor.
