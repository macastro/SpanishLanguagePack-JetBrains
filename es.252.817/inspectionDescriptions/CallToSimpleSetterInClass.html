Informa sobre llamadas a setters simples de propiedades desde dentro de la clase que posee la propiedad.

Un setter simple de propiedad se define como aquel que solo asigna el valor de su parámetro a un campo, sin realizar otros cálculos. Este tipo de llamadas a setters simples pueden ser inlineadas de manera segura. Algunos estándares de codificación también recomiendan evitar el uso de setters simples para mantener el código más claro.

Ejemplo:


  class Foo {
  private int index;
    public Foo(int idx) {
       setIndex(idx);
    }
    public void setIndex(int idx) {
       index = idx;
    }
  }

Después de aplicar la corrección rápida:


  class Foo {
    private int index;
    public Foo(int idx) {
      index = idx;
    }
    public void setIndex(int idx) {
      index = idx;
    }
  }

La inspección puede configurarse con las siguientes opciones:

  • Si se debe informar solo sobre llamadas a setters en this y no en objetos del mismo tipo pasados como parámetros.
  • Si se deben ignorar los setters no private.
