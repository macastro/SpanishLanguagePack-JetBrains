Inspecciona llamadas a un setter de propiedad simple en la clase de la propiedad.

Un setter de propiedad simple está definido como aquel que simplemente asigna el valor de su parámetro a un campo y no realiza ningún otro cálculo. Tales llamadas simples a setter pueden ser interpoladas de forma segura. Algunos estándares de codificación también sugieren no utilizar setters simples por razones de claridad de código.

Ejemplo:


  class Foo {
  private int index;
    public Foo(int idx) {
       setIndex(idx);
    }
    public void setIndex(int idx) {
       index = idx;
    }
  }

Después de que la corrección rápida es aplicada:


  class Foo {
    private int index;
    public Foo(int idx) {
      index = idx;
    }
    public void setIndex(int idx) {
      index = idx;
    }
  }

Utiliza las siguientes opciones para configurar la inspección:

  • Si se debe informar solo de llamadas a setter en this, no en objetos del mismo tipo pasados como parámetro.
  • Si se deben ignorar los setter no private.
