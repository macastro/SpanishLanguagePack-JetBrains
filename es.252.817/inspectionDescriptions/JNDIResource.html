Informa sobre recursos JNDI que no se han cerrado de manera segura. Esta inspección reporta recursos JNDI que incluyen javax.naming.InitialContext y javax.naming.NamingEnumeration.

Por defecto, esta inspección asume que los recursos pueden cerrarse mediante cualquier método que contenga 'close' o 'cleanup' en su nombre.

Ejemplo:


  Object findObject(Properties properties, String name) throws NamingException {
    Context context = new InitialContext(properties); //contexto no cerrado
    return context.lookup(name);
  }

Configure la inspección con las siguientes opciones:

  • Si se permite abrir recursos JNDI dentro de un bloque try. Este estilo es menos preferible ya que es más verboso que abrir el recurso antes del bloque try.
  • Si los recursos pueden cerrarse mediante cualquier llamada a método que tome el recurso como argumento.
