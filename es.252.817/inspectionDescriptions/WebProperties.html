Reporta los siguientes problemas en los descriptores utilizados para desplegar módulos Web en el servidor:
  • Referencias a clases no instanciables
  • Referencias a clases que no extienden las clases requeridas
  • Referencias a clases con ámbito inadecuado
  • Etiquetas y valores de atributos vacíos
  • Etiquetas y valores de atributos que no coinciden con el esquema requerido (por ejemplo, identificadores Java)
  • Etiquetas que no contienen las subetiquetas o atributos requeridos
  • Etiquetas que definen objetos con nombres duplicados

Ejemplo:



<web-app>
  <filter-mapping>
    <filter-name>CustomFilter</filter-name> // Error: debe definirse una subetiqueta servlet-name o url-pattern
  </filter-mapping>
  <servlet-mapping>
    <servlet-name>CustomServletName</servlet-name>
    <url-pattern>/ baseUrl</url-pattern> // Error: el patrón URI no debe contener espacios
  </servlet-mapping>
</web-app>
