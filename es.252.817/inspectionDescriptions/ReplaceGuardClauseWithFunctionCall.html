Informa sobre cláusulas condicionales que pueden ser reemplazadas por llamadas a funciones.

Ejemplo:

  fun test(foo: Int?) {
      if (foo == null) throw IllegalArgumentException("foo") // Cláusula reemplazable
  }

Después de aplicar la corrección rápida:

  fun test(foo: Int?) {
      checkNotNull(foo)
  }
