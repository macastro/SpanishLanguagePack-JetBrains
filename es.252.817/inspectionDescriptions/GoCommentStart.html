Informa sobre comentarios que no comienzan con el nombre del elemento exportado.

De acuerdo con Comment Sentences en github.com/golang, es una convención comenzar los comentarios con el nombre del elemento exportado.

Ejemplo:

// representa una solicitud para ejecutar un comando.
type Request struct {}

El comentario comienza con la descripción de la estructura en lugar del nombre de la estructura. Para cumplir con la convención, puede aplicar la corrección rápida agregar prefijo al comentario. Después de aplicar la corrección rápida, el comentario se verá así:

// Request representa una solicitud para ejecutar un comando.
type Request struct {} // mejor
