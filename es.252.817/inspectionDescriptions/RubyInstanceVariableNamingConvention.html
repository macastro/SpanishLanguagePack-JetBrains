Reporta variables de instancia con nombres demasiado cortos, demasiado largos o que no coinciden con el patrón de expresión regular especificado.

Utilice el campo patrón para especificar la expresión regular que se usará para validar los nombres de las variables de instancia.

Utilice los campos longitud mínima y longitud máxima para especificar la longitud mínima y máxima permitida para los nombres, respectivamente.
