Informa sobre casos donde expresiones restringidas por conceptos son asignadas a variables declaradas como auto y sugiere agregar restricciones de tipo.

Ejemplo:


  template<class T>
  concept Foo = true;
  Foo auto func() { return true; }
  void bar() {
      auto f = func(); //advertencia aquí sobre la necesidad de agregar la restricción 'Foo'
  }
