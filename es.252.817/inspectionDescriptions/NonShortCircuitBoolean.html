Informa sobre el uso de formas no de cortocircuito de los operadores booleanos 'and' y 'or' (&, |, &= y |=). Aunque las versiones sin cortocircuito ocasionalmente son útiles, en la mayoría de los casos se deberían usar las formas de cortocircuito (&& y ||), y estos usos no intencionados pueden llevar a errores difíciles de detectar.

Se sugiere utilizar las versiones de cortocircuito mediante una corrección rápida.

Ejemplo:


  void foo(boolean x, boolean y, boolean z) {
    if (x | y) { x |= z; }
  }

Después de aplicar la corrección rápida:


  void foo(boolean x, boolean y) {
    if (x || y) { x = x || z; }
  }
