Informa sobre cualquier llamada a métodos en la clase junit.framework.Assert. Esta clase está obsoleta y estas llamadas pueden reemplazarse por llamadas a métodos en la clase org.junit.Assert.

Por ejemplo:


  import org.junit.*;
  public class NecessaryTest {
    @Test
    public void testIt() {
      junit.framework.Assert.assertEquals("expected", "actual");
    }
  }

Después de aplicar la corrección rápida, el resultado se muestra así:


  import org.junit;
  public class NecessaryTest {
    public void testIt() {
      Assert.assertEquals("expected", "actual");
    }
  }
