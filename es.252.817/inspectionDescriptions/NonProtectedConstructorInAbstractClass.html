Reporta constructores public en clases abstract.

Los constructores de clases abstract solo pueden ser llamados desde constructores de sus subclases, por lo que declararlos como public puede resultar confuso.

Esta corrección rápida cambiará estos constructores a protected.

Ejemplo:


  public abstract class Foo {
    public Foo () { // advertencia: tiene modificador 'public'
        /* ... */
    }
  }

Después de aplicar la corrección rápida:


  public abstract class Foo {
    protected Foo () {
        /* ... */
    }
  }

Configuración de la inspección:

Use la opción Ignorar clases no públicas siguiente para ignorar constructores public en clases no públicas.
