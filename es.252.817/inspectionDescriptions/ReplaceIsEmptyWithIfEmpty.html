Reporta llamadas a isEmpty, isBlank, isNotEmpty o isNotBlank en declaraciones if para asignar valores predeterminados.

Esta corrección rápida reemplaza la condición if con llamadas a ifEmpty o ifBlank.

Ejemplo:


  fun test(list: List<Int>): List<Int> {
      return if (list.isEmpty()) {
          println()
          foo()
      } else {
          list
      }
  }

Después de aplicar la corrección rápida:


  fun test(list: List<Int>): List<Int> {
      return list.ifEmpty {
          println()
          foo()
      }
  }

Esta inspección solo se reporta si la versión del lenguaje Kotlin del proyecto o módulo es 1.3 o superior.
