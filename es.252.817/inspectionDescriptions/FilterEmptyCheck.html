Devolver errores de colecciones filtradas vacías.

Los arreglos rápidos usan exists o forall para revisar si están vacíos.

Ejemplo:


  var x: Seq[Int]
  x.filter(p).size == 0
  x.filter(p).length > 0
  x.filterNot(p).isEmpty
  x.filterNot(p).nonEmpty

Después de aplicar el arreglo rápido:


  var x: Seq[Int]
  !x.exists(p)
  x.exists(p)
  x.forall(p)
  !x.forall(p)
