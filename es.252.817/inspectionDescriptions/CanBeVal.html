Inspecciona variables locales declaradas con la palabra clave var que nunca son modificadas.

Kotlin fomenta la declaración de variables prácticamente inmutables usando la palabra clave val, asegurando que su valor nunca cambiará.

Ejemplo:


  fun ejemplo() {
      var numerosPrimos = listOf(1, 2, 3, 5, 7, 11, 13)
      var numerosFibonacci = listOf(1, 1, 2, 3, 5, 8, 13)
      print("Números iguales: " + numerosPrimos.intersect(numerosFibonacci))
  }

La corrección rápida reemplaza la palabra clave var con val:


  fun ejemplo() {
      val numerosPrimos = listOf(1, 2, 3, 5, 7, 11, 13)
      val numerosFibonacci = listOf(1, 1, 2, 3, 5, 8, 13)
      print("Números iguales: " + numerosPrimos.intersect(numerosFibonacci))
  }
