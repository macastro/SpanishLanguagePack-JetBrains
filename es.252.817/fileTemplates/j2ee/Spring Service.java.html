Esta plantilla integrada es editable. Además de expresiones y comentarios de Java, puede usar variables predefinidas (que se enumeran a continuación), que se expanden a los valores correspondientes como macros.

También puede especificar variables personalizadas. Las variables personalizadas usan el formato ${VARIABLE_NAME}, donde VARIABLE_NAME es el nombre de la variable (por ejemplo, ${MY_CUSTOM_FUNCTION_NAME}). Antes de que el IDE cree un archivo nuevo con variables personalizadas, aparecerá un cuadro de diálogo donde puede definir los valores de las variables personalizadas de la plantilla.

Usando la directiva #parse, puede incluir plantillas desde la pestaña Incluir. Para incluir una plantilla, especifique el nombre completo de la plantilla como un parámetro entre comillas (por ejemplo, #parse("File Header.java")).
Las variables predefinidas toman los siguientes valores:
${PACKAGE_NAME} Nombre del paquete donde se crea la nueva clase
${NAME} Nombre de la nueva clase que especificó en el cuadro de diálogo Crear nueva clase
${USER} Nombre de inicio de sesión del sistema del usuario actual
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${MONTH_NAME_SHORT} Las 3 primeras letras del nombre del mes actual (Jan, Feb, etc.)
${MONTH_NAME_FULL} Nombre completo del mes actual (January, February, etc.).
${DAY} Día del mes actual
${HOUR} Hora actual
${MINUTE} Minuto actual
${PROJECT_NAME} Nombre del proyecto actual
