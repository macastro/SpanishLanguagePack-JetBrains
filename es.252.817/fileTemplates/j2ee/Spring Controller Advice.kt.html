Esta plantilla integrada es editable. Además de las expresiones y comentarios Java, puede usar variables predefinidas (listadas a continuación), que se expandirán a sus valores correspondientes como macros.

También puede especificar variables personalizadas. Las variables personalizadas usan el siguiente formato: ${VARIABLE_NAME}, donde VARIABLE_NAME es el nombre de la variable (por ejemplo, ${MY_CUSTOM_FUNCTION_NAME}). Antes de que el IDE use variables personalizadas para crear un archivo nuevo, verá un diálogo donde podrá definir los valores de las variables personalizadas en la plantilla.

Mediante el uso de la directiva #parse, puede incluir plantillas desde la pestaña Incluir. Para incluir una plantilla, especifique el nombre completo de la plantilla como parámetro entre comillas (por ejemplo, #parse("File Header.java")).
Las variables predefinidas toman los siguientes valores:
${PACKAGE_NAME} El nombre del paquete donde se crea la nueva clase
${NAME} El nombre de la nueva clase especificado en el diálogo Crear nueva clase
${USER} El nombre de inicio de sesión del sistema del usuario actual
${DATE} La fecha actual del sistema
${TIME} La hora actual del sistema
${YEAR} El año actual
${MONTH} El mes actual
${MONTH_NAME_SHORT} Las primeras 3 letras del nombre del mes actual (Jan, Feb, etc.)
${MONTH_NAME_FULL} El nombre completo del mes actual (January, February, etc.)
${DAY} El día del mes
${HOUR} La hora actual
${MINUTE} El minuto actual
${PROJECT_NAME} El nombre del proyecto actual
