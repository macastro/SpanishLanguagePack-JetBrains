Se aplica a los nuevos Spring Controller Advice creados al invocar Nuevo | Componente de Spring | Controller Advice en la ventana de herramientas Proyecto.
Esta plantilla integrada es editable. Además de las expresiones y comentarios de Java, puede utilizar variables predefinidas (que se enumeran a continuación), que se expanden a sus valores correspondientes, como macros.

También puede especificar variables personalizadas. Las variables personalizadas utilizan el siguiente formato: ${VARIABLE_NAME}, donde VARIABLE_NAME es el nombre de la variable (por ejemplo, ${MY_CUSTOM_FUNCTION_NAME}). Antes de que el IDE cree un archivo nuevo utilizando variables personalizadas, verá un cuadro de diálogo donde puede definir los valores de las variables personalizadas en la plantilla.

Puede incluir plantillas desde la pestaña Incluir utilizando la directiva #parse. Para incluir una plantilla, especifique el nombre completo de la plantilla como parámetro entre comillas (por ejemplo, #parse("File Header.java")).
Las variables predefinidas toman los siguientes valores:
${PACKAGE_NAME} Nombre del paquete donde se crea la nueva clase
${NAME} Nombre de la nueva clase que especifica en el cuadro de diálogo Crear nueva clase
${USER} Nombre de inicio de sesión del sistema del usuario actual
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${MONTH_NAME_SHORT} Los primeros 3 caracteres del nombre del mes actual (Ene, Feb, etc.)
${MONTH_NAME_FULL} Nombre completo del mes actual (Enero, Febrero, etc.).
${DAY} Día del mes
${HOUR} Hora actual
${MINUTE} Minuto actual
${PROJECT_NAME} Nombre del proyecto actual
