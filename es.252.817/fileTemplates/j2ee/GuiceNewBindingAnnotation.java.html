Esta es la plantilla incorporada que se usa cada vez que crea una nueva anotación de enlace Guice.
Use la directiva #parse para incluir una plantilla en la pestaña de inclusión especificando el nombre completo de la plantilla requerida como un parámetro entre comillas. Por ejemplo:
#parse("File Header.java")
Las variables predefinidas tomarán los siguientes valores:
${PACKAGE_NAME}   Nombre del paquete donde se crea la nueva clase
${NAME}   Nombre de la nueva clase especificado en el diálogo Crear Nueva Clase
${USER}   Nombre de inicio de sesión del usuario actual del sistema
${DATE}   Fecha actual del sistema
${TIME}   Hora actual del sistema
${YEAR}   Año actual
${MONTH}   Mes actual
${MONTH_NAME_SHORT}   Primeras 3 letras del nombre del mes actual. Por ejemplo: Jan, Feb, etc.
${MONTH_NAME_FULL}   Nombre completo del mes actual. Por ejemplo: January, February, etc.
${DAY}   Día actual
${HOUR}   Hora actual
${MINUTE}   Minuto actual
${PROJECT_NAME}   Nombre del proyecto actual
