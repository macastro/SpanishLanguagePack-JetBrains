Ésta es una plantilla integrada que se utiliza cada vez que se crea una nueva anotación de ámbito Guice.
Utilizando la directiva #parse, puede incluir plantillas en la pestaña Incluidos especificando el nombre completo de la plantilla deseada como un parámetro entre comillas. Por ejemplo:
#parse("File Header.java")
Las variables predefinidas toman los siguientes valores:
${PACKAGE_NAME} Nombre del paquete donde se va a crear la nueva clase
${NAME} Nombre de la nueva clase que especifique el usuario en el diálogo Crear nueva clase
${USER} Nombre de inicio de sesión del sistema del usuario actual
${DATE} Fecha del sistema actual
${TIME} Hora del sistema actual
${YEAR} Año actual
${MONTH} Mes actual
${MONTH_NAME_SHORT} Las primeras tres letras del nombre del mes actual. Por ejemplo: Ene, Feb, etcétera.
${MONTH_NAME_FULL} El nombre completo del mes actual. Por ejemplo: Enero, Febrero, etcétera.
${DAY} Día del mes actual
${HOUR} Hora actual
${MINUTE} Minuto actual
${PROJECT_NAME} Nombre del proyecto actual
