Go File es una plantilla integrada que se usa cada vez que se crea un nuevo archivo Go. Para usar esta plantilla, vaya a Archivo | Nuevo, seleccione Go File en la categoría Go y haga clic en Archivo vacío.

Esta plantilla es editable. En la plantilla, puede usar las variables predefinidas que se enumeran a continuación. Estas variables se expandirán a sus valores correspondientes en el editor más tarde.

También puede especificar variables personalizadas. Las variables personalizadas usan el siguiente formato: ${VARIABLE_NAME}, donde VARIABLE_NAME es el nombre de la variable (por ejemplo, ${MY_CUSTOM_FUNCTION_NAME}). Antes de que el IDE cree un nuevo archivo usando variables personalizadas, verá un cuadro de diálogo donde puede definir los valores de las variables personalizadas en la plantilla.

Mediante el uso de la directiva #parse, puede incluir plantillas de la pestaña Incluir. Para incluir una plantilla, especifique el nombre completo de la plantilla entre comillas como parámetro (por ejemplo, #parse("File Header")).
Lista de variables predefinidas
${GO_PACKAGE_NAME_WITH_TEST_SUFFIX} El nombre del paquete donde se crea el nuevo archivo de prueba. El archivo debe tener el sufijo _test. El formato del nombre es: package_test.
${GO_PACKAGE_NAME} El nombre del paquete donde se crea el nuevo archivo.
${DATE} Fecha actual del sistema.
${DAY} Día del mes actual.
${DAY_NAME_SHORT} Las 3 primeras letras del nombre del día actual (por ejemplo, Lun, Mar, etc.).
${DAY_NAME_FULL} Nombre completo del día actual (por ejemplo, Lunes, Martes, etc.).
${DIR_PATH} La ruta del directorio del nuevo archivo. Esta ruta es relativa a la raíz del proyecto.
${DS} Símbolo del dólar ($). Esta variable se usa para escapar el carácter dólar, de modo que no se trate como prefijo de variables de plantilla.
${FILE_NAME} El nombre del nuevo archivo.
${HOUR} Hora actual.
${MINUTE} Minuto actual.
${SECOND} Segundo actual.
${MONTH} Mes actual.
${MONTH_NAME_SHORT} Las 3 primeras letras del nombre del mes actual (por ejemplo, Ene, Feb, etc.).
${MONTH_NAME_FULL} Nombre completo del mes actual (por ejemplo, Enero, Febrero, etc.).
${NAME} El nombre de la nueva entidad: archivo, tipo, interfaz, etc.
${PRODUCT_NAME} El nombre del IDE.
${PROJECT_NAME} El nombre del proyecto actual.
${TIME} Hora actual del sistema.
${USER} Nombre de inicio de sesión del sistema del usuario actual.
${YEAR} Año actual.
