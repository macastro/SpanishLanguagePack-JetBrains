Plantilla integrada para un archivo de plugin principal de WordPress. Se usa cada vez que se crea un nuevo proyecto de tipo 'Plugin de WordPress'.
La plantilla es editable. Además de las sentencias, expresiones y comentarios de PHP, puede usar las variables predefinidas listadas a continuación. Cada variable se expande a su valor correspondiente, como si fuera una macro.
Usando la directiva #parse, puede incluir plantillas de la pestaña Includes especificando el nombre completo de la plantilla deseada como un parámetro entre comillas.
Las variables predefinidas toman los siguientes valores:
${PLUGIN_NAME} Nombre del plugin actual derivado del nombre del directorio
${FILE_NAME} Nombre del archivo actual
${USER} Nombre de inicio de sesión del usuario actual en el sistema
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${DAY} Día del mes actual
${HOUR} Hora actual
${MINUTE} Minuto actual
${PRODUCT_NAME} Nombre del IDE actual
${PROJECT_NAME} Nombre del proyecto actual
${DS} Símbolo del dólar, se evalúa al carácter '$' normal.
