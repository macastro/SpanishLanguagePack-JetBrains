Una plantilla integrada para traits de PHP. Se utiliza cada vez que se crea un nuevo trait de PHP seleccionando Nuevo | Clase PHP en el menú contextual de una de las vistas de proyecto.
La plantilla es editable. Además de sentencias, expresiones y comentarios de PHP, puede utilizar las variables predefinidas que se enumeran a continuación. Cada variable se expande a su valor correspondiente, como una macro.
Mediante la directiva #parse, puede incluir plantillas desde la pestaña Includes especificando el nombre completo de la plantilla deseada como parámetro entre comillas.
Las variables predefinidas toman los siguientes valores:
${FILE_NAME} Nombre del archivo actual
${USER} Nombre de inicio de sesión del usuario actual
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${DAY} Día del mes actual
${HOUR} Hora actual
${MINUTE} Minuto actual
${PRODUCT_NAME} Nombre del IDE actual
${DS} Signo de dólar, se evalúa al carácter '$' normal.
${PROJECT_NAME} Nombre del proyecto actual
