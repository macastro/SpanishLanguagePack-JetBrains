Plantilla integrada para módulos Ruby. Esta plantilla se utiliza cada vez que se crea un nuevo módulo Ruby seleccionando Nuevo | Archivo/Clase Ruby en el menú contextual de alguna de las vistas de proyecto. Además de sentencias, expresiones y comentarios de Ruby, puede utilizar las variables predefinidas que se enumeran a continuación. Cada variable se expande a su valor correspondiente, de forma similar a una macro. Mediante la directiva #parse, puede incluir plantillas de la pestaña Include especificando el nombre completo de la plantilla deseada como un parámetro entre comillas.
Las variables predefinidas toman los siguientes valores:
${FILE_NAME} Nombre del archivo actual
${USER} Nombre de inicio de sesión del usuario actual en el sistema
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${DAY} Día actual
${HOUR} Hora actual
${MINUTE} Minuto actual
${PRODUCT_NAME} Nombre del IDE actual
${PROJECT_NAME} Nombre del proyecto actual
