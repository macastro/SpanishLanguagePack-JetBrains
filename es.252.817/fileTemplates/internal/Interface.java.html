Se aplica a las nuevas interfaces Java creadas mediante Nuevo | Clase Java | Interfaz en la ventana de herramientas Proyecto.

Esta plantilla incorporada es editable. Además de las expresiones Java y comentarios, puede usar variables predefinidas (listadas a continuación) que se expandirán como macros a sus valores correspondientes.

También puede especificar variables personalizadas. Las variables personalizadas utilizan el siguiente formato: ${VARIABLE_NAME}, donde VARIABLE_NAME es el nombre de la variable (por ejemplo, ${MY_CUSTOM_FUNCTION_NAME}). Antes de que el IDE cree un nuevo archivo usando variables personalizadas, verá un diálogo donde podrá definir los valores para las variables personalizadas en la plantilla.

Usando la directiva #parse, puede incluir plantillas de la pestaña Include. Para incluir una plantilla, especifique su nombre completo como un parámetro entre comillas (por ejemplo, #parse("File Header.java")).
Las variables predefinidas aceptan los siguientes valores:
${PACKAGE_NAME}   Nombre del paquete donde se crea la nueva interfaz
${NAME}   Nombre de la nueva interfaz que especifica en el diálogo Crear Nueva Clase
${USER}   Nombre de inicio de sesión del usuario actual
${DATE}   Nombre de inicio de sesión del usuario actual
${TIME}   Hora actual del sistema
${YEAR}   Año actual
${MONTH}   Mes actual
${MONTH_NAME_SHORT}   Primeras 3 letras del nombre del mes actual (Jan, Feb, etc.)
${MONTH_NAME_FULL}   Nombre completo del mes actual (January, February, etc.)
${DAY}   Día del mes actual
${HOUR}   Hora actual
${MINUTE}   Minuto actual
${PROJECT_NAME}   Nombre del proyecto actual
