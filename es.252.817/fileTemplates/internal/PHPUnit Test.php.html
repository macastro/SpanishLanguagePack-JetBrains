Una plantilla integrada para pruebas PHPUnit de la versión 5 o anterior. Esta plantilla se utiliza cada vez que crea una nueva prueba PHPUnit, seleccionando Nuevo | Prueba PHP desde el menú contextual de una de las vistas del proyecto, y eligiendo 'PHPUnit < 6' como la biblioteca de pruebas.
La plantilla es editable. Además de sentencias, expresiones y comentarios de PHP, también puede usar las variables predefinidas que se enumeran a continuación. Cada variable se expande a su valor correspondiente como una macro.
Usando la directiva #parse, puede incluir una plantilla de la pestaña Includes especificando el nombre completo de la plantilla deseada como un parámetro entre comillas.
Las variables predefinidas toman los siguientes valores:
${TESTED_NAME} Nombre de la clase probada
${TESTED_NAMESPACE} Espacio de nombres de la clase probada
${NAME} Nombre de la clase de prueba
${NAMESPACE} Espacio de nombres de la clase de prueba
${FILE_NAME} Nombre del archivo actual
${USER} Nombre de inicio de sesión del usuario actual
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${DAY} Día del mes actual
${HOUR} Hora actual
${MINUTE} Minuto actual
${PRODUCT_NAME} Nombre del IDE actual
${DS} Símbolo del dólar, se evalúa como el carácter simple '$'.
${PROJECT_NAME} Nombre del proyecto actual
