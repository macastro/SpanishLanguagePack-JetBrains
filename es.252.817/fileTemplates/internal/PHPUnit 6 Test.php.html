Una plantilla integrada para pruebas de PHPUnit de la versión 6 o superior. Se utiliza cada vez que se crea una nueva prueba de PHPUnit. Seleccione Nuevo | Prueba de PHP desde el menú emergente de una de las vistas de proyecto y luego seleccione "PHPUnit" como biblioteca de pruebas.
La plantilla es editable. Además de sentencias, expresiones y comentarios de PHP, también puede usar las variables predefinidas que se enumeran a continuación. Cada variable se expande como una macro a su valor correspondiente.
Mediante la directiva #parse, se pueden incluir plantillas desde la pestaña Includes especificando el nombre completo de la plantilla deseada como parámetro entre comillas.
Las variables predefinidas aceptan los siguientes valores:
${TESTED_NAME} Nombre de la clase probada
${TESTED_NAMESPACE} Espacio de nombres de la clase probada
${NAME} Nombre de la clase de prueba
${NAMESPACE} Espacio de nombres de la clase de prueba
${FILE_NAME} Nombre del archivo actual
${USER} Nombre de inicio de sesión del usuario actual
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${DAY} Día del mes actual
${HOUR} Hora actual
${MINUTE} Minuto actual
${PRODUCT_NAME} Nombre del IDE actual
${DS} Signo de dólar, se evalúa como el carácter '$' normal.
${PROJECT_NAME} Nombre del proyecto actual
