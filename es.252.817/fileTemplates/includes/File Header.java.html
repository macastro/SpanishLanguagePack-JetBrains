Contiene un fragmento de código que se puede incluir en plantillas de archivos (pestaña Templates) con la ayuda de la directiva #parse.
Esta plantilla integrada es editable. Además de texto estático, código y comentarios, también puede utilizar variables predefinidas, que se expandirán posteriormente a los valores correspondientes, al igual que las macros.
Las variables predefinidas aceptan los siguientes valores:
${PACKAGE_NAME} Nombre del paquete donde se crea el nuevo archivo
${USER} Nombre de inicio de sesión del usuario actual en el sistema
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${MONTH_NAME_SHORT} Las 3 primeras letras del nombre del mes actual. Ej: Jan, Feb, etc.
${MONTH_NAME_FULL} Nombre completo del mes actual. Ej: January, February, etc.
${DAY} Día del mes actual
${DAY_NAME_SHORT} Las 3 primeras letras del nombre del día actual. Ej: Mon, Tue, etc.
${DAY_NAME_FULL} Nombre completo del día actual. Ej: Monday, Tuesday, etc.
${HOUR} Hora actual
${MINUTE} Minuto actual
${PROJECT_NAME} Nombre del proyecto actual
