Contiene un fragmento de código que se puede incluir en las plantillas de archivo (pestaña Plantillas) utilizando la directiva #parse.
Esta plantilla es editable. Además de texto estático, código y comentarios, también puede utilizar variables predefinidas, que se expandirán posteriormente a los valores correspondientes, como si fueran macros.
Las variables predefinidas adoptarán los siguientes valores:
${FILE_NAME} Nombre del archivo actual
${USER} Nombre de inicio de sesión del sistema del usuario actual
${USER_NAME} Nombre completo del usuario actual
${ORGANIZATION_NAME} Nombre de la organización del proyecto de Xcode
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${MONTH_NAME_SHORT} Los 3 primeros caracteres del nombre del mes actual. Ejemplos: Jan, Feb, etc.
${MONTH_NAME_FULL} El nombre completo del mes actual. Ejemplos: January, February, etc.
${DAY} Día actual
${DAY_NAME_SHORT} Los 3 primeros caracteres del nombre del día actual de la semana. Ejemplos: Mon, Tue, etc.
${DAY_NAME_FULL} El nombre completo del día actual de la semana. Ejemplos: Monday, Tuesday, etc.
${HOUR} Hora actual
${MINUTE} Minuto actual
${PROJECT_NAME} Nombre del proyecto actual
${COMPILER_VERSION_MAJOR} Versión principal del compilador de Swift
${COMPILER_VERSION_MINOR} Versión secundaria del compilador de Swift
${PREFER_EXPLICIT_RETURN} Valor de la opción Preferir retorno explícito (Preferencias | Editor | Estilo de código | Swift | Generación de código)
