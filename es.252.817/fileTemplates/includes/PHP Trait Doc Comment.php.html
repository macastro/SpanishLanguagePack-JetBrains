Una plantilla integrada para comentarios PHPDoc de un trait de PHP. Contiene un fragmento de código que se puede incluir en plantillas de archivo (pestaña Templates) con la ayuda de la directiva #parse.
La plantilla es editable. Además de texto estático, código y comentarios, puede utilizar variables predefinidas que se expandirán a sus valores correspondientes, como macros.
Las variables predefinidas aceptan los siguientes valores:
${NAME} Nombre del trait.
${NAMESPACE} Nombre completamente cualificado de la clase del trait, sin la barra inicial.
${DS} Signo de dólar, se evalúa al carácter ordinario "$".
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${MONTH_NAME_SHORT} Las 3 primeras letras del nombre del mes actual. Ej: Ene, Feb, etc.
${MONTH_NAME_FULL} Nombre completo del mes actual. Ej: Enero, Febrero, etc.
${DAY} Día del mes actual
${HOUR} Hora actual
${MINUTE} Minuto actual
${SECOND} Segundo actual
${PROJECT_NAME} Nombre del proyecto actual
${USER} Usuario actual
${CARET} Marca la posición a la que debe moverse el cursor de texto después de añadir el comentario.
