Plantilla integrada para comentarios PHPDoc de una función PHP. Contiene un fragmento de código que se puede incluir en plantillas de archivo (pestaña Templates) con la ayuda de la directiva #parse.
La plantilla es editable. Además de texto estático, código y comentarios, también puede utilizar variables predefinidas que se expandirán a sus valores correspondientes, como si fueran macros.
Las variables predefinidas aceptan los siguientes valores:
${NAME} Nombre de la función.
${TYPE_HINT} Sugerencia de tipo (type hint) para el valor de retorno de la función. Se evalúa como 'void' si no se infiere ningún tipo de retorno.
${PARAM_DOC}

Comentarios PHPDoc para los parámetros.

Se generan como una serie de líneas '* @param type name'. Se evalúa como contenido vacío si no hay parámetros.
${THROWS_DOC}

Comentarios de documentación (doc comment) para las excepciones.

Se generan como una serie de líneas '* @throws type'. Se evalúa como contenido vacío si no se lanzan excepciones.
${STATIC} Toma el valor 'static' si la función/método es estático, y una cadena vacía en caso contrario. Por ejemplo: 
      #if (${STATIC} == "static") * @static
      #end
${DS} El signo de dólar, se evalúa como el carácter '$' normal.
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${MONTH_NAME_SHORT} Los primeros 3 caracteres del nombre del mes actual. Ejemplo: Ene, Feb, etc.
${MONTH_NAME_FULL} Nombre completo del mes actual. Ejemplo: Enero, Febrero, etc.
${DAY} Día del mes actual
${HOUR} Hora actual
${MINUTE} Minuto actual
${SECOND} Segundo actual
${PROJECT_NAME} Nombre del proyecto actual
${USER} Usuario actual
${CARET} Marca la posición a la que debe moverse el cursor de texto después de añadir el comentario.
