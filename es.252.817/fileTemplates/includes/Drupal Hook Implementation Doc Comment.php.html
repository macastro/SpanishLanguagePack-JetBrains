Una plantilla incorporada para comentarios PHPDoc de una implementación de hook de Drupal. Contiene un fragmento de código que puede incluirse en una plantilla de archivo (etiqueta Templates) con la ayuda de la directiva #parse.
La plantilla es editable. Además del texto estático, código y comentarios, puede usar variables predefinidas que se expandirán como macros a sus valores correspondientes.
Las variables predefinidas aceptan los siguientes valores:
${HOOK_NAME}   Nombre del hook.
