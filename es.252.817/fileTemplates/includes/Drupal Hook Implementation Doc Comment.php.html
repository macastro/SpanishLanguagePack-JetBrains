Una plantilla integrada para comentarios PHPDoc que implementan un hook de Drupal. Contiene un fragmento de código que se puede incluir en plantillas de archivo (pestaña Templates) con la ayuda de la directiva #parse.
La plantilla es editable. Además de texto estático, código y comentarios, puede usar variables predefinidas, que se expandirán a sus valores correspondientes como si fueran macros.
Las variables predefinidas aceptan los siguientes valores:
${HOOK_NAME} Nombre del hook.
