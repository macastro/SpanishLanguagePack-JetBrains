Plantilla incorporada para comentarios PHPDoc de una clase PHP. Contiene un fragmento de código que puede incluirse en plantillas de archivo (pestaña Templates) con ayuda de la directiva #parse.
La plantilla es editable. Además del texto estático, código y comentarios, puede usar variables predefinidas que se expandirán como macros a sus valores correspondientes.
Las variables predefinidas aceptan los siguientes valores:
${NAME}   Nombre de la clase.
${NAMESPACE}   Nombre completamente calificado del espacio de nombres de la clase, sin barra diagonal inicial.
${DS}   Símbolo del dólar, evaluado como el carácter "$" normal.
${DATE}   Fecha actual del sistema
${TIME}   Hora actual del sistema
${YEAR}   Año actual
${MONTH}   Mes actual
${MONTH_NAME_SHORT}   Primeras 3 letras del nombre del mes actual. Por ejemplo: Jan, Feb, etc.
${MONTH_NAME_FULL}   Nombre completo del mes actual. Por ejemplo: January, February, etc.
${DAY}   Día del mes
${HOUR}   Hora actual
${MINUTE}   Minuto actual
${SECOND}   Segundo actual
${PROJECT_NAME}   Nombre del proyecto actual
${USER}   Usuario actual
${CARET}   Marca la posición donde debe moverse el cursor después de añadir el comentario.
