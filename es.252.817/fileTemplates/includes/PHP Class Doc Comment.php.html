Una plantilla integrada para comentarios PHPDoc de clases PHP. Incluye un fragmento de código que puede ser incluido en plantillas de archivo (pestaña Templates) con la ayuda de la directiva #parse.
La plantilla es editable. Además de texto estático, código y comentarios, también puede usar variables predefinidas, que se expandirán a los valores correspondientes como si fueran macros.
Las variables predefinidas aceptan los siguientes valores:
${NAME} Nombre de la clase.
${NAMESPACE} Nombre completamente cualificado del espacio de nombres de la clase, sin barra inicial.
${DS} Signo de dólar, se evalúa al carácter ordinario '$'.
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${MONTH_NAME_SHORT} Primeras 3 letras del nombre del mes actual. Ej.: Jan, Feb, etc.
${MONTH_NAME_FULL} Nombre completo del mes actual. Ej.: January, February, etc.
${DAY} Día del mes
${HOUR} Hora actual
${MINUTE} Minuto actual
${SECOND} Segundo actual
${PROJECT_NAME} Nombre del proyecto actual
${USER} Usuario actual
${CARET} Marca la posición a la que debe moverse el cursor de texto después de añadir el comentario.
