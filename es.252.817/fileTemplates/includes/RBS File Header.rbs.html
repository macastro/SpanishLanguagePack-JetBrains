En esta plantilla, puede especificar fragmentos de código que se incluirán en la plantilla de archivo (pestaña de archivo) usando la directiva #parse. Una plantilla puede ser compartida por múltiples plantillas de archivo. Además de las declaraciones RBS, expresiones y comentarios, puede usar variables predefinidas que luego se expandirán como macros a sus valores correspondientes.
Las variables predefinidas aceptan los siguientes valores:
${FILE_NAME}   nombre del archivo actual
${USER}   nombre de inicio de sesión del usuario actual
${DATE}   fecha actual del sistema
${TIME}   hora actual del sistema
${YEAR}   año actual
${MONTH}   mes actual
${DAY}   día del mes actual
${HOUR}   hora actual
${MINUTE}   minuto actual
${PRODUCT_NAME}   nombre del IDE actual
${PROJECT_NAME}   nombre del proyecto actual
