Plantilla integrada para encabezado de archivo PHP. Contiene un fragmento de código que se puede incluir en plantillas de archivo (pestaña Templates) con la ayuda de la directiva #parse.
La plantilla es editable. Además de texto estático, código y comentarios, puedes usar variables predefinidas que se expandirán a los valores correspondientes como macros.
Las variables predefinidas toman los siguientes valores:
${FILE_NAME} Nombre del archivo actual
${USER} Nombre de inicio de sesión del usuario actual
${DATE} Fecha del sistema actual
${TIME} Hora del sistema actual
${YEAR} Año actual
${MONTH} Mes actual
${MONTH_NAME_SHORT} Las 3 primeras letras del nombre del mes actual. Ejemplos: Jan, Feb, etc.
${MONTH_NAME_FULL} Nombre completo del mes actual. Ejemplos: January, February, etc.
${DAY} Día del mes actual
${HOUR} Hora actual
${MINUTE} Minuto actual
${SECOND} Segundo actual
${PRODUCT_NAME} Nombre del IDE actual
${PROJECT_NAME} Nombre del proyecto actual
