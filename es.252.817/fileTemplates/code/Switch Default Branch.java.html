Genera la rama default para expresiones o sentencias switch utilizadas en acciones de corrección rápida como insertar rama 'default'.
Esta plantilla integrada es editable. Por ejemplo, puedes cambiar la excepción lanzada o llamar a un framework de logging en su lugar.
Las variables predefinidas aceptan los siguientes valores:
${EXPRESSION} Texto de la expresión selectora del switch
${EXPRESSION_TYPE} Tipo de la expresión selectora del switch
