Genera una rama default para la expresión o declaración switch utilizada en correcciones rápidas como insertar rama 'default'.
Esta plantilla incorporada es editable. Por ejemplo, puede modificar la excepción lanzada o llamar al framework de registro.
Las variables predefinidas aceptan los siguientes valores:
${EXPRESSION}   Texto de la expresión del selector switch
${EXPRESSION_TYPE}   Tipo de la expresión del selector switch
