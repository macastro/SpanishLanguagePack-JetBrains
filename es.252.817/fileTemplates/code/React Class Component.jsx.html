Esta plantilla integrada se utiliza para crear componentes de clase de React (por ejemplo, mediante la refactorización Extraer componente). Puede personalizar esta plantilla, por ejemplo, usando Component en lugar de PureComponent o cambiando la definición de los tipos de prop. También puede usar variables predefinidas. Al aplicar la plantilla, estas variables serán reemplazadas por los valores correspondientes.
Las variables predefinidas aceptan los siguientes valores:
${NAME} Nombre del componente
${COMPONENT_BODY} Cuerpo del componente
${PROP_TYPES} Definición de tipo de prop (usando el módulo prop-types)
${PROP_TYPES} Valores prop por defecto que React usará cuando no se proporcione ningún valor prop.
