Genera el cuerpo del método setter anulado de una propiedad al generarlo, por ejemplo, al invocar la acción Code | Override Method.
La plantilla es editable. Además de expresiones y comentarios de Objective-C, puede usar variables predefinidas que se expandirán a sus valores correspondientes.
Las variables predefinidas toman los siguientes valores:
${CALL_SUPER} Llamada al método de la superclase: para métodos void — [super methodName], con o sin parámetros formales; para otros métodos — return [super methodName], con o sin parámetros formales
${RETURN_TYPE} Tipo de retorno del método creado
${DEFAULT_RETURN_VALUE} Valor que el método devuelve por defecto
${CONTAINING_CLASS} Nombre de la clase en la que se anula el método
${IVAR_IS_AVAILABLE} Es "true" cuando la propiedad tiene una variable de instancia
${IVAR} Nombre de la variable de instancia
${PARAM} Nombre del parámetro del setter
${SETTER_SEMANTICS} En modo ARC: "copyArc" cuando el atributo copy está configurado, "assign" en caso contrario.
En modo no ARC: "copy" cuando el atributo copy está configurado, "retain" cuando el atributo retain está configurado, "assign" en caso contrario.
Todas las variables predefinidas de la plantilla de C File Header (pestaña Include) también están disponibles (por ejemplo, ${USER}, ${DATE} , etc.).
