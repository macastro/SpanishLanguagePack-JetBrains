Crea el cuerpo del método -copyWithZone: implementado que se genera cuando se genera, por ejemplo, al invocar la acción Código | Generar | -copyWithZone:.
La plantilla es editable. Además de expresiones y comentarios de Objective-C, puede usar variables predefinidas que se expandirán a sus valores correspondientes.
Las variables predefinidas tomarán los siguientes valores:
${RETURN_TYPE} Tipo de valor de retorno del método creado
${DEFAULT_RETURN_VALUE} Valor que el método devuelve por defecto
${CONTAINING_CLASS} Nombre de la clase donde se implementa el método
${CUSTOM_CODE} Cuerpo del método generado
Todas las variables predefinidas de la plantilla 'C File Header' (pestaña Include) también están disponibles (por ejemplo, ${USER}, ${DATE} , etc.).
