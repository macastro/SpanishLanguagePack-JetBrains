Rellena el cuerpo de un subíndice implementado al generarlo (por ejemplo, usando la acción Code | Generate | Implement Methods).
La plantilla es editable. Además de expresiones y comentarios de Swift, puedes usar variables predefinidas que se expandirán a los valores correspondientes.
Las variables predefinidas tomarán los siguientes valores:
${RETURN_TYPE} Tipo de retorno del subíndice creado
${DEFAULT_RETURN_VALUE} Valor predeterminado sugerido para el valor de retorno (puede estar vacío)
${SIGNATURE} Firma del subíndice
${HAS_SET} Es "true" si el subíndice tiene un setter
${GETTER_EFFECTS} Efectos del getter para el getter de la propiedad (async, throws)
${GET_ATTRIBUTES} Atributos para el getter del subíndice
${SET_ATTRIBUTES} Atributos para el setter del subíndice
Todas las variables predefinidas de la plantilla Swift File Header (pestaña Include) también están disponibles (por ejemplo, ${USER}, ${DATE}, etc.).
