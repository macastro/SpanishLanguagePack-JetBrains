Una plantilla integrada para un método de prueba de PHPSpec, utilizada, por ejemplo, cuando se selecciona la casilla de verificación para generar el método en el diálogo 'Nuevo Test'.
Las variables predefinidas toman los siguientes valores:
${NAME} Nombre del método.
${CAPITALIZED_NAME} Nombre del método con la primera letra en mayúscula.
${TESTED_NAME} Nombre de la clase que se está probando.
