Go Test es una plantilla integrada que se puede usar para crear funciones de prueba de Go en archivos de prueba. Esta plantilla solo funciona para archivos de prueba con el sufijo _test. Para usar la plantilla, haga clic en Código | Generar y seleccione Prueba.
Lista de variables predefinidas
${NAME} Nombre de la función de prueba
${PARAM_NAME} Nombre del parámetro de la función
${PARAM_TYPE} Tipo del parámetro de la función
