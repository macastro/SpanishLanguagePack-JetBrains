Go Test es una plantilla incorporada que se puede usar para crear funciones de prueba Go en archivos de test. Esta plantilla solo está disponible para archivos de prueba con el sufijo _test. Para usar esta plantilla, haga clic en Código | Generar y seleccione Test.
Lista de variables predefinidas
${NAME}   Nombre de la función de prueba
${PARAM_NAME}   Nombre del parámetro de la función
${PARAM_TYPE}   Tipo del parámetro de la función
