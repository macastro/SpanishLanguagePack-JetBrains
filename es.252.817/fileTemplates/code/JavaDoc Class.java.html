Define las plantillas utilizadas para generar JavaDoc para clases, interfaces o records, por ejemplo, al aplicar la acción de intención Agregar Javadoc.

Además de las expresiones y comentarios de Java, puede usar variables predefinidas que luego se expandirán a sus valores correspondientes, de forma similar a las macros.

Mediante el uso de la directiva #parse, es posible incluir plantillas de la pestaña Includes. Para incluir una plantilla, especifique su nombre completo entre comillas como argumento (por ejemplo, #parse("File Header.java")).
Las variables predefinidas aceptan los siguientes valores:
${RECORD_COMPONENTS} Lista de componentes del record
${TYPE_PARAMS} Lista de nombres de parámetros de tipo
${CONTAINING_CLASS} Nombre de la clase contenedora
${INTERFACE} Devuelve true si el propietario del JavaDoc es una interfaz
${ENUM} Devuelve true si el propietario del JavaDoc es una enumeración
${RECORD} Devuelve true si el propietario del JavaDoc es un record
${PACKAGE_NAME} Nombre del paquete donde se crea la nueva enumeración
${NAME} Nombre de la nueva enumeración especificada en el diálogo Crear nueva clase
${USER} Nombre de inicio de sesión del usuario actual
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${MONTH_NAME_SHORT} Primeras 3 letras del nombre del mes actual (Ene, Feb, etc.)
${MONTH_NAME_FULL} Nombre completo del mes actual (Enero, Febrero, etc.)
${DAY} Día del mes
${HOUR} Hora actual
${MINUTE} Minuto actual
${PROJECT_NAME} Nombre del proyecto actual
