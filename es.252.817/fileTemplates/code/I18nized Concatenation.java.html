Reemplaza la concatenación de cadenas (como String s = "Hello " + name;) con una expresión de código Java internacionalizada (como String s = MessageFormat.format(​myResourceBundle.getString(​"hello.0"), name);).
Esta plantilla integrada es editable. Además de las expresiones y comentarios de Java, puedes usar variables predefinidas que luego se expandirán a los valores correspondientes.
Las variables predefinidas aceptan los siguientes valores:
${RESOURCE_BUNDLE} Expresión de tipo java.util.ResourceBundle; una instancia de este tipo está disponible en este contexto.
${PARAMETERS} Variables utilizadas en la concatenación de cadenas, que normalmente se pasan como argumentos a la expresión internacionalizada.
${PROPERTY_KEY} Nombre de la clave de propiedad definida en el archivo de propiedades correspondiente.
Generalmente, es el valor utilizado para el parámetro del método ResourceBundle.getString().
${PROPERTY_VALUE} Valor de la propiedad definida en el archivo de propiedades correspondiente.
Este es el valor literal original de la cadena Java.
