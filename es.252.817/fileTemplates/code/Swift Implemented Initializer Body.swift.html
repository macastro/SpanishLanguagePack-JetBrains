Rellena el cuerpo del inicializador cuando se generan inicializadores implementados (por ejemplo, usando la acción Código | Generar | Implementar métodos).
La plantilla es editable. Además de expresiones y comentarios de Swift, también puede usar variables predefinidas, que se expandirán a los valores correspondientes.
Las variables predefinidas toman los siguientes valores:
${SIGNATURE} Firma del inicializador
Todas las variables predefinidas de la plantilla Encabezado de archivo de Swift (pestaña Incluir) también están disponibles (por ejemplo, ${USER}, ${DATE}, etc.).
