Una plantilla incorporada para el constructor de una clase PHP.
Las variables predefinidas aceptan los siguientes valores:
${PARAM_LIST}   Lista de parámetros del constructor separados por comas, correspondiente a cada propiedad seleccionada.
${PARAM_DOC}   Comentarios PHPDoc para los parámetros.
Se genera como una serie de líneas '* @param type name'. Si no hay parámetros, se evalúa como contenido vacío.
${THROWS_DOC}   Comentarios PHPDoc para las excepciones.
Se genera como una serie de líneas '* @throws type'. Si no hay excepciones lanzadas, se evalúa como contenido vacío.
${CLASS_NAME}   Nombre de la clase.
${BODY}   Cuerpo del constructor que contiene las asignaciones '$this->property=$property;'.
