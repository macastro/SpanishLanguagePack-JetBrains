Plantilla incorporada para un constructor de clase de PHP.
Las variables predefinidas aceptan los siguientes valores:
${PARAM_LIST} Lista de parámetros del constructor separada por comas, uno para cada propiedad seleccionada.
${PARAM_DOC} Comentario PHPDoc para los parámetros.
Se genera como una serie de líneas '* @param type name'. Si no hay parámetros, se evalúa a contenido vacío.
${THROWS_DOC} Comentario PHPDoc para las excepciones.
Se genera como una serie de líneas '* @throws type'. Si no se lanzan excepciones, se evalúa a contenido vacío.
${CLASS_NAME} Nombre de la clase.
${BODY} Cuerpo del constructor que contiene las asignaciones como '$this->property=$property;'.
