Define la plantilla utilizada para generar JavaDoc para constructores, por ejemplo, al aplicar la acción de intención Agregar Javadoc.

Además de expresiones y comentarios de Java, también puede utilizar variables predefinidas, que se expandirán a los valores correspondientes como macros.

Puede incluir plantillas de la pestaña Includes utilizando la directiva #parse. Para incluir una plantilla, especifique el nombre completo de la plantilla entre comillas como parámetro (por ejemplo, #parse("File Header.java")).
Las variables predefinidas toman los siguientes valores:
${PARAMS} Lista de nombres de parámetros
${TYPE_PARAMS} Lista de nombres de parámetros de tipo
${THROWS} Lista de excepciones lanzadas
${CONTAINING_CLASS} Nombre de la clase que la contiene
${PACKAGE_NAME} Nombre del paquete en el que se crea el nuevo enum
${NAME} Nombre del nuevo enum que especificó en el diálogo Crear nueva clase
${USER} Nombre de inicio de sesión del sistema del usuario actual
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${MONTH_NAME_SHORT} Las primeras 3 letras del nombre del mes actual (Jan, Feb, etc.)
${MONTH_NAME_FULL} Nombre completo del mes actual (January, February, etc.)
${DAY} Día del mes actual
${HOUR} Hora actual
${MINUTE} Minuto actual
${PROJECT_NAME} Nombre del proyecto actual
