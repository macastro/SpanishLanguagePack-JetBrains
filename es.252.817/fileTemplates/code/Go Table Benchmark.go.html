Go Table Benchmark es una plantilla incorporada que se puede utilizar para crear funciones de benchmark tabulares de Go en archivos de prueba. Esta plantilla solo está disponible para archivos de prueba con el sufijo _test. Para usar esta plantilla, haga clic en Código | Generar y seleccione Benchmark Tabular.
Lista de variables predefinidas
${NAME}   Nombre de la función de benchmark tabular
${PARAM_NAME}   Nombre del parámetro formal de la función
${PARAM_TYPE}   Tipo del parámetro formal de la función
