Este es una plantilla integrada utilizada para rellenar el cuerpo de un método Groovy cada vez que el programa genera uno (por ejemplo, usando la acción de intención Crear método desde uso).
La plantilla es editable. Además de expresiones y comentarios Groovy, también puede usar variables predefinidas que se expandirán a sus valores correspondientes.
Las variables predefinidas tomarán los siguientes valores:
${RETURN_TYPE} El tipo de retorno del método creado
${DEFAULT_RETURN_VALUE} El valor que devuelve el método por defecto
${METHOD_NAME} El nombre del método creado
${CLASS_NAME} El nombre cualificado de la clase donde se crea el método
${SIMPLE_CLASS_NAME} El nombre no cualificado de la clase donde se implementa el método
