Rellena el bloque else después de desenvolver un valor opcional con una sentencia guard. Esta plantilla es editable. Además de expresiones y comentarios de Swift, puedes usar variables predefinidas que se expandirán a sus valores correspondientes.
Las variables predefinidas tomarán los siguientes valores:
${RETURN_TYPE} El tipo de retorno de la función circundante (si existe)
${DEFAULT_RETURN_VALUE} Valor de retorno predeterminado sugerido (puede estar vacío)
Todas las variables predefinidas de la plantilla Encabezado de archivo Swift (pestaña Incluir) también están disponibles (por ejemplo, ${USER}, ${DATE}, etc.).
