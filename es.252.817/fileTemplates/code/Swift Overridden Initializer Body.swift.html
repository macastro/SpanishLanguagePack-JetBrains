Rellena el cuerpo de un inicializador cuando se genera un inicializador anulado (por ejemplo, utilizando la acción Código | Generar | Anular métodos).
La plantilla es editable. Además de expresiones Swift y comentarios, también puede utilizar variables predefinidas que se expandirán a los valores correspondientes.
Las variables predefinidas adoptan los siguientes valores:
${CALL_SUPER} Llamada al inicializador super: super.init() con o sin parámetros.
${SIGNATURE} La firma del método.
Todas las variables predefinidas de la plantilla Encabezado de archivo Swift (en la pestaña Incluir) también están disponibles (por ejemplo, ${USER}, ${DATE}, etc.).
