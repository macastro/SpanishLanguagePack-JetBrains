Rellena el cuerpo del método al generar métodos implementados (por ejemplo, usando la acción Código | Generar | Implementar métodos).
La plantilla es editable. Además de expresiones y comentarios de Swift, también puede usar variables predefinidas, que se expandirán a sus valores correspondientes.
Las variables predefinidas aceptan los siguientes valores:
${RETURN_TYPE} Tipo del valor de retorno del método creado
${DEFAULT_RETURN_VALUE} Valor predeterminado sugerido para el valor de retorno (puede estar vacío)
${SIGNATURE} Firma del método
Todas las variables predefinidas de la plantilla Encabezado de archivo Swift (pestaña Include) también están disponibles (por ejemplo, ${USER}, ${DATE}, etc.).
