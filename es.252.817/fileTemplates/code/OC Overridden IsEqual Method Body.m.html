Crea el cuerpo del método -isEqual: sobrescrito al generar (por ejemplo, al invocar las acciones Código | Sobrescribir método o Código | Generar | -isEqual: and -hash).
La plantilla es editable. Además de las expresiones y comentarios de Objective-C, también puede utilizar variables predefinidas, que luego se expandirán a los valores correspondientes.
Las variables predefinidas toman los siguientes valores:
${CALL_SUPER} Llamada al método super: para métodos void — [super nombreMétodo], con o sin parámetros; para otros métodos — return [super nombreMétodo], con o sin parámetros
${RETURN_TYPE} Tipo de retorno del método creado
${DEFAULT_RETURN_VALUE} Valor devuelto por el método por defecto
${CONTAINING_CLASS} Nombre de la clase en la que se sobrescribe el método
${CUSTOM_CODE} Nombre del método de comparación `isEqualToTipo:` específico del tipo (si se genera)
Todas las variables predefinidas de la plantilla de Encabezado de archivo C (pestaña Include) también están disponibles (por ejemplo, ${USER}, ${DATE}, etc.)
