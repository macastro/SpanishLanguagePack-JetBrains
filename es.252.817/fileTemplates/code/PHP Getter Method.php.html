Plantilla integrada para un método getter de PHP. Se utiliza, por ejemplo, al invocar la acción rápida "Añadir getter y setter".
Las variables predefinidas aceptan los siguientes valores:
${STATIC} Toma el valor 'static' si la propiedad es estática, de lo contrario, una cadena vacía.
${GET_OR_IS} El prefijo 'get/is'. Se evalúa a la cadena 'is' si una propiedad tiene un tipo booleano (sugerencia de tipo), de lo contrario, se evalúa a la cadena 'get'.
${NAME} Nombre del método en estilo camelCase sin el prefijo "get" y con el primer carácter en mayúscula. Por ejemplo, el nombre de propiedad '_my_property' se convierte en 'MyProperty'.
${CLASS_NAME} Contiene el nombre de la clase contenedora.
${RETURN_TYPE} Contiene la sugerencia de tipo de retorno si el nivel del lenguaje es PHP 7.0 o superior.
${FIELD_NAME} El nombre original de la propiedad tal cual.
${TYPE_HINT} La sugerencia de tipo de la propiedad, o una cadena vacía si no se especifica.
${DS} El signo de dólar, se evalúa al carácter '$' normal.
