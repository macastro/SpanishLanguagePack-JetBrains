Una plantilla incorporada para un método getter de PHP. Se utiliza, por ejemplo, cuando se invoca la corrección rápida "Agregar getter y setter".
Las variables predefinidas aceptan los siguientes valores:
${STATIC}   Evalúa a 'static' si la propiedad es estática, de lo contrario es una cadena vacía.
${GET_OR_IS}   Prefijo 'get/is'. Evalúa a 'is' si una propiedad tiene un tipo booleano (type hint), de lo contrario evalúa a la cadena 'get'.
${NAME}   Nombre del método en estilo camelCase, sin el prefijo "get" y con la primera letra en mayúscula. Por ejemplo, el nombre de propiedad '_my_property' se convierte en 'MyProperty'.
${CLASS_NAME}   Contiene el nombre de la clase.
${RETURN_TYPE}   Contiene el type hint de retorno si el nivel de lenguaje es PHP 7.0 o superior
${FIELD_NAME}   El nombre original de la propiedad sin modificaciones.
${TYPE_HINT}   El type hint de la propiedad, o cadena vacía si no está especificado.
${DS}   El símbolo del dólar, evaluado como el carácter "$" simple.
