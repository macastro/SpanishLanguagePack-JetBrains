Crea el cuerpo del método **-description** implementado cuando se genera para clases sin variables de instancia (por ejemplo, al invocar la acción **Code | Generate | -description**).
Esta plantilla es editable. Además de expresiones y comentarios de Objective-C, puede usar variables predefinidas que se expandirán a los valores correspondientes.
Las variables predefinidas toman los siguientes valores:
${RETURN_TYPE} Tipo de valor de retorno del método creado
${DEFAULT_RETURN_VALUE} Valor que devuelve el método por defecto
${CONTAINING_CLASS} Nombre de la clase en la que se implementa el método
Todas las variables predefinidas de la plantilla **C File Header** (pestaña **Include**) también están disponibles (por ejemplo, ${USER}, ${DATE}, etc.)
