Rellena el cuerpo del método cuando se generan métodos sobreescritos (por ejemplo, usando la acción Código | Generar | Sobreescribir métodos).
La plantilla es editable. Además de las expresiones y comentarios de Swift, puede usar variables predefinidas que luego se expandirán a los valores correspondientes.
Las variables predefinidas tomarán los siguientes valores:
${CALL_SUPER} Llamada al método super, para métodos Void: super.methodName() con o sin parámetros; para otros métodos: return super.methodName() con o sin parámetros
${RETURN_TYPE} Tipo de valor de retorno del método creado
${DEFAULT_RETURN_VALUE} Valor predeterminado que devuelve el método
${SIGNATURE} Firma del método
Todas las variables predefinidas de la plantilla Encabezado de archivo de Swift (pestaña Incluir) también están disponibles (por ejemplo, ${USER}, ${DATE}, etc.).
