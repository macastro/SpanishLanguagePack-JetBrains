Esta plantilla incorporada se utiliza para rellenar el cuerpo del método cada vez que se genera uno (por ejemplo, mediante la acción de intención Crear método a partir del uso).
La plantilla es editable. Además de las expresiones y comentarios de Javascript, también puede utilizar variables predefinidas. Estas variables se sustituirán por los valores correspondientes cuando se aplique la plantilla.
Las variables predefinidas adoptarán los siguientes valores:
${IS_ASSIGNMENT_TARGET} Devuelve 'true' si el método creado es un objetivo de asignación.
${IS_NEW_CALL} Devuelve 'true' si el método creado está bajo una llamada 'new'.
${HAS_EXPECTED_TYPE} Devuelve 'true' si el método creado está en una posición donde se espera un tipo.
${DEFAULT_RETURN_VALUE} Valor predeterminado para el tipo esperado del método creado ('undefined' si no hay un tipo esperado)
${METHOD_NAME} Nombre del método creado
${SEMICOLON} Punto y coma o vacío, basado en las opciones de estilo de código
Todas las variables predefinidas de la plantilla Encabezado de archivo (pestaña Incluir) también están disponibles (por ejemplo: ${USER}, ${DATE}, etc.).
