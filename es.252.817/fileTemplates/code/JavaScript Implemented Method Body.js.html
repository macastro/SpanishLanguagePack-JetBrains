Esta plantilla incorporada se utiliza para rellenar el cuerpo del método cada vez que se genera un método (por ejemplo, mediante la acción de intención crear método desde uso).
La plantilla es editable. Además de expresiones Javascript y comentarios, puede utilizar variables predefinidas. Estas variables serán reemplazadas por sus valores correspondientes cuando se aplique la plantilla.
Las variables predefinidas aceptarán los siguientes valores:
${IS_ASSIGNMENT_TARGET}   Devuelve 'true' si el método creado es un objetivo de asignación.
${IS_NEW_CALL}   Devuelve 'true' si el método creado está bajo una llamada 'new'.
${HAS_EXPECTED_TYPE}   Devuelve 'true' si el método creado está en una posición donde se espera un tipo.
${DEFAULT_RETURN_VALUE}   El valor predeterminado del tipo esperado para el método creado (o 'undefined' si no hay tipo esperado)
${METHOD_NAME}   El nombre del método creado
${SEMICOLON}   Punto y coma o vacío, según las opciones de estilo de código
Todas las variables predefinidas de la plantilla Encabezado de archivo (pestaña Include) también están disponibles. (Por ejemplo: ${USER}, ${DATE}, etc.)
