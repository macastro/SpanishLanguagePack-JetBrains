Esta plantilla integrada se utiliza para crear componentes de clase de React (por ejemplo, a través de la refactorización Extraer componente). Puede personalizar esta plantilla, por ejemplo, usando Component en lugar de PureComponent o cambiando la definición del tipo de prop. También puede utilizar variables predefinidas. Cuando se aplica esta plantilla, estas variables se sustituirán por los valores correspondientes. Esta plantilla se utiliza en TypeScript y Flow.
Las variables predefinidas tomarán los siguientes valores:
${NAME} Nombre del componente
${COMPONENT_BODY} Cuerpo del componente
${PROP_TYPES} Definición del tipo de props
${PROP_TYPES} React usará los valores predeterminados de prop cuando no se proporcione ningún valor de prop.
