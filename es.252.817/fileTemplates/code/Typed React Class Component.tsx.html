Esta plantilla incorporada se utiliza para crear componentes de clase React (por ejemplo, a través de la refactorización 提取组件). Puede personalizar esta plantilla, por ejemplo, usando Component en lugar de PureComponent o cambiando la definición de tipos de prop. También puede utilizar variables predefinidas. Cuando se aplica la plantilla, estas variables se reemplazarán con sus valores correspondientes. Esta plantilla se utiliza en TypeScript y Flow.
Las variables predefinidas aceptarán los siguientes valores:
${NAME}   Nombre del componente
${COMPONENT_BODY}   Cuerpo del componente
${PROP_TYPES}   Definición de tipos de props
${PROP_TYPES}   React utilizará valores predeterminados de prop cuando no se proporcionen valores de prop.
