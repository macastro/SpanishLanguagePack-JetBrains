Crea el cuerpo de las funciones implementadas durante la generación (por ejemplo, usando la acción Código | Implementar métodos).
Al editar la plantilla, puede usar expresiones C++, comentarios y las siguientes variables predefinidas:
${RETURN_TYPE}   El tipo de retorno de la función.
${DEFAULT_RETURN_VALUE}   El valor de retorno predeterminado de la función.
Todas las variables predefinidas de la plantilla Encabezado de archivo C (pestaña Incluir), como ${USER} y ${DATE}, también están disponibles.
