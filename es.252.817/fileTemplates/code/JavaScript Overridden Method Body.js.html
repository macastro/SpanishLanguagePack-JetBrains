Esta plantilla integrada se utiliza para rellenar el cuerpo de un método cada vez que se genera un método sobrescrito (por ejemplo, al invocar la función Código | Sobrescribir métodos...).
La plantilla es editable. Además de expresiones JavaScript y comentarios, también puede utilizar variables predefinidas. Estas variables se reemplazarán por los valores correspondientes cuando se aplique la plantilla.
Las variables predefinidas tomarán los siguientes valores:
${CALL_SUPER} Llamada al método super. Para métodos void: super.method_name(); con o sin parámetros; para otros métodos: return super.method_name(); con o sin parámetros.
${PLAIN_CALL_SUPER} Llamada simple al método super: super.method_name(); con o sin parámetros.
${RETURN_TYPE} El tipo de retorno del método creado.
${DEFAULT_RETURN_VALUE} El valor por defecto devuelto por el método.
${METHOD_NAME} El nombre del método sobrescrito.
Todas las variables predefinidas de la plantilla de Cabecera de archivo (pestaña Incluir) también están disponibles. (por ejemplo: ${USER}, ${DATE}, etc.).
