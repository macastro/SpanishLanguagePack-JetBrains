Genera el cuerpo del método getter sobrescrito de una propiedad, creado al generar (por ejemplo, al invocar la acción Code | Override Method).
La plantilla es editable. Además de expresiones y comentarios de Objective-C, también puede usar variables predefinidas, que luego se expandirán a sus valores correspondientes.
Las variables predefinidas toman los siguientes valores:
${CALL_SUPER} Llamada al método super: para métodos void — [super methodName], con o sin parámetros; para otros métodos — return [super methodName], con o sin parámetros
${RETURN_TYPE} Tipo del valor de retorno del método creado
${DEFAULT_RETURN_VALUE} Valor devuelto por defecto por el método
${CONTAINING_CLASS} Nombre de la clase en la que se sobrescribe el método
${IVAR_IS_AVAILABLE} Es "true" si la propiedad tiene una variable de instancia
${IVAR} Nombre de la variable de instancia
${SETTER_SEMANTICS} En modo ARC: "copyArc" si el atributo copy está configurado, de lo contrario "assign"
En modo no ARC: "copy" si el atributo copy está configurado, "retain" si el atributo retain está configurado, de lo contrario "assign"
Todas las variables predefinidas del plantilla C File Header (pestaña Include) también están disponibles (por ejemplo, ${USER}, ${DATE} , etc.)
