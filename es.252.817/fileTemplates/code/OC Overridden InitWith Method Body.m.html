Crea el cuerpo del método -init sobreescrito cuando se genera, por ejemplo, al invocar las acciones Code (Código) | Override Method (Sobreescribir método) o Code (Código) | Generate (Generar) | -initWith.
Esta plantilla es editable. Además de expresiones y comentarios de Objective-C, también puede usar variables predefinidas que se expandirán a los valores correspondientes.
Las variables predefinidas tomarán los siguientes valores:
${CALL_SUPER} Llamada al método de la superclase: para métodos void — [super methodName], con o sin parámetros; para otros métodos — return [super methodName], con o sin parámetros
${RETURN_TYPE} Tipo de retorno del método que se está creando
${DEFAULT_RETURN_VALUE} Valor predeterminado devuelto por el método
${CONTAINING_CLASS} Nombre de la clase en la que se sobreescribe el método
${CUSTOM_CODE} Cuerpo del método generado
Todas las variables predefinidas de la plantilla C File Header (Cabecera de archivo C) (pestaña Include (Incluir)) también están disponibles (${USER}, ${DATE}, etc.)
