Esta es la plantilla incorporada utilizada para rellenar el cuerpo del método cada vez que se genera un método dinámico de FreeMarker.
La plantilla es editable. Además de las expresiones Java y comentarios, puede utilizar variables predefinidas que luego se expandirán a sus valores correspondientes.
Las variables predefinidas aceptarán los siguientes valores:
${RETURN_TYPE}   El tipo de retorno del método creado
${DEFAULT_RETURN_VALUE}   El valor predeterminado que retorna el método
${METHOD_NAME}   El nombre del método creado
${CLASS_NAME}   El nombre cualificado de la clase donde se crea el método
