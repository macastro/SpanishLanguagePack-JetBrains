Esta es una plantilla para crear métodos de prueba HTTP para Ktor 2.x.x en una clase de prueba de JUnit 4.
Las variables predefinidas aceptan los siguientes valores:
${NAME} El nombre del método creado
${HTTP_METHOD} El método HTTP utilizado (GET, POST, etc.).
${ROUTE} La ruta relativa al servicio actual que necesita ser probado
${CONFIGURE_SERVER} Un método de extensión de ruta utilizado para configurar el árbol de enrutamiento para la parte del servidor que necesita ser probada
${BODY} El cuerpo del método generado
