Rellena el parámetro catch cuando se genera, por ejemplo, cuando se invoca la función Código | Rodear con....
Esta plantilla integrada es editable. Por ejemplo, puede hacer que el parámetro generado sea final y usar siempre un nombre predefinido, independientemente del nombre cualificado de la excepción capturada.
Las variables predefinidas toman los siguientes valores:
${EXCEPTION} El nombre de la variable Exception especificada como el parámetro catch derivado del tipo de excepción capturada e si el nombre cualificado termina en Exception.
${EXCEPTION_TYPE} El tipo del parámetro catch.
