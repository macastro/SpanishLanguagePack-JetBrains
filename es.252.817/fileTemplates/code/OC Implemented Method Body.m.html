Crea el cuerpo del método implementado cuando se genera (por ejemplo, al invocar la acción Código | Implementar método).
Esta plantilla es editable. Además de las expresiones y comentarios de Objective-C, también puede utilizar variables predefinidas que se expandirán a los valores correspondientes.
Las variables predefinidas tomarán los siguientes valores:
${RETURN_TYPE} El tipo de valor de retorno del método creado
${DEFAULT_RETURN_VALUE} El valor que el método devuelve por defecto
${CONTAINING_CLASS} El nombre de la clase en la que se implementa el método
Todas las variables predefinidas de la plantilla Cabecera de archivo C (pestaña Incluir) también están disponibles (por ejemplo, ${USER}, ${DATE} , etc.).
