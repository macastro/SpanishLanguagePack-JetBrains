Define la plantilla utilizada para generar Javadoc para métodos, por ejemplo, cuando se aplica la acción de intención Añadir Javadoc.

Además de expresiones y comentarios Java, también puede usar variables predefinidas, que luego se expanden como macros a su valor correspondiente.

Mediante el uso de la directiva #parse, se pueden incluir plantillas desde la pestaña Incluidos. Para incluir una plantilla, especifique su nombre completo como parámetro entre comillas (por ejemplo, #parse("File Header.java")).
Las variables predefinidas aceptan los siguientes valores:
${PARAMS} Lista de nombres de parámetros
${TYPE_PARAMS} Lista de nombres de parámetros de tipo
${THROWS} Lista de excepciones lanzadas
${RETURN_TYPE} El tipo devuelto por el método
${ELEMENT_NAME} El nombre del miembro de clase propietario del Javadoc generado
${CONTAINING_CLASS} El nombre de la clase padre
${PACKAGE_NAME} El nombre del paquete donde se crea el nuevo enum
${NAME} El nombre del nuevo enum especificado en el diálogo Crear Nueva Clase
${USER} El nombre de inicio de sesión del usuario actual
${DATE} La fecha actual del sistema
${TIME} La hora actual del sistema
${YEAR} El año actual
${MONTH} El mes actual
${MONTH_NAME_SHORT} Las 3 primeras letras del nombre del mes actual (Ene, Feb, etc.)
${MONTH_NAME_FULL} Nombre completo del mes actual (Enero, Febrero, etc.).
${DAY} El día del mes
${HOUR} La hora actual
${MINUTE} El minuto actual
${PROJECT_NAME} El nombre del proyecto actual
