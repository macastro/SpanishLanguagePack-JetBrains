Esta plantilla integrada se utiliza para completar cuerpos de métodos al generar métodos (por ejemplo, a través de la acción de intención Crear método a partir del uso).
La plantilla es editable. Además de expresiones y comentarios de TypeScript, también puede usar variables predefinidas. Cuando se aplica la plantilla, estas variables se reemplazarán por los valores correspondientes.
Las variables predefinidas toman los siguientes valores:
${RETURN_TYPE} Tipo de retorno del método creado
${DEFAULT_RETURN_VALUE} Valor devuelto por defecto por el método
${METHOD_NAME} Nombre del método creado
${SEMICOLON} Punto y coma o vacío, según las opciones de estilo de código
Todas las variables predefinidas de la plantilla Encabezado de archivo (pestaña Incluir) también están disponibles. (por ejemplo: ${USER}, ${DATE}, etc.)
