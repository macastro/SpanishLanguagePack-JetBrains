Crea el cuerpo del método -copyWithZone: sobrescrito cuando se genera (por ejemplo, al invocar la acción Código | Generar | -copyWithZone:).
La plantilla es editable. Además de las expresiones y comentarios de Objective-C, puede utilizar variables predefinidas que se expandirán a sus valores correspondientes.
Las variables predefinidas aceptan los siguientes valores:
${CALL_SUPER} Llamada al método super. Para métodos void: [super methodName], con o sin parámetros; para otros métodos: return [super methodName], con o sin parámetros
${RETURN_TYPE} Tipo de retorno del método creado
${DEFAULT_RETURN_VALUE} Valor que devuelve el método por defecto
${CONTAINING_CLASS} Nombre de la clase en la que se sobrescribe el método
${CUSTOM_CODE} Cuerpo del método generado
Todas las variables predefinidas de la plantilla Cabecera de archivo C (pestaña Include) también están disponibles (por ejemplo, ${USER}, ${DATE} , etc.).
