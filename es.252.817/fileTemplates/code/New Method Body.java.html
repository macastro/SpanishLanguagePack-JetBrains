Rellena el cuerpo del método cuando se genera un método (por ejemplo, al aplicar la acción de intención Crear método a partir de uso).
Esta plantilla integrada es editable. Además de las expresiones y comentarios de Java, también puede utilizar variables predefinidas que se expandirán a los valores correspondientes.
Las variables predefinidas aceptan los siguientes valores:
${RETURN_TYPE} Tipo de valor de retorno del método creado
${DEFAULT_RETURN_VALUE} Valor que el método devuelve por defecto
${METHOD_NAME} Nombre del método creado
${CLASS_NAME} Nombre cualificado de la clase donde se crea el método
${SIMPLE_CLASS_NAME} Nombre no cualificado de la clase donde se implementa el método
Todas las variables predefinidas de la plantilla Encabezado de archivo (pestaña Incluir) también están disponibles (por ejemplo: ${USER}, ${DATE}, etc.).
