Rellena el cuerpo del método al generarlo (por ejemplo, al aplicar la intención Crear método desde uso).
Esta plantilla incorporada es editable. Además de expresiones Java y comentarios, puede usar variables predefinidas que luego se expandirán a sus valores correspondientes.
Las variables predefinidas aceptan los siguientes valores:
${RETURN_TYPE}   Tipo de retorno del método creado
${DEFAULT_RETURN_VALUE}   Valor de retorno predeterminado del método
${METHOD_NAME}   Nombre del método creado
${CLASS_NAME}   Nombre cualificado de la clase donde se crea el método
${SIMPLE_CLASS_NAME}   Nombre no cualificado de la clase que implementa el método
Todas las variables predefinidas de la plantilla Encabezado de archivo (pestaña Include) también están disponibles. (Por ejemplo: ${USER}, ${DATE}, etc.)
