Reemplaza las expresiones literales de cadena (por ejemplo, String s = "Hello";) por expresiones de código Java internacionalizado (por ejemplo, String s = myResourceBundle.getString("hello");).
Esta plantilla integrada es editable. Además de expresiones Java y comentarios, puede usar variables predefinidas que se expandirán a sus valores correspondientes.
Las variables predefinidas aceptan los siguientes valores:
${RESOURCE_BUNDLE} Una expresión de tipo java.util.ResourceBundle, cuya instancia está disponible en este contexto.
${PROPERTY_KEY} El nombre de la clave de propiedad especificada en el archivo de propiedades correspondiente. Generalmente, es el valor utilizado para el parámetro del método ResourceBundle.getString().
${PROPERTY_VALUE} El valor de la propiedad definido en el archivo de propiedades correspondiente. Este es el valor literal de cadena de Java original.
