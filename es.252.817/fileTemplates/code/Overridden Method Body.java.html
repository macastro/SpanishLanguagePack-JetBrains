Rellena el cuerpo del método cada vez que el programa genera un método sobreescrito (por ejemplo, al llamar a la función Código | Sobreescribir métodos...).
Esta plantilla integrada es editable. Además de las expresiones y comentarios de Java, también puede usar variables predefinidas que se expandirán a los valores correspondientes.
Las variables predefinidas toman los siguientes valores:
${CALL_SUPER} Llamada al método super. Para métodos void - super.method_name();, con o sin parámetros; para otros métodos - return super.method_name();, con o sin parámetros
${PLAIN_CALL_SUPER} Llamada al método super, super.method_name();, con o sin parámetros
${RETURN_TYPE} Tipo de retorno del método creado
${DEFAULT_RETURN_VALUE} Valor que devuelve el método por defecto
${METHOD_NAME} Nombre del método sobreescrito
${CLASS_NAME} Nombre cualificado de la clase donde se sobreescribe el método
${SIMPLE_CLASS_NAME} Nombre no cualificado de la clase que implementa el método
Todas las variables predefinidas de la plantilla de Encabezado de archivo (pestaña Incluir) también están disponibles. (por ejemplo: ${USER}, ${DATE}, etc.)
