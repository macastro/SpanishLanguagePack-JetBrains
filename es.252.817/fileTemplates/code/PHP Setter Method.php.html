Plantilla integrada para un método setter de PHP. Se utiliza, por ejemplo, cuando se llama a la corrección rápida "Agregar getters y setters".
Las variables predefinidas toman los siguientes valores:
${STATIC} Toma el valor 'static' si la propiedad es estática, o una cadena vacía en caso contrario.
${NAME} Nombre del método con estilo camelCase, sin el prefijo "set" y con el primer carácter en mayúscula. Por ejemplo, un nombre de propiedad '_my_property' se convierte en 'MyProperty'.
${CLASS_NAME} Nombre de la clase contenedora.
${SCALAR_TYPE_HINT} Contiene la sugerencia de tipo escalar si el nivel de lenguaje es PHP 7.0 o superior.
${FIELD_NAME} El nombre original de la propiedad.
${PARAM_NAME} Nombre del parámetro del setter. Casi idéntico a ${FIELD_NAME}, pero se elimina el primer guion bajo si lo hay. Por ejemplo, un nombre de propiedad '_myProperty' se convierte en 'myProperty'.
${TYPE_HINT} Sugerencia de tipo (type hint) para la propiedad, una cadena vacía si no se especifica.
${VOID_RETURN_TYPE} Indica si el tipo de retorno void está disponible para el nivel de lenguaje PHP del proyecto.
${DS} El signo de dólar, que se evalúa como un carácter '$' normal.
