Crea el cuerpo del método sobrescrito cuando se genera (por ejemplo, al invocar la acción Código | Sobrescribir métodos).
La plantilla es editable. Además de expresiones y comentarios de Objective-C, puede usar variables predefinidas que luego se expandirán a sus valores correspondientes.
Las variables predefinidas tomarán los siguientes valores:
${CALL_SUPER}   Llamada al método super, para métodos void — [super methodName], con o sin parámetros formales; para otros métodos — return [super methodName], con o sin parámetros formales
${RETURN_TYPE}   Tipo de retorno del método creado
${DEFAULT_RETURN_VALUE}   Valor de retorno predeterminado del método
${CONTAINING_CLASS}   Nombre de la clase en la que se sobrescribe el método
Todas las variables predefinidas disponibles en la plantilla de Encabezado C (pestaña Includes) también están disponibles (por ejemplo, ${USER}, ${DATE}, etc.)
