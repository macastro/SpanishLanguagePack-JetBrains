El cuerpo del método -description que se crea durante la generación (por ejemplo, al invocar la acción Código | Generar | -description).
La plantilla es editable. Además de expresiones y comentarios de Objective-C, puede usar variables predefinidas que luego se expandirán a sus valores correspondientes.
Las variables predefinidas tomarán los siguientes valores:
${RETURN_TYPE}   El tipo de retorno del método creado
${DEFAULT_RETURN_VALUE}   El valor predeterminado que retorna el método
${CONTAINING_CLASS}   El nombre de la clase donde se implementa el método
${CUSTOM_CODE}   El cuerpo del método generado
Todas las variables predefinidas de la plantilla Encabezado de archivo C (pestaña Incluir) también están disponibles (por ejemplo, ${USER}, ${DATE}, etc.)
