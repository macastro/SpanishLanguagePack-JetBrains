Una plantilla integrada para métodos setter de PHP. Por ejemplo, se utiliza cuando se invoca la corrección rápida "agregar getters y setters".
Las variables predefinidas toman los siguientes valores:
${NAME} Nombre del método en estilo camel case, sin el prefijo "set" y con el primer carácter en mayúscula. Por ejemplo, el nombre de propiedad '_my_property' se convierte en 'MyProperty'.
${CLASS_NAME} El nombre de la clase contenedora.
${SCALAR_TYPE_HINT} Contiene la sugerencia de tipo escalar si el nivel del lenguaje es PHP 7.0 o superior.
${RETURN_TYPE} Indica si el nivel del lenguaje es PHP 7.0 o superior, permitiendo así especificar un tipo de retorno para la función/método.
${FIELD_NAME} El nombre original de la propiedad tal cual.
${PARAM_NAME} El nombre del parámetro del setter. Casi igual que ${FIELD_NAME}, pero el primer guion bajo (si lo hay) se elimina. Por ejemplo, el nombre de propiedad '_myProperty' se convierte en 'myProperty'.
${TYPE_HINT} La sugerencia de tipo de la propiedad. Si no se especifica, es una cadena vacía.
${DS} El signo de dólar, se evalúa como el carácter '$' normal.
