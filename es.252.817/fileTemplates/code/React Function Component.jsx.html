Esta plantilla integrada se utiliza para crear componentes funcionales de React (por ejemplo, a través de la refactorización Extraer componente). Puede personalizar esta plantilla, por ejemplo, definir el componente como una función flecha o cambiar la definición de los tipos de prop. También puede utilizar variables predefinidas. Cuando se aplica la plantilla, estas variables se reemplazan por los valores correspondientes.
Las variables predefinidas tomarán los siguientes valores:
${NAME} Nombre del componente
${HAS_PROPS} true si el componente tiene props
${COMPONENT_BODY} Cuerpo del componente
${PROP_TYPES} Definición de tipos de prop (usando el módulo prop-types)
