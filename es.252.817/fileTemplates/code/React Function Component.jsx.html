Esta plantilla incorporada se utiliza para crear componentes funcionales de React (por ejemplo, a través de la refactorización Extract Component). Puede personalizar esta plantilla, por ejemplo, definiendo el componente como una función flecha o cambiando la definición del tipo de prop. También puede utilizar variables predefinidas. Estas variables serán reemplazadas por sus valores correspondientes cuando se aplique la plantilla.
Las variables predefinidas aceptarán los siguientes valores:
${NAME}   Nombre del componente
${HAS_PROPS}   true si el componente tiene props
${COMPONENT_BODY}   Cuerpo del componente
${PROP_TYPES}   Definición de tipos de props (usando el módulo prop-types)
