Define la plantilla utilizada para generar JavaDoc para campos, por ejemplo, cuando se aplica la acción de intención Agregar JavaDoc.

Además de las expresiones y comentarios Java, puede usar variables predefinidas que posteriormente se expandirán a sus valores correspondientes, de forma similar a como lo hacen las macros.

Mediante el uso de la directiva #parse, puede incluir plantillas de la pestaña Includes. Para incluir una plantilla, especifique su nombre completo como un parámetro entre comillas (por ejemplo, #parse("File Header.java")).
Las variables predefinidas aceptan los siguientes valores:
${ELEMENT_NAME} Nombre del miembro de clase que es propietario del JavaDoc generado
${CONTAINING_CLASS} Nombre de la clase contenedora
${PACKAGE_NAME} Nombre del paquete en el que se crea la nueva enumeración
${NAME} Nombre de la nueva enumeración que especifica en el diálogo Crear nueva clase
${USER} Nombre de inicio de sesión del sistema del usuario actual
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${MONTH_NAME_SHORT} Primeras 3 letras del nombre del mes actual (Jan, Feb, etc.)
${MONTH_NAME_FULL} Nombre completo del mes actual (January, February, etc.)
${DAY} Día del mes actual
${HOUR} Hora actual
${MINUTE} Minuto actual
${PROJECT_NAME} Nombre del proyecto actual
