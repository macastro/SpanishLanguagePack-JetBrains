En las plantillas de archivo, puede usar texto, código, comentarios y variables predefinidas. A continuación se proporciona una lista de variables predefinidas. Cuando utiliza estas variables en una plantilla, se expandirán posteriormente a sus valores correspondientes en el editor.

También puede especificar variables personalizadas. Las variables personalizadas utilizan el siguiente formato: ${VARIABLE_NAME}, donde VARIABLE_NAME es el nombre de la variable (por ejemplo, ${MY_CUSTOM_FUNCTION_NAME}). Antes de que el IDE cree un nuevo archivo usando variables personalizadas, verá un diálogo donde podrá definir los valores para las variables personalizadas en la plantilla.

Mediante la directiva #parse, puede incluir plantillas de la pestaña Include. Para incluir una plantilla, especifique su nombre completo como parámetro entre comillas (por ejemplo, #parse("File Header")).
Lista de variables predefinidas
${DATE}   Fecha actual del sistema
${DAY}   Día del mes
${DAY_NAME_SHORT}   Primeras 3 letras del nombre del día de la semana actual (por ejemplo, Mon, Tue, etc.)
${DAY_NAME_FULL}   Nombre completo del día de la semana actual (Monday, Tuesday, etc.)
${DIR_PATH}   Ruta del directorio del nuevo archivo (relativa a la raíz del proyecto)
${DS}   Símbolo del dólar ($). Esta variable se usa para escapar el carácter dólar para que no se interprete como prefijo de una variable de plantilla.
${FILE_NAME}   Nombre del nuevo archivo
${HOUR}   Hora actual
${MINUTE}   Minuto actual
${SECOND}   Segundo actual
${MONTH}   Mes actual
${MONTH_NAME_SHORT}   Primeras 3 letras del nombre del mes actual (Jan, Feb, etc.)
${MONTH_NAME_FULL}   Nombre completo del mes actual (January, February, etc.)
${NAME}   Nombre de la nueva entidad (archivo, tipo, interfaz, etc.)
${ORGANIZATION_NAME}   Nombre de la organización especificado en la configuración del proyecto
${PRODUCT_NAME}   Nombre del IDE
${PROJECT_NAME}   Nombre del proyecto actual
${TIME}   Hora actual del sistema
${USER}   Nombre de inicio de sesión del usuario actual en el sistema
${YEAR}   Año actual
