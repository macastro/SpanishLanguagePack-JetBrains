Genera código repetitivo para el manejo idiomático de errores.

Se puede invocar en variables y expresiones de tipo error.

  • Reemplaza la variable err con la siguiente construcción:

    if err != nil {
    return value1, value2, ...
  }

  • Reemplaza la expresión expr con la siguiente construcción:

    if err := expr; err != nil {
    return value1, value2, ...
  }

Cada value es o bien la variable err o bien el valor predeterminado del tipo de retorno correspondiente de la función contenedora. Puedes invocar esta plantilla en expresiones con múltiples valores.
