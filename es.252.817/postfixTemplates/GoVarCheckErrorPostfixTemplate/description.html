Genera código repetitivo para el manejo idiomático de errores.

Puede invocar la plantilla en cualquier llamada que devuelva un error o en cualquier expresión de tipo error. La plantilla introduce nuevas variables, les asigna la expresión y verifica si el error no es nil con el siguiente código: 

if err != nil {
  return value1, value2, ...
}

Cada value es ya sea la variable err o el valor predeterminado del tipo de retorno correspondiente de la función contenedora.
