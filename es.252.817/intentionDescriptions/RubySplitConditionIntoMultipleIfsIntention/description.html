Divide una condición conjuntiva o disyuntiva en múltiples sentencias if.

Una condición conjuntiva se divide en dos if anidados, mientras que una condición disyuntiva se divide en dos sentencias if secuenciales con bloques duplicados.

Dividir una condición en múltiples sentencias if puede ser útil para mejorar la legibilidad cuando la condición se hace demasiado larga o para editarla si es necesario modificarla.
