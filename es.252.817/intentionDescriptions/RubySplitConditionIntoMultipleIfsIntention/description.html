Divide las condiciones de conjunción o disyunción en múltiples sentencias if.

Las condiciones de conjunción se dividen en dos sentencias if anidadas, mientras que las condiciones de disyunción se dividen en una secuencia de dos sentencias if con bloques repetidos.

Dividir una condición en múltiples sentencias if puede ser útil para mejorar la legibilidad cuando las condiciones se vuelven demasiado extensas, o para facilitar la edición cuando se necesitan modificar las condiciones.
