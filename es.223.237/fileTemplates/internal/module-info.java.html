Se aplica a los nuevos archivos module-info.java que se crean invocando Nuevo | module-info.java en la ventana de herramientas de Proyecto en el menú superior.

Esta plantilla incorporada se puede editar. Junto con las expresiones de Java y los comentarios, también se pueden usar las variables predefinidas (que se enumeran a continuación) que luego se expandirán en valores correspondientes.

También es posible especificar variables personalizadas. Las variables personalizadas utilizan el siguiente formato: ${NOMBRE_DE_LA_VARIABLE}, donde NOMBRE_DE_LA_VARIABLE es un nombre para su variable (por ejemplo, ${NOMBRE_DE_LA_FUNCION_PERSONALIZADA}). Antes de que el IDE cree un nuevo archivo con variables personalizadas, verá un cuadro de diálogo donde podrá definir valores para variables personalizadas en la plantilla.

Mediante la directiva #parse, se pueden incluir plantillas de la pestaña Incluye. Para incluir una plantilla, especifique el nombre completo de la plantilla como un parámetro entre comillas (por ejemplo, #parse("File Header.java").
Las variables predefinidas toman los siguientes valores:
${NOMBRE_DEL_MODULO} Nombre del módulo en el que se crea un nuevo archivo module-info.java
${USUARIO} Nombre de inicio de sesión del sistema del usuario actual
${FECHA} Fecha actual del sistema
${HORA} Hora actual del sistema
${AÑO} Año actual
${MES} Mes actual
${NOMBRE_DEL_MES_CORTO} Las primeras 3 letras del nombre del mes actual (ene, feb, etc.)
${NOMBRE_DEL_MES_COMPLETO} Nombre completo del mes actual (enero, febrero, etc.)
${DÍA} Día actual del mes
${HORA} Hora actual
${MINUTO} Minuto actual
${NOMBRE_DEL_PROYECTO} Nombre del proyecto actual
