Plantilla integrada para pruebas funcionales de Codeception. Esta se utilizará cada vez que se cree una nueva prueba funcional de Codeception (seleccionando Nuevo | Prueba PHP en el menú emergente de una de las vistas del proyecto y escogiendo 'Codeception Functional' como librería de pruebas).
La plantilla se puede editar. Además de las declaraciones, expresiones y comentarios PHP, también puedes utilizar las variables predefinidas que aparecen a continuación. Cada variable se expandirá al valor correspondiente al igual que una macro.
Utilizando la directiva #parse, puedes incluir plantillas de la pestaña Includes especificando el nombre completo de la plantilla que desees como parámetro entre comillas.
Las variables predefinidas toman los siguientes valores:
${TESTED_NAME} Nombre de la clase donde se realizan las pruebas
${TESTED_NAMESPACE} Espacio de nombres de la clase donde se realizan las pruebas
${NAME} Nombre de la clase de prueba
${NAMESPACE} Espacio de nombres de la clase de prueba
${FILE_NAME} Nombre del archivo actual
${USER} Nombre de inicio de sesión del sistema del usuario actual
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${DAY} Día del mes actual
${HOUR} Hora actual
${MINUTE} Minutos actuales
${PRODUCT_NAME} Nombre del IDE actual
${DS} El símbolo del dólar se interpretará como el carácter normal '$'.
${PROJECT_NAME} Nombre del proyecto actual
