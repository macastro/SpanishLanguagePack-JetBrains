Reemplace una concatenación de String (por ejemplo String s = "Hello " + name;) por una expresión de código Java internacionalizada (por ejemplo String s = MessageFormat.format(​myResourceBundle.getString(​"hello.0"), name);).
Esta plantilla incorporada es editable. Junto con las expresiones y comentarios de Java, también puede usar las variables predefinidas que luego se expandirán a los valores correspondientes.
Las variables predefinidas toman los siguientes valores:
${RESOURCE_BUNDLE} Expresión del tipo java.util.ResourceBundle, cuya instancia está disponible en este contexto.
${PARAMETERS} Variables usadas en la concatenación de cadenas, generalmente pasan como argumentos a la expresión internacionalizada.
${PROPERTY_KEY} Nombre de clave de propiedad que se define en el archivo de propiedades correspondiente. Normalmente es el valor que se utiliza para el parámetro del método ResourceBundle.getString().
${PROPERTY_VALUE} Valor de propiedad que se define en el archivo de propiedades correspondiente. Este es el valor literal de cadena de Java original.
