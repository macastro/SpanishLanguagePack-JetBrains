Define la plantilla que se usa para generar JavaDoc para los métodos de anulación, p. ej. cuando se aplica la acción intencional Agregar Javadoc.

Junto con comentarios y expresiones Java, también puedes usar las variables predefinidas que luego se expandirán como macros en los valores correspondientes.

Al usar la directiva #parse, puedes incluir plantillas de la ficha Incluye. Para incluir una plantilla, especifica el nombre completo de la plantilla como parámetro entre comillas (por ejemplo, #parse("File Header.java")).
Las variables predefinidas toman los siguientes valores:
${PARAMS} Lista de nombres de parámetros
${TYPE_PARAMS} Lista de nombres de parámetros tipo
${THROWS} Lista de excepciones lanzadas
${RETURN_TYPE} Tipo devuelto por el método
${PARAMS_INHERITED} Lista de parámetros con descripciones del método super.
${ELEMENT_NAME} Nombre del miembro de clase que es propietario del JavaDoc generado
${CONTAINING_CLASS} Nombre de la clase padre
${SUPER_METHOD} Nombre completo del método super
${PACKAGE_NAME} Nombre del paquete en el que se crea una nueva enumeración
${NAME} Nombre de la nueva enumeración especificada por ti en el cuadro de diálogo Crear nueva clase
${USER} Nombre de inicio de sesión del sistema del usuario actual
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${MONTH_NAME_SHORT} Primeras 3 letras del nombre del mes actual (ene, feb, etc.)
${MONTH_NAME_FULL} Nombre completo del mes actual (enero, febrero, etc.)
${DAY} Día actual del mes
${HOUR} Hora actual
${MINUTE} Minuto actual
