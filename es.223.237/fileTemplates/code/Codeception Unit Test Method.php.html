Plantilla incorporada para el método de prueba unitaria Codeception utilizado cuando la casilla de verificación para generar el método está seleccionada en el cuadro de diálogo 'Nueva prueba', por ejemplo.
Las variables predefinidas toman los siguientes valores:
${NAME} Nombre del método.
${CAPITALIZED_NAME} Nombre del método que comienza con mayúscula.
${TESTED_NAME} Nombre de la clase probada.
