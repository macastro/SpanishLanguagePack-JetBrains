Crea el cuerpo de un método implementado -copyWithZone: cuando se genera, por ejemplo, al llamar a la acción Código | Generar | -copyWithZone:.
La plantilla se puede editar. Junto con las expresiones y los comentarios de Objective-C, se pueden utilizar variables predefinidas que luego se expandirán a los valores correspondientes.
Las variables predefinidas serán los siguientes valores:
${RETURN_TYPE} el tipo de retorno del método creado
${DEFAULT_RETURN_VALUE} el valor que el método devuelve por defecto
${CONTAINING_CLASS} el nombre de la clase donde se implementa el método
${CUSTOM_CODE} el cuerpo del método generado
Todas las variables predefinidas de la plantilla C File Header (la pestaña Incluye) también están disponibles (${USER}, ${DATE}, etc.)
