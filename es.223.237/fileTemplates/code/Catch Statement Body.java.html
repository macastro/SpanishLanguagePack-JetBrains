Rellena el cuerpo de un bloque catch cuando se genera, por ejemplo, cuando se aplica la función Código | Rodear con ....
Esta plantilla incorporada es editable. Además de comentarios y expresiones Java, también puedes utilizar las variables predefinidas, que se expandirán a los valores correspondientes.
Las variables predefinidas toman los siguientes valores:
${EXCEPTION} Nombre de la variable de excepción especificada como parámetro de catch
${EXCEPTION_TYPE} Tipo del parámetro de catch
