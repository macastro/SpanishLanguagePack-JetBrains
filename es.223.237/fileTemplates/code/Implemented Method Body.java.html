Rellena el cuerpo de un método implementado cuando se genera, por ejemplo, cuando se aplican las acciones de intención de Implementar clase abstracta o Implementar método abstracto o cuando se llama a la función Código | Implementar métodos....
Esta plantilla integrada es editable. Junto con expresiones y comentarios Java, también puedes utilizar las variables predefinidas que se expandirán entonces a los valores correspondientes.
Las variables predefinidas toman los siguientes valores:
${RETURN_TYPE} Tipo de valor devuelto del método creado
${DEFAULT_RETURN_VALUE} Valor devuelto por el método por defecto
${METHOD_NAME} Nombre del método que se implementa
${CLASS_NAME} Nombre calificado de la clase en la que se implementa el método
${SIMPLE_CLASS_NAME} Nombre no calificado de la clase en la que se implementa el método
Todas las variables predefinidas de la plantilla Encabezado de archivo (pestaña Incluye) también están disponibles (por ejemplo, ${USER}, ${DATE}, etc.).
