Define la plantilla que se utiliza para generar el JavaDoc para los constructores, por ejemplo cuando se aplica la acción de intenciones Add Javadoc(Añadir Javadoc).

Junto con las expresiones y los comentarios de Java, también puedes usar las variables predefinidas que luego se expandirán como macros en los valores correspondientes.

Mediante el uso de la directiva #parse, puedes incluir plantillas de la pestaña Includes(Incluye). Para incluir una plantilla, especifica el nombre completo de la plantilla como parámetro entre comillas (por ejemplo, #parse("File Header.java")).
Las variables predefinidas toman los siguientes valores:
${PARAMS} Lista de nombres de parámetros
${TYPE_PARAMS} Lista de nombres de parámetros de tipos
${THROWS} Lista de excepciones lanzadas
${CONTAINING_CLASS} Nombre de la clase padre
${PACKAGE_NAME} Nombre del paquete en el que se crea una nueva enumeración
${NAME} Nombre de la nueva enumeración que especificó en el diálogo Create New Class (Crear nueva clase)
${USER} Nombre de inicio de sesión del sistema del usuario actual
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${MONTH_NAME_SHORT} Primeras 3 letras del nombre del mes actual (Ene., Feb., etc.)
${MONTH_NAME_FULL} Nombre completo del mes actual (enero, febrero, etc.)
${DAY} Día actual del mes
${HOUR} Hora actual
${MINUTE} Minuto actual
${PROJECT_NAME} Nombre del proyecto actual
