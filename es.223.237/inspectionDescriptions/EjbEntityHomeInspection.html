Se informa de las siguientes definiciones incorrectas de elementos relacionados con EJB de entidad:
  • La clase no implementa las interfaces necesarias (por ejemplo, EJBLocalObject o EJBObject)
  • Los modificadores de la clase EJB son incorrectos
  • La clase no define un constructor público sin argumentos
  • Se utiliza una clase en lugar de una interfaz (o viceversa)

Ejemplo:


public final class SessionEJB implements javax.ejb.SessionBean {
  private SessionEJB() {} // Error: la clase debe definir un constructor público sin argumentos
}
