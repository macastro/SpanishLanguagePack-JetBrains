Informa sobre cualquier método que no requiere la creación de una instancia de clase y que se puede hacer estático.

Ejemplo:


class MyClass(object):
    def my_method(self, x):
        print(x)

Si se aplica una corrección rápida Hacer función desde método, el código cambia a:


def my_method(x):
    print(x)


class MyClass(object):
    pass

Si selecciona la corrección rápida Hacer método static, se agrega el decorador @staticmethod:


class MyClass(object):
    @staticmethod
    def my_method(x):
        print(x)
