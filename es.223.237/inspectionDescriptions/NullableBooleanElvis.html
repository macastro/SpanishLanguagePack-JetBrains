Informa de los casos en los que debería utilizarse una comprobación de igualdad en lugar del operador "elvis".

Ejemplo:


fun check(a: Boolean? == null) {
    if (a ?: false) throw IllegalStateException()
}

Después de aplicar la revisión rápida:


fun check(a: Boolean? == null) {
    if (a == true) throw IllegalStateException()
}
