Detecta sentencias no permitidas antes del bloque 'plugins {}'.

Debido a las limitaciones de Gradle Plugin DSL, sólo un conjunto restringido de sentencias Groovy está disponible antes del bloque 'plugins {}'. Las únicas opciones son 'buildscript {}', 'pluginManagement {}' y otros 'plugins {}'.
Ver documentación de Gradle

Ejemplo:


import foo.bar.Baz
plugins {} // reporta 'plugins'
plugins {
  foo() // reporta 'foo'
  id 'java'
}
