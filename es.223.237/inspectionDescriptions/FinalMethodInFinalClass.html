Reporta métodos finales en clases finales.

Ya que las clases finales no pueden ser heredadas, marcar un método como final puede resultar innecesario y confuso.

Ejemplo:

record Bar(int a, int b) {
  public final int sumar() { 
     return a + b;
  }
}

Después de aplicar el arreglo rápido:

record Bar(int a, int b) {
  public int sumar() { 
     return a + b;
  }
}

Como se muestra en el ejemplo, una clase puede marcarse como final de forma explícita o implícita.
