Informa usos de clases concretas en lugar de interfaces. Estas declaraciones pueden representar una falla de abstracción y pueden dificultar las pruebas.

Esta inspección no informará las declaraciones cuyas clases provengan de bibliotecas del sistema o de terceros. No se informan conversiones, instancias ni variables locales en las implementaciones del método equals(). Además, no se informan las conversiones en las implementaciones del método clone().

Ejemplo:


  interfaz Entidad {}
  clase EntidadImplementa implements Entidad {}

  anular processObject(Objeto obj) {
    // advertencia: instancia de la clase concreta
    si (obj instanceof EntidadImplementa) {
      // advertencia: conversión a la clase concreta,
      // en lugar de la interfaz
      processEntity((EntidadImplementa)obj);
    }
  }
  // advertencia: parámetro de clase concreta
  anular processEntity(EntidadImplementa obj) {
  }

Usa la opción Ignorar tipo de clase abstracta para ignorar conversiones a clases abstractas.
