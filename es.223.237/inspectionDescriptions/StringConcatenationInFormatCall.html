Informa sobre concatenaciones de cadenas no constantes que se usan como argumentos de cadenas de formato.

Aunque ocasionalmente se hace con intención, normalmente es un uso erróneo de un método de formateo y puede incluso provocar problemas de seguridad si las variables usadas en la cadena concatenada contienen caracteres especiales como %.

Además, a veces puede ser el resultado de concatenar por error un argumento de formato de cadena cuando se escribió + en lugar de ,.

Ejemplo:


  static String formatGreeting(String userName) {
    return String.format("Hello, " + userName);
  }

Aquí, userName se interpretará como parte de la cadena de formato, lo que puede resultar en IllegalFormatException (por ejemplo, si userName es "%") o en el uso de una gran cantidad de memoria (por ejemplo, si userName es "%2000000000%"). La llamada debería reemplazarse por String.format("Hello, %s", userName);.
