Informa los parámetros @PathVariable declarados en la firma del método pero ausentes en la ruta de la URL o viceversa. El quick-fix agrega el parámetro faltante.

Ejemplo:


  @RequestMapping("/path/{myVariable}/")
  public String handler(@PathVariable String name_is_not_equal_to_myVariable) {
    return "...";
  }

Después de aplicar el quick-fix, el resultado se verá así:


  @RequestMapping("/path/{myVariable}/")
  public String handler(@PathVariable String myVariable) {
    return "...";
  }
