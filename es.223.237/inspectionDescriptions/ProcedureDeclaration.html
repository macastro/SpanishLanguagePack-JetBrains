Informa sobre declaraciones de métodos con sintaxis de procedimiento.

No se recomienda que los desarrolladores usen sintaxis de tipo procedimiento para métodos.

Es inconsistente, puede generar errores y se eliminará en Scala 3.

La corrección rápida convierte la declaración de sintaxis de procedimiento a sintaxis de función.

Ejemplo:


  trait Test {
    def method()
  }

Después de aplicar la corrección rápida:


  trait Test {
    def method(): Unit
  }

Sugerencia: puedes usar Analizar / Ejecutar inspección por nombre (Ctrl+Alt+Shift+I) para aplicar esta inspección a todo el proyecto.
