Reporta expresiones booleanas negadas que pueden simplificarse.

El arreglo rápido simplifica la expresión booleana.

Ejemplo:


    fun test(n: Int) {
        !(0 == 1)
    }

Después de aplicar el arreglo rápido:


    fun test(n: Int) {
        0 != 1
    }

Ten en cuenta que esta acción puede cambiar la semántica del código si hay valores IEEE-754 NaN involucrados:


    fun main() {
        println(!(Double.NaN >= 0)) // true
    }

Después de aplicar el arreglo rápido:


    fun main() {
        println(Double.NaN < 0) // false
    }
