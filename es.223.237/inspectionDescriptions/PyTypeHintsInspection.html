타입 힌트의 유효하지 않은 사용 사례를 보고합니다.

예:

from typing import TypeVar

T0 = TypeVar('T1') # 'TypeVar'의 인수는 'T0'여야 함


def b(p: int) -> int:  # 주석 및 어노테이션 모두에 지정된 타입
    # type: (int) -> int
    pass


def c(p1, p2): # 타입 시그니처의 인수가 너무 많음
    # type: (int) -> int
    pass

사용 가능한 빠른 수정은 다양한 액션을 제의합니다. 문제가 있는 요소를 이름 변경, 제거, 이동할 수 있습니다. 타입 선언을 수동으로 수정하여 경고가 표시되지 않게 할 수 있습니다.
