Informa sobre sentencias con etiqueta dentro de sentencias switch. Si bien ocasionalmente es intencional, esta construcción a menudo es el resultado de un error tipográfico.

Ejemplo:


  switch (x) {
      case 1:
      case2:   // advertencia: etiqueta de texto 'case2:' en sentencia 'switch'
      case 3:
          break;
  }
