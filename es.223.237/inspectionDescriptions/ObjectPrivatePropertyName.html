Informa las propiedades que no siguen las convenciones de nomenclatura.

Se informan las siguientes propiedades:

  • Propiedades privadas en objetos y objetos compañeros

Puede especificar el patrón requerido en las opciones de inspección.

Convenciones de nomenclatura recomendadas: deben comenzar con un guión bajo o una letra mayúscula, utilizar notación camello.

Ejemplo:


  class Persona {
    companion object {
      // propiedad en el objeto acompañante
      private val SIN_NOMBRE = Persona()
    }
  }
