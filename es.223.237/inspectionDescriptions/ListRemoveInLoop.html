Informa sobre List.remove(index) llamadas en un bucle que se puede reemplazar con List.subList().clear().

El reemplazo es más eficiente para la mayoría de las implementaciones de List cuando se eliminan muchos elementos.

Ejemplo:


  void removeRange(List<String> list, int from, int to) {
    for (int i = from; i < to; i++) {
        list.remove(from);
    }
  }

Después de aplicar la solución rápida:


  void removeRange(List<String> list, int from, int to) {
    if (to > from) {
      list.subList(from, to).clear();
    }
  }

La solución rápida agrega automáticamente una verificación de rango para evitar una posible IndexOutOfBoundsException cuando el valor mínimo es mayor que el valor máximo. Se puede eliminar si tal situación es imposible en su código.
