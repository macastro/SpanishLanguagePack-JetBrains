Informa funciones y propiedades public y protected que tienen un tipo de retorno implícito. Por motivos de estabilidad API, se recomienda especificar dichos tipos de manera explícita.

Ejemplo:


  fun publicFunctionWhichAbusesTypeInference() =
      otherFunctionWithNotObviousReturnType() ?: yetAnotherFunctionWithNotObviousReturnType()

Después de aplicar la solución rápida:


  fun publicFunctionWhichAbusesTypeInference(): Api =
      otherFunctionWithNotObviousReturnType() ?: yetAnotherFunctionWithNotObviousReturnType()
