Informa sobre accesos a clases internas y anidadas en las que la llamada se califica mediante una subclase de la clase declarativa, en lugar de la propia clase declarativa.

Java permite dicha calificación, pero dichos accesos pueden indicar una sutil confusión de herencia y anulación.

Ejemplo:


  class Super {
    static class Inner {}
  }

  class Sub extends Super {
    void test() {
      Sub.Inner s = new Sub.Inner(); // La clase 'Inner' se declara en la clase 'Super', pero se referencia a través de la clase 'Sub'
    }
  }

Después de aplicar la revisión rápida:


  class Super {
    static class Inner {}
  }

  class Sub extends Super {
    void test() {
      Super.Inner s = new Super.Inner();
    }
  }
