Informa campos de documentos no resueltos especificados dentro de la anotación de org.springframework.data.mongodb.repository.Query.

Ejemplo:


@Document
class User {
  @Field
  String name;
}

interface UserRepository extends Repository<User, Long> {
  @Query("{ age: { $gt: ?0 } }") // 'age' está resaltado como no resuelto
  List<User> findUsersOlderThan(Integer age);
}
