Reporta llamadas sin cláusulas de argumentos que se refieren a métodos con una cláusula de parámetros vacía.

La convención es que un método debe tener paréntesis si tiene efectos secundarios. Si bien es posible omitir las cláusulas de argumentos en las llamadas al método (para adaptar el principio de acceso uniforme para Java), se recomienda que los usuarios agreguen una cláusula de argumento vacía cuando el método invocado representa más que una propiedad.

La corrección rápida agrega una cláusula de argumento vacía.

Ejemplo:


  def test(): Int = ???

  test

Después de que se apliquen las correcciones rápidas:


  def test(): Int = ???

  test()
