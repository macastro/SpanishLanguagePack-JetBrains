Reporta declaraciones o expresiones switch en los cuales la cláusula default está posicionada antes de otro caso. Un constructo así es innecesariamente confuso. Se provee una corrección rápida para mover la cláusula default a la última posición, si es posible.

Ejemplo:


  switch (n) {
      default:
          System.out.println();
          break;
      case 1:
          break;
  }

Después de aplicar la corrección rápida:


  switch (n) {
    case 1:
        break;
    default:
        System.out.println();
        break;
  }
