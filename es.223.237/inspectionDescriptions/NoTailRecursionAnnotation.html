Informa de métodos recursivos terminales sin la anotación @tailrec que verifica que el método será compilado con optimización de llamada de cola.

Ten en cuenta que un método debe ser efectivamente final para que se muestre esta inspección.

Ejemplo:


  def factorial(n: Long, acc: Long = 1): Long =
    if (n <= 1) acc else factorial(n - 1, acc * n)

Después de que el arreglo rápido es aplicado:


  @tailrec
  def factorial(n: Long, acc: Long = 1): Long =
    if (n <= 1) acc else factorial(n - 1, acc * n)
