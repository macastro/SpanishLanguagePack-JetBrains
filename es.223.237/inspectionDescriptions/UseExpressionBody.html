Informa a las expresiones return (de una sola línea o when) que pueden reemplazarse con la sintaxis del cuerpo de la expresión.

La sintaxis de cuerpo de la expresión es recomendada por la guía de estilo.

Se puede usar la corrección rápida "Convertir al cuerpo de la expresión" para modificar el código automáticamente.

Ejemplo:


  fun sign(x: Int): Int {
      return when { // <== puede simplificarse
          x < 0 -> -1
          x > 0 -> 1
          else -> 0
      }
  }

Después de aplicar la corrección rápida:


  fun sign(x: Int): Int = when {
      x < 0 -> -1
      x > 0 -> 1
      else -> 0
  }
