Reporta llamadas a métodos abstract de la clase actual durante la construcción de objetos.

Un método es llamado durante la construcción de objetos si se encuentra dentro de:

  • Constructor
  • Inicializador de instancia no estático
  • Inicializador de campo no estático
  • Método clone()
  • Método readObject()
  • Método readObjectNoData()

Dichas llamadas pueden provocar errores sutiles, ya que la inicialización del objeto puede ocurrir antes de la llamada al método.

Ejemplo:


  abstract class Parent {
    abstract void abstractMethod();
  }

  class Child extends Parent {
    Child() {
      abstractMethod();
    }
  }

Esta inspección comparte la funcionalidad con las siguientes inspecciones:

  • Método sobrecargable llamado durante la construcción del objeto
  • Método sobrecargado llamado durante la construcción del objeto

Solo se debe habilitar una inspección a la vez para evitar la duplicación de advertencias.
