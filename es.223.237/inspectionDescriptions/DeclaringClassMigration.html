Informa llamadas a la propiedad 'declaringClass' en tipos Enum, que producirán un error de compilación desde la versión 1.9.

'Enum.getDeclaringClass' se encuentra entra las funciones Java "ocultas", que no se ven normalmente al resolver. Sin embargo, es visible mediante una propiedad sintética, lo que es un fallo de interfaz.

Más detalles: KT-49653 Desuso y eliminación de la propiedad sintética Enum.declaringClass

La corrección rápida reemplaza la llamada con 'declaringJavaClass'.

Ejemplo:


  fun <E: Enum<E>> foo(values: Array<E>) {
    EnumSet.noneOf(values.first().declaringClass)
  }

Después de aplicar la corrección rápida:


  fun <E: Enum<E>> foo(values: Array<E>) {
    EnumSet.noneOf(values.first().declaringJavaClass)
  }

Esta inspección solo se informa si el nivel de lenguaje Kotlin del proyecto o módulo es 1.7 o superior.
