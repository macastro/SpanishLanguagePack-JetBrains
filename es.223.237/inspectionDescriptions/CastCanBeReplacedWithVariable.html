Señala las operaciones de conversión de tipo de datos que se podrían sustituir por una variable local o variable de patrón existente con el mismo valor.

Ejemplo:


  void foo(Object obj) {
    String s = (String) obj;
    System.out.println(((String) obj).trim());
  }

Después de aplicar la solución rápida:


  void foo(Object obj) {
    String s = (String) obj;
    System.out.println(s.trim());
  }
