Reporta verificaciones de longitud de arreglo innecesarias seguidas de una iteración de arreglo. Cuando la longitud del arreglo es cero, la iteración se saltará de todos modos, por lo que no es necesario verificar la longitud explícitamente.

Ejemplo:


  void f(String[] array) {
    if (array.length != 0) { // verificación innecesaria
      for (String str : array) {
        System.out.println(str);
      }
    }
  }

Se sugiere una corrección rápida para desenvolver o eliminar la verificación de longitud:


  void f(String[] array) {
    for (String str : array) {
      System.out.println(str);
    }
  }
