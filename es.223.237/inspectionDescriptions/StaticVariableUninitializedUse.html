Informa variables static que son leídas antes de la inicialización.

La inspección ignora las comprobaciones de igualdad con null.

Ejemplo:


  class Foo {
    public static int bar;

    public static void main(String[] args) {
      System.out.println(bar);
    }
  }

Obsérvese que esta inspección utiliza un algoritmo de flujo de datos muy conservador y puede informar incorrectamente las variables static como no inicializadas. Las variables que se informan como inicializadas siempre se inicializarán.

Utiliza la opción Ignorar campos primitivos para ignorar los campos primitivos no inicializados.
