Reporta variables estáticas cuyos nombres son muy cortos, muy largos o no siguen el patrón de expresión regular especificado.

Esta inspección no revisa las constantes, es decir, variables de tipo inmutable declaradas como static final

Utilice los campos provistos más abajo para especificar la longitud mínima, longitud máxima y expresión regular esperada para los nombres de variables estáticas.

Las expresiones regulares deben especificarse en el formato estándar java.util.regex.
