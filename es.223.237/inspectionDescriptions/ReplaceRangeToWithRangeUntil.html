Los reportes llaman a rangeTo u operador .. en vez de llamadas a ..<.

Usar las funciones correspondientes hace tu código más simple.

El arreglo rápido reemplaza el rangeTo o llamada .. con ..<.

Ejemplo:


  fun foo(a: Int) {
      for (i in 0..a - 1) {

      }
  }

Después que el arreglo rápido es aplicado:


  fun foo(a: Int) {
      for (i in 0..<a) {

      }
  }
