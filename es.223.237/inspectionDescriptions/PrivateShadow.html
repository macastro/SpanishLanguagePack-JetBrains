Informa de los parámetros de clase que sombrean una var de superclase.

Ejemplo:

class Animal {
  var name: String = ""
}

class Cat(name: String = "") extends Animal {
  // Convierte `name` en un val private[this] que sombrea la var
  def usage = name
}

Al extender una clase con una var no private, el usuario podría esperar que un parámetro de clase de la subclase anulara la var de la superclase, pero de hecho solo la sombreará. Esto a veces puede provocar errores no triviales.

Tenga en cuenta que el resaltado aparecerá solo si el valor se utiliza realmente.
