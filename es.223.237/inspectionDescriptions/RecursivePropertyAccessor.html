Reporta llamadas del accesor de propiedades recursivas que pueden terminar en un StackOverflowError. Tales llamadas usualmente se confunden con el acceso al campo de respaldo.

Ejemplo:


  var counter: Int = 0
      set(value) {
          counter = if (value < 0) 0 else value
      }

Luego de aplicar la corrección rápida:


  var counter: Int = 0
      set(value) {
          field = if (value < 0) 0 else value
      }
