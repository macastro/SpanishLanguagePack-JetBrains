Informa las comparaciones con null que van seguidas de una llamada a equals() con un argumento constante.

Ejemplo:


  if (s != null && s.equals("literal")) {}

Después de aplicar la corrección rápida:


  if ("literal".equals(s)) {}

Usa la configuración de la inspección para informar las llamadas a equals() con un argumento no constante cuando se demuestra que el argumento para equals() no es null.
