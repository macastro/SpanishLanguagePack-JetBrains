Reportea métodos con aridad variable, los cuales pueden ser anotados como @SafeVarargs. La anotación @SafeVarargs suprime advertencias desmarcadas sobre la creación de arreglos parametrizados en los sitios de llamadas.

Ejemplo:


  public class Foo<T> {
    private List<T> list = new ArrayList<>();

    public final void safeVarargs(T... elements) {
      Collections.addAll(list, elements);
    }
  }

Después de aplicar la corrección rápida:


  public class Foo<T> {
    private List<T> list = new ArrayList<>();

    @SafeVarargs
    public final void safeVarargs(T... elements) {
      Collections.addAll(list, elements);
    }
  }

Esta anotación no es soportada en Java 1.6 o JVM más viejas.
