Reporta la encapsulación de valores ya encapsulados.

Esta es una operación redundante puesto que cualquier valor encapsulado será primero desempaquetado automáticamente antes de encapsular el valor de nuevo. Si se hace dentro de un ciclo interno, dicho código puede causar problemas de rendimiento.

Ejemplo:


  Integer value = 1;
  method(Integer.valueOf(value));

Después de aplicar la corrección rápida:


  Integer value = 1;
  method(value);
