Informa de las llamadas a java.lang.String.concat().

Dichas llamadas se pueden sustituir por el operador + para más claridad y, posiblemente, para mayor rendimiento si el método se ha invocado sobre una constante con un argumento constante.

Ejemplo:


  String foo(String nombre) {
    return nombre.concat("foo");
  }

Después de aplicar la solución rápida:


  String foo(String nombre) {
    return nombre + "foo";
  }
