Informa sobre los intentos de acceso a la columna 0 de java.sql.ResultSet o java.sql.PreparedStatement. Por razones históricas, las columnas de java.sql.ResultSet y java.sql.PreparedStatement se numeran empezando por 1, en lugar de por 0, y acceder a la columna 0 es un error común en la programación JDBC.

Ejemplo:


  String getName(ResultSet rs) {
    return rs.getString(0);
  }
