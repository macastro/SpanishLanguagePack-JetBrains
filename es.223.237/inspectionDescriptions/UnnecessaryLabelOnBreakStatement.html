Reporta sentencias break con etiquetas innecesarias. Dichas etiquetas no cambian el flujo de control pero dificultan el seguimiento del código.

Ejemplo:


  etiqueta:
  for(int i = 0; i < 10; i++) {
    if (shouldBreak()) break etiqueta;
    //hacerAlgo
  }

Después de aplicar la corrección rápida:


  etiqueta:
  for(int i = 0; i < 10; i++) {
    if (shouldBreak()) break;
    //hacerAlgo
  }
