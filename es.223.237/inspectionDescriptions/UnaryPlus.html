Informa sobre usos del operador unitario +. El signo más unitario suele ser una operación nula y su presencia puede ser muestra de un error de codificación. Por ejemplo, en una combinación con el operador de incremento (como en +++) o con el operador de igual (como en =+).

Ejemplo:


  void unaryPlus(int i) {
    int x = + +i;
  }

Se sugieren las siguientes correcciones rápidas:

  • Eliminar los operadores + antes de la variable i:

    
  void unaryPlus(int i) {
    int x = i;
  }

  • Sustituir los operadores + por el operador de incremento de prefijo:

    
  void unaryPlus(int i) {
    int x = ++i;
  }

Utiliza la siguiente casilla de verificación para informar sobre los signos más unitarios que se utilizan junto con una expresión binaria o unitaria. Esto significa que la inspección no informará de situaciones en las que se utilice una expresión de signo más unitario en expresiones de inicialización de matriz o como argumento de un método.
