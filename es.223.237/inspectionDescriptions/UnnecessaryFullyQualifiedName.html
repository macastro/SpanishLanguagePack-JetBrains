Informa de los nombres de clase totalmente calificados que pueden acortarse.

El arreglo rápido acorta los nombres totalmente calificados y añade sentencias de importación si es necesario.

Ejemplo:


  class ListWrapper {
    java.util.List<Integer> l;
  }

Después de aplicar el arreglo rápido:


  import java.util.List;
  class ListWrapper {
    List<Integer> l;
  }

Configurar la inspección:

Utiliza la opción **Ignorar en sentencias de módulo de Java 9** para ignorar los nombres totalmente calificados dentro de las sentencias de módulo provides y uses de Java 9.

En Ajustes | Editor | Estilo de código | Java | Importaciones utiliza las siguientes opciones para configurar la inspección:

  • Utiliza la opción **Insertar importaciones para clases internas** si las referencias a clases internas deben calificarse con la clase externa.
  • Utiliza la opción **Usar nombres de clase totalmente calificados en JavaDoc** para permitir nombres totalmente calificados en Javadocs.
