Informa operandos de cadena vacía en concatenaciones de cadenas. La concatenación con la cadena vacía puede usarse para convertir objetos o primitivas que no son String en String, pero puede ser más claro usar una llamada al método String.valueOf().

Se sugiere una solución rápida para simplificar la concatenación.

Ejemplo:


  void foo(int x, int y) {
    String s = "" + x + " ; " + y;
  }

Después de aplicar la solución rápida:


  void foo(int x, int y) {
    String s = x + " ; " + y;
  }

Usa la opción Informar solo en caso de que se puedan eliminar cadenas vacías sin otros cambios para ignorar los casos en los que eliminar la cadena vacía requerirá agregar una conversión String.valueOf() de otro operando.
