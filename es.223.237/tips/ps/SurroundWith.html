Detectar partes del código

Para detectar un bloque de código rápidamente con una instrucción válida, selecciona el bloque de código en concreto del editor, y presiona &shortcut:SurroundWith; ( Código | Detectar ).

La lista de opciones disponibles o métodos auxiliares dependerá del contexto o el lenguaje. Por ejemplo, puedes llevar a cabo acciones como detectar un bloque html con etiquetas.
