Administrar solicitudes de incorporación de cambios

Puedes administrar solicitudes de incorporación de cambios de GitHub entrantes directamente desde PhpStorm. Para ello, selecciona "Git | GitHub | Mostrar solicitudes de incorporación de cambios" en el menú principal. Con PhpStorm, puedes asignar y combinar solicitudes de incorporación de cambios, ver la cronología y los comentarios insertados, enviar comentarios y revisiones y aprobar cambios sin tener que salir del IDE.
