Barra de navegación

La barra de navegación es un reemplazo para la ventana de herramientas 'Proyecto'. Permite buscar la estructura del proyecto, abrir archivos específicos o navegar a elementos de código específicos en el archivo actual.

Para mostrar la barra de navegación, presiona &shortcut:ShowNavBar; . Utiliza las flechas de dirección para desplazarte por archivos y carpetas.
