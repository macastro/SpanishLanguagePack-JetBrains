Depuración con una configuración de Rails predefinida

Para proyectos Rails, RubyMine crea automáticamente una configuración de ejecución/depuración development usando el entorno Rails correspondiente. Para comenzar a depurar tu aplicación usando la configuración de ejecución/depuración development, presiona Control dos veces para invocar el popup "Ejecutar acción". Luego, comienza a escribir development , selecciónalo de la lista y presiona la tecla Intro mientras mantienes presionada la tecla Mayús (el título del diálogo cambiará a "Depurar").
