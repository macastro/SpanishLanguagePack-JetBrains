Compleción de código

Cuando se utiliza la función de compleción de código, presionar &shortcut:EditorChooseLookupItemReplace; permite aceptar el elemento seleccionado que actualmente está resaltado en la lista emergente.

A diferencia de aceptar con &shortcut:EditorChooseLookupItem;, el resto del nombre a la derecha del cursor se sobrescribe con el nombre seleccionado. Esto resulta particularmente útil cuando se reemplaza un método o nombre de variable por otro nombre de método o variable.
