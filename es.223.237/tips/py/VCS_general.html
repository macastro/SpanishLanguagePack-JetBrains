Control de versiones

PyCharm se puede integrar rápidamente con los sistemas de control de versiones Git, Mercurial, Subversion y Perforce.

Al utilizar el control de versiones para tus aplicaciones, no sólo puedes colaborar, sino también eliminar el riesgo de guardar toda tu base de código localmente. Presiona &shortcut:ActivateVersionControlToolWindow; para ver el historial de cambios del proyecto y presiona &shortcut:CheckinProject; para enviar los cambios locales al repositorio.
