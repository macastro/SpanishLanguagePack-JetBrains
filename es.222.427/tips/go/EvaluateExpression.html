Evaluación de expresión

Mientras el script está detenido en un punto de interrupción, puedes ejecutar una expresión o reasignar un valor a una variable. La función "Evaluación de expresión " es útil para obtener información adicional sobre el estado del programa o para probar distintos escenarios en tiempo de ejecución.

Para evaluar y mostrar rápidamente el valor actual de una parte del código, coloca el cursor en el fragmento de código y presiona &shortcut:QuickEvaluateExpression;. O bien, ve a "Ejecución | Acciones de depuración | Evaluación rápida de expresión ".
