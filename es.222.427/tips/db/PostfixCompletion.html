Finalización de código postfija

La Finalización de código postfija se aplica de derecha a izquierda y evita que el cursor se mueva hacia atrás mientras se codifica. Escriba un punto . después del fragmento de código que desea modificar y seleccione la opción deseada.

También puede pulsar &shortcut:InsertLiveTemplate; para mostrar una lista de sugerencias que incluyen plantillas postfijas.
