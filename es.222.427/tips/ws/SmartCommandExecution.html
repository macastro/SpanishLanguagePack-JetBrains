Ejecución de las funciones del IDE desde la terminal

Si un comando está resaltado en la terminal, no lo ejecutes todavía. En lugar de ello, pulsa Control+Enter (Windows y Linux) / Control+Enter (macOS) para activar la función correspondiente del IDE.

Por ejemplo, prueba git log para mostrar la pestaña "Registro" en la ventana de herramientas Git o ejecuta "Mostrar diferencias" en varios nombres de archivo y ábrelos en el visor de diferencias.
