Administración de solicitudes de incorporación de cambios

Puedes administrar solicitudes de incorporación de cambios de GitHub recibidas directamente desde WebStorm. Para hacerlo, selecciona en el menú principal:Git | GitHub | Mostrar solicitudes de incorporación de cambios. Con WebStorm, puedes asignar y combinar solicitudes de incorporación de cambios, mostrar líneas de tiempo y comentarios en línea, enviar comentarios y revisiones y aprobar cambios sin salir del IDE.
