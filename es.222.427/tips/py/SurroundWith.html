Rodear la sección del código

Para rodear rápidamente un bloque de código con un constructo válido, selecciona el bloque de código en el editor y presiona &shortcut:SurroundWith; ( Código | Rodear ).

La lista de opciones o envolturas disponibles varía según el contexto o el lenguaje. Por ejemplo, puedes realizar acciones como rodear un bloque html con etiquetas.
