Previsualización incorporada

La previsualización incorporada de RubyMine muestra el resultado del código HTML en una pestaña de edición independiente. RubyMine guarda los cambios realizados en el archivo HTML o en los archivos CSS o JavaScript vinculados y vuelve a cargar automáticamente la previsualización.

Para abrir la previsualización, coloca el cursor sobre el código HTML y haz clic en el ícono de RubyMine en el menú emergente del navegador.
