Esta es una plantilla incorporada que se utiliza para reemplazar el texto en GSP. Ejemplo:
<html> <body> Hello ${'world'}! </body> </html>
Usando etiquetas personalizadas para GSP internacionalizadas, es: Ejemplo:
<html> <body> <g:message code="HelloKey" args="['world']"/> </body> </html> .

La plantilla es editable. Además de las expresiones permitidas en el contexto de GSP, puedes usar variables predefinidas que se expandirán a su valor correspondiente.

Las variables predefinidas usan los siguientes valores:
${PROPERTY_KEY} El nombre de clave de la propiedad que se especifica en el archivo de propiedades correspondiente.
${PROPERTY_VALUE} El valor de la propiedad que se especifica en el archivo de propiedades correspondiente. Este es el valor del texto de GSP original seleccionado.
${ARGS_KEY} Argumentos
