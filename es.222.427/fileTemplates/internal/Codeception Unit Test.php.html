Plantilla incorporada para las pruebas de unidad de Codeception. Se utiliza cada vez que se crea una nueva prueba de unidad de Codeception (seleccione Nuevo | Prueba PHP en el menú emergente en una de las vistas del proyecto y seleccione 'Codeception Unit' como biblioteca de pruebas).
La plantilla se puede editar. Además de las expresiones, declaraciones y comentarios de PHP, también se pueden utilizar las variables predefinidas que se enumeran a continuación. Cada variable se expandirá al valor correspondiente, como una macro.
La directiva #parse permite incluir plantillas ubicadas en la pestaña Incluye especificando el nombre completo de la plantilla deseada entre comillas como parámetro.
Las variables predefinidas toman los siguientes valores:
${TESTED_NAME} Nombre de la clase bajo prueba
${TESTED_NAMESPACE} Espacio de nombres de la clase bajo prueba
${NAME} Nombre de la clase de prueba
${NAMESPACE} Espacio de nombres de la clase de prueba
${FILE_NAME} Nombre del archivo actual
${USER} Nombre de inicio de sesión del sistema del usuario actual
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${DAY} Día del mes actual
${HOUR} Hora actual
${MINUTE} Minuto actual
${PRODUCT_NAME} Nombre del IDE actual
${DS} El signo del dólar se evalúa en el caracter '$' normal.
${PROJECT_NAME} Nombre del proyecto actual
