Se aplica a las nuevas enumeraciones de Java que se crean invocando Nuevo | Clase Java | Enumeración en la ventana de herramientas Proyecto.

Esta plantilla integrada es editable. Junto con las expresiones y comentarios de Java, también puedes utilizar las variables predefinidas (que se enumeran a continuación) que luego se ampliarán como macros en los valores correspondientes.

También es posible especificar variables personalizadas. Las variables personalizadas utilizan el siguiente formato: ${NOMBRE_VARIABLE}, donde NOMBRE_VARIABLE es un nombre para tu variable (por ejemplo, ${NOMBRE_MI_FUNCION_PERSONALIZADA}). Antes de que el IDE cree un nuevo archivo con variables personalizadas, ves un cuadro de diálogo donde puedes definir valores para variables personalizadas en la plantilla.

Mediante la directiva #parse puedes incluir plantillas de la pestaña Incluye. Para incluir una plantilla, especifica el nombre completo de la plantilla como un parámetro entre comillas (por ejemplo, #parse("Encabezado de archivo.java").
Las variables predefinidas toman los siguientes valores:
${PACKAGE_NAME} Nombre del paquete en el que se crea una nueva enumeración
${NAME} Nombre de la nueva enumeración que especificaste en el diálogo Crear nueva clase
${USER} Nombre de inicio de sesión del sistema del usuario actual
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año en curso
${MONTH} Mes actual
${MONTH_NAME_SHORT} Primeras 3 letras del nombre del mes actual (Ene, Feb, etc.)
${MONTH_NAME_FULL} Nombre completo del mes actual (enero, febrero, etc.)
${DAY} Día actual del mes
${HOUR} Hora actual
${MINUTE} Minuto actual
${PROJECT_NAME} Nombre del proyecto actual
適用於通過在專案工具視窗中呼叫新建 | Java 類別 | 列舉所建立的新 Java 列舉。

您也可以指定自訂變數。自訂變數採用下列格式：${VARIABLE_NAME}，其中 VARIABLE_NAME 是變數的名稱（例如，${MY_CUSTOM_FUNCTION_NAME}） 。 在 IDE 建立包含自訂變數的新檔案之前，您將會看到一個對話方塊，您可以在其中定義範本中自訂變數的值。
