Informa sobre los bucles for que iteran sobre una colección usando la función withIndex() y que no utilizan la variable de índice.

Utiliza la corrección rápida "Eliminar índices en el bucle 'for'" para optimizar el código.

Ejemplos:


  fun foo(bar: List<String>) {
     for ((index : Int, value: String) in bar.withIndex()) { // <== 'index' no se usa
         println(value)
     }
  }

Después de aplicar la corrección rápida:


  fun foo(bar: List<String>) {
      for (value: String in bar) { // <== '.withIndex()' e 'index' se han eliminado
          println(value)
      }
  }
