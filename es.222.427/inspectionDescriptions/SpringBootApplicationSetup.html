Informa de @SpringBootApplication en el paquete predeterminado y de anotaciones redundantes @EnableAutoConfiguration o @ComponentScan.

La corrección rápida elimina las anotaciones redundantes.

Ejemplo:


@SpringBootApplication
@ComponentScan // Se informa de 'Declaración redundante: @SpringBootApplication ya implica @ComponentScan'
public class DemoApplication {
    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
}

Después de aplicar la corrección rápida:


  @SpringBootApplication
  public class DemoApplication {
    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
  }
