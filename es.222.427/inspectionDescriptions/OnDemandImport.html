Informa de cualquier sentencia import que abarca todo el paquete ('* imports').

Algunos estándares de codificación prohíben este tipo de sentencias import.

Puedes configurar IntelliJ IDEA para que detecte y corrija este tipo de sentencias utilizando el comando Optimizar import. Vaya a Configuración | Editor | Estilo de código | Java | Importar, asegúrese de que está habilitada la opción Utilizar importación de clase única y especifique un valor en los campos Número de clases con importación '*' y Número de nombres con importación static '*'. Por lo tanto, esta inspección resulta útil principalmente para informes sin conexión para una base de código que no busca cambios.
