Informa sobre referencias a tipos genéricos con un número incorrecto de argumentos de tipo. El uso de un tipo genérico sin pasar el número correcto de argumentos de tipo es un error.

Hay una corrección rápida disponible para corregir el número de argumentos de tipo pasados.

Ejemplo:

class Example[A]
end

# Se han proporcionado demasiados argumentos de tipo
type example = Example[untyped, untyped]

Después de la corrección rápida:

class Example[A]
end

type example = Example[untyped]
