Reporta llamadas a funciones miembro virtuales desde constructores o destructores. Como la construcción comienza con la clase base y se mueve a las clases derivadas, los recursos de la clase derivada aún no se han inicializado. La destrucción se realiza en orden inverso, por lo que llamar a una función virtual puede provocar el acceso a los recursos que ya han sido destruidos.

Ejemplo:


  class Test {
  public:
      void virtual setUp();

      Test() {
          setUp();
      }
  };
