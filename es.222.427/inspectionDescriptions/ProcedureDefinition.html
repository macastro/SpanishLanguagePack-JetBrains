Reporta definiciones de métodos con sintaxis de procedimiento.

No se recomienda que los desarrolladores usen sintaxis tipo procedimiento para métodos. Es inconsistente, puede generar errores y se eliminará en Scala 3.

El arreglo rápido convierte la definición de sintaxis de procedimiento a sintaxis de función.

Ejemplo:


  class Test {
    def method() { }
  }

Después de aplicar el arreglo rápido:


  class Test {
    def method(): Unit = { }
  }

Sugerencia: Puede usar Analizar / Ejecutar inspección por nombre (Ctrl+Alt+Shift+I) para aplicar esta inspección a todo el proyecto.
