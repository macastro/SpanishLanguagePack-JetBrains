if informe de sentencia en la que se puede extraer una parte común de la rama.

Estas partes comunes son independientes de la condición y dificultan la comprensión del contenido de la if sentencia.

Ejemplo:


  if (x > 12) {
    doSomethingBefore();
    doSomethingDifferent1();
    doSomethingAfter();
  } else {
    doSomethingBefore();
    doSomethingDifferent2();
    doSomethingAfter();
  }

Después de aplicar la corrección rápida:


  doSomethingBefore();
  if (x > 12) {
    doSomethingDifferent1();
  } else {
    doSomethingDifferent2();
  }
  doSomethingAfter();

Esta es una función actualizada en 2018.1
