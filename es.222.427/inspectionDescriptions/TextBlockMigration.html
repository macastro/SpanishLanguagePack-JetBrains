Informa concatenaciones de String que se pueden simplificar reemplazándolas con bloques de texto.

Requisitos:

  • \n ocurre dos o más veces.
  • Los bloques de texto no están concatenados.

Utilice la opción Informe literales de cadena únicos para resaltar literales únicos que contengan saltos de línea. La corrección rápida seguirá estando disponible incluso cuando esta opción esté deshabilitada.

Ejemplo:


  String html = "<html>\n" +
                "    <body>\n" +
                "        <p>Hello, world</p>\n" +
                "    </body>\n" +
                "</html>\n";

Después de aplicar la corrección rápida:


  String html = """
                <html>
                    <body>
                        <p>Hello, world</p>
                    </body>
                </html>
                """;
