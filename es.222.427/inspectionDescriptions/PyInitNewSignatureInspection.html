Informa sobre firmas incompatibles de los métodos __new__ y __init__.

Ejemplo:


class MyClass(object):
    def __new__(cls, arg1):
        return super().__new__(cls)

    def __init__(self):
        pass

Si __new__ e __init__ tienen argumentos diferentes, no se puede instanciar MyClass.

Como solución, el IDE ofrece aplicar la refactorización Cambiar firma.
