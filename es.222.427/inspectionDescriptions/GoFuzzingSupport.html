Informa la presencia de pruebas fuzz cuando la versión del SDK de Go es menor a 1.18

Las pruebas fuzz son un método de pruebas automatizadas que implican una búsqueda dirigida de datos de entrada que puedan causar un bloqueo del programa o exponer un comportamiento no válido. Go admite las pruebas fuzz a partir de Go 1.18.

Ejemplo de una prueba fuzz:


func Div(a, b int) int {
	return a / b
}

func FuzzDiv(f *testing.F) {
	f.Fuzz(func(t *testing.T, a, b int) {
		Div(a, b) // informa un error de tiempo de ejecución: división entera entre cero
	})
}

Consulta Go Fuzzing para obtener más información.
