Inspecciona el código que usa == o != para comparar cadenas.

Estos operadores determinan igualdad referencial en lugar de comparar contenido. En la mayoría de los casos, las cadenas deberían compararse usando equals(), lo que realiza una comparación carácter a carácter cuando las cadenas son objetos diferentes.

Ejemplo:


  void foo(String s, String t) {
    final boolean b = t == s;
  }

Si se sabe que t no es null, entonces es seguro aplicar la revisión rápida "insegura" y obtener un resultado similar al siguiente:


  void foo(String s, String t) {
    final boolean b = t.equals(s);
  }
