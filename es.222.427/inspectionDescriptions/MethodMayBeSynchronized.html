Reporta métodos cuyo cuerpo contiene una sola expresión `synchronized`. La expresión de bloqueo para esta expresión `synchronized` debe ser igual a `this` para métodos de instancia o `[NombreClase].class` para métodos estáticos.

Para mejorar la legibilidad de este tipo de métodos, puedes eliminar el contenedor `synchronized` y marcar el método como `synchronized`.

Ejemplo:


  public int generateInt(int x) {
    synchronized (this) {
      return 1;
    }
  }

Después de aplicar la correción rápida:


  public synchronized int generateInt(int x) {
    return 1;
  }
