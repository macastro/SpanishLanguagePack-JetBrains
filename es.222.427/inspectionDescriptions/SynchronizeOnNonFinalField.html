Reporta expresiones de bloqueo de sentencias synchronized que consisten en una referencia de campo no final. Es poco probable que tales sentencias tengan una semántica útil, ya que diferentes subprocesos pueden adquirir bloqueos diferentes incluso cuando operan sobre el mismo objeto.

Ejemplo:


  private Object o;
  public void foo() {
    synchronized (o) // sincronización en un campo no final
    { }
  }
