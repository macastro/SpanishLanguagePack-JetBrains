Informe del "boxing" explícito que envuelve valores primitivos de objetos.

A partir de Java 5 no es necesario el "boxing" manual explícito, y se puede eliminar de manera segura.

Ejemplo:

  • Integer i = new Integer(1); → Integer i = Integer.valueOf(1);
  • int i = Integer.valueOf(1); → int i = 1;

Utiliza la opción Informar sólo expresiones "boxing" realmente innecesarias para informar sólo de los "boxing" realmente innecesarios, en los que el valor "boxing" se "unboxing" inmediatamente, implícita o explícitamente. En estos casos, se puede eliminar todo el paso de "boxing" y "unboxing". Esta inspección no informa sobre lo explícito y simple "boxing".

Esta inspección sólo informa si el nivel de idioma del proyecto o módulo es 5 o superior.
