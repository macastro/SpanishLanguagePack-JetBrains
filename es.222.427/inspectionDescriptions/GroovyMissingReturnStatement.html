Informa sobre la ausencia de sentencias return al final de los métodos cuyo tipo de retorno no es void. El final del método debe ser alcanzable mediante el flujo de ejecución del método.

Ejemplo:


String foo(int a) {
  if (a > 0) {
    return "más de cero"
  }
} // foo(-1) retornará 'null'

int bar(int a) {
  if (a > 0) {
    return a
  }
} // bar(-1) fallará con una excepción en tiempo de ejecución
