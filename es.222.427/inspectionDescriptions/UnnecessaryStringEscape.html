Inspecciona innecesariamente caracteres escapados en literales String y, opcionalmente, en char.

Los caracteres de tabulación escapados \t no se inspeccionan, porque son invisibles.

Ejemplos:


  String s = "\'Scare\' quotes";
  String t = """
    All you need is\n\tLove\n""";

Después de aplicar la solución rápida:


  String s = "'Scare' quotes";
  String t = """
    All you need is
    \tLove
  """;

Nuevo en la versión 2019.3
