Informa expresiones afectadas por conversión de desempaquetado (desempaquetado automático de objetos en valores primitivos). Intenta no utilizar objetos en lugar de primitivos. Podría afectar significativamente al rendimiento.

Ejemplo:

int x = new Integer(42);

La solución rápida explicita la conversión:

int x = new Integer(42).intValue();

Desempaquetado automático apareció en Java 5. Esta inspección puede ayudar a realizar una degradación para compatibilidad inversa con versiones anteriores de Java.
