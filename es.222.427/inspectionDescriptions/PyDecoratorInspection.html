Informa sobre el uso de decoradores @classmethod o @staticmethod en métodos fuera de una clase.

Ejemplo:


class State(object):

    @classmethod
    def my_state(cls, name):
        cls.name = name


@classmethod
def change_state(self):
    pass

El método change_state no debe usar el decorador @classmethod o debe ser movido a la declaración de la clase State.

Si se aplica la acción Eliminar decorador, el código cambia a:


class State(object):

    @classmethod
    def my_state(cls, name):
        cls.name = name


def change_state(self):
    pass
