if—else La inspección reporta claves redundantes else en las cadenas de sentencias y sentencias.

La clave else es redundante cuando todas las ramas anteriores terminan en sentencias return, throw, break o continue. En este caso, la sentencia de la rama else se puede mover después de la sentencia if y la clave else se puede eliminar.

Ejemplo:


  if (name == null) {
      throw new IllegalArgumentException();
  } else {
    System.out.println(name);
  }

Después de aplicar la corrección rápida:


  if (name == null) {
    throw new IllegalArgumentException();
  }
  System.out.println(name);

Deshabilita la opción Reportar cuando no hay sentencias después de la sentencia 'if' para ignorar los casos en que la sentencia if—else es la última sentencia en un bloque de código.
