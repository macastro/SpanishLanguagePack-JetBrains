Reporta constructores redundantes declarados dentro de los registros de Java.

Ejemplo 1:


  record Point(int x, int y) {
    public Point {} // podría eliminarse
  }
  
  record Point(int x, int y) {
    public Point(int x, int y) { // podría eliminarse
      this.x = x;
      this.y = y;
    }
  }

El quick-fix elimina los constructores redundantes.

Ejemplo 2:


   // podría convertirse en un constructor compacto
  record Range(int from, int to) {
    public Range(int from, int to) {
      if (from > to) throw new IllegalArgumentException();
      this.from = from;
      this.to = to;
    }
  }

El quick-fix convierte este código en un constructor compacto.
