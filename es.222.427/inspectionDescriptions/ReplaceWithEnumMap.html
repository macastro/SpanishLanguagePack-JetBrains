Reporta llamadas a la función `hashMapOf` o al constructor `HashMap` que puedan ser reemplazadas por una llamada al constructor `EnumMap`.

Utilizar el constructor `EnumMap` simplifica el código.

La corrección rápida reemplaza a la llamada a la función con la llamada al constructor `EnumMap`.

Ejemplo:


  enum class E {
      A, B
  }

  fun getMap(): Map<E, String> = hashMapOf()

Después de aplicar la corrección rápida:


  enum class E {
      A, B
  }

  fun getMap(): Map<E, String> = EnumMap(E::class.java)
