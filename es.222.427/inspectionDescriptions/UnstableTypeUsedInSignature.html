Informa de las declaraciones de clases, métodos y campos que hacen referencia a un tipo de API inestable en la firma, pero que no están marcados con la misma anotación inestable.

Esta inspección garantiza que las firmas de una API pública no expongan ningún tipo inestable (interno, experimental). Por ejemplo, si un método devuelve una clase experimental, el método en sí se considera experimental porque los cambios incompatibles del tipo (supresión o traslado a otro paquete) llevan a cambios incompatibles de firma de método.
