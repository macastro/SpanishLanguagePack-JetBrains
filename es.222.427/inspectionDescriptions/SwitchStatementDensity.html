Informa de las sentencias u expresiones de switch con una proporción demasiado baja de etiquetas de switch a sentencias ejecutables.

Estas sentencias de switch pueden resultar confusas y probablemente deban refactorizarse.

Ejemplo:


    switch (i) { // un caso y 5 sentencias ejecutables -> densidad del 20%
        case 1:
            System.out.println("1");
            System.out.println("2");
            System.out.println("3");
            System.out.println("4");
            System.out.println("5");
            break;
    }

Usa el campo Densidad mínima de ramas para especificar la proporción permitida de etiquetas de switch a sentencias ejecutables.
