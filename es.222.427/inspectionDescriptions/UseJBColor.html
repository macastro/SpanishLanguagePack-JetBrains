Se informan los usos de java.awt.Color.

Esto puede no detectar temas 'oscuros' (por ejemplo, el 'Darcula' incluido), lo que puede resultar en una IU de apariencia deficiente.

La corrección rápida reemplazará estos usos con JBColor, que puede definir variantes de color 'oscuras'.

Ejemplos:


// Malo:
Color darkGreen = new Color(12, 58, 27);
Color blue = Color.BLUE;

// Bueno:
Color darkGreen = new JBColor(12, 58, 27);
Color blue = JBColor.BLUE;
Color green = new JBColor(new Color(12, 58, 27), new Color(27, 112, 39));
