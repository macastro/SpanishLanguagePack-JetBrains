Informa sobre modificadores redundantes y sugiere eliminarlos. El código resultante será más breve, no obstante, el comportamiento y el significado seguirá siendo el mismo.

Ejemplo 1:


  // Todo el código es implícitamente strictfp en Java 17 y superior
  strictfp class X {

    // Las enumeraciones internas son implícitamente estáticas
    static enum Inner {
        A, B, C
    }

    // Los registros internos son implícitamente estáticos
    static record R() {
    }
  }

Ejemplo 2:


  final record R() {
    // Todos los registros son implícitamente finales
  }

Ejemplo 3:


  // Todas las interfaces son implícitamente abstractas
  abstract interface Printer {

    // Todos los miembros de la interfaz son implícitamente públicos
    public int size();

    // Todas las clases internas de las interfaces son implícitamente estáticas
    static class Inner {}
  }
