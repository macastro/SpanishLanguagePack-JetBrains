Informa de llamadas a las funciones to que se pueden reemplazar con la forma infija.

Usar la forma infija hace que tu código sea más simple.

El arreglo rápido reemplaza to con la forma infija.

Ejemplo:


  fun foo(a: Int, b: Int) {
      val pair = a.to(b)
  }

Después de aplicar el arreglo rápido:


  fun foo(a: Int, b: Int) {
      val pair = a to b
  }
