Se informa de una referencia no válida dentro de la anotación `org.springframework.shell.standard.ShellMethodAvailability` de Spring Shell.

Ejemplo:


@ShellComponent
public class MyCommands {

    @ShellMethod("sum")
    @ShellMethodAvailability("sumAvailability") // El valor de la anotación se resalta como un símbolo no resuelto
    public int sum(int a, int b) {
        return a + b;
    }

    @ShellMethodAvailability("sumMethod") // El valor de la anotación se resalta como un símbolo no resuelto
    public Availability isSumAvailable() {
        return Availability.available();
    }
}
