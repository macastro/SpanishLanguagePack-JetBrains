Reporta expresiones de máscara de bits que están garantizadas para evaluarse como true o false.

La inspección verifica las expresiones de la forma (var & constant1) == constant2 o (var | constant1) == constant2, donde constant1 y constant2 son constantes de máscara de bits incompatibles.

Ejemplo:

  // Máscara incompatible: como la máscara termina en 00,
  // el resultado podría ser 0x1200 pero no 0x1234
  if ((máscara & 0xFF00) == 0x1234) {...}
