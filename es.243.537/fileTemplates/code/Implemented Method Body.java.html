Rellena el cuerpo del método cuando se genera un método implementado (por ejemplo, cuando se aplica la acción de intención Implementar clase abstracta o Implementar método abstracto, o cuando se llama a la función Código | Implementar métodos...).
Esta plantilla incorporada es editable. Además de las expresiones y comentarios de Java, puede utilizar variables predefinidas que se expandirán a los valores correspondientes.
Las variables predefinidas reciben los siguientes valores:
${RETURN_TYPE}   Tipo de retorno del método que se crea.
${DEFAULT_RETURN_VALUE}   Valor devuelto por defecto del método.
${METHOD_NAME}   Nombre del método implementado.
${CLASS_NAME}   Nombre completo (cualificado) de la clase que implementa el método.
${SIMPLE_CLASS_NAME}   Nombre simple (no cualificado) de la clase que implementa el método.
Todas las variables predefinidas en la plantilla Encabezado de archivo (pestaña Incluye) también están disponibles (por ejemplo, ${USER}, ${DATE}, etc.).
