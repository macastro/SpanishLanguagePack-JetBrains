Rellena el cuerpo del bloque catch cuando se genera, por ejemplo, cuando se aplica la función Código | Rodear con....
Esta plantilla incorporada es editable. Además de expresiones y comentarios Java, puede usar variables predefinidas que se expandirán a los valores correspondientes.
Las variables predefinidas reciben los siguientes valores:
${EXCEPTION}   Nombre de la variable Exception especificada como parámetro del catch.
${EXCEPTION_TYPE}   Tipo del parámetro del catch.
