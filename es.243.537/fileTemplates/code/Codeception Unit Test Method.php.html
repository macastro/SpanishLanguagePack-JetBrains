Una plantilla integrada para los métodos de prueba unitaria de Codeception, utilizada, por ejemplo, cuando se selecciona la casilla de verificación para el método generado en el diálogo "Nueva prueba".
Las variables predefinidas tomarán los siguientes valores:
${NAME}   Nombre del método.
${CAPITALIZED_NAME}   Nombre del método con el primer carácter en mayúscula.
${TESTED_NAME}   Nombre de la clase probada.
