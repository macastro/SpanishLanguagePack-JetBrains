Crea el cuerpo de un método -description sobreescrito cuando se genera, por ejemplo, al llamar a la acción Código | Generar | -description.
La plantilla es editable. Junto con las expresiones y comentarios de Objective-C, puede usar variables predefinidas que se expandirán a los valores correspondientes.
Las variables predefinidas tomarán los siguientes valores:
${CALL_SUPER}   Una llamada al método superior, para métodos void — [super nombreMétodo] con o sin parámetros; para otros métodos — return [super nombreMétodo] con o sin parámetros
${RETURN_TYPE}   El tipo de retorno del método creado
${DEFAULT_RETURN_VALUE}   El valor retornado por el método por defecto
${CONTAINING_CLASS}   El nombre de la clase donde se sobreescribe el método
${CUSTOM_CODE}   El cuerpo del método generado
Todas las variables predefinidas de la plantilla Encabezado de Archivo C (la pestaña Incluye) también están disponibles (${USER}, ${DATE}, etc.)
