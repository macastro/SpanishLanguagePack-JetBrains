Esta plantilla integrada se utiliza para completar el cuerpo de un método sobrescrito cada vez que se genera, por ejemplo, cuando se llama a la función Código | Sobrescribir método....
La plantilla es editable. Además de expresiones y comentarios JavaScript, también puede usar variables predefinidas. Estas variables se reemplazarán con los valores correspondientes cuando se aplique la plantilla.
Las variables predefinidas tomarán los siguientes valores:
${CALL_SUPER}   Una llamada al método superior, para métodos void - super.nombre_del_método(); con o sin parámetros; para otros métodos - return super.nombre_del_método(); con o sin parámetros.
${PLAIN_CALL_SUPER}   Una llamada al método superior, super.nombre_del_método(); con o sin parámetros.
${RETURN_TYPE}   Tipo de retorno del método creado.
${DEFAULT_RETURN_VALUE}   Valor devuelto por el método por defecto.
${METHOD_NAME}   Nombre del método que se sobrescribe.
Todas las variables predefinidas de la plantilla Encabezado de archivo (pestaña Incluye) también están disponibles (ej. ${USER}, ${DATE}, etc.).
