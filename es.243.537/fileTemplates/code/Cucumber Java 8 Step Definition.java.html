Crea definiciones de pasos en Cucumber para archivos Java.
Las variables predefinidas toman los siguientes valores:
${STEP_KEYWORD}   Anotación de la definición del paso. Generalmente, se usa una palabra clave para invocar un paso (Given, Then, And, etc.).
${STEP_REGEXP}   Expresión regular generada para aceptar el paso.
${PARAMETERS}   Parámetros del método generado.
${BODY}   Cuerpo del método generado.
