Esta es la plantilla incorporada que se utiliza cada vez que se crea una nueva clase de módulo Guice.
Si utiliza la directiva #parse, podrá incluir plantillas que se encuentran en la pestaña Incluye especificando el nombre completo de la plantilla deseada como un parámetro entre comillas. Ejemplo:
#parse("File Header.java")
Las variables predefinidas toman los siguientes valores:
${PACKAGE_NAME} El nombre del paquete en el que se crea la nueva clase
${NAME} El nombre de la nueva clase que el usuario especifica en el diálogo Crear nueva clase
${USER} El nombre del inicio de sesión del sistema del usuario actual
${DATE} La fecha actual del sistema
${TIME} La hora actual del sistema
${YEAR} El año actual
${MONTH} El mes actual
${MONTH_NAME_SHORT} Las primeras 3 letras del nombre del mes actual. Por ejemplo: Ene, Feb, etc.
${MONTH_NAME_FULL} El nombre completo del mes actual. Por ejemplo: Enero, Febrero, etc.
${DAY} El día del mes actual
${HOUR} La hora actual
${MINUTE} El minuto actual
${PROJECT_NAME} El nombre del proyecto actual
