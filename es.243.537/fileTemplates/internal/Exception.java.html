Esta plantilla incorporada es editable. Además de expresiones y comentarios de Java, también puede utilizar variables predefinidas (como se enumera a continuación), que luego se expandirán como una macro a los valores correspondientes.

También puede especificar variables personalizadas. Las variables personalizadas utilizan el siguiente formato: ${NOMBRE_VARIABLE}, donde NOMBRE_VARIABLE es el nombre de la variable (por ejemplo, ${MY_CUSTOM_FUNCTION_NAME}). Antes de que el IDE cree un nuevo archivo con variables personalizadas, verá un cuadro de diálogo en el que puede definir los valores de las variables personalizadas en la plantilla.

Mediante el uso de la instrucción #parse, se pueden incluir plantillas en la ficha Incluir. Para incluir una plantilla, especifique el nombre completo de la plantilla como parámetro entre comillas (por ejemplo, #parse("File Header.java")).
Las variables predefinidas reciben los siguientes valores:
${PACKAGE_NAME} El nombre del paquete en el que se crea la nueva clase de excepción
${NAME} El nombre de la nueva excepción que especifique en el cuadro de diálogo Crear nueva clase
${USER} El nombre de inicio de sesión del sistema del usuario actual
${DATE} El nombre de inicio de sesión del sistema del usuario actual
${TIME} La hora actual del sistema
${YEAR} El año actual
${MONTH} El mes actual
${MONTH_NAME_SHORT} Las 3 primeras letras del nombre del mes actual (por ejemplo, Jan, Feb)
${MONTH_NAME_FULL} El nombre completo del mes actual (por ejemplo, January, February)
${DAY} El día del mes actual
${HOUR} La hora actual
${MINUTE} El minuto actual
${PROJECT_NAME} El nombre del proyecto actual
