Se aplica a las nuevas clases Java que se crean invocando Nuevo | Clase Java | Clase en la ventana de herramientas Proyecto.
Esta plantilla incorporada es editable. Junto con las expresiones y comentarios de Java puedes también usar las variables predefinidas (listadas abajo) que luego se expandirán como macros en los valores correspondientes.

También es posible especificar variables personalizadas. Las variables personalizadas usan el siguiente formato: ${VARIABLE_NAME}, donde VARIABLE_NAME es un nombre para tu variable (por ejemplo, ${MY_CUSTOM_FUNCTION_NAME}). Antes de que el IDE cree un nuevo archivo con variables personalizadas, verás un cuadro de diálogo donde puedes definir los valores para las variables personalizadas en la plantilla.

Usando la directiva #parse, puedes incluir plantillas desde la pestaña Incluir. Para incluir una plantilla, especifica el nombre completo de la plantilla como un parámetro entre comillas (por ejemplo, #parse("File Header.java").
Las variables predefinidas toman los siguientes valores:
${PACKAGE_NAME} Nombre del paquete en el cual una nueva clase es creada
${NAME} Nombre de la nueva clase especificada por ti en el cuadro de diálogo Crear nueva clase
${USER} Nombre de login de sistema del usuario actual
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${MONTH_NAME_SHORT} Primeras 3 letras del nombre del mes actual (Ene, Feb, etc.)
${MONTH_NAME_FULL} Nombre completo del mes actual (Enero, Febrero, etc.)
${DAY} Día actual del mes
${HOUR} Hora actual
${MINUTE} Minuto actual
${PROJECT_NAME} Nombre del proyecto actual
