Se aplica a nuevas anotaciones Java que se crean al invocar Nuevo | Clase Java | Anotación en la ventana de herramienta Proyecto.

Esta plantilla integrada es editable. Junto con comentarios y expresiones Java, también puede utilizar las variables predefinidas (indicada a continuación) que luego se expandirán como macros en valores correspondientes.

También es posible especificar variables personalizadas. Las variables personalizadas utilizan el siguiente formato: ${NOMBRE_VARIABLE}, donde NOMBRE_VARIABLE es un nombre para su variable (por ejemplo, ${MY_CUSTOM_FUNCTION_NAME}). Antes de que el IDE cree un nuevo archivo con variables personalizadas, verá un cuadro de diálogo donde puede definir valores para variables personalizadas en la plantilla.

Mediante el uso de la directiva #parse, puede incluir plantillas de la pestaña Incluye. Para incluir una plantilla, especifique el nombre completo de la plantilla como parámetro entre comillas (por ejemplo, #parse("File Header.java").
Las variables predefinidas toman los siguientes valores:
${PACKAGE_NAME} Nombre del paquete en el que se crea una nueva anotación
${NAME} Nombre de la nueva anotación especificada por usted en el cuadro de diálogo Crear nueva clase
${USER} Nombre de inicio de sesión del sistema del usuario actual
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${MONTH_NAME_SHORT} Las primeras 3 letras del nombre del mes actual (Lun, Mar, etc.)
${MONTH_NAME_FULL} Nombre completo del mes actual (Enero, Febrero, Marzo, etc.)
${DAY} Día actual del mes
${HOUR} Hora actual
${MINUTE} Minuto actual
${PROJECT_NAME} Nombre del proyecto actual
