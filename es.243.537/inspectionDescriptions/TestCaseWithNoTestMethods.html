Inspecciona los casos de prueba que no son abstract sin métodos de prueba. Dichos casos de prueba suelen indicar código sin terminar o podrían ser un resto de refactorización que debería eliminarse.

Ejemplo:


  public class CrucialTest {
    @Before
    public void setUp() {
      System.out.println("setting up");
    }
  }

Utiliza la opción Ignorar casos de prueba que tengan superclases con métodos de prueba para ignorar casos de prueba que tengan superclases con métodos de prueba.
