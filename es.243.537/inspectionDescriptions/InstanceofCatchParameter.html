Informa de los casos en los que se utiliza una expresión instanceof para probar el tipo de un parámetro en un bloque catch.

Normalmente, es mejor probar el tipo de parámetros catch con bloques catch separados en lugar de utilizar instanceof.

Ejemplo:


  void foo(Runnable runnable) {
    try {
        runnable.run();
    } catch (Throwable throwable) {
        if (throwable instanceof NoClassDefFoundError) { // advertencia: 'instanceof' en el parámetro 'throwable' de 'catch'
            System.out.println("¡Clase no encontrada!");
        }
    }
  }
