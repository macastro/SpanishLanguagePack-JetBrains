Reporta cadenas largas de transformaciones de Reactive Streams.

Cada llamada de método operador, como map() o filter(), crea algunos objetos para esos operadores. Llamar a una cadena larga de operadores en cada suscripción, para cada elemento de la secuencia, puede causar una sobrecarga de rendimiento. Para evitarlo, combine una cadena larga de llamadas en una sola llamada de operador cuando sea posible.

Ejemplo:


Flux.just(1, 2, 3)
        .map(it -> it + 1)
        .map(it -> it + 2)
        .map(it -> it + 3)

Después de aplicar la corrección rápida:


Flux.just(1, 2, 3)
        .map(it -> it + 1 + 2 + 3)
