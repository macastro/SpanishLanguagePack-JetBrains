java.util.Optional<T>, java.util.OptionalDouble, java.util.OptionalInt, java.util.OptionalLong o com.google.common.base.Optional se utiliza como tipo de campo o parámetro.

Optional está diseñado para proporcionar un mecanismo limitado para el tipo de retorno de los métodos de la biblioteca donde es necesario expresar claramente "ningún resultado".

También presenta un problema usar campos con tipos java.util.Optional cuando se requiere que la clase sea Serializable, ya que java.util.Optional no es serializable.

Ejemplo:


  class MyClass {
    Optional<String> name; // Campo Optional

    // Parámetro Optional
    void setName(Optional<String> name) {
      this.name = name;
    }
  }
