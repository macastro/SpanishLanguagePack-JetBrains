Reporta las llamadas a hashCode() y equals() en el objeto java.net.URL.

Los métodos equals() y hashCode() de URL pueden realizar búsquedas de DNS para resolver el nombre de host. Esto puede ocasionar retrasos significativos dependiendo de la disponibilidad y velocidad de la red y de los servidores DNS. Usar java.net.URI en lugar de java.net.URL evitará las búsquedas de DNS.

Ejemplo:


  int hashCodeEquals(URL url1, URL url2) {
    return url1.hashCode() == url2.hashCode();
  }
