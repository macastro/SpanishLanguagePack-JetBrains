Informa las expresiones new FileInputStream() o new FileOutputStream() que se pueden reemplazar con llamadas Files.newInputStream() o Files.newOutputStream(), respectivamente.
Las transmisiones creadas usando métodos Files usualmente son más eficientes que aquellas creadas por los constructores de transmisión.

Ejemplo:


  InputStream is = new BufferedInputStream(new FileInputStream(file));

Después de que se aplica el arreglo rápido:


  InputStream is = new BufferedInputStream(Files.newInputStream(file.toPath()));

Esta inspección no muestra advertencia si el nivel de lenguaje es 10 o más alto, pero el arreglo rápido aún está disponible.

Esta inspección solo informa si el nivel de lenguaje del proyecto o módulo es 7 o más alto.
