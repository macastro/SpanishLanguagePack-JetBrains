Informa de las expresiones de acceso a variables de apuntares o arrays cuando el índice puede estar fuera de rango del array o del búfer asignado.

Ejemplo:


  void foo() {
      int buffer[100];

      for (int i = 0; i <= 100; i ++)
          buffer[i] = 0; // desbordamiento del búfer cuando i es igual a 100
  }
