Reportan expresiones de bits sin sentido.

Tales expresiones incluyen aplicar el operador & al valor máximo para el tipo dado, aplicar el operador or a cero y desplazar por cero. Tales expresiones pueden ser el resultado de refactorizaciones automáticas no seguidas hasta su finalización y es poco probable que hayan sido originalmente intencionadas.

Ejemplos:


  // Advertencia: la operación no tiene sentido y puede ser reemplazada sólo con `flags`
  // 0xFFFF_FFFF es el valor máximo para un entero, y ambos literales son tratados
  // como literales enteros de 32 bits.
  int bits = flags & 0xFFFF_FFFF;

  // Advertencia: la operación no tiene sentido y puede ser reemplazada sólo con `bits`
  // OR con 0 siempre devuelve el otro operando.
  int or = bits | 0x0;

  // Advertencia: la operación no tiene sentido, ya que siempre resulta en 0
  int xor = or ^ or;
