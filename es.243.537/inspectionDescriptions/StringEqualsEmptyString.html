String denunciado por llamar a equals() para comparar con una cadena vacía. En este caso, es preferible usar .isEmpty() debido a que hace más claro lo que se está comprobando.

Ejemplo:


  void checkString(String s){
    if ("".equals(s)) throw new IllegalArgumentException();
  }

Tras aplicar el arreglo rápido:


  void checkString(String s){
    if (s != null && s.isEmpty()) throw new IllegalArgumentException();
  }

"".equals(str) devuelve false si str es nulo. Por seguridad, el arreglo rápido de esta inspección inserta un control de nulo explícito cuando el argumento de equals() puede ser nulo. Use las opciones para hacer que la inspección ignore dichos casos.
