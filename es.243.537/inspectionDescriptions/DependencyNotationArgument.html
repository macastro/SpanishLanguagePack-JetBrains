Detecta notaciones incorrectas de dependencias.

Los siguientes tipos/formatos son soportados por Gradle:
  • Instancias de Dependency;
  • Valores String o CharSequence, por ejemplo 'org.gradle:gradle-core:1.0';
  • Mapas, por ejemplo [group: 'org.gradle', name: 'gradle-core', version: '1.0'];
  • FileCollections, por ejemplo files('some.jar', 'someOther.jar');
  • Proyectos, por ejemplo project(':some:project:path');
  • ClassPathNotation, por ejemplo gradleApi();
  • Listas de notaciones de dependencias, por ejemplo ['org.gradle:gradle-core:1.0'];
  • (Gradle 7.0+) Accesores a catálogos de versiones, por ejemplo libs.groovy.core.
Ver documentación de Gradle

Ejemplo:


dependencies {
  implementation(1) // reporta '1'
}
