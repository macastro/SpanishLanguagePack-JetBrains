Reporta los bucles while que iteran sobre colecciones y que pueden reemplazarse con un bucle for mejorado (sintaxis de iteraci√≥n foreach).

Ejemplo:


  Iterator it = c.iterator();
  while(it.hasNext()) {
    Object obj = it.next();
    System.out.println(obj);
  }

Se puede sustituir por:


  for (Object obj : c) {
    System.out.println(obj);
  }
