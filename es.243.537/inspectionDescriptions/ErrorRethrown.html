Reporta declaraciones try que capturan java.lang.Error o cualquiera de sus subclases y no vuelven a lanzar el error.

Las declaraciones que capturan java.lang.ThreadDeath no son reportadas.

Ejemplo:


  try {
    executeTests(request);
  }
  catch (OutOfMemoryError ex) { // advertencia: el error 'ex' no fue relanzado
    return false;
  }
