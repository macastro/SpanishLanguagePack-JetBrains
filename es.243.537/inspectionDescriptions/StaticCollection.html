Reporta campos estáticos de un tipo Collection. Si bien esto no es necesariamente un problema, las colecciones estáticas a menudo provocan fugas de memoria y, por lo tanto, algunos estándares de codificación las prohíben.

Ejemplo:


  public class Example {
    static List<String> list = new ArrayList<>();

  }

Configurar la inspección:

  • Usa la opción Ignorar colecciones o mapas débiles estáticos para ignorar los campos del tipo java.util.WeakHashMap.
