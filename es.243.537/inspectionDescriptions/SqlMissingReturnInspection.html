Informa de las funciones que no tienen sentencias RETURN.

Ejemplo (Oracle):

CREATE FUNCTION foo RETURN int AS
BEGIN
END;

La función foo debe retornar el valor entero pero el cuerpo de la función no retorna nada. Para arreglar el error, agrega una sentencia RETURN (por ejemplo, return 1;).

CREATE FUNCTION foo RETURN int AS
BEGIN
 RETURN 1;
END;
