Informa caracteres de continuación faltantes en el comando RUN.

En la forma de shell del comando RUN debes usar una '\' (barra invertida) para continuar una instrucción RUN sola en la siguiente línea. De otra forma, la compilación del Docker fallará.

Ejemplos:


  # the command below will fail
  RUN /bin/bash -c 'source $HOME/.bashrc;
      echo $HOME'

Después de aplicar la corrección rápida:


  RUN /bin/bash -c 'source $HOME/.bashrc; \
      echo $HOME'
