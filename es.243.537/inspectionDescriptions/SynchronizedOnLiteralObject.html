Reporta bloques synchronized que se bloquean en objetos inicializados con un literal.

Los literales de cadena son internados y Character, Boolean y Number pueden ser asignados desde un caché. Debido a esto, es posible que alguna otra parte del sistema, que usa un objeto inicializado con el mismo literal, esté sosteniendo una referencia al mismo objeto. Esto puede crear situaciones de interbloqueo inesperadas, si el objeto de bloqueo fue considerado privado.

Ejemplo:


  class Main {
    final String mutex = "Mutex";
    void method() {
      synchronized (mutex) {
      }
    }
  }

Usa la opción Advertir sobre todos los posibles literales para reportar cualquier sincronización en String, Character, Boolean y Number objetos.
