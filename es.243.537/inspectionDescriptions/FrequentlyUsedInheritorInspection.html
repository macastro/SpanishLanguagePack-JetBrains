Reporta clases o interfaces que pueden ser reemplazados con una implementación o extensión de una clase o interfaz más específica y comúnmente utilizada.

Para que esta inspección funcione, una superclase necesita estar en los archivos fuente del proyecto y el proyecto necesita usar el sistema de compilación de IntelliJ IDEA.

Ejemplo:


  clase MyInheritor implements A {} // Se sugiere B en la referencia A

  interfaz A {}

  clase abstracta B implements A {}

  clase abstracta C1 extiende B {}
  clase abstracta C2 extiende B {}
  clase abstracta C3 extiende B {}
  clase abstracta C4 extiende B {}
  clase abstracta C5 extiende B {}

Por defecto, esta inspección no resalta problemas en el editor, pero sólo proporciona una solución rápida.
