Informa las expresiones de desplazamiento que coinciden o exceden el ancho del número entero.

Todos los bits del operando de la izquierda se desplazan con dicha expresión, lo que resulta en un valor constante. El valor constante indica que el desplazamiento de la posición está incorrecto o bien la expresión es redundante.

Ejemplo:


  func shift(i int8) {
    fmt.Println(i << 8) // siempre imprime 0
  }
