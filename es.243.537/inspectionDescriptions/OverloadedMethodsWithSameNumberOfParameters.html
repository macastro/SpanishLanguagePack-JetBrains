Informa de métodos declarados en la misma clase, con el mismo nombre y el mismo número de parámetros. Estas sobrecargas pueden resultar muy confusas, porque puede no estar claro qué sobrecarga se invoca.

Ejemplo:


  clase Main {
    public static void execute(Runnable r) {}
    public static <T> void execute(RunnableFuture<T> c) {}
  }

Utiliza la opción para ignorar métodos sobrecargados cuyos tipos de parámetro son definitivamente incompatibles.
