Informa los usos de true | false que pueden ser reemplazados con bool. La palabra clave bool es una abreviatura que ayuda a la lectura.

Existe una corrección rápida disponible para convertir cualquier combinación de true | false con bool.

Ejemplo:

# Se puede simplificar 'true | false'
type example = true | false

Después de la corrección rápida:

type example = bool
