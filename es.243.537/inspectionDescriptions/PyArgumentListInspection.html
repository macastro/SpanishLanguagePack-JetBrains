Informa sobre discrepancias entre los parámetros declarados, los argumentos reales, y los argumentos incorrectos (por ejemplo, argumentos nombrados duplicados, orden incorrecto de los argumentos).

Ejemplo:


class Foo:
    def __call__(self, p1: int, *, p2: str = "%"):
        return p2 * p1


bar = Foo()
bar.__call__() # parámetro no ingresado
bar(5, "#") # argumento inesperado

El fragmento de código correcto se vería así:


class Foo:
    def __call__(self, p1: int, *, p2: str = "%"):
        return p2 * p1


bar = Foo()
bar.__call__(5)
bar(5, p2="#")
