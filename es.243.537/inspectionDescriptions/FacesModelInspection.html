Informará de elementos incorrectos en el archivo faces-config.xml de Java Server Faces.

  • Referencias a clases que no pueden instanciarse
  • Referencias a clases que no extienden las clases necesarias
  • Referencias a clases con un ámbito inadecuado
  • Etiquetas y valores de atributos vacíos
  • Valores de etiquetas y atributos que no coinciden con los patrones requeridos (p. ej., identificadores Java)
  • Etiquetas que no contienen las etiquetas o atributos secundarios necesarios
  • Etiquetas que definen objetos con nombres duplicados

Ejemplo:


  <faces-config >
     <application>
        <action-listener>java.lang.String</action-listener>  <!-- 'java.lang.String' no se puede asignar a 'javax.faces.event.ActionListener' -->
    </application>
    <navigation-rule>
        <from-view-id>myView.xhtml</from-view-id>
        <navigation-case>
            <to-view-id>unknown.xhtml</to-view-id>  <!-- No se puede resolver 'unknown.xhtml' -->
        </navigation-case>
    </navigation-rule>
  </faces-config>
