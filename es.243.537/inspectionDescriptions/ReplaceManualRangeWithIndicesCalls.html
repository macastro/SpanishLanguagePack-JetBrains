Reporta los operadores until y rangeTo que pueden ser reemplazados con Collection.indices o iteración sobre la colección dentro del ciclo for.

Usar azúcar sintáctica hace tu código más simple.

La corrección rápida reemplaza el rango manual con la construcción correspondiente.

Ejemplo:


  fun main(args: Array<String>) {
      for (index in 0..args.size - 1) {
          println(args[index])
      }
  }

Después de aplicar la corrección rápida:


  fun main(args: Array<String>) {
      for (element in args) {
          println(element)
      }
  }
