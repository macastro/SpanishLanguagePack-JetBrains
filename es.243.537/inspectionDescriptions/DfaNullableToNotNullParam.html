Esta inspección identifica llamadas de método donde un argumento que potencialmente podría ser null se pasa a un parámetro que está anotado con @NotNull, ayudando a detectar NullPointerExceptions potenciales antes de que ocurran en tiempo de ejecución.

Ejemplo:


def test(@Nullable saludo: String): Unit = {
  decirHola(saludo)
}

def decirHola(@NotNull saludo: String): Unit = {
  println(saludo.toLowerCase)
}
