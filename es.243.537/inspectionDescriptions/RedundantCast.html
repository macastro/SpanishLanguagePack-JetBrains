Informa de expresiones de conversión innecesarias.

Ejemplo:


  static Object toObject(String s) {
    return (Object) s;
  }

Utiliza la casilla de verificación siguiente para ignorar conversiones aclaratorias, por ejemplo, conversiones en llamadas de colección donde se espera un Object: 


  static void removeFromList(List<String> l, Object o) {
    l.remove((String)o);
  }
