Informa sobre configuraciones de contexto Spring incorrectas.

  • Archivos y directorios no resueltos en atributos locations y atributos alias correspondientes de metaanotaciones @ContextConfiguration
  • Falta archivo de contexto de aplicación predeterminado

Para más información, consulta @ContextConfiguration.

Ejemplo:


  @ContextConfiguration(locations = "classpath:META-INF/unknown-context.xml") // informa "No se puede resolver el archivo 'unknown-context.xml'"
  class MyTests {...}
