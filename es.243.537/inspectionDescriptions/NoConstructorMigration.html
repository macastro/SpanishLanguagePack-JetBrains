Informa sobre llamadas a constructores en supertipos funcionales que darán lugar a un error de compilación a partir de la versión 1.9.

Tipos de motivaciones:

  • La implementación no cumple con una especificación o documentación publicada

Más detalles: KT-46344: No se produce error en una llamada al constructor de una superclase en una interfaz de función en la lista de supertipos

La corrección rápida elimina una llamada al constructor.

Ejemplo:


  abstract class A : () -> Int()

Tras aplicar la corrección rápida:


  abstract class A : () -> Int

Esta inspección solo informa si el nivel del lenguaje Kotlin del proyecto o módulo es 1.7 o superior.
