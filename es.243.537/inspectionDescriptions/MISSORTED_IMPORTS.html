Reporta declaraciones import no ordenadas de acuerdo al estilo de código actual (ver Ajustes | Editor | Estilo de Código).

Ejemplo:


  import java.util.List;
  import java.util.ArrayList;

  public class Ejemplo {
    List lista = new ArrayList();
  }

Después de aplicar la corrección rápida "Optimizar Imports": 


  import java.util.ArrayList;
  import java.util.List;

  public class Ejemplo {
    List lista = new ArrayList();
  }
