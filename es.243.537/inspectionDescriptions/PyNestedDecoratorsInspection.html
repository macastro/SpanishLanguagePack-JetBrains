Informa de problemas con la anidación de decoradores. Esta inspección resalta los casos en que se aplica classmethod o staticmethod antes de otro decorador.

Ejemplo:


def innocent(f):
    return f


class A:
    @innocent  # El decorador no va a recibir una función
    @classmethod
    def f2(cls):
        pass

    @innocent  # El decorador no va a recibir una función
    @staticmethod
    def f1():
        pass

Como solución rápida, el IDE ofrece eliminar el decorador.
