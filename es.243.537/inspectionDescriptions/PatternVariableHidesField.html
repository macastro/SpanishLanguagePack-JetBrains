Reporta variables de patrón nombradas de manera idéntica a un campo de una clase circundante. Como resultado de dicha denominación, puede utilizar accidentalmente la variable de patrón cuando se pretende utilizar el campo con el mismo nombre.

Se sugiere una solución rápida para cambiar el nombre de la variable.

Ejemplo:


  class Pointless {
    Point p = new Point();

    public void test(Object a) {
      if (a instanceof Point p) {
        System.out.print("a is a point (" + p.x + ", " + p.y + ')');
      } else {
        System.out.print("p is a point (" + p.x + ", " + p.y + ')');
      }
    }
  }
