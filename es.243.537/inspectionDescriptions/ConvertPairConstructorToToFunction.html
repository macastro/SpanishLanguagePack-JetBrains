Informa de una invocación de constructor Pair que se puede reemplazar por una llamada a función infija to().

Las invocaciones explícitas de constructores pueden añadir verbosidad, especialmente si se usan múltiples veces. Reemplazar llamadas de constructor con to() hace que el código sea más fácil de leer y mantener.

Ejemplo:

 
  val countries = mapOf( 
      Pair("France", "Paris"), 
      Pair("Spain", "Madrid"), 
      Pair("Germany", "Berlin") 
  )

Tras aplicar la corrección rápida:

 
  val countries = mapOf( 
      "France" to "Paris", 
      "Spain" to "Madrid", 
      "Germany" to "Berlin" 
  )
