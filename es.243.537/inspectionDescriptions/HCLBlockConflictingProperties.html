Informa sobre propiedades conflictivas dentro de un bloque HCL.

Ejemplo del problema:


provider "kubernetes" {
  version = "~> 1.25"
  config_path = "~/.kube/config"
  version = "~> 1.24"
}

En este ejemplo, la propiedad version se declara dos veces dentro del bloque del proveedor kubernetes, lo que es incorrecto.

Ejemplo de correcci√≥n:


provider "kubernetes" {
  version = "~> 1.25"
  config_path = "~/.kube/config"
}
