Inspecciona vérificaciones de tipo is para un objeto que no son idiomaticas.

Se recomienda reemplazar esas verificaciones con una comparación de referencias.

Ejemplo:


  object Foo

  fun foo(arg: Any) = when {
      arg is Foo -> ...
      arg !is Foo -> ...
  }

Después de aplicar la solución rápida:


  object Foo

  fun foo(arg: Any) = when {
      arg === Foo -> ...
      arg !== Foo -> ...
  }
