Informa variables locales con parámetros de tipo.

Un parámetro de tipo para una variable local no tiene sentido porque no se puede especializar.

Ejemplo:


  fun main() {
      val <T> x = ""
  }

Después de aplicar la revisión rápida:


  fun main() {
      val x = ""
  }

Esta inspección solo informa si el nivel de lenguaje Kotlin del proyecto o módulo es 1.4 o superior.
