Informa los parámetros catch que no se utilizan en sus bloques correspondientes. Esta inspección no informará ningún parámetro catch denominado "ignore" o "ignored".

Ejemplo:


  try {
      def arr = new int[3]
      arr[5] = 5
  } catch(Exception ex) {
      println('Captura de excepción')
  }

Aquí el parámetro ex nunca se usa en el bloque catch.

Después de aplicar la solución rápida:


  try {
      def arr = new int[3]
      arr[5] = 5
  } catch(Exception ignored) {
      println('Captura de excepción')
  }
