Reporta ciclos for que no contienen ni componentes de inicialización ni de actualización, y sugiere convertirlos en ciclos while. Esto hace que el código sea más fácil de leer.

Ejemplo:


  for(; exitCondition(); ) {
    process();
  }

Después de aplicar la solución rápida:


  while(exitCondition()) {
    process();
  }

La solución rápida también está disponible para otros ciclos for, por lo que puede reemplazar cualquier ciclo for con un ciclo while.

Utilice la opción Ignorar ciclos 'infinite' for sin condiciones si desea ignorar ciclos for con condiciones triviales o inexistentes.
