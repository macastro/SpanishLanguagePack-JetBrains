Se reportan declaraciones de comandos Spring Shell no válidas:

  • Falta la descripción del comando
  • El comando no está dentro de org.springframework.shell.standard.@ShellComponent

Ejemplo:


// Se reportarán todos los métodos porque falta @ShellComponent
class MathCommands {

  @ShellMethod("")
  public int sum(int a, int b) { // Se reportará el método porque la descripción está vacía
    return a + b;
  }

  @ShellMethod
  public int mul(int a, int b) { // Se reportará el método porque falta la descripción
    return a * b;
  }
}
