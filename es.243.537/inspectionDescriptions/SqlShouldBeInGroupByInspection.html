Informa de las columnas que no están en la cláusula GROUP BY o dentro de una llamada a una función de agregación.

Ejemplo (Microsoft SQL Server):

CREATE TABLE t1 (a INT, b INT);
SELECT a, b FROM t1 GROUP BY a;

Si ejecuta la consulta SELECT, recibirá un error porque Microsoft SQL Server espera que la columna b esté en GROUP BY o utilizada dentro de una función de agregación. Los dos ejemplos siguientes corrigen el error.

SELECT a, b FROM t1 GROUP BY a, b;
SELECT a, max(b) max_b FROM t1 GROUP BY a;
