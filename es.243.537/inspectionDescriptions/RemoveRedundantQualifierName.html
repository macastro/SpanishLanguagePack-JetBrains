Informa sobre calificadores redundantes (o sus partes) en nombres de clase, funciones y propiedades.

Un nombre completamente calificado es un identificador inequívoco que especifica a qué objeto, función o propiedad hace referencia una llamada. En los contextos donde el nombre puede acortarse, la inspección informa sobre la oportunidad y la solución rápida asociada 'Quitar nombre de calificador redundante' permite modificar el código.

Ejemplos:


  package my.simple.name
  import kotlin.Int.Companion.MAX_VALUE

  class Foo

  fun main() {
      val a = my.simple.name.Foo()    // 'Foo' reside en el paquete declarado 'my.simple.name', el calificador es redundante
      val b = kotlin.Int.MAX_VALUE    // Puede reemplazarse con 'MAX_VALUE' ya que está importado
      val c = kotlin.Double.MAX_VALUE // Puede reemplazarse con 'Double.MAX_VALUE' ya que los tipos incorporados se importan automáticamente
  }

Después de que se aplique la corrección rápida:


  package my.simple.name
  import kotlin.Int.Companion.MAX_VALUE

  class Foo

  fun main() {
      val a = Foo()
      val b = MAX_VALUE
      val c = Double.MAX_VALUE
  }
