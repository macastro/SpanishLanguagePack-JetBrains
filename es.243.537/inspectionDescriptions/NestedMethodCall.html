Reporta una llamada a un método que se utiliza como parámetro de otra llamada a un método.

Esta corrección rápida introduce una variable, simplificando el código y facilitando la depuración.

Ejemplo:


  public int y() { return 1; }
  public int f(int x) { return 2 * x; }

  public void foo() {
      int x = f(y());
  }

Después de aplicar la corrección rápida:


  public int y() { return 1; }
  public int f(int x) { return 2 * x; }

  public void foo() {
      int y = y();
      int x = f(y);
  }

Utiliza las opciones de inspección para cambiar los informes de:

  • Llamadas a métodos en inicializadores de campo.
  • Llamadas a métodos estáticos.
  • Llamadas a getters simples.
