Informa sobre las variables locales cuyos nombres son demasiado cortos, demasiado largos o no coinciden con el patrón de expresión regular especificado.

Utiliza el campo Patrón para especificar una expresión regular para validar los nombres de variables locales.

Utiliza los campos Longitud mínima y Longitud máxima para especificar la longitud mínima y máxima de un nombre en consecuencia.
