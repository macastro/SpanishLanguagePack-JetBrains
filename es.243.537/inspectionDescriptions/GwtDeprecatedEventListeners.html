Reporta el código que utiliza interfaces de escucha de eventos deprecated en GWT 1.6.

Las interfaces de escucha del paquete com.google.gwt.user.client.ui (por ejemplo: com.google.gwt.user.client.ui.ClickListener) están desaprobadas desde la versión 1.6 de GWT.

La corrección rápida migra el código para utilizar las interfaces *Handler correspondientes.
