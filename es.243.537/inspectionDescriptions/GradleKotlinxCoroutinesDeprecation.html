Informa de las dependencias de la biblioteca kotlinx.coroutines en Gradle que se deben actualizar para que sean compatibles con Kotlin 1.3+.

Ejemplo:


  dependencies {
      implementation 'org.jetbrains.kotlinx:kotlinx-coroutines-core:0.0.1'
  }

La corrección rápida cambia la versión de la biblioteca kotlinx.coroutines a una compatible con Kotlin 1.3:


  dependencies {
      implementation 'org.jetbrains.kotlinx:kotlinx-coroutines-core:0.27.0-eap13'
  }
