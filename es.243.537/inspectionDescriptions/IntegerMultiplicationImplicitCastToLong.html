Informes de multiplicaciones y desplazamientos a la izquierda enteros que se convierten implícitamente a tipo long.

Ejemplo:


  void f(int i) {
    long val = 65536 * i;
  }

Después de que se aplica una corrección rápida, el código cambia a:


  void x(int i) {
    long val = 65536L * i;
  }

Ejemplo:


  void f(int i) {
    long value = i << 24;
  }

Después de que se aplica una corrección rápida, el código cambia a:


  void f(int i) {
    long value = (long) i << 24;
  }

Tales multiplicaciones suelen ser errores, ya que pueden provocar un truncamiento de desbordamiento inesperado. El problema se puede corregir convirtiendo el literal int en un literal long (65536L).
