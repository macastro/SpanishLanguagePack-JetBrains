Se están reportando intentos de verificar reflexivamente la presencia de anotaciones que no están en tiempo de ejecución.

Usar Class.isAnnotationPresent() para probar anotaciones que tienen su política de retención establecida como SOURCE o CLASS (valor predeterminado) siempre resultará en un resultado negativo. Este error es fácil de pasar por alto.

Ejemplo:


  {
    getClass().isAnnotationPresent(SourceAnnotation.class); //Siempre devuelve falso
  }

  @Retention(RetentionPolicy.SOURCE)
  @interface SourceAnnotation {}
