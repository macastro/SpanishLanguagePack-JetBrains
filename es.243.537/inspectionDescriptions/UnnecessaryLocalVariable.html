Se informan las siguientes variables locales innecesarias que no contribuyen a la comprensión del método:
  • Variables locales que se devuelven inmediatamente y.
  • Variables locales que se asignan inmediatamente a otra variable y que después no se usan.
  • Variables locales que siempre tienen el mismo valor que otra variable local o parámetro.

Ejemplo:


  boolean yes() {
    boolean b = true;
    return b;
  }

Después de aplicar la corrección rápida:


  boolean yes() {
    return true;
  }

Configuración de la inspección:

  • Utiliza la opción "Ignorar variables que se devuelven o se procesan inmediatamente" para ignorar las variables que se devuelven o se procesan inmediatamente. Algunos estilos de código recomiendan utilizar este tipo de variables para mayor claridad y facilidad de depuración.
  • Utiliza la opción "Ignorar variables con anotaciones" para ignorar las variables que tienen anotaciones.
