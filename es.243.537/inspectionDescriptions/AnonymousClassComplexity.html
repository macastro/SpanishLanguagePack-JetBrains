Informa de clases internas anónimas cuya complejidad total excede el máximo especificado.

La complejidad total de una clase es la suma de las complejidades ciclomáticas de todos los métodos e inicializadores que declara la clase. Los métodos e inicializadores heredados no se cuentan para la complejidad total.

Las clases anónimas deberían tener una complejidad muy baja, de lo contrario serán difíciles de entender y deberían promoverse para que se conviertan en clases internas con nombre.

Utiliza el campo Límite de complejidad ciclomática para especificar la complejidad máxima permitida para una clase.
