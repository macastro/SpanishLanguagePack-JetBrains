Inspecciona constructores que no comienzan invocando al constructor "super" u otro constructor dentro de la misma clase.

Se puede considerar que estos constructores comienzan implícitamente con una invocación a super(). Algunos estándares de codificación prefieren que se especifique explícitamente esta invocación a super().

Ejemplo:


  class Foo {
    Foo() {}
  }

Después de aplicar la corrección rápida:


  class Foo {
    Foo() {
      super();
    }
  }

Utiliza las opciones de la inspección para ignorar las clases que extienden directamente de Object. Por ejemplo: 


  class Foo {
    Foo() {} // No se informa
  }

  class Bar extends Foo {
    Bar() {} // Invocación implícita a 'super()'
  }
