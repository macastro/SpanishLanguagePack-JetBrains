Detecta expresiones condicionales que tienen ramas then y else muy similares que se pueden mover hacia dentro. Esta acción reduce el código.

Ejemplo:


  double g(int a, int b) {
    return a == b ? Math.cos(0) : Math.cos(1);
  }

Después de aplicar la corrección rápida:


  double g(int a, int b) {
    return Math.cos(a == b ? 0 : 1);
  }
