Informa de dos cadenas de llamada que pueden reemplazarse por una sola llamada.

Puede ayudar a evitar la ejecución redundante de código.

La corrección rápida reemplaza la cadena de llamada por una sola llamada.

Ejemplo:


  fun main() {
      listOf(1, 2, 3).filter { it > 1 }.count()
  }

Después de aplicar la corrección rápida:


  fun main() {
      listOf(1, 2, 3).count { it > 1 }
  }
