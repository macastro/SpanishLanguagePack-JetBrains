Reporta un lazo con una sentencia if que puede terminar con break sin cambiar la semántica. De esta manera previene una iteración redundante del lazo.

Ejemplo:


  boolean found = false;
  for (int i = 0; i < arr.length; i++) {
    if (Objects.equals(value, arr[i])) {
      found = true;
    }
  }

Después de aplicar el arreglo rápido:


  boolean found = false;
  for (int i = 0; i < arr.length; i++) {
    if (Objects.equals(value, arr[i])) {
      found = true;
      break;
    }
  }
