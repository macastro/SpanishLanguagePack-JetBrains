Informa del modificador inner en una clase como redundante si no hace referencia a miembros de su clase externa.

Ejemplo:


  class Foo {
      inner class InnerClass { // modificador `inner` redundante
          fun hello() {
              println("¡Hola!")
          }
      }
  }

  class List {
      val objects = Array<Any>(42) { Any() }

      inner class Iterator { // modificador `inner` no redundante
          fun next(): Any {
              return objects[0]
          }
      }
  }

Tras aplicar la corrección rápida:


  class Foo {
      class InnerClass { // modificador `inner` redundante
          fun hello() {
              println("¡Hola!")
          }
      }
  }

  class List {
      val objects = Array<Any>(42) { Any() }

      inner class Iterator { // modificador `inner` no redundante
          fun next(): Any {
              return objects[0]
          }
      }
  }
