Informa los módulos usados en cláusulas de superclase. Las cláusulas de superclase sólo están permitidas para referenciar clases.

Hay una solución rápida disponible para eliminar la cláusula de superclase inválida.

Ejemplo:

# Módulo 'Kernel' usado como una superclase
class Example < Kernel
end

Después de la solución rápida:

class Example
end
