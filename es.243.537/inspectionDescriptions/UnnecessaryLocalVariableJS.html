Informa una variable local innecesaria que no hace que una función sea más comprensible:
  • una variable local que se devuelve inmediatamente
  • una variable local que se asigna inmediatamente a otra variable y ya no se utiliza
  • una variable local que siempre tiene el mismo valor que otra variable local o parámetro

Utiliza la casilla de verificación que aparece a continuación para hacer que esta inspección ignore las variables que son devueltas o lanzadas de inmediato. Algunos estilos de codificación sugieren el uso de dichas variables para mayor claridad y facilidad de depuración.
