El informe compara directamente los errores, por ejemplo, con ==, y sugiere usar errors.Is en su lugar.

Desde Go 1.13, los errores se pueden envolver utilizando la función fmt.Errorf con el verbo %w. Por lo tanto, la comparación directa de errores utilizando la comparación de igualdad falla en los errores envueltos. La forma preferida de verificar un error específico es utilizar la función errors.Is de la biblioteca estándar, ya que esta función recorre la cadena de errores envueltos mientras busca un error específico.

Para obtener más información sobre errors.Is, consulte Cómo trabajar con errores en Go 1.13.

La solución rápida reemplaza la comparación directa de errores por una llamada a errors.Is.

Ejemplo:


if err == MyError {
  fmt.Println("MyErr")
}

Después de aplicar la solución rápida.


if errors.Is(err, MyError) {
  fmt.Println("MyErr")
}
