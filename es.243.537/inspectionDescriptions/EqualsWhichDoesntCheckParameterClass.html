Reporta los métodos equals() que no comprueban el tipo de su parámetro.

No poder comprobar el tipo del parámetro en el método equals() puede tener como resultado errores latentes si el objeto se usa en una colección no tipada.

Ejemplo:


  class MyClass {
    int x;

    @Override
    public boolean equals(Object obj) {
      // el método equals debe devolver false si obj no es MyClass
      return ((MyClass)obj).x == x;
    }
  }
