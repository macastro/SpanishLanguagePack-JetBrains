Informa los tipos de conversiones que se pueden eliminar si el tipo de variable se reduce al tipo de conversión.

Ejemplo:


  Object x = "  string  ";
  System.out.println(((String)x).trim());

Aquí, cambiar el tipo de x a String hace que la conversión sea redundante. La solución rápida sugerida actualiza el tipo de variable y elimina todas las conversiones redundantes en esa variable: 


  String x = "  string  ";
  System.out.println(x.trim());
