Reporta las propiedades Range.start y Range.endInclusive **empaquetadas**.

Estas propiedades se pueden reemplazar por las propiedades empaquetadas first y last para evitar llamadas redundantes.

El arreglo rápido reemplaza las propiedades start y endInclusive por las propiedades correspondientes first y last.

Ejemplo:


  fun foo(range: CharRange) {
      val lastElement = range.endInclusive
  }

Después de aplicar el arreglo rápido:


  fun foo(range: CharRange) {
      val lastElement = range.last
  }
