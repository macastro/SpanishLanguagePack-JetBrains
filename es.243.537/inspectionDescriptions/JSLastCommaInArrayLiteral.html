Informa un uso de coma final en un literal de arreglo.

La advertencia solo se informa cuando la versión del lenguaje JavaScript está establecida en ECMAScript 5.1.

Aunque las comas finales en arreglos están permitidas por la especificación, algunos navegadores pueden lanzar un error cuando se utiliza una coma final.

Puede configurar las opciones de formato para las comas finales en Estilo de código | JavaScript o TypeScript | Puntuación.
