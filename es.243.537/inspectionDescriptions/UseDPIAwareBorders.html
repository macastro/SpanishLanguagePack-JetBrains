Reporta instancias simplificables de javax.swing.border.EmptyBorder y JBUI.Borders.emptyXyz().

Las instancias de EmptyBorder no son sensibles a DPI y pueden causar problemas con el diseño de IU.

La corrección rápida reemplazará con JBUI.Borders.empty() o simplificará la expresión.

Ejemplo:


// Incorrecto:
Border border1 = new EmptyBorder(1, 2, 3, 4);
Border border2 = new EmptyBorder(1, 2, 1, 2);
Border border3 = new EmptyBorder(1, 0, 0, 0);

// Correcto:
Border border1 = JBUI.Borders.empty(1, 2, 3, 4);
Border border2 = JBUI.Borders.empty(1, 2);
Border border3 = JBUI.Borders.emptyTop(1);
