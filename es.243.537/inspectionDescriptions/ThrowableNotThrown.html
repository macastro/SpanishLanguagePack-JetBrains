Reporta instanciaciones de Throwable o sus subclases, donde el Throwable creado nunca es realmente arrojado. Adicionalmente, esta inspección reporta llamadas de método que retornan instancias de Throwable o sus subclases, cuando el resultado de la llamada de método no es arrojado.

Llamadas a métodos anotados con la anotación de Error Prone o AssertJ @CanIgnoreReturnValue no será reportada.

Ejemplo:


    void check(String s) {
        if (s == null) {
            new NullPointerException("s");
        }
        // ...
    }
