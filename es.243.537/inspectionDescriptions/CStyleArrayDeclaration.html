Informa de las declaraciones de array escritas en sintaxis de estilo C, en las que los corchetes de array están situados después del nombre de la variable o de la lista de parámetros del método. Muchos estilos de código prefieren las declaraciones de array en estilo Java, en los que los corchetes de array se colocan después del nombre del tipo, como se ve a continuación:

Ejemplo:


  public String process(String value[])[] {
    return value;
  }

Después de aplicar la corrección rápida:


  public String[] process(String[] value) {
    return value;
  }

Configuración de la inspección:

Utiliza la opción Omitir declaraciones de estilo C en variables para informar solo de las declaraciones de array de estilo C que se declaran en el tipo de valor devuelto del método.
