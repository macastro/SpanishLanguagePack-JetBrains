Informa sobre las definiciones con una anotación @targetName que anulan definiciones sin la anotación @targetName.

Consulta la anotación @targetName.

La solución rápida elimina la anotación @targetName.

Ejemplo:


  class A:
    def foo: Int = 1

  class B extends A:
    @targetName("overriddenFoo")
    override def foo: Int = 2

Después de aplicar la solución rápida:


  class A:
    def foo: Int = 1

  class B extends A:
    override def foo: Int = 2
