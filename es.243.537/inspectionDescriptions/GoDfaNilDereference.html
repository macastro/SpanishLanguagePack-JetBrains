Informa instancias donde punteros que pueden ser nil son desreferenciados.

Ejemplo:


type Person struct{ Name string }

func _(person *Person) {
	if person != nil || person.Name != "Peter" {
	}
}

En el ejemplo proveído, person es un puntero a una estructura Person, pero es nil ya que no se le ha asignado un valor. Intentar desreferenciar a person para acceder a Name sin verificar si person es nil resultará en un error de ejecución.
