Informa de los casos en que se comparan colecciones de Scala de diferentes categorías.

Las colecciones de Scala de diferentes categorías (Set, Seq, Map o Iterator) no pueden ser iguales y tampoco deben ser comparadas con arrays. Esta inspección sugiere convertir una de las colecciones a la misma categoría.

Ejemplo:


  seq == iterator
  seq != array

Después de aplicar la corrección rápida:


  seq.toIterator == iterator or seq == iterator.toSeq
  seq != array.toSeq
