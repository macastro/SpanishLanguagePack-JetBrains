Informa sobre las definiciones de método con el estilo incorrecto de paréntesis. Se esperan paréntesis en las definiciones de método con parámetros, lo opuesto a las definiciones de método sin parámetros.

Ejemplo:

# Mala práctica
def example_method(); end
def example_method_with_parameters param1, param2; end
Hay soluciones rápidas disponibles para eliminar los paréntesis o agregarlos respectivamente. Después de aplicar las soluciones rápidas, el resultado se ve así: 
# Buena práctica
def example_method; end
def example_method_with_parameters(param1, param2); end
