Informa las cadenas de condiciones en las que se verifica un rango de valores y estas cadenas de condiciones pueden simplificarse a una sola verificación.

El arreglo rápido reemplaza una cadena de condición con una expresión simplificada:

Ejemplo:


  x > 2 && x < 4

Después de que se aplica el arreglo rápido:


  x == 3

Ejemplo:


  arr.length == 0 || arr.length > 1

Después de que se aplica el arreglo rápido:


  arr.length != 1
