Informa los campos o métodos que no son `public` y `/o no estáticos` y que usan la anotación `@DataPoint`. Las clases de prueba de Teorías con miembros `@DataPoint` mal formados no se pueden ejecutar.

Ejemplo:


  @RunWith(Theories.class)
  public class SeriousTest {
    @DataPoint
    private String dataPoint = "value";

    @DataPoint("generated")
    private String generatedDataPoint() {
      return "generated value";
    }

    @Theory
    public void theoryMethod(String param) {
      // ...
    }
  }

Hay una corrección rápida para modificar los modificadores:


  @RunWith(Theories.class)
  public class SeriousTest {
    @DataPoint
    public static String dataPoint = "value";

    @DataPoint("generated")
    public static String generatedDataPoint() {
      return "generated value";
    }

    @Theory
    public void theoryMethod(String param) {
      // ...
    }
  }
