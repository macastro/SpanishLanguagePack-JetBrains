Informa de métodos sin cláusulas de parámetros que anulan métodos con cláusulas de parámetros vacías.

La convención es que un método debe tener paréntesis si tiene efectos secundarios. De acuerdo con el principio de sustitución de Liskov, cuando un método anulado tiene una cláusula de parámetro, que indica un efecto secundario, el método anulado también debe declararse como un método con efectos secundarios.

La corrección rápida añade una cláusula de parámetro vacía.

Ejemplo:


  class Base {
    def x() = 1
  }
  class Impl extends Base {
    override def x = 2
  }

Después de aplicar las correcciones rápidas:


  class Base {
    def x() = 1
  }
  class Impl extends Base {
    override def x() = 2
  }
