Informa de métodos getter estándares que pueden remplazarse por la anotación de Lombok @Getter.

Ejemplo:


  import java.util.Date;

  public class MyClass {
    /**
     * La fecha del proceso.
     */
    private Date processDate;

    /**
     * Devuelve la fecha.
     *
     * @return La fecha
     */
    public Date getProcessDate() {
      return processDate;
    }
  }

Después de aplicar la corrección/limpieza rápida:


  import lombok.Getter;
  import java.util.Date;

  @Getter
  public class MyClass {
    /**
     * La fecha del proceso.
     * -- GETTER --
     * Devuelve la fecha.
     *
     * @return La fecha
     */
    private Date processDate;
  }

Solo informa cuando la biblioteca Lombok está configurada. Para desconfigurar Lombok, consulta el sitio web de Lombok.
