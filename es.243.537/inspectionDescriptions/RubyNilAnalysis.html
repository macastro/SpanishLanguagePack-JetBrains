Informa posibles desreferencias de nil. La desreferenciación de nil puede provocar un NoMethodError.

Utiliza la lista de receivers ignorados para especificar las llamadas al método utilizadas como receivers que deben ignorarse (por ejemplo, Foo#call o Foo::call para un método de instancia o un método singleton respectivamente).

Ejemplo:

def foo
  rand(2) == 1 ? nil : 1
end

a = foo
a.nomethod # la invocación del método puede producir un NoMethodError
