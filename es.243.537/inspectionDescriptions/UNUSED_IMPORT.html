Informa sobre las instrucciones redundantes de import.

Las instrucciones regulares de import no son necesarias cuando no se usan clases y paquetes importados en el archivo fuente. Lo mismo se aplica a los campos y métodos static importados que no se usan en el archivo fuente.

Ejemplo:


  import java.util.ArrayList;
  public class Example {
    public static void main(String[] args) {
      System.out.println("Hello World!");
    }
  }

Después de aplicar la corrección rápida: 


  public class Example {
    public static void main(String[] args) {
      System.out.println("Hello World!");
    }
  }
