Reporta paréntesis vacíos redundantes en llamadas a funciones cuyo único parámetro es un lambda que se encuentra fuera de los paréntesis.

Usa la corrección rápida "Quitar paréntesis innecesarios de llamadas a funciones con lambda" para limpiar el código.

Ejemplos:


  fun foo() {
      listOf(1).forEach() {  }
  }

Después de aplicar la corrección rápida:


  fun foo() {
      listOf(1).forEach {  }
  }
