Informa de instancias en las que las variables pueden tener nil o un valor inesperado debido a que el error asociado no se comprueba para que no sea nil, como en v, err := foo().

El arreglo rápido elimina la función del alcance de esta inspección.

Ejemplo:


func _() {
    file, err := os.Open("file.txt")
    // Aquí se omite la comprobación de errores
    name := file.Name()
    print(name, err)
}

En el ejemplo proporcionado, file podría ser nil o tener un valor inesperado si err no es nil.
