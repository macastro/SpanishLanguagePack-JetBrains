Informa las definiciones sin una anotación @targetName las cuales sobrescriben definiciones con anotación @targetName.

Ver La anotación @targetName.

La corrección rápida agrega una anotación @targetName con el nombre externo de la definición sobreescrita.

Ejemplo:


  trait A:
    @targetName("multiply")
    def *(i: Int): A

  class B extends A:
    override def *(i: Int): A = ???

Después de aplicar la corrección rápida:


  trait A:
    @targetName("multiply")
    def *(i: Int): A

  class B extends A:
    @targetName("multiply")
    override def *(i: Int): A = ???
