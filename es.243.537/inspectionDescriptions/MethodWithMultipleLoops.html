Inspecciones de IDE: Métodos con múltiples declaraciones de bucle Informa métodos que contienen más de una declaración de bucle.

Ejemplo:

El siguiente método se informará porque contiene dos bucles:


  void methodWithTwoLoops(int n1, int n2) {
    for (int i = 0; i < n1; i++) {
      System.out.println(i);
    }

    int j = 0;
    while (j < n2) {
      System.out.println(j);
      j++;
    }
  }

El siguiente método también se informará porque contiene un bucle anidado:


  void methodWithNestedLoop(int n1, int n2) {
    for (int i = 0; i < n1; i++) {
      for (int j = 0; j < n2; j++) {
        System.out.println(i + j);
      }
    }
  }
