Reporta cualquier llamada a métodos de las clases junit.framework.Assert, org.junit.Assert, o org.junit.Assume dentro de pruebas de JUnit 5.

Aunque las pruebas funcionan correctamente, la migración a org.junit.jupiter.api.Assertions/org.junit.jupiter.api.Assumptions te ayudará a evitar dependencias en versiones anteriores de JUnit.

Ejemplo:


  import org.junit.Assert;
  import org.junit.jupiter.api.Test;

  public class MyTest {
    @Test
    public void simpleTest() {
      Assert.assertEquals(4, 2 + 2);
    }
  }

Después de aplicar la solución rápida:


  import org.junit.jupiter.api.Assertions;
  import org.junit.jupiter.api.Test;

  public class MyTest {
    @Test
    public void simpleTest() {
      Assertions.assertEquals(4, 2 + 2);
    }
  }
