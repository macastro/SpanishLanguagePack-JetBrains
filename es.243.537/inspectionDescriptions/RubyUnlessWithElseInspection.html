Informa afirmaciones unless usadas con else. Es preferible poner la rama positiva primero.

Ejemplo:

# Mala práctica
a = 1
unless a > 2
  puts 'a es 2 o menos'
else
  puts 'a es mayor que 2'
end
Hay una corrección rápida disponible para convertir una unless en if. Después de aplicar la corrección, el resultado tendrá este aspecto: 
# Buena práctica
a = 1
if a > 2
  puts 'a es mayor que 2'
else
  puts 'a es 2 o menos'
end
