Reporta campos de matrices que son declarados volatile. Estas declaraciones pueden ser confusas debido a que acceder al propio arreglo sigue las reglas para campos volatile, pero acceder al contenido del arreglo no.

Ejemplo:


  class Data {
    private volatile int[] idx = new int[0];
  }

    private final AtomicIntegerArray idx = new AtomicIntegerArray(new int[0]);
  }
