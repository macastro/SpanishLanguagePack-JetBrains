@UiFieldInforma sobre la asignación a un campo marcado por la anotación.

Estos campos se inicializan de forma predeterminada mediante @UiField, por lo que el código del usuario no debería cambiarlos. Si se desea inicializar un campo de este tipo en el código propio, se debe establecer en true el atributo de @UiField provided.

La corrección rápida establece en true el atributo provided de la anotación @UiField.
