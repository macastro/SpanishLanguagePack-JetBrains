ChangeScope informa propiedades no asignadas al crear una Entity con las funciones new u getOrCreate.

Es obligatorio asignar todas las propiedades no nulas de la Entity, o bien etiquetarlas con la anotación @Many. Si no, se producirá una excepción en tiempo de ejecución.

Ejemplo:


  interfaz A: Entidad {
    var x: Int
    var y: Int?
    @Muchos var z: Int
    var p: Int
  }

  fun ChangeScope.foo() {
    nueva (A::class) {
      esta.p = 0
    }
  }

El arreglo rápido generará una asignación para la propiedad obligatoria faltante


  fun ChangeScope.foo() {
    nueva (A::class) {
      esta.x = x
      esta.p = 0
    }
  }
