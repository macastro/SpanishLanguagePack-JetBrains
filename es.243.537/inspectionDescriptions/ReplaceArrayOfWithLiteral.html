Informa de llamadas al método arrayOf que pueden reemplazarse por literales de matriz [...].

Ejemplos:

  annotation class MyAnnotation(val strings: Array<String>)

  @MyAnnotation(arrayOf("alpha", "beta", "omega")) // 'arrayOf()' reemplazable
  class MyClass

Después de aplicar la corrección rápida:

  annotation class MyAnnotation(val strings: Array<String>)

  @MyAnnotation(["alpha", "beta", "omega"])
  class MyClass
