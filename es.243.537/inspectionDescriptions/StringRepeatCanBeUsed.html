Informa de bucles que se pueden reemplazar con un único método String.repeat() (disponible desde Java 11).

Ejemplo:


  void append(StringBuilder sb, int count, Object obj) {
    for (int i = 0; i < count; i++) {
      sb.append(obj);
    }
  }

Tras aplicar la corrección rápida:


  void append(StringBuilder sb, int count, Object obj) {
    sb.append(String.valueOf(obj).repeat(Math.max(0, count)));
  }

De forma predeterminada, la inspección puede encerrar count con Math.max(0, count) si no puede demostrar estáticamente que count no es negativo. Esto se hace para evitar un posible cambio de semántica, ya que String.repeat() rechaza los números negativos. Usa la opción Añadir Math.max(0,count) para evitar un posible cambio de semántica para deshabilitar este comportamiento si es necesario.

Del mismo modo, una cadena que quieras repetir se puede encerrar en String.valueOf para evitar una posible NullPointerException si se desconoce si puede ser null.

Esta inspección solo informa si el nivel de lenguaje del proyecto o módulo es 11 o superior.
