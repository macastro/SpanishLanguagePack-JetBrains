Informa sobre fragmentos de código Java que son idénticos a los métodos static existentes, sugiriendo reutilizar dichos métodos static. La reutilización de los métodos existentes acorta el código y lo hace más legible.

Ejemplo:


  static List<String> readFileAndTrim(Path path) throws IOException {
    List<String> lines = Files.readAllLines(path);
    return lines.stream().map(String::trim).toList();
  }
  
  static List<String> readFileAndTrim(String path) throws IOException {
    Path p = Path.of(path);
    List<String> lines = Files.readAllLines(p);
    return lines.stream().map(String::trim).toList();
  }
Aquí, el segundo método es bastante parecido al primero, y el primero puede reutilizarse en su implementación. Después de aplicar la corrección rápida, el resultado será el siguiente: 

  static List<String> readFileAndTrim(Path path) throws IOException {
    List<String> lines = Files.readAllLines(path);
    return lines.stream().map(String::trim).toList();
  }

  static List<String> readFileAndTrim(String path) throws IOException {
    Path p = Path.of(path);
    return readFileAndTrim(p);
  }
