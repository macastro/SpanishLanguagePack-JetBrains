Informa de las anotaciones @Aspect en las versiones de Spring anteriores a la 2.5 si no hay ningún elemento <aop:aspectj-autoproxy/> en la configuración XML.

Ejemplo:


@Aspect // No se ha habilitado el soporte de '@AspectJ
class MyAspect {
   ...
}


<beans>
   <!-- <aop:aspectj-autoproxy/> -->
</beans>
