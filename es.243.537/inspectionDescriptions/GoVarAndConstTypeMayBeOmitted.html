Informa sobre los tipos en declaraciones de variables y constantes que se pueden omitir, ya que el compilador puede inferirlos. Tales tipos son redundantes y su omisión puede mejorar la legibilidad del código.

Ejemplo:

var s string = fmt.Sprintln("hi")

El tipo string en la declaración de variable puede omitirse. Para eliminar el tipo, usa la acción rápida Eliminar tipo. Después de aplicar la acción rápida:

var s = fmt.Sprintln("hi")
