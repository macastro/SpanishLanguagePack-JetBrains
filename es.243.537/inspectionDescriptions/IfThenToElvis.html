Inspección: Reporta las expresiones `if-then` que pueden ser convertidas a expresiones Elvis (`?:`)

Ejemplo:


  fun puedeSerFoo(): String? = "foo"

  var foo = puedeSerFoo()
  val bar = if (foo == null) "hola" else foo

La corrección rápida convierte la expresión `if-then` a una expresión Elvis (`?:`):


  fun puedeSerFoo(): String? = "foo"

  var foo = puedeSerFoo()
  val bar = foo ?: "hola"
