Informa de los servlets que no tienen una asignación asociada.

Ejemplo:


  <web-app>
    <servlet>
      <servlet-name>foo</servlet-name> // Error: el servlet debe tener una asignación
      <servlet-class>javax.servlet.Servlet</servlet-class>
    </servlet>
  
    <filter>
      <filter-name>bar</filter-name> // Error: el filtro debe tener una asignación
      <filter-class>java.lang.String</filter-class>
    </filter>
  </web-app>
