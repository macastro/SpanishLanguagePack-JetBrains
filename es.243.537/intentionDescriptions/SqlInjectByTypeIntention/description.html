Inyecta un dialecto especifico y asócialo con un tipo de dato SQL seleccionado. Aplica este dialecto a todas las apariciones del mismo tipo (p. ej., query).

En el ejemplo Before, puedes invocar la acción de intención Inyectar por tipo en el valor 'SELECT * FROM foo', luego inyectar este tipo de objeto y asociarlo con cualquier dialecto SQL. Este valor tiene un tipo de objeto query (especificado por q query DEFAULT 'SELECT * FROM foo'). Por lo tanto, la próxima vez que uses el tipo query en el código, se inyectará automáticamente el dialecto seleccionado.
