Genera código cliché para manejo de errores.

Se puede invocar en variables y expresiones del tipo error.

  • Sustituye la variable err por la siguiente sintaxis:

    if err != nil {
    return value1, value2, ...
  }

  • Sustituye la expresión expr por la siguiente sintaxis:

    if err := expr; err != nil {
    return value1, value2, ...
  }

Cada valor es la variable err o el valor predeterminado del tipo del valor de retorno correspondiente de la función envolvente. Esta plantilla se puede invocar en una expresión con múltiples valores.
