Existen cuatro tipos de módulos:

  • Idiomas que definen múltiples aspectos de los idiomas que luego pueden ser utilizados para escribir código en soluciones
  • Generadores que definen las reglas de conversión para transformar código a código en otro idioma
  • Soluciones que guardan código (programas o bibliotecas)
  • DevKits que agrupan múltiple idiomas y soluciones juntas, de manera que los usuarios pueden expresar dependencias con una sola declaración de dependencia

Las soluciones en sí pueden ser de tres tipos básicos:

  • Sandbox - programas de usuario final
  • Soluciones de tiempo de ejecución - bibliotecas de código reutilizable, frecuentemente cargadas de fuentes externas (p. ej. archivos jar)
  • Soluciones de plugin - código que extiende la funcionalidad de IDE (p. ej. nuevas entradas de menú, acciones, ventanas de herramientas, etc.)
