**Administración de solicitudes de extracción** Puedes administrar las solicitudes de extracción de GitHub entrantes directamente desde IntelliJ IDEA Community. Para ello, selecciona **Git | GitHub | Mostrar solicitudes de extracción** en el menú principal. IntelliJ IDEA Community te permite asignar y combinar solicitudes de extracción, ver el cronograma y los comentarios en línea, enviar comentarios y revisiones y aprobar cambios sin salir del IDE.
