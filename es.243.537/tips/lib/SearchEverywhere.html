Buscar en todas partes

Presione Shift dos veces para buscar archivos, acciones, símbolos, elementos UI, ramas Git y comentarios presentes en tu proyecto. Pulsando dos veces Shift nuevamente, extenderás la búsqueda a elementos que no pertenecen al proyecto.

Utiliza pestañas o métodos abreviados directos &shortcut:GotoClass; para clases, &shortcut:GotoFile; para archivos, &shortcut:GotoSymbol; para símbolos, y &shortcut:GotoAction; para acciones para acotar los resultados de tu búsqueda.
