Después de haber escrito la parte significativa de una declaración o una sentencia, puedes presionar &shortcut:EditorCompleteStatement; para insertar automáticamente elementos de sintaxis necesarios (paréntesis de cierre, llaves, punto y coma, etc.) y colocarte en la posición donde puedes continuar escribiendo.

Por ejemplo, si has escrito

public void Foo(string input/*caret*/

puedes presionar &shortcut:EditorCompleteStatement; para obtener

public void Foo(string input) { /*caret*/ }
