DDL de objeto de base de datos, datos y ubicación del Explorador de base de datos

Presionar &shortcut:GotoDeclaration; abre el DDL del objeto seleccionado. Por ejemplo, presionar este atajo en el nombre de una tabla en una declaración SELECT mostrará el DDL de dicha tabla (sentencia CREATE TABLE).

Presionar &shortcut:EditSource; abre los datos del objeto. Por ejemplo, presionar este atajo en el nombre de una tabla en una declaración SELECT mostrará el contenido de esa tabla.

Presionar Alt+Shift+B se mueve al objeto en la ventana de herramientas "Explorador de base de datos".
