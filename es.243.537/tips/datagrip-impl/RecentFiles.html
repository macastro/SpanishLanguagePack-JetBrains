Ver archivos recientes

Si accidentalmente cerraste una consola de consulta, un script SQL o cualquier otra pestaña, presiona &shortcut:RecentFiles;. El diálogo de Archivos recientes muestra una lista de archivos editados recientemente.

Presiona &shortcut:RecentFiles; (Ver | Archivos recientes) para ver la lista de archivos abiertos recientemente.
