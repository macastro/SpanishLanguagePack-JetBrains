Archivos SQL como fuentes de datos DDL

Los archivos SQL que contienen instrucciones CREATE pueden utilizarse como fuentes de datos. Como resultado, puede crear diagramas y referenciar todas las tablas, columnas y otros objetos definidos en dichos archivos en el editor.

Para crear una fuente de datos DDL, arrastre el archivo DDL al Explorador de base de datos (Ver | Ventanas de herramientas | Explorador de base de datos) desde el administrador de archivos.
