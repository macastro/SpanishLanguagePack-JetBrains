Administración de Pull Requests

Puedes administrar directamente los Pull Requests (PR) entrantes de GitHub desde Aqua. Para esto, selecciona "Git | GitHub | Mostrar Pull Requests " en el menú principal. Con Aqua, puedes asignar y fusionar PR, mostrar la línea de tiempo y los comentarios en línea, enviar comentarios y revisiones, y aprobar cambios sin salir del IDE.
