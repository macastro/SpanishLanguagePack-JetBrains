Utilizando el complemento Big Data Tools Plugin es posible extraer un job de Spark de un cuaderno Zeppelin a otro archivo.

  1. Seleccionar el fragmento de código del job de Spark en el cuaderno.
  2. Clic derecho sobre el código seleccionado y seleccionar desde el menú contextual Refactor | Extract Spark Job.
  3. Al guardar el archivo Scala, se abre en otra pestaña del editor.
