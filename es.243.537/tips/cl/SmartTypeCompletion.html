Finalización de código compatible con el tipo

La finalización de código compatible con el tipo analiza el tipo esperado de expresión completa y ayuda a encontrar métodos y variables aplicables en el contexto actual. Esta finalización de código funciona después de la palabra clave return, en asignaciones, en listas de argumentos de llamada a métodos, etc. Puedes presionar &shortcut:SmartTypeCompletion; ( Código | Finalización de código | Compatible con el tipo ) para restringir la lista de finalización.
