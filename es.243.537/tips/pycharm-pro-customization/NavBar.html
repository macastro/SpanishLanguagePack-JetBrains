Barra de navegación

La barra de navegación es una alternativa rápida a la ventana de herramientas Proyecto donde puedes recorrer la estructura de tu proyecto, abrir archivos específicos y saltar a elementos de código específicos en el archivo actual.

Pulsa &shortcut:ShowNavBar; para mostrar la barra de navegación y utiliza las teclas de flecha para cambiar entre archivos y carpetas.
