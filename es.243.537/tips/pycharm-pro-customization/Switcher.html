Switcher para la navegación

Utiliza el &shortcut:Switcher; para navegar entre archivos abiertos y ventanas de herramientas. Mantén pulsada la tecla Ctrl para dejar abierto el cuadro emergente del switcher. Para navegar, utiliza Tab y Mayús+Tab.

Pulsa Borrar o Retroceso para cerrar una pestaña del editor u ocultar una ventana de herramientas.
