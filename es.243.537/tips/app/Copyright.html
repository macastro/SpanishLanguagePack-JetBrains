Agrega avisos de copyright

En IntelliJIDEACommunity, puedes configurar diferentes avisos de copyright para diferentes conjuntos de archivos. Para tu comodidad, puedes usar variables. Por ejemplo, usa $today.year para mantener el año actualizado.

En el cuadro de diálogo Settings/Preferences (&shortcut:ShowSettings;), selecciona Editor | Copyright | Copyright Profiles, crea un nuevo perfil y agrega el texto para tu aviso de copyright usando variables.
