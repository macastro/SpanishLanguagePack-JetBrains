Comparar archivos

Para comparar dos archivos .jar, .zip o .phar o incluso archivos dentro de un archivo, selecciónelos en la ventana de herramientas Proyecto y presione &shortcut:CompareTwoFiles;.

La función Comparar archivos está integrada con el descompilador de bytecode Java y le permite ver qué ha cambiado exactamente entre dos versiones diferentes de una biblioteca.
