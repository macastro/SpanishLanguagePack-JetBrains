Utiliza el selector para navegar

Utiliza el selector &shortcut:Switcher; para navegar entre archivos abiertos y ventanas de herramientas. Mantén pulsada la tecla Ctrl para dejar el selector abierto. Para navegar, utiliza Tab y Shift+Tab.

Pulsa la tecla Delete o BackSpace para cerrar una pestaña de editor u ocultar una ventana de herramienta.
