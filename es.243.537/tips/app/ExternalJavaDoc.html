Documentación externa

Para abrir tu navegador con la página de documentación para el elemento en el caret, presiona &shortcut:ExternalJavaDoc; o selecciona Tools | Web Browsers and Preview del menú principal.

Asegúrate de establecer un navegador predeterminado en la configuración. Para verificar el navegador predeterminado, abre la configuración presionando &shortcut:ShowSettings; y navega a Tools | Web Browsers and Preview.
