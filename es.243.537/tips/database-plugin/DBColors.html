Usa colores para fuentes de datos

Puedes usar colores para distinguir entre tus fuentes de datos y sus elementos. Para establecer un color para una fuente de datos o su objeto, haz clic derecho en el elemento en la ventana de herramientas de Base de Datos (Ver | Ventanas de Herramientas | Base de Datos) y selecciona Herramientas | Establecer Color.
