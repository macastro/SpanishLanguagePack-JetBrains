Trabajar con archivos SQL

Haga doble clic en un archivo SQL para abrirlo en el IDE. Para ejecutar una consulta desde este archivo, llame a acciones de intención (Opción+Enter para macOS, Alt+Enter para Windows y Linux) y seleccione Ejecutar consulta en la consola. En la lista Sesiones, seleccione una consola existente o cree una nueva.

Tenga en cuenta que una nueva consola de consulta significa una nueva conexión a una fuente de datos.
