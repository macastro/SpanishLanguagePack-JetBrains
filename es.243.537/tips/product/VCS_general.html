Control de versiones

IntelliJ IDEA ofrece integración lista para usar con los siguientes sistemas de control de versiones: Git, Mercurial, Subversion y Perforce.

Realiza versiones de tu aplicación para colaborar en ella, así como para eliminar los riesgos de almacenar localmente toda tu base de código. Presiona &shortcut:ActivateVersionControlToolWindow; para ver el historial de los cambios en el proyecto o presiona &shortcut:CheckinProject; para enviar tus cambios locales al repositorio.
