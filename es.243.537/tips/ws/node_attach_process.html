Depuración de aplicaciones Node.js en ejecución

Si la aplicación Node.js se inició con las marcas --inspect o --inspect-brk, puede iniciar una depuración desde las ventanas de herramientas integradas "Terminal", "Ejecutar" o "Depurar".

En la ventana de herramientas "Terminal" se muestra el mensaje informativo Debugger listening <host>:<port>. El valor predeterminado del puerto es 9229. Para iniciar la depuración, presione Control+Shift mientras hace clic en el enlace.
