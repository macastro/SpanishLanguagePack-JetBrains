Switcher de navegación

Para moverte entre archivos abiertos y ventanas de herramientas, utiliza el Switcher &shortcut:Switcher;. Para salir del popup del Switcher, mantén pulsada la tecla Control. Para navegar, utiliza Tab y Shift+Tab.

Para cerrar una pestaña del editor, o para ocultar una ventana de herramientas, pulsa Delete o BackSpace.
