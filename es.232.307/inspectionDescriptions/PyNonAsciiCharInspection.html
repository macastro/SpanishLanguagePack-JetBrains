Informa cuando un archivo contiene caracteres no ASCII y no tiene una declaración de codificación en la parte superior en Python 2.

Ejemplo:


class A(object):
# №5
    def __init__(self):
        pass

En este ejemplo, el IDE informa un símbolo no ASCII en un comentario y una falta de declaración de codificación. Aplica la corrección rápida propuesta para añadir una declaración de codificación faltante:


# coding=utf-8
class A(object)
# №5
    def __init__(self):
        pass
