Reporta expresiones redundantes de Unit.

Unit en Kotlin se puede utilizar como un tipo de retorno de las funciones que no devuelven nada significativo. El tipo Unit solo tiene un valor posible, el cual es el objeto Unit.

Ejemplos:


  fun redundantA(): Unit {
      return Unit // redundante, 'Unit' se devuelve como predeterminado y coincide con el tipo de retorno esperado
  }

  fun requiredA(condition: Boolean): Any {
      if (condition) return "hola"
      return Unit // 'Unit' explícito es requerido, puesto que el tipo esperado es 'Any'
  }

  fun redundantB(condition: Boolean): Any = if (condition) {
      fun auxiliar(): Int = 1
      println("${ancillary()}")
      Unit // redundante, puesto que la última expresión es de tipo 'Unit'
  } else {
      println("sino")
  }

  fun requiredB(condition: Boolean): Any = if (condition) {
      1024
      Unit // requerido, de lo contrario '1024' (Int) sería el valor de retorno
  } else {
      println("sino")
  }
