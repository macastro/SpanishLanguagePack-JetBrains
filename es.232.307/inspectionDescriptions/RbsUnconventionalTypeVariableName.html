Reporta variables de tipo con nombres que son ya sea demasiado cortos, demasiado largos o no coinciden con el patrón de expresión regular especificado.

Usa el campo Patrón para especificar una expresión regular para validar nombres de variables de tipo.

Usa los campos Longitud mínima y Longitud máxima para especificar la longitud mínima y máxima de un nombre respectivamente.
