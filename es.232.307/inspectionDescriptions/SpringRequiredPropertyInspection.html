Informa las propiedades bean configuradas por @Required que no han sido inyectadas o cableadas automáticamente.

Ejemplo:


<beans>
  <bean class="beans.MyComponent" id="foo"/>
  <!-- informa "Faltan propiedades requeridas: 'port'" -->
</beans>


public class MyComponent {
   @Required
   public void setPort(int port) // informa "La propiedad requerida 'port' no está inyectada"
   {...}
}
