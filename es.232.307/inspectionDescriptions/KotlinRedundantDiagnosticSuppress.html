Informes de usos de anotaciones @Suppress que pueden eliminarse de forma segura porque el diagnóstico del compilador que afectan ya no es aplicable en este contexto.

Ejemplo:


  fun doSmth(@Suppress("UNUSED_PARAMETER") used: Int) {
    println(used)
  }

Después de aplicar la solución rápida:


  fun doSmth(used: Int) {
    println(used)
  }
