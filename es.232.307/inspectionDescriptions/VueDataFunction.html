Informa de una propiedad ``data``` de un componente Vue que no es una función. Sugiere envolver un literal de objeto con una función.

Al definir un componente, ``data``` debe declararse como una función que devuelve el objeto de datos inicial, porque la misma definición se utilizará para crear numerosas instancias. Si se sigue utilizando un objeto sin formato para ``data```, ese mismo objeto se compartirá por referencia en todas las instancias creadas. Con una función ``data```, cada vez que se crea una nueva instancia, podemos simplemente llamarla para devolver una nueva copia de los datos iniciales.
