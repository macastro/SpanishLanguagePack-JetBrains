Informa de llamadas a la función errors.As cuando el segundo argumento no es un puntero a una interfaz o a un tipo que implementa un error.

Estas llamadas provocan pánico en tiempo de ejecución.

Para obtener más información sobre la función As, consulta func As en go.dev.

Ejemplo:

_, err := os.Open("non-existing")
var pathError *fs.PathError
if errors.As(err, pathError) { // se requiere un puntero a *fs.PathError
}

Después de aplicar la corrección rápida Añadir prefijo '&':

_, err := os.Open("non-existing")
var pathError *fs.PathError
if errors.As(err, &pathError) {
}

Esta inspección solo informa si la versión del lenguaje es 1.13 o superior.
