Informa de comprobaciones de null que pueden ser sustituidas por una llamada a un método estático desde Objects o Stream.

Ejemplo:


  if (message == null) {
    application.messageStorage().save(new EmptyMessage());
  } else {
    application.messageStorage().save(message);
  }

Después de aplicar la corrección rápida:


  application.messageStorage()
    .save(Objects.requireNonNullElseGet(message, () -> new EmptyMessage()));

Utiliza la opción No advertir si la sustitución es más larga que el original para ignorar los casos en los que la sustitución es más larga que el código original.
