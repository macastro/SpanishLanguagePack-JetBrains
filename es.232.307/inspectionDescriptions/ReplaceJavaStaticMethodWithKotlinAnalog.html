Informa de una llamada al método Java que puede reemplazarse por una función Kotlin, por ejemplo, System.out.println().

Reemplazar el código elimina la dependencia de Java y convierte el código en Kotlin idiomático.

La corrección rápida reemplaza las llamadas al método Java por la misma llamada Kotlin.

Ejemplo:


  import java.util.Arrays

  fun main() {
      val a = Arrays.asList(1, 3, null)
  }

Después de aplicar la corrección rápida:


  fun main() {
      val a = listOf(1, 3, null)
  }
