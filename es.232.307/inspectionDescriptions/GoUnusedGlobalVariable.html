Reporta variables globales que están definidas pero nunca son usadas en el código.

Si tienes variables no utilizadas, el código no se compilará. Para más información sobre variables e importaciones no utilizadas, consulta Importaciones y variables no utilizadas en go.dev. 

func main() {
 a := 422
}

El código en el ejemplo no se compilará. Por lo tanto, se resalta como un error. Puedes aplicar dos soluciones rápidas para tales casos: Eliminar variable y Renombrar a _. La primera solución rápida elimina la variable, la segunda convertirá la variable a un identificador en blanco.

Después de que se aplique la solución rápida Renombrar a _:

func main() {
 _ := 422
}
