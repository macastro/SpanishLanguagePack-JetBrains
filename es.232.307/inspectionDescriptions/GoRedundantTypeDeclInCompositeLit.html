Reporta declaraciones de tipo redundantes en literales compuestos.

Ejemplo:

nums := [][]int{[]int{1}, []int{2}}

Tenemos una rebanada de rebanadas del tipo int. En este caso, puedes usar una definición más corta. Puedes corregir este código manualmente o usar el arreglo rápido Eliminar tipo redundante. Después de aplicar el arreglo rápido, el código queda de la siguiente manera:

nums := [][]int{{1},{2}}

Para obtener más información sobre los literales compuestos, consulta Go Language Specification: Composite Literals en go.dev.
