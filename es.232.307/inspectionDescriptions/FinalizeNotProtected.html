Informa cualquier implementación del método Object.finalize() que esté declarado como public.

De acuerdo con el contrato del Object.finalize(), sólo el recolector de basura llama a este método. Hacer público este método puede generar confusión, porque significa que el método puede ser usado desde otro código.

Se proporciona una corrección rápida para hacer protected al método, para evitar que sea invocado desde otras clases.

Ejemplo:


  class X {
    public void finalize() {
      /* ... */
    }
  }

Después de aplicar la corrección rápida:


  class X {
    protected void finalize() {
      /* ... */
    }
  }
