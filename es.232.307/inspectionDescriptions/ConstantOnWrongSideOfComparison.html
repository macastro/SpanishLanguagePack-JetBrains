Reporta operaciones de comparación donde el valor constante está en el lado incorrecto.

Algunas convenciones de codificación especifican que las constantes deben estar en un lado específico de una comparación, ya sea izquierda o derecha.

Ejemplo:


  boolean compare(int x) {
    return 1 > x; // Constante '1' en el lado izquierdo de la comparación
  }

Después de aplicar la revisión rápida:


  boolean compare(int x) {
    return x < 1;
  }

Utiliza la configuración de inspección para escoger el lado de las constantes en las comparaciones y si advertir si los literales null están en el lado incorrecto.
