Inspecciona los métodos `createOptionsPanel()` en implementaciones de inspección para informar aquellos que se pueden convertir automáticamente a `getOptionsPane()`.

Se recomienda describir controles de forma declarativa en `getOptionsPane()`, y está en desuso crear controles de opciones de inspección mediante `createOptionsPanel()`. Esta inspección revisa implementaciones de `createOptionsPanel()`, y cuando la implementación utiliza `InspectionOptionsPanel` o sus descendientes y es suficientemente simple, sugiere convertirla automáticamente a la nueva API.

Esta inspección actualmente solo admite Java y Kotlin. No se puede convertir código que utilice métodos Kotlin como `apply` o `also`. Intenta incrustarlo.
