Se reportan los usos del operador == o != para arrays que deberían ser reemplazados con contentEquals().

Los operadores == y != comparan las referencias de los arrays en lugar de su contenido.

Ejemplos:

  fun test() {
      val a = arrayOf(1, 2, 3)
      val b = arrayOf(1, 2, 3)
      println(a == b) // comparación de referencias
  }

Después de aplicar la solución rápida:

  fun test() {
      val a = arrayOf(1, 2, 3)
      val b = arrayOf(1, 2, 3)
      println(a.contentEquals(b))
  }
