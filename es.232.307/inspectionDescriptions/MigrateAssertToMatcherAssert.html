Informa sobre las llamadas a los métodos Assert.assertEquals(), Assert.assertTrue(), etc. que pueden migrarse a llamadas declarativas de Assert.assertThat() de estilo Hamcrest.

Por ejemplo:


  public class SubstantialTest {
    @Test
    public void testContents(Collection<String> c, String s) {
      Assert.assertTrue(c.contains(s));
      Assert.assertEquals(c, s);
      Assert.assertNotNull(c);
      Assert.assertNull(c);
      Assert.assertFalse(c.contains(s));
    }
  }

Se proporciona una corrección rápida para realizar la migración:


  public class SubstantialTest {
    @Test
    public void testContents(Collection<String> c, String s) {
      assertThat(c, hasItem(o));
      assertThat(o, is(c));
      assertThat(c, notNullValue());
      assertThat(c, nullValue());
      assertThat(c, not(hasItem(o)));
    }
  }

Esta inspección requiere que la biblioteca Hamcrest esté disponible en la ruta de clase.

Utiliza la opción Importar estáticamente los métodos del emparejador para especificar si deseas que la corrección rápida importe estáticamente los métodos de emparejamiento Hamcrest.
