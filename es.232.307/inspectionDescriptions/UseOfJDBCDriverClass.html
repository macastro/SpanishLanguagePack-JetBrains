Informa sobre el uso de clases específicas de controlador JDBC. El uso de dichas clases vinculará tu proyecto a una base de datos y controlador específico, frustrando el propósito de JDBC y resultando en una pérdida de portabilidad.

Ejemplo:


  import java.sql.Driver;

  abstract class Sample implements Driver {
    public void foo() {
        Sample sample;
    }
  }
