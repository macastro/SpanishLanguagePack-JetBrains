Informa de variables static que pueden no estar inicializadas durante la inicialización de clase.

Ejemplo:


  class Foo {
    public static int bar;

    static { }
  }

Ten en cuenta que esta inspección usa un algoritmo de flujo de datos muy conservador y puede informar incorrectamente de que las variables static no están inicializadas. Las variables que se informa que están inicializadas siempre lo estarán.

Usa la opción Ignorar campos primitivos para ignorar campos primitivos no inicializados.
