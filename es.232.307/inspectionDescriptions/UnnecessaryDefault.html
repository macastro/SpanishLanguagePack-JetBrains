Reporta enumeración switch declaraciones o expresiones con ramas default que nunca pueden tomarse, porque todos los valores posibles están cubiertos por una rama case.

Tales elementos son redundantes, especialmente para expresiones switch, porque no se compilan cuando todas las constantes de enumeración no están cubiertas por una rama case.

El nivel de idioma debe configurarse a 14 para reportar expresiones switch.

La corrección rápida provista quita ramas default.

Ejemplo:


  enum E { A, B }
  int foo(E e) {
    return switch (e) {
      case A -> 1;
      case B -> 2;
      default -> 3;
    };
  }

La corrección rápida ya aplicada:


  enum E { A, B }
  int foo(E e) {
    return switch (e) {
      case A -> 1;
      case B -> 2;
    };
  }

Use la opción Sólo reportar expresiones switch para reportar sólo ramas default redundantes en expresiones switch.
