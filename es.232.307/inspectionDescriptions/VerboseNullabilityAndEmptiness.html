Informa sobre combinaciones de verificaciones de null y de vacío que pueden simplificarse en una sola verificación.

La corrección rápida reemplaza las comprobaciones resaltadas con una llamada a una comprobación combinada, como isNullOrEmpty().

Ejemplo:


  fun test(list: List<Int>?) {
      if (list == null || list.isEmpty()) {
          println("La lista está vacía.")
      } else {
          println(list.joinToString())
      }
  }

Después de aplicar la corrección rápida:


  fun test(list: List<Int>?) {
      if (list.isNullOrEmpty()) {
          println("La lista está vacía.")
      } else {
          println(list.joinToString())
      }
  }
