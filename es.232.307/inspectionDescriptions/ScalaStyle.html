Informa de problemas encontrados por Scalastyle.

Scalastyle examina tu código Scala e indica problemas potenciales con él.

Coloca el archivo de configuración en <project-root>, <project-root>/.idea, o <project-root>/project.

El nombre del archivo de configuración puede ser scalastyle-config.xml o scalastyle_config.xml. Los archivos de configuración para fuentes de prueba se pueden sobrescribir mediante scalastyle-test-config.xml o scalastyle_test_config.xml.

La documentación completa está disponible en el sitio web de Scalastyle.
