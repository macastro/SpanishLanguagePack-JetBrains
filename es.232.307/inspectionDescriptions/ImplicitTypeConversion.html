Informa conversiones implícitas entre los tipos XPath predefinidos STRING, NUMBER, BOOLEAN y NODESET. Ayuda a escribir scripts XSLT que son más expresivos sobre tipos, y evita errores sutiles:

Ejemplo:


  <xsl:if test="foo" />
no es lo mismo que 

  <xsl:if test="string(foo)" />

La primera prueba verifica si el elemento "foo" existe (count(foo) > 0); mientras que la última solo es verdadera si el elemento de hecho contiene texto (string-length(foo) > 0). Sugiere que la conversión de tipo sea más explícita.

Utiliza las siguientes opciones para configurar la inspección:

  • Habilita o deshabilita las conversiones implícitas entre ciertos tipos
  • Informa siempre las conversiones explícitas que no resultan en el tipo realmente esperado, por ejemplo, <xsl:if test="number(foo)" />
  • Ignora la conversión de NODESET a BOOLEAN usando la función string() como un atajo para escribir string-length() > 0.
