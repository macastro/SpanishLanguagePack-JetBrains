Informa sobre comparaciones de igualdad encadenadas.

Este tipo de comparaciones pueden resultar confusas: a == b == c significa (a == b) == c, pero es posible que se pretenda a == b && a == c.

Ejemplo:


  boolean chainedEquality(boolean a, boolean b, boolean c) {
    return a == b == c;
  }

Puedes utilizar paréntesis para que la comparación resulte menos confusa:


  boolean chainedEquality(boolean a, boolean b, boolean c) {
    return (a == b) == c;
  }
