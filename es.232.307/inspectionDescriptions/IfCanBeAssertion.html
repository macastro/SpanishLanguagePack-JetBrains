Reporta sentencias if que arrojan únicamente java.lang.Throwable desde una rama then y que no tienen una rama else. Estas sentencias se pueden convertir en sentencias assert más compactas.

La inspección también reporta Preconditions.checkNotNull() de Guava. Pueden reemplazarse por una llamada a Objects.requireNonNull() para la cual es posible que no se necesite una biblioteca.

Ejemplo:


  if (x == 2) throw new RuntimeException("falla");
  if (y == null) throw new AssertionError();
  Preconditions.checkNotNull(z, "z");

Después de aplicar la corrección rápida:


  assert x != 2 : "falla";
  Objects.requireNonNull(y);
  Objects.requireNonNull(z, "z");

De forma predeterminada, esta inspección proporciona una corrección rápida en el editor sin resaltado de código.
