Reporta expresiones `when` sólo con una rama `else` que pueden simplificarse.

La corrección rápida "Simplificar expresión" puede usarse para corregir el código automáticamente.

Ejemplo:


  fun redundante() {
      val x = when { // <== redundante, la corrección rápida simplifica la expresión when a "val x = 1"
          else -> 1
      }
  }
