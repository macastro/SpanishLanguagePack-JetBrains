Vista previa integrada

La vista previa integrada de WebStorm muestra el resultado de un código HTML en una pestaña de editor independiente. WebStorm guarda los cambios realizados en los archivos HTML o archivos CSS o JavaScript vinculados y recarga la vista previa automáticamente.

Para abrir la vista previa, coloca el cursor sobre el código HTML y haz clic en el ícono de WebStorm de la ventana emergente del navegador.
