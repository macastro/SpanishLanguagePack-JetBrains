Mostrar anotaciones Git Blame

Las anotaciones de Git muestran información detallada sobre la procedencia de cada línea de código (haz clic con el botón derecho en el margen y selecciona "Annotate with Git Blame").

Haz clic con el botón derecho en una anotación y selecciona "Show Diff" para ver la diferencia entre la versión actual del archivo y la versión anterior.
