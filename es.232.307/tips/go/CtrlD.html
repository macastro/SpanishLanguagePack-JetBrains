Duplicar un bloque de código o una línea

Presiona Duplicar bloque (editor) en el editor para duplicar el bloque de código seleccionado o la línea actual si no se seleccionó ningún bloque.

Si deseas agregar un salto de línea entre la parte duplicada y el texto original, presiona Ir a la acción y escribe "Duplicar líneas completas". Puedes asignar un atajo a esta acción en la configuración del elemento de menú "Mapa de teclas" Mostrar configuración.
