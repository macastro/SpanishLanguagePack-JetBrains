Completado de código

Utilizando la finalización de código, puedes presionar &shortcut:EditorChooseLookupItemReplace; para aceptar el elemento seleccionado que esté resaltado en la lista emergente.

A diferencia de aceptar con &shortcut:EditorChooseLookupItem;, el resto del nombre a la derecha del cursor se sobrescribirá con el nombre seleccionado. Esto es particularmente útil cuando reemplazas un nombre de método o variable por otro nombre de método o variable.
