Es la plantilla incorporada para enumerados PHP. Se usa cada vez que se crea un nuevo enumerado PHP al seleccionar Nuevo | Clase PHP del menú emergente en una de las vistas de proyecto.
Esta plantilla se puede editar. Además de las declaraciones, expresiones y comentarios de PHP, también se pueden usar las variables predefinidas que se indican a continuación. Cada variable se expande al valor correspondiente de forma similar a las macros.
Se pueden incluir las plantillas que están en la pestaña Incluye, especificando el nombre completo de la plantilla que se desea entre comillas como parámetro, mediante la directiva #parse.
Las variables predefinidas toman los siguientes valores:
${FILE_NAME} El nombre del archivo actual
${USER} El nombre de inicio de sesión de sistema del usuario actual
${DATE} La fecha actual del sistema
${TIME} La hora actual del sistema
${YEAR} El año actual
${MONTH} El mes actual
${DAY} El día del mes actual
${HOUR} La hora actual
${MINUTE} El minuto actual
${PRODUCT_NAME} El nombre del IDE actual
${DS} El símbolo del dólar se evalúa como el carácter '$' normal.
${PROJECT_NAME} El nombre del proyecto actual
