Una plantilla integrada para una clase Ruby. Se usa siempre que se crea una nueva clase Ruby seleccionando Nuevo | Archivo/Clase Ruby en el menú emergente en una de las vistas del proyecto. Junto con las sentencias, expresiones y comentarios Ruby, también se pueden usar las variables predefinidas listadas a continuación. Cada variable se expande como macros en el valor correspondiente. Por medio de la #parse directiva, se pueden incluir plantillas desde la pestaña Incluye especificando el nombre completo de la plantilla deseada como un parámetro entre comillas.
Las variables predefinidas toman los siguientes valores:
${FILE_NAME} Nombre del archivo actual
${USER} Nombre de inicio de sesión del sistema del usuario actual
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${DAY} Día actual del mes
${HOUR} Hora actual
${MINUTE} Minuto actual
${PRODUCT_NAME} Nombre actual del IDE
${PROJECT_NAME} Nombre del proyecto actual
