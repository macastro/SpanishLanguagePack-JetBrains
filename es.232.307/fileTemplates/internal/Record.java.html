Se aplica a los registros de Java creados al invocar Nuevo | Clase de Java | Registro en la ventana de herramientas Proyecto.
Esta plantilla integrada se puede editar. Además de las expresiones y comentarios de Java, también puedes utilizar las variables predefinidas (enumeradas a continuación) que se expandirán como macros en los valores correspondientes.

También es posible especificar variables personalizadas. Las variables personalizadas usan el siguiente formato: ${NOMBRE_DE_VARIABLE}, donde NOMBRE_DE_VARIABLE es un nombre para tu variable (por ejemplo, ${NOMBRE_DE_LA_FUNCION_PERSONALIZADA}). Antes de que el IDE cree un archivo nuevo con variables personalizadas, verás un cuadro de diálogo en el que puedes definir valores para variables personalizadas en la plantilla.

Con la directiva #parse, puedes incluir plantillas de la pestaña Incluidos. Para incluir una plantilla, especifica el nombre completo de la plantilla como parámetro entre comillas (por ejemplo, #parse("File Header.java").
Las variables predefinidas toman los siguientes valores:
${NOMBRE_DEL_PAQUETE} Nombre del paquete en el que se crea un registro nuevo
${NOMBRE} Nombre del nuevo registro que especificaste en el cuadro de diálogo Crear clase nueva
${USUARIO} Nombre de inicio de sesión del sistema del usuario actual
${FECHA} Fecha actual del sistema
${HORA} Hora actual del sistema
${AÑO} Año actual
${MES} Mes actual
${NOMBRE_DEL_MES_CORTO} Las primeras 3 letras del nombre del mes actual (Ene, Feb, etc.)
${NOMBRE_DEL_MES_COMPLETO} Nombre completo del mes actual (Enero, Febrero, etc.)
${DIA} Día actual del mes
${HORA} Hora actual
${MINUTO} Minuto actual
${NOMBRE_DEL_PROYECTO} Nombre del proyecto actual
