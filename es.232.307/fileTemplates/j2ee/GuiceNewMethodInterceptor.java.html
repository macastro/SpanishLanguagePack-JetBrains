Esta es una plantilla integrada que se utiliza cada vez que crea una clase interceptor de métodos de Guice nueva.
Puede utilizar la directiva #parse para incluir plantillas en la pestaña Incluir, especificando el nombre completo de la plantilla necesaria entre comillas como parámetro. Por ejemplo:
#parse("File Header.java")
Las variables predefinidas toman los siguientes valores:
${PACKAGE_NAME} El nombre del paquete en el que se crea la nueva clase
${NAME} El nombre de la nueva clase especificado por el usuario en el cuadro de diálogo Crear clase nueva
${USER} El nombre de inicio de sesión del sistema del usuario actual
${DATE} La fecha actual del sistema
${TIME} La hora actual del sistema
${YEAR} El año actual
${MONTH} El mes actual
${MONTH_NAME_SHORT} Los primeros 3 caracteres del nombre del mes actual. Por ejemplo: ene, feb, etc.
${MONTH_NAME_FULL} El nombre completo del mes actual. Por ejemplo: enero, febrero, etc.
${DAY} El día del mes actual
${HOUR} La hora actual
${MINUTE} El minuto actual
${PROJECT_NAME} El nombre del proyecto actual
