Esta plantilla que viene predeterminada se puede utilizar para crear componentes de funciones React, por ejemplo, mediante la refactorización Extraer componente. Puedes personalizar la plantilla, por ejemplo, definir componentes como funciones de flecha o cambiar las definiciones de tipos de prop. También puedes utilizar variables predefinidas. Estas variables se reemplazarán por los valores correspondientes cuando se aplique la plantilla.
Las variables predefinidas tomarán los siguientes valores:
${NAME} nombre del componente
${HAS_PROPS} verdadero si el componente tiene props
${COMPONENT_BODY} cuerpo del componente
${PROP_TYPES} definición de tipos de prop (usando el módulo prop-types)
