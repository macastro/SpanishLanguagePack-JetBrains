Genera la rama default para la expresión o declaración switch utilizada por acciones de corrección rápida como Insertar rama 'default'.
Esta plantilla incorporada es editable. Por ejemplo, puede cambiar la excepción lanzada o llamar a un marco de registro en su lugar.
Las variables predefinidas toman los siguientes valores:
${EXPRESSION} Texto de expresión del selector de switch
${EXPRESSION_TYPE} Tipo de expresión del selector de switch
