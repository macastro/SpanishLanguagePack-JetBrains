Después de desenrollar un valor opcional con una declaración guard, rellena el bloque else. La plantilla es editable. Además de las expresiones y los comentarios de Swift, puedes utilizar variables predefinidas que se expandirán a sus valores correspondientes.
Las variables predefinidas toman los valores siguientes:
${RETURN_TYPE} Tipo de valor devuelto de la función envolvente (si existe)
${DEFAULT_RETURN_VALUE} Valor predeterminado sugerido para el valor devuelto (puede estar vacío)
Todas las variables predefinidas de la plantilla Encabezado de archivo Swift (pestaña Incluir) también están disponibles (por ejemplo: ${USER}, ${DATE}, ).
