Esta es una plantilla para crear métodos de prueba HTTP para Ktor en una clase de pruebas JUnit 4.
Las variables predefinidas toman los siguientes valores:
${NAME} El nombre del método que se creará.
${HTTP_METHOD} El método HTTP usado (por ejemplo, GET, POST).
${ROUTE} La ruta relativa al servicio actual que se debe probar.
${CONFIGURE_SERVER} Un método de extensión de ruta para configurar el árbol de rutas para la parte del servidor que se debe probar.
${BODY} El cuerpo del método generado.
