Se aplica a los nuevos controladores Spring creados invocando Nuevo | Componente Spring | Controlador en la ventana de herramienta Proyecto.
Esta plantilla integrada es editable. Además de expresiones y comentarios de Java, también puede usar variables predefinidas (enumeradas a continuación), que se expandirán posteriormente a sus valores correspondientes, como macros.

También puede especificar variables personalizadas. Las variables personalizadas usan el siguiente formato: ${VARIABLE_NAME}, donde VARIABLE_NAME es el nombre de la variable (por ejemplo, ${MY_CUSTOM_FUNCTION_NAME}). Antes de que el IDE cree un nuevo archivo usando variables personalizadas, verá un diálogo donde podrá definir los valores para las variables personalizadas en la plantilla.

Mediante el uso de la directiva #parse, puede incluir plantillas de la pestaña Incluir. Para incluir una plantilla, especifique el nombre completo de la plantilla como un parámetro entre comillas (por ejemplo, #parse("File Header.java")).
Las variables predefinidas toman los siguientes valores:
${PACKAGE_NAME} Nombre del paquete donde se crea la nueva clase
${NAME} El nombre de la nueva clase que especificó en el diálogo Crear nueva clase
${USER} Nombre de inicio de sesión del sistema del usuario actual
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${MONTH_NAME_SHORT} Las 3 primeras letras del nombre del mes actual (Ene, Feb, etc.)
${MONTH_NAME_FULL} El nombre completo del mes actual (Enero, Febrero, etc.)
${DAY} Día del mes
${HOUR} Hora actual
${MINUTE} Minuto actual
${PROJECT_NAME} Nombre del proyecto actual
