Se aplica a los nuevos Controladores REST creados invocando Nuevo | Componente Spring | Controlador REST en la ventana de herramientas Proyecto.
Esta plantilla incorporada es editable. Además de expresiones y comentarios Java, puede usar variables predefinidas (listadas a continuación), las cuales se expandirán como macros a los valores correspondientes.

También puede especificar variables personalizadas. Las variables personalizadas usan el formato: ${VARIABLE_NAME}, donde VARIABLE_NAME es el nombre de la variable (por ejemplo, ${MY_CUSTOM_FUNCTION_NAME}). Antes de que el IDE cree un nuevo archivo usando variables personalizadas, verá un diálogo donde podrá definir los valores para las variables personalizadas en la plantilla.

Usando la directiva #parse, puede incluir plantillas de la pestaña Incluir. Para incluir una plantilla, especifique su nombre completo como un parámetro entre comillas (por ejemplo, #parse("File Header.java")).
Las variables predefinidas aceptan los siguientes valores:
${PACKAGE_NAME} El nombre del paquete donde se crea la nueva clase
${NAME} El nombre de la nueva clase especificado por el usuario en el diálogo Crear nueva clase
${USER} El nombre de inicio de sesión del sistema del usuario actual
${DATE} La fecha del sistema actual
${TIME} La hora del sistema actual
${YEAR} El año actual
${MONTH} El mes actual
${MONTH_NAME_SHORT} Los 3 primeros caracteres del nombre del mes actual (Jan, Feb, etc.)
${MONTH_NAME_FULL} El nombre completo del mes actual (January, February, etc.).
${DAY} El día del mes actual
${HOUR} La hora actual
${MINUTE} Los minutos actuales
${PROJECT_NAME} El nombre del proyecto actual
