Se aplica a un nuevo Controlador Spring creado invocando Nuevo | Componente Spring | Controlador en la ventana de herramientas Proyecto.
Esta plantilla incorporada es editable. Además de expresiones y comentarios Java, puede usar variables predefinidas (listadas a continuación) que se expanden a los valores correspondientes como macros.

También puede especificar variables personalizadas. Las variables personalizadas usan el formato: ${NOMBRE_VARIABLE}, donde NOMBRE_VARIABLE es el nombre de la variable (p. ej., ${MI_NOMBRE_FUNCION_PERSONALIZADA}). Antes de que el IDE cree un nuevo archivo usando variables personalizadas, verá un diálogo donde puede definir los valores para las variables personalizadas en la plantilla.

Usando la directiva #parse, puede incluir plantillas de la pestaña Incluir. Para incluir una plantilla, especifique el nombre completo de la plantilla como parámetro entre comillas (p. ej., #parse("File Header.java")).
Las variables predefinidas toman los siguientes valores:
${PACKAGE_NAME} Nombre del paquete donde se crea la nueva clase
${NAME} Nombre de la nueva clase que especificó en el diálogo Crear nueva clase
${USER} Nombre de inicio de sesión del sistema del usuario actual
${DATE} Fecha actual del sistema
${TIME} Hora actual del sistema
${YEAR} Año actual
${MONTH} Mes actual
${MONTH_NAME_SHORT} Primeras 3 letras del nombre del mes actual (Ene, Feb, etc.)
${MONTH_NAME_FULL} Nombre completo del mes actual (Enero, Febrero, etc.)
${DAY} Día del mes actual
${HOUR} Hora actual
${MINUTE} Minuto actual
${PROJECT_NAME} Nombre del proyecto actual
