Aplicable a las nuevas configuraciones de Spring creadas invocando Nuevo | Componente de Spring | Configuración en la ventana de herramientas de Proyecto.
Esta plantilla integrada es editable. Además de expresiones y comentarios de Java, también puede usar variables predefinidas (listadas a continuación), las cuales se expanden posteriormente a los valores correspondientes, como las macros.

También puede especificar variables personalizadas. Las variables personalizadas usan el siguiente formato: ${NOMBRE_VARIABLE}, donde NOMBRE_VARIABLE es el nombre de la variable (por ejemplo, ${MI_NOMBRE_FUNCION_PERSONALIZADO}). Antes de que el IDE cree un nuevo archivo usando variables personalizadas, verá un diálogo donde puede definir los valores para las variables personalizadas en la plantilla.

Usando la directiva #parse, puede incluir plantillas desde la pestaña Incluir. Para incluir una plantilla, especifique el nombre completo de la plantilla como parámetro entre comillas (por ejemplo, #parse("File Header.java")).
Las variables predefinidas toman los siguientes valores:
${PACKAGE_NAME} el nombre del paquete donde se crea la nueva clase
${NAME} el nombre de la nueva clase que especifica en el diálogo Crear nueva clase
${USER} el nombre de inicio de sesión en el sistema del usuario actual
${DATE} la fecha actual del sistema
${TIME} la hora actual del sistema
${YEAR} el año actual
${MONTH} el mes actual
${MONTH_NAME_SHORT} las 3 primeras letras del nombre del mes actual (Jan, Feb, etc.)
${MONTH_NAME_FULL} el nombre completo del mes actual (January, February, etc.).
${DAY} el día del mes
${HOUR} la hora actual
${MINUTE} el minuto actual
${PROJECT_NAME} el nombre del proyecto actual
