Informa sobre las asignaciones en las que el valor asignado nunca se utiliza posteriormente en el código.

Ejemplo:


  fun foo(): Int = 42

  fun example() {
      var local = 0
      print(local)
      local = 42  // el valor asignado nunca se lee
  }
  • Después de la corrección rápida, se elimina la asignación:

  fun foo(): Int = 42

  fun example() {
      var local = 0
      print(local)
  }
  • Si la expresión del lado derecho (RHS) puede tener efectos secundarios, la corrección rápida también sugerirá mantener la expresión RHS para preservar dichos efectos secundarios: 

  fun foo(): Int = 42

  fun example() {
      var local = 0
      print(local)
      foo()  // esta llamada a la función se mantiene por sus efectos secundarios
  }
