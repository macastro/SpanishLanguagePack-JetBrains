Informa sobre las llamadas a map { it.await() } en una colección de Deferred.

Esto se puede reemplazar por una llamada a awaitAll() más concisa y potencialmente más eficiente.

Para obtener más información, consulta la documentación de kotlinx.coroutines.awaitAll.

Ejemplo:


  suspend fun waitForResult(results: List<Deferred<String>>): List<String> {
    return results.map { it.await() }
  }

Después de aplicar la corrección rápida:


  suspend fun waitForResult(results: List<Deferred<String>>): List<String> {
    return results.awaitAll()
  }
