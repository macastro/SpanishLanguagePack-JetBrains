Reporta todas las cadenas con prefijo de interpolación.

La corrección rápida de la inspección elimina el prefijo de interpolación, actualiza las entradas de interpolación y escapa los caracteres de dólar no seguros.

Ejemplo:


val foo = $$"""
{
    "foo": $ref,
    "bar": {
        "baz": "$20"
        "boo": "$${100 + 200}"
    }
}
"""

Después de aplicar la corrección rápida:


val foo = """
{
    "foo": ${'$'}ref,
    "bar": {
        "baz": "$20"
        "boo": "${100 + 200}"
    }
}
"""
