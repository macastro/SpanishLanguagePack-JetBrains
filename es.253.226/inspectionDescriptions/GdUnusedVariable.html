Informa sobre las variables y constantes de GDScript que han sido declaradas pero no se utilizan nunca.

Esta inspección ayuda a mantener los scripts limpios al detectar declaraciones de variables locales y constantes que no tienen usos dentro del ámbito del proyecto. Si existe una indicación de tipo pero hace referencia a un tipo no resuelto, la declaración se ignora para evitar falsos positivos.

Ejemplo:


var tmp = 42 # No utilizado, se informará
const PI2 = PI * 2 # No utilizado, se informará

var enemy: UnknownType # Si UnknownType no se puede resolver, se ignora

Acciones de corrección rápida:

  • Eliminar las variables o constantes no utilizadas.

Esta inspección no informará sobre declaraciones que estén referenciadas en cualquier lugar del proyecto. También omite las declaraciones con tipos no resueltos para evitar falsas alertas.
