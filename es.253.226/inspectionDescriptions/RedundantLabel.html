Informa sobre las etiquetas redundantes que no se pueden referenciar desde las sentencias 'break', 'continue' o 'return'.

Estas etiquetas se pueden eliminar de forma segura.

Ejemplo:


fun foo(c: Boolean) {
    b@ while (true) a@ {
        while (true) {
            break@b
        }
    }
}

Después de aplicar la corrección rápida:


fun foo(c: Boolean) {
    b@ while (true) {
        while (true) {
            break@b
        }
    }
}
