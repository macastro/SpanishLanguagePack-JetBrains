Informa de redundancias en la declaración del nombre de @PathVariable cuando este coincide con el nombre del parámetro. La corrección rápida permite eliminar el atributo de nombre explícito de la anotación.

Ejemplo:


  @GetMapping("/{orderId}")
  public Order getOrder(@PathVariable("orderId") Long orderId)
    return return orderRepository.findOne(orderId);
  }

Tras aplicar la corrección rápida, el resultado será el siguiente:


  @GetMapping("/{orderId}")
  public Order getOrder(@PathVariable Long orderId)
    return orderRepository.findOne(orderId);
  }
