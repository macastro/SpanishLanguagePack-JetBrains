Informa sobre el uso de kotlin.coroutines.suspendCoroutine y sugiere reemplazarlo por kotlinx.coroutines.suspendCancellableCoroutine cuando la dependencia kotlinx.coroutines esté presente.

La variante suspendCancellableCoroutine proporciona soporte de cancelación integrado, lo cual es importante para la concurrencia estructurada y una gestión de recursos adecuada.

Ejemplo:


    suspend fun usage(): String {
        return suspendCoroutine { continuation ->
            continuation.resume("Hello!")
        }
    }

Tras aplicar la corrección rápida:


    suspend fun usage(): String {
        return suspendCancellableCoroutine { continuation ->
            continuation.resume("Hello!")
        }
    }
