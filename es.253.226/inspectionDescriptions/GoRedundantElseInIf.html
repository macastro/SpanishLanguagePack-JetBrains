Reporta cláusulas else redundantes en sentencias if que contienen un return.

Ejemplo:


  if someCode() {
      fmt.Println("branch A")
      return nil
  } else {
      fmt.Println("branch B")
  }

Después de aplicar la corrección rápida:


  if someCode() {
      fmt.Println("branch A")
      return nil
  }
  fmt.Println("branch B")
