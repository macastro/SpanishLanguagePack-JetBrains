Informa sobre el uso de Thread.ofVirtual en lugar de IntelliJVirtualThreads.ofVirtual.

Thread.ofVirtual es una API genérica de Java para crear hilos virtuales que no es adecuada para las necesidades de la plataforma IntelliJ.

La corrección rápida sustituirá el uso por IntelliJVirtualThreads.ofVirtual.

Ejemplo: 


// Incorrecto:
Thread thread = Thread.ofVirtual().start();

// Correcto:
Thread thread = IntelliJVirtualThreads.ofVirtual().start();
