Reporta conversiones no seguras que utilizan return en asignaciones de variables o argumentos de valor que pueden reemplazarse por una conversión segura.

La corrección rápida reemplaza la conversión no segura por una conversión segura.

Ejemplo


    fun test(x: Any): Any? {
        val y = x as String ?: return null
    }

Después de aplicar la corrección rápida:


  fun test(x: Any): Any? {
      val y = x as? String ?: return null
  }
