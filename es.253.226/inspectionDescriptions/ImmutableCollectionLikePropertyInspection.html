Informa sobre el uso de colecciones inmutables en entidades JPA.

Detecta propiedades anotadas con @OneToMany o @ManyToMany con tipos de colección inmutables (List, Set, Map). Sugiere usar tipos de colección mutables (MutableList, MutableSet, MutableMap) en su lugar para garantizar la compatibilidad con Hibernate.

Fragmento de código: 


@Entity
open class User(
  @Id
  val id: Long = 0L,
  @OneToMany
  val orders: List = listOf(),
)
Después de aplicar la corrección rápida: 

@Entity
open class User(
  @Id
  val id: Long = 0L,
  @OneToMany
  val orders: MutableList = mutableListOf(),
)
