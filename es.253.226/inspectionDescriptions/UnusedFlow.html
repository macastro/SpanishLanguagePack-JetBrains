Informa sobre los Flows de Kotlinx Coroutines que se han creado pero nunca se consumen ni se pasan a otras funciones.

El código que define un Flow no se ejecutará a menos que el Flow se recolecte utilizando un operador terminal.

Ejemplo:


fun foo() {
    flowOf(1).onEach { println(it) }
}

Consulte algunos ejemplos de operadores terminales.
