Informa sobre el uso de la propiedad kotlin.coroutine.coroutineContext en código que depende de la biblioteca kotlinx.coroutines.

Cuando tanto kotlin.coroutine.coroutineContext como kotlinx.coroutines.CoroutineScope.coroutineContext pueden estar presentes en el código, se puede generar confusión y posibles errores.

La biblioteca kotlinx.coroutines proporciona la función currentCoroutineContext() como una alternativa más clara que se debería preferir, incluso cuando no hay un conflicto explícito.

Para obtener más información, consulta la documentación de kotlin.coroutine.coroutineContext y kotlinx.coroutines.currentCoroutineContext.

Ejemplo:


  suspend fun getCurrentJob(): Job? {
    return coroutineContext[Job]
  }

Tras aplicar la corrección rápida:


  suspend fun getCurrentJob(): Job? {
    return currentCoroutineContext()[Job]
  }
