Informa sobre expresiones no utilizadas, lo que suele indicar que el código está incompleto o que falta una instrucción return.

Si la expresión no utilizada es la última instrucción de una función y su tipo coincide con el tipo de retorno de la función, una corrección rápida añadirá automáticamente la palabra clave return.

Ejemplo:


fun example(): Int {
    val x = 10
    x // Advertencia: la expresión no se utiliza
}

Después de aplicar la corrección rápida:


fun example(): Int {
    val x = 10
    return x
}
