Informa sobre las instrucciones import no utilizadas en el código Python.

Esta inspección detecta las instrucciones import del código que no se utilizan y que pueden eliminarse de forma segura. Eliminar las importaciones no utilizadas ayuda a mantener el código limpio y reduce el riesgo de colisiones de nombres.

Ejemplo:


import os           # Importación no utilizada
import sys         # Importación utilizada
from math import pi # Importación no utilizada

print(sys.version)

La inspección proporciona una corrección rápida para optimizar las importaciones que elimina todas las instrucciones import no utilizadas.

Tenga en cuenta que algunas importaciones pueden usarse indirectamente (p. ej., por sus efectos secundarios) y no deberían eliminarse. Si ciertas importaciones son necesarias por sus efectos secundarios, puede suprimir esta inspección para ellas.
