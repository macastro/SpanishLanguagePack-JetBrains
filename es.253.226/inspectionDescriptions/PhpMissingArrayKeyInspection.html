Informa sobre los arrays que no proporcionan todas las claves o entradas obligatorias definidas en la forma del array.

Esta inspección ayuda a identificar escenarios en los que la instanciación de un array podría causar problemas en tiempo de ejecución debido a la falta de entradas obligatorias, garantizando una mejor consistencia del código y el cumplimiento de la estructura definida.

Fragmento de código embebido: 


      function connect(#[ArrayShape([
          'path' => 'string',
          'host' => [
              'name' => 'string',
              'port' => 'int'
          ]
      ])] array $options = []) { };
      connect(['host' => ['name' => 'localhost', 'port' => 1234]]); // Falta la clave 'path'
