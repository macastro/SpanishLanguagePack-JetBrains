Informa de las palabras clave return redundantes que aparecen en la última expresión de una rama if o when cuando se devuelve la propia expresión if/when completa.

Estas palabras clave no aportan ningún valor y solo ensucian el código.

La corrección rápida permite eliminar la palabra clave return redundante, dejando solo la expresión.

Ejemplo:


fun foo(flag: Boolean): String {
  return if (flag) {
    "foo"
  } else {
    return "bar"  // palabra clave 'return' redundante
  }
}

Después de aplicar la corrección rápida:


fun foo(flag: Boolean): String {
  return if (flag) {
    "foo"
  } else {
    "bar"
  }
}
