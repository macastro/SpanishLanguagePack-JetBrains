Informa sobre las expresiones lambda que no se invocan ni se asignan.

Las expresiones lambda que no se invocan ni se asignan a una variable no tienen ningún efecto en la ejecución del programa y es probable que no sean intencionadas.

Ejemplo:


  fun foo() {
    {} // Advertencia: la expresión lambda nunca se invoca
  }

Después de aplicar el arreglo rápido:


  fun foo() {
    run {} // La expresión lambda ahora se invoca mediante la función 'run'
  }
