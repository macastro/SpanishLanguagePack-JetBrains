Informa sobre las variables de ruta no utilizadas en el DSL de enrutador de Kotlin, así como los usos de variables de ruta no declaradas.

Las variables de ruta deben declararse en la ruta (path) antes de acceder a ellas. El intento de acceder a una variable de ruta no declarada provocará un error de tiempo de ejecución.

Ejemplo:


@Configuration
class RouterConfiguration {
    @Bean
    fun myRouter() = router {
        GET("/test/{var}") { ServerResponse.ok().body("${it.pathVariable("bar")}") }
    }
}

Después de aplicar la corrección:


@Configuration
class RouterConfiguration {
    @Bean
    fun myRouter() = router {
        GET("/test/{var}") { ServerResponse.ok().body("${it.pathVariable("var")}") }
    }
}
