Reporta llamadas a forEach { it.join() } en colecciones de Job.

Esto se puede reemplazar con una llamada a joinAll(), que es más concisa y completamente equivalente.

Para obtener más información, consulte la documentación de kotlinx.coroutines.joinAll.

Ejemplo:


  suspend fun waitForJobs(jobs: List<Job>) {
    jobs.forEach { it.join() }
  }

Después de aplicar la corrección rápida:


  suspend fun waitForJobs(jobs: List<Job>) {
    jobs.joinAll()
  }
