Informa sobre los métodos main de instancia declarados dentro de una clase anidada.

Groovy no procesa directamente las llamadas al método main dentro de una clase anidada; sin embargo, es posible delegando la ejecución a Java. En este caso, el código solo se ejecutará correctamente si la versión de Java es lo suficientemente reciente. Considere el siguiente ejemplo: 


class Outer {
  static class Nested {
    void main() {
      println 'Hello from nested'
    }
  }
}
El código se ejecuta correctamente en Java 25, pero falla en Java 17.
