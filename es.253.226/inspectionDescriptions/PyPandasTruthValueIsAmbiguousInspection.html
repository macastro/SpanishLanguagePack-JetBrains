Informa sobre el uso ambiguo de un DataFrame o Series de pandas en un contexto booleano (como if, while o expresiones lógicas). Esto suele provocar el siguiente error de tiempo de ejecución: ValueError: The truth value of a DataFrame is ambiguous.

En pandas, expresiones como df o df == other no devuelven un único valor booleano, sino un DataFrame o Series de valores booleanos. El uso de estas expresiones en el flujo de control sin una reducción explícita (por ejemplo, .any(), .all() o .empty) es ambiguo y provocará una excepción.

Ejemplo:

if df:  # ❌ Lanza ValueError: El valor de verdad de un DataFrame es ambiguo
    print("DataFrame exists")

if not df.empty:  # ✅ Comprueba si el DataFrame tiene alguna fila
    print("DataFrame exists")

Tras aplicar la corrección rápida, la condición se sustituirá por un reductor adecuado según el contexto, como .any(), .all() o .empty.
