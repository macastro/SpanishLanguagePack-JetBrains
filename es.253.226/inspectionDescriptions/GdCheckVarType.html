Informa sobre la falta de sugerencias de tipo explícitas en las declaraciones de variables y constantes de GDScript cuando el tipo es inferible, y marca el uso de tipos en asignaciones con :=.

Nota:

  • Sugiere añadir tipos cuando la expresión del inicializador tiene un tipo concreto que no es Variant ni null.
  • Ignora los tipos similares a recursos que no se pueden resolver con precisión.
  • La asignación := no se puede utilizar junto con sugerencias de tipo explícitas. Use = cuando el tipo esté especificado.

Ejemplo:


var hp = 100          # Sugerencia: var hp: int = 100
var hp: int := 100    # Error: la asignación ":=" no puede tener un tipo especificado
var hp: int = 100     # Correcto
