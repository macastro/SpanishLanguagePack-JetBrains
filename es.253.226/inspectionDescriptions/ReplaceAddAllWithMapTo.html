Informa sobre las llamadas a MutableCollection.addAll / += donde el argumento es una llamada a Iterable.map/filter.

Este código asigna un objeto List adicional que puede evitarse utilizando mapTo/filterTo.

La corrección rápida permite reemplazar la llamada por mapTo/filterTo.

Ejemplo:


    coll1.addAll(coll2.map { transform(it) })
    coll1 += coll2.map { transform(it) }

Tras aplicar la corrección rápida:


    coll2.mapTo(coll1) { transform(it) }
