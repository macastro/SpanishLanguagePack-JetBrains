Informa los tipos de parámetros especificados explícitamente en los establecedores de propiedades.

El tipo de parámetro del establecedor siempre coincide con el tipo de propiedad, por lo que no es necesario que sea explícito. La acción rápida 'Quitar especificación de tipo explícito' permite modificar el código en conformidad.

Ejemplos:


  fun process(x: Int) {}

  var x: Int = 0
      set(value: Int) = process(value) // <== Se puede omitir de forma segura la especificación 'Int'

Después de aplicar la acción rápida:


  fun process(x: Int) {}

  var x: Int = 0
      set(value) = process(value)
