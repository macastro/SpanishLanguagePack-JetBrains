Informar de cualquier método de prueba de JUnit declarado como static, no declarado como public, que no devuelva void o que tenga parámetros formales. Estos métodos de prueba pueden crearse fácilmente por error, pero el ejecutor de pruebas de JUnit no los ejecutará.

Ejemplo:


  @Test
  void test() {
    assertEquals(1, 1);
  }

Después de aplicar la corrección rápida:


  @Test
  public void test() {
    assertEquals(1, 1);
  }
Puede especificar anotaciones mediante las opciones de la inspección. No se informará de parámetros formales que tengan anotaciones con una de estas anotaciones.
