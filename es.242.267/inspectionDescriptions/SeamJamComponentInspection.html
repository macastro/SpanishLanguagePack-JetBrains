Informes de problemas de configuración de las clases de Seam anotadas

Ejemplo:


@Nombre ("authenticator")
clase SeanComponent {
  @Crear // No se permite el uso de métodos estáticos
  estático privado anular crear() {...}

   @Destruir // No se permite el uso de métodos estáticos
   estático privado anular destruir() {...}

   @ModeloDatos // No es getter
   mapa método privado() {...}

   @Fábrica // Debe ser público
   mapa privado obtenerUniverso2() {...}

   @Desenvolver // No debe tener parámetros de forma
   mapa estático privado crearUniverso (int días) { ... }
}
