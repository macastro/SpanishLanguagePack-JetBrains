Reporta parámetros de métodos de recursos (anotados con @PathParam, @QueryParam, etc.) con tipos no soportados.

El tipo del parámetro, campo o propiedad anotado debe cumplir con uno de los siguientes requisitos:

  1. Ser un tipo primitivo
  2. Tener un constructor que acepte un único argumento String
  3. Tener un método estático llamado valueOf() o fromString() que acepte un único argumento String, por ejemplo, Integer.valueOf(String).
  4. Tener una implementación registrada de ParamConverterProvider JAX-RS SPI de extensión que devuelva una instancia de ParamConverter capaz de convertir una cadena a un tipo particular.
  5. Ser List<T>, Set<T> o SortedSet<T>, donde T cumple 2, 3 o 4 arriba. La colección resultante es de solo lectura.
