Esta inspección identifica llamadas a métodos donde un argumento que potencialmente podría ser null se pasa a un parámetro que no está anotado con @Nullable, ayudando a detectar posibles NullPointerExceptions antes de que ocurran durante la ejecución.

Esta inspección solo se muestra para métodos específicos que están en el código fuente del proyecto.

Ejemplo:


def test(@Nullable greeting: String): Unit = {
  sayHello(greeting)
}

def sayHello(greeting: String): Unit = {
  println(greeting.toLowerCase)
}
