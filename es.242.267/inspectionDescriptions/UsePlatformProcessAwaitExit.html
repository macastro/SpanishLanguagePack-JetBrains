Reporta el uso de Process.waitFor() y Process.onExit() dentro de un contexto de corrutina. El uso de estos métodos está prohibido dentro de un contexto de corrutina.

Usa com.intellij.util.io.awaitExit() en su lugar.

Ejemplo:


suspend fun doSomething(process: Process) {
  val exitCode = process.waitFor() // Incorrecto
  // ...
}

suspend fun doSomething(process: Process) {
  val exitCode = process.awaitExit() // Correcto
  // ...
}
