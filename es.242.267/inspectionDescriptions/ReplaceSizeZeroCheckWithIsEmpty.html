Informe las comprobaciones de size == 0 de Collections/Array/String que deberían reemplazarse con isEmpty().

El uso de isEmpty() simplifica el código.

El arreglo rápido reemplazará las comprobaciones de tamaño por isEmpty().

Ejemplo:


  fun foo() {
      val arrayOf = arrayOf(1, 2, 3)
      arrayOf.size == 0
  }

Después de aplicar el arreglo rápido:


  fun foo() {
      val arrayOf = arrayOf(1, 2, 3)
      arrayOf.isEmpty()
  }
