Informa sobre concatenaciones de String que pueden simplificarse reemplazándolas por una plantilla de cadena.

Ejemplo:


  String name = "Bob";
  String greeting = "Hola, " + name + ". Tienes " + 29 + " años.";

Después de aplicar la corrección rápida:


  String name = "Bob";
  String greeting = STR."Hola, \{name}. Tienes 29 años.";
