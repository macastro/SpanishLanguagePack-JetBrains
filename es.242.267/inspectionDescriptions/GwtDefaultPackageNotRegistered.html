Reporta un descriptor de módulo GWT que no registra el directorio predeterminado client o public, pero sí registra un directorio fuente o público personalizado.

GWT solo trata implícitamente a la carpeta client como una ruta de origen del lado del cliente si el archivo gwt.xml no tiene las etiquetas source o super-source. Por lo tanto, es probable que sea un error si el directorio client existe pero no está registrado en el archivo gwt.xml y ese archivo gwt.xml declara una etiqueta source o super-source personalizada.

De manera similar, también puede ser un error si el archivo gwt.xml declara un directorio public personalizado, pero no registra una carpeta existente con el nombre predeterminado.

Arreglo rápido agrega un registro explícito para la carpeta predeterminada en el archivo gwt.xml.
