Se informa de clases de pruebas JUnit que no pueden construirse por el ejecutor de pruebas JUnit estándar.

Las clases de prueba JUnit 4 deben ser public e incluir un constructor public sin argumentos o no incluir constructor alguno (constructor predeterminado implícito), y no deben tener otros constructores public. Las clases de prueba Junit 3 deben ser public y requieren un constructor public sin argumentos o un constructor public con un único parámetro de tipo String (que llame al superconstructor coincidente). De lo contrario, las clases de prueba no pueden ser ejecutadas por el ejecutor de pruebas JUnit estándar.

Ejemplo:


public class MyTest {

  private MyTest() {} // el constructor sin argumentos es private

  @Test
  public void testSomething() {
    assertEquals(1, 1);
  }
}
