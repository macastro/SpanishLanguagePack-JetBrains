Reporta llamados a métodos estáticos donde el llamado es calificado por una subclase de la clase declarada, en lugar de por la propia clase declarada.

Java permite tal calificación para clases, pero tales llamadas pueden indicar una sutil confusión de herencia y anulación.

Ejemplo:


  class Padre {
    public static void print(String str) {}
  }
  class Hijo extends Padre {}

  Hijo.print("Hello, world!");

Después de aplicar la solución rápida:


  Padre.print("Hello, world!");
