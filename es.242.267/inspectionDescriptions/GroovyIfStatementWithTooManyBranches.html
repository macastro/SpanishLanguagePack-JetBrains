Reporta sentencias if con demasiadas ramas. Dichas sentencias pueden resultar confusas y a menudo son el síntoma de niveles inadecuados de abstracción de diseño.

Ejemplo:


if (a) {
  print "foo"
} else if (b) {
  print "bar"
} else if (c) {
  print "baz"
} else if (d) {
  print "Demasiadas ramas"
}

Utiliza el campo Número máximo de ramas para especificar el número máximo de ramas esperado.
