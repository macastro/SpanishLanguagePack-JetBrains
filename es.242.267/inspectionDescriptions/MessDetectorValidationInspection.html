Informa los problemas de estilo de código detectados por PHP Mess Detector.

La inspección requiere que PHP Mess Detector esté correctamente instalado y configurado en el IDE en Configuración | PHP | Herramientas de calidad | Mess Detector.

Para obtener más información sobre la instalación de PHP Mess Detector, consulta Instalación de PHP Mess Detector (phpmd.org) para obtener más detalles.

Puedes utilizar un conjunto de reglas predefinido o agregar el tuyo especificando archivos válidos ruleset.xml. Consulta el índice de reglas (phpmd.org) y Cómo crear un conjunto de reglas personalizado (phpmd.org) para obtener más detalles.
