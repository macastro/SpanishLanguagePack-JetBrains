Esta inspección informa de clases de caso de prueba de Arquillian que definen varios despliegues.

Según la Documentación de Arquillian, las clases de prueba de Arquillian deberían definir los archivos de despliegue utilizando métodos public static con una signatura determinada y anotarlos con @Deployment. Cada caso de prueba debería tener un único despliegue, contando también los métodos definidos en las superclases del caso de prueba.
