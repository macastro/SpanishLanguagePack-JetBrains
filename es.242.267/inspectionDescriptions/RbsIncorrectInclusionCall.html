Reporta llamadas de inclusión de referencias de clase (include, prepend y extend). Las llamadas de inclusión pueden hacer referencia sólo a módulos o interfaces.

Puedes usar una corrección rápida para eliminar la llamada de inclusión incorrecta.

Ejemplo:

class Example
  # se usa la clase 'Object' en la llamada 'include'
  include Object
end

Después de aplicar la corrección rápida:

class Example
end
