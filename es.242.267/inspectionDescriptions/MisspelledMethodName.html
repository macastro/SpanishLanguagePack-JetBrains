Informa de los casos en los que varios métodos de una clase tienen nombres que sólo difieren en el uso de mayúsculas o minúsculas. Estos nombres pueden resultar muy confusos.

Ejemplo: 


  public int hashcode() { // informado; ¿debería ser hashCode probablemente?
    return 0;
  }

El editor es el único que puede utilizar la corrección rápida para renombrar estos métodos.

Utiliza la opción Ignorar métodos que reemplacen o implementen un método de superclase para ignorar los métodos que reemplacen o implementen un método de la superclase.
