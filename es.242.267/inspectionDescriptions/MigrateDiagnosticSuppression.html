Inspecciona las supresiones con viejos nombres de diagnósticos, como @Suppress("HEADER_WITHOUT_IMPLEMENTATION").

Algunos de los diagnósticos de Kotlin 1.2 y anteriores son ahora obsoletos, haciendo que dichas supresiones sean redundantes.

Ejemplo:


@Suppress("HEADER_DECLARATION_WITH_BODY")
expect fun connection() {
  // ...
}

Después de aplicar la corrección rápida:


@Suppress("EXPECTED_DECLARATION_WITH_BODY")
expect fun connection() {
  // ...
}
