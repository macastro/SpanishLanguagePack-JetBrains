Informa sobre operaciones no seguras con this en la construcción de objetos, incluidas:
  • Acceder a una propiedad no final durante la inicialización de la clase: desde un constructor o inicialización de propiedad
  • Llamar a una función no final durante la inicialización de la clase
  • Usar this como argumento de función en un constructor de una clase no final

Si otras clases heredan de la clase dada, es posible que no estén completamente inicializadas en el momento en que se realiza una operación no segura.

Ejemplo:


  abstract class Base {
      val code = calculate()
      abstract fun calculate(): Int
  }

  class Derived(private val x: Int) : Base() {
      override fun calculate() = x
  }

  fun testIt() {
      println(Derived(42).code) // Esperado: 42, real: 0
  }
