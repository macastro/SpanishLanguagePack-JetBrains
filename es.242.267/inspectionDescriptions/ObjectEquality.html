Reporta código que usa == o != en lugar de equals() para probar la igualdad entre objetos.

Comparar objetos usando == o != suele ser un error, ya que compara objetos por identidad en vez de igualdad. No se reportan comparaciones con null.

Comparaciones de arreglos, String y Number son reportadas por inspecciones separadas.

Ejemplo:

  if (list1 == list2) {
    return;
  }

Después de aplicar la corrección rápida:

  if (Objects.equals(list1, list2)) {
    return;
  }

Usa las configuraciones de inspección para configurar excepciones para esta inspección.
