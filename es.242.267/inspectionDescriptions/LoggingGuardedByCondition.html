Reporte de llamadas de registro rodeadas de condiciones de guarda. Esta inspección puede usarse para ajustar estilos de código personalizados.

Ejemplo:


  public class TestObject {
    void test(Object object) {
      if(LOG.isDebugEnabled()){
        LOG.debug("some logging " + expensiveCalculation(1));
      }
    }
  }

Después de aplicar una corrección rápida:


  public class TestObject {
    void test(Object object) {
      LOG.debug("some logging " + expensiveCalculation(1));
    }
  }

Esta inspección soporta los frameworks de registro Log4j2 y SLF4J (excepto los constructores).
