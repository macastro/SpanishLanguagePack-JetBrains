Reporta una expresión de máscara de bits que seguro se evalúa en true o false. Las expresiones tienen la forma (var & constante1) == constante2 ó (var | constante1) == constante2, donde constante1 y constante2 son constantes de máscara de bits incompatibles.

Ejemplo:


// Máscara incompatible: como el último byte de la máscara es cero,
// algo como 0x1200 sería posible, pero no 0x1234
if ((mask & 0xFF00) == 0x1234) {...}
