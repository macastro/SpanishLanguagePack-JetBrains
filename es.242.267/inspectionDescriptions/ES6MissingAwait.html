Informa una llamada a función async sin un prefijo await esperado dentro de una función async. Tal llamada devuelve una Promise y el flujo de control continúa inmediatamente.

Ejemplo:


async function bar() { /* ... */ }
async function foo() {
    bar(); // malo
}

Tras aplicar la corrección rápida, se añade el prefijo await:


async function bar() { /* ... */ }
async function foo() {
    await bar(); // bueno
}

Cuando está seleccionada la casilla de verificación "Informe de promesas en instrucciones return", también sugiere añadir await en las instrucciones return.
Aunque generalmente no es necesario, aporta dos ventajas principales.
  • No olvidarás añadir await cuando rodees tu código con try-catch.
  • Un await explícito ayuda al tiempo de ejecución de V8 a proporcionar seguimientos de pila asincrónicos.
