Informa sobre el uso de tipos dinámicos (instance, class y self) dentro de contextos estáticos. El uso de tipos dinámicos en contextos estáticos puede resultar confuso, ya que el tipo nunca cambia.

Puedes utilizar la corrección rápida para reemplazar el uso de tipos dinámicos a sus equivalentes estáticos.

Ejemplo:

class Example
  # 'instance' se utiliza en un contexto estático
  INSTANCE: instance
  # 'class' se utiliza en un contexto estático
  CLASS: class
end

Después de aplicar la corrección rápida:

class Example
  INSTANCE: Example
  CLASS: singleton(Example)
end
