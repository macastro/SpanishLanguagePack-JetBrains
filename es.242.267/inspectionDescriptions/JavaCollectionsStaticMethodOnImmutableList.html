Reporta llamadas a métodos mutadores (como `fill`, `reverse`, `shuffle`, `sort`) de Java en una colección inmutable de Kotlin.

Esto puede producir una `UnsupportedOperationException` en tiempo de ejecución.

Ejemplo:


  import java.util.Collections

  fun test() {
      val listaInmutable = listOf(1, 2)
      Collections.reverse(listaInmutable)
  }

Para solucionar el problema, haz que la lista sea mutable.
