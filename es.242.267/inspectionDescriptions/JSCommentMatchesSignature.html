Informa de la falta de coincidencia entre los nombres y el número de parámetros en un comentario JSDoc y los parámetros reales de una función. Sugiere actualizar los parámetros en el comentario JSDoc.

Ejemplo:


/**
 * @param height Altura en píxeles
 */
function sq(altura, width) {} // width no está documentado

Tras aplicar la solución rápida:


/**
 * @param height Altura en píxeles
 * @param width
 */
function sq(altura, width) {}
