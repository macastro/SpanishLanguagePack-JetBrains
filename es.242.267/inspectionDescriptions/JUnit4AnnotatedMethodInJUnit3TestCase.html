Inspecciona métodos anotados con JUnit 4 @Test en el heredero junit.framework.TestCase. Mezclar API de JUnit puede causar problemas al ejecutar pruebas.

Ejemplo:


  public class MyTest extends TestCase {
      @Test //El nombre no comienza con "test", por lo que se ignorará
      public void wouldBeIgnored() {}
  
      @Test   //El nombre comienza con "test"
      @Ignore //Por lo tanto, se ejecutará incluso con la anotación @Ignore
      public void testWouldBeExecuted() {}
  }

Correcciones proporcionadas:

  • Elimine la anotación @Ignore y cambie el nombre del método de prueba para que no comience con "test".
  • Convierta la clase de prueba JUnit 3 a JUnit 4.
