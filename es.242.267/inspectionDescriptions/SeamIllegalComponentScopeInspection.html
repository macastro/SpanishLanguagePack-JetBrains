Informa de anotaciones @Scope, @Role y @DataModel que especifican un ámbito no compatible para un componente determinado.

La siguiente tabla enumera los ámbitos compatibles para cada tipo de componente (los ámbitos predeterminados están en **negrita**):

Bean de sesión sin estado Bean de sesión con estado Bean de entidad Bean Java
STATELESS X x
EVENT x x x
PAGE x x
CONVERSATION X X X
SESSION x x x
BUSINESS_PROCESS x x x
APPLICATION x x x

Las variables de contexto @DataModel tienen diferentes restricciones en su ámbito: si el ámbito no se especifica, el ámbito es el mismo que el ámbito del componente envolvente (EVENT si el componente está en ámbito STATELESS). Alternativamente, se puede especificar un ámbito PAGE para las variables de contexto @DataModel.
