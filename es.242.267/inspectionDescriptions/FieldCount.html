Informa sobre clases que sobrepasan el número máximo de campos especificado.

Las clases que contienen una gran cantidad de campos suelen estar intentando realizar demasiados procesos. Intenta dividir estas clases en varias clases más pequeñas.

Configuración de la inspección:

  • Utiliza el campo Límite de campos para especificar el número máximo de campos permitidos en una clase.
  • Utiliza la opción Incluir campos constantes en el recuento para especificar si se deben contar los campos constantes.
  • Por defecto, solo los objetos static final inmutables se cuentan como constantes. Utiliza la opción Contar campos 'static final' como constantes para contar cualquier campo static final como constante.
  • Utiliza la opción Incluir constantes de enumeración en el recuento para especificar si se deben contar las constantes de enum en las clases enum.
