Reporta usos de primitivas funcionales de Guava que pueden migrarse a llamadas de API de Java estándar.

Por ejemplo, la inspección reporta usos de clases e interfaces como FluentIterable, Optional, Function, Predicate o Supplier.

Ejemplo:


  ImmutableList<String> results = FluentIterable.from(List.of(1, 2, 3)).transform(Object::toString).toList();

Después de aplicar corrección rápida:


  List<String> results = List.of(1, 2, 3).stream().map(Object::toString).collect(Collectors.toList());

La corrección rápida podría cambiar las semánticas. Algunos iterables de Guava evaluados perezosamente pueden transformarse en evaluados con entusiasmo.
