Reporta literales de objeto anónimos que implementan una interfaz Java con un solo método abstracto que puede ser convertido en una llamada con una expresión lambda.

Ejemplo:


class SomeService {
  val threadPool = Executors.newCachedThreadPool()
    
  fun foo() {
    threadPool.submit(object : Runnable {
      override fun run() {
        println("hola")
      }
    })
  }
}

Después de aplicar la corrección rápida:


  fun foo() {
    threadPool.submit { println("hola") }
  }
