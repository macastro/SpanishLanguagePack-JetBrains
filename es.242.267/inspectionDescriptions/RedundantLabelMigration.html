A partir de Kotlin 1.4 informa de etiquetas redundantes que causan errores de compilación.

Desde Kotlin 1.0, cualquier instrucción puede etiquetarse con una etiqueta:


  fun foo() {
    L1@ val x = L2@bar()
  }
Pero dichas etiquetas solo pueden ser referenciadas de pocas formas específicas:
  • Interrupción/continuación de un bucle
  • Retorno no local desde una lambda en línea o una función anónima en línea
    • sssss

Dichas etiquetas están prohibidas a partir de Kotlin 1.4.

Esta inspección solo informa si el nivel de lenguaje de Kotlin del proyecto o módulo es 1.4 o superior.
