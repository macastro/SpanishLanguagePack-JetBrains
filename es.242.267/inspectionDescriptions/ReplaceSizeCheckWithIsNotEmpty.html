Inspecciona comprobaciones de tamaño de Collections/Array/String que deberían reemplazarse con isNotEmpty().

Usar isNotEmpty() simplifica tu código.

El arreglo rápido reemplaza la comprobación de tamaño con isNotEmpty().

Ejemplo:


  fun foo() {
      val arrayOf = arrayOf(1, 2, 3)
      arrayOf.size > 0
  }

Después de aplicar el arreglo rápido:


  fun foo() {
      val arrayOf = arrayOf(1, 2, 3)
      arrayOf.isNotEmpty()
  }
