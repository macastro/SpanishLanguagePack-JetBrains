Los reportes están habilitados para registrar condiciones de sentencias if que no coinciden con el nivel de registro de la llamada de registro contenida.

Por ejemplo: 


  if (LOG.isTraceEnabled()) {
    // se registró nivel de depuración, pero se verificó nivel de seguimiento
    LOG.debug("some log message");
  }

Esta inspección entiende los frameworks de registro java.util.logging, Log4j, Log4j2, Apache Commons Logging y SLF4J.
