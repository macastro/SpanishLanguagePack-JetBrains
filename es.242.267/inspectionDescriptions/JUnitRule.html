Informa sobre el uso incorrecto de las anotaciones @Rule o @ClassRule. No se pueden ejecutar las pruebas que tienen una regla cuyo formato es incorrecto.
  • Los miembros anotados con @Rule deben ser public y de tipo secundario de org.junit.rules.TestRule o org.junit.rules.MethodRule
  • Los miembros anotados con @ClassRule deben ser public y static y de tipo secundario de org.junit.rules.TestRule

Ejemplo:


  public class MomentousTest {
    @Rule
    private final TemporaryFolder tempFolder =
      new TemporaryFolder();

    // ... Las pruebas pasan por aquí
  }

El arreglo rápido corrige los modificadores:


  public class MomentousTest {
    @Rule
    public final TemporaryFolder tempFolder =
      new TemporaryFolder();

    // ... Las pruebas pasan por aquí
  }
