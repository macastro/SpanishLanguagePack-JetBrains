Reporta una comprobación de igualdad con Float.NaN o Double.NaN que debería reemplazarse con una comprobación de isNaN().

De acuerdo con IEEE 754, la comprobación de igualdad contra NaN siempre retorna false, incluso para NaN == NaN. Por lo tanto, es probable que tal comprobación sea un error.

La corrección rápida reemplaza la comparación con la comprobación isNaN() que usa una técnica de comparación diferente y maneja correctamente los valores NaN.

Ejemplo:


  fun check(value: Double): Boolean {
      return Double.NaN == value
  }

Después de aplicar la corrección:


  fun check(value: Double): Boolean {
      return value.isNaN()
  }
