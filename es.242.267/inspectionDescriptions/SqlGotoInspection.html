Reporta usos de sentencias GOTO hacia atrás y sentencias GOTO utilizadas para salir de un ciclo.

El uso extensivo de sentencias GOTO generalmente no se recomienda. Para más información, consulte Sentencia GOTO en procedimientos SQL en ibm.com.

En lugar de retroceder a una sentencia previa usando GOTO, considere usar un ciclo.

En lugar de salir del ciclo WHILE con GOTO, considere utilizar otras sentencias de control de flujo (por ejemplo, RETURN o BREAK).

Ejemplo (Oracle):

CREATE PROCEDURE test(n INT) AS
DECLARE
    x INT;
BEGIN
    x := 0;
    GOTO a;
    <<a>> x := 1;
    IF (n = 0) THEN
        GOTO a;
    END IF;
    WHILE TRUE
        LOOP
            GOTO b;
        END LOOP;
    <<b>> x := 3;
END;
