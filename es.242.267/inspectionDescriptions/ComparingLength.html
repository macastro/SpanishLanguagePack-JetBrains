Informa sobre la comparación .length o .size y sugiere reemplazarla por .sizeIs.

Ejemplo:


  var seq: Seq[Int]
  seq.length == n
  seq.length != n
  seq.length < n
  seq.length > n

  seq.size == n
  seq.size != n
  seq.size < n
  seq.size > n

Tras aplicar la corrección rápida:


  var seq: Seq[Int]
  seq.sizeIs == n
  seq.sizeIs != n
  seq.sizeIs < n
  seq.sizeIs > n

  seq.sizeIs == n
  seq.sizeIs != n
  seq.sizeIs < n
  seq.sizeIs > n
