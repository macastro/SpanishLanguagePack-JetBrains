Reporta llamadas a System.out.println() con una excepción como argumento.

El uso de sentencias de impresión para registrar excepciones oculta el seguimiento de la pila, lo que puede complicar la investigación del problema. Se recomienda usar un registrador en su lugar.

Las llamadas a System.out.print(), System.err.println() y System.err.print() con un argumento de excepción también son reportadas. Es mejor usar un registrador para registrar excepciones en su lugar.

Por ejemplo, en lugar de:


  try {
      foo();
  } catch (Exception e) {
      System.out.println(e);
  }

use el siguiente código:


  try {
      foo();
  } catch (Exception e) {
      logger.warn(e); // la llamada al registrador puede ser diferente
  }
