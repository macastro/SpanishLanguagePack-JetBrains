Reporta problemas con los argumentos de property() y funciones anotadas con @property.


class C:
    @property
    def abc(self):  # Getter debería retornar o generar algo
        pass

    @abc.setter
    def foo(self, value):  # Los nombres de la función y el decorador no coinciden
        pass

    @abc.setter
    def abc(self, v1, v2):  # La firma de Setter debe ser (self, value)
        pass

    @abc.deleter
    def abc(self, v1):  # La firma de Delete debe ser (self)
        pass

Una solución rápida ofrece actualizar los parámetros.
