Informa de problemas esenciales para este archivo (por ejemplo, errores de sintaxis) en el resultado de una ejecución de inspección de código por lotes. Estos problemas suelen estar siempre resaltados en el editor y no se pueden configurar, a diferencia de las inspecciones. Estas opciones controlan el ámbito de las comprobaciones realizadas por esta inspección:
  • Opción "Informar de errores de sintaxis": informar de problemas relacionados con el analizador.
  • Opción "Informar de problemas de anotadores específicos del lenguaje": informar de problemas encontrados por anotadores configurados para el idioma correspondiente. Consulta Soporte para idiomas personalizados: anotadores para obtener más información.
  • Opción "Informar de otros problemas de resaltado": informar de problemas específicos del idioma del archivo actual (por ejemplo, errores de implementación o excepciones no notificadas). Consulta Soporte para idiomas personalizados: resaltado para obtener más información.
