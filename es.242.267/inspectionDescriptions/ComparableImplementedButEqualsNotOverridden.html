java.lang.Comparable está implementado pero no se ha sobrescrito equals().

Si no se sobrescribe equals(), la implementación de equals() no coincide con la implementación de compareTo(). Si se añaden objetos de esta clase a una colección como java.util.SortedSet, esta colección infringe el contrato de java.util.Set definido en el término equals().

Ejemplo:


  class Longitud implements Comparable< Longitud > {
    private int cm = 0;

    @Override
    public int compareTo(@NotNull Longitud o) {
      if (cm == o.cm) {
        return 0;
      }
      return cm < o.cm ? -1 : 1;
    }
  }

Después de aplicar la corrección rápida:


  class Longitud implements Comparable< Longitud > {
    private int cm = 0;

    @Override
    public int compareTo(@NotNull Longitud o) {
      if (cm == o.cm) {
        return 0;
      }
      return cm < o.cm ? -1 : 1;
    }

    @Override
    public boolean equals(Object o) {
      return o instanceof Longitud && compareTo((Longitud) o) == 0;
    }
  }
