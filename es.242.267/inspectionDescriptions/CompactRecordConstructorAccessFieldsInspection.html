Estas invocaciones parecen sospechosas porque los campos del registro pueden no estar inicializados en el momento de estas invocaciones.

Ejemplo:


public record MyRecord(String name, int id) {
    public MyRecord {
        validateMyFields(); // Parece sospechoso
    }

        if (this.name.isEmpty()) {
            throw new IllegalArgumentException();
        }
    }
}
