Reporta etiquetas personalizadas inexistentes.

Puede crear etiquetas personalizadas para su aplicación usando archivos de plantilla simples que se almacenan en el directorio app/views/tags. El nombre de archivo de la plantilla se utiliza como nombre de la etiqueta. Por ejemplo, para crear una etiqueta hola, simplemente cree el archivo app/views/tags/hola.html. Para hacer referencia a la etiqueta directamente: #{hola /}.

Esta inspección reportará la referencia #{hola /} si app/views/tags/hola.html no existe.
