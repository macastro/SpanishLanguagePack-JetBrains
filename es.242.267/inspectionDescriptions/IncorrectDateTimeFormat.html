Reporta patrones de formato de fecha y hora no válidos.

Reporta los siguientes errores:

  • Caracteres de patrón no soportados como "TT"
  • Uso de caracteres reservados como "#"
  • Relleno usado incorrectamente
  • Corchetes no balanceados
  • Cantidad incorrecta de caracteres de patrón consecutivos

Ejemplos:


  DateTimeFormatter.ofPattern("[][]]"); // ']' de cierre sin '[' de apertura
  DateTimeFormatter.ofPattern("TT"); // Carácter de patrón ilegal 'T'
  DateTimeFormatter.ofPattern("{"); // Se usa el carácter reservado '{'
  DateTimeFormatter.ofPattern("MMMMMM"); // Demasiados caracteres de patrón 'M' consecutivos
