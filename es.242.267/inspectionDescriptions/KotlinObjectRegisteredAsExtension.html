Inspecciona los objetos Kotlin registrados como extensiones de plugins.

El ciclo de vida de la extensión lo administra la plataforma IntelliJ. Utilizar objetos Kotlin para el registro de extensiones puede dar lugar a la creación de instancias de extensión innecesarias y a que el plugin no pueda desinstalarse.

Ejemplo

Registro de extensión:


<annotator language="myLang" implementationClass="com.example.MyAnnotator"/>

Implementación de extensión:


// Incorrecto:
object MyAnnotator : Annotator {
  ...
}

// Correcto:
class MyAnnotator : Annotator {
  ...
}
