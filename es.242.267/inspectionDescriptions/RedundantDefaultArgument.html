Informa sobre los argumentos que tienen el mismo valor que los predeterminados.

La corrección rápida eliminará las conversiones redundantes.

Ejemplo:


  def f(x: Int, y: Int = 0, z: Int) {}
  f(1, y = 0, z = 1)

Después de aplicar la corrección rápida:


  def f(x: Int, y: Int = 0, z: Int) {}
  f(1, z = 1)
