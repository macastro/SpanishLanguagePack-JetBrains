Reporta cualquier archivo package.html que se use para documentar paquetes.

Desde JDK 1.5, se recomienda que use archivos package-info.java en su lugar, ya que dichos archivos también pueden contener anotaciones de paquetes. De esta manera, package-info.java se convierte en un único repositorio para anotaciones y documentación a nivel de paquete.

Ejemplo: package.html


<html>
  <body>
    Documentation example.
  </body>
</html>

Después de aplicar la solución rápida: package-info.java


/**
 * Documentation example.
 */
package com.sample;
