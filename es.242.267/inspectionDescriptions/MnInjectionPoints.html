Reporta puntos de inyección en clases de bean que tienen dependencias faltantes o ambiguas.

Ejemplo:


  interface Engine {}

  @Singleton
  class GasEngine implements Engine {}

  @Singleton
  class ElectricEngine implements Engine {}

  @Singleton
  class Car {

    @Inject
    Engine engine; // Dependencia ambigua: hay múltiples beans que coinciden con el punto de inyección
  }
