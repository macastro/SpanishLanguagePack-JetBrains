Reporta llamadas en las que los argumentos posicionales van después de los nombrados. Funciona en PostgreSQL, Oracle y Db2.

Ejemplo (En PostgreSQL):

CREATE FUNCTION foo(a int, b int, c int) RETURNS int
    LANGUAGE plpgsql AS
$$
BEGIN
    RETURN a + b + c;
END
$$;
SELECT foo(a => 1, b => 2, c => 3);
  -- `3` va después del argumento nombrado
SELECT foo(1, b => 2, 3);
  -- `1` y `3` van después del argumento nombrado
SELECT foo(b => 2, 1, 3);
