in_array($needle, $haystack, $strict) la llamada se desenvuelve en una disyunción correspondiente.

La disyunción resultante se envuelve en paréntesis si es parte de una expresión lógica. Si $haystack contiene un elemento con efectos secundarios, se incluye en la primera expresión de igualdad. No se sugiere esta intención si $haystack contiene múltiples elementos con efectos secundarios.
