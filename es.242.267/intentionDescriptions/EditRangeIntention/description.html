Puede agregar una anotación @Range o editar su valor. Las anotaciones @Range se pueden agregar a parámetros de método, campos o valores de retorno de método y herramientas de análisis estático (por ejemplo, inspecciones) las utilizan para validar el código.

Se puede especificar un rango para un valor de tipo long, int, short, char o byte . Un rango se define especificando el límite inferior (inclusive) from y el límite superior (inclusive) to.
