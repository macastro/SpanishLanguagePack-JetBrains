Genera un código obligatorio para tratar errores habituales.

Puede llamarse sobre variables y expresiones de tipo error.

  • Reemplaza la variable err con la siguiente construcción:

    if err != nil {
    return value1, value2, ...
  }

  • Reemplaza la expresión expr con la siguiente construcción:

    if err := expr; err != nil {
    return value1, value2, ...
  }

Cada valor puede ser la variable err o el valor por defecto correspondiente al tipo de retorno de la función envolvente. Es posible llamar a esta plantilla sobre una expresión con múltiples valores.
