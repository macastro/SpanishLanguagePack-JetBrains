Plantilla integrada para una prueba unitaria de Codeception. Esta plantilla se utiliza cada vez que se crea una nueva prueba unitaria de Codeception. Seleccione Nuevo | Prueba PHP en el menú emergente de una de las vistas del proyecto y elija "Unidad Codeception" como biblioteca de pruebas.
La plantilla es editable. Además de las sentencias, expresiones y comentarios PHP, también puede utilizar las variables predefinidas que se enumeran a continuación. Cada variable se expande como una macro al valor correspondiente.
La directiva #parse permite incluir plantillas de la pestaña Inclusiones especificando el nombre completo de la plantilla requerida como parámetro entre comillas.
Variables predefinidas:
${TESTED_NAME}   Nombre de la clase probada
${TESTED_NAMESPACE}   Espacio de nombres de la clase probada
${NAME}   Nombre de la clase de prueba
${NAMESPACE}   Espacio de nombres de la clase de prueba
${FILE_NAME}   Nombre del archivo actual
${USER}   Nombre de inicio de sesión del sistema del usuario actual
${DATE}   Fecha actual del sistema
${TIME}   Hora actual del sistema
${YEAR}   Año actual
${MONTH}   Mes actual
${DAY}   Día del mes actual
${HOUR}   Hora actual
${MINUTE}   Minuto actual
${PRODUCT_NAME}   Nombre del IDE actual
${DS}   Símbolo de dólar, se evalúa como el carácter normal '$'.
${PROJECT_NAME}   Nombre del proyecto actual
