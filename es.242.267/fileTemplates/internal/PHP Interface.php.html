Esta es una plantilla integrada para una nueva interfaz PHP. Se usa cada vez que crea una nueva interfaz PHP seleccionando Nuevo | Clase PHP en el menú emergente de una de las vistas del proyecto.
La plantilla es editable. Además de sentencias, expresiones y comentarios PHP, puede usar las variables predefinidas que se listan a continuación. Cada variable se expande a su valor correspondiente, como una macro.
La directiva #parse permite incluir una plantilla de la pestaña Inclusiones especificando el nombre completo de la plantilla deseada como parámetro entre comillas.
Variables predefinidas:
${FILE_NAME}   Nombre del archivo actual
${USER}   Nombre de usuario del sistema
${DATE}   Fecha actual del sistema
${TIME}   Hora actual del sistema
${YEAR}   Año actual
${MONTH}   Mes actual
${DAY}   Día del mes actual
${HOUR}   Hora actual
${MINUTE}   Minuto actual
${PRODUCT_NAME}   Nombre del IDE actual
${DS}   Signo de dólar, se evalúa como el carácter '$'.
${PROJECT_NAME}   Nombre del proyecto actual
