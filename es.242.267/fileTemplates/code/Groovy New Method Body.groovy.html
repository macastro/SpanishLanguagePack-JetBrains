Esta es una plantilla incorporada que se utiliza para rellenar el cuerpo de un método Groovy cada vez que el programa lo genera (por ejemplo, al usar la acción de intención Crear método a partir del uso).
La plantilla es editable. Además de expresiones y comentarios Groovy, puede utilizar variables predefinidas que luego se expandirán a los valores correspondientes.
Las variables predefinidas reciben los siguientes valores:
${RETURN_TYPE}   Tipo de retorno del método creado.
${DEFAULT_RETURN_VALUE}   Valor devuelto por defecto por el método.
${METHOD_NAME}   Nombre del método creado.
${CLASS_NAME}   Nombre completo (cualificado) de la clase en la que se crea el método.
${SIMPLE_CLASS_NAME}   Nombre simple (no cualificado) de la clase que implementa el método.
