Esta plantilla integrada se utiliza para completar el cuerpo de un método cada vez que se genera, por ejemplo, mediante la acción de intención Crear método a partir del uso.
La plantilla es editable. Junto con las expresiones y comentarios de JavaScript, también puede utilizar variables predefinidas. Estas variables se reemplazarán con los valores correspondientes cuando se aplique la plantilla.
Las variables predefinidas tomarán los siguientes valores:
${IS_ASSIGNMENT_TARGET}   Devuelve 'true' si el método creado es un objetivo de asignación.
${IS_NEW_CALL}   Devuelve 'true' si el método creado está bajo una llamada 'new'.
${HAS_EXPECTED_TYPE}   Devuelve 'true' si el método creado está en una ubicación con tipo esperado.
${DEFAULT_RETURN_VALUE}   Valor predeterminado del tipo esperado para el método creado ('undefined' si no hay tipo esperado).
${METHOD_NAME}   Nombre del método creado.
${SEMICOLON}   Punto y coma o vacío, según las opciones de estilo de código.
Todas las variables predefinidas de la plantilla Encabezado de archivo (pestaña Incluye) también están disponibles (p. ej., ${USER}, ${DATE}, etc.).
