Plantilla incorporada para un constructor de clase PHP.
Las variables predefinidas toman los siguientes valores:
${PARAM_LIST}   Lista de parámetros del constructor, separados por comas, uno por cada propiedad seleccionada.
${PARAM_DOC}   Comentario PHPDoc de los parámetros.
Se genera como una serie de líneas '* @param tipo nombre'. Si no hay parámetros, se evalúa como contenido vacío.
${THROWS_DOC}   Comentario PHPDoc de las excepciones.
Se genera como una serie de líneas '* @throws tipo'. Si no se lanza ninguna excepción, se evalúa como contenido vacío.
${CLASS_NAME}   Nombre de la clase.
${BODY}   Cuerpo del constructor que contiene las asignaciones '$this->propiedad=propiedad;'.
