Crea el cuerpo de un método -description implementado cuando se genera para una clase que no tiene variables de instancia, por ejemplo, al llamar a la acción Código | Generar | -description.
La plantilla es editable. Junto con expresiones y comentarios de Objective-C, puede usar variables predefinidas que luego se expandirán a los valores correspondientes.
Las variables predefinidas tomarán los siguientes valores:
${RETURN_TYPE}   el tipo de retorno del método creado
${DEFAULT_RETURN_VALUE}   el valor retornado por el método por defecto
${CONTAINING_CLASS}   el nombre de la clase donde se implementa el método
Todas las variables predefinidas de la plantilla Encabezado de archivo C (la pestaña Incluye) también están disponibles (${USER}, ${DATE}, etc.)
