Crea el cuerpo de un método setter sobreescrito para una propiedad cuando se genera, por ejemplo, al llamar a la acción Código | Sobreescribir Método.
La plantilla es editable. Junto con las expresiones y comentarios de Objective-C, puede usar variables predefinidas que luego se expandirán a los valores correspondientes.
Las variables predefinidas tomarán los siguientes valores:
${CALL_SUPER}   una llamada al método super, para métodos void — [super nombreMétodo] con o sin parámetros; para otros métodos — return [super nombreMétodo] con o sin parámetros
${RETURN_TYPE}   el tipo de retorno del método creado
${DEFAULT_RETURN_VALUE}   el valor retornado por el método por defecto
${CONTAINING_CLASS}   el nombre de la clase donde se sobreescribe el método
${IVAR_IS_AVAILABLE}   "true" cuando la propiedad tiene una variable de instancia
${IVAR}   el nombre de la variable de instancia
${PARAM}   el nombre del parámetro del setter
${SETTER_SEMANTICS}   en modo ARC: "copyArc" cuando el atributo copy está establecido, "assign" en caso contrario
en modo no ARC: "copy" cuando el atributo copy está establecido, "retain" cuando el atributo retain está establecido, "assign" en caso contrario
Todas las variables predefinidas de la plantilla Encabezado de Archivo C (la pestaña Incluye) también están disponibles (${USER}, ${DATE}, etc.)
