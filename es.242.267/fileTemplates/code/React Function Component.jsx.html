Esta plantilla integrada se utiliza para crear componentes de función React, por ejemplo, a través de la refactorización Extraer Componente. Puede personalizar la plantilla, por ejemplo, definir componentes como funciones flecha o cambiar las definiciones de los tipos de prop. También puede usar variables predefinidas. Estas variables se reemplazarán con los valores correspondientes cuando se aplique la plantilla.
Las variables predefinidas tomarán los siguientes valores:
${NAME}   nombre del componente
${HAS_PROPS}   true si el componente tiene props
${COMPONENT_BODY}   cuerpo del componente
${PROP_TYPES}   definición de tipos de prop (usando el módulo prop-types)
