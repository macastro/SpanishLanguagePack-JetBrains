Rellena el parámetro catch al generarse, por ejemplo, cuando se llama a la función Código | Envolver con....
Esta plantilla incorporada es editable. Por ejemplo, puede hacer que el parámetro generado sea final, usando siempre un nombre predefinido independientemente del nombre cualificado de la excepción capturada.
Variables predefinidas reciben los siguientes valores:
${EXCEPTION}   Si el nombre cualificado termina en Exception, el nombre de la variable Exception se especifica como el parámetro catch derivado del tipo de excepción capturada e.
${EXCEPTION_TYPE}   El tipo del parámetro catch.
