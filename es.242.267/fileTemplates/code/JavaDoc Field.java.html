Define la plantilla que se utiliza para generar Javadoc para campos, por ejemplo, cuando se aplica la acción de intención Añadir Javadoc.

Además de las expresiones y comentarios de Java, puede utilizar variables predefinidas que se expandirán a sus valores correspondientes como macros.

Mediante la directiva #parse, puede incluir plantillas de la pestaña Inclusiones. Para incluir una plantilla, especifique el nombre completo de la plantilla entre comillas como parámetro (por ejemplo, #parse("Encabezado de archivo.java")).
Las variables predefinidas reciben los siguientes valores:
${ELEMENT_NAME}   Nombre del miembro de la clase que es el propietario del Javadoc generado.
${CONTAINING_CLASS}   Nombre de la clase contenedora.
${PACKAGE_NAME}   Nombre del paquete en el que se crea la nueva enumeración.
${NAME}   Nombre de la nueva enumeración especificado en el cuadro de diálogo Crear nueva clase.
${USER}   Nombre de inicio de sesión del sistema del usuario actual.
${DATE}   Fecha actual del sistema.
${TIME}   Hora actual del sistema.
${YEAR}   Año actual.
${MONTH}   Mes actual.
${MONTH_NAME_SHORT}   Las 3 primeras letras del nombre del mes actual (ene, feb, etc.).
${MONTH_NAME_FULL}   Nombre completo del mes actual (enero, febrero, etc.).
${DAY}   Día del mes actual.
${HOUR}   Hora actual.
${MINUTE}   Minuto actual.
${PROJECT_NAME}   Nombre del proyecto actual.
