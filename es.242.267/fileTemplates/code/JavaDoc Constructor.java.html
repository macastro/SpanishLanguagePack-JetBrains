Define la plantilla que se utiliza para generar Javadoc para constructores, por ejemplo, cuando se aplica la acción de intención Añadir Javadoc.

Además de las expresiones y comentarios de Java, puede usar variables predefinidas que luego se expandirán como macros a los valores correspondientes.

Mediante la directiva #parse, se pueden incluir plantillas de la pestaña Incluye. Para incluir una plantilla, especifique el nombre completo de la plantilla entre comillas como parámetro (por ejemplo, #parse("Encabezado de archivo.java")).
Las variables predefinidas reciben los siguientes valores:
${PARAMS}   Lista de nombres de parámetros
${TYPE_PARAMS}   Lista de nombres de parámetros de tipo
${THROWS}   Lista de excepciones lanzadas
${CONTAINING_CLASS}   Nombre de la clase contenedora
${PACKAGE_NAME}   Nombre del paquete en el que se crea la nueva enumeración
${NAME}   Nombre de la nueva enumeración especificado en el diálogo Crear nueva clase
${USER}   Nombre de inicio de sesión del sistema del usuario actual
${DATE}   Fecha actual del sistema
${TIME}   Hora actual del sistema
${YEAR}   Año actual
${MONTH}   Mes actual
${MONTH_NAME_SHORT}   Primeras 3 letras del nombre del mes actual (ene, feb, etc.)
${MONTH_NAME_FULL}   Nombre completo del mes actual (enero, febrero, etc.)
${DAY}   Día del mes actual
${HOUR}   Hora actual
${MINUTE}   Minuto actual
${PROJECT_NAME}   Nombre del proyecto actual
