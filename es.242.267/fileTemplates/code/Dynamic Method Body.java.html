Esta es la plantilla incorporada que se utiliza para rellenar el cuerpo del método cada vez que el programa genera un método dinámico FreeMarker.
La plantilla es editable. Además de las expresiones y comentarios de Java, también puede utilizar variables predefinidas que se expandirán a sus valores correspondientes.
Las variables predefinidas reciben los siguientes valores:
${RETURN_TYPE}   Tipo de retorno del método creado.
${DEFAULT_RETURN_VALUE}   Valor de retorno predeterminado del método.
${METHOD_NAME}   Nombre del método creado.
${CLASS_NAME}   Nombre completo (cualificado) de la clase donde se crea el método.
