Plantilla incorporada para un método setter de PHP. Se usa, por ejemplo, cuando se invoca la corrección rápida "Agregar getter y setter".
Variables predefinidas que reciben los siguientes valores:
${NAME}   Nombre del método en estilo CamelCase, sin el prefijo "set", con la primera letra en mayúscula. Por ejemplo, el nombre de propiedad '_my_property' se convierte en 'MyProperty'.
${CLASS_NAME}   Nombre de la clase contenedora.
${SCALAR_TYPE_HINT}   Contiene la pista de tipo escalar si el nivel de lenguaje es PHP 7.0 o superior.
${RETURN_TYPE}   Indica si el nivel de lenguaje es PHP 7.0 o superior, por lo que se puede especificar el tipo de retorno para la función/método.
${FIELD_NAME}   Nombre de la propiedad original, sin modificaciones.
${PARAM_NAME}   Nombre del parámetro del setter. Casi igual a ${FIELD_NAME}, pero se elimina el primer guion bajo, si lo hay. Por ejemplo, el nombre de propiedad '_myProperty' se convierte en 'myProperty'.
${TYPE_HINT}   Pista de tipo de la propiedad; cadena vacía si no se especifica.
${DS}   Signo de dólar, se evalúa como el carácter '$' normal.
