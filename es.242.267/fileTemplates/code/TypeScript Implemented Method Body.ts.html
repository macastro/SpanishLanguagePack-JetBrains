Esta plantilla integrada se utiliza para rellenar el cuerpo de un método cuando se genera, por ejemplo, mediante la acción de intención Crear método a partir del uso.
La plantilla es editable. Junto con las expresiones y comentarios de TypeScript, también puede usar variables predefinidas. Estas variables se reemplazarán con los valores correspondientes cuando se aplique la plantilla.
Las variables predefinidas tomarán los siguientes valores:
${RETURN_TYPE}   Tipo de retorno del método creado
${DEFAULT_RETURN_VALUE}   Valor devuelto por el método por defecto
${METHOD_NAME}   Nombre del método creado
${SEMICOLON}   Punto y coma o vacío, según las opciones de estilo de código
Todas las variables predefinidas de la plantilla Encabezado de archivo (pestaña Incluye) también están disponibles (ej.: ${USER}, ${DATE}, etc.)
