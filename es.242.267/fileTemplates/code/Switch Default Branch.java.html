Genera una rama default para una sentencia o expresión switch, utilizada por acciones de arreglos rápidos como Insertar rama 'default'.
Esta plantilla integrada es editable. Por ejemplo, puede cambiar la excepción que se lanza o llamar al framework de registro.
Las variables predefinidas reciben los siguientes valores:
${EXPRESSION}   Texto de la expresión del selector switch
${EXPRESSION_TYPE}   Tipo de la expresión del selector switch
