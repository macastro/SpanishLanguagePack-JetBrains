Plantilla incorporada para el cuerpo de un método PHP redefinido.
Variables predefinidas que reciben los siguientes valores:
${NAME}   Nombre del método.
${PARAM_LIST}   Una lista de parámetros separados por comas, similar a la del método original.
${RETURN}   Si el método original tiene una sentencia return, esta variable contiene la cadena 'return'; de lo contrario, contiene una cadena vacía.
