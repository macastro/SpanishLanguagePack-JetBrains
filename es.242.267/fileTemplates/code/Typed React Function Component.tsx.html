Esta plantilla integrada se utiliza para crear componentes de función React, por ejemplo, a través de la refactorización Extraer Componente. Puede personalizar esta plantilla, por ejemplo, definir componentes como funciones flecha o cambiar las definiciones de tipos de prop. También puede usar variables predefinidas. Estas variables se reemplazarán con los valores correspondientes cuando se aplique la plantilla. Esta plantilla se utiliza en TypeScript y Flow.
Las variables predefinidas tomarán los siguientes valores:
${NAME}   nombre del componente
${COMPONENT_BODY}   cuerpo del componente
${PROP_TYPES}   definición de tipo de props
