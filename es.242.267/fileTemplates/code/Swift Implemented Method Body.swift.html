Rellena el cuerpo del método cuando se generan métodos implementados (por ejemplo, usando la acción Código | Generar | Implementar métodos).
La plantilla es editable. Además de expresiones y comentarios Swift, puedes usar variables predefinidas que se expandirán a sus valores correspondientes.
Las variables predefinidas reciben los siguientes valores:
${RETURN_TYPE}   El tipo de retorno del método creado.
${DEFAULT_RETURN_VALUE}   El valor de retorno predeterminado sugerido (puede estar vacío).
${SIGNATURE}   La firma del método.
Todas las variables predefinidas en la plantilla Encabezado de archivo Swift (pestaña Incluye) también están disponibles (por ejemplo, ${USER}, ${DATE}, etc.).
