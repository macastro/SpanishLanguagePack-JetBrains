Reemplaza la concatenación de cadenas (ej. String s = "Hello " + name;) con una expresión de código Java internacionalizada (ej. String s = MessageFormat.format(​myResourceBundle.getString(​"hello.0"), name);).
Esta plantilla integrada es editable. Además de expresiones y comentarios Java, puedes usar variables predefinidas que se expandirán a sus valores correspondientes.
Las variables predefinidas reciben los siguientes valores:
${RESOURCE_BUNDLE}   Una expresión de tipo java.util.ResourceBundle, cuya instancia está disponible en este contexto.
${PARAMETERS}   Variables utilizadas en la concatenación de cadenas, normalmente pasadas como argumentos a la expresión internacionalizada.
${PROPERTY_KEY}   El nombre de la clave de propiedad definida en el archivo de propiedades correspondiente. Generalmente, es el valor utilizado como parámetro del método ResourceBundle.getString().
${PROPERTY_VALUE}   El valor de la propiedad definida en el archivo de propiedades correspondiente. Este es el valor literal de la cadena Java original.
