Plantilla incorporada para un comentario PHPDoc de una implementación de gancho de Drupal. Contiene un fragmento de código que se puede incluir en las plantillas de archivo (pestaña Plantillas) con la ayuda de la directiva #parse.
La plantilla es editable. Además de texto estático, código y comentarios, puede usar variables predefinidas, que se expandirán como macros a sus valores correspondientes.
Las variables predefinidas reciben los siguientes valores:
${HOOK_NAME}   Nombre del gancho.
