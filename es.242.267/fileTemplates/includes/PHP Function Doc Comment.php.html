Plantilla incorporada para comentarios PHPDoc de una función PHP. Contiene un fragmento de código que se puede incluir en las plantillas de archivo (pestaña Plantillas) con la ayuda de la directiva #parse.
La plantilla es editable. Además de texto estático, código y comentarios, puede usar variables predefinidas que se expandirán a los valores correspondientes como macros.
Las variables predefinidas reciben los siguientes valores:
${NAME}   Nombre de la función.
${TYPE_HINT}   Tipo de retorno de la función. Se evalúa como 'void' si no se infiere ningún tipo de retorno.
${PARAM_DOC}  

Comentarios PHPDoc para los parámetros.

Se genera como una serie de líneas '* @param tipo nombre'. Se evalúa como contenido vacío si no hay parámetros.

${THROWS_DOC}  

Comentarios de documentación para las excepciones.

Se genera como una serie de líneas '* @throws tipo'. Se evalúa como contenido vacío si no se lanza ninguna excepción.

${STATIC}   Toma el valor 'static' si la función/método es estático, o una cadena vacía en caso contrario. Ejemplo: 
      #if (${STATIC} == "static") * @static
      #end
${DS}   Signo de dólar, se evalúa como el carácter '$' normal.
${DATE}   Fecha actual del sistema.
${TIME}   Hora actual del sistema.
${YEAR}   Año actual.
${MONTH}   Mes actual.
${MONTH_NAME_SHORT}   Las 3 primeras letras del nombre del mes actual. Por ejemplo: Ene, Feb, etc.
${MONTH_NAME_FULL}   Nombre completo del mes actual. Por ejemplo: Enero, Febrero, etc.
${DAY}   Día del mes actual.
${HOUR}   Hora actual.
${MINUTE}   Minuto actual.
${SECOND}   Segundo actual.
${PROJECT_NAME}   Nombre del proyecto actual.
${USER}   Usuario actual.
${CARET}   Marca la posición a la que se debe mover el cursor de texto después de añadir el comentario.
