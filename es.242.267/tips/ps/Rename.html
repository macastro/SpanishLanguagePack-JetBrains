Refactorización de cambio de nombre

El nombre de las clases, métodos y variables se puede cambiar fácilmente actualizando todas las ubicaciones donde se utilizan de forma automática.

Coloca el cursor en el símbolo cuyo nombre deseas cambiar y presiona &shortcut:RenameElement; (Refactorizar | Cambiar nombre). Después de escribir el nuevo nombre, presiona &shortcut:EditorEnter;.

Presiona &shortcut:RenameElement; nuevamente para abrir el diálogo "Cambiar nombre". En este diálogo puedes buscar texto especificado en comentarios o cadenas.
