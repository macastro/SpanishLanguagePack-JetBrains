Selector de navegación

Para navegar entre los archivos abiertos y las ventanas de herramientas, utilice el selector &shortcut:Switcher;. Para mantener abierto el selector emergente, mantenga presionada la tecla Control. Para navegar, utilice las teclas Tab y Shift+Tab.

Para cerrar una pestaña del editor o esconder una ventana de herramientas, presione las teclas Delete o BackSpace.
