Formatear código

Para formatear código de acuerdo a la configuración del estilo de código actual, selecciona Código | Formatear código desde el menú principal o presiona &shortcut:ReformatCode; .

También puedes utilizar Código | Optimizar importaciones para ordenar afirmaciones de importación de acuerdo a la configuración del estilo de código del proyecto y eliminar importaciones no utilizadas.
