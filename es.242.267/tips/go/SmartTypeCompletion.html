Autocompletado de tipo de coincidencia

El autocompletado de "tipo de coincidencia" analiza el tipo esperado de la expresión completa y ayuda a encontrar métodos y variables aplicables en el contexto actual. Este autocompletado funciona después de la palabra clave return, en la asignación, en las listas de argumentos de las llamadas a métodos, etc.&shortcut:SmartTypeCompletion; ( Código | Autocompletado de código | Tipo de coincidencia ) se puede presionar para filtrar la lista de autocompletado.
