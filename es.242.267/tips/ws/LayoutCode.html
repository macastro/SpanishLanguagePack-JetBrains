Formatear el código

Para dar formato al código de acuerdo con las opciones de estilo de código actuales, seleccione "Código | Formatear código" en el menú principal o presione &shortcut:ReformatCode;.

También puede usar "Código | Optimizar importaciones" para ordenar las sentencias import de acuerdo a las opciones de estilo de código del proyecto y eliminar las importaciones que no se usan.
