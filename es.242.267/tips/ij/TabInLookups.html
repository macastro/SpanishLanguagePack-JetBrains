Autocompletar código

Al utilizar el autocompletado de código, puede pulsar &shortcut:EditorChooseLookupItemReplace; para aceptar el elemento seleccionado actualmente resaltado en la lista emergente.

A diferencia de la aceptación con &shortcut:EditorChooseLookupItem;, el resto del nombre a la derecha del cursor se sobrescribirá con el nombre seleccionado. Esto es especialmente útil cuando se reemplaza un nombre de método o variable por otro nombre de método o variable.
