Ver anotaciones Git Blame

Las anotaciones Git ofrecen información detallada sobre el origen de cada línea de código (clic con el botón derecho en el canal y seleccionar "Anotar con Git Blame ").

Haz clic con el botón derecho en la anotación y selecciona "Mostrar diferencia " para ver el diff entre la versión actual del archivo y la versión anterior.
