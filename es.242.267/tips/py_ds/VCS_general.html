Control de versiones

DataSpell se puede integrar fácilmente con sistemas de control de versiones como Git, Mercurial, Subversion y Perforce.

Controlar las versiones de las aplicaciones no sólo permite colaborar, sino que elimina el riesgo de almacenar toda la base de código localmente. Presiona &shortcut:ActivateVersionControlToolWindow; para ver el historial de cambios del proyecto y &shortcut:CheckinProject; para enviar los cambios locales al repositorio.
