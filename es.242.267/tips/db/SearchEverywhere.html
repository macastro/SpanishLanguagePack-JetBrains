Búsqueda en cualquier lugar

Presiona Shift 2 veces para buscar archivos, acciones, símbolos, elementos de la interfaz de usuario, ramas de Git y comentarios en todo el proyecto. Presiona Shift 2 veces nuevamente para buscar también elementos que no están en el proyecto.

Utiliza las pestañas para acotar los resultados de la búsqueda o utiliza directamente el acceso directo que necesites (para objetos de base de datos, &shortcut:GotoClass; para archivos, &shortcut:GotoFile; para símbolos, &shortcut:GotoSymbol; para acciones, &shortcut:GotoAction;).
