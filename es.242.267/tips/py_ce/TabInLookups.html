Completado de código

Al usar el completado de código presionando &shortcut:EditorChooseLookupItemReplace; se puede aceptar la selección actualmente resaltada en la lista desplegable.

A diferencia de aceptarla con &shortcut:EditorChooseLookupItem;, esta sobrescribirá el resto del nombre a la derecha del cursor con el nombre seleccionado. Esto es especialmente útil para reemplazar un nombre de método o variable por otro.
