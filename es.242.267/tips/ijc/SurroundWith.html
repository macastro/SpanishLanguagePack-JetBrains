Rodear fragmento de código

Para rodear un bloque de código rápidamente con un constructo válido, selecciona el bloque de código de destino en el editor y presiona &shortcut:SurroundWith; (Code | Surround With).

La lista de opciones o envolturas disponibles varía según el contexto y el lenguaje. Por ejemplo, puedes realizar acciones como rodear un bloque html con etiquetas <div>.
