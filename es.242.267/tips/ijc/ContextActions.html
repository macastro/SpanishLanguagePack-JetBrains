Mejora el código con acciones contextuales

Al presionar &shortcut:ShowIntentionActions; en el editor, puedes corregir errores y advertencias resaltadas, mejorar la estructura del código u optimizarlo. Para algunas acciones contextuales, puedes presionar &shortcut:QuickJavaDoc; (Ver | Documentación rápida) para obtener una vista previa o una descripción de la acción.
